# 每個人都有自己的因果

這週有兩則老故事一直在我心裡打轉。它們簡單得近乎童稚，卻有一種讓我反覆咀嚼的重量。

第一則是山頂上的老君廟。廟裡的神像日日受香火跪拜，廟前的石獅卻風吹日曬、還被鳥糞砸頭。石獅忍不住抱怨：「你我同是一塊石頭、同一位工匠、同一天完工，為何你受供奉，我卻在外頭受苦？」神像淡淡地回答：「當年鑿子第一次落在你身上，你就喊石頭太硬、天氣太熱。你千刀未完就崩了角，匠人只好把你粗粗修圓。而我忍著一道道刻痕，心想挨得過去就是一尊像。你不是輸在出身，是輸在每一個選擇上。」

第二則是井底的青蛙。牠一生住在井底，抬頭只見一小塊天，成天怨嘆世界為何這麼小。一隻大雁飛過井口，邀牠出來看看山與海。青蛙費盡力氣跳出井口，被遼闊的天地晃花了眼，興奮地跳進旁邊的稻田——沒幾天稻田乾旱，牠幾乎渴死，只好奮力跳回井裡，喘著氣說：「原來你有你的廣闊，我有我的井水。」大雁繼續南飛，青蛙繼續在井底過冬。

讓我在意的是，這兩則故事都沒有把責任推給命運。石獅不是被石頭注定的，而是被牠面對每一次鑿擊的態度，一刀一刀塑成的；青蛙也沒有做錯什麼，牠本就屬於那口井，而那口井在廣闊天地會渴死牠的時候，恰恰保住了牠的命。

我想，我們常常花很大的力氣去比較別人的路——羨慕那受供奉的神像、那能飛越大海的雁。我們只看見別人「到了哪裡」，卻忘了把他們送到那裡的千萬個微小抉擇，也忘了每一種天性各有其適得其所的位置：他的適合他，我的適合我。

因果，其實不是寫在星象裡的懲罰，反倒更像是沉積：每一個念頭、每一次反應、每一回我們是退縮還是承受，都會緩緩落下，慢慢成為我們腳下所站的那片地。今天的我，是無數個當時不以為意的昨天，一層層堆疊起來的。

所以我試著同時對兩件事溫柔一點。對鑿子溫柔一點——把困難當成雕琢，而不是傷害，去承受那些從前會讓我崩角的刻痕；也對自己的那口井溫柔一點——別再怨恨它的井壁，並且明白，滋養我的水，未必是滋養別人的那一泓。

每個人都有自己的因果，也都有自己該去的歸處。石獅蹲著，大雁南飛，香火依舊。這些都不是不公平，而只是當初種下什麼、如今便收成什麼罷了。而值得慶幸的是——就在今天，我們都還來得及，種下一點更善良的東西。
