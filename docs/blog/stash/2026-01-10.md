# Getting Started with VitePress

VitePress is a static site generator powered by Vite and Vue 3. It is designed to be fast, minimal, and developer-friendly — perfect for building documentation sites, blogs, and knowledge bases.

## Why VitePress?

Unlike heavier solutions, VitePress compiles your Markdown into fully static HTML at build time. The result is a site that loads instantly, ranks well in search engines, and costs nothing to host on platforms like GitHub Pages, Vercel, or Netlify.

## Installation

Start by creating a new project folder and initializing it:

```bash
mkdir my-docs && cd my-docs
npm init -y
npm install -D vitepress
```

Then run the setup wizard:

```bash
npx vitepress init
```

Follow the prompts and you will have a working docs site in under two minutes.

## Writing Your First Page

Create a Markdown file anywhere inside your `docs/` folder:

```md
# Hello World

Welcome to my first VitePress page.
```

Run the dev server to see it live:

```bash
npm run docs:dev
```

## Next Steps

- Configure your sidebar and navbar in `config.ts`
- Add dark mode, search, and social links
- Deploy to Vercel with zero configuration
