# ASA 麻醉風險分級

**ASA Anesthesia Risk Classification**

## 概述

**ASA 身體狀況分類系統**（ASA Physical Status Classification System）由美國麻醉醫師學會（American Society of Anesthesiologists）於 1941 年首次提出，並歷經多次修訂（最近一次重大更新為 2020 年），是目前全球麻醉科及外科最廣泛使用的**術前麻醉風險評估工具**。ASA 分級系統以患者的整體健康狀況及系統性疾病的嚴重程度為依據，將患者分為 1–6 等級，為麻醉計劃制定、術前風險告知及研究標準化提供共通語言。

## 分級系統

### 管路依賴（Tube Dependent，Level 1–3）

ASA 分為六個等級，並可加上「E」後綴表示緊急手術：

| 等級    | 說明               | 臨床範例                           | 手術死亡率 |
| ------- | ------------------ | ---------------------------------- | ---------- |
| ASA I   | 正常健康患者       | 健康成人，無任何疾病               | < 0.1%     |
| ASA II  | 輕度系統性疾病     | 控制良好的高血壓、糖尿病、輕度肥胖 | 0.1–0.2%   |
| ASA III | 重度系統性疾病     | 控制不良的糖尿病、COPD、病態性肥胖 | 1.8–4.0%   |
| ASA IV  | 危及生命的系統疾病 | 近期心肌梗塞、嚴重心衰、敗血症     | 7.8–23.0%  |
| ASA V   | 瀕死患者           | 腹主動脈瘤破裂、嚴重顱內出血       | 9.4–50.7%  |
| ASA VI  | 腦死器官捐贈       | 腦死確認、器官捐贈手術             | N/A        |

## 緊急手術加註（E modifier）

若手術為**緊急性質**，在 ASA 等級後加註「**E**」（Emergency），例如：

- **ASA 2E**：有輕度系統性疾病的患者進行緊急手術
- **ASA 3E**：有重度系統性疾病的患者進行緊急手術

> **緊急手術通常使手術死亡率提高 2–4 倍。** 緊急程度的定義：手術若延遲將導致嚴重傷害或死亡風險顯著增加。

## 各等級詳細說明

### ASA I — 正常健康

完全健康的患者，無任何系統性疾病、器官異常或心理精神疾患。不吸菸或偶爾吸菸（不依賴），不飲酒或輕度社交性飲酒，BMI < 30。

**臨床範例**：無任何慢性病的年輕健康成人接受擇期手術。

### ASA II — 輕度系統性疾病

有輕度系統性疾病，但對日常生活**無實質功能限制**。包括：

- 控制良好的高血壓（血壓穩定，藥物控制佳）
- 控制良好的第 2 型糖尿病（HbA1c < 8%）
- 輕度肺部疾病（不影響日常活動）
- BMI 30–40（肥胖但非病態性）
- 懷孕
- 目前吸菸者
- 社交性飲酒
- 控制良好的癲癇

### ASA III — 重度系統性疾病

有**實質性功能限制**的嚴重系統性疾病，但尚未直接危及生命。包括：

- 控制不良的糖尿病或高血壓
- 慢性阻塞性肺病（COPD）
- 病態性肥胖（BMI ≥ 40）
- 活動性肝炎、酒精依賴
- 植入心律調節器
- 中等程度腎功能不全
- 早產兒（矯正胎齡未滿 60 週）
- 距今超過 3 個月的心肌梗塞、腦中風、TIA 或冠心病病史

### ASA IV — 危及生命的系統性疾病

有**持續性、直接危及生命**的嚴重疾病。包括：

- 近期（3 個月內）急性心肌梗塞、腦中風或 TIA
- 嚴重冠狀動脈疾病或持續性心肌缺血
- 嚴重心臟瓣膜疾病或嚴重心臟衰竭（EF < 25%）
- 敗血症
- DIC（瀰漫性血管內凝血）
- ARD（急性呼吸窘迫症候群）
- 末期腎臟病（未規律接受透析）

### ASA V — 瀕死患者

不接受手術則預期無法存活，手術為**最後的存活機會**。包括：

- 腹主動脈瘤破裂
- 大量顱內出血伴腦疝氣
- 廣泛性腸繫膜缺血
- 嚴重多重創傷
- 心因性休克

### ASA VI — 腦死器官捐贈

已透過標準程序確認腦死，進行器官捐贈的手術。

## 臨床應用

ASA 分級廣泛應用於以下情境：

- **術前麻醉評估**：麻醉科醫師制定麻醉計劃的標準依據
- **術前風險告知**：向患者及家屬說明手術風險的共同語言
- **手術適應症判斷**：ASA IV–V 的患者是否適合擇期手術需謹慎評估
- **研究標準化**：多中心研究的患者基線資料標準化
- **醫療品質監測**：術後併發症與死亡率的風險調整基準
- **醫療資源分配**：高風險患者的術後加護病房規劃

## ASA 與手術風險的關係

ASA 等級與術後 30 天死亡率有顯著相關，並受以下因素調整：

| 調整因素      | 影響                                 |
| ------------- | ------------------------------------ |
| 緊急手術（E） | 死亡率提高 2–4 倍                    |
| 手術部位      | 心臟、胸腔、神經手術風險高於四肢手術 |
| 麻醉方式      | 全身麻醉 vs. 區域麻醉                |
| 手術時間      | 超過 3 小時風險顯著增加              |
| 術者經驗      | 中心量與術者經驗影響結果             |

## ASA 分級的限制

ASA 是主觀評估工具，存在以下限制：

- **評分者間變異性**：不同醫師對同一患者的評分可能有出入，特別是 ASA II 與 ASA III 之間
- **未納入手術因素**：ASA 僅評估患者健康狀況，不包含手術本身的風險（手術部位、複雜度）
- **不反映特定器官功能**：如肺功能、心臟射出分率等客觀數據
- **動態變化**：急性病情變化可能使 ASA 等級在短時間內改變

## 注意事項

::: warning ASA 等級需由麻醉科醫師確認
ASA 分級應由**麻醉科醫師**在術前評估後正式確認，外科醫師的初步評估可作為參考，但最終分級以麻醉科醫師判斷為準。
:::

::: warning ASA IV–V 患者的手術決策需多科討論
ASA IV 或 ASA V 的患者進行非緊急手術前，應召開**多科討論（MDT）**，由外科、麻醉科、心臟科、重症醫學科等相關科室共同評估手術風險效益比，確保患者及家屬充分了解風險後再做決定。
:::

本計算器僅供術前風險評估參考與教育目的，不能取代麻醉科醫師的完整術前評估。手術風險評估應由合格的麻醉科及外科醫師根據患者的完整病史、體格檢查及相關檢查結果綜合判定。
