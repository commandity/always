# ASA 麻醉風險分級

**ASA Physical Status Classification · 美國麻醉醫師學會身體狀態分級**

<AsaCalculator />

## ASA 分級系統

**ASA Physical Status Classification**（ASA-PS，美國麻醉醫師學會 1941，最近修訂 2020）依患者**術前整體健康狀態**分為 I–VI 六級，是全球麻醉與外科最廣泛使用的術前風險評估共通語言。分級愈高，圍術期併發症與死亡風險愈高。

| 等級    | 定義                   | 臨床範例                             | 圍術期死亡率參考 |
| ------- | ---------------------- | ------------------------------------ | ---------------- |
| ASA I   | 正常健康患者           | 無系統性疾病、不吸菸、BMI < 30       | < 0.1%           |
| ASA II  | 輕度系統性疾病         | 控制良好的高血壓/糖尿病、吸菸、懷孕   | 0.1–0.2%         |
| ASA III | 重度系統性疾病         | 控制不良糖尿病、COPD、BMI ≥ 40       | 1.8–4.0%         |
| ASA IV  | 持續危及生命的系統疾病 | 近 3 月心肌梗塞、EF < 25%、敗血症    | 7.8–23.0%        |
| ASA V   | 瀕死患者               | 主動脈瘤破裂、大量顱內出血、心因休克 | 9.4–50.7%        |
| ASA VI  | 腦死器官捐贈           | 腦死確認、器官捐贈手術               | N/A              |

### 分級意義與麻醉考量

| 等級      | 風險層級 | 意義與麻醉處置                                             |
| --------- | -------- | ---------------------------------------------------------- |
| I–II      | 低風險   | 一般麻醉風險低，常規術前評估即可                           |
| III       | 中度風險 | 需優化共病、加強監測；視情況安排術前照會                   |
| IV        | 高風險   | 圍術期風險顯著上升，需完整評估、風險告知與加護後送準備     |
| V         | 極高風險 | 手術為挽救生命的最後手段，須充分知情同意                   |
| VI        | 特殊     | 僅用於腦死器官捐贈手術                                     |

> **ASA 分級是主觀臨床判斷，而非精確風險計算**：它反映患者的「基礎健康狀態」，不納入手術種類、術式難度或年齡本身。分級具一定觀察者間差異，應由麻醉醫師於術前綜合病史與理學檢查判定。分級愈高，死亡率呈**指數上升**，III 級起即需個別化的麻醉與監測計畫。

## 緊急手術加註（E modifier）

若手術為**緊急性質**（延遲將顯著增加傷害或死亡風險），於等級後加註「**E**」，例如 **ASA IIIE**。

| 標示    | 手術性質 | 說明                             |
| ------- | -------- | -------------------------------- |
| （無 E） | 擇期手術 | 可安排、非時間緊迫               |
| E       | 緊急手術 | 延遲將導致嚴重傷害或死亡風險上升 |

> **緊急手術通常使圍術期死亡率提高約 2–4 倍。** 相同疾病嚴重度下，加註「E」代表風險層級更高；例如 ASA IIIE 的風險高於 ASA III。

## 注意事項

::: warning 分級不等於風險計算
ASA-PS 為術前健康狀態的分級，非個人化的死亡率預測；實際圍術期風險尚需結合手術類型、術式與麻醉方式綜合評估。
:::

::: info 免責聲明
本工具僅供臨床輔助與教學參考，最終麻醉風險評估與分級應由麻醉專科醫師負責判定。
:::
