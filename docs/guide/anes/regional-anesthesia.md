## 1. 頸部 Head & Neck

### 1.1 浅頸神經叢阻斷 Superficial Cervical Plexus Block (SCPB)

**適應症 Indications**

- 頸動脈內膜切除術（Carotid endarterectomy）
- 頸部淺表手術（淋巴結切片、甲狀腺手術輔助）
- 頸部皮膚手術

**解剖 Anatomy**

- 浅頸神經叢由 C1–C4 前支腹側支組成
- 穿出胸鎖乳突肌（SCM）後緣中點（Erb's point）
- 分布於頸前、頸側、耳廓、鎖骨上皮膚

**體表定位 Landmarks**

| 標記             | 描述                     |
| ---------------- | ------------------------ |
| **Erb's point**  | SCM 後緣中點，C2–C3 水平 |
| **SCM 後緣**     | 沿後緣觸摸               |
| **乳突 Mastoid** | SCM 起始端上方           |

**超音波引導 Ultrasound-guided**

| 項目     | 內容                                                      |
| -------- | --------------------------------------------------------- |
| **探頭** | 高頻線性 10–15 MHz                                        |
| **位置** | SCM 後緣中點，橫切面                                      |
| **目標** | SCM 後緣深筋膜淺層（superficial to investing layer）      |
| **進針** | 從後方向前方進針（lateral-to-medial），平面內（in-plane） |
| **藥物** | 0.25% Bupivacaine 或 0.5% Ropivacaine 15–20 mL            |
| **確認** | 藥物擴散於 SCM 後緣皮下，不應見深筋膜深層擴散             |

**併發症 Complications**

- 局部血腫（頸動脈、頸靜脈穿刺）
- 藥物誤入血管
- 頸交感神經阻斷（Horner's syndrome）
- 迷走神經阻斷（心搏過緩）
- 膈神經阻斷（同側膈肌麻痺）
- 氣胸（極少見）

---

### 1.2 深頸神經叢阻斷 Deep Cervical Plexus Block (DCPB)

**適應症 Indications**

- 頸動脈內膜切除術（與 SCPB 聯合或單獨使用）
- 深部頸部手術

**解剖 Anatomy**

- C1–C4 前支，位於椎旁橫突溝內
- 位於中斜角肌（posterior）與前斜角肌（anterior）之間
- 深於 SCPB，覆蓋頸深筋膜

**超音波引導 Ultrasound-guided**

| 項目     | 內容                                                   |
| -------- | ------------------------------------------------------ |
| **探頭** | 高頻線性 10–15 MHz                                     |
| **位置** | SCM 後緣，C2–C4 水平橫切面                             |
| **目標** | 中斜角肌深面、橫突溝內（prevertebral fascia 下）       |
| **進針** | 後外側向內進針（posterior-to-anterior），平面內        |
| **藥物** | 0.25% Bupivacaine 或 0.5% Ropivacaine 5–8 mL per level |
| **確認** | 藥物擴散至中斜角肌深面，包圍神經根                     |

**併發症 Complications**

- 藥物誤入椎動脈（ seizures 立即發生）
- 藥物誤入蛛網膜下腔或硬膜外腔
- 膈神經阻斷（100% 單側使用時發生率高）
- 頸交感神經阻斷
- 脊髓缺血（大量藥物）
- 喉返神經阻斷

---

### 1.3 枕大/枕小神經阻斷 Greater/Lesser Occipital Nerve Block

**適應症 Indications**

- 後頭部手術
- 枕部神經痛
- 緊張型頭痛

**超音波引導 Ultrasound-guided**

| 項目     | 內容                                                          |
| -------- | ------------------------------------------------------------- |
| **探頭** | 高頻線性 10–15 MHz                                            |
| **位置** | 乳突與枕外隆凸連線中點後方                                    |
| **目標** | 枕動脈旁（ Greater occipital nerve 與 occipital artery 伴行） |
| **進針** | 從外側進針，平面內                                            |
| **藥物** | 1% Lidocaine 或 0.25% Bupivacaine 3–5 mL                      |

---

### 1.4 顎神經阻斷 Infraorbital Nerve Block

**適應症 Indications**

- 上唇、上顎、鼻側皮膚手術
- 上顎前牙拔牙

**解剖**

- 顎神經（V2 分支）從 infraorbital foramen 穿出
- Foramen 位於眶下緣下方約 0.5–1 cm

**超音波引導 Ultrasound-guided**

| 項目     | 內容                                |
| -------- | ----------------------------------- |
| **探頭** | 高頻線性 10–15 MHz                  |
| **位置** | 眶下緣下方，infraorbital foramen 旁 |
| **目標** | Foramen 前方，maxilla 表面          |
| **進針** | 從外側向內進針                      |
| **藥物** | 1% Lidocaine 2–3 mL                 |

**注意**：避免注入眶內，不要穿入 foramen

## 2. 上肢 Upper Extremity

### 2.1 臂神經叢阻斷 Brachial Plexus Blocks

#### 2.1.1 肩胛上神經阻斷 Suprascapular Nerve Block

**適應症 Indications**

- 肩關節手術（與 interscalene 聯合可減少膈神經麻痺）
- 肩關節疼痛
- 肩袖修補術後 analgesia

**解剖**

- 起自 C5–C6，從上幹分出
- 經肩胛上切跡（suprascapular notch）進入肩胛上窩
- 支配岡上肌、岡下肌、肩關節囊

**超音波引導 Ultrasound-guided**

| 項目     | 內容                                          |
| -------- | --------------------------------------------- |
| **探頭** | 高頻線性 10–15 MHz                            |
| **位置** | 肩胛上切跡水平（supraspinatus fossa）         |
| **目標** | 肩胛上切跡內，suprascapular artery 旁         |
| **進針** | 從外側向內進針，平面內                        |
| **藥物** | 0.25% Bupivacaine 或 0.5% Ropivacaine 5–10 mL |
| **確認** | 藥物擴散包圍 notch 內神經                     |

**併發症**

- 藥物誤入肩胛上動脈
- 氣胸（針尖太深）
- 膈神經阻斷（若藥物向上擴散）

---

#### 2.1.2 肋鎖臂神經叢阻斷 Interscalene Brachial Plexus Block (ISB)

**適應症 Indications**

- 肩關節手術（rotator cuff repair、肩關節置換、肩關節脫臼復位）
- 肱骨近端骨折
- 肩關節疼痛管理

**解剖 Anatomy**

- C5–C6（有時 C4、C7）神經根
- 位於前斜角肌與中斜角肌間溝內（interscalene groove）
- 肩胛上神經（C5–C6）、腋神經（C5–C6）同由 ISB 涵蓋

**體表定位 Landmarks**

| 標記                    | 描述                                |
| ----------------------- | ----------------------------------- |
| **Erb's point**         | SCM 後緣中點                        |
| **Interscalene groove** | 沿 SCM 後緣深觸，患者轉頭對側可觸及 |
| **C6 tubercle**         | 水平位置（Chassaignac's tubercle）  |

**超音波引導 Ultrasound-guided**

| 項目     | 內容                                                      |
| -------- | --------------------------------------------------------- |
| **探頭** | 高頻線性 10–15 MHz                                        |
| **位置** | 環狀軟骨水平（C6），SCM 後方                              |
| **目標** | 前斜角肌與中斜角肌間，可見 C5–C6 神經根（圆形低回音結構） |
| **進針** | 後外側向內進針（lateral-to-medial），平面內               |
| **藥物** | 0.25% Bupivacaine 或 0.5% Ropivacaine 15–20 mL            |
| **確認** | 藥物擴散包圍神經根，fill 位於兩斜角肌間                   |

**常用配方**

- 0.25% Bupivacaine 20 mL
- 0.5% Ropivacaine 15–20 mL
- 1% Lidocaine + 0.25% Bupivacaine 混合 20 mL（加速起始）

**併發症 Complications**

- **膈神經阻斷**：單側 ISB 發生率 100%（高位注射時），膈肌麻痺→肺功能下降 25%
- **Horner's syndrome**：頸交感神經阻斷
- **喉返神經阻斷**：聲音沙啞
- **椎動脈內注射**： seizures
- **蛛網膜下腔/硬膜外腔注射**：脊髓麻醉或硬膜外擴散
- **氣胸**
- **全脊髓麻醉**（total spinal anesthesia）

**改良技術：Selective ISB / Nerve-sparing ISB**

- 低容量（5 mL）精準注射至目標神經根
- 減少膈神經阻斷發生率

---

#### 2.1.3 鎖骨上阻斷 Supraclavicular Brachial Plexus Block

**適應症 Indications**

- 前臂及手部手術（"spinal of the arm"）
- 肘部以下手術
- 肱骨中遠端骨折
- 手部外傷

**解剖 Anatomy**

- 臂神經叢的上、中、下幹（trunks/divisions）
- 位於鎖骨上方，第一肋骨外側
- 與鎖骨下動脈緊鄰（神經在動脈外上方）

**超音波引導 Ultrasound-guided**

| 項目     | 內容                                           |
| -------- | ---------------------------------------------- |
| **探頭** | 高頻線性 10–15 MHz                             |
| **位置** | 鎖骨上方窝，鎖骨中點上方                       |
| **目標** | 鎖骨下動脈外上方，第一肋骨表面                 |
| **進針** | 從外側向內進針（lateral-to-medial），平面內    |
| **藥物** | 0.25% Bupivacaine 或 0.5% Ropivacaine 20–25 mL |
| **確認** | 藥物擴散包圍動脈外上方的神經束（呈"葡萄串"狀） |

**關鍵解剖**

- 鎖骨下動脈為主要標記
- 第一肋骨為深層標記
- 胸膜頂位於第一肋骨內側（氣胸風險）

**併發症 Complications**

- **氣胸**：最重要併發症，因針尖接近胸膜頂
- 藥物誤入鎖骨下動脈
- 膈神經阻斷（30–50%）
- 臂叢神經損傷
- 血腫

---

#### 2.1.4 鎖骨下阻斷 Infraclavicular Brachial Plexus Block

**適應症 Indications**

- 前臂及手部手術
- 肘部以下手術
- 肘關節手術
- 適合門診手術（較 supraclavicular 安全）

**解剖 Anatomy**

- 臂神經叢的三束（lateral, medial, posterior cords）
- 位於鎖骨下，胸大肌深面
- 腋動脈為主要標記

**超音波引導 Ultrasound-guided**

| 項目     | 內容                                             |
| -------- | ------------------------------------------------ |
| **探頭** | 高頻線性 10–15 MHz                               |
| **位置** | 鎖骨下方，coracoid process 水平                  |
| **目標** | 腋動脈周圍（三束呈倒三角形包圍動脈）             |
| **進針** | 從外側向內進針（lateral-to-medial），平面內      |
| **藥物** | 0.25% Bupivacaine 或 0.5% Ropivacaine 20–30 mL   |
| **確認** | 藥物擴散包圍腋動脈，呈"甜甜圈"徵象（donut sign） |

**併發症**

- 氣胸（較 supraclavicular 低但仍有可能）
- 藥物誤入腋動脈
- 膈神經阻斷（較 supraclavicular 低，約 30%）
- 血腫

---

#### 2.1.5 腋神經阻斷 Axillary Brachial Plexus Block

**適應症 Indications**

- 前臂及手部手術
- 桓骨/尺骨骨折
- 手部外傷
- 門診手術首選（安全性最高）

**解剖 Anatomy**

- 臂神經叢的終末分支（正中神經、尺神經、橈神經、肌皮神經）
- 位於腋窩，腋動脈周圍
- 被腋鞘（axillary sheath）包圍

**超音波引導 Ultrasound-guided**

| 項目     | 內容                                                 |
| -------- | ---------------------------------------------------- |
| **探頭** | 高頻線性 10–15 MHz                                   |
| **位置** | 腋窩，腋動脈最高搏動點                               |
| **目標** | 腋動脈周圍四個象限內的神經                           |
| **進針** | 多點注射（multi-injection），平面內                  |
| **藥物** | 0.25% Bupivacaine 或 0.5% Ropivacaine 20–30 mL total |

**神經分布**

| 神經         | 位置            | 支配                 |
| ------------ | --------------- | -------------------- |
| **正中神經** | 腋動脈前方/外側 | 前臂屈肌、魚際肌     |
| **尺神經**   | 腋動脈內下方    | 前臂尺側屈肌、手內肌 |
| **橈神經**   | 腋動脈後方      | 伸肌群               |
| **肌皮神經** | 腋鞘外，喙突旁  | 前臂外側皮膚         |

**注意**：肌皮神經常位於腋鞘外，需單獨阻斷

**併發症**

- 藥物誤入腋動脈/腋靜脈
- 血腫
- 神經損傷（發生率低）
- 腋動脈瘤/假性動脈瘤（罕見）

## 3. 胸壁與軀幹 Chest Wall & Trunk

### 3.1 椎旁阻斷 Paravertebral Block (PVB)

**適應症 Indications**

- 乳房手術（mastectomy、lumpectomy）
- 肋骨骨折
- 胸腔手術（VATS、thoracotomy）
- 帶狀疱疹疼痛
- 腹壁手術（單側）

**解剖 Anatomy**

- 椎體兩側的楔形空間（paravertebral space）
- 前界：壁層胸膜
- 後界：肋間膜（ costo-transverse ligament）
- 內側界：椎體及椎間盤
- 外側界：肋間後膜
- 包含：脊神經前支、後支、交通支、交感神經鏈、肋間動靜脈

**超音波引導 Ultrasound-guided**

| 項目     | 內容                                                              |
| -------- | ----------------------------------------------------------------- |
| **探頭** | 高頻線性 10–15 MHz（深部可用低頻曲面）                            |
| **位置** | 後正中線旁 2–3 cm，目標肋骨水平                                   |
| **目標** | 肋橫突韌帶深面，paravertebral space                               |
| **進針** | 從外側向內進針（lateral-to-medial），平面內                       |
| **藥物** | 0.25% Bupivacaine 或 0.5% Ropivacaine 3–5 mL/level                |
| **確認** | 藥物擴散使胸膜下移（lifting sign），可見 paravertebral space 擴大 |

**技術要點**

- 穿過 costotransverse ligament 後有"pop"感
- 回抽確認無血液/腦脊液
- 可在超音波下觀察藥物擴散

**併發症 Complications**

- 氣胸（1–2%）
- 硬膜外腔擴散（可能導致雙側阻斷）
- 蛛網膜下腔注射
- 血管內注射
- 低血壓（交感神經阻斷）

---

### 3.2 肋間神經阻斷 Intercostal Nerve Block

**適應症 Indications**

- 肋骨骨折
- 胸腔手術後止痛
- 帶狀疱疹
- 胸壁手術

**解剖 Anatomy**

- 肋間神經（T1–T11）位於肋骨下緣的肋溝內（costal groove）
- 在肋間內肌與肋間最內肌之間
- 伴行肋間動靜脈

**超音波引導 Ultrasound-guided**

| 項目     | 內容                                               |
| -------- | -------------------------------------------------- |
| **探頭** | 高頻線性 10–15 MHz                                 |
| **位置** | 目標肋骨水平，後外側                               |
| **目標** | 肋骨下緣深面（costal groove 內）                   |
| **進針** | 從內側向外側（medial-to-lateral），平面內          |
| **藥物** | 0.25% Bupivacaine 或 0.5% Ropivacaine 3–5 mL/level |
| **確認** | 藥物擴散於肋骨下緣深面                             |

**注意**

- 每次最多阻斷 3–4 個肋間（藥物毒性風險）
- 單側使用（避免雙側阻斷影響呼吸功能）
- 藥物吸收快速（肋間血管豐富）

**併發症**

- 氣胸
- 血管內注射（肋間動脈/靜脈）
- 藥物毒性（多節段阻斷時）
- 局部血腫

---

### 3.3 胸肌神經阻斷 Pectoral Nerve Block (PECs I & II)

**適應症 Indications**

- 乳房手術（augmentation、mastectomy、reconstruction）
- 腋窩淋巴結清創
- 植入物相關手術
- 胸壁前外側手術

**解剖 Anatomy**

**PECs I（Pectoral Nerve Block Type I）**

- 胸外側神經（lateral pectoral nerve, C5–C7）
- 胸內側神經（medial pectoral nerve, C8–T1）
- 位於胸大肌深面，胸小肌淺面

**PECs II（加做）**

- 胸外側神經（lateral pectoral nerve）
- 肋間臂神經（intercostobrachial nerve, T2）
- 位於胸小肌深面，前鋸肌淺面

**超音波引導 Ultrasound-guided**

| 項目                | 內容                                           |
| ------------------- | ---------------------------------------------- |
| **探頭**            | 高頻線性 10–15 MHz                             |
| **位置（PECs I）**  | 鎖骨下方，胸大肌與胸小肌之間                   |
| **目標（PECs I）**  | 胸大肌深面、胸小肌淺面                         |
| **藥物（PECs I）**  | 0.25% Bupivacaine 或 0.5% Ropivacaine 10–15 mL |
| **位置（PECs II）** | 胸小肌深面，前鋸肌淺面                         |
| **目標（PECs II）** | 胸小肌與前鋸肌之間                             |
| **藥物（PECs II）** | 0.25% Bupivacaine 或 0.5% Ropivacaine 10–15 mL |

**併發症**

- 血管內注射
- 氣胸（罕見）
- 局部血腫

---

### 3.4 前鋸肌平面阻斷 Serratus Anterior Plane Block (SAPB)

**適應症 Indications**

- 肋骨骨折
- 連枷胸（flail chest）
- 乳房手術
- 胸腔手術（VATS）
- 腋窩手術
- 帶狀疱疹

**解剖 Anatomy**

- 前鋸肌（serratus anterior）位於胸壁外側
- 深面為肋骨與肋間肌
- 肋間神經的外側皮支（lateral cutaneous branch）穿過前鋸肌

**超音波引導 Ultrasound-guided**

| 項目     | 內容                                            |
| -------- | ----------------------------------------------- |
| **探頭** | 高頻線性 10–15 MHz                              |
| **位置** | 腋中線，第 4–5 肋骨水平                         |
| **目標** | 前鋸肌深面（deep plane）或淺面（shallow plane） |
| **進針** | 從前向後（anterior-to-posterior），平面內       |
| **藥物** | 0.25% Bupivacaine 或 0.5% Ropivacaine 20–30 mL  |
| **確認** | 藥物擴散使前鋸肌與肋間肌分離                    |

**併發症**

- 氣胸
- 血管內注射
- 局部血腫

### 3.5 背伸肌筋膜平面阻斷 Erector Spinae Plane Block (ESP)

**適應症 Indications**

- 肋骨骨折
- 連枷胸
- 胸腔手術
- 帶狀疱疹
- 腹壁手術（多節段）

**解剖 Anatomy**

- 背伸肌（erector spinae）位於胸椎旁
- 深面為肋骨與橫突

| **位置** | 後正中線旁 2–3 cm，T5–T7 水平                        |
| **目標** | 背伸肌深面、橫突表面                                 |
| **進針** | 從外側向內進針（lateral-to-medial），平面內          |
| **藥物** | 0.25% Bupivacaine 或 0.5% Ropivacaine 20–30 mL       |
| **確認** | 藥物擴散使背伸肌與橫突分離                           |

**併發症**

- 氣胸（針尖過深）
- 蛛網膜下腔注射（罕見）
- 血管內注射
- 局部血腫

## 4. 腹壁 Abdominal Wall

### 4.1 腹橫肌平面阻斷 Transversus Abdominis Plane Block (TAP)

**適應症 Indications**

- 腹壁手術（appendectomy、cholecystectomy、hernia repair）
- 剖腹產
- 下腹壁疼痛
- 椎管內麻醉禁忌時的替代

**解剖 Anatomy**

- 腹內斜肌（internal oblique）與腹橫肌（transversus abdominis）之間的平面
- 肋間神經的前支（T7–L1）走行於此平面

**超音波引導 Ultrasound-guided**

| 項目     | 內容                                             |
| -------- | ------------------------------------------------ |
| **探頭** | 高頻線性 10–15 MHz                               |
| **位置** | 髂前上棘與肋骨之間（腋中線）                     |
| **目標** | 腹內斜肌與腹橫肌之間                             |
| **進針** | 從外側向內進針（lateral-to-medial），平面內      |
| **藥物** | 0.25% Bupivacaine 或 0.5% Ropivacaine 20 mL/side |
| **確認** | 藥物擴散使腹內斜肌與腹橫肌分離，呈"三明治"徵象   |

**併發症**

- 腹腔內穿刺（腸穿孔，罕見）
- 肝/脾穿刺（上腹阻斷時）
- 血管內注射
- 局部血腫

### 4.3 髂腹下/髂腹股溝神經阻斷 Ilioinguinal/Iliohypogastric Nerve Block

### 5.3 收肌管阻斷 Adductor Canal Block (ACB)

**適應症 Indications**

- 藥物誤入血管
- 血腫

### 5.4 坐骨神經阻斷 Sciatic Nerve Block

| **Supragluteal**           | 臀大肌淺面，坐骨結節上方 | 較少使用      |
| **Popliteal**              | 膕窩上方                 | 足部/腳踝手術 |

**超音波引導 Ultrasound-guided（Subgluteal approach）**

| 項目     | 內容                                           |
| -------- | ---------------------------------------------- |
| **探頭** | 低頻曲面 2–5 MHz（深部）或高頻線性 10–15 MHz   |
| **位置** | 坐骨結節外側，臀大肌深面                       |
| **目標** | 坐骨神經（呈橢圓形，低回音束狀結構）           |
| **進針** | 從外側向內進針（lateral-to-medial），平面內    |
| **藥物** | 0.25% Bupivacaine 或 0.5% Ropivacaine 20–30 mL |
| **確認** | 藥物擴散包圍坐骨神經                           |

**併發症**

- 坐骨神經損傷
- 血管內注射
- 局部血腫

## 7. 椎管內 Neuraxial

### 7.1 硬膜外麻醉 Epidural Anesthesia

**適應症 Indications**

- 下腹部手術（appendectomy、hysterectomy、caesarean section）
- 下肢手術
- 胸腔手術（thoracic epidural）
- 產科鎮痛
- 慢性疼痛管理

**解剖 Anatomy**

- 硬膜外腔位於黃韌帶（ligamentum flavum）與硬脊膜之間
- 胸段：T1–T12（胸椎棘突陡峭）
- 腰段：L1–L4（腰椎棘突較水平）
- 骶段：骶管

**穿刺位置**

| 位置     | 棘突方向         | 穿刺角度   |
| -------- | ---------------- | ---------- |
| **胸段** | 陡峭（60–70°）   | 向頭側傾斜 |
| **腰段** | 較水平（15–30°） | 較垂直     |
| **骶段** | 骶裂孔           | 向頭側     |

**穿刺技術**

| 項目             | 內容                                        |
| ---------------- | ------------------------------------------- |
| **方法**         | 正中法（midline）或旁正中法（paramedian）   |
| **阻力消失**     | Loss of resistance（LOR）至空氣或生理食鹽水 |
| **Confirmation** | 回抽無血液/腦脊液，負壓試驗陽性             |

**常用藥物**

| 藥物            | 濃度              | 起始時間  | 持續時間   |
| --------------- | ----------------- | --------- | ---------- |
| **Bupivacaine** | 0.5% (isobaric)   | 15–20 min | 1.5–2.5 hr |
| **Bupivacaine** | 0.5% (hyperbaric) | 5–10 min  | 1.5–2 hr   |
| **Lidocaine**   | 2%                | 5–10 min  | 40–60 min  |
| **Ropivacaine** | 0.5%              | 15–20 min | 1.5–2.5 hr |

**併發症**

- 硬膜穿破後頭痛（post-dural puncture headache, PDPH）
- 脊髓或神經根損傷
- 硬膜外血腫
- 硬膜外膿瘍
- 藥物誤入血管
- 全脊髓麻醉
- 低血壓（交感神經阻斷）

---

### 7.2 蛛網膜下腔麻醉 Spinal Anesthesia (Subarachnoid Block)

**適應症 Indications**

- 下腹部手術（appendectomy、caesarean section、hernia repair）
- 下肢手術
- 肛門手術
- 剖腹產

**解剖 Anatomy**

- 蛛網膜下腔位於蛛網膜與軟腦膜之間
- 含腦脊液（CSF）
- 脊髓終止於 L1–L2（成人），故腰椎穿刺選 L3–L4 或 L4–L5

**穿刺位置**

| 項目     | 內容                   |
| -------- | ---------------------- |
| **位置** | L3–L4 或 L4–L5 間隙    |
| **方向** | 正中法或旁正中法       |
| **確認** | 腦脊液流出（清晰液體） |

**常用藥物**

| 藥物                         | 濃度/類型 | 劑量      | 起始時間  | 持續時間   |
| ---------------------------- | --------- | --------- | --------- | ---------- |
| **Bupivacaine (hyperbaric)** | 0.5%      | 7.5–15 mg | 3–5 min   | 1–1.5 hr   |
| **Bupivacaine (isobaric)**   | 0.5%      | 10–15 mg  | 10–15 min | 1.5–2.5 hr |
| **Lidocaine (hyperbaric)**   | 5%        | 50–100 mg | 1–2 min   | 45–60 min  |
| **Tetracaine (hyperbaric)**  | 1%        | 5–10 mg   | 5–10 min  | 1.5–2 hr   |

**併發症**

- 低血壓（最常見，交感神經阻斷）
- 心搏過緩（迷走神經反射）
- 背痛
- 硬膜穿破後頭痛（PDPH）
- 尿滯留
- 馬尾症候群（cauda equina syndrome，罕見）
- 感染

### 7.3 腰椎硬膜外聯合 Combined Spinal-Epidural (CSE)

**適應症 Indications**

- 剖腹產（快速起效 + 長期止痛）
- 產科鎮痛（early analgesia + continuous infusion）
- 下腹部手術

**技術**

1. 先行硬膜外穿刺（loss of resistance 確認）
2. 經硬膜外針插入腰椎穿刺針
3. 蛛網膜下腔注藥（subarachnoid block）
4. 退出腰椎穿刺針
5. 留置硬膜外導管

**常用配方**

- Spinal：0.5% hyperbaric Bupivacaine 7.5–12.5 mg + Fentanyl 10–25 mcg
- Epidural：0.25% Bupivacaine or 0.2% Ropivacaine 3–5 mL initial dose，後 5–8 mL/hr infusion

**併發症**

- 硬膜穿破後頭痛（PDPH）
- 藥物誤入血管
- 低血壓
- 背痛

### D. 臨床小提示 Clinical Pearls

1. **ISB 高容量 vs 低容量**：高容量（20 mL）提供完全阻斷但膈神經麻痺率 100%；低容量（5 mL）可保留膈功能但可能不完全
2. **Supraclavicular vs ISB**：supraclavicular 阻斷更完全且起效快，但氣胸風險較高
3. **收肌管 vs 股神經**：收肌管阻斷保留股四頭肌力量，適合門診 TKA
4. **TAP vs PVB**：PVB 提供更好的胸段止痛，TAP 適合下腹壁
5. **臀部坐骨神經**：subgluteal approach 在超音波下最易辨認
6. **踝神經阻斷**：最後選擇，但若超音波導引得當可提供完全足部麻醉
7. **CSE**：適合需要快速起效 + 長期止痛的剖腹產
8. **ESP block**：新興技術，操作簡單且安全，適合多節段肋骨骨折
