# CAT 慢性阻塞性肺病評估測試

**CAT COPD Assessment Test**

## 概述

慢性阻塞性肺病（COPD）是全球第三大死亡原因，準確評估其對患者日常生活的影響是治療決策的核心。COPD 評估測試（CAT）是一個簡短、經過驗證的患者自評問卷，量化 COPD 症狀對生活品質的衝擊，並整合至 GOLD 2023+ 的 ABE 分類框架中指導治療選擇。

## 什麼是 CAT？

CAT（COPD Assessment Test）由 Jones 等人於 2009 年開發，是一個 8 題的患者自填問卷，涵蓋 COPD 最重要的症狀與功能面向：咳嗽、痰液、胸悶、喘息、居家活動、外出信心、睡眠品質及精力。每題 0–5 分，總分 0–40 分，分數越高代表 COPD 對生活的衝擊越大。

### 八個評估面向

| 題號 | 面向     | 0 分（最佳）  | 5 分（最差）     |
| ---- | -------- | ------------- | ---------------- |
| Q1   | 咳嗽     | 從不咳嗽      | 一直在咳嗽       |
| Q2   | 痰液     | 胸腔完全無痰  | 胸腔充滿痰液     |
| Q3   | 胸悶     | 完全無壓迫感  | 非常嚴重的壓迫感 |
| Q4   | 喘息     | 爬坡/爬梯不喘 | 爬坡/爬梯非常喘  |
| Q5   | 居家活動 | 不受任何限制  | 受到非常大的限制 |
| Q6   | 外出信心 | 有信心外出    | 完全沒信心外出   |
| Q7   | 睡眠     | 睡得很好      | 睡得很差         |
| Q8   | 精力     | 精力充沛      | 完全沒有精力     |

## 評分詮釋

### 四級影響分類

| CAT 總分  | 影響程度   | 臨床描述                                                    |
| --------- | ---------- | ----------------------------------------------------------- |
| **0–9**   | 低度影響   | COPD 症狀輕微，偶爾咳嗽或活動時略感喘息，整體功能良好       |
| **10–20** | 中度影響   | COPD 對日常生活有明顯影響，多數日子喘息，晨起常有胸悶和痰液 |
| **21–30** | 高度影響   | COPD 嚴重影響日常功能，持續疲憊、喘息，睡眠及活動均受限     |
| **31–40** | 極高度影響 | COPD 對生活品質造成極嚴重衝擊，幾乎每天受症狀困擾           |

### 最小臨床重要差異（MCID）

CAT 分數變化 **≥ 2 分**被認為具有臨床意義。在臨床試驗和真實世界研究中，治療有效時通常可見 CAT 分數降低 2–4 分。建議每 **2–3 個月**複評一次，追蹤疾病進展或治療反應。

## GOLD 2023+ ABE 分類框架

GOLD（全球慢性阻塞性肺病倡議組織）2023 年起將原 ABCD 分類簡化為 **ABE** 分類，CAT 分數是症狀評估的核心工具之一。

### ABE 分類邏輯

```
                    症狀評估
                    CAT < 10     CAT ≥ 10
                   (少症狀)      (多症狀)
                  ┌──────────┬──────────┐
  急性惡化 0–1次  │ Group A  │ Group B  │
  (無住院)        │          │          │
                  ├──────────┴──────────┤
  急性惡化 ≥2次   │        Group E       │
  OR 住院 ≥1次    │  (高急性惡化風險)    │
                  └─────────────────────┘
```

### 各組治療建議（GOLD 2023）

| GOLD 組別   | CAT  | 急性惡化史             | 初始治療建議                                          |
| ----------- | ---- | ---------------------- | ----------------------------------------------------- |
| **Group A** | < 10 | 0–1 次（無住院）       | 支氣管擴張劑（單藥，按需或定期）                      |
| **Group B** | ≥ 10 | 0–1 次（無住院）       | **LABA + LAMA** 雙效吸入治療                          |
| **Group E** | 任何 | ≥2 次中度 OR ≥1 次住院 | **LABA + LAMA**；若血液嗜酸性粒細胞 ≥300 考慮加入 ICS |

**注意：** CAT ≥ 10 本身不足以確定 Group E，需同時評估急性惡化史。

## CAT 在 COPD 管理中的角色

### 初始評估

初診 COPD 患者（FEV1/FVC < 0.70 確認診斷後）完成 CAT，了解症狀負擔並建立基線值，用於：

- ABE 分類指導初始治療選擇
- 識別需優先介入的症狀領域
- 與患者溝通病情嚴重程度

### 定期監測

每 2–3 個月複評，重點追蹤：

- **CAT 分數上升**：症狀惡化，考慮升階治療或評估共病
- **CAT 分數下降 ≥2**：治療有效，維持現有方案
- **各題分數變化**：識別最困擾患者的面向（如睡眠惡化提示夜間支氣管痙攣）

### 急性惡化後評估

急性惡化緩解後 4–6 週重新評估 CAT，確認是否恢復至基線水準。未能恢復者提示肺功能持續下降，需重新評估治療策略。

## CAT 與其他評估工具比較

| 工具     | 題數  | 時間       | 評估面向             | 用途                        |
| -------- | ----- | ---------- | -------------------- | --------------------------- |
| **CAT**  | 8 題  | 2–3 分鐘   | 多面向 COPD 影響     | GOLD ABE 分類、全面症狀評估 |
| **mMRC** | 1 題  | < 1 分鐘   | 僅喘息               | 快速篩查，ABE 分類替代指標  |
| **CCQ**  | 10 題 | 2–4 分鐘   | 症狀、功能狀態、心理 | 研究用，臨床較少用          |
| **SGRQ** | 50 題 | 10–15 分鐘 | 詳細健康狀況         | 臨床試驗金標準              |

**CAT 優勢：** 題數少、完成快、信效度良好（與 SGRQ 相關係數 r = 0.84）、對變化敏感、可在初級醫療廣泛使用。

### CAT vs mMRC

mMRC 只評估喘息（0–4 分級），優點是更快速；缺點是忽略咳嗽、痰液、睡眠、精力等同樣重要的面向。GOLD 允許以 mMRC ≥ 2 替代 CAT ≥ 10 作為 Group B 的分類標準，但 CAT 提供更全面的症狀圖像，臨床上更有助於識別治療目標。

## 肺復健與 CAT

肺復健（Pulmonary Rehabilitation, PR）是目前 COPD 最有效的非藥物治療，已證實可顯著降低 CAT 分數：

- 完成 8–12 週肺復健後，CAT 平均降低 **3–5 分**
- Group B 及 Group E 患者均可從肺復健中獲益
- CAT ≥ 10 是轉介肺復健的重要指標之一
- 肺復健包含：運動訓練、呼吸訓練、患者教育、戒菸支持

## 注意事項

::: warning CAT 不能診斷 COPD
CAT 評估 COPD 的症狀負擔，**不能用於診斷 COPD**。診斷需以肺功能確認：吸入支氣管擴張劑後 FEV1/FVC < 0.70（氣流限制不完全可逆）。CAT 在未確診患者中僅反映症狀，不代表 COPD 存在。
:::

::: warning GOLD ABE 分類需整合急性惡化史
CAT ≥ 10 僅代表症狀多，屬 Group B；若同時有 ≥2 次中度急性惡化或 ≥1 次住院，則應分類為 **Group E**，治療策略有所不同（需考慮 ICS 加入）。臨床分類務必同時詢問過去 12 個月急性惡化史。
:::

::: info 患者自評工具的局限
CAT 為患者主觀自評，可能受認知功能、教育程度、文化背景及情緒狀態影響。憂鬱症合併 COPD 患者的 CAT 分數常系統性偏高，不完全反映氣道阻塞程度。建議結合肺功能（FEV1%）、6 分鐘步行距離及臨床評估綜合判斷。
:::

::: info 追蹤頻率建議
穩定期 COPD 患者建議每 **2–3 個月**完成一次 CAT，急性惡化後 **4–6 週**重評。台灣 COPD 照護指引建議將 CAT 納入每次回診的常規評估，建立長期趨勢曲線，提高患者對病情管理的參與感。
:::

::: info 免責聲明
本工具依據 Jones PW et al. Eur Respir J 2009 原始驗證研究及 GOLD 2023 指引設計，僅供臨床教育及輔助評估參考，**不能取代呼吸科醫師的完整評估**。COPD 治療決策須整合肺功能、急性惡化史、共病評估及患者意願。
:::
