# CAT 慢性阻塞性肺病評估測試

**COPD Assessment Test · 慢性阻塞性肺病症狀影響量表**

<CATCalculator />

## CAT 量表

**COPD Assessment Test（CAT）** 為 8 題自填量表，評估 COPD 對日常生活的整體影響，每題 0–5 分，總分 0–40。

| 項目           | 0 分（無症狀）           | 5 分（極嚴重）             |
| -------------- | ------------------------ | -------------------------- |
| 咳嗽           | 我從不咳嗽               | 我一直在咳嗽               |
| 痰液           | 胸腔完全沒有痰           | 胸腔裡全是痰               |
| 胸悶           | 胸部完全沒有壓迫感       | 胸部有非常嚴重的壓迫感     |
| 爬坡/爬樓梯喘息 | 爬一層樓梯時不會喘       | 爬一層樓梯時非常喘         |
| 居家活動受限   | 做家裡的事不受任何限制   | 做家裡的事受到非常大限制   |
| 離家信心       | 對出門有信心             | 對離家一點信心都沒有       |
| 睡眠           | 我睡得很好               | 因肺部疾病睡得很差         |
| 精力           | 我精力充沛               | 我完全沒有精力             |

### 分數意義與處置

| 總分  | 影響程度   | 意義與建議處置                                       |
| ----- | ---------- | ---------------------------------------------------- |
| 0–10  | 低度影響   | 症狀影響輕微，維持戒菸、規律用藥與運動，定期追蹤     |
| 11–20 | 中度影響   | 明顯影響日常，檢視吸入劑技巧與治療是否足夠           |
| 21–30 | 高度影響   | 嚴重影響功能，強化藥物治療並評估肺復原               |
| ≥ 31  | 極高度影響 | 極嚴重影響生活品質，積極治療、肺復原並評估合併症     |

> **CAT ≥ 10 分**通常代表症狀負擔明顯，是 GOLD 分組（結合症狀與惡化史）中「症狀較多」的分界之一，常與 mMRC 呼吸困難量表併用。CAT 的價值在於**追蹤同一病人的分數變化**：分數上升 ≥ 2 分具臨床意義，可反映惡化或治療反應不佳；分數為病人主觀感受，應與肺功能（FEV₁）及惡化史綜合判讀。

## 注意事項

::: warning 症狀量表非肺功能
CAT 反映症狀對生活的影響，不等同肺功能檢查（如 FEV₁）；COPD 分級與治療應結合肺量計與惡化史。
:::

::: info 免責聲明
本工具僅供臨床輔助與教學參考，COPD 診斷、分組與治療應由醫師依完整評估判定。
:::
