# PEF 峰值呼氣流速預測

**Predicted Peak Expiratory Flow · 峰值呼氣流速預測與氣喘行動分區**

<PEFCalculator />

## PEF 預測方程式

**峰值呼氣流速（Peak Expiratory Flow, PEF）** 依性別、年齡與身高預測；本工具依年齡自動選用成人或兒童方程式。

| 族群         | 方程式（PEF, L/min）                     |
| ------------ | ---------------------------------------- |
| 成人男性     | 5.48 × 身高(cm) − 1.93 × 年齡 − 454      |
| 成人女性     | 3.72 × 身高(cm) − 2.24 × 年齡 + 2        |
| 兒童男（5–17）| 5.49 × 身高(cm) − 441（身高導向）        |
| 兒童女（5–17）| 3.94 × 身高(cm) − 281（身高導向）        |

> 成人採 Nunn & Gregg（BMJ 1989）方程式，適用約 18–85 歲；兒童 PEF 主要由身高決定，採 Knudson 身高導向估計。

## 綠 / 黃 / 紅 氣喘行動分區

輸入實測 PEF 後，以「實測 ÷ 預測 × 100%」判定分區：

| 分區 | % 預測值 | 意義與建議處置                               |
| ---- | -------- | -------------------------------------------- |
| 綠色 | ≥ 80%    | 氣道控制良好，維持現行治療                   |
| 黃色 | 50–79%   | 警示區，依氣喘行動計畫調整用藥、加強監測     |
| 紅色 | < 50%    | 緊急區，立即使用急救吸入劑並儘速就醫         |

> **PEF 主要用於監測氣道阻塞的變化趨勢**，而非診斷。個人「最佳值」（個人最佳 PEF）常比族群預測值更能反映控制狀態，建議在症狀穩定時建立個人基準。綠 / 黃 / 紅分區是氣喘自我管理計畫（Asthma Action Plan）的核心；**PEF < 50% 預測值或個人最佳值**代表嚴重氣流阻塞，屬急性發作警訊。

## 注意事項

::: warning 預測值僅供參考
族群預測方程式存在個體差異；臨床判讀應優先參考病人的個人最佳 PEF 與症狀。
:::

::: info 免責聲明
本工具僅供臨床輔助與教學參考，急性發作處置與用藥調整應依醫囑與氣喘行動計畫執行。
:::
