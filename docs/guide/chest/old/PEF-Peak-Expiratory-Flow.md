# PEF 峰值呼氣流速預測

**PEF Peak Expiratory Flow**

## 概述

峰值呼氣流速（Peak Expiratory Flow, PEF）是評估氣道阻塞程度的簡便工具，廣泛應用於哮喘監測、急診分診及肺功能篩查。正確解讀 PEF 需要與個人化預測值比較，而非僅依賴絕對數值。

## 什麼是峰值呼氣流速？

PEF 是最大用力呼氣初始瞬間的最高流速，以公升/分鐘（L/min）表示。它主要反映大氣道（中央氣道）的通暢程度，在哮喘、COPD 及其他阻塞性肺病中常見降低。

### 為什麼需要預測值？

PEF 個體間差異極大，受年齡、性別、身高影響顯著。單一絕對數值無法判斷是否正常，必須與**個人化預測值**或**個人最佳值**比較，才能評估肺功能狀態。

## 參考方程式

### 成人（18–85 歲）：Nunn & Gregg 1989

Nunn 與 Gregg 於 1989 年整合兩項研究，建立了適用於 15–85 歲成人的 PEF 預測迴歸方程式，是目前歐洲及英國臨床最廣泛採用的標準。

```
男性：PEF = (5.48 × 身高cm) − (1.93 × 年齡) − 454
女性：PEF = (3.72 × 身高cm) − (2.24 × 年齡) + 2
```

**驗證族群：** 英格蘭南部非吸菸健康成人，使用 Wright 峰流速計（已校正至 EU/EN 13826 刻度）。

**特點：**

- 同時納入年齡（負效應）和身高（正效應）
- 年齡效應在男性（−1.93/yr）略低於女性（−2.24/yr）
- 男性因肺容量較大，預測值系統性高於女性

### 兒童（5–<18 歲）：Knudson 1983

Knudson 等人（1983）的研究確立了以身高為主要預測因子的兒童 PEF 方程式，被 Clement Clarke EU 刻度峰流速計兒童諾莫圖採用。

```
男孩：PEF = (5.49 × 身高cm) − 441
女孩：PEF = (3.94 × 身高cm) − 281
```

**特點：**

- 兒童 PEF 主要由身高決定，年齡效應相對次要
- 青春期前（< 150 cm）男女差異不顯著，青春期後男孩 PEF 顯著高於女孩
- 正常兒童 PEF 隨身高線性增加，15 歲後男孩超過女孩約 100–150 L/min

## 三色管理分區系統

哮喘管理採用「交通燈」（Traffic Light）三色分區：

| 區域        | % 個人最佳值或預測值 | 描述     | 建議行動                      |
| ----------- | -------------------- | -------- | ----------------------------- |
| 🟢 **綠色** | ≥ 80%                | 良好控制 | 維持現有治療，繼續記錄        |
| 🟡 **黃色** | 50–79%               | 警示區   | 使用 SABA，評估誘因，諮詢醫師 |
| 🔴 **紅色** | < 50%                | 緊急區   | 立即使用 SABA，無效立即就醫   |

### 重要概念：個人最佳值 vs 預測值

在哮喘管理中，**個人最佳值**（Personal Best）比人口預測值更具臨床意義：

- **預測值**：基於年齡/性別/身高的族群統計值，用於初始評估
- **個人最佳值**：患者在病情穩定控制期（通常 2–3 週無症狀）多次測量的最高值

建立個人最佳值後，後續的急性發作評估應以**個人最佳值**為基準計算三色分區，而非預測值。

## 影響 PEF 的主要因素

### 生理因素

| 因素         | 影響方向    | 備註                                |
| ------------ | ----------- | ----------------------------------- |
| 身高         | ↑ 正相關    | 最強預測因子，兒童尤為顯著          |
| 年齡（成人） | ↓ 負相關    | 成人每增加 1 歲，PEF 下降約 2 L/min |
| 性別         | 男 > 女     | 男性因胸腔容積較大，PEF 系統性偏高  |
| 種族         | 亞洲 < 歐洲 | 亞洲人群約低 10–20%；臺灣臨床應注意 |
| 體能訓練     | ↑ 輕度      | 呼吸肌訓練可提高 PEF                |

### 病理因素

**降低 PEF：**

- 哮喘（支氣管痙攣）
- COPD（不可逆氣流阻塞）
- 急性上/下呼吸道感染
- 過敏性鼻炎急性期
- 吸菸（慢性影響）
- 胸廓畸形、神經肌肉疾病

**不反映 PEF 的問題：**

- 小氣道病變（PEF 為大氣道指標）
- 限制性肺疾病（需 FVC 評估）

## PEF 與螺旋測定法（Spirometry）比較

| 特性               | PEF                | FEV1 / Spirometry |
| ------------------ | ------------------ | ----------------- |
| 設備               | 簡便峰流速計       | 螺旋肺量計        |
| 技術難度           | 低                 | 中等（需訓練）    |
| 費用               | 低                 | 較高              |
| 診斷哮喘           | 輔助（支持性）     | **首選診斷工具**  |
| 監測已知哮喘       | ✓ 適合居家監測     | 臨床定期複查      |
| 小氣道評估         | ✗ 不敏感           | ✓ FEF25-75%       |
| 可逆性測試         | ✓ 支氣管擴張劑前後 | ✓ 更精確          |
| 年輕兒童（< 5 歲） | ✗ 技術困難         | ✗ 亦困難          |

## 臨床應用場景

### 哮喘急性發作嚴重度評估

| 嚴重度 | PEF（% 預測/最佳值） | 其他臨床表現       |
| ------ | -------------------- | ------------------ |
| 輕度   | > 70%                | 可完整句子說話     |
| 中度   | 40–70%               | 短句說話困難       |
| 重度   | < 40%                | 呼吸困難，說話困難 |
| 瀕危   | < 25% 或無法測量     | 意識改變，靜默肺   |

### GINA 哮喘管理建議

GINA（全球哮喘倡議組織）對 PEF 監測的建議：

- **確診哮喘**：結合症狀、體徵、可逆性氣流阻塞（FEV1 改善 ≥12%，≥200 mL）
- **居家監測**：推薦有嚴重哮喘、對症狀感知差、或有急性加重史的患者每日監測
- **急診評估**：PEF < 40% 預測值為重度發作，需立即處置

### 職業性哮喘篩查

連續 2 週以上的居家 PEF 記錄（早晚各一次），若顯示工作日降低、休假改善的規律性變化，高度提示職業性哮喘。

## 兒童 PEF 特殊考量

### 青春期的性別分歧

- 10 歲以前：男女 PEF 差異不顯著（均以身高為主）
- 青春期後（約 12–14 歲起）：男孩 PEF 加速增長，15 歲時超過同身高女孩約 50–100 L/min
- 女孩 PEF 約於 15–16 歲達到峰值；男孩持續增長至 18–20 歲

### 技術注意事項（兒童）

- **年齡下限**：多數兒童 5 歲起可配合正確操作
- **三次取最高**：兒童更易出現測量間變異，務必取三次最高值
- **站姿**：站立比坐姿可獲得更高且更可重複的 PEF 值
- **鼓勵**：兒童需充分口頭鼓勵以獲得最大用力呼氣

## 亞洲人群的校正考量

Nunn & Gregg（1989）及 Knudson（1983）均以歐洲白人族群建立，亞洲人群（包括臺灣）的 PEF 參考值系統性偏低：

- 亞洲成人 PEF 約較歐洲預測值低 **10–20%**
- 部分臨床建議對亞洲人群採用 **×0.8–0.85** 的校正係數
- 更準確的做法是建立患者**個人最佳值**作為基準

目前尚無廣泛驗證的臺灣本土 PEF 參考方程式；臨床使用歐洲方程式時應注意系統性偏高可能。

## 操作技術規範

正確的 PEF 測量步驟：

1. 直立站立（或坐直）
2. 深吸一口氣至全肺容量（TLC）
3. 將峰流速計咬嘴含好，嘴唇緊密包覆
4. **盡可能快速、用力**地呼氣（爆發性而非緩慢吹氣）
5. 記錄讀數，歸零，重複三次
6. 記錄三次中的**最高值**

**常見錯誤：**

- 呼氣不夠用力（最常見）
- 舌頭阻塞咬嘴
- 未站立（坐姿測量偏低）
- 未充分深吸氣就開始呼氣
- 僅測一次（單次測量重現性差）

## 注意事項

::: warning 預測值的種族差異
本工具使用的 Nunn & Gregg（1989）方程式基於英格蘭白人族群，亞洲人群（含臺灣）的 PEF 預測值系統性偏高約 10–20%。臨床建議優先建立患者**個人最佳值**作為哮喘管理基準，而非依賴人口統計預測值。
:::

::: warning PEF 不等於哮喘診斷
PEF 降低僅提示氣流阻塞，無法區分哮喘、COPD、支氣管炎等病因。哮喘確診需符合 GINA 標準：症狀 + 可逆性氣流阻塞（吸入 SABA 後 FEV1 改善 ≥12% 且 ≥200 mL）或氣道過度反應性陽性。PEF 作為輔助監測工具，不能單獨作為診斷依據。
:::

::: info 兒童 5 歲以下替代方案
5 歲以下幼兒通常無法正確配合 PEF 測量。對此年齡群，哮喘評估應改用：臨床症狀評估（wheezing 頻率、嚴重度）、試驗性治療反應、脈氧飽和度監測，以及特殊兒科肺功能測試（如嬰兒肺功能測試）。
:::

::: info 居家監測建議
哮喘患者建議：每日早晨（起床後、用藥前）及傍晚各測一次，連續記錄至少 2 週建立個人最佳值。「晨間最低值（Morning Dip）」是哮喘控制不佳的早期信號，即使無症狀也可能出現明顯下降。
:::

::: info 免責聲明
本工具依據 Nunn & Gregg（1989）及 Knudson（1983）參考方程式設計，僅供臨床教育及輔助評估參考，**不能取代呼吸科醫師正式評估及螺旋測定法**。急性哮喘發作管理須依據完整臨床評估，不應單獨依賴 PEF 數值決策。
:::
