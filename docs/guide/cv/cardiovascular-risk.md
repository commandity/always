# 心血管風險評估工具

**Cardiovascular Risk Assessment · CHA₂DS₂-VA & AHA PREVENT**

<CardiovascularRisk />

## CHA₂DS₂-VA 心房顫動中風風險

**CHA₂DS₂-VA**（2024 ESC）用於非瓣膜性心房顫動患者的缺血性中風年度風險分層，指引口服抗凝血劑（OAC）決策。2024 年版**移除性別（Sc）項**，改稱 CHA₂DS₂-VA。

| 項目                           | 分數 |
| ------------------------------ | ---- |
| C — 充血性心臟衰竭             | 1    |
| H — 高血壓                     | 1    |
| A₂ — 年齡 ≥ 75 歲             | 2    |
| D — 糖尿病                     | 1    |
| S₂ — 中風 / TIA / 血栓栓塞病史 | 2    |
| V — 血管疾病                   | 1    |
| A — 年齡 65–74 歲             | 1    |

### 分數意義與處置

| 總分 | 建議                                       |
| ---- | ------------------------------------------ |
| 0    | 不需抗凝血，血栓風險低                     |
| 1    | 依個別風險考慮 OAC                         |
| ≥ 2  | 建議口服抗凝血劑（除非禁忌）               |

> **≥ 2 分即建議 OAC**（NOAC 優先於 warfarin）。分數愈高年度中風率愈高（如 4 分約 4%/年）。抗凝血決策亦須併同評估出血風險（如 HAS-BLED），但出血風險高不應單獨作為不抗凝的理由，而是提示需矯正可逆的出血因子。

## AHA PREVENT 心血管風險

**PREVENT**（AHA 2023，Predicting Risk of cardiovascular disease EVENTs）為 30–79 歲、無既有 ASCVD 成人估計 **10 年與 30 年**心血管風險的方程式，可估 ASCVD、整體 CVD 與心衰（HF）風險，並納入 eGFR（腎功能）等代謝-腎-心因子。

| 主要預測因子 | 說明                             |
| ------------ | -------------------------------- |
| 年齡、性別   | 30–79 歲                         |
| 血脂         | 總膽固醇、HDL-C                  |
| 血壓         | 收縮壓、是否服用降壓藥           |
| 腎功能       | eGFR                             |
| 其他         | 吸菸、糖尿病、是否服用 statin    |

### 風險分層（10 年 ASCVD）

| 10 年風險 | 分層     | 意義與建議處置                             |
| --------- | -------- | ------------------------------------------ |
| < 5%      | 低風險   | 生活型態介入為主                           |
| 5–< 7.5%  | 邊際     | 依風險增強因子個別化決策                   |
| 7.5–< 20% | 中等     | 考慮 statin，共同決策                       |
| ≥ 20%     | 高風險   | 建議 statin 及積極危險因子管理             |

> **PREVENT 是首個納入 eGFR 並不含種族的美國心血管風險方程式**，較舊的 Pooled Cohort Equations 更符合當代族群，通常給出較低的估計值。10 年風險用於當前治療決策，30 年風險有助於年輕族群的長期溝通；治療門檻應結合風險增強因子（如家族史、Lp(a)、CAC）共同決策。

## 注意事項

::: warning 風險估計非個別保證
兩者皆為族群統計模型的機率估計，不能取代個別臨床判斷與共同決策；適用族群、驗證條件不同，套用於範圍外對象時需謹慎。
:::

::: info 免責聲明
本工具僅供臨床輔助與教學參考，抗凝血或降脂治療決策應由醫師依完整評估判定。
:::
