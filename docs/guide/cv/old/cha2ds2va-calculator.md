# CHA₂DS₂-VA 心房顫動中風風險評估

**CHA₂DS₂-VA Atrial Fibrillation Stroke Risk Assessment**

## 概述

**CHA₂DS₂-VA** 是評估**非瓣膜性心房顫動（AF）患者中風風險**的臨床決策工具，用於指導口服抗凝血藥物（OAC）的使用決策。原版 **CHA₂DS₂-VASc**（2010）包含女性性別（Sex category）作為風險因子，但 2024 年 ESC 心房顫動指引依據最新實證研究，將女性性別從評分中移除，更名為 **CHA₂DS₂-VA**，總分最高 **9 分**。

## 評分說明

CHA₂DS₂-VA 共 7 項臨床因子，其中 **A₂** 與 **S₂** 各計 **2 分**，其餘各計 **1 分**：

| 字母   | 因子                    | 定義                                 | 分數   |
| ------ | ----------------------- | ------------------------------------ | ------ |
| **C**  | 充血性心臟衰竭          | 近期心衰竭症狀或 EF ≤ 40%            | +1     |
| **H**  | 高血壓                  | 靜息 BP > 140/90 mmHg 或正在服降壓藥 | +1     |
| **A₂** | 年齡 ≥ 75 歲            | 高齡為最強中風危險因子之一           | **+2** |
| **D**  | 糖尿病                  | 空腹血糖 > 125 mg/dL 或正在服降糖藥  | +1     |
| **S₂** | 中風 / TIA / 血栓栓塞史 | 過去曾有缺血性中風、TIA 或周邊栓塞   | **+2** |
| **V**  | 血管疾病                | 冠心病、PAD、主動脈斑塊或心肌梗塞史  | +1     |
| **A**  | 年齡 65–74 歲           | 65–74 歲（與 A₂ 不重複計分）         | +1     |

> **年齡注意**：A₂（≥ 75 歲）與 A（65–74 歲）**不重複計分**，請擇一勾選。75 歲以上者計 **+2**，65–74 歲者計 **+1**，64 歲以下不計分。

## 2024 ESC 指引更新：為何移除女性性別？

2024 年 ESC 指引依據多項大型研究的最新分析，得出以下結論：

| 問題                                  | 結論                                                           |
| ------------------------------------- | -------------------------------------------------------------- |
| 女性是否為獨立中風危險因子？          | **不是**——女性中風風險升高主要由其他共病（高齡、高血壓等）解釋 |
| 0 分女性（其他因子全無）的中風風險？  | 接近 0%，無需抗凝                                              |
| CHA₂DS₂-VA vs CHA₂DS₂-VASc 預測準確度 | CHA₂DS₂-VA 特異性更高，減少不必要的抗凝治療                    |

> **臨床影響**：過去 CHA₂DS₂-VASc 下，僅憑女性性別即得 1 分的患者（無其他危險因子），依新指引 CHA₂DS₂-VA = 0，不需抗凝治療，避免了不必要的出血風險。

## 各評分因子的臨床意義

### C — 充血性心臟衰竭

心衰竭導致心房擴大、血液停滯及心房顫動持續，顯著增加心內血栓形成及中風風險。左心室收縮功能減退（EF ≤ 40%）或近期心衰竭失代償均計分。

### H — 高血壓

高血壓是心房顫動最常見的合併症（約 70–80% AF 患者合併高血壓），也是中風最重要的可改變危險因子。包含控制良好但正在服藥的患者。

### A₂ — 年齡 ≥ 75 歲（+2 分）

高齡是 AF 中風最強的獨立危險因子，此項計 2 分反映其重要性。75 歲以上患者的年中風率比 65 歲以下患者高出 3–5 倍。

### D — 糖尿病

糖尿病透過促進動脈粥樣硬化、微血管病變及凝血異常增加中風風險。需注意糖尿病前期不計分。

### S₂ — 中風 / TIA / 血栓栓塞史（+2 分）

過去曾有缺血性中風或 TIA 是未來中風最強的獨立預測因子，計 2 分。此類患者無論其他分數高低，均應積極使用 OAC。

### V — 血管疾病

冠狀動脈疾病、周邊動脈疾病或主動脈斑塊反映系統性動脈粥樣硬化，是中風的協同危險因子。單純靜脈血栓（如 DVT）不計分。

### A — 年齡 65–74 歲（+1 分）

65–74 歲族群的中風風險較年輕族群顯著升高，但低於 75 歲以上。與 A₂ 不重複計分。

## 抗凝血藥物選擇

### 非維他命 K 口服抗凝藥（NOAC）——首選

| 藥物        | 靶點       | 劑量（標準） | 特色                         |
| ----------- | ---------- | ------------ | ---------------------------- |
| Apixaban    | 第 Xa 因子 | 5 mg bid     | 出血風險最低，腎功能要求寬鬆 |
| Rivaroxaban | 第 Xa 因子 | 20 mg qd     | 每日一次，依從性佳           |
| Dabigatran  | 凝血酶     | 150 mg bid   | 有解毒劑（Idarucizumab）     |
| Edoxaban    | 第 Xa 因子 | 60 mg qd     | 每日一次                     |

### Warfarin（維他命 K 拮抗劑）

僅在以下情況使用 Warfarin：

- 中重度二尖瓣狹窄
- 機械性人工心臟瓣膜
- NOAC 過敏或無法取得
- 目標 INR：2.0–3.0（機械瓣膜 2.5–3.5）

### 劑量調整情況

| 條件                                                    | Apixaban 調整 | Rivaroxaban 調整 | Dabigatran 調整 |
| ------------------------------------------------------- | ------------- | ---------------- | --------------- |
| 年齡 ≥ 80 歲 + 體重 ≤ 60 kg + Cr ≥ 1.5 mg/dL（任 2 項） | 2.5 mg bid    | —                | —               |
| CrCl 30–50 mL/min                                       | —             | 15 mg qd         | 75–110 mg bid   |
| CrCl < 30 mL/min                                        | 慎用          | 禁用             | 禁用            |

## 出血風險評估：HAS-BLED

在決定是否抗凝時，需同時評估出血風險：

| HAS-BLED 因子              | 定義                      | 分數  |
| -------------------------- | ------------------------- | ----- |
| **H** Hypertension         | 收縮壓 > 160 mmHg         | +1    |
| **A** Abnormal renal/liver | 腎功能或肝功能異常        | +1 各 |
| **S** Stroke history       | 中風病史                  | +1    |
| **B** Bleeding             | 出血史或出血傾向          | +1    |
| **L** Labile INR           | INR 不穩定（TTR < 60%）   | +1    |
| **E** Elderly              | 年齡 > 65 歲              | +1    |
| **D** Drugs/Alcohol        | 抗血小板藥、NSAIDs 或酒精 | +1 各 |

> **重要**：HAS-BLED ≥ 3 代表高出血風險，但**不是停用抗凝藥的理由**，而是提示需要**積極處理可逆性出血危險因子**（如控制血壓、停用 NSAIDs、監測 INR）。

## 心房顫動與中風

### AF 相關中風的特徵

- 通常較嚴重，致死率和致殘率比非 AF 相關中風高 2 倍
- 大面積梗塞更常見（心源性栓塞往往造成較大血栓）
- 復發率高——未治療的 AF 患者年中風復發率可達 12%

### 抗凝治療的效益

- OAC 可降低 AF 相關中風風險約 **60–70%**
- 與阿斯匹靈相比，NOAC 降低中風風險更顯著，且顱內出血更少
- **阿斯匹靈不能替代 OAC**——2024 ESC 指引明確建議不應以阿斯匹靈預防 AF 相關中風

## 注意事項

::: warning CHA₂DS₂-VA 僅適用於非瓣膜性心房顫動
CHA₂DS₂-VA 評分系統是針對**非瓣膜性**心房顫動設計的。中重度二尖瓣狹窄或機械性人工心臟瓣膜患者屬於「瓣膜性 AF」，這類患者中風風險極高，**無論分數高低均應使用 Warfarin**（NOAC 在此族群禁用或療效未經充分驗證）。
:::

::: warning 年齡分組不重複計分
A₂（≥ 75 歲，+2 分）與 A（65–74 歲，+1 分）**不可同時勾選**。75 歲以上患者計 A₂（+2），不再加算 A（+1）。工具中已設計互斥邏輯防止重複計分。
:::

::: warning 出血風險高不等於停用抗凝藥
HAS-BLED 高分患者應積極**處理可改變的出血危險因子**（控制血壓、停用非必要的 NSAIDs 和抗血小板藥、戒酒、穩定 INR），而非停用抗凝治療。對大多數中高風險 AF 患者，抗凝治療的中風預防效益遠大於出血風險。
:::

::: info 免責聲明
本計算器依據 2024 ESC 心房顫動指引建立，僅供臨床評估參考。抗凝血治療的決策應由心臟科或相關專科醫師根據患者完整病史、出血風險、腎功能及個別偏好綜合判斷。
:::
