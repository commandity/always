## Dupixent 適應症評估

針對合併鼻息肉之慢性鼻竇炎（CRSwNP）患者，評估是否符合生物製劑杜避炎（Dupilumab/Dupixent）健保給付條件。

| 項目             | 說明                        |
| ---------------- | --------------------------- |
| 組織嗜酸性球     | 組織切片 ≥ 10/HPF           |
| 血液嗜酸性球     | ≥ 250 cells/μL 或 ≥ 5%      |
| 總 IgE           | ≥ 100 IU/mL                 |
| 全身性類固醇需求 | 每年 ≥ 2 療程或長期使用禁忌 |
| 生活品質下降     | SNOT-22 ≥ 40/110            |

## SNOT-22 鼻竇生活品質量表

**Sino-Nasal Outcome Test-22** 為 22 題自填問卷，每題 0–5 分，總分 0–110 分。

| 面向           | 題數 | 評估內容                     |
| -------------- | ---- | ---------------------------- |
| 鼻部症狀       | 8 題 | 鼻塞、流鼻水、嗅覺喪失等     |
| 耳部與顏面症狀 | 4 題 | 耳悶、頭暈、面部疼痛         |
| 睡眠功能       | 5 題 | 入睡困難、夜間醒來、白天疲倦 |
| 心理與情緒功能 | 5 題 | 生產力、注意力、情緒影響     |
