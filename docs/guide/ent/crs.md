## Dupixent 適應症評估

針對合併鼻息肉之慢性鼻竇炎（CRSwNP）患者，評估是否符合生物製劑杜避炎（Dupilumab/Dupixent）健保給付條件。

| 項目             | 說明                        |
| ---------------- | --------------------------- |
| 組織嗜酸性球     | 組織切片 ≥ 10/HPF           |
| 血液嗜酸性球     | ≥ 250 cells/μL 或 ≥ 5%      |
| 總 IgE           | ≥ 100 IU/mL                 |
| 全身性類固醇需求 | 每年 ≥ 2 療程或長期使用禁忌 |
| 生活品質下降     | SNOT-22 ≥ 40/110            |

### 判定結果意義

| 符合項數 | 判定         | 意義                                                 |
| -------- | ------------ | ---------------------------------------------------- |
| 0–1 項   | 尚未符合     | 未達第二型發炎標準，暫不符合給付條件                 |
| 2 項     | 接近門檻     | 差 1 項達標，建議補齊檢驗資料（切片、抽血）再評估    |
| ≥ 3 項   | 符合適應症   | 達第二型發炎生物標記標準（≥ 3/5），可依規範申請給付 |

> 上表五項生物標記中需符合 **≥ 3/5 項**，此門檻用於確立「第二型發炎」體質——Dupilumab 阻斷 IL-4/IL-13 路徑，僅對第二型發炎主導的鼻息肉有效，故給付規範以此篩選最可能受益的患者。

## SNOT-22 鼻竇生活品質量表

**Sino-Nasal Outcome Test-22** 為 22 題自填問卷，每題 0–5 分，總分 0–110 分。

| 面向           | 題數 | 評估內容                     |
| -------------- | ---- | ---------------------------- |
| 鼻部症狀       | 8 題 | 鼻塞、流鼻水、嗅覺喪失等     |
| 耳部與顏面症狀 | 4 題 | 耳悶、頭暈、面部疼痛         |
| 睡眠功能       | 5 題 | 入睡困難、夜間醒來、白天疲倦 |
| 心理與情緒功能 | 5 題 | 生產力、注意力、情緒影響     |

### SNOT-22 分數解讀

| 總分  | 疾病負擔 | 意義與建議處置                             |
| ----- | -------- | ------------------------------------------ |
| 0–19  | 輕微     | 鼻竇症狀對生活影響輕微，藥物治療通常足夠   |
| 20–39 | 輕度     | 症狀已影響生活品質，建議規則藥物治療並追蹤 |
| 40–59 | 中度     | 顯著疾病負擔，評估手術或生物製劑等進階治療 |
| ≥ 60  | 重度     | 生活品質嚴重受損，建議積極的多模式治療     |

> **SNOT-22 ≥ 40 分**是健保 Dupixent 給付條件中「生活品質下降」的門檻，也常作為考慮鼻竇內視鏡手術的參考值。治療前後比較時，下降 **≥ 8.9 分（MCID）**才代表患者可實際感受到的改善；健康人平均約 7 分。
