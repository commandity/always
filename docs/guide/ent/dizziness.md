## 眩暈鑑別診斷工具

四步驟系統性評估流程：分級判斷（Triage）→ HINTS 檢查→ BPPV 診斷→ AUVP 與前庭性偏頭痛/梅尼爾氏症鑑別。

| 步驟 | 名稱           | 目的                                         |
| ---- | -------------- | -------------------------------------------- |
| 1    | Triage 分級    | 區分急性前庭症候群（AVS）vs 誘發性 vs 自發性 |
| 2    | HINTS 檢查     | 中樞性 vs 周圍性眩暈鑑別                     |
| 3    | BPPV 診斷      | 良性陣發性姿勢性眩暈（耳石症）               |
| 4    | AUVP / VM / MD | 急性前庭神經炎／前庭性偏頭痛／梅尼爾氏症鑑別 |

## DHI 頭暈障礙量表

**Dizziness Handicap Inventory**（Jacobson 1991）共 25 題，計分方式為 Yes（4 分）/ Sometimes（2 分）/ No（0 分），總分 0–100 分。

| 面向                 | 題數 | 核心評估                       |
| -------------------- | ---- | ------------------------------ |
| Functional（功能性） | 9 題 | 頭暈對日常活動與功能的限制     |
| Emotional（情緒）    | 9 題 | 頭暈引起的挫折、焦慮等情緒反應 |
| Physical（身體）     | 7 題 | 特定動作誘發的頭暈與平衡問題   |

### DHI 分數解讀

| 總分  | 障礙程度   | 意義與建議處置                                 |
| ----- | ---------- | ---------------------------------------------- |
| 0–16  | 輕微障礙   | 頭暈對日常生活影響輕微，追蹤觀察即可           |
| 17–34 | 輕至中度   | 已造成可察覺的功能限制，建議就醫評估           |
| 35–60 | 中度障礙   | 明顯前庭功能障礙，建議完整前庭評估與前庭復健   |
| > 60  | 重度障礙   | 嚴重障礙且跌倒風險升高，強烈建議積極評估與復健 |

> **DHI ≤ 16 分**視為正常範圍；**> 60 分**與反覆跌倒風險顯著相關，屬高風險族群。治療前後比較時，變化達 **18 分以上**才代表有臨床意義的改善（MCID）。Physical 面向偏高提示動作誘發性眩暈（如 BPPV），可配合上方鑑別診斷工具進一步定位病因。
