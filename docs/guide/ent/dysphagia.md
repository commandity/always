## FOIS 功能性口腔攝食量表

**FOIS（Functional Oral Intake Scale）** 由 Crary、Mann 與 Groher 於 2005 年發表，是一套用於評估吞嚥障礙患者實際經口進食功能的七點序數量表，記錄患者的實際進食行為而非症狀嚴重程度。FOIS 具有優異的評分者間信度（ICC > 0.90），是目前急性期與復健期最廣泛使用的吞嚥功能分級工具。

### FOIS 等級意義

| 等級    | 進食方式                     | 臨床意義                                     |
| ------- | ---------------------------- | -------------------------------------------- |
| Level 1 | 完全管路進食                 | 完全依賴鼻胃管或胃造口，經口進食有高度風險   |
| Level 2 | 管路依賴，少量嘗試           | 仍以管路為主，僅少量嘗試性經口進食           |
| Level 3 | 管路依賴，一致性經口補充     | 管路與經口並行，為撤管前的過渡期             |
| Level 4 | 完全經口（單一質地）         | 脫離管路，但飲食質地嚴格受限                 |
| Level 5 | 完全經口（多質地，需特別準備）| 需切碎、增稠等準備，飲食自由度部分受限      |
| Level 6 | 完全經口（無特別準備）       | 接近正常，僅需迴避特定食物                   |
| Level 7 | 完全正常經口進食             | 無任何限制                                   |

> **Level 1–3 與 Level 4–7 是關鍵分水嶺**：Level 1–3 表示患者仍依賴管路營養，需積極吞嚥復健與營養介入；達到 Level 4 以上表示已可完全經口進食，後續目標為逐步放寬質地限制。等級提升 1 級即具臨床意義，常作為復健成效與撤管決策的依據。

### FOIS 的臨床應用

| 疾病/情境       | 應用                                               |
| --------------- | -------------------------------------------------- |
| 急性腦中風      | 建立基線（Level 1–3 需立即管路介入），追蹤功能恢復 |
| 頭頸部癌症      | 手術或放射治療後功能追蹤，規劃管路撤除時機         |
| 巴金森病        | 功能惡化早期預警，提前介入飲食調整                 |
| 重症加護（ICU） | 拔管後吞嚥功能快速分級                             |

## EAT-10 吞嚥篩檢量表

**EAT-10（Eating Assessment Tool-10）** 由 Belafsky 等人於 2008 年發表，是目前全球最廣泛使用的吞嚥障礙快速自填篩檢工具，已被翻譯成 30 種以上語言，具有良好的信度、效度與敏感度。

### 臨床切截點

| EAT-10 總分 | 臨床意義           | 建議                                           |
| ----------- | ------------------ | ---------------------------------------------- |
| 0–2         | 正常範圍           | 無顯著吞嚥障礙                                 |
| ≥ 3         | 異常，需進一步評估 | 建議轉介語言治療師或耳鼻喉科                   |
| 3–10        | 輕度吞嚥障礙       | 追蹤觀察，考慮預防性介入                       |
| 11–20       | 中度吞嚥障礙       | 建議臨床吞嚥評估（CSE）或儀器評估（MBSS/FEES） |
| > 20        | 重度吞嚥障礙       | 立即就醫，評估誤嗆風險及鼻胃管需求             |

> **重要**：EAT-10 ≥ 3 分即視為陽性，建議進一步完整評估。

> **Q8 和 Q9 是誤嗆風險的重要指標**：食物卡喉及進食時咳嗽是顯性誤嗆的典型症狀。

## FOIS 與 EAT-10 的比較

| 特性         | FOIS                     | EAT-10                     |
| ------------ | ------------------------ | -------------------------- |
| 填答者       | 臨床醫師 / 語言治療師    | 患者自填                   |
| 評估內容     | 實際進食功能（能做什麼） | 吞嚥症狀嚴重度（感覺如何） |
| 格式         | 7 等級單選               | 10 題 Likert 量表          |
| 總分         | Level 1–7                | 0–40 分                    |
| 優點         | 客觀、直接反映功能       | 快速、患者主觀感受         |
| 缺點         | 需臨床人員判斷           | 不反映客觀功能             |
| 最佳使用時機 | 功能追蹤、管路撤除決策   | 症狀篩檢、主觀影響評估     |

> 臨床上建議**合併使用** FOIS 與 EAT-10：FOIS 提供客觀功能等級，EAT-10 補充患者的主觀感受，兩者互補可獲得更完整的吞嚥評估圖像。

## 注意事項

::: warning FOIS 需由受訓臨床人員評估
FOIS 屬於臨床觀察性量表，需由語言治療師或受過吞嚥評估訓練的臨床醫師根據患者的實際進食觀察結果填寫，不適合作為患者自填問卷。
:::

::: warning EAT-10 ≥ 3 分需進一步評估
EAT-10 是篩檢工具，不能作為吞嚥障礙的診斷依據。任何 EAT-10 ≥ 3 分的患者，均應由語言治療師或耳鼻喉科醫師進行完整的評估。
:::

::: warning 進食時咳嗽或嗆到請立即就醫
若患者在進食或喝水時有頻繁咳嗽、聲音沙啞（濕聲）、吞嚥後食物殘留或反覆發燒，這些可能是誤嗆的警示症狀，應優先安排完整吞嚥評估。
:::

::: info 免責聲明
本計算器僅供臨床篩檢參考，不具診斷效力，不能取代完整的吞嚥評估。若有吞嚥困難症狀，請諮詢耳鼻喉科醫師或語言治療師進行完整評估。
:::
