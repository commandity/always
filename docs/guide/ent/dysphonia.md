## VoiSS 嗓音症狀量表

**Voice Symptom Scale**（Deary 2003）共 30 題，每題 0（從不）至 4（總是），總分 0–120 分，分三個面向評估嗓音問題的影響。

| 面向               | 題數  | 核心評估                         |
| ------------------ | ----- | -------------------------------- |
| Impairment（損害） | 15 題 | 發聲功能障礙、嗓音品質變化       |
| Emotional（情緒）  | 8 題  | 嗓音問題引起的心理反應、社交迴避 |
| Physical（身體）   | 7 題  | 喉部不適感、咽喉症狀             |

### VoiSS 分數解讀

| 總分  | 臨床意義 | 建議處置                       |
| ----- | -------- | ------------------------------ |
| 0–14  | 正常範圍 | 嗓音症狀在正常範圍，無顯著障礙 |
| 15–29 | 輕度障礙 | 嗓音衛教與追蹤觀察             |
| 30–49 | 中度障礙 | 建議就醫評估，考慮嗓音治療     |
| ≥ 50  | 重度障礙 | 強烈建議喉科評估與積極治療     |

> **VoiSS ≥ 15 分**即超出正常範圍。分數同時反映功能損害與心理社會衝擊：若 Emotional 面向分數偏高，表示嗓音問題已影響社交與情緒，治療計畫應納入心理支持；Physical 面向偏高則需排除喉部器質性病變或咽喉逆流。

## VHI-10 嗓音障礙指數

**Voice Handicap Index-10**（Rosen 2004）為 VHI-30 的簡化版，10 題快速篩檢嗓音問題對生活品質的影響，每題 0–4 分，總分 0–40 分。

| 總分範圍 | 臨床意義           | 建議處置                             |
| -------- | ------------------ | ------------------------------------ |
| 0–10     | 正常或輕微嗓音影響 | 通常不需介入，可追蹤觀察             |
| 11–20    | 中度嗓音障礙       | 嗓音問題已影響生活品質，建議就醫評估 |
| > 20     | 重度嗓音障礙       | 強烈建議嗓音治療或手術評估           |

> **VHI-10 ≥ 11 分**視為異常：正常人平均僅約 2–3 分，超過 11 分表示患者主觀感受到嗓音問題對日常溝通、工作與情緒造成實質妨礙。治療前後比較時，下降 6 分以上（MCID）才代表有臨床意義的改善。

## GRBAS 嗓音品質評估

**GRBAS Scale**（Hirano 1981）為國際通用的聽覺感知嗓音評估量表，由臨床醫師依聽覺判斷評分（0–3）。

| 參數 | 名稱               | 評估內容                 |
| ---- | ------------------ | ------------------------ |
| G    | Grade 整體嘶啞度   | 嗓音偏離正常之總體程度   |
| R    | Roughness 粗糙度   | 聲帶不規則振動之粗糙音質 |
| B    | Breathiness 氣息聲 | 聲門閉合不全之漏氣聲     |
| A    | Asthenia 無力感    | 嗓音微弱無力             |
| S    | Strain 緊縮感      | 過度用力或緊繃的發聲     |

### GRBAS 評分意義

| 分數 | 嚴重度 | 意義                             |
| ---- | ------ | -------------------------------- |
| 0    | 正常   | 該項音質特徵無異常               |
| 1    | 輕度   | 偶爾或輕微可察覺的異常           |
| 2    | 中度   | 明顯且持續的異常                 |
| 3    | 重度   | 嚴重異常，音質明顯偏離正常       |

> **G（整體嘶啞度）為總結性指標**，是五項中最常用於追蹤治療成效的分數；R 與 B 分別指向聲帶振動不規則（如聲帶結節、息肉）與聲門閉合不全（如聲帶麻痺、萎縮）兩類不同病理方向，可協助縮小鑑別診斷範圍。

## 注意事項

::: warning GRBAS 需由受訓人員執行
GRBAS 為聽覺感知評估，需由語言治療師或嗓音專科醫師進行主觀判斷。
:::
