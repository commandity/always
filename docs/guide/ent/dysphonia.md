## VoiSS 嗓音症狀量表

**Voice Symptom Scale**（Deary 2003）共 30 題，分三個面向評估嗓音問題的影響。

| 面向               | 題數  | 核心評估                         |
| ------------------ | ----- | -------------------------------- |
| Impairment（損害） | 15 題 | 發聲功能障礙、嗓音品質變化       |
| Emotional（情緒）  | 8 題  | 嗓音問題引起的心理反應、社交迴避 |
| Physical（身體）   | 7 題  | 喉部不適感、咽喉症狀             |

## VHI-10 嗓音障礙指數

**Voice Handicap Index-10**（Rosen 2004）為 VHI-30 的簡化版，10 題快速篩檢嗓音問題對生活品質的影響，每題 0–4 分，總分 0–40 分。

| 總分範圍 | 臨床意義           |
| -------- | ------------------ |
| 0–10     | 正常或輕微嗓音影響 |
| 11–20    | 中度嗓音障礙       |
| > 20     | 重度嗓音障礙       |

## GRBAS 嗓音品質評估

**GRBAS Scale**（Hirano 1981）為國際通用的聽覺感知嗓音評估量表，由臨床醫師依聽覺判斷評分（0–3）。

| 參數 | 名稱               | 評估內容                 |
| ---- | ------------------ | ------------------------ |
| G    | Grade 整體嘶啞度   | 嗓音偏離正常之總體程度   |
| R    | Roughness 粗糙度   | 聲帶不規則振動之粗糙音質 |
| B    | Breathiness 氣息聲 | 聲門閉合不全之漏氣聲     |
| A    | Asthenia 無力感    | 嗓音微弱無力             |
| S    | Strain 緊縮感      | 過度用力或緊繃的發聲     |

## 注意事項

::: warning GRBAS 需由受訓人員執行
GRBAS 為聽覺感知評估，需由語言治療師或嗓音專科醫師進行主觀判斷。
:::

::: info 免責聲明
本計算器僅供評分記錄與教育參考，嗓音障礙診斷與治療需由語言治療師或耳鼻喉科醫師進行完整評估。
:::
