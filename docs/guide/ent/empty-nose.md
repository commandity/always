## ENS6Q 空鼻症問卷

**Empty Nose Syndrome 6-Item Questionnaire**（Velasquez 2017）為 6 題自填問卷，每題 0–5 分，總分 0–30 分。

| 題號 | 項目                   |
| ---- | ---------------------- |
| Q1   | 鼻腔乾燥感             |
| Q2   | 缺乏氣流通過鼻腔的感覺 |
| Q3   | 鼻腔窒息感或呼吸困難   |
| Q4   | 鼻腔感覺過度寬敞或空洞 |
| Q5   | 鼻腔結痂               |
| Q6   | 鼻腔灼熱感             |

### ENS6Q 分數解讀

| 總分  | 臨床意義       | 建議處置                                           |
| ----- | -------------- | -------------------------------------------------- |
| 0–10  | 空鼻症可能性低 | 症狀未達篩查陽性標準，觀察追蹤，尋找其他鼻部病因   |
| 11–17 | 中度疑似       | 達陽性門檻，建議棉花測試（Cotton Test）與內視鏡評估 |
| 18–23 | 重度疑似       | 症狀負擔重，建議內視鏡、影像學及棉花測試確認診斷   |
| ≥ 24  | 極重度疑似     | 積極評估介入治療（黏膜下植入物等重建手術）         |

> **ENS6Q ≥ 11 分（陽性門檻）**：原始驗證研究顯示此切截值鑑別空鼻症患者的敏感度約 87%、特異度約 96%，特別適用於下鼻甲手術後出現「鼻腔通暢卻自覺吸不到氣」（矛盾性鼻塞，Q4）的患者。**MCID = 7 分**——治療（如棉花測試暫時填塞、植入物手術）前後下降 7 分以上，才代表患者可實際感受到的改善。
