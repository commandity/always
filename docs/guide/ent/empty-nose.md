## ENS6Q 空鼻症問卷

**Empty Nose Syndrome 6-Item Questionnaire**（Velasquez 2017）為 6 題自填問卷，每題 0–5 分，總分 0–30 分。

| 題號 | 項目                   |
| ---- | ---------------------- |
| Q1   | 鼻腔乾燥感             |
| Q2   | 缺乏氣流通過鼻腔的感覺 |
| Q3   | 鼻腔窒息感或呼吸困難   |
| Q4   | 鼻腔感覺過度寬敞或空洞 |
| Q5   | 鼻腔結痂               |
| Q6   | 鼻腔灼熱感             |

| 總分 | 臨床意義       |
| ---- | -------------- |
| < 11 | 空鼻症可能性低 |
| ≥ 11 | 高度懷疑空鼻症 |

> MCID（最小臨床重要差異）= 7 分。

## 注意事項

::: info 免責聲明
本計算器僅供臨床篩檢參考，空鼻症診斷需結合鼻內視鏡及患者病史綜合判斷。
:::
