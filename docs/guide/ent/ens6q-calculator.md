# ENS6Q 空鼻症症狀評估

**ENS6Q Empty Nose Syndrome Symptom Assessment**

## 概述

**ENS6Q**（Empty Nose Syndrome 6-Item Questionnaire）由 Velasquez、Thamboo、Habib、Huang 及 Nayak 等人於 2016 年發表於 _International Forum of Allergy & Rhinology_，是目前**唯一經過效度驗證**的空鼻症（Empty Nose Syndrome, ENS）專屬患者自填問卷（Patient-Reported Outcome Measure，PROM）。本量表設計簡潔，共 6 題，可作為 SNOT-22 的輔助篩查工具，協助臨床醫師辨識疑似 ENS 患者。

## 量表架構

ENS6Q 共 **6 個問題**，評估患者目前的鼻腔症狀嚴重程度：

| 題號 | 評估症狀               | 英文                                                   |
| ---- | ---------------------- | ------------------------------------------------------ |
| Q1   | 鼻腔乾燥感             | Nasal dryness                                          |
| Q2   | 缺乏氣流通過鼻腔的感覺 | Lack of air sensation going through the nasal cavities |
| Q3   | 鼻腔窒息感或呼吸困難   | Nasal suffocation or difficulty breathing through nose |
| Q4   | 鼻腔感覺過度寬敞或空洞 | Nose feeling too open or wide                          |
| Q5   | 鼻腔結痂               | Nasal crusting                                         |
| Q6   | 鼻腔灼熱感             | Nasal burning sensation                                |

## 評分與計算

每題以 6 點 Likert 量表評分：

| 分數 | 描述                  |
| ---- | --------------------- |
| 0    | 無症狀（No symptom）  |
| 1    | 極輕微（Very mild）   |
| 2    | 輕度（Mild）          |
| 3    | 中度（Moderate）      |
| 4    | 重度（Severe）        |
| 5    | 極嚴重（Very severe） |

**計算公式：**

**ENS6Q 分數 = 六題原始分數加總**

- 分數範圍：**0–30**（分數越高，症狀越嚴重）
- **切截點（Cut-off）：≥ 11 分**提示可能為空鼻症（Velasquez et al., 2017）

## 分級與臨床意義

| ENS6Q 總分 | 嚴重程度     | 臨床建議                                                    |
| ---------- | ------------ | ----------------------------------------------------------- |
| 0–10       | 低度疑似 ENS | 目前症狀不符合 ENS 篩查陽性標準，建議觀察追蹤               |
| 11–17      | 中度疑似 ENS | 符合 ENS 疑似陽性，建議安排棉花測試及耳鼻喉科評估           |
| 18–23      | 重度疑似 ENS | 重度 ENS 症狀，強烈建議鼻腔內視鏡、影像學及棉花測試確認診斷 |
| 24–30      | 極重度疑似   | 極重度症狀，需積極評估手術（黏膜下植入物）或其他介入治療    |

> **MCID（最小臨床重要差異）**：ENS6Q 分數改善 **≥ 7 分** 視為具有臨床意義的改善（Tam et al., _Laryngoscope_ 2018）。此標準常用於手術療效評估。

## 認識空鼻症（Empty Nose Syndrome）

空鼻症（ENS）是一種後天性鼻腔功能障礙，主要發生於接受鼻甲切除手術（turbinectomy 或 turbinoplasty）後的患者。儘管鼻腔在解剖學上更為通暢，患者卻矛盾地感到呼吸困難、窒息感及氣流感消失。

### 病理機轉

正常下鼻甲（inferior turbinate）負責：

- 調節吸入氣流的溫度與濕度
- 提供鼻黏膜的氣流阻力感（air resistance sensation）
- 驅動黏膜纖毛清除功能

鼻甲組織過度切除後，以上功能受損，導致：

- 鼻腔失去加溫加濕功能 → **乾燥、結痂**
- 氣流感應受體（flow receptors）失去刺激 → **缺乏氣流感、窒息感**
- 矛盾性鼻阻塞（paradoxical nasal obstruction） → **鼻腔感覺過寬但仍無法舒適呼吸**

### 高風險族群

- 接受**全鼻甲切除術（total turbinectomy）**者風險最高
- 接受鼻甲部分切除術（partial turbinoplasty）者風險次之
- 鼻中隔矯正手術若同時過度處理鼻甲亦可能引發

## 各題臨床意義

### Q1：鼻腔乾燥感

下鼻甲的主要功能之一是對吸入空氣加濕。切除後，鼻黏膜失去足夠的表面積，吸入乾燥空氣直接接觸較深層鼻腔，造成持續乾燥感，嚴重者合併出血及反覆感染。

### Q2：缺乏氣流通過鼻腔的感覺

鼻腔內有豐富的冷受體（TRPM8）及機械性受體，感應氣流通過時的涼爽與流速。鼻甲切除後，這些受體失去足夠的氣流刺激，患者主觀上感受不到「有空氣通過」，即使客觀鼻腔阻力已下降。此為 ENS 最核心的症狀之一。

### Q3：鼻腔窒息感或呼吸困難

矛盾性症狀：鼻腔解剖上已相當通暢，患者卻感到窒息或呼吸困難，尤其在夜間或平躺時加重。機轉涉及鼻腔氣流感應異常，大腦無法接收正常「呼吸完成」的反饋訊號。此症狀與焦慮、憂鬱的共病率極高。

### Q4：鼻腔感覺過度寬敞或空洞

患者自覺鼻腔「空洞」、「太寬」，氣流流動異常（層流變亂流，失去聚焦效應）。這是 ENS 最具特異性的症狀，也是「空鼻」一名稱的由來。棉花測試（cotton test）在此症狀的確認診斷中扮演關鍵角色。

### Q5：鼻腔結痂

鼻腔加濕功能喪失導致分泌物快速乾燥形成硬殼，常需反覆以生理食鹽水沖洗或手動清除，若合併感染可出現異味及出血。

### Q6：鼻腔灼熱感

黏膜保護機制喪失（正常黏液層的潤滑及緩衝作用減弱），乾燥空氣直接刺激暴露的神經末梢，產生灼熱或刺痛感，類似乾眼症患者的眼睛症狀。

## 診斷流程

ENS6Q ≥ 11 分僅為篩查工具，確診需綜合以下評估：

```
ENS6Q ≥ 11 分
    ↓
詳細病史（鼻甲手術史、症狀發生時間點）
    ↓
鼻腔內視鏡檢查（黏膜萎縮、鼻腔過寬、結痂）
    ↓
鼻竇 CT（鼻甲體積評估）
    ↓
棉花測試（Cotton Test）✓ → 症狀改善 → 支持 ENS 診斷
    ↓
ENS 診斷確立 → 討論治療選項
```

### 棉花測試（Cotton Test）

辦公室內即可進行的簡易測試：將濕棉花置於鼻腔缺損區域（模擬鼻甲體積），若患者的 ENS6Q 相關症狀（尤其 Q2、Q3、Q4）明顯改善，高度支持 ENS 診斷，並提示手術重建可能有效。

## 治療選項

### 保守治療

| 治療方式           | 作用                       |
| ------------------ | -------------------------- |
| 高頻率鹽水沖洗     | 保濕、清除結痂             |
| 鼻腔保濕劑/凝膠    | 減輕乾燥感                 |
| 鼻腔滴油（芝麻油） | 潤滑黏膜（有限的研究支持） |
| 加濕器環境調整     | 降低乾燥氣流刺激           |
| 心理支持/CBT       | 針對合併焦慮憂鬱的輔助治療 |

### 手術治療

對棉花測試陽性且保守治療無效的重度患者：

| 手術方式                    | 說明                                               |
| --------------------------- | -------------------------------------------------- |
| 鼻甲黏膜下植入物（Implant） | 置入矽膠、骨粉或生物材料以重建鼻甲體積（主流術式） |
| 鼻甲再生術（Turbinoplasty） | 利用自體或異體組織重建下鼻甲                       |
| 富血小板血漿（PRP）注射     | 促進黏膜再生（初步研究結果令人鼓舞）               |

> 術後以 ENS6Q 追蹤療效，MCID ≥ 7 分為有效標準。

## ENS6Q 與 SNOT-22 的比較

| 特性     | ENS6Q                     | SNOT-22                    |
| -------- | ------------------------- | -------------------------- |
| 題數     | 6 題                      | 22 題                      |
| 施測時間 | 1–2 分鐘                  | 5–10 分鐘                  |
| 評分範圍 | 0–30（每題 0–5）          | 0–110（每題 0–5）          |
| 評估範圍 | ENS 核心症狀（專一性高）  | 鼻竇炎全面症狀 + 生活品質  |
| 切截點   | ≥ 11 分                   | 無單一 ENS 切截點          |
| MCID     | ≥ 7 分                    | ≥ 8.9 分                   |
| 主要用途 | ENS 篩查、術後療效評估    | CRS 評估、Dupilumab 適應症 |
| 驗證族群 | ENS vs CRSsNP vs 健康對照 | 各種鼻竇疾病               |

> ENS6Q 與 SNOT-22 具有中等程度相關（r = 0.682），兩者可互補使用於術前評估及術後追蹤。

## 心理健康共病

研究顯示 ENS 患者的焦慮（GAD-7）與憂鬱（PHQ-9/BDI-II）共病率極高：

- ENS6Q 分數與 BDI-II 憂鬱量表呈顯著正相關
- ENS6Q 分數與 BAI 焦慮量表呈顯著正相關
- 術前需常規篩查心理健康狀態，必要時轉介心理科合併治療

## 注意事項

::: warning ENS6Q 為篩查工具，非確診標準
ENS6Q ≥ 11 分提示疑似 ENS，但確診仍需結合完整病史（鼻甲手術史）、鼻腔內視鏡、影像學及棉花測試。分數低於切截點不能完全排除 ENS 的可能性。
:::

::: warning 評估時間窗
ENS6Q 評估的是患者**目前**的症狀感受，與 NOSE 量表（過去一個月）或 SNOT-22（過去兩週）的時間窗不同，使用時需向患者說明清楚。
:::

::: warning ENS 的心理社會影響
ENS 可嚴重影響患者的睡眠、社交功能及情緒健康。症狀嚴重者有較高的憂鬱及自傷風險，臨床評估時需注意心理健康狀態並提供適當支持。
:::
