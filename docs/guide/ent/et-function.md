# 耳咽管功能障礙評估

**Eustachian Tube Dysfunction · ETDQ-7**

<EtFunctionCalculator />

## ETDQ-7 耳咽管功能障礙問卷

**Eustachian Tube Dysfunction Questionnaire**（McCoul 2012）為 7 題自填問卷，每題 1–7 分，總分 7–49 分。

| 題號 | 項目                       |
| ---- | -------------------------- |
| Q1   | 耳朵壓迫或悶塞感           |
| Q2   | 耳朵疼痛感                 |
| Q3   | 耳鳴                       |
| Q4   | 耳內液體感或啪啪聲         |
| Q5   | 吞嚥或打哈欠後症狀改善     |
| Q6   | 感冒/鼻竇炎/過敏時症狀加重 |
| Q7   | 氣壓變化時耳朵不適         |

### ETDQ-7 分數解讀

計分以**各題平均分數**（總分 ÷ 7）為準：

| 平均分數  | 總分約當   | 臨床意義     | 建議處置                       |
| --------- | ---------- | ------------ | ------------------------------ |
| < 2.1     | < 14.5     | 正常範圍     | 耳咽管功能正常，無需介入       |
| 2.1–2.9   | 14.5–20    | 輕度功能障礙 | 追蹤觀察，處理鼻部過敏等誘因   |
| 3.0–4.4   | 21–31      | 中度功能障礙 | 建議耳鼻喉科評估（鼓室圖、內視鏡）|
| ≥ 4.5     | ≥ 31.5     | 重度功能障礙 | 積極治療，考慮耳咽管氣球擴張術等介入 |

> **總分 ≥ 14.5（平均 ≥ 2.1）**為診斷切截值：原始驗證研究中，此門檻區分阻塞性耳咽管功能障礙患者與健康人的敏感度與特異度均達 100%。分數越高代表耳壓調節功能越差，症狀（耳悶、氣壓變化不適）對生活的干擾越明顯；亦可用於治療前後的成效比較。

## 注意事項

::: info 免責聲明
本計算器僅供臨床篩檢參考，耳咽管功能障礙診斷需結合鼓室圖、聽力檢查及鼻咽內視鏡綜合評估。
:::
