# 耳咽管功能障礙評估

**Eustachian Tube Dysfunction · ETDQ-7**

<EtFunctionCalculator />

## ETDQ-7 耳咽管功能障礙問卷

**Eustachian Tube Dysfunction Questionnaire**（McCoul 2012）為 7 題自填問卷，每題 1–7 分，總分 7–49 分。

| 題號 | 項目                       |
| ---- | -------------------------- |
| Q1   | 耳朵壓迫或悶塞感           |
| Q2   | 耳朵疼痛感                 |
| Q3   | 耳鳴                       |
| Q4   | 耳內液體感或啪啪聲         |
| Q5   | 吞嚥或打哈欠後症狀改善     |
| Q6   | 感冒/鼻竇炎/過敏時症狀加重 |
| Q7   | 氣壓變化時耳朵不適         |

> **ETDQ-7 ≥ 14.5 分**高度提示耳咽管功能障礙。

## 注意事項

::: info 免責聲明
本計算器僅供臨床篩檢參考，耳咽管功能障礙診斷需結合鼓室圖、聽力檢查及鼻咽內視鏡綜合評估。
:::
