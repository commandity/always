# FOIS 功能性口腔攝食量表

**FOIS Functional Oral Intake Scale**

## 概述

**FOIS**（Functional Oral Intake Scale）由 Crary、Mann 與 Groher 於 2005 年發表，是一套用於評估**吞嚥障礙患者實際經口進食功能**的七點序數量表。FOIS 的特色在於它記錄的是患者的**實際進食行為**（能做什麼），而非症狀嚴重程度，因此特別適合追蹤腦中風、頭頸部癌症手術後或其他神經疾病患者的吞嚥復健進展。FOIS 具有優異的評分者間信度（ICC > 0.90），是目前急性期與復健期最廣泛使用的吞嚥功能分級工具。

## 量表架構

FOIS 分為兩大類、七個等級：

### 管路依賴（Tube Dependent，Level 1–3）

| 等級    | 名稱                 | 說明                                                   |
| ------- | -------------------- | ------------------------------------------------------ |
| Level 1 | 完全管路進食         | 完全不經口，全依賴鼻胃管（NGT）或胃造口（PEG/PEJ）     |
| Level 2 | 管路依賴，少量嘗試   | 主要依賴管路，偶爾嘗試少量經口進食（如口腔刺激或試吃） |
| Level 3 | 管路依賴，一致性補充 | 仍需管路，但可規律地經口進食特定食物作為補充           |

### 完全經口進食（Total Oral Intake，Level 4–7）

| 等級    | 名稱                   | 說明                                                     |
| ------- | ---------------------- | -------------------------------------------------------- |
| Level 4 | 單一質地               | 完全不需管路，但只能接受單一質地（如純軟食或僅增稠液體） |
| Level 5 | 多種質地，需特別準備   | 可進食多種質地，但需切碎、去骨或避免特定食物             |
| Level 6 | 多種質地，無需特別準備 | 幾乎正常，但有特定食物需迴避                             |
| Level 7 | 完全正常進食           | 無任何限制，可進食所有質地食物及液體                     |

## 各等級詳細說明

### Level 1 — 完全管路進食

患者完全不經口進食，所有營養、水分及藥物均透過鼻胃管（NGT）、胃造口（PEG）或其他腸道外方式供給。常見於急性期嚴重吞嚥障礙、意識不清或完全失去吞嚥功能的患者。

### Level 2 — 管路依賴，少量嘗試

主要仍依賴管路，但可在語言治療師監督下進行少量試吞，目的多為口腔刺激、感覺訓練或評估性試吃，而非攝取足夠的營養與水分。

### Level 3 — 管路依賴，一致性補充

患者有足夠的吞嚥功能進行規律的經口補充，但整體熱量與水分仍主要仰賴管路，經口攝取佔總量不足。常見於處於復健過渡期的患者。

### Level 4 — 完全經口進食，單一質地

患者已可完全依賴經口進食維持營養，但只接受一種食物質地（如：僅能進食泥狀食物，或僅接受濃稠液體）。此等級的患者可能仍有誤嗆風險，飲食須嚴格控管。

### Level 5 — 完全經口進食，多種質地，需特別準備

患者可進食多種質地食物，但需要廚房端的特殊準備（如：切碎肉類、避免堅果、去骨魚肉），反映吞嚥功能有改善但仍有部分限制。

### Level 6 — 完全經口進食，無需特別準備

接近正常飲食，家庭飲食通常不需特殊處理，但仍有特定食物（如難以控制的小顆粒食物、特別乾燥的食物）需要避免。

### Level 7 — 完全正常經口進食

吞嚥功能完全恢復，可進食所有質地食物及液體，無任何飲食限制。

## FOIS 的臨床應用

### 急性腦中風

腦中風後吞嚥障礙（Post-Stroke Dysphagia）發生率高達 37–78%。FOIS 可在急性期迅速建立基線（Level 1–3 需立即管路介入），並在復健過程中追蹤吞嚥功能恢復（Level 3→4→5→7 的進展）。

研究顯示，腦中風後 FOIS 等級每提升一級，吸入性肺炎風險顯著降低，住院天數縮短。

### 頭頸部癌症

手術（全喉切除、喉部分切除、舌根切除）或放射治療後，患者的吞嚥功能可能從 Level 7 急劇下降至 Level 1–2。FOIS 提供清楚的功能性描述，協助規劃管路撤除（Decannulation / Tube Withdrawal）的時機。

### 巴金森病與神經退化疾病

在漸進性疾病中，FOIS 等級的下降（如從 Level 6 退至 Level 5 或 4）能早期提示吞嚥功能惡化，有助於提前介入飲食調整及防範誤嗆。

### 重症加護（ICU）

長期插管（> 48 小時）後，約 40–60% 患者有拔管後吞嚥障礙（Post-Extubation Dysphagia），FOIS 可作為拔管後評估的快速功能分級工具。

## FOIS 與 EAT-10 的比較

| 特性         | FOIS                     | EAT-10                     |
| ------------ | ------------------------ | -------------------------- |
| 填答者       | 臨床醫師 / 語言治療師    | 患者自填                   |
| 評估內容     | 實際進食功能（能做什麼） | 吞嚥症狀嚴重度（感覺如何） |
| 格式         | 7 等級單選               | 10 題 Likert 量表          |
| 總分         | Level 1–7                | 0–40 分                    |
| 優點         | 客觀、直接反映功能       | 快速、患者主觀感受         |
| 缺點         | 需臨床人員判斷           | 不反映客觀功能             |
| 最佳使用時機 | 功能追蹤、管路撤除決策   | 症狀篩檢、主觀影響評估     |

> 臨床上建議**合併使用** FOIS 與 EAT-10：FOIS 提供客觀功能等級，EAT-10 補充患者的主觀感受，兩者互補可獲得更完整的吞嚥評估圖像。

## 管路撤除的參考標準

管路撤除（Tube Withdrawal）是吞嚥復健的重要里程碑，通常在以下條件具備時考慮：

- FOIS 達到 **Level 4** 以上（可完全經口進食）
- 儀器吞嚥評估（MBSS 或 FEES）確認誤嗆風險在可接受範圍
- 患者可攝取足夠的熱量與水分維持基本營養需求
- 意識清楚，能配合飲食質地限制

## 注意事項

::: warning FOIS 需由受訓臨床人員評估
FOIS 屬於**臨床觀察性量表**，需由語言治療師（SLP）或受過吞嚥評估訓練的臨床醫師根據患者的實際進食觀察結果填寫，不適合作為患者自填問卷。
:::

::: warning Level 1–3 患者需定期重新評估
管路依賴患者（Level 1–3）應定期（至少每週）接受吞嚥功能再評估，以決定是否可以進展至下一等級並開始試吞計畫，避免不必要的長期管路依賴。
:::

::: warning 不能單獨作為管路撤除的依據
FOIS 等級改善是管路撤除的重要參考，但最終決策應結合**儀器吞嚥評估（MBSS/FEES）**、臨床吞嚥評估及整體醫療狀況綜合判斷，不能單憑 FOIS 等級決定。
:::

本計算器僅供臨床評分記錄與教育參考，不能取代完整的吞嚥評估。吞嚥障礙的診斷、功能分級及治療計畫應由語言治療師或耳鼻喉科醫師共同制定。
:::
