# GRBAS 嗓音品質評估

**GRBAS Voice Quality Assessment**

## 概述

**GRBAS 量表**由日本語音學者 Hirano 於 1981 年提出，並經日本語音語言障礙研究委員會（Japan Society of Logopedics and Phoniatrics）標準化，是目前全球最廣泛使用的**聽知覺嗓音品質分析工具**。GRBAS 透過訓練有素的臨床醫師或語言治療師，依據聆聽患者嗓音後的主觀感知，對五個嗓音維度進行量化評分，可快速且有效地描述嗓音障礙的性質與嚴重程度。

## 量表架構

GRBAS 為五個英文字母的縮寫，代表五個獨立的嗓音品質面向：

| 字母  | 中文名稱   | 英文        | 核心概念                                   |
| ----- | ---------- | ----------- | ------------------------------------------ |
| **G** | 整體嘶啞度 | Grade       | 嗓音整體偏離正常的程度，為綜合性嚴重度指標 |
| **R** | 粗糙度     | Roughness   | 聲帶不規則振動產生的聲音不穩定感           |
| **B** | 氣息聲     | Breathiness | 聲門閉合不全造成的漏氣成分                 |
| **A** | 無力感     | Asthenia    | 聲音微弱、無力、音量小且音調偏低           |
| **S** | 緊縮感     | Strain      | 聲音過度用力或緊繃的超功能性特質           |

## 評分標準

每個參數均以 **0–3 分**的四點量表評分：

| 分數 | 程度 | 臨床描述                   |
| ---- | ---- | -------------------------- |
| 0    | 正常 | 無此特質，嗓音品質正常     |
| 1    | 輕度 | 輕微可察覺，但不明顯       |
| 2    | 中度 | 明顯存在，清楚可辨         |
| 3    | 重度 | 非常明顯，嚴重影響嗓音品質 |

評分結果通常以字串形式記錄，例如 **G2R2B1A0S1**，清楚呈現各維度的嚴重程度。

## 各參數詳細說明

### G — 整體嘶啞度（Grade）

G 是 GRBAS 中最重要的參數，代表**嗓音整體障礙的嚴重程度**，是其他四個參數的綜合表現。G 分數通常等於或高於其他任一參數。臨床上常以 G 分數單獨作為嗓音障礙嚴重度的快速指標：

- **G0**：嗓音正常
- **G1**：輕度嗓音障礙（多數人能辨識但不明顯）
- **G2**：中度障礙（明顯嘶啞，影響溝通）
- **G3**：重度障礙（嚴重嘶啞或失聲）

### R — 粗糙度（Roughness）

R 反映聲帶**振動不規則性**，感知上呈現為聲音的顫抖感、不穩定感或粗糙質感。主要對應聲學參數中的 **Jitter**（基頻微擾）與 **Shimmer**（振幅微擾）升高。

常見於：

- 聲帶息肉（Vocal Cord Polyp）
- 聲帶結節（Vocal Nodules）
- 聲帶腫瘤（Laryngeal Tumor）
- 聲帶麻痺（Vocal Cord Paralysis）

### B — 氣息聲（Breathiness）

B 反映**聲門閉合不全**（Glottal Insufficiency），使氣流在發聲時洩漏，產生氣息音成分。對應聲學參數中的**諧噪比**（HNR）降低及噪音成分增加。

常見於：

- 聲帶萎縮（Vocal Fold Atrophy / Presbyphonia）
- 單側聲帶麻痺
- 聲帶溝（Sulcus Vocalis）
- 喉部腫瘤（大型病灶）

### A — 無力感（Asthenia）

A 反映聲音整體的**音量不足與能量虛弱**，聲音細小、無力，缺乏共鳴。不同於氣息聲，無力感主要源於肌肉力量不足而非結構問題。

常見於：

- 肌肉張力性低下型發聲障礙
- 重症肌無力（Myasthenia Gravis）
- 系統性疾病或長期臥床後的去功能化
- 老年嗓音（Presbyphonia）

### S — 緊縮感（Strain）

S 反映聲音的**超功能性（Hyperfunctional）特質**，聲音緊繃、費力、高度緊張感，音調可能偏高。與 A（無力）呈對立面。

常見於：

- 肌肉張力性高張型發聲障礙（MTD）
- 痙攣性發聲障礙（Spasmodic Dysphonia）
- 聲帶結節（過度補償性發聲）
- 功能性發聲障礙（心因性）

## 典型嗓音疾病的 GRBAS 特徵

| 疾病           | 典型 GRBAS 特徵          | 說明                         |
| -------------- | ------------------------ | ---------------------------- |
| 聲帶結節       | G1–2, R1–2, B0–1, A0, S1 | 輕中度嘶啞，伴粗糙，輕微費力 |
| 聲帶息肉       | G2–3, R2–3, B1, A0, S0–1 | 明顯粗糙，不規則振動顯著     |
| 單側聲帶麻痺   | G2–3, R1, B2–3, A1–2, S0 | 氣息聲為主，閉合不全         |
| 聲帶萎縮       | G1–2, R0–1, B2, A2, S0   | 老年型，氣息聲與無力感並存   |
| 痙攣性發聲障礙 | G2, R1–2, B0, A0, S2–3   | 緊縮感顯著，頻繁聲音中斷     |
| 聲帶腫瘤       | G3, R2–3, B1–2, A1, S0–1 | 多面向嚴重障礙               |

## 臨床應用

GRBAS 廣泛應用於以下場合：

- **初始嗓音評估**：建立基線嗓音品質描述
- **喉部疾病診斷輔助**：配合喉鏡（Laryngoscopy / Stroboscopy）判斷病因
- **治療反應監測**：嗓音治療（Voice Therapy）、手術或 Botox 注射前後比較
- **手術適應症評估**：聲帶顯微手術、聲帶注射填充術前評估
- **嗓音職業評估**：歌手、教師、演講者等嗓音高需求職業的定期評估
- **研究與教學**：嗓音障礙的標準化描述工具

## 評估注意事項

GRBAS 屬於**聽知覺主觀評估**，評分結果受以下因素影響：

- **評分者訓練程度**：需有嗓音評估經驗的語言治療師或耳鼻喉科醫師
- **錄音品質**：建議在安靜環境以標準錄音設備錄製持續母音（如 /a/）
- **評分者間信度**：G 參數的信度較高，A 與 S 的評分者間信度相對較低
- **情境效應**：應同時評估對話語音與持續母音，以獲得更完整的資訊

## 與客觀聲學分析的關係

GRBAS 常與客觀聲學分析（Acoustic Analysis）互補使用：

| GRBAS 參數 | 對應聲學指標                            |
| ---------- | --------------------------------------- |
| G（整體）  | HNR（諧噪比）、NHR（噪諧比）            |
| R（粗糙）  | Jitter（基頻微擾）、Shimmer（振幅微擾） |
| B（氣息）  | HNR 降低、噪音能量增加                  |
| A（無力）  | 最大發聲時間（MPT）縮短、音量降低       |
| S（緊縮）  | 基頻（F0）偏高、頻譜噪音特徵            |

## 注意事項

::: warning GRBAS 需由受訓的臨床人員評估
GRBAS 是**聽知覺評估工具**，需由受過嗓音評估訓練的語言治療師或耳鼻喉科醫師執行，不適合作為患者自填問卷使用。本工具提供臨床評分記錄與教育說明，評分結果的準確性取決於評估者的臨床經驗。
:::

::: warning 建議搭配喉鏡檢查
G ≥ 2 或有 R、B 明顯升高的患者，強烈建議接受**喉鏡或頻閃喉鏡**（Stroboscopy）檢查，以確認聲帶結構異常（如息肉、結節、腫瘤或麻痺）並排除惡性疾病。
:::

::: warning 嗓音持續嘶啞超過兩週請就醫
成人嗓音嘶啞持續超過**兩週**，尤其合併吸菸史、頸部腫塊或吞嚥困難，應盡快就醫排除喉部惡性腫瘤。
:::

::: info 免責聲明
本計算器僅供臨床評分記錄與教育參考，不具診斷效力。嗓音障礙的診斷與治療應由耳鼻喉科醫師或嗓音語言治療師進行完整評估。
:::
