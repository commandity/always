# 聽覺障礙鑑定計算器（b230）

**Hearing Impairment Assessment Calculator (b230)**
<HearingHandicapCalculator />

## 計算公式

### 第一步：平均聽力閾值（PTA）

```
PTA = ( 500 Hz + 1000 Hz + 2000 Hz + 4000 Hz ) ÷ 4
```

- 採用**純音氣導聽閾**（Pure-Tone Air-Conduction Threshold，dB HL）
- 右耳（右耳 PTA）與左耳（左耳 PTA）分別計算
- 結果四捨五入至小數點後一位

### 第二步：單耳聽障比率（超過 100% 以 100% 計算）

**單耳比率 = ( PTA − 25 ) × 1.5%**

- 若 PTA ≤ 25 dB → 單耳比率 = 0%
- 若計算結果 > 100% → 上限為 100%
- 右耳與左耳各自計算

### 第三步：雙耳整體障礙比率

```
雙耳整體比率 = ( 優耳比率 × 5 + 劣耳比率 × 1 ) ÷ 6
```

- **優耳**：聽力閾值較低（聽力較佳）的耳朵
- **劣耳**：聽力閾值較高（聽力較差）的耳朵
- 優耳佔較大權重（5:1）

## 障礙等級鑑定標準（b230 編碼）

依衛生福利部官方標準，以**雙耳整體障礙比率**判定編碼：

| 編碼   | 障礙等級         | 條件                                                                                                       |
| ------ | ---------------- | ---------------------------------------------------------------------------------------------------------- |
| b230.0 | 未達下列基準     | 雙耳整體障礙比率 < 45%，且不符合 b230.1 單耳條件                                                           |
| b230.1 | 輕度聽覺功能障礙 | 雙耳整體障礙比率 **45.0%–70.0%**；或一耳聽力關值超過 90 dB（含）以上，且另一耳聽力關值超過 48 dB（含）以上 |
| b230.2 | 中度聽覺功能障礙 | 雙耳整體障礙比率 **70.1%–90.0%**                                                                           |
| b230.3 | 重度聽覺功能障礙 | 雙耳整體障礙比率 **> 90.1%**                                                                               |

> **備註**：如無法取得純音聽力閾值者，以 **ABR（聽性腦幹反應）聽力閾值**作為純音聽力閾值計算。

## 計算範例

**範例**：右耳 500 Hz = 60、1k Hz = 70、2k Hz = 80、4k Hz = 90；左耳全部 = 40

| 步驟     | 右耳                                       | 左耳                          |
| -------- | ------------------------------------------ | ----------------------------- |
| PTA      | (60+70+80+90)/4 = **75.0 dB**              | (40+40+40+40)/4 = **40.0 dB** |
| 單耳比率 | (75−25)×1.5 = **75.0%**                    | (40−25)×1.5 = **22.5%**       |
| 雙耳整體 | (22.5×5 + 75×1)/6 = **31.25%** → **31.3%** | —                             |

> 優耳為左耳（PTA 較低），劣耳為右耳。雙耳整體 31.3% < 45%，且右耳 PTA = 75 dB（未達 90 dB），故**未達身心障礙標準（b230.0）**。

## 使用說明

1. 輸入右耳四個頻率（500、1000、2000、4000 Hz）的純音氣導聽閾（dB HL）
2. 按 **Enter 鍵**自動跳至下一格
3. 輸入左耳四個頻率的數值
4. 系統即時計算 PTA、單耳比率、雙耳整體比率並判定障礙等級
5. 點選「查看評估結果」可查看完整計算明細
6. 點選「複製 Markdown 結果」可將報告複製至剪貼簿

## 注意事項

::: warning 正式鑑定須由醫師執行
聽覺障礙之正式鑑定須由**具資格的醫師**（通常為耳鼻喉科），依據合格聽力檢查機構（需符合信度標準）出具的純音聽力檢查報告，填寫《身心障礙者鑑定表》後送衛生局核定。本計算器結果僅供參考，不具法律效力。
:::

::: warning 氣導聽閾的有效性
本計算使用純音**氣導聽閾**。若懷疑有傳導性聽損（氣骨導差 > 10 dB）、或偽聾（聲導抗結果不一致），正式鑑定時應同時參考**骨導聽閾**及客觀聽力檢查（如 ABR、ASSR），以確保數值的可靠性。
:::

:::
