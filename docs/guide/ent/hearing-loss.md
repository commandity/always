# 聽覺障礙鑑定計算器

**Hearing Handicap Assessment · Pure Tone Audiometry**

<HearingLossCalculator />

## 計算方式

本工具依據純音聽力檢查結果，分別計算雙耳各頻率（500、1000、2000、4000 Hz）之平均聽力閾值，並依臺灣身心障礙鑑定基準判定障礙等級。

| 障礙等級 | 基準                    |
| -------- | ----------------------- |
| 輕度     | 優耳聽力損失 55–69 dBHL |
| 中度     | 優耳聽力損失 70–89 dBHL |
| 重度     | 優耳聽力損失 ≥ 90 dBHL  |

## 注意事項

::: warning 需由聽力師執行檢查
純音聽力檢查應由聽力師在標準隔音室中執行，結果僅供臨床參考。
:::

::: info 免責聲明
本計算器僅供教育參考，正式鑑定需由衛生福利部指定機構執行。
:::
