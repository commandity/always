# 聽覺障礙鑑定計算器

**Hearing Handicap Assessment · Pure Tone Audiometry**

<HearingLossCalculator />

## 計算方式

本工具依據純音聽力檢查結果，分三步驟計算「雙耳整體障礙比率」，並依臺灣身心障礙鑑定基準（ICF 編碼 b230）判定障礙等級：

| 步驟 | 計算             | 說明                                                  |
| ---- | ---------------- | ----------------------------------------------------- |
| 1    | 各耳 PTA         | 500、1000、2000、4000 Hz 四頻率聽力閾值之平均（dBHL） |
| 2    | 單耳障礙比率     | （PTA − 25）× 1.5%；PTA ≤ 25 dBHL 視為 0%（正常聽力） |
| 3    | 雙耳整體障礙比率 | （優耳比率 × 5 ＋ 劣耳比率）÷ 6，優耳權重較高         |

### 障礙等級判定與意義

| 整體障礙比率 | 障礙等級         | ICF 編碼 | 意義                                           |
| ------------ | ---------------- | -------- | ---------------------------------------------- |
| < 45%        | 未達身心障礙標準 | b230.0   | 聽力損失未達法定障礙程度                       |
| 45–70%       | 輕度聽覺功能障礙 | b2301    | 達身心障礙資格門檻，可申請鑑定與相關福利       |
| 70.1–90%     | 中度聽覺功能障礙 | b2302    | 日常口語溝通明顯受限，通常需助聽器輔助         |
| > 90%        | 重度聽覺功能障礙 | b2303    | 接近全聾，需評估人工電子耳或手語等替代溝通方式 |

> **45% 是身心障礙資格的關鍵門檻**。另有單耳條款：單耳 PTA ≥ 90 dBHL 且對側耳 PTA ≥ 48 dBHL 時，即使整體比率未達 45%，仍符合輕度障礙（b2301）。優耳加權（× 5）反映日常聽覺功能主要取決於較好的那隻耳朵——單側全聾但對側正常者，整體比率仍偏低。

## 注意事項

::: warning 需由聽力師執行檢查
純音聽力檢查應由聽力師在標準隔音室中執行，結果僅供臨床參考。
:::

::: info 免責聲明
本計算器僅供教育參考，正式鑑定需由衛生福利部指定機構執行。
:::
