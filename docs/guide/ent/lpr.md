## RFS 逆流表現評分

**Reflux Finding Score**（Belafsky 2001）為內視鏡下評估咽喉逆流相關喉部徵象的客觀評分系統，由臨床醫師依喉鏡檢查結果填寫，共 8 項發現，總分 0–26 分。

| 面向     | 項目                                                                                      | 給分範圍        |
| -------- | ----------------------------------------------------------------------------------------- | --------------- |
| 喉部徵象 | 聲門下水腫、喉室消失、紅斑/充血、聲帶水腫、瀰漫性喉部水腫、後聯合部肥厚、肉芽腫、喉部黏液 | 0–4（依各項目） |

### RFS 分數解讀

| 總分  | 臨床意義 | 建議處置                                 |
| ----- | -------- | ---------------------------------------- |
| 0–7   | 正常範圍 | 喉部徵象不足以支持 LPR，考慮其他病因     |
| 8–11  | 輕度 LPR | 配合 RSI 問卷評估，考慮試驗性 PPI 治療   |
| 12–17 | 中度 LPR | 建議積極 PPI 治療，考慮 24 小時 pH 監測  |
| ≥ 18  | 重度 LPR | 積極治療並密切追蹤，評估進一步介入之需求 |

> **RFS ≥ 7 分**高度提示咽喉逆流：原始驗證研究顯示，RFS > 7 者有 95% 的機率經雙探頭 pH 監測證實存在逆流。分數越高，代表喉部黏膜受逆流物傷害的內視鏡徵象越多、越嚴重；治療有效時 RFS 應隨追蹤逐步下降。

## RSI 逆流症狀指數

**Reflux Symptom Index**（Belafsky 2002）為 9 題自填問卷，評估咽喉逆流相關症狀嚴重度，每題 0（無）至 5（極嚴重），總分 0–45 分。

| 題號 | 症狀             | 面向   |
| ---- | ---------------- | ------ |
| Q1   | 聲音嘶啞         | 嗓音   |
| Q2   | 清嗓動作         | 咽喉   |
| Q3   | 痰多或鼻涕倒流   | 咽喉   |
| Q4   | 吞嚥困難         | 吞嚥   |
| Q5   | 進食或躺下後咳嗽 | 呼吸道 |
| Q6   | 呼吸困難或窒息感 | 呼吸道 |
| Q7   | 令人煩惱的咳嗽   | 呼吸道 |
| Q8   | 喉嚨異物感       | 咽喉   |
| Q9   | 心灼熱/胃酸逆流  | 消化道 |

### RSI 分數解讀

| 總分  | 臨床意義    | 建議處置                                   |
| ----- | ----------- | ------------------------------------------ |
| 0–13  | 正常範圍    | 症狀不足以支持 LPR                         |
| 14–20 | 輕至中度    | 建議就醫評估，考慮試驗性治療與生活型態調整 |
| 21–30 | 中度 LPR    | 建議喉鏡評估（RFS），考慮 PPI 治療         |
| > 30  | 重度 LPR    | 積極治療並追蹤，考慮 24 小時 pH 監測       |

> **RSI ≥ 13 分**高度提示咽喉逆流：健康人 RSI 平均約 11.6 分，超過 13 分即超出正常上限（95 百分位）。分數反映患者主觀症狀負擔——分數越高，症狀對生活的干擾越明顯；有效治療後 RSI 通常於 2–6 個月內明顯下降。

## 注意事項

::: warning RFS 需由臨床醫師執行
RFS 為內視鏡觀察評分，需由耳鼻喉科醫師或受訓醫療人員執行喉鏡檢查後填寫，不適用於患者自評。
:::

::: warning RSI 為主觀症狀篩檢
RSI 雖為患者自填，但咽喉逆流症狀與其他疾病（過敏性鼻炎、氣喘、慢性咳嗽）高度重疊，陽性結果需結合 RFS 及聽覺感知評估綜合判斷。
:::

:::
