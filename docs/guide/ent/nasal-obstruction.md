# 鼻腔阻塞症狀評估

**Nasal Obstruction Symptom Evaluation · NOSE**

<NasalObstructionCalculator />

## NOSE 量表

**Nasal Obstruction Symptom Evaluation**（Stewart 2004）為 5 題自填問卷，每題 0–4 分，原始總分 × 5 得標準化總分 0–100 分。

| 題號 | 項目                   |
| ---- | ---------------------- |
| Q1   | 鼻塞或鼻腔充血感       |
| Q2   | 鼻腔阻塞感             |
| Q3   | 無法用鼻子呼吸         |
| Q4   | 睡眠困擾               |
| Q5   | 運動時無法獲得足夠空氣 |

### NOSE 分數解讀

| 總分   | 嚴重度 | 意義與建議處置                                     |
| ------ | ------ | -------------------------------------------------- |
| 0–25   | 輕微   | 鼻塞對生活影響小，觀察或保守治療（鼻噴劑）即可     |
| 26–50  | 中度   | 症狀已影響睡眠或活動，建議就醫評估治療選項         |
| 51–75  | 重度   | 明顯影響生活品質，藥物效果不佳時評估手術矯正       |
| 76–100 | 極重度 | 強烈建議手術評估（鼻中隔矯正、下鼻甲縮小術等）     |

> 分數反映患者主觀鼻塞負擔：**> 50 分**通常被視為考慮結構性手術的參考區間——原始驗證研究中，接受鼻中隔成形術的患者術前平均約 65 分，術後多降至 25 分以下。夜間症狀（Q4）與運動症狀（Q5）偏高，分別提示睡眠呼吸障礙與動態鼻閥塌陷的可能，可作為進一步檢查的方向。
