# 鼻腔阻塞症狀評估

**Nasal Obstruction Symptom Evaluation · NOSE**

<NasalObstructionCalculator />

## NOSE 量表

**Nasal Obstruction Symptom Evaluation**（Stewart 2004）為 5 題自填問卷，每題 0–4 分，原始總分 × 5 得標準化總分 0–100 分。

| 題號 | 項目                   |
| ---- | ---------------------- |
| Q1   | 鼻塞或鼻腔充血感       |
| Q2   | 鼻腔阻塞感             |
| Q3   | 無法用鼻子呼吸         |
| Q4   | 睡眠困擾               |
| Q5   | 運動時無法獲得足夠空氣 |

| 總分   | 嚴重度 |
| ------ | ------ |
| 0–25   | 輕微   |
| 26–50  | 中度   |
| 51–75  | 重度   |
| 76–100 | 極重度 |

## 注意事項

::: info 免責聲明
本計算器僅供臨床篩檢參考，鼻腔阻塞診斷需結合鼻內視鏡及影像學檢查。
:::
