# NOSE 鼻腔阻塞症狀評估

**NOSE Nasal Obstruction Symptom Evaluation**
<NoseCalculator />

## 量表架構

NOSE 量表共 **5 個問題**，評估過去**一個月內**鼻腔阻塞對患者生活的影響：

| 題號 | 評估症狀                     | 英文                                                  |
| ---- | ---------------------------- | ----------------------------------------------------- |
| Q1   | 鼻塞或鼻腔充血感             | Nasal congestion or stuffiness                        |
| Q2   | 鼻腔阻塞感                   | Nasal blockage or obstruction                         |
| Q3   | 無法用鼻子呼吸               | Unable to breathe through the nose                    |
| Q4   | 睡眠困擾                     | Trouble sleeping                                      |
| Q5   | 運動或用力時無法獲得足夠空氣 | Unable to get enough air through nose during exercise |

## 評分與計算

每題以 5 點 Likert 量表評分：

| 分數 | 描述                           |
| ---- | ------------------------------ |
| 0    | 不是問題（Not a problem）      |
| 1    | 極輕微（Very mild problem）    |
| 2    | 輕度（Mild or slight problem） |
| 3    | 中度（Moderate problem）       |
| 4    | 嚴重（Severe problem）         |

**計算公式：**

**NOSE 分數 = 五題原始分數加總 × 5**

- 原始分數範圍：0–20
- NOSE 分數範圍：**0–100**（分數越高，阻塞越嚴重）

## 嚴重度分級

| NOSE 分數 | 嚴重程度   | 臨床意義                       |
| --------- | ---------- | ------------------------------ |
| 0         | 無阻塞     | 無鼻腔阻塞症狀                 |
| 5–25      | 輕度阻塞   | 輕微影響，建議保守治療觀察     |
| 30–50     | 中度阻塞   | 中等程度影響生活，建議就醫評估 |
| 55–75     | 重度阻塞   | 明顯影響生活品質，建議積極治療 |
| 80–100    | 極重度阻塞 | 嚴重影響，強烈建議手術介入評估 |

> **MCID（最小臨床重要差異）**：治療後 NOSE 分數改善 **≥ 15–20 分**，視為具有臨床意義的改善。此標準常用於術後療效評估。

## 各題臨床意義

### Q1：鼻塞或鼻腔充血感

涵蓋主觀上的鼻腔「塞住」感，可能源於鼻中隔彎曲、鼻甲肥大、黏膜腫脹或鼻息肉。

### Q2：鼻腔阻塞感

評估鼻腔在靜態下的通暢程度，較 Q1 更偏向物理性阻塞感。與 Q1 高度相關但各有不同的患者感受角度。

### Q3：無法用鼻子呼吸

直接評估鼻腔呼吸功能，是 NOSE 量表中對整體分數貢獻最大的題目（荷蘭語版驗證研究結果）。分數高者常見代償性口呼吸。

### Q4：睡眠困擾

鼻腔阻塞是睡眠障礙的常見原因之一。夜間鼻腔塌陷、張口呼吸、打鼾乃至阻塞性睡眠呼吸暫停（OSA）都可能與此相關，此題對生活品質的衝擊尤為重要。

### Q5：運動時呼吸困難

評估動態活動中的鼻腔功能限制，對年輕、活動量大的患者特別有鑑別價值。

## 常見適應症與臨床應用

| 適應症                              | NOSE 的角色                            |
| ----------------------------------- | -------------------------------------- |
| 鼻中隔彎曲（Deviated Nasal Septum） | 術前嚴重度評估、鼻中隔矯正術後療效追蹤 |
| 下鼻甲肥大                          | 鼻甲縮小術（射頻、電刀、切除）前後比較 |
| 過敏性鼻炎                          | 藥物治療或免疫治療反應評估             |
| 慢性鼻炎                            | 保守治療或手術前後的症狀變化記錄       |
| 功能性鼻整形術                      | 手術適應症確立與術後功能改善評估       |
| 慢性鼻竇炎（CRS）                   | 常與 SNOT-22 合併使用，評估阻塞成分    |

## NOSE 與 SNOT-22 的比較

| 特性       | NOSE                     | SNOT-22                   |
| ---------- | ------------------------ | ------------------------- |
| 題數       | 5 題                     | 22 題                     |
| 施測時間   | 1–2 分鐘                 | 5–10 分鐘                 |
| 評估範圍   | 鼻腔阻塞（專一性高）     | 鼻竇炎全面症狀 + 生活品質 |
| 評估時間窗 | 過去一個月               | 過去兩週                  |
| 最高分     | 100 分                   | 110 分                    |
| 主要用途   | 鼻阻塞、鼻整形、鼻甲手術 | CRS、Dupilumab 適應症評估 |
| MCID       | ≥ 15–20 分               | ≥ 8.9 分                  |

> NOSE 與 SNOT-22 可互補使用：NOSE 評估鼻腔阻塞的嚴重度，SNOT-22 評估完整的鼻竇炎症狀與生活品質影響。

## 手術決策參考

研究顯示 NOSE 分數與手術必要性的關聯：

- **NOSE < 30**：鼻腔阻塞較輕微，手術效益有限，建議先嘗試保守治療
- **NOSE 30–50**：中等程度，可考慮手術，需結合客觀評估（鼻腔內視鏡、CT）
- **NOSE ≥ 55**：重度以上阻塞，手術介入通常有顯著改善

術後追蹤建議：術後 **3 個月**、**6 個月**及 **12 個月** 重複施測，以客觀記錄功能恢復。

## 注意事項

::: warning 評估時間窗為過去一個月
NOSE 量表詢問的是過去**一個月**的症狀感受，不同於 SNOT-22 的兩週時間窗。若患者近期有急性上呼吸道感染或過敏發作，建議在症狀穩定後重新評估。
:::

::: warning NOSE 為單一症狀量表
NOSE 僅評估鼻腔阻塞這一核心症狀，不涵蓋嗅覺、流鼻水、後鼻滴、顏面疼痛或情緒功能等面向。對於慢性鼻竇炎患者，建議同時使用 SNOT-22 進行全面評估。
:::
