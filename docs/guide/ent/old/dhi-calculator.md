# DHI 頭暈障礙量表

**DHI Dizziness Handicap Inventory**
<DhiCalculator />

## 評分說明

每題三個選項，依症狀對生活的影響頻率作答：

| 分數 | 中文 | 英文      | 說明                         |
| ---- | ---- | --------- | ---------------------------- |
| 4    | 總是 | Yes       | 此情況總是發生，造成顯著影響 |
| 2    | 有時 | Sometimes | 此情況偶爾發生               |
| 0    | 從不 | No        | 此情況從未發生或不適用       |

**總分範圍：0–100 分**（F 面向 0–36，E 面向 0–36，P 面向 0–28）

## 臨床切截點

| DHI 總分 | 嚴重程度     | 臨床建議                                 |
| -------- | ------------ | ---------------------------------------- |
| 0–16     | 輕微障礙     | 頭暈對日常生活影響輕微，可觀察追蹤       |
| 17–34    | 輕至中度障礙 | 建議就醫評估，考慮前庭功能檢查           |
| 35–60    | 中度障礙     | 建議積極評估，安排前庭復健治療           |
| > 60     | 重度障礙     | 嚴重障礙，強烈建議完整前庭評估及長期復健 |

> **關鍵切截值：DHI > 16** 提示頭暈造成顯著障礙（Jacobson & Newman 1990）。部分研究以 **> 18** 為篩選門檻。治療後 DHI 改善 **≥ 18 分**被視為臨床有意義的變化（MCID）。

## 三個面向的臨床意義

| 面向      | 題數 | 最高分 | 評估重點                                     |
| --------- | ---- | ------ | -------------------------------------------- |
| 功能（F） | 9 題 | 36 分  | 頭暈對日常活動、工作、旅行、家務的功能性限制 |
| 情緒（E） | 9 題 | 36 分  | 頭暈引起的沮喪、焦慮、恐懼、尷尬等情緒影響   |
| 身體（P） | 7 題 | 28 分  | 特定頭部動作或姿勢改變誘發頭暈的身體反應     |

> **身體面向**（P）主要反映前庭系統對頭部動作的敏感度，在良性陣發性姿勢性頭暈（BPPV）患者中常見 P 面向分數較高；**情緒面向**（E）則與焦慮、恐慌症及慢性主觀性頭暈（PPPD）密切相關。

| 題號 | 面向 | 症狀描述             | 臨床意義                            |
| ---- | ---- | -------------------- | ----------------------------------- |
| Q1   | P    | 抬頭向上看           | 耳石或半規管功能障礙誘發            |
| Q2   | E    | 沮喪                 | 慢性頭暈的情緒負擔                  |
| Q3   | F    | 限制旅行             | 日常生活功能限制的重要指標          |
| Q4   | P    | 超市走道行走困難     | 視覺依賴性頭暈（Visual Dependency） |
| Q5   | F    | 起床或上床困難       | 姿勢轉換誘發頭暈（常見於 BPPV）     |
| Q6   | F    | 限制社交             | 社交退縮，影響生活品質              |
| Q7   | F    | 閱讀困難             | 眼球追蹤引起的視覺誘發性頭暈        |
| Q8   | P    | 費力活動加劇         | 前庭功能代償能力下降                |
| Q9   | E    | 害怕獨自外出         | 跌倒恐懼與行動限制                  |
| Q10  | E    | 社交場合尷尬         | 心理社交障礙                        |
| Q11  | P    | 快速移動頭部         | 前庭敏感度升高的直接測試            |
| Q12  | F    | 避免高處             | 高度相關的恐懼與功能限制            |
| Q13  | P    | 床上翻身             | BPPV 的高度特異性症狀               |
| Q14  | F    | 家務或庭院工作困難   | 日常生活功能指標                    |
| Q15  | E    | 害怕被誤認為醉酒     | 步態不穩引起的社交焦慮              |
| Q16  | F    | 散步困難             | 基本行動功能受損                    |
| Q17  | P    | 人行道行走加劇       | 地面不平引起的本體感覺挑戰          |
| Q18  | E    | 注意力困難           | 認知負荷影響，與慢性頭暈相關        |
| Q19  | F    | 黑暗中室內行走困難   | 視覺剝奪下的平衡代償失能            |
| Q20  | E    | 害怕獨自在家         | 安全感喪失、行動限制的焦慮指標      |
| Q21  | F    | 感覺殘障             | 整體生活品質與自我效能              |
| Q22  | E    | 人際關係壓力         | 頭暈對家庭與社交關係的衝擊          |
| Q23  | E    | 憂鬱                 | 慢性頭暈引起的情緒低落              |
| Q24  | F    | 工作或家庭責任受影響 | 職業與生活功能的全面評估            |
| Q25  | P    | 彎腰加劇             | 頭部位置改變誘發，常見於 BPPV       |

> **Q13-床上翻身**和 **Q25-彎腰**在 BPPV 患者中幾乎必然為「總是」，是臨床快速篩查 BPPV 的重要線索。

## 常見頭暈疾病的 DHI 特徵

| 疾病                           | 典型 DHI 分數 | 面向特徵               |
| ------------------------------ | ------------- | ---------------------- |
| 良性陣發性姿勢性頭暈（BPPV）   | 20–50         | P 面向顯著升高         |
| 前庭神經炎急性期               | 50–80         | 三個面向均高           |
| 梅尼爾氏病                     | 40–70         | F 和 E 面向顯著        |
| 持續性姿勢－知覺性頭暈（PPPD） | 50–90         | E 面向顯著，慢性化     |
| 前庭偏頭痛                     | 30–60         | 發作性，治療後改善明顯 |
| 中樞性頭暈                     | 50–100        | 因病灶位置而異         |

## 前庭復健治療的監測

### 治療後再評估

- **治療前**：建立基準 DHI 分數，記錄三個面向分布
- **治療後 4–8 週**：重新評估，改善 **≥ 18 分**視為有臨床意義的進步
- **長期追蹤**：每 3–6 個月評估，監測慢性化傾向

### 治療反應預測

| 特徵                 | 預後                         |
| -------------------- | ---------------------------- |
| 初始 DHI < 36        | 前庭復健反應較快             |
| P 面向分數高         | 前庭習慣化訓練效果好         |
| E 面向分數高（> 24） | 需合併心理治療或認知行為療法 |
| BPPV 確診後耳石復位  | DHI 通常在 1–4 週內顯著下降  |

### 藥物與非藥物治療

- **前庭抑制劑（如 Dimenhydrinate、Meclizine）**：急性期緩解症狀，不建議長期使用
- **前庭復健訓練（Vestibular Rehabilitation Therapy, VRT）**：慢性前庭功能障礙的主要治療
- **耳石復位術（Epley Maneuver）**：BPPV 首選治療，成功率 > 85%
- **認知行為治療（CBT）**：情緒面向（E）分數高者，尤其是 PPPD

## 注意事項

::: warning DHI 是自填問卷，不能單獨診斷前庭疾病
DHI 高分提示頭暈造成顯著障礙，但不能區分 BPPV、梅尼爾氏病、前庭神經炎、中樞性頭暈等不同病因。確診需結合病史、神經學檢查、眼震電圖（VNG）、聽力檢查及影像學評估。
:::

::: warning 高齡患者解讀須謹慎
65 歲以上長者因正常老化造成前庭功能退化，基礎 DHI 分數可能略高。合併多重用藥、視力問題或骨關節疾病者，頭暈症狀來源需仔細鑑別，避免單純歸因於前庭疾病。
:::

::: warning E 面向高分需評估焦慮與 PPPD
情緒面向（E）分數 ≥ 24 分，尤其是在前庭功能檢查正常或已代償的患者，需考慮持續性姿勢－知覺性頭暈（PPPD）或恐慌症。此類患者以前庭復健單獨治療效果有限，需合併心理治療或 SSRI 藥物。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不具診斷效力。頭暈的診斷及治療應由耳鼻喉科、神經科或前庭專科醫師根據完整病史、神經學檢查及相關前庭功能評估綜合判斷。
:::
