# Dupixent 杜避炎適應症評估

**Dupixent Indication Assessment**
<DupixentCalculator />

## 適應症：慢性鼻竇炎合併鼻息肉（CRSwNP）

Dupilumab 用於治療在鼻用類固醇（INCS）治療下控制不佳的**成人慢性鼻竇炎合併鼻息肉**（CRSwNP）患者。

### 核心條件：第二型發炎生物標記（需符合 ≥ 3/5 項）

第二型發炎（Type 2 Inflammation）是 CRSwNP 的主要病理機轉，由 IL-4、IL-5、IL-13 等細胞激素驅動，臨床上可透過以下生物標記確認：

| 生物標記                   | 閾值                                     | 臨床意義               |
| -------------------------- | ---------------------------------------- | ---------------------- |
| 組織嗜酸性球（Tissue Eos） | ≥ 10/HPF                                 | 組織切片確認第二型發炎 |
| 血液嗜酸性球（Blood Eos）  | ≥ 250 cells/μL 或 ≥ 5%                   | 系統性嗜酸性球增多     |
| 總 IgE（Total IgE）        | ≥ 100 IU/mL                              | 過敏/第二型免疫反應    |
| 全身性類固醇依賴或禁忌     | 每年 ≥ 2 療程，或長期 > 3 個月，或有禁忌 | 藥物難治型疾病嚴重度   |
| 生活品質顯著下降           | SNOT-22 ≥ 40/110                         | 疾病對日常生活的影響   |

> 符合 **≥ 3/5 項**即達 Dupilumab 生物標記適應門檻。

### 附加臨床條件

以下條件雖非主要判斷標準，但強烈支持使用 Dupilumab：

- **嗅覺明顯喪失（Anosmia）**：嗅覺測試確認嗅覺功能喪失，反映鼻息肉嚴重程度
- **合併氣喘診斷**：已診斷合併氣喘且需定期使用吸入性類固醇（ICS），Dupilumab 可同時治療兩種適應症

## 全身性類固醇標準說明

「全身性類固醇需求或禁忌」符合以下任一條件即算：

| 條件          | 詳細說明                                   |
| ------------- | ------------------------------------------ |
| 每年 ≥ 2 療程 | 每療程 Prednisolone 0.5–1 mg/kg，持續 5 天 |
| 長期使用      | 連續使用超過 3 個月                        |
| 禁忌症        | 如糖尿病控制不良、骨質疏鬆症、精神科疾病等 |

## SNOT-22 量表簡介

**SNOT-22**（Sinonasal Outcome Test-22）是評估鼻竇炎相關生活品質的標準化問卷，共 22 題，每題 0–5 分，總分 0–110 分，分數越高代表症狀越嚴重、生活品質越差。

| 分數範圍 | 臨床意義                   |
| -------- | -------------------------- |
| < 20     | 輕微症狀                   |
| 20–39    | 中度症狀                   |
| ≥ 40     | 顯著症狀，生活品質明顯受損 |
| ≥ 50     | 嚴重症狀，建議積極介入     |

> **MCID（最小臨床重要差異）**：SNOT-22 改善 ≥ 8.9 分視為有臨床意義的改善。

## 用法用量

- **給藥方式**：皮下注射（subcutaneous injection）
- **起始劑量**：600 mg（分兩處各 300 mg）
- **維持劑量**：300 mg，每 4 週一次（q4w）
- **注射部位**：大腿、腹部或上臂（需輪換注射部位）
- **自我注射**：可由患者或照顧者自行注射，需完成訓練

## 療效證據

**SINUS-24 與 SINUS-52** 三期臨床試驗（pivotal trials）結果：

- **鼻息肉評分（NPS）**：相較安慰劑顯著改善（平均改善 ≈ 1.8 分）
- **鼻塞評分（NC）**：顯著改善
- **SNOT-22**：平均改善約 19–20 分（遠超 MCID）
- **嗅覺改善**：Dupilumab 組嗅覺恢復比率顯著高於安慰劑組
- **減少類固醇使用**：全身性類固醇使用率降低約 75%
- **手術避免**：減少需要鼻竇手術的比例

## 安全性與副作用

Dupilumab 的整體安全性良好，主要副作用包括：

| 副作用       | 發生率         | 備註                          |
| ------------ | -------------- | ----------------------------- |
| 注射部位反應 | 10–18%         | 通常輕微，隨時間減少          |
| 結膜炎       | 2–5%（CRSwNP） | 異位性皮膚炎適應症較高（14%） |
| 頭痛         | 約 9%          | 多為輕度                      |
| 上呼吸道感染 | 常見           | 非嚴重副作用                  |

> **相較其他生物製劑，Dupilumab 無需定期監測血球計數或肝功能，且無免疫抑制相關的嚴重感染或惡性腫瘤風險增加**，安全性profile 在生物製劑中屬於相對良好。

## 使用禁忌與注意事項

::: warning 使用前確認

- **過敏反應**：對 dupilumab 或任何賦形劑有過敏史者禁用
- **寄生蟲感染**：使用前應治療現有的蠕蟲（Helminth）感染；若治療期間發生感染且無法控制，應暫停 Dupilumab
- **活性感染**：不建議在急性嚴重感染期間使用
- **疫苗接種**：使用期間可接種不活化疫苗；**避免使用活疫苗**
  :::

::: warning 嗜酸性球暫時性升高
部分患者使用初期可能出現血液嗜酸性球暫時性升高，通常於治療後 4–8 週達峰值後下降。若嗜酸性球持續顯著升高（> 1500 cells/μL）且伴隨相關症狀，需進一步評估。
:::

::: info 台灣健保給付狀態
Dupilumab 用於 CRSwNP 在台灣已取得 TFDA 核准，健保給付狀態請以最新健保署公告為準。使用前建議確認患者是否符合當前健保給付條件，或評估自費治療選項。
:::
