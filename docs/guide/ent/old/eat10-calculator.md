# EAT-10 吞嚥篩檢量表

**EAT-10 Swallowing Screening Tool**
<Eat10Calculator />

## 評分說明

每題依嚴重程度評分：

| 分數 | 描述     | 說明                       |
| ---- | -------- | -------------------------- |
| 0    | 沒有問題 | 完全不受影響               |
| 1    | 輕微     | 偶爾有輕微困擾             |
| 2    | 中度     | 中等程度影響               |
| 3    | 重度     | 明顯影響日常吞嚥           |
| 4    | 非常嚴重 | 嚴重影響，生活品質顯著下降 |

**總分範圍：0–40 分**

## 臨床切截點

| EAT-10 總分 | 臨床意義           | 建議                                               |
| ----------- | ------------------ | -------------------------------------------------- |
| 0–2         | 正常範圍           | 無顯著吞嚥障礙                                     |
| ≥ 3         | 異常，需進一步評估 | 建議轉介語言治療師或耳鼻喉科進行完整吞嚥評估       |
| 3–10        | 輕度吞嚥障礙       | 追蹤觀察，考慮預防性介入                           |
| 11–20       | 中度吞嚥障礙       | 建議臨床吞嚥評估（CSE）或儀器吞嚥評估（MBSS/FEES） |
| > 20        | 重度吞嚥障礙       | 立即就醫，評估誤嗆風險、飲食質地調整及鼻胃管需求   |

> **重要**：EAT-10 ≥ 3 分即視為陽性（具臨床意義），建議進一步完整評估，**不論分數高低均需評估誤嗆風險**。

## 各題評估面向

EAT-10 的 10 個題目涵蓋吞嚥障礙的多個面向：

| 題號 | 面向               | 核心評估             |
| ---- | ------------------ | -------------------- |
| Q1   | 體重與營養         | 吞嚥問題導致體重減輕 |
| Q2   | 社交功能           | 影響外出用餐的能力   |
| Q3   | 安全性             | 喝液體的困難程度     |
| Q4   | 安全性             | 吃固體食物的困難程度 |
| Q5   | 功能性             | 吞嚥藥丸的困難程度   |
| Q6   | 疼痛               | 吞嚥時是否感到疼痛   |
| Q7   | 生活品質           | 進食樂趣的影響       |
| Q8   | 安全性（誤嗆風險） | 食物卡住喉嚨的感覺   |
| Q9   | 安全性（誤嗆風險） | 進食時咳嗽           |
| Q10  | 情緒/心理          | 吞嚥帶來的壓力感     |

> **Q8 和 Q9 是誤嗆風險的重要指標**：食物卡喉及進食時咳嗽是顯性誤嗆（Overt Aspiration）的典型症狀，任一題得分 ≥ 2 應高度警覺。

## 吞嚥評估流程

EAT-10 屬於**篩檢工具**，陽性結果應進行以下進一步評估：

### 臨床吞嚥評估（CSE — Clinical Swallowing Evaluation）

由語言治療師執行，包含口腔動作評估、不同質地食物的試吞觀察、進食時咳嗽與嗆咳監測。

### 儀器吞嚥評估

| 工具 | 全名                                                                 | 優點                       |
| ---- | -------------------------------------------------------------------- | -------------------------- |
| MBSS | Modified Barium Swallow Study（改良式鋇劑吞嚥攝影）                  | 金標準，可觀察整個吞嚥過程 |
| FEES | Fiberoptic Endoscopic Evaluation of Swallowing（軟式內視鏡吞嚥評估） | 床邊可執行，無輻射         |

## 常見適應症與臨床應用

EAT-10 廣泛用於以下患者族群的吞嚥篩檢：

| 疾病/情境     | EAT-10 的角色                                |
| ------------- | -------------------------------------------- |
| 腦中風（CVA） | 急性期及復健期吞嚥障礙篩檢                   |
| 頭頸部癌症    | 放射治療或手術後吞嚥功能監測                 |
| 巴金森病      | 疾病進展中的吞嚥障礙追蹤                     |
| 失智症        | 晚期吞嚥安全性評估                           |
| 老年衰弱症    | 肌少症性吞嚥障礙（Sarcopenic Dysphagia）篩檢 |
| 頸椎手術後    | 前路手術後吞嚥功能評估                       |
| 插管後        | 拔管後吞嚥功能恢復評估                       |
| 一般老年門診  | 吞嚥障礙常規篩檢                             |

## 誤嗆與吸入性肺炎的風險

吞嚥障礙最嚴重的併發症是**誤嗆**（Aspiration），即食物或液體進入氣道，可導致：

- **吸入性肺炎**（Aspiration Pneumonia）：特別在老年或免疫功能低下患者中死亡率高
- **慢性肺部損傷**：長期少量誤嗆（Silent Aspiration）可造成慢性肺部發炎
- **體重減輕與脫水**：因進食困難導致營養與水分攝取不足

> **沉默性誤嗆**（Silent Aspiration）：部分患者誤嗆時無明顯咳嗽反射，EAT-10 無法直接偵測此情形，需儀器評估（MBSS/FEES）確認。

## EAT-10 的信效度

- **內部一致性**：Cronbach's α = 0.94（高度一致）
- **再測信度**：相關係數 r = 0.72（良好）
- **對吞嚥障礙的敏感度**：約 89%
- **對正常吞嚥的特異度**：約 82%
- **MCID（最小臨床重要差異）**：≥ 3 分的變化視為具臨床意義

## 注意事項

::: warning EAT-10 ≥ 3 分需進一步評估
EAT-10 是**篩檢工具**，不能作為吞嚥障礙的診斷依據。任何 EAT-10 ≥ 3 分的患者，均應由語言治療師或耳鼻喉科醫師進行完整的臨床吞嚥評估，必要時安排儀器評估（MBSS 或 FEES）。
:::

::: warning 進食時咳嗽或嗆到請立即就醫
若患者在進食或喝水時有頻繁咳嗽、聲音沙啞（濕聲）、吞嚥後食物殘留或反覆發燒，這些可能是**誤嗆的警示症狀**，應優先安排完整吞嚥評估，勿延誤就醫。
:::

::: info 飲食質地調整建議
確認有吞嚥障礙後，可依據國際吞嚥障礙飲食標準化倡議（IDDSI）建議，調整食物與液體的質地（如：切碎食物、增稠液體），以降低誤嗆風險，並在語言治療師指導下進行吞嚥復健訓練。
:::

::: info 免責聲明
本計算器僅供臨床篩檢參考，不具診斷效力，不能取代完整的吞嚥評估。若有吞嚥困難症狀，請諮詢耳鼻喉科醫師或語言治療師進行完整評估。
:::
