# ESS 愛普沃斯嗜睡量表

**ESS Epworth Sleepiness Scale**
<ESSCalculator />

## 量表架構

ESS 共 **8 個問題**，評估在以下日常情境中打瞌睡或入睡的可能性：

| 題號 | 情境               | 說明                 |
| ---- | ------------------ | -------------------- |
| Q1   | 坐著閱讀           | 靜態低刺激活動       |
| Q2   | 看電視             | 靜態被動娛樂         |
| Q3   | 在公共場所靜坐     | 如劇院、會議室       |
| Q4   | 連續乘車超過一小時 | 非駕駛，車程未停歇   |
| Q5   | 下午躺下休息       | 臥床但非就寢時間     |
| Q6   | 坐著與人交談       | 社交互動             |
| Q7   | 午飯後安靜坐著     | 未飲酒               |
| Q8   | 駕車停紅燈時       | 最能偵測危險程度嗜睡 |

## 評分說明

每題依打瞌睡的可能性評分：

| 分數 | 描述       | 說明                   |
| ---- | ---------- | ---------------------- |
| 0    | 從不打瞌睡 | 在此情境中絕對不會入睡 |
| 1    | 輕微可能   | 偶爾有可能，但機率低   |
| 2    | 中等可能   | 有一定可能性           |
| 3    | 高度可能   | 在此情境幾乎必定打瞌睡 |

**總分範圍：0–24 分**（分數越高，日間嗜睡越嚴重）

## 嚴重度分級

| ESS 總分 | 嚴重程度   | 臨床意義與建議                               |
| -------- | ---------- | -------------------------------------------- |
| 0–10     | 正常嗜睡   | 日間嗜睡程度在正常範圍                       |
| 11–12    | 輕度嗜睡   | 輕微超出正常範圍，建議改善睡眠習慣           |
| 13–15    | 中度嗜睡   | 中度日間嗜睡，建議就醫評估，排除睡眠呼吸障礙 |
| 16–17    | 重度嗜睡   | 重度嗜睡，建議多項睡眠生理檢查（PSG）        |
| 18–24    | 極重度嗜睡 | 極重度嗜睡，強烈建議睡眠專科評估             |

> **注意**：ESS > 10 通常視為臨床上有意義的日間嗜睡（Excessive Daytime Sleepiness，EDS）。不同文獻對切截點的定義略有差異，部分研究以 **ESS ≥ 11** 作為異常標準。

## 各題臨床重點

### Q1–Q3：低刺激靜態情境

坐著閱讀、看電視及在公共場所靜坐，屬於最容易引發生理性嗜睡的低刺激情境，正常人在這些情境偶爾感到睏倦是可以接受的。

### Q4、Q5：放鬆被動情境

作為乘客乘車及下午躺下，是測試「被動放鬆」狀態下嗜睡程度的情境，分數高者常見於慢性睡眠不足或睡眠呼吸障礙患者。

### Q6、Q7：社交與餐後情境

與人交談及餐後靜坐，正常人在這些情境幾乎不應出現嗜睡。若此兩題分數偏高，提示嗜睡程度已超出一般生理性波動。

### Q8：駕車停紅燈時

這是 ESS 中**最具安全警示意義**的題目。在停車等紅燈時打瞌睡，代表患者在需要高度警覺的情境下仍無法維持清醒，與**駕車危險及交通事故**風險高度相關。若此題得 2–3 分，應強烈建議患者暫停駕車直到接受完整評估與治療。

## 臨床應用

### 阻塞性睡眠呼吸暫停（OSA）篩檢

ESS 是 OSA 門診評估的標準問卷之一，常與以下工具合併使用：

| 工具                     | 功能                         |
| ------------------------ | ---------------------------- |
| STOP-Bang 問卷           | OSA 風險快速篩檢             |
| 多項睡眠生理檢查（PSG）  | 金標準診斷工具               |
| 居家睡眠呼吸監測（HSAT） | 中至高風險患者的門診替代選項 |
| 呼吸中止指數（AHI）      | 客觀嚴重度指標               |

### 猝睡症（Narcolepsy）

猝睡症患者 ESS 通常顯著偏高（常 > 15），且日間嗜睡難以控制。需配合多次入睡潛伏時間測試（MSLT）確診。

### 治療反應監測

持續性正壓呼吸器（CPAP）治療 OSA 後，ESS 分數通常顯著下降。治療後 ESS 仍 > 10，應評估是否有殘餘嗜睡（Residual EDS）或其他共病（如憂鬱、甲狀腺功能低下）。

### 藥物評估

用於評估促醒藥物（如 Modafinil、Armodafinil）或其他治療對日間清醒度的療效。

## ESS 的限制

ESS 為**主觀自填量表**，結果受多種因素影響：

- **低估傾向**：部分患者因習慣慢性嗜睡而低估自身困倦程度（所謂「sleepiness denial」）
- **高估傾向**：焦慮症或憂鬱症患者可能因主觀疲憊感而高報嗜睡
- **不反映驅車危險**：ESS 不能直接預測某一患者的具體駕車危險程度
- **無時間窗限制**：不同於 PHQ-9 或 SNOT-22，ESS 詢問的是「一般情況下」而非特定時間窗
- **不能獨立診斷**：高 ESS 分數需配合客觀睡眠檢查才能確診病因

## 注意事項

::: warning Q8 駕車相關安全警告
若患者在 Q8（駕車停紅燈時）得分 2 分或 3 分，或 ESS 總分 ≥ 16 分，應明確告知患者存在**駕車危險**，並建議暫停駕車，直到接受完整評估與有效治療。各國對嗜睡症駕車的法規規定不同，請依當地法規處理。
:::

::: warning 猝睡症患者請立即就醫
若患者在靜坐與他人交談（Q6）或停紅燈（Q8）等情境得 3 分，且 ESS 總分顯著偏高（通常 > 16），應盡快轉介睡眠專科評估，以排除猝睡症或其他中樞性嗜睡疾病。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不具診斷效力，不能取代完整的睡眠醫學評估。若有日間嗜睡、睡眠障礙或相關症狀，請諮詢睡眠醫學科、胸腔科、神經科或耳鼻喉科醫師進行完整評估。
:::
