# ETDQ-7 耳咽管功能障礙問卷

**ETDQ-7 Eustachian Tube Dysfunction Questionnaire**
<Etdq7Calculator />

## 耳咽管功能簡介

**耳咽管**（Eustachian Tube）是連接中耳與鼻咽部的管狀結構，主要功能為：

- **壓力平衡**：調節中耳腔氣壓與外界平衡
- **黏液引流**：將中耳分泌物引流至鼻咽
- **保護功能**：防止聲音震動與細菌進入中耳

當耳咽管功能障礙時，中耳氣壓失衡，引發一系列耳部症狀，包括耳悶、耳壓、耳鳴及聽力下降。

## 評分說明

每題依症狀嚴重程度評分，採 **1–7 點 Likert 量表**：

| 分數 | 描述     | 臨床意義             |
| ---- | -------- | -------------------- |
| 1    | 沒有問題 | 完全不受影響         |
| 2    | 極輕微   | 幾乎可忽略的症狀     |
| 3    | 輕度     | 輕微影響，偶爾注意到 |
| 4    | 中度     | 明顯影響，需要注意   |
| 5    | 重度     | 嚴重影響日常生活     |
| 6    | 最嚴重   | 極度困擾             |
| 7    | 極嚴重   | 完全無法忍受         |

**計算方式**：將 7 題分數加總後除以 7，得到**平均分數**（Mean Score）。

## 臨床切截點

| 平均分數 | 嚴重程度     | 臨床意義                   |
| -------- | ------------ | -------------------------- |
| < 2.1    | 正常範圍     | 耳咽管功能正常，無顯著障礙 |
| 2.1–2.9  | 輕度功能障礙 | 建議追蹤觀察，必要時就醫   |
| 3.0–4.4  | 中度功能障礙 | 建議耳鼻喉科完整評估       |
| ≥ 4.5    | 重度功能障礙 | 建議積極治療或手術評估     |

> **臨床閾值**：平均分數 **≥ 2.1**（總分 ≥ 14.5）為異常，敏感度約 88%，特異度約 83%，是建議進一步評估的切截點。

## 各題評估重點

| 題號 | 症狀               | 臨床意義                                   |
| ---- | ------------------ | ------------------------------------------ |
| Q1   | 耳朵壓迫或悶塞感   | 最常見的 ETD 主訴，源於中耳負壓            |
| Q2   | 耳朵疼痛           | 中耳積液或急性耳道壓力改變                 |
| Q3   | 耳鳴（嗡嗡聲）     | 中耳壓力失衡或積液引起的聲音感知改變       |
| Q4   | 耳內積液感或啪啪聲 | 積液移動或耳咽管開放時的聲音               |
| Q5   | 吞嚥或打哈欠後改善 | 耳咽管一過性開放的特徵性症狀，高度提示 ETD |
| Q6   | 感冒或過敏時加重   | 黏膜水腫堵塞耳咽管，急性加重因素           |
| Q7   | 氣壓變化時症狀加重 | 被動開管（Passive Opening）功能缺損        |

> **Q5 是 ETD 的高度特異性症狀**：若患者報告吞嚥或打哈欠能暫時緩解耳朵不適，強烈提示耳咽管功能性阻塞（Obstructive ETD），而非中耳炎或其他耳部疾病。

## ETD 的分類

根據症狀特點，ETD 主要分為兩型：

### 阻塞型 ETD（Obstructive ETD）

最常見類型，耳咽管無法正常開放，導致中耳負壓。常見原因：

- 過敏性鼻炎（最常見）
- 慢性鼻竇炎
- 腺樣體肥大（兒童）
- 鼻息肉
- 耳咽管周圍組織腫脹

### 開放型 ETD（Patulous ETD）

耳咽管過度開放，患者可能聽到自己的呼吸聲（自聲增強，Autophony）或說話聲在耳中迴響。常見原因：

- 體重快速減輕
- 慢性疲勞
- 懷孕
- 神經肌肉疾病

> ETDQ-7 主要針對阻塞型 ETD，對開放型 ETD 的評估需結合其他臨床工具。

## 臨床應用

ETDQ-7 廣泛應用於以下情境：

- **ETD 初始篩檢**：在耳鼻喉科門診快速評估耳咽管功能
- **手術適應症評估**：耳咽管擴張手術（Eustachian Tube Balloon Dilation，ETBD）術前評估
- **術後療效追蹤**：手術前後 ETDQ-7 分數比較，評估治療效果
- **過敏性鼻炎治療監測**：抗過敏治療對 ETD 症狀的改善程度
- **研究標準化**：ETD 相關臨床研究的標準化患者報告工具

## 耳咽管擴張手術（ETBD）

當保守治療（鼻用類固醇、抗組織胺、耳咽管通氣等）無效時，可考慮**耳咽管球囊擴張手術**：

- **適應症**：ETDQ-7 平均分 ≥ 2.1，保守治療 > 3 個月無效
- **原理**：以充氣球囊擴張耳咽管軟骨段，改善黏膜功能與管腔通暢度
- **療效**：術後 ETDQ-7 平均改善約 1.5–2.0 分，多數患者有顯著改善
- **安全性**：微創手術，在全身或局部麻醉下進行，風險低

## 與相關疾病的鑑別

ETDQ-7 需與以下疾病鑑別，因症狀可能重疊：

| 疾病                                   | 與 ETD 的鑑別點                              |
| -------------------------------------- | -------------------------------------------- |
| 中耳積液（Otitis Media with Effusion） | 驗聽（傳導性聽損）+ 鼓室圖（Type B）有助區分 |
| 慢性中耳炎                             | 耳鏡可見鼓膜穿孔或中耳病變                   |
| 顳顎關節障礙（TMJ）                    | 疼痛位置在耳前，咀嚼時加重                   |
| 開放型 ETD                             | 自聲增強為主要症狀，低頭時改善               |
| 梅尼爾氏症                             | 三主症：眩暈發作、單側聽損、耳鳴             |

## 注意事項

::: warning ETDQ-7 ≥ 2.1 建議完整評估
ETDQ-7 平均分 ≥ 2.1 為陽性，但確診 ETD 需要結合完整耳鼻喉科評估，包括：**耳鏡檢查、純音聽力測試、鼓室圖（Tympanogram）及鼻腔內視鏡**，以排除中耳炎、鼓膜穿孔及鼻咽部腫瘤等疾病。
:::

::: warning Q7 高分（≥ 4）需注意鼻咽癌
若患者對 Q7（氣壓變化時症狀加重）評分偏高，且合併單側耳症狀、頸部腫塊或鼻出血，應優先排除**鼻咽癌（Nasopharyngeal Carcinoma）**，因鼻咽癌可透過壓迫耳咽管開口引起 ETD 樣症狀。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不具診斷效力，不能取代完整的耳鼻喉科評估。若有持續耳悶、耳壓或聽力下降症狀，請諮詢耳鼻喉科醫師進行完整評估，包括聽力測試及鼓室圖檢查。
:::
