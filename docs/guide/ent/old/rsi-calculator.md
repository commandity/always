# RSI 逆流症狀指數

**RSI Reflux Symptom Index**
<RsiCalculator />

## 評分說明

每題依症狀對生活的困擾程度評分：

| 分數 | 描述     | 說明                   |
| ---- | -------- | ---------------------- |
| 0    | 沒有問題 | 完全沒有此症狀         |
| 1    | 極輕微   | 幾乎可忽略的輕微症狀   |
| 2    | 輕度     | 輕微，偶爾注意到       |
| 3    | 中度     | 中等程度，有時影響生活 |
| 4    | 重度     | 嚴重，頻繁影響日常生活 |
| 5    | 最嚴重   | 極度困擾，完全無法忍受 |

**總分範圍：0–45 分**

## 臨床切截點

| RSI 總分 | 嚴重程度     | 臨床建議                                     |
| -------- | ------------ | -------------------------------------------- |
| ≤ 13     | 正常範圍     | 喉咽逆流症狀在正常範圍內                     |
| 14–20    | 輕至中度 LPR | 建議就醫評估，考慮試驗性 PPI 治療            |
| 21–30    | 中度 LPR     | 建議喉鏡評估（RFS），積極 PPI 治療           |
| > 30     | 重度 LPR     | 嚴重 LPR，建議積極治療並考慮 24 小時 pH 監測 |

> **關鍵切截值：RSI > 13** 為異常，敏感度約 62%，特異度約 96%，對 LPR 的診斷特異性高。若合併 RFS > 7（喉鏡客觀發現），診斷準確性更高。

## 各題評估重點

| 題號 | 症狀                 | LPR 相關性                                      |
| ---- | -------------------- | ----------------------------------------------- |
| Q1   | 聲音嘶啞             | 胃酸侵蝕喉部黏膜，聲帶水腫或紅斑                |
| Q2   | 清嗓動作             | 最常見的 LPR 症狀，逆流刺激喉部引起的反射性動作 |
| Q3   | 過多黏液或鼻涕倒流   | 喉部黏液分泌增加，與慢性喉炎有關                |
| Q4   | 吞嚥困難             | 喉部水腫或食道功能障礙                          |
| Q5   | 進食或躺下後咳嗽     | 逆流物接觸喉部引起的咳嗽反射，典型 LPR 症狀     |
| Q6   | 呼吸困難或窒息感     | 聲門下水腫或喉痙攣（Laryngospasm）              |
| Q7   | 令人煩惱的咳嗽       | 慢性咳嗽，LPR 是常見原因之一                    |
| Q8   | 喉嚨異物感（梅核氣） | 最常被提及的 LPR 症狀，但特異性相對較低         |
| Q9   | 心灼熱或胃酸逆流     | 傳統 GERD 症狀，在 LPR 患者中可能較不明顯       |

> **LPR 的特殊性**：與傳統胃食道逆流（GERD）不同，LPR 患者常以 Q2（清嗓）、Q7（咳嗽）、Q8（異物感）為主訴，而**不一定有 Q9**（燒心感）——這也是 LPR 容易被漏診的原因。

## 喉咽逆流（LPR）vs. 胃食道逆流（GERD）

| 特性             | LPR                          | GERD                   |
| ---------------- | ---------------------------- | ---------------------- |
| 主要症狀         | 聲音沙啞、清嗓、咳嗽、梅核氣 | 燒心感、胃酸逆流、胸痛 |
| 燒心感           | 較少見（< 40%）              | 主要症狀（> 80%）      |
| 發作時間         | 白天直立時較多               | 夜間躺下時較多         |
| 上食道括約肌功能 | 可能受損                     | 通常正常               |
| 食道黏膜損傷     | 較少                         | 常見（食道炎）         |
| 診斷工具         | RSI + RFS + 喉鏡             | 胃鏡、24 小時 pH 監測  |
| 主要治療         | PPI + 飲食/生活習慣調整      | PPI ± 抗酸藥           |

## RSI 與 RFS 合併使用

**RSI** 評估患者的主觀症狀，**RFS**（Reflux Finding Score）則是由喉鏡評估客觀的喉部逆流徵象：

| RFS 項目                                       | 臨床意義             |
| ---------------------------------------------- | -------------------- |
| 聲帶下水腫（Subglottic edema）                 | 逆流的高度特異性發現 |
| 後聯合肥厚（Posterior commissure hypertrophy） | 常見逆流發現         |
| 喉室消失（Ventricular obliteration）           | 提示喉部水腫嚴重     |
| 紅斑（Erythema / Hyperemia）                   | 炎症反應             |
| 聲帶水腫                                       | 影響聲音品質         |
| 瀰漫性喉部水腫                                 | 較嚴重的逆流損傷     |
| 後聲門肥厚（Posterior glottic hypertrophy）    | 慢性逆流的典型發現   |
| 厚黏液（Thick endolaryngeal mucus）            | 黏液分泌增加         |

> **診斷標準**：RSI > 13 且 RFS > 7，診斷 LPR 的特異性可達 95% 以上。

## 治療與監測

### 質子幫浦抑制劑（PPI）

- **標準治療**：每日 2 次（餐前 30 分鐘），療程至少 3–6 個月
- **療效評估**：治療後重新填寫 RSI，改善 ≥ 50% 視為有效
- **注意**：LPR 對 PPI 的反應較 GERD 慢，需要較長療程

### 飲食與生活習慣調整

- 避免：咖啡、茶、酒精、碳酸飲料、巧克力、番茄製品、油炸食物
- 睡前 3 小時不進食
- 抬高床頭 15–30 公分
- 控制體重
- 戒菸

### 手術治療

保守治療無效者，可考慮：

- **抗逆流手術**（Nissen Fundoplication）：適用於有明確解剖異常（如裂孔疝）的患者
- **內視鏡抗逆流治療**：較新的微創選項

## 注意事項

::: warning RSI 是篩檢工具，不能單獨診斷 LPR
RSI > 13 高度提示 LPR，但確診需結合喉鏡評估（RFS）、24 小時雙探頭 pH 監測或阻抗 pH 監測（Impedance-pH Monitoring）。建議由耳鼻喉科或腸胃科醫師進行完整評估。
:::

::: warning 排除其他喉部疾病
清嗓、聲音沙啞及異物感也可能源於聲帶息肉、結節、腫瘤或甲狀腺疾病。RSI 高分患者應接受完整喉部內視鏡檢查，以排除其他喉部病因。
:::

::: warning Q5 與 Q6 高分需警覺誤嗆
Q5（進食或躺下後咳嗽）及 Q6（呼吸困難或窒息感）高分者，需評估是否合併**吞嚥障礙**或**誤嗆**（Aspiration），必要時安排吞嚥評估（EAT-10 或儀器吞嚥評估）。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不具診斷效力。喉咽逆流的診斷及治療應由耳鼻喉科或腸胃科醫師根據完整病史、喉鏡及相關檢查綜合判斷。
:::
