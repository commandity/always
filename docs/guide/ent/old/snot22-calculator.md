# SNOT-22 鼻竇生活品質量表

**SNOT-22 Sinonasal Outcome Test**
<Snot22Calculator />

## 量表架構

SNOT-22 共 **22 個問題**，涵蓋四大領域：

| 領域           | 題號    | 題數 | 最高分 | 評估面向                                                 |
| -------------- | ------- | ---- | ------ | -------------------------------------------------------- |
| 鼻部症狀       | Q1–Q8   | 8    | 40     | 擤鼻、鼻塞、噴嚏、流鼻水、鼻涕倒流、濃稠鼻涕、嗅覺、咳嗽 |
| 耳部與顏面症狀 | Q9–Q12  | 4    | 20     | 耳悶、頭暈、耳痛、臉部疼痛壓迫感                         |
| 睡眠功能       | Q13–Q17 | 5    | 25     | 入睡困難、夜醒、睡眠品質、醒來疲累、白天倦怠             |
| 心理與情緒功能 | Q18–Q22 | 5    | 25     | 生產力、注意力、沮喪、悲傷、尷尬                         |

**總分範圍：0–110 分**（分數越高，症狀越嚴重、生活品質越差）

## 評分說明

每題依過去**兩週**內症狀的嚴重程度與頻率綜合評估：

| 分數 | 描述   | 臨床意義                 |
| ---- | ------ | ------------------------ |
| 0    | 無問題 | 完全不受影響             |
| 1    | 極輕微 | 偶爾出現，幾乎不影響生活 |
| 2    | 輕度   | 輕微影響，但尚可接受     |
| 3    | 中度   | 明顯影響日常活動或睡眠   |
| 4    | 重度   | 嚴重干擾日常生活         |
| 5    | 最嚴重 | 完全無法正常生活         |

## 嚴重度分級

| SNOT-22 總分 | 嚴重程度 | 臨床建議                                                |
| ------------ | -------- | ------------------------------------------------------- |
| 0–19         | 輕微     | 症狀輕微，定期追蹤                                      |
| 20–39        | 中度     | 症狀中度影響生活，建議就醫評估                          |
| 40–59        | 顯著     | 生活品質明顯受損，**達 Dupilumab 生物標記門檻（≥ 40）** |
| 60–110       | 嚴重     | 症狀嚴重，強烈建議積極治療介入                          |

> **MCID（最小臨床重要差異）**：SNOT-22 改善 **≥ 8.9 分**視為有臨床意義的治療反應。術後或治療後追蹤時，以此標準判定是否有效。

## SNOT-22 ≥ 40 的臨床意義

SNOT-22 ≥ 40 分是**Dupilumab**（Dupixent）用於慢性鼻竇炎合併鼻息肉（CRSwNP）的**生物標記適應條件**之一（5 項中的第 5 項）。此閾值代表患者的生活品質顯著受損，單靠鼻用類固醇無法充分控制疾病，需考慮生物製劑治療。

## 各領域臨床重點

### 鼻部症狀（Q1–Q8）

最直接反映鼻竇炎的核心症狀。其中 **Q7 嗅覺/味覺喪失** 在慢性鼻竇炎合併鼻息肉（CRSwNP）患者中特別重要，嗅覺障礙不僅影響生活品質，也是第二型發炎嚴重度的指標。

### 耳部與顏面症狀（Q9–Q12）

臉部疼痛和壓迫感在急性加重期間尤為突出。耳悶和頭暈可能與耳咽管功能障礙有關。

### 睡眠功能（Q13–Q17）

CRS 患者普遍有睡眠障礙，鼻塞導致睡眠呼吸道阻塞、夜間鼻涕倒流引起咳嗽，都會嚴重影響睡眠品質。此領域分數高低與患者白天功能狀態密切相關。

### 心理與情緒功能（Q18–Q22）

慢性病程帶來的情緒影響常被低估。長期鼻竇炎造成的疲勞、注意力下降和社交尷尬（因鼻部症狀）都會影響工作表現和人際關係。

## 臨床應用

SNOT-22 廣泛應用於以下情境：

- **CRS 初始評估**：建立基線生活品質分數
- **治療前後比較**：藥物治療或手術介入的療效評估
- **生物製劑適應症評估**：SNOT-22 ≥ 40 是 Dupilumab 適應條件之一
- **術後追蹤**：功能性鼻竇內視鏡手術（FESS）後的病程監測
- **臨床試驗**：SINUS-24/52 等大型試驗的主要結果指標
- **研究標準化**：跨中心、跨語言的 CRS 症狀比較

## 五大最重要症狀（Top 5 Items）

SNOT-22 原始問卷要求患者勾選「對自己最重要的 5 個症狀」，這有助於個別化了解患者最關心的症狀面向，引導治療目標的設定與醫病溝通。常見的前五重要症狀包括：

- 鼻塞
- 嗅覺/味覺喪失
- 睡眠品質不佳
- 疲倦/精力下降
- 鼻涕倒流

## 注意事項

::: warning 請依據過去兩週的實際感受填寫
SNOT-22 的評估時間窗為**過去兩週**。急性加重期、手術後早期或剛完成藥物療程後的分數，可能無法代表患者的慢性基線狀態，應在穩定狀態下重複評估。
:::

::: warning SNOT-22 不作為診斷工具
SNOT-22 為症狀嚴重度與生活品質評估工具，**不具診斷功能**，不能單獨作為慢性鼻竇炎或其他疾病的診斷依據。診斷需結合病史、內視鏡檢查及影像學（CT）綜合判定。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，結果不能取代完整的耳鼻喉科評估與臨床判斷。若您有長期鼻竇症狀、生活品質受影響，請諮詢耳鼻喉科或鼻科專科醫師進行完整評估。
:::
