# STOP-BANG 睡眠呼吸中止症量表

**STOP-BANG Sleep Apnea Screening**
<StopBangCalculator />

## 評分說明

STOP-BANG 共 8 個問題，每個「是」計 **1 分**，總分 **0–8 分**：

| 字母                        | 問題                           | 評分條件                         |
| --------------------------- | ------------------------------ | -------------------------------- |
| **S** — 打鼾 Snoring        | 您打鼾聲音很大嗎？             | 響到隔著房門可聽到，或讓床伴困擾 |
| **T** — 疲倦 Tired          | 您白天常感疲倦、昏睡嗎？       | 日間過度嗜睡，即使夜間睡眠足夠   |
| **O** — 觀察 Observed       | 有人觀察到您睡眠中呼吸暫停嗎？ | 床伴或家人曾目擊睡眠呼吸中止     |
| **P** — 血壓 Pressure       | 您有高血壓病史或正在治療嗎？   | 確診高血壓，含服藥中             |
| **B** — BMI Body Mass Index | 您的 BMI 超過 35 kg/m² 嗎？    | 第二級肥胖以上                   |
| **A** — 年齡 Age            | 您的年齡超過 50 歲嗎？         | 50 歲以上                        |
| **N** — 頸圍 Neck           | 您的頸圍超過 40 cm 嗎？        | 甲狀軟骨下緣水平測量             |
| **G** — 性別 Gender         | 您是男性嗎？                   | 生理男性                         |

## 臨床切截點

| STOP-BANG 分數 | 風險等級 | AHI ≥ 5 機率 | AHI ≥ 30 機率 | 建議處置               |
| -------------- | -------- | ------------ | ------------- | ---------------------- |
| 0–2            | 低風險   | 約 17%       | 約 3%         | 症狀治療，通常不需 PSG |
| 3–4            | 中等風險 | 約 50%       | 約 20%        | 考慮轉介睡眠評估       |
| 5–8            | 高風險   | > 70%        | 約 40%        | 強烈建議 PSG 或 HSAT   |

> **AHI**（Apnea-Hypopnea Index，呼吸中止-低通氣指數）：每小時睡眠中呼吸事件數。AHI ≥ 5 = OSA 診斷；AHI ≥ 15 = 中重度；AHI ≥ 30 = 重度 OSA。

## Chung 2012 進階高風險標準

2012 年 Chung 等人提出針對外科手術患者的進階高風險標準，符合以下**任一項**即為高風險：

| 高風險條件                          |
| ----------------------------------- |
| STOP-BANG 總分 **≥ 5**              |
| STOP-BANG ≥ 2 且**男性**            |
| STOP-BANG ≥ 2 且 **BMI > 35 kg/m²** |
| STOP-BANG ≥ 2 且**頸圍 > 40 cm**    |

> 進階標準對重度 OSA（AHI ≥ 30）的靈敏度可達 90% 以上，特別適用於高風險手術前評估。

## STOP-BANG 各項的臨床意義

### STOP — 症狀評估

**S（打鼾）** 是 OSA 最普遍的症狀，反映睡眠中上呼吸道部分阻塞。需注意不是所有打鼾者都有 OSA，但重度 OSA 患者幾乎都有打鼾症狀。

**T（日間疲倦）** 反映睡眠片段化造成的睡眠品質下降，進一步可用 Epworth 嗜睡量表（ESS）量化嗜睡程度。

**O（被觀察到呼吸中止）** 是對 OSA 診斷特異性最高的問題。床伴目擊的呼吸暫停直接反映完全性上呼吸道阻塞，是 OSA 最重要的臨床警示訊號。

**P（高血壓）** 與 OSA 呈強烈雙向關係。頑固性高血壓（使用 3 種以上降壓藥仍控制不佳）的患者中，OSA 盛行率可達 80%。

### BANG — 解剖與人口學特徵

**B（BMI > 35）** 肥胖是 OSA 最重要的可改變危險因子。體重每增加 10%，OSA 風險增加約 32%；體重減少 10–15%，AHI 可降低 50%。

**A（年齡 > 50）** 隨年齡增長，咽部肌肉張力下降，睡眠結構改變，OSA 盛行率從 30 歲的 5% 上升到 60 歲的 20–30%。

**N（頸圍 > 40 cm）** 頸部脂肪堆積直接壓迫咽部氣道。男性頸圍 > 43 cm、女性 > 41 cm 為重度 OSA 的獨立預測因子。

**G（男性）** 男性罹患 OSA 的風險是女性的 2–3 倍，主因脂肪分布（男性傾向頸部及內臟脂肪堆積）、咽部解剖及黃體素（Progesterone）的保護作用差異。女性停經後 OSA 風險顯著上升。

## OSA 嚴重度分類

| AHI（次/小時） | 嚴重度   | 臨床意義                     |
| -------------- | -------- | ---------------------------- |
| 0–4            | 正常     | 無 OSA                       |
| 5–14           | 輕度 OSA | 輕微症狀，可考慮非 CPAP 治療 |
| 15–29          | 中度 OSA | 建議 CPAP 或口腔矯正器       |
| ≥ 30           | 重度 OSA | 建議積極 CPAP 治療           |

## 診斷工具

### 多道睡眠生理檢查（PSG，Polysomnography）

- **金標準診斷工具**
- 在睡眠實驗室進行，監測腦波（EEG）、眼動、肌電、心電、血氧、呼吸氣流、胸腹運動
- 適用：中高風險患者、複雜合併症、懷疑非 OSA 睡眠障礙

### 居家睡眠呼吸監測（HSAT，Home Sleep Apnea Testing）

- 輕便設備在家中進行
- 靈敏度略低，但便利、低成本
- 適用：中高風險且無明顯合併症的患者

## 治療選項

### CPAP（持續正壓呼吸器）

**首選治療**，適用於所有中重度 OSA 及輕度但有症狀者。規律使用（每晚 ≥ 4 小時）可顯著改善嗜睡、血壓及生活品質。

### 口腔矯正器（Mandibular Advancement Device）

適用於輕中度 OSA，或不耐受 CPAP 者。將下顎前移以擴大咽部氣道。

### 生活方式改變

| 措施                 | 效果                    |
| -------------------- | ----------------------- |
| 體重減輕 10–15%      | AHI 降低約 50%          |
| 避免仰睡（側睡訓練） | 輕度 OSA 可降低 AHI 50% |
| 戒酒（尤其睡前）     | 減少咽部肌肉放鬆        |
| 戒菸                 | 降低上呼吸道發炎        |
| 睡前避免鎮靜藥物     | 減少肌肉張力下降        |

### 手術治療

適用於特定解剖異常（扁桃腺肥大、鼻中隔彎曲、顱顏畸形）。懸雍垂咽顎整形術（UPPP）對重度 OSA 效果有限。

## OSA 的系統性影響

未治療的 OSA 與多種嚴重系統疾病相關：

| 系統     | 影響                                                  |
| -------- | ----------------------------------------------------- |
| 心血管   | 高血壓、心房顫動、冠狀動脈疾病、心衰竭風險增加 2–3 倍 |
| 腦血管   | 中風風險增加 1.5–3 倍                                 |
| 代謝     | 胰島素阻抗、第二型糖尿病風險增加                      |
| 神經認知 | 注意力下降、記憶障礙、失智風險增加                    |
| 駕駛安全 | 交通事故風險增加 2–7 倍                               |
| 圍手術期 | 術後肺部併發症、插管困難、復甦延遲                    |

## 注意事項

::: warning STOP-BANG 是篩查工具，不能確診 OSA
STOP-BANG 高分（≥ 3）提示 OSA 風險升高，但確診需要多道睡眠生理檢查（PSG）或居家睡眠呼吸監測（HSAT）。分數低不能完全排除 OSA，尤其是症狀不典型的女性患者。
:::

::: warning 女性患者可能被低估
OSA 在女性中的症狀表現不同——更常見失眠、頭痛、憂鬱而非典型的打鼾和嗜睡。STOP-BANG 中男性（G = 男性得分）的設計可能低估女性風險，臨床評估時需保持警覺。
:::

::: warning 術前評估特別重要
OSA 患者在全身麻醉後有較高的氣道管理困難、術後低氧及心肺併發症風險。STOP-BANG 高風險患者在術前應通知麻醉科，並準備相應的氣道管理措施及術後加強監測。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不能取代完整病史、身體檢查及正式睡眠評估。OSA 的診斷及治療應由睡眠醫學科、耳鼻喉科或胸腔科醫師根據患者個別狀況綜合判斷。
:::
