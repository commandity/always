# 睡眠呼吸中止症評估工具

**Obstructive Sleep Apnea Screening · STOP-BANG & ESS**

<OsaCalculator />

## STOP-BANG 睡眠呼吸中止篩檢

**STOP-BANG**（Chung 2008）為 8 項二元篩檢工具（是/否），每項 1 分，總分 0–8 分。

| 項目                 | 內容                   |
| -------------------- | ---------------------- |
| S — Snoring（打鼾）  | 鼾聲大（關門外可聽到） |
| T — Tired（疲倦）    | 白天常感疲倦或嗜睡     |
| O — Observed（觀察） | 被觀察到睡眠中呼吸中止 |
| P — Pressure（血壓） | 有高血壓病史或治療中   |
| B — BMI              | BMI > 35 kg/m²         |
| A — Age（年齡）      | > 50 歲                |
| N — Neck（頸圍）     | > 40 cm                |
| G — Gender（性別）   | 男性                   |

| 總分 | OSA 風險 |
| ---- | -------- |
| 0–2  | 低風險   |
| 3–4  | 中度風險 |
| 5–8  | 高風險   |

## ESS 嗜睡量表

**Epworth Sleepiness Scale**（Johns 1991）為 8 情境自填量表，評估日間嗜睡程度，每題 0–3 分，總分 0–24 分。

| 總分  | 嗜睡程度 |
| ----- | -------- |
| 0–10  | 正常範圍 |
| 11–14 | 輕度嗜睡 |
| 15–18 | 中度嗜睡 |
| ≥ 19  | 重度嗜睡 |

## 注意事項

::: warning 篩檢工具無法取代睡眠檢查
STOP-BANG 及 ESS 為篩檢工具，OSA 確診需依賴 polysomnography（PSG）睡眠檢查。
:::

本計算器僅供臨床篩檢參考，OSA 診斷及治療需由睡眠專科醫師進行完整評估。
:::
