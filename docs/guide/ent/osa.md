# 睡眠呼吸中止症評估工具

**Obstructive Sleep Apnea Screening · STOP-BANG & ESS**

<OsaCalculator />

## STOP-BANG 睡眠呼吸中止篩檢

**STOP-BANG**（Chung 2008）為 8 項二元篩檢工具（是/否），每項 1 分，總分 0–8 分。

| 項目                 | 內容                   |
| -------------------- | ---------------------- |
| S — Snoring（打鼾）  | 鼾聲大（關門外可聽到） |
| T — Tired（疲倦）    | 白天常感疲倦或嗜睡     |
| O — Observed（觀察） | 被觀察到睡眠中呼吸中止 |
| P — Pressure（血壓） | 有高血壓病史或治療中   |
| B — BMI              | BMI > 35 kg/m²         |
| A — Age（年齡）      | > 50 歲                |
| N — Neck（頸圍）     | > 40 cm                |
| G — Gender（性別）   | 男性                   |

| 總分 | OSA 風險 |
| ---- | -------- |
| 0–2  | 低風險   |
| 3–4  | 中度風險 |
| 5–8  | 高風險   |

### STOP-BANG 分數意義

| 總分 | OSA 風險 | 意義與建議處置                                               |
| ---- | -------- | ------------------------------------------------------------ |
| 0–2  | 低風險   | 中重度 OSA 可能性低，若臨床仍高度懷疑可安排居家睡眠檢測      |
| 3–4  | 中度風險 | 建議進一步評估，安排 PSG 或居家睡眠檢測                      |
| 5–8  | 高風險   | 中重度 OSA 機率高，建議優先安排 PSG；麻醉前需特別註記        |

> **≥ 3 分即為篩檢陽性**：偵測中重度 OSA（AHI ≥ 15）的敏感度約 93%，適合「寧可多驗、不可漏掉」的術前與高風險族群篩檢；但特異度較低，陽性不等於確診。另一提高特異度的判定：STOP 四項中 ≥ 2 項陽性**且**合併男性、BMI > 35 或頸圍 > 40 cm 之一，亦屬高風險。

## ESS 嗜睡量表

**Epworth Sleepiness Scale**（Johns 1991）為 8 情境自填量表，評估日間嗜睡程度，每題 0–3 分，總分 0–24 分。

| 總分  | 嗜睡程度   | 意義與建議處置                                       |
| ----- | ---------- | ---------------------------------------------------- |
| 0–10  | 正常範圍   | 日間嗜睡在正常範圍，無需特別處置                     |
| 11–12 | 輕度嗜睡   | 略高於正常，先檢視睡眠時數與睡眠衛生                 |
| 13–15 | 中度嗜睡   | 建議就醫評估，排除睡眠呼吸障礙等病因                 |
| 16–17 | 重度嗜睡   | 積極就醫，通常需要 PSG 檢查                          |
| ≥ 18  | 極重度嗜睡 | 強烈建議睡眠專科評估，注意猝睡症或重度 OSA 的可能    |

> **ESS > 10 分**即表示日間嗜睡超出正常範圍。ESS 測量的是「打瞌睡的傾向」而非疲倦感，與 OSA 嚴重度（AHI）僅中度相關——ESS 正常不能排除 OSA，因此臨床上常與 STOP-BANG 併用：前者反映主觀嗜睡負擔與治療後症狀改善，後者估計 OSA 的驗前機率。**≥ 16 分**者應暫避免長途駕駛與操作危險機械，直到病因釐清。

## 注意事項

::: warning 篩檢工具無法取代睡眠檢查
STOP-BANG 及 ESS 為篩檢工具，OSA 確診需依賴 polysomnography（PSG）睡眠檢查。
:::

::: info 免責聲明
本計算器僅供臨床篩檢參考，OSA 診斷及治療需由睡眠專科醫師進行完整評估。
:::
