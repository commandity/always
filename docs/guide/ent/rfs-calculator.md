# RFS 逆流表現評分

**RFS Reflux Finding Score**
<RfsCalculator />

## 評分說明

RFS 由醫師透過**軟式或硬式喉鏡**評估下列 8 項喉部所見：

| 項目             | 英文                             | 計分範圍  | 說明                                       |
| ---------------- | -------------------------------- | --------- | ------------------------------------------ |
| 聲門下水腫       | Subglottic Edema                 | 0 / 2     | 假聲帶下方黏膜水腫形成偽裂（pseudosulcus） |
| 喉室消失         | Ventricular Obliteration         | 0 / 2 / 4 | 喉室因水腫部分（2分）或完全（4分）填塞     |
| 紅斑／充血       | Erythema / Hyperemia             | 0 / 2 / 4 | 僅杓狀軟骨（2分）或瀰漫性紅斑（4分）       |
| 聲帶水腫         | Vocal Fold Edema                 | 0–4       | 0=無，1=輕度，2=中度，3=重度，4=息肉樣變性 |
| 瀰漫性喉部水腫   | Diffuse Laryngeal Edema          | 0–4       | 0=無，1=輕度，2=中度，3=重度，4=阻塞性     |
| 後壁增生肥厚     | Posterior Commissure Hypertrophy | 0–4       | 0=無，1=輕度，2=中度，3=重度，4=阻塞性     |
| 肉芽腫或肉芽組織 | Granuloma / Granulation Tissue   | 0 / 2     | 杓狀軟骨或後聯合的肉芽腫                   |
| 喉部黏液附著     | Thick Endolaryngeal Mucus        | 0 / 2     | 黏稠黏液附著或聲帶間黏液絲                 |

**總分範圍：0–26 分**

## 臨床切截點

| RFS 總分 | 嚴重程度 | 臨床建議                                |
| -------- | -------- | --------------------------------------- |
| ≤ 7      | 正常範圍 | 喉鏡所見無顯著 LPR 跡象                 |
| 8–11     | 輕度 LPR | 建議配合 RSI 評估，考慮試驗性 PPI 治療  |
| 12–17    | 中度 LPR | 建議積極 PPI 治療，考慮 24 小時 pH 監測 |
| ≥ 18     | 重度 LPR | 重度喉咽逆流，建議積極治療及密切追蹤    |

> **關鍵切截值：RFS > 7** 為異常（Belafsky 2001）。部分臨床文獻採用 **> 8** 作為較嚴格的切截值以提高特異度。若合併 RSI > 13（患者自填問卷），診斷 LPR 的準確性可達 95% 以上。

## 各項評估重點

| 項目             | LPR 相關性                                                 |
| ---------------- | ---------------------------------------------------------- |
| 聲門下水腫       | LPR 的高度特異性發現，假聲帶下方黏膜因逆流持續刺激形成偽裂 |
| 喉室消失         | 喉室黏膜水腫，提示慢性逆流損傷的嚴重程度                   |
| 紅斑／充血       | 胃酸刺激引起的急慢性炎症反應，杓狀軟骨充血最具代表性       |
| 聲帶水腫         | 直接影響聲音品質，逆流是慢性聲帶水腫的重要原因之一         |
| 瀰漫性喉部水腫   | 廣泛黏膜水腫提示逆流程度嚴重，需積極治療                   |
| 後壁增生肥厚     | 後聯合是逆流物最先接觸的部位，最常見的慢性 LPR 所見        |
| 肉芽腫或肉芽組織 | 接觸性肉芽腫（contact granuloma）與逆流及聲帶使用過度有關  |
| 喉部黏液附著     | 黏液分泌增加或清除能力下降，常引起慢性清嗓衝動             |

> **後壁增生肥厚**是最常見的 LPR 喉鏡所見，幾乎出現於所有 RFS 異常的患者；**聲門下水腫（偽裂）**則是最具特異性的發現，正常人幾乎不會出現。

## 喉咽逆流（LPR）vs. 胃食道逆流（GERD）

| 特性     | LPR                          | GERD                   |
| -------- | ---------------------------- | ---------------------- |
| 主要症狀 | 聲音沙啞、清嗓、咳嗽、梅核氣 | 燒心感、胃酸逆流、胸痛 |
| 燒心感   | 較少見（< 40%）              | 主要症狀（> 80%）      |
| 發作時間 | 白天直立時較多               | 夜間躺下時較多         |
| 喉部損傷 | 常見（RFS 升高）             | 較少                   |
| 食道損傷 | 較少                         | 常見（食道炎）         |
| 診斷工具 | RSI + RFS + 喉鏡             | 胃鏡、24 小時 pH 監測  |
| 主要治療 | PPI + 飲食／生活習慣調整     | PPI ± 抗酸藥           |

## RFS 與 RSI 合併使用

**RFS** 評估醫師觀察到的客觀喉部所見，**RSI** 則記錄患者的主觀症狀，兩者互補：

|              | RSI                | RFS                    |
| ------------ | ------------------ | ---------------------- |
| **評估對象** | 患者自填           | 醫師喉鏡檢查           |
| **性質**     | 主觀症狀           | 客觀所見               |
| **切截值**   | > 13 為異常        | > 7 為異常             |
| **總分**     | 0–45               | 0–26                   |
| **優點**     | 快速篩檢，無需儀器 | 客觀、不受患者主觀影響 |

> **診斷建議**：同時滿足 RSI > 13 且 RFS > 7，診斷 LPR 的特異性可達 95% 以上，可作為試驗性 PPI 治療的依據。

## 治療與監測

### 質子幫浦抑制劑（PPI）

- **標準治療**：每日 2 次（餐前 30 分鐘），療程至少 3–6 個月
- **療效評估**：治療後重新評估 RFS，改善 ≥ 50% 視為有效
- **注意**：LPR 對 PPI 的反應較 GERD 慢，喉部黏膜修復需更長時間

### 飲食與生活習慣調整

- 避免：咖啡、茶、酒精、碳酸飲料、巧克力、番茄製品、油炸食物
- 睡前 3 小時不進食
- 抬高床頭 15–30 公分
- 控制體重
- 戒菸

### 手術治療

保守治療無效者，可考慮：

- **抗逆流手術**（Nissen Fundoplication）：適用於有明確解剖異常（如裂孔疝）的患者
- **內視鏡抗逆流治療**：較新的微創選項

## 注意事項

::: warning RFS 為醫師評估工具，需搭配喉鏡檢查
RFS 必須由耳鼻喉科醫師透過喉鏡檢查後填寫，不適合患者自行評估。RFS 升高僅提示喉部可能有逆流相關損傷，確診仍需結合臨床症狀（RSI）及必要時的 24 小時雙探頭 pH 阻抗監測（Impedance-pH Monitoring）。
:::

::: warning 排除其他喉部疾病
聲帶水腫、後壁肥厚及肉芽腫也可能源於聲帶過度使用、感染、過敏或腫瘤。RFS 高分患者應由醫師進行完整評估，排除其他喉部病因後再進行 LPR 治療。
:::

::: warning 肉芽腫需區分成因
接觸性肉芽腫可由 LPR、長期插管或聲帶過度使用引起。單純以 RFS 肉芽腫項目判定 LPR 時，需結合其他項目及病史綜合判斷，避免過度診斷。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不具診斷效力。喉咽逆流的診斷及治療應由耳鼻喉科或腸胃科醫師根據完整病史、喉鏡及相關檢查綜合判斷。
:::
