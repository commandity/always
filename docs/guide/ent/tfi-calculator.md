# TFI 耳鳴功能指數

**TFI Tinnitus Functional Index**

## 概述

**TFI**（Tinnitus Functional Index）由 Meikle 等人於 2012 年在美國奧勒岡健康與科學大學（OHSU）發展，是目前**對治療反應最敏感的耳鳴評估工具**。TFI 共 25 題，涵蓋 8 個面向，每題以 0–10 連續量尺評分，提供比傳統三選一（如 THI）更細緻的障礙評估。TFI 具有優異的內部一致性（Cronbach's α = 0.97）及再測信度，且最小臨床有意義差值（MCID）僅 13 分，使其成為監測治療效果的理想工具。

## 評分說明

每題以 **0–10 連續量尺**評分，依耳鳴對生活影響的嚴重程度作答：

| 分數 | 說明                    |
| ---- | ----------------------- |
| 0    | 完全沒有影響 / 從不發生 |
| 1–3  | 輕微影響                |
| 4–6  | 中度影響                |
| 7–9  | 重度影響                |
| 10   | 影響最嚴重 / 一直發生   |

**TFI 總分 = 25 題平均分 × 10（轉換為 0–100 分）**

## 臨床切截點

| TFI 總分 | 嚴重程度 | 臨床建議                             |
| -------- | -------- | ------------------------------------ |
| < 25     | 輕微影響 | 耳鳴對生活影響輕微，通常不需積極治療 |
| 25–49    | 中度影響 | 建議就醫評估，考慮聲音治療或諮商     |
| 50–74    | 重大影響 | 耳鳴造成顯著障礙，建議積極多面向治療 |
| ≥ 75     | 嚴重影響 | 嚴重障礙，強烈建議多學科耳鳴治療計劃 |

> **MCID（最小臨床有意義差值）**：治療後 TFI 改善 **≥ 13 分**，即視為具有臨床意義的改善（Meikle 2012）。此數值遠低於 THI 的 20 分，使 TFI 對治療效果更加敏感。

## 八個面向的臨床意義

| 面向     | 英文             | 題數 | 最高分 | 評估重點                                 |
| -------- | ---------------- | ---- | ------ | ---------------------------------------- |
| 侵擾性   | Intrusive        | 3    | 30     | 耳鳴的出現頻率、響度強度、難以忽視的程度 |
| 控制感   | Sense of Control | 3    | 30     | 對耳鳴的掌控感、被壓倒感、應對努力程度   |
| 認知影響 | Cognitive        | 3    | 30     | 注意力集中、清晰思考、忘記耳鳴的困難程度 |
| 睡眠影響 | Sleep            | 3    | 30     | 入睡困難、夜間再入睡、睡眠充足性的干擾   |
| 聽覺影響 | Auditory         | 3    | 30     | 噪音環境聆聽、欣賞媒體、語音理解的干擾   |
| 放鬆影響 | Relaxation       | 3    | 30     | 放鬆、平靜感受、休憩恢復的困難程度       |
| 生活品質 | Quality of Life  | 3    | 30     | 社交、嗜好、工作與家庭職責的干擾程度     |
| 情緒影響 | Emotional        | 4    | 40     | 焦慮、煩躁、憂鬱及整體生活影響           |

> **情緒面向**（E）有 4 題（最高 40 分），權重略高，反映耳鳴對情緒的全面影響；**侵擾性面向**（I）是最直接反映耳鳴本身特性的面向，在治療早期最先出現改善。

## TFI 與 THI 比較

|                    | TFI                | THI                   |
| ------------------ | ------------------ | --------------------- |
| **發表年份**       | 2012               | 1996                  |
| **題目數**         | 25 題              | 25 題                 |
| **計分方式**       | 0–10（連續量尺）   | 0 / 2 / 4（三選一）   |
| **面向數**         | 8 個面向           | 3 個面向（F / E / C） |
| **總分**           | 0–100              | 0–100                 |
| **對治療的敏感度** | 高（MCID = 13 分） | 中（MCID = 20 分）    |
| **臨床用途**       | 精細評估、療效監測 | 初步篩查、分級        |
| **填寫時間**       | 約 5–8 分鐘        | 約 5 分鐘             |

> **臨床建議**：初次就診使用 THI 快速分級，後續療效追蹤使用 TFI 細緻監測各面向變化。兩者合併使用提供最完整的耳鳴評估。

## 各面向的治療目標

| 面向     | 主要干預策略                                                          |
| -------- | --------------------------------------------------------------------- |
| 侵擾性   | 聲音治療（白噪音、助聽器）降低耳鳴與背景聲對比                        |
| 控制感   | 耳鳴再訓練治療（TRT）、正念冥想（Mindfulness）、接受與承諾治療（ACT） |
| 認知影響 | 認知行為治療（CBT）、注意力訓練                                       |
| 睡眠影響 | 睡眠衛教、白噪音機、必要時短期藥物輔助                                |
| 聽覺影響 | 助聽器擬合、聽覺復健、溝通策略訓練                                    |
| 放鬆影響 | 漸進式肌肉放鬆、正念減壓（MBSR）                                      |
| 生活品質 | 活動計劃、社交支持、職業諮商                                          |
| 情緒影響 | CBT、SSRI（合併憂鬱/焦慮）、心理支持                                  |

## 治療後評估指引

### 療效判斷標準

| TFI 改善幅度 | 臨床意義         |
| ------------ | ---------------- |
| < 13 分      | 無顯著改善       |
| 13–20 分     | 輕度有意義的改善 |
| 21–40 分     | 中度有意義的改善 |
| > 40 分      | 顯著改善         |

### 追蹤時程

- **治療開始後 4 週**：評估侵擾性與睡眠面向的早期改善
- **治療後 8–12 週**：完整 TFI 重新評估，調整治療計劃
- **長期追蹤（6 個月、1 年）**：監測習慣化成效與慢性化風險

## 注意事項

::: warning TFI 不能單獨診斷耳鳴病因
TFI 評估耳鳴對 8 個生活面向的影響，但無法區分耳鳴的原因。高分患者應接受完整聽覺評估（純音聽力、耳鳴音調匹配、ABR）及必要時的影像學檢查，以排除聽神經瘤或其他嚴重病因。
:::

::: warning 睡眠面向高分需特別關注
TFI 睡眠面向（S）> 20 分者，耳鳴引起的睡眠障礙可能形成惡性循環，使耳鳴感知更強烈。此類患者應優先處理睡眠問題，必要時轉介睡眠醫學評估。
:::

::: warning 情緒面向高分需同步心理評估
TFI 情緒面向（E）> 30 分，合併憂鬱或焦慮症狀者，耳鳴治療需與精神科或心理科合作。未處理的情緒障礙是耳鳴治療失敗的最常見原因之一。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不具診斷效力。耳鳴的診斷及治療應由耳鼻喉科、聽力師或相關專科醫師根據完整病史、聽力檢查及相關評估綜合判斷。
:::
