# THI 耳鳴障礙量表

**THI Tinnitus Handicap Inventory**

## 概述

**THI**（Tinnitus Handicap Inventory）由 Newman 等人於 1996 年在美國底特律亨利福特醫院發展，是目前全球**最廣泛使用的耳鳴自填評估工具**。THI 共 25 題，涵蓋功能（F）、情緒（E）、災難化（C）三個面向，評估耳鳴對日常生活的整體影響程度，具有優異的內部一致性（Cronbach's α = 0.93）及再測信度（r = 0.94），並被翻譯為超過 20 種語言廣泛應用於臨床與研究。

## 評分說明

每題三個選項，依耳鳴症狀發生的頻率作答：

| 分數 | 中文 | 英文      | 說明           |
| ---- | ---- | --------- | -------------- |
| 0    | 從不 | No        | 此情況從未發生 |
| 2    | 有時 | Sometimes | 此情況偶爾發生 |
| 4    | 總是 | Yes       | 此情況總是發生 |

**總分範圍：0–100 分**（F 面向 0–44，E 面向 0–36，C 面向 0–20）

## 臨床切截點（五級分類）

| THI 總分 | 障礙等級 | 嚴重程度   | 臨床建議                             |
| -------- | -------- | ---------- | ------------------------------------ |
| 0–16     | Grade 1  | 輕微障礙   | 衛教、聽覺追蹤，提供耳鳴資訊         |
| 18–36    | Grade 2  | 輕度障礙   | 就醫評估，考慮聲音治療或助聽器       |
| 38–56    | Grade 3  | 中度障礙   | 聲音治療合併諮商，考慮 TRT           |
| 58–76    | Grade 4  | 重度障礙   | 積極耳鳴再訓練治療（TRT），心理支持  |
| 78–100   | Grade 5  | 極重度障礙 | 多學科團隊評估，TRT + CBT + 藥物治療 |

> **關鍵切截值：THI > 16** 提示耳鳴造成顯著障礙，需要進一步評估與介入。治療後 THI 改善 **≥ 20 分**被視為臨床有意義的進步（MCID）。

## 三個面向的臨床意義

| 面向        | 題數  | 最高分 | 評估重點                                               |
| ----------- | ----- | ------ | ------------------------------------------------------ |
| 功能（F）   | 11 題 | 44 分  | 耳鳴對注意力、聽覺、睡眠、閱讀、工作與社交的功能性影響 |
| 情緒（E）   | 9 題  | 36 分  | 耳鳴引起的憤怒、沮喪、憂鬱、焦慮、不安全感等情緒反應   |
| 災難化（C） | 5 題  | 20 分  | 對耳鳴的無望感、無力感、無法忍受感與嚴重疾病認知       |

> **災難化面向**（C）雖只有 5 題，卻是預測耳鳴慢性化與治療反應的重要指標。C 面向分數高（≥ 12）者往往對耳鳴產生災難化認知，需優先進行認知行為治療（CBT）。

## 耳鳴的常見病因與 THI 特徵

| 病因                      | 典型 THI | 面向特徵             |
| ------------------------- | -------- | -------------------- |
| 噪音性聽損（NIHL）        | 20–50    | F 面向為主           |
| 老年性聽損（Presbycusis） | 15–40    | F 面向，E 面向相對低 |
| 梅尼爾氏病                | 40–70    | F + E，與眩暈並存    |
| 特發性突發性耳聾後        | 50–80    | 三個面向均高         |
| 聽神經瘤術後              | 40–70    | F + C 面向           |
| 焦慮症合併耳鳴            | 60–100   | E + C 面向顯著       |
| 慢性中耳炎                | 20–45    | 治療後常改善         |

## THI 與 TFI 比較

|              | THI                 | TFI                                                                |
| ------------ | ------------------- | ------------------------------------------------------------------ |
| **發表年份** | 1996                | 2012                                                               |
| **題目數**   | 25 題               | 25 題                                                              |
| **計分方式** | 0 / 2 / 4（三選一） | 0–10（連續量尺）                                                   |
| **面向數**   | 3（F / E / C）      | 8（侵擾性 / 控制感 / 認知 / 睡眠 / 聽覺 / 放鬆 / 生活品質 / 情緒） |
| **總分**     | 0–100               | 0–100                                                              |
| **優點**     | 快速填寫，分類明確  | 面向細分，對治療反應更敏感                                         |
| **切截值**   | > 16 為顯著障礙     | < 25 輕微 / 25–49 中度 / ≥ 50 顯著                                 |
| **MCID**     | ≥ 20 分             | ≥ 13 分                                                            |

> **臨床建議**：THI 適合初步篩查與分級，TFI 適合精細評估各面向影響及治療效果監測。兩者合併使用可提供最完整的耳鳴障礙評估。

## 耳鳴治療與監測

### 聲音治療（Sound Therapy）

- **白噪音遮蔽**：降低耳鳴與環境聲的對比度，減少侵擾感
- **助聽器**：合併聽損者優先，改善聽覺功能同時減輕耳鳴
- **耳鳴再訓練治療（TRT）**：結合低強度聲音治療與諮商，訓練大腦習慣化（Habituation）耳鳴

### 認知行為治療（CBT）

- 適用對象：THI 情緒（E）或災難化（C）面向分數高者
- 療程：8–12 週，每週一次
- 效果：改善耳鳴相關焦慮、憂鬱，降低耳鳴困擾程度

### 藥物治療

- **抗焦慮藥**（如 Benzodiazepines）：短期緩解，不建議長期使用
- **抗憂鬱藥**（如 SSRI）：合併憂鬱或焦慮症者考慮
- **褪黑激素**：改善耳鳴相關睡眠障礙
- **注意**：目前無 FDA 核准的耳鳴特效藥物

### 治療後再評估

- **治療前**：建立基準 THI 分數，記錄三個面向分布
- **治療後 4–8 週**：重新評估，改善 **≥ 20 分**視為有臨床意義
- **長期追蹤**：每 3–6 個月評估，監測慢性化趨勢

## 注意事項

::: warning THI 是自填問卷，不能單獨診斷耳鳴病因
THI 評估耳鳴對生活的障礙程度，但無法區分病因（如耳蝸性、聽神經性、中樞性）。THI 高分患者應接受完整聽覺評估（包括純音聽力檢查、耳鳴音調與響度匹配、聽腦幹反應）及必要時的影像學檢查，以排除嚴重病因。
:::

::: warning 高 C 面向分數需優先心理支持
災難化面向（C）≥ 12 分者，對耳鳴的災難化認知可能妨礙所有治療效果。此類患者應優先轉介認知行為治療（CBT），單純聲音治療效果有限。
:::

::: warning 合併憂鬱或焦慮需同步處理
THI 情緒（E）面向分數 ≥ 24 分，且合併情緒障礙者，耳鳴治療需與精神科或心理科合作。憂鬱未處理者，耳鳴治療反應往往較差。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不具診斷效力。耳鳴的診斷及治療應由耳鼻喉科、聽力師或相關專科醫師根據完整病史、聽力檢查及相關評估綜合判斷。
:::
