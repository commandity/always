## THI 耳鳴障礙量表

**Tinnitus Handicap Inventory**（Newman 1996）共 25 題，計分方式為 Yes（4 分）/ Sometimes（2 分）/ No（0 分），總分 0–100 分。

| 面向                   | 題數  | 核心評估               |
| ---------------------- | ----- | ---------------------- |
| Functional（功能性）   | 12 題 | 耳鳴對日常功能的影響   |
| Emotional（情緒）      | 9 題  | 耳鳴引起的情緒反應     |
| Catastrophic（災難性） | 4 題  | 絕望感與無法擺脫的感受 |

### THI 分數解讀

| 總分   | 等級（Grade）     | 意義與建議處置                                       |
| ------ | ----------------- | ---------------------------------------------------- |
| 0–16   | 一級・無或輕微    | 僅在安靜環境察覺耳鳴，衛教與聽覺追蹤即可             |
| 18–36  | 二級・輕度        | 環境音可蓋過耳鳴，建議就醫評估，考慮聲音治療         |
| 38–56  | 三級・中度        | 背景噪音中仍察覺耳鳴，建議聲音治療合併諮商           |
| 58–76  | 四級・重度        | 耳鳴幾乎持續干擾睡眠與日常活動，建議耳鳴再訓練（TRT）|
| 78–100 | 五級・極重度      | 耳鳴影響所有活動，強烈建議多學科完整評估與治療       |

> **THI ≤ 16 分**視為正常範圍；分數越高，代表耳鳴對功能、情緒的侵擾越深。治療前後比較時，下降 **20 分以上**才代表有臨床意義的改善（MCID）。若 Catastrophic 面向分數偏高（絕望感強烈），需特別留意情緒共病並考慮心理轉介。

## TFI 耳鳴功能指數

**Tinnitus Functional Index**（Meikle 2012）共 25 題，每題 0–10 分，總分 0–250 分（標準化為 0–100 分）。

| 面向                        | 題數 |
| --------------------------- | ---- |
| Intrusive（干擾性）         | 3 題 |
| Sense of Control（控制感）  | 3 題 |
| Cognitive（認知）           | 3 題 |
| Sleep（睡眠）               | 3 題 |
| Auditory（聽覺）            | 3 題 |
| Relaxation（放鬆）          | 3 題 |
| Quality of Life（生活品質） | 4 題 |
| Emotional（情緒）           | 3 題 |

### TFI 分數解讀

| 總分  | 影響程度 | 意義與建議處置                             |
| ----- | -------- | ------------------------------------------ |
| 0–24  | 輕微影響 | 耳鳴對生活影響小，通常不需積極治療         |
| 25–49 | 中度影響 | 有明確困擾，建議就醫評估，考慮聲音治療或諮商 |
| 50–74 | 重大影響 | 耳鳴造成顯著功能障礙，建議積極治療         |
| ≥ 75  | 嚴重影響 | 強烈建議多學科耳鳴治療計畫                 |

> **TFI ≥ 25 分**為「需要介入」的參考門檻；**下降 13 分以上**代表治療產生有意義的改善（MCID）。TFI 對治療反應較 THI 敏感，適合用於療效追蹤；八個子面向分數則可指出最需優先處理的向度（如睡眠、情緒）。

## 注意事項

:::
