# VHI-10 嗓音障礙指數

**VHI-10 Voice Handicap Index**

## 概述

**VHI-10**（Voice Handicap Index–10）是 Rosen 等人於 2004 年從完整版 **VHI-30** 中篩選出的精簡版嗓音生活品質量表，保留了 VHI-30 中最具代表性、信效度最高的 10 個題目。VHI-10 評估嗓音障礙對患者**日常功能**與**情緒**的主觀影響，是目前嗓音門診中使用最廣泛的患者自填嗓音障礙問卷，施測時間僅需 2–3 分鐘。

## 量表架構

VHI-10 共 **10 個問題**，涵蓋兩個面向：

### 功能面向（Functional Domain，F）

評估嗓音障礙對**日常溝通功能**的影響，包括電話使用、被聽見的困難及工作影響：

| 題號 | 評估內容               |
| ---- | ---------------------- |
| Q1   | 嗓音讓別人難以聽清楚   |
| Q2   | 嘈雜環境中更難被理解   |
| Q3   | 家人在另一房間難以聽到 |
| Q4   | 減少使用電話           |
| Q10  | 影響收入、工作或晉升   |

### 情緒面向（Emotional Domain，E）

評估嗓音障礙帶來的**心理與情緒影響**：

| 題號 | 評估內容             |
| ---- | -------------------- |
| Q5   | 因嗓音問題減少外出   |
| Q6   | 覺得自己有殘缺       |
| Q7   | 被要求重複時感到煩躁 |
| Q8   | 說話時感到焦慮       |
| Q9   | 感到無能為力         |

> **注意**：完整版 VHI-30 另含**身體面向（Physical Domain，P）**，評估嗓音的感知特質（如聲音用力費力、不穩定等）。VHI-10 省略了身體面向，專注於功能與情緒兩個對生活影響最直接的面向。

## 評分說明

每題依頻率評分：

| 分數 | 描述   | 頻率說明       |
| ---- | ------ | -------------- |
| 0    | 從不   | 從未發生       |
| 1    | 幾乎不 | 偶爾，極少發生 |
| 2    | 有時   | 有時發生       |
| 3    | 經常   | 頻繁發生       |
| 4    | 總是   | 幾乎每次都發生 |

**總分範圍：0–40 分**（分數越高，嗓音障礙對生活影響越大）

## 嚴重度分級

| VHI-10 總分 | 嚴重程度 | 臨床建議                                     |
| ----------- | -------- | -------------------------------------------- |
| 0–10        | 輕度障礙 | 嗓音問題對生活影響輕微，可觀察或基礎嗓音衛教 |
| 11–20       | 中度障礙 | 中等程度影響，建議嗓音評估及嗓音治療         |
| 21–40       | 重度障礙 | 嚴重影響生活與工作，強烈建議積極治療介入     |

> **MCID（最小臨床重要差異）**：VHI-10 分數改善 **≥ 5 分**視為具臨床意義的改善，常用於嗓音治療或手術前後療效評估。

## VHI-10 vs VHI-30

| 特性     | VHI-10               | VHI-30                          |
| -------- | -------------------- | ------------------------------- |
| 題數     | 10 題                | 30 題                           |
| 施測時間 | 2–3 分鐘             | 5–10 分鐘                       |
| 面向     | 功能（F）、情緒（E） | 功能（F）、情緒（E）、身體（P） |
| 總分範圍 | 0–40                 | 0–120                           |
| 臨床使用 | 門診快速篩檢、追蹤   | 研究、詳細分析                  |
| MCID     | ≥ 5 分               | ≥ 18 分                         |

## 臨床應用

VHI-10 廣泛應用於以下情境：

- **嗓音障礙初始評估**：建立患者主觀嗓音品質基線
- **嗓音治療療效追蹤**：治療前後比較，監測改善程度
- **手術適應症評估**：聲帶顯微手術、聲帶注射前後比較
- **痙攣性發聲障礙（SD）**：Botox 注射療效評估
- **嗓音復健**：甲狀腺手術、心臟手術後聲帶麻痺的功能影響評估
- **嗓音職業人士**：教師、歌手、演員等高嗓音需求職業的職業嗓音健康評估
- **研究標準化**：國際嗓音研究的通用患者報告結果指標（PROM）

## 嗓音障礙常見疾病的 VHI-10 特徵

| 嗓音疾病       | 典型 VHI-10 特點                              |
| -------------- | --------------------------------------------- |
| 聲帶結節       | F 分數偏高（工作影響明顯），整體中度          |
| 聲帶麻痺       | 功能面向 F 顯著（電話、被聽見困難），整體較高 |
| 痙攣性發聲障礙 | 情緒面向 E 顯著（焦慮、社交逃避），整體偏高   |
| 老年嗓音萎縮   | F 分數中等（溝通功能受影響），情緒面向相對低  |
| 喉癌術後       | 兩個面向均顯著偏高，整體常達重度              |
| 心因性失聲     | E 分數顯著偏高（情緒、焦慮），F 可能高低不定  |

## 注意事項

::: warning VHI-10 為患者主觀自填問卷
VHI-10 反映患者的**主觀感受**，可能與客觀的嗓音聲學指標（如 GRBAS、聲學分析）不完全一致。部分患者因適應慢性嗓音問題而低估障礙程度（adaptation bias），建議同時使用 GRBAS 及客觀聲學分析進行全面評估。
:::

::: warning 情緒面向高分需關注心理狀態
VHI-10 情緒面向（E）分數顯著偏高（≥ 12）的患者，可能合併焦慮、憂鬱或社交恐懼。評估時應留意相關心理症狀，必要時轉介心理諮商或身心科評估。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不具診斷效力。嗓音障礙的診斷與治療應由耳鼻喉科醫師（嗓音專科）或語言治療師進行完整評估，包括喉鏡檢查、頻閃喉鏡及客觀聲學分析。
:::
