# VoiSS 嗓音症狀量表

**VoiSS Voice Symptom Scale**
<VoissCalculator />

## 量表架構

VoiSS 共 **30 個題目**，分為三個面向：

| 面向          | 英文                    | 題數   | 最高分  | 評估核心                         |
| ------------- | ----------------------- | ------ | ------- | -------------------------------- |
| 功能限制（I） | Impairment / Limitation | 15     | 60      | 嗓音品質改變、溝通困難、嗓音疲勞 |
| 情緒影響（E） | Emotional Impact        | 8      | 32      | 沮喪、焦慮、社交迴避、自我意識   |
| 身體症狀（P） | Physical Symptoms       | 7      | 28      | 喉嚨黏液、清嗓、喉痛、異物感     |
| **總分**      | **Total**               | **30** | **120** |                                  |

## 評分說明

每題依過去**一週**症狀出現的頻率評分：

| 分數 | 描述   | 頻率         |
| ---- | ------ | ------------ |
| 0    | 從不   | 從未發生     |
| 1    | 幾乎不 | 極少發生     |
| 2    | 有時   | 偶爾發生     |
| 3    | 經常   | 頻繁發生     |
| 4    | 總是   | 幾乎每次都有 |

**總分範圍：0–120 分**（分數越高，嗓音症狀越嚴重）

## 嚴重度分級

| VoiSS 總分 | 嚴重程度 | 臨床建議                                 |
| ---------- | -------- | ---------------------------------------- |
| 0–14       | 正常範圍 | 嗓音症狀在正常範圍，無顯著障礙           |
| 15–29      | 輕度障礙 | 輕度嗓音症狀，建議嗓音衛教與生活調整     |
| 30–49      | 中度障礙 | 中度嗓音障礙，建議就醫評估與嗓音治療     |
| ≥ 50       | 重度障礙 | 重度嗓音障礙，強烈建議積極喉科評估與治療 |

> **診斷準確性**：VoiSS 對嗓音障礙的**敏感度約 83–86%，特異度約 63–67%**，是具備良好臨床效度的篩檢工具。

## 各面向詳細說明

### 功能限制（I — Impairment，15 題，最高 60 分）

功能限制面向是 VoiSS 中**題數最多、權重最大**的部分，涵蓋患者的實際嗓音功能改變：

- **嗓音品質**：嗓音變化、嘶啞、粗糙感、聲音時好時壞
- **溝通效能**：在嘈雜環境難以被聽清、需要重複、說話費力
- **嗓音耐力**：說話後聲音疲勞、聲音在長時間使用後變差
- **嗓音行為改變**：避免使用電話、難以大聲說話

功能限制面向分數高，通常反映有客觀的聲帶病理（如聲帶息肉、結節、麻痺）。

### 情緒影響（E — Emotional，8 題，最高 32 分）

情緒面向評估嗓音障礙對**心理與社交層面**的影響：

- 沮喪、焦慮、挫折感
- 因嗓音問題感到窘迫或無能為力
- 減少社交活動、公共場合的自我意識

情緒面向分數高，特別常見於**痙攣性發聲障礙（Spasmodic Dysphonia）**、職業歌手及需要高度嗓音表現的職業人士，也可能合併焦慮症或憂鬱症。

### 身體症狀（P — Physical，7 題，最高 28 分）

身體症狀面向評估**喉嚨與聲道的局部感覺症狀**，這些症狀往往是患者就醫的直接主訴：

- 喉嚨黏液或痰（最常見）
- 清嗓衝動
- 說話時喉嚨疼痛或刺激感
- 咽喉異物感（梅核氣）
- 喉嚨緊繃感

身體症狀面向高分常見於**慢性喉炎、胃食道逆流性喉炎（LPR）、喉部過敏**及**肌肉張力性發聲障礙（MTD）**。

## VoiSS 與其他嗓音量表的比較

| 特性       | VoiSS         | VHI-10   | VHI-30     | GRBAS         |
| ---------- | ------------- | -------- | ---------- | ------------- |
| 填答者     | 患者          | 患者     | 患者       | 臨床醫師      |
| 題數       | 30            | 10       | 30         | 5 參數        |
| 面向數     | 3（I/E/P）    | 2（F/E） | 3（F/E/P） | 5             |
| 身體症狀   | ✓（喉嚨症狀） | ✗        | ✓          | 間接（R/B/S） |
| 最高分     | 120           | 40       | 120        | 15            |
| 評估時間窗 | 過去一週      | 無特定   | 無特定     | 當下          |
| 施測時間   | 8–10 分鐘     | 2–3 分鐘 | 8–10 分鐘  | 5 分鐘        |

> VoiSS 的最大特色在於包含**身體症狀（P）面向**，能捕捉到 VHI-10 所遺漏的喉嚨局部症狀，對咽喉合併症狀（如 LPR、慢性咳嗽）的評估更為完整。

## 臨床應用

VoiSS 適用於以下情境：

- **嗓音障礙初始全面評估**：建立三面向基線分數
- **治療前後療效比較**：嗓音治療、手術、Botox 注射的療效監測
- **嗓音職業健康篩檢**：教師、歌手、演員等職業的定期評估
- **胃食道逆流性喉炎（LPR）**：身體症狀（P）面向對喉嚨黏液、清嗓、刺激感的評估特別敏感
- **痙攣性發聲障礙（SD）**：情緒面向（E）分數偏高是此病的特徵
- **研究標準化**：多面向嗓音障礙研究的患者報告結果指標

## 注意事項

::: warning P 面向高分需排除喉咽逆流（LPR）
身體症狀面向（P）中的喉嚨黏液、清嗓、刺激感及異物感，是**胃食道逆流性喉炎**（LPR）的典型主訴。若 P 面向分數顯著偏高，建議同時評估逆流症狀（如可搭配 RSI 量表），並考慮喉鏡檢查。
:::

::: warning E 面向高分需關注心理狀態
情緒面向（E）分數顯著偏高的患者，可能合併焦慮症、憂鬱症或社交恐懼症。嗓音治療師或耳鼻喉科醫師應留意，必要時轉介心理諮商。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不具診斷效力。嗓音障礙的完整評估應包括喉鏡檢查、頻閃喉鏡、客觀聲學分析及 GRBAS 聽知覺評估，由耳鼻喉科醫師或嗓音語言治療師綜合判斷。
:::
