# BHT & AD8 認知功能複評工具

**BHT & AD8 Cognitive Screening Tools**

## 概述

**BHT**（Blessed Orientation-Memory-Concentration Test，Blessed 定向記憶集中測驗）與 **AD8**（8-item Informant Interview to Differentiate Aging and Dementia）是 WHO ICOPE 認知功能初評異常後，進行**第二階段複評**的標準工具。當長者在 ICOPE 認知初評（記憶力或定向力任一題回答「否」）出現異常時，應依照填答者選擇合適版本進行複評。

## ICOPE 認知功能篩查流程

依據 **112 年國民健康署 ICOPE 長者功能評估**，認知功能域的初評與複評流程如下：

### 初評（ICOPE 認知三題）

| 題目           | 評估方式                                                   | 異常判定     |
| -------------- | ---------------------------------------------------------- | ------------ |
| 記憶力         | 說出「鉛筆、汽車、書」三項，第三題後再詢問，能否回憶全部？ | 無法回憶     |
| 定向力（時間） | 詢問「今天的日期？」（年月日）                             | 答錯或不知道 |
| 定向力（地點） | 詢問「您現在在哪裡？」                                     | 答錯或不知道 |

> 任一答案為「否（異常）」→ 進入複評流程

### 複評工具選擇

```
ICOPE 認知初評異常
       ↓
 長者能自行作答？
   ├─ 是 → BHT（由長者本人接受施測）
   └─ 否 → AD8（由家屬或主要照顧者填答）
              ↓ 複評陽性
       轉介 + 社會照護支持評估
              ↓
   神經內科 / 精神科 / 老年醫學科
   完整失智症評估（MMSE、MoCA、影像）
```

## BHT（Blessed 定向記憶集中測驗）

### 量表簡介

BHT 由 **Blessed、Tomlinson 及 Roth（1968）**研發，是最早系統化評估老年認知功能的工具之一。量表共 **6 題**，採**加權計分**，依題目難度給予不同權重，總分 **0–28 分**。

### 計分方式

| 題目 | 評估內容                                   | 加權 | 計分說明                     |
| ---- | ------------------------------------------ | ---- | ---------------------------- |
| B1   | 今年是幾年？                               | ×4   | 錯誤得 4 分                  |
| B2   | 現在是幾月？                               | ×3   | 錯誤得 3 分                  |
| B3   | 現在大約幾點？（不看時鐘，誤差 ≤1 小時）   | ×3   | 錯誤得 3 分                  |
| B4   | 從 20 倒數到 1                             | ×2   | 每個錯誤 ×2（最高 5 個錯誤） |
| B5   | 從 12 月倒念到 1 月                        | ×2   | 每個錯誤 ×2（最高 5 個錯誤） |
| B6   | 記憶詞回憶（姓名、數字、地名）五分鐘後詢問 | ×2   | 每項無法回憶 ×2（最高 3 項） |

> **計分原則**：正確作答 = 0 分（不加分）。**分數越高代表認知功能越差。**

### 分級解釋

| BHT 總分 | 認知狀態             | 建議處置                     |
| -------- | -------------------- | ---------------------------- |
| 0–4      | 正常                 | 6–12 個月定期追蹤            |
| 5–9      | 輕度認知障礙（疑似） | 建議進一步 MoCA 或 MMSE 評估 |
| 10–16    | 中度認知障礙         | 轉介神經內科或老年醫學科     |
| ≥ 17     | 重度認知障礙         | 立即安排完整神經認知評估     |

> 原始文獻（Blessed et al., 1968）以 ≥ 10 分作為認知異常的切截點，對應病理組織中神經纖維纏結的密度。

### 施測注意事項

**B4、B5（倒數計算）：** 記錄錯誤次數，包含跳過、順序錯誤、或停頓超過 10 秒。若長者完全無法開始，記為 5 個錯誤。

**B6（記憶詞）：** 施測前先讓長者重複三次確認記住，進行其他題目後約五分鐘再詢問。以台灣常用版本：「**張偉明、45、高雄**」（姓名算一項、數字算一項、地名算一項，最多 3 項錯誤）。

## AD8（8 題知情者訪談）

### 量表簡介

AD8 由 **Galvin、Roe、Xiong 及 Morris（2005）**於華盛頓大學研發，全名為 _8-item Informant Interview to Differentiate Aging and Dementia_。屬**公共領域**（public domain），可自由使用。

AD8 的最大特點是：**由家屬或主要照顧者填答**，評估長者認知功能相較於過去數年間的**改變情況**，不受長者當下配合度或認知狀態影響，是長者認知能力不足以自行作答時的首選替代工具。

### 計分方式

每題有三個選項：

| 選項         | 分數   | 含義               |
| ------------ | ------ | ------------------ |
| 是，有改變   | 1 分   | 觀察到認知功能退步 |
| 否，沒有改變 | 0 分   | 未觀察到退步       |
| 不知道       | 不計分 | 無法判斷           |

**總分 = 回答「是，有改變」的題數（0–8 分）**

### 分級解釋

| AD8 總分 | 臨床意義                       | 建議處置                             |
| -------- | ------------------------------ | ------------------------------------ |
| 0–1      | 認知功能正常（或未觀察到改變） | 定期追蹤，6–12 個月後再評估          |
| 2–3      | 疑似輕度認知障礙               | 建議安排 MoCA 或 MMSE，追蹤 3–6 個月 |
| ≥ 4      | 疑似失智症（建議完整評估）     | 轉介神經內科、精神科或老年醫學科     |

> AD8 ≥ 2 分的敏感性為 **84%**、特異性為 **86%**，對輕度失智症（CDR 0.5）有良好篩查效力（Galvin et al., 2005）。

### AD8 的八大評估面向

| A#  | 評估面向                 | 臨床意義                             |
| --- | ------------------------ | ------------------------------------ |
| A1  | 判斷力（做決定有困難）   | 執行功能退化的早期指標               |
| A2  | 興趣降低（活動和嗜好）   | 冷漠（apathy）常見於早期失智         |
| A3  | 重複（問題、故事、話題） | 情節記憶受損的典型表現               |
| A4  | 學習新器具有困難         | 程序性記憶及學習能力退化             |
| A5  | 忘記年月日               | 定向力障礙的家屬觀察指標             |
| A6  | 複雜財務有困難           | 執行功能及工具性日常活動（IADL）     |
| A7  | 記不住約定               | 前瞻性記憶（prospective memory）退化 |
| A8  | 每天思考/記憶問題        | 整體認知功能的全面性退化             |

### 為何需要家屬填答？

老年認知障礙患者常有**病識感缺乏（anosognosia）**，無法準確評估自身認知功能的退步。家屬或主要照顧者透過長期相處，往往能更早、更準確地察覺細微變化，AD8 正是利用這一特性設計。

> 研究顯示，AD8 由知情家屬填答的準確性顯著優於患者自填，尤其在早期（CDR 0.5）失智症的偵測上。

## 複評陽性後的處理流程

### 介入與轉介（參考國健署 ICOPE 計畫步驟）

```
BHT ≥ 5 或 AD8 ≥ 2（複評陽性）
              ↓
評估社會照護與支持的需求
（如：社區據點、長照服務）
              ↓
照護個案以支持個人日常生活活動
              ↓
轉介醫療資源（擇一或多項）：
  ├─ 神經內科（認知/失智專科門診）
  ├─ 精神科（老年精神醫學）
  ├─ 失智共照中心
  ├─ 老年醫學科
  └─ 家庭醫學科（後續轉介）
              ↓
追蹤個案是否成功轉介或進行介入
              ↓
3–6 個月後再次評估異常項目的變化
```

### 完整失智症評估項目

轉介後，醫療機構通常進行以下標準評估：

| 評估類型     | 常用工具             | 說明                     |
| ------------ | -------------------- | ------------------------ |
| 認知功能     | MMSE、MoCA、ADAS-Cog | 量化認知功能受損程度     |
| 日常功能     | ADL、IADL            | 評估生活自理能力         |
| 行為症狀     | NPI、GDS             | 評估精神行為症狀及憂鬱   |
| 影像學       | 頭顱 MRI 或 CT       | 排除腦血管病變、腫瘤     |
| 血液檢查     | TSH、B12、葉酸、CBC  | 排除可治療的認知障礙原因 |
| 神經心理測驗 | 視空間、語言、執行   | 確認失智症類型           |

## 兩工具比較

| 比較項目       | BHT                           | AD8                               |
| -------------- | ----------------------------- | --------------------------------- |
| 填答者         | 長者本人                      | 家屬 / 主要照顧者                 |
| 題數           | 6 題（加權）                  | 8 題（等分）                      |
| 施測時間       | 3–5 分鐘                      | 2–4 分鐘                          |
| 計分範圍       | 0–28 分                       | 0–8 分                            |
| 切截點（陽性） | ≥ 5 分                        | ≥ 2 分                            |
| 適用情境       | 長者能配合作答                | 長者無法/不宜自填                 |
| 不適用情境     | MMSE < 15、失語、重度認知障礙 | 無長期知情家屬、獨居              |
| 敏感性         | ~82%                          | 84%                               |
| 特異性         | ~81%                          | 86%                               |
| 版權           | 公共領域                      | 公共領域（Washington University） |

## 老年認知障礙背景

### 失智症流行病學（台灣）

- 65 歲以上老年人失智症盛行率：約 **8%**
- 80 歲以上老年人：盛行率高達 **20–30%**
- 台灣 2023 年估計失智人口超過 **31 萬人**，但確診率不足 50%
- 輕度認知障礙（MCI）盛行率：約 **15–20%**，每年約 10–15% 轉化為失智症

### 及早發現的重要性

早期篩查與介入可帶來以下效益：

- 排除**可逆性認知障礙**（甲狀腺低下、B12 缺乏、憂鬱症、藥物副作用等）
- 延緩失智症病程進展（非藥物介入：認知訓練、運動、社會參與）
- 提早規劃**預立醫療決定（ACP）**與財務法律安排
- 連結長照資源，減輕照顧者負擔
- 藥物治療（如乙醯膽鹼酯酶抑制劑）在輕度至中度失智症效果最佳

### 認知障礙的常見可逆原因

ICOPE 及 BHT/AD8 陽性後，轉介評估時須優先排除以下可治療原因：

| 類別        | 常見原因                                                 |
| ----------- | -------------------------------------------------------- |
| 代謝/內分泌 | 甲狀腺功能低下、低血糖、電解質異常（低血鈉）、腎功能不全 |
| 營養缺乏    | 維生素 B12 缺乏、葉酸缺乏、維生素 B1 缺乏（Wernicke）    |
| 藥物        | 抗膽鹼藥物、苯二氮平類、抗組織胺、多重用藥（≥ 5 種）     |
| 感染        | 泌尿道感染、肺炎（急性譫妄，非失智症）                   |
| 精神疾患    | 憂鬱症（假性失智）、焦慮症                               |
| 神經疾患    | 常壓性水腦、腦膜瘤、慢性硬膜下血腫                       |

## 注意事項

::: warning BHT 不適用嚴重認知障礙者
MMSE < 10 或無法配合施測（失語、重度焦慮、急性譫妄）的長者，BHT 結果不可靠。應改由家屬填答 AD8，或待急性狀況緩解後再評估。
:::

::: warning 區分譫妄（Delirium）與失智症
急性發作、注意力波動、意識混亂的認知改變，可能為**譫妄**而非失智症。譫妄需優先排除感染、藥物、代謝等急性原因，不應在急性期以 BHT 或 AD8 診斷認知障礙。
:::

::: warning AD8 需要可靠的知情者
若家屬與長者相處時間不足（如分開居住、關係疏遠），或家屬本身有認知問題，AD8 的結果可信度降低。此時應盡量尋找接觸最多的主要照顧者，或考慮其他評估方式。
:::

::: info ICOPE 認知評估位置
本工具為 ICOPE 初評後的**複評（第二步）**。初評使用三題記憶力及定向力問題；複評陽性後進入**介入/轉介（第三步）**，再進行**追蹤（第四步，3–6 個月後測）**。
:::

::: info 版權說明
BHT 原版由 Blessed、Tomlinson 及 Roth 於 1968 年發表，屬公共領域。AD8 由美國華盛頓大學 Galvin 等人研發，屬公共領域，可自由用於非商業臨床及研究目的。中文翻譯版請參考台灣失智症協會或國健署提供之官方譯本。
:::

::: info 免責聲明
本計算器依據 BHT 及 AD8 量表設計，僅供臨床教育及輔助評估參考，不能取代完整的精神科或神經科診斷訪談。複評陽性者請轉介至適當醫療機構進行確認診斷。
:::
