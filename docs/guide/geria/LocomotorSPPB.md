# SPPB 行動功能複評工具

**SPPB Short Physical Performance Battery**

## 概述

**SPPB**（Short Physical Performance Battery，簡易身體表現功能量表）是 WHO ICOPE 行動功能初評異常後，進行**第二階段複評**的標準工具。由 Guralnik 等人於 1994 年研發，涵蓋三個客觀測試：平衡測試、步行速度測試及椅子起身測試，總分 0–12 分，是目前評估老年人下肢功能與跌倒風險最廣泛使用的工具之一。

## ICOPE 行動功能篩查流程

依據 **112 年國民健康署 ICOPE 長者功能評估**，行動功能域的初評與複評流程如下：

### 初評（ICOPE 椅子起身單題）

| 評估方式                                                 | 異常判定             |
| -------------------------------------------------------- | -------------------- |
| 雙手抱胸，從椅子連續起立坐下五次，計時是否在 12 秒內完成 | 超過 12 秒或無法完成 |

> 初評異常（椅子起身 > 12 秒）→ 進入 SPPB 複評流程

### 複評工具選擇

```
ICOPE 行動功能初評異常
（椅子起身 > 12 秒或無法完成）
              ↓
        複評：SPPB
（平衡 + 步行速度 + 椅子起身三項）
              ↓
      深入評估（依複評結果）
   ├─ 潛在疾病或問題：
   │    多重用藥、骨關節炎、骨質疏鬆、
   │    衰弱症或肌少症、疼痛
   └─ 環境社會支持：
        跌倒風險評估（外在環境、居家環境、行動輔具）
              ↓
      介入 / 轉介
   ├─ 醫療資源：骨科、復健科、老年醫學科、
   │             家庭醫學科、營養科或其他
   └─ 社區資源：預防延緩失能據點、長照、
                 輔具中心或其他
              ↓
      追蹤（個案是否成功轉介或進行介入）
              ↓
      3–6 個月後再次評估異常項目的變化
```

## SPPB 三項子測試

### 測試一：平衡測試（0–4 分）

平衡測試採**循序施測**，A 未通過即停止，不進行 B 與 C。

#### A. 並腳站立（Side-by-side stand）

| 結果         | 分數                      |
| ------------ | ------------------------- |
| 維持 ≥ 10 秒 | 通過（繼續 B）            |
| 少於 10 秒   | 未通過 → 平衡分 = 0，停止 |

#### B. 半並排站立（Semi-tandem stand）

一腳腳跟靠著另一腳大趾旁站立：

| 結果         | 分數                      |
| ------------ | ------------------------- |
| 維持 ≥ 10 秒 | 通過（繼續 C）            |
| 少於 10 秒   | 未通過 → 平衡分 = 1，停止 |

#### C. 直線站立（Tandem stand）

一腳腳跟緊接另一腳腳趾前端站立，計時：

| 持續時間  | 分數 |
| --------- | ---- |
| ≥ 10 秒   | 4 分 |
| 3–9.99 秒 | 3 分 |
| < 3 秒    | 2 分 |

> **施測注意**：A/B 兩腳皆可，每次嘗試最長計時 10 秒。站立期間若受測者移動腳步、抓扶任何物品或需要施測者協助，即視為未通過。

### 測試二：步行速度測試（0–4 分）

測量受測者以**平常步速**行走指定距離所需的秒數。

- **標準距離：4 公尺**（優先使用）
- **替代距離：3 公尺**（測試空間不足時）

#### 4 公尺計分表

| 時間（秒）  | 分數 |
| ----------- | ---- |
| < 4.82 s    | 4 分 |
| 4.82–6.20 s | 3 分 |
| 6.21–8.70 s | 2 分 |
| > 8.70 s    | 1 分 |
| 無法完成    | 0 分 |

#### 3 公尺計分表

| 時間（秒）  | 分數 |
| ----------- | ---- |
| < 3.62 s    | 4 分 |
| 3.62–4.65 s | 3 分 |
| 4.66–6.52 s | 2 分 |
| > 6.52 s    | 1 分 |
| 無法完成    | 0 分 |

> **施測注意**：受測者站立起始線後，聽到指令後開始行走，計時從第一步踏出開始，走過終點線時停止計時。通常進行兩次取最佳時間。行走時可使用輔具（拐杖、助行器），但需記錄。

### 測試三：椅子起身測試（0–4 分）

受測者**雙手交叉抱胸**，從標準高度椅子（約 43 cm）連續起立坐下 **5 次**，記錄總時間。

| 時間（秒）        | 分數 |
| ----------------- | ---- |
| < 11.19 s         | 4 分 |
| 11.2–13.69 s      | 3 分 |
| 13.7–16.69 s      | 2 分 |
| 16.7–59.9 s       | 1 分 |
| > 60 s 或無法完成 | 0 分 |

> **施測注意**：計時從受測者開始起身時開始，完成第 5 次完全站立後停止。若受測者無法在不用手臂支撐的情況下完成單次起身，即記為「無法完成（0 分）」。

## SPPB 總分解釋

| 總分  | 行動功能狀態 | 臨床建議                             |
| ----- | ------------ | ------------------------------------ |
| 10–12 | 正常         | 維持規律運動，6–12 個月後追蹤        |
| 7–9   | 輕度行動障礙 | 多模態運動介入，3–6 個月後追蹤       |
| 4–6   | 中度行動障礙 | 轉介復健科或老年醫學科，評估跌倒風險 |
| 0–3   | 重度行動障礙 | 立即轉介復健科，評估肌少症及長照需求 |

> SPPB 總分每減少 1 分，失能風險增加約 10–20%；總分 ≤ 9 分者，未來一年內跌倒風險顯著增加（OR ≈ 2.3）。

## 深入評估：潛在疾病與環境因素

SPPB 複評陽性後，應同步評估以下可能造成行動障礙的原因：

### 潛在疾病或問題

| 類別       | 常見原因                             | 說明                                        |
| ---------- | ------------------------------------ | ------------------------------------------- |
| 多重用藥   | ≥ 5 種藥物                           | 抗膽鹼藥、苯二氮平類、降壓藥過量 → 影響平衡 |
| 骨關節疾患 | 骨關節炎、骨質疏鬆、髖/膝關節問題    | 疼痛限制活動，步態改變                      |
| 肌少症     | 肌肉量減少、握力下降                 | 與跌倒、失能密切相關                        |
| 衰弱症     | Frailty（CFS ≥ 5）                   | 體重下降、疲憊、活動力下降                  |
| 疼痛       | 慢性疼痛（下背、膝、踝）             | 直接影響步態與起身能力                      |
| 神經疾患   | 帕金森氏症、中風後遺症、周邊神經病變 | 協調與平衡受損                              |

### 跌倒風險：環境社會支持評估

| 評估面向 | 常見問題                                 |
| -------- | ---------------------------------------- |
| 外在環境 | 地面不平、光線不足、缺乏扶手             |
| 居家環境 | 浴室滑倒、床鋪過高、地毯邊緣             |
| 行動輔具 | 未使用或使用不當（拐杖高度、助行器規格） |

## 介入策略

### 依 SPPB 分數選擇介入

#### SPPB 7–9 分（輕度）：運動介入為主

**多模態運動計畫（每週 3–5 次）：**

| 運動類型 | 建議頻率        | 說明                                               |
| -------- | --------------- | -------------------------------------------------- |
| 阻力訓練 | 每週 2–3 次     | 彈力帶、啞鈴或自重訓練，針對下肢（股四頭肌、臀肌） |
| 平衡訓練 | 每週 3 次       | 單腳站立（30秒×3組）、踏步練習、Otago 運動計畫     |
| 有氧運動 | 每週 ≥ 150 分鐘 | 快走、游泳、腳踏車                                 |
| 太極拳   | 每週 3–5 次     | 改善平衡、降低跌倒風險（有大量 RCT 支持）          |

#### SPPB 4–6 分（中度）：運動 + 醫療評估

除上述運動介入外，建議：

- 轉介**復健科**進行物理治療評估，制定個人化運動處方
- 評估**輔具需求**（拐杖、四腳拐、助行器、踝足矯具）
- 進行**居家環境改造**評估（衛浴扶手、防滑地墊、去除門檻）
- 評估並處理**疼痛**（非類固醇消炎藥、局部注射、物理治療）
- **肌少症篩查**（握力 + 小腿圍 / BIA / DXA）

#### SPPB 0–3 分（重度）：立即轉介 + 長照評估

- 立即轉介**復健科**或**老年醫學科**進行完整評估
- 申請**長期照顧 2.0 服務**（ADL 評估、居家照服員、日間照顧）
- 評估**輔具補助**（長照輔具補助方案）
- 考慮**漸進式復健**（床上運動 → 坐姿運動 → 站立平衡 → 步行訓練）
- 評估**營養狀態**（蛋白質攝取 ≥ 1.2 g/kg/day，補充維生素 D）

## 肌少症（Sarcopenia）與 SPPB 的關係

SPPB 是診斷肌少症的重要功能指標之一：

| 診斷標準          | 指標              | 切截點                         |
| ----------------- | ----------------- | ------------------------------ |
| AWGS 2019（亞洲） | 握力              | 男 < 28 kg，女 < 18 kg         |
|                   | SPPB              | ≤ 9 分（支持嚴重肌少症）       |
|                   | 步行速度          | < 1.0 m/s（4m 測試）           |
|                   | 肌肉量（BIA/DXA） | 男 < 7.0 kg/m²，女 < 5.4 kg/m² |

SPPB ≤ 9 分且握力低下，應安排 BIA 或 DXA 確認肌肉量，以完整診斷肌少症。

## 注意事項

::: warning 安全性優先
施測前應確認受測者近期無骨折、急性關節炎發作或嚴重疼痛。若受測者有跌倒風險，應在施測旁邊提供支撐，但不實際協助完成測試。
:::

::: warning SPPB 不等同於 ICOPE 初評椅子起身
ICOPE 初評椅子起身僅測 5 次起身是否在 12 秒內完成（二元結果）；SPPB 椅子起身測試記錄實際秒數並對應 0–4 分的加權計分，兩者計分方式不同。
:::

::: info 步行距離選擇
4 公尺為國際標準建議距離。空間不足時可使用 3 公尺替代，但需在報告中註明，因兩種距離的切截點不同，不可直接比較。
:::

::: info 台灣 ICOPE 推動現況
國健署 ICOPE 計畫將行動功能（椅子起身）列為六大初評域之一。初評異常者由基層醫療轉介進行 SPPB 複評，並視情況轉介至長照管理中心、社區復健據點或醫療機構。
:::

::: info 免責聲明
本計算器依據 SPPB 量表設計，僅供臨床教育及輔助評估參考，不能取代正式的醫療評估。複評異常者請轉介至適當醫療或長照資源進行後續評估與照護。
:::
