# MNA-SF 營養不良複評工具

**MNA-SF Mini Nutritional Assessment Short Form**

## 概述

**MNA-SF**（Mini Nutritional Assessment Short-Form，迷你營養評估簡式版）是 WHO ICOPE 營養不良初評異常後，進行**第二階段複評**的標準工具。由 Guigoz、Vellas 等人於 1994 年研發（Nestlé 資助），2009 年修訂，共 6 題，完成時間約 3–5 分鐘，最高 14 分。本工具是目前全球使用最廣泛的老年營養篩查工具之一，已於 100 多個國家完成驗證。

## ICOPE 營養不良篩查流程

依據 **112 年國民健康署 ICOPE 長者功能評估**，營養域的初評與複評流程如下：

### 初評（ICOPE 兩題）

| 評估問題                                        | 異常判定   |
| ----------------------------------------------- | ---------- |
| 過去三個月，體重是否在無意中減輕了 3 公斤以上？ | 回答「是」 |
| 過去三個月，是否曾經食慾不振？                  | 回答「是」 |

> 初評任一異常 → 進入 MNA-SF 複評流程

### 複評工具選擇

```
ICOPE 營養初評異常
（非刻意體重減輕 > 3 kg 或食慾下降）
              ↓
        複評：MNA-SF
（食量、體重、活動、疾病、精神、BMI/小腿圍）
              ↓
      深入評估（依複評結果）
   ├─ 潛在疾病或問題：
   │    衰弱症或肌少症、急性或潛在疾病
   │    （包含口腔健康）、壓力事件
   └─ 環境社會支持：
        導致攝取不足的環境社會因素
        （如無人備餐、憂鬱、孤獨感）
              ↓
      介入 / 轉介
   ├─ 醫療資源：老年醫學科、家庭醫學科、
   │             營養科或其他
   └─ 社區資源：社區照護據點、
                 長照或其他
              ↓
      追蹤（個案是否成功轉介或進行介入）
              ↓
      3–6 個月後再次評估異常項目的變化
```

## MNA-SF 六項評估題目

### 題目 A：食量變化（0–2 分）

過去三個月，是否因食慾不振、消化問題、咀嚼或吞嚥困難而減少食量？

| 選項         | 分數 |
| ------------ | ---- |
| 食量嚴重減少 | 0 分 |
| 食量中度減少 | 1 分 |
| 食量沒有改變 | 2 分 |

### 題目 B：體重下降（0–3 分）

過去三個月內體重下降的情況？

| 選項                            | 分數 |
| ------------------------------- | ---- |
| 體重下降大於 3 公斤（6.6 磅）   | 0 分 |
| 不知道                          | 1 分 |
| 體重下降 1–3 公斤（2.2–6.6 磅） | 2 分 |
| 體重沒有下降                    | 3 分 |

### 題目 C：活動能力（0–2 分）

| 選項                           | 分數 |
| ------------------------------ | ---- |
| 需長期臥床或坐輪椅             | 0 分 |
| 可以下床或離開輪椅，但不能外出 | 1 分 |
| 可以外出                       | 2 分 |

### 題目 D：心理創傷或急性疾病（0 或 2 分）

過去三個月內是否有受到心理創傷或患上急性疾病？

| 選項 | 分數 |
| ---- | ---- |
| 有   | 0 分 |
| 沒有 | 2 分 |

> **注意**：此題採二元計分（0 或 2 分），不存在 1 分。

### 題目 E：精神心理問題（0–2 分）

| 選項             | 分數 |
| ---------------- | ---- |
| 嚴重癡呆或抑鬱   | 0 分 |
| 輕度癡呆         | 1 分 |
| 沒有精神心理問題 | 2 分 |

### 題目 F：身體質量指數 / 小腿圍（0–3 分）

**F1（優先使用）：身體質量指數 BMI（公斤/公尺²）**

| BMI 範圍       | 分數 |
| -------------- | ---- |
| BMI < 19       | 0 分 |
| BMI 19 至 < 21 | 1 分 |
| BMI 21 至 < 23 | 2 分 |
| BMI ≥ 23       | 3 分 |

**F2（替代使用）：小腿圍 CC（公分）— 無法取得 BMI 時才使用**

| 小腿圍         | 分數 |
| -------------- | ---- |
| 小腿圍 < 31 cm | 0 分 |
| 小腿圍 ≥ 31 cm | 3 分 |

> 如果無法取得 BMI（如臥床患者），請以小腿圍（CC）代替 F1 填答 F2。如已完成 F1，請勿填答 F2。

## MNA-SF 總分解釋

| 總分         | 營養狀況            | 臨床建議                                                           |
| ------------ | ------------------- | ------------------------------------------------------------------ |
| **12–14 分** | ✅ 正常營養狀況     | 無立即介入需求，建議維持均衡飲食，每 3–6 個月定期追蹤              |
| **8–11 分**  | ⚠️ 有營養不良的風險 | 轉介營養師，進行完整 MNA（29 題）評估及飲食介入計畫                |
| **0–7 分**   | 🔴 營養不良         | 立即轉介老年醫學科或營養師，積極介入（口服營養補充、排查潛在病因） |

> MNA-SF 敏感性約 **96%**、特異性約 **98%**（以完整 MNA 29 題為標準），在社區與住院老年人群中均表現優異。

## 深入評估：潛在疾病與社會環境因素

MNA-SF 複評陽性後，應同步評估以下可能導致營養不良的原因：

### 潛在疾病或問題

| 類別           | 常見原因                            | 說明                                              |
| -------------- | ----------------------------------- | ------------------------------------------------- |
| 衰弱症         | Frailty（CFS ≥ 5）                  | 體重下降、肌力減少、疲憊感三者與營養不良相互強化  |
| 肌少症         | 肌肉量減少、握力下降                | 蛋白質攝取不足為主因之一；應加測握力與小腿圍      |
| 口腔健康       | 齲齒、義齒不合、吞嚥困難            | 直接影響食物攝取量與種類，常被忽略                |
| 急性或慢性疾病 | 感染、腫瘤、慢性腎病、心臟衰竭      | 疾病代謝需求增加或食慾抑制                        |
| 多重用藥       | ≥ 5 種藥物                          | 影響食慾、味覺、腸胃功能（如 metformin、digoxin） |
| 壓力事件       | 喪偶、住院、手術後恢復期            | 急性應激反應導致食慾明顯下降                      |
| 內分泌疾病     | 甲狀腺功能亢進/低下、腎上腺功能不全 | 影響基礎代謝與食慾調節                            |

### 環境與社會支持評估

| 評估面向     | 常見問題                                            |
| ------------ | --------------------------------------------------- |
| 備餐能力     | 獨居、行動不便、認知退化導致無法自行備餐            |
| 經濟因素     | 食物採購困難、食物多樣性不足                        |
| 社交孤立     | 獨食導致食慾下降；孤獨感與憂鬱相互強化              |
| 憂鬱情緒     | 憂鬱為營養不良的重要促因，應同步使用 GDS-15 評估    |
| 認知退化     | 遺忘進食、無法表達飢餓感，應同步使用 BHT / AD8 評估 |
| 宗教飲食限制 | 特殊飲食規定可能導致特定營養素缺乏                  |

## 介入策略

### 依 MNA-SF 分數選擇介入

#### MNA-SF 8–11 分（有風險）：飲食諮詢為主

**飲食介入目標：**

| 介入面向   | 建議                                            |
| ---------- | ----------------------------------------------- |
| 熱量攝取   | 每日 ≥ 30 kcal/kg（衰弱老人可增至 35 kcal/kg）  |
| 蛋白質攝取 | 每日 ≥ 1.2 g/kg；腎功能正常者可至 1.5 g/kg      |
| 餐次設計   | 少量多餐（每日 4–6 次），避免一次大量進食       |
| 食物質地   | 視咀嚼/吞嚥能力調整（軟質、泥狀、流質分級）     |
| 補充營養品 | 考慮口服營養補充品（ONS）每日 200–400 kcal      |
| 維生素 D   | 補充至血清 25(OH)D ≥ 30 ng/mL；每日 800–2000 IU |

#### MNA-SF 0–7 分（營養不良）：積極介入 + 醫療評估

除上述飲食介入外，建議：

- 轉介**老年醫學科**進行完整營養評估（含完整 MNA 29 題）
- 排查並治療**潛在疾病**（感染、腫瘤、甲狀腺疾病等）
- 評估**吞嚥功能**（必要時轉介語言治療師進行吞嚥評估）
- 強化**口腔照護**（轉介牙科處理齲齒、義齒問題）
- 考慮**管灌飲食**評估（若口服攝取嚴重不足且有適應症）
- 進行**肌少症篩查**（握力 + 小腿圍 / BIA / DXA）
- 同步評估**憂鬱**（GDS-15）與**認知功能**（BHT / AD8）

### 社區資源連結

| 需求     | 建議資源                               |
| -------- | -------------------------------------- |
| 餐食服務 | 社區照護據點送餐服務、日間照顧中心供餐 |
| 營養諮詢 | 衛生所或醫院營養科門診、社區藥局       |
| 長照服務 | 長期照顧管理中心（ADLS 評估後申請）    |
| 社工支持 | 轉介社工師處理獨居、經濟困難、備餐支援 |
| 口腔照護 | 社區牙醫師、牙科到宅服務               |

## 肌少症（Sarcopenia）與營養的關係

MNA-SF 評估結果與肌少症診斷密切相關：

| 診斷標準          | 指標              | 切截點                         |
| ----------------- | ----------------- | ------------------------------ |
| AWGS 2019（亞洲） | 握力              | 男 < 28 kg，女 < 18 kg         |
|                   | 步行速度          | < 1.0 m/s（4m 測試）           |
|                   | SPPB              | ≤ 9 分（支持嚴重肌少症）       |
|                   | 肌肉量（BIA/DXA） | 男 < 7.0 kg/m²，女 < 5.4 kg/m² |

MNA-SF 陽性且握力低下者，建議同步轉介進行 **SPPB 行動功能評估**與肌肉量測定，以完整診斷肌少症與衰弱症。

> **衰弱循環（Frailty Cycle）**：營養不良 → 肌力下降 → 行動障礙 → 活動減少 → 食慾進一步下降，三者形成相互強化的惡性循環，需同步評估與介入。

## 注意事項

::: warning 安全性優先
MNA-SF 涉及體重與 BMI 資訊，施測前應確認體重測量的可靠性。無法站立者可採臥床或坐輪椅量測，並在記錄中說明測量方式。
:::

::: warning MNA-SF 不等同於 ICOPE 初評
ICOPE 初評採兩個二元問題（是/否），MNA-SF 採加權多選題計分（最高 14 分），兩者設計目的不同：初評快速判斷是否異常，MNA-SF 則量化嚴重程度並指引介入方向。
:::

::: info F1 與 F2 的選擇
優先使用 F1（BMI）。對於臥床、水腫或無法正確測量體重/身高的患者，改用 F2（小腿圍）替代。兩者不同時填答。小腿圍測量應在非慣用側膝蓋彎曲 90° 時，以軟尺環繞小腿最粗處測量。
:::

::: info 認知障礙患者的施測
輕至中度認知障礙患者仍可完成 MNA-SF，但建議在照顧者陪同下填答，尤其是關於食量、體重變化及活動能力等問題，照顧者的觀察往往更為準確。
:::

::: info 台灣 ICOPE 推動現況
國健署 ICOPE 計畫將營養（非刻意體重減輕與食慾下降）列為六大初評域之一。初評異常者由基層醫療轉介進行 MNA-SF 複評，並視情況轉介至營養科、老年醫學科或社區長照資源。
:::

::: info 免責聲明
本計算器依據 MNA-SF 量表設計，僅供臨床教育及輔助評估參考，不能取代正式的醫療評估。MNA® 為 Nestlé 於 1994 年開發，2009 年修訂，供臨床免費使用。複評異常者請轉介至適當醫療或長照資源進行後續評估與照護。
:::
