# WHO 近距離視力測試卡

**WHO Near Vision Test Card**

## 概述

**WHO 簡單視力圖**（WHO Simple Vision Chart）是 WHO ICOPE 視力障礙初評異常後，進行**第二階段複評**的標準工具。採用 Tumbling E 字（翻轉 E）設計，無需患者識字，適用於所有年齡層及教育背景的受測者。測試距離 **40 cm**，共 4 列符號（6/36、6/18、6/12、6/6），通過標準為第 2 列（6/18）答對 ≥ 3/4。

## ICOPE 視力障礙篩查流程

依據 **112 年國民健康署 ICOPE 長者功能評估**，視力域的初評與複評流程如下：

### 初評（ICOPE 單題）

| 評估方式                                                                                               | 異常判定           |
| ------------------------------------------------------------------------------------------------------ | ------------------ |
| 詢問：「您的眼睛看遠、看近或閱讀是否有困難？」（包括正在治療的眼疾，如糖尿病、高血壓引起的視網膜病變） | 回答「是，有困難」 |

> 初評異常（自述有視力困難）→ 進入 WHO 視力測試卡複評流程

### 複評流程

```
ICOPE 視力初評異常
（自述看遠、看近或閱讀有困難）
              ↓
     複評：WHO 近距離視力測試卡
（4 列 Tumbling E，測試距離 40 cm）
              ↓
  6/18 行答對 ≥ 3/4？
   ├─ 是（通過）→ 近距視力正常
   │              建議 6–12 個月後追蹤
   └─ 否（未通過）→ 視力受損
                    深入評估（潛在疾病或問題）
                    ├─ 白內障
                    ├─ 青光眼（眼內壓異常）
                    ├─ 糖尿病視網膜病變
                    ├─ 老年性黃斑部病變
                    ├─ 屈光不正（近視、遠視、老花）
                    └─ 其他眼疾（角膜、玻璃體病變）
              ↓
      介入 / 轉介
   ├─ 醫療資源：眼科轉介、屈光矯正（眼鏡 / 隱形眼鏡）
   └─ 社區資源：視障輔具、社區視力保健
              ↓
      追蹤（個案是否成功轉介或進行介入）
              ↓
      3–6 個月後再次評估異常項目的變化
```

## 測試卡規格

### 物理尺寸與測試距離

本測試卡採用 WHO 國際標準 Tumbling E 視標，各列符號高度依據 Snellen 視力公式計算，**SVG 單位 = mm，列印時請選「實際大小（100%）」**，不可縮放。

| 列別        | Snellen  | 符號高度（mm） | 臨床意義                   |
| ----------- | -------- | -------------- | -------------------------- |
| 第 1 列     | **6/36** | 9.89 mm        | 嚴重受損才會看不見，暖身列 |
| **第 2 列** | **6/18** | **4.96 mm**    | **WHO 臨界值 — 判讀關鍵**  |
| 第 3 列     | **6/12** | 3.30 mm        | 正常近距視力下限           |
| 第 4 列     | **6/6**  | 1.65 mm        | 優良視力                   |

> 符號高度計算公式：`h = D × tan(5 × VA_ratio × arcmin)`，其中 D = 測試距離（mm）= 400 mm，VA_ratio = Snellen 分母/分子。

### 通過標準

| 評估結果              | 判讀                   | 臨床建議         |
| --------------------- | ---------------------- | ---------------- |
| 6/18 行答對 **≥ 3/4** | ✅ 通過 — 近距視力正常 | 6–12 個月後追蹤  |
| 6/18 行答對 **< 3/4** | ⚠️ 未通過 — 視力受損   | 轉介眼科完整評估 |

### Tumbling E 四個方向

| 符號 | 方向     | 患者回答方式      |
| ---- | -------- | ----------------- |
| E→   | 開口朝右 | 指向右 / 說「右」 |
| E←   | 開口朝左 | 指向左 / 說「左」 |
| E↑   | 開口朝上 | 指向上 / 說「上」 |
| E↓   | 開口朝下 | 指向下 / 說「下」 |

> 不識字或無法言語的患者可用**手指比方向**，或使用箭頭指示板輔助回答。

## 施測標準程序

### 測試前準備

| 準備事項 | 說明                                                  |
| -------- | ----------------------------------------------------- |
| 列印規格 | A4 紙，印表機設定「實際大小」，禁止「縮放至頁面」     |
| 照明     | 充足室內照明（避免逆光或陰暗角落）                    |
| 距離量測 | 使用繩子或尺量出精確 40 cm（建議預先製作 40 cm 標尺） |
| 眼鏡     | **戴鏡者於戴鏡狀態下測試**，記錄矯正（CC）視力        |
| 遮眼     | 遮蓋一眼（勿壓迫眼球），分別測試雙眼                  |

### 施測步驟

1. **說明指示**：向患者說明測試目的，示範 E 字方向回答方式（可先用大符號練習）。
2. **從第 1 列開始**（最大符號，6/36），逐列向下測試。
3. **逐列記錄**每個符號的回答，答對打 ✓，答錯記錄實際回答方向。
4. **第 2 列（6/18）為判讀關鍵**，答對 ≥ 3/4 即通過，可選擇繼續測試第 3、4 列。
5. **換眼重複**同樣程序。
6. 若患者**猶豫超過 5 秒**視為不確定，鼓勵猜測；若仍無法回答，記錄為錯誤。

## 視力測試結果解釋

### 4 級判讀

| 測試結果               | 嚴重程度        | 臨床建議                                 |
| ---------------------- | --------------- | ---------------------------------------- |
| 6/18 未通過（< 3/4）   | 🔴 視力受損     | 立即轉介眼科，進行完整裂隙燈及視網膜檢查 |
| 6/18 通過，6/12 未通過 | ⚠️ 輕度視力障礙 | 建議眼科評估，考慮老花眼鏡               |
| 6/12 通過，6/6 未通過  | 🟡 視力尚可     | 近距視力可接受，定期追蹤                 |
| 6/6 通過               | ✅ 視力正常     | 6–12 個月後定期追蹤                      |

### 視力與 Snellen 分數對照

| Snellen 視力 | logMAR | 臨床描述 | 日常功能影響           |
| ------------ | ------ | -------- | ---------------------- |
| 6/6          | 0.00   | 正常視力 | 無影響                 |
| 6/12         | 0.30   | 輕度下降 | 細字閱讀困難           |
| 6/18         | 0.48   | 中度下降 | 一般閱讀困難           |
| 6/36         | 0.78   | 重度下降 | 計算找零、辨識面孔困難 |
| 6/60         | 1.00   | 嚴重受損 | 日常生活嚴重受限       |

## 深入評估：潛在眼疾

WHO 視力測試卡異常後，應轉介眼科評估以下可能病因：

### 老年常見眼疾

| 疾病                        | 盛行率             | 臨床特徵                 | 介入方向                       |
| --------------------------- | ------------------ | ------------------------ | ------------------------------ |
| **白內障**                  | 60 歲以上 50%↑     | 視力模糊、畏光、色調改變 | 手術（白內障摘除＋人工水晶體） |
| **老年性黃斑部病變（AMD）** | 75 歲以上 10–15%   | 中央視野扭曲、暗點       | 抗 VEGF 注射、低視力輔具       |
| **青光眼**                  | 40 歲以上 2–3%     | 初期無症狀、週邊視野缺損 | 眼壓控制（點眼藥、雷射、手術） |
| **糖尿病視網膜病變**        | 糖尿病患者 30–40%  | 視力波動、飛蚊症         | 血糖控制、雷射光凝固、手術     |
| **老花眼（Presbyopia）**    | 40 歲以上幾乎 100% | 近距閱讀困難             | 老花眼鏡（閱讀眼鏡）           |
| **屈光不正（未矯正）**      | 全年齡             | 遠視或近視模糊           | 配戴眼鏡或隱形眼鏡             |

### 眼科評估項目

| 評估項目                | 目的                                    |
| ----------------------- | --------------------------------------- |
| 最佳矯正視力（BCVA）    | 確認視力下降是否可矯正                  |
| 眼內壓（IOP）           | 青光眼篩查（正常 < 21 mmHg）            |
| 裂隙燈檢查              | 角膜、前房、水晶體評估（白內障分期）    |
| 散瞳眼底檢查            | 視網膜、視神經盤評估（AMD、DR、青光眼） |
| 視野測試                | 青光眼視野缺損                          |
| 光學同調斷層掃描（OCT） | 黃斑部及視網膜神經纖維層影像            |

## 介入策略

### 依複評結果選擇介入

#### 6/18 通過（近距視力正常）：追蹤為主

- 建議 **6–12 個月**後重新評估
- 若患者自述有其他視覺症狀（飛蚊、閃光、周邊視野缺損），仍應轉介眼科
- 強調定期眼科健檢的重要性（含散瞳眼底檢查）

#### 6/18 未通過（視力受損）：轉介眼科

| 疑似病因         | 優先評估項目 | 介入建議                                   |
| ---------------- | ------------ | ------------------------------------------ |
| 白內障           | 裂隙燈分期   | 手術時機評估（視力 ≤ 6/18 且影響日常生活） |
| AMD              | OCT + 眼底   | 抗 VEGF（溼性）；低視力輔具（乾性）        |
| 青光眼           | IOP + 視野   | 點眼藥降壓；必要時手術                     |
| 糖尿病視網膜病變 | 散瞳眼底     | 轉介眼科 + 內分泌科控制血糖                |
| 未矯正屈光不正   | 驗光         | 配鏡（老花眼鏡、多焦鏡片）                 |

### 社區視力保健資源

| 資源類型     | 說明                                    |
| ------------ | --------------------------------------- |
| 社區眼科篩查 | 衛生所定期眼疾篩查活動                  |
| 低視力輔具   | 放大鏡、電子助視器、有聲設備            |
| 長照輔具補助 | 長照 2.0 輔具補助方案                   |
| 視障社會福利 | 視障手冊申請（視力 ≤ 0.1 或視野 ≤ 20°） |

## 注意事項

::: warning 列印規格嚴格要求
本視力卡**必須以原尺寸（100%）列印**，否則符號大小不符標準，所有通過與否判讀均失效。列印時請在印表機設定中選擇「實際大小」或「100%」，切勿選擇「縮放至頁面」、「符合紙張」等選項。建議使用尺量測列印後第 1 列（6/36）符號高度是否等於 9.89 mm 以確認列印正確。
:::

::: warning 必須分別測試雙眼
近距視力測試須分別遮蓋左眼、右眼各測一次，並分別記錄。雙眼同時測試無效，因為好的一眼會補償較差的一眼。施測者應確認遮眼板確實遮蓋，且未壓迫眼球（以免暫時改變視力）。
:::

::: info 與 ICOPE 遠距視力初評的區別
ICOPE 初評為**主觀自述**（是否有困難），本測試為**客觀功能評估**（實際辨識能力）。部分患者初評正常但客觀視力已下降（代償性適應），或初評異常但客觀測試通過（習慣不良閱讀環境）。兩者互補，複評仍以客觀測試為準。
:::

::: info 測試距離的精確性
40 cm 是 WHO 設計此卡的標準距離，偏差會影響判讀：距離縮短（如 35 cm）使符號顯得較大，低估視力障礙；距離拉長（如 50 cm）使符號顯得較小，高估視力障礙。建議預製 40 cm 繩索或標記尺，確保每次測試距離一致。
:::

::: info 台灣 ICOPE 推動現況
國健署 ICOPE 計畫將視力（看遠、看近或閱讀困難）列為六大初評域之一。初評異常者由基層醫療轉介進行 WHO 視力測試卡複評，並視情況轉介至眼科、社區視力保健據點或低視力輔具中心。
:::

::: info 免責聲明
本計算器依據 WHO 簡單視力圖設計，僅供臨床教育及輔助評估參考，不能取代眼科正式視力評估。WHO 視力測試卡由世界衛生組織開發，供基層醫療視力篩查使用。複評異常者請轉介至眼科醫師進行完整評估。
:::
