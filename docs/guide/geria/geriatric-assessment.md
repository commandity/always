# 長者功能評估工具

**Geriatric Function Assessment · ICOPE & AD8 & SPPB & MNA-SF & GDS & Vision**

<GeriatricAssessment />

## WHO ICOPE 長者功能篩查

**Integrated Care for Older People**（WHO）以六大功能域（認知、行動、營養、視力、聽力、憂鬱）進行初評，任一異常即進入複評與個人化照護計畫。本工具整合初評與各域複評，初評異常會自動於對應複評分頁亮起提示。

| 功能域 | 初評異常判定                             | 複評工具（本頁分頁） |
| ------ | ---------------------------------------- | -------------------- |
| 認知   | 三詞回憶 / 時間定向答錯                   | BHT・AD8             |
| 行動   | 椅子起立測試（5 次）未達標               | SPPB                 |
| 營養   | 近三月體重減輕 > 3 kg 或食慾不振         | MNA-SF               |
| 視力   | 自述看遠/看近困難                        | 近距離視力測試       |
| 聽力   | 自述聽力困難                             | （轉介聽力檢查）     |
| 憂鬱   | 情緒低落 / 失去興趣（任一）              | GDS                  |

## BHT・AD8 認知複評

三詞回憶（BHT）＋ **AD8**（8 題知情者問卷）。AD8 **≥ 2 分**提示認知功能減退，建議進一步失智症評估（MMSE/MoCA、影像學）。

## SPPB 行動功能

**Short Physical Performance Battery**：平衡、4 公尺步行速度、5 次椅子起立三項各 0–4 分，總分 0–12。

| SPPB 總分 | 行動功能   | 意義                         |
| --------- | ---------- | ---------------------------- |
| 10–12     | 良好       | 維持運動習慣                 |
| 7–9       | 中等       | 建議肌力與平衡訓練           |
| 4–6       | 較差       | 跌倒與失能風險升高，介入復健 |
| 0–3       | 嚴重受限   | 高度失能風險，完整評估與支持 |

> SPPB ≤ 9 與跌倒、住院及死亡風險上升相關；步行速度 < 0.8 m/s 是衰弱的重要指標。

## MNA-SF 營養複評

**Mini Nutritional Assessment – Short Form**，6 題，總分 0–14。

| MNA-SF 總分 | 營養狀態   | 建議                         |
| ----------- | ---------- | ---------------------------- |
| 12–14       | 正常       | 維持均衡飲食                 |
| 8–11        | 有風險     | 營養衛教、監測體重           |
| 0–7         | 營養不良   | 轉介營養師，評估進食與吞嚥   |

## GDS 老年抑鬱量表

**Geriatric Depression Scale**，提供 GDS-4 / GDS-15 / GDS-30 版本：

| 版本   | 題數 | 切截點  | 用途             |
| ------ | ---- | ------- | ---------------- |
| GDS-4  | 4    | ≥ 2 分  | 急診/門診快速篩查 |
| GDS-15 | 15   | ≥ 5 分  | 標準臨床篩查     |
| GDS-30 | 30   | ≥ 10 分 | 完整評估與追蹤   |

> 超過切截點提示可能有憂鬱，需臨床會談確認；GDS 為篩檢工具，不作診斷用。

## WHO 近距離視力測試卡

以標準符號卡逐列測試近距離視力，判定是否達轉介眼科的視力障礙門檻。測試需在適當照明、慣用閱讀距離（約 40 cm）下進行，配戴平常的閱讀眼鏡。

## 注意事項

::: warning 篩檢不等於診斷
本頁各量表為 ICOPE 架構下的功能篩檢/複評工具；異常結果需由醫師結合完整病史、理學檢查與專科評估確認。
:::

::: info 免責聲明
本計算器僅供臨床輔助與衛教參考，長者照護計畫需由醫療團隊個別化制定。
:::
