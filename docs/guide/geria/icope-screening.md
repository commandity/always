# WHO ICOPE 長者功能篩查

**WHO ICOPE Integrated Care for Older People Screening**

## 概述

**WHO ICOPE**（Integrated Care for Older People，長者整合照護）是世界衛生組織於 2017 年發布的老年健康照護框架，核心目標是透過**系統性功能篩查**，在老年人失能或依賴發生之前，及早辨識功能下降並介入，以維持或提升老年人的內在能力（Intrinsic Capacity）與功能發揮（Functional Ability）。本工具依據 **112 年國民健康署 ICOPE 長者功能評估初評量表**建置，涵蓋六大功能域初評、用藥評估及社會照護評估。

## 評估架構總覽

ICOPE 篩查採「初評 → 複評」兩階段設計：

| 階段     | 說明                                                      |
| -------- | --------------------------------------------------------- |
| **初評** | 以簡短問題/測試快速篩查六大功能域，任一異常即需進一步複評 |
| **複評** | 使用標準化量表或轉介專科確認，制定個人化照護計畫          |
| **附加** | 任一初評異常，同時進行用藥評估及社會照護與支持評估        |

## 六大功能域初評內容

### 1. 認知功能（Cognitive Decline）

| 評估項目       | 測試方式                                                              | 異常判定          |
| -------------- | --------------------------------------------------------------------- | ----------------- |
| 記憶力         | 說出 3 項物品（鉛筆、汽車、書），請長者重複並記住，第三題後再詢問一次 | 無法回憶全部 3 項 |
| 定向力（時間） | 詢問「今天的日期？」（含年月日）                                      | 答錯或不知道      |
| 定向力（地點） | 詢問「您現在在哪裡？」                                                | 答錯或不知道      |

**異常 → 複評工具：BHT 或 AD8 量表**

### 2. 行動功能（Limited Mobility）

| 評估項目     | 測試方式                                                   | 異常判定             |
| ------------ | ---------------------------------------------------------- | -------------------- |
| 椅子起身測試 | 雙手抱胸，從椅子上連續起立坐下五次，計時是否在 12 秒內完成 | 超過 12 秒或無法完成 |

**異常 → 複評工具：SPPB 量表**

### 3. 營養不良（Malnutrition）

| 評估項目 | 問題                                            | 異常判定   |
| -------- | ----------------------------------------------- | ---------- |
| 體重下降 | 過去三個月，體重是否在無意中減輕了 3 公斤以上？ | 回答「是」 |
| 食慾不振 | 過去三個月，是否曾經食慾不振？                  | 回答「是」 |

**異常 → 複評工具：MNA-SF 量表**

### 4. 視力障礙（Visual Impairment）

| 評估項目 | 問題                                 | 異常判定   |
| -------- | ------------------------------------ | ---------- |
| 視力自評 | 您的眼睛看遠、看近或閱讀是否有困難？ | 回答「是」 |

**異常 → 複評：有困難者轉介眼科；無困難但懷疑者進行 WHO 簡單視力量表**

### 5. 聽力障礙（Hearing Loss）

| 評估項目 | 測試方式                                                    | 異常判定     |
| -------- | ----------------------------------------------------------- | ------------ |
| 氣音測試 | 施測者遮住口部，以氣音輕聲說「6、1、9」，長者能否正確跟著唸 | 未能正確重複 |

**異常 → 複評：轉介醫療院所接受正式聽力檢測（純音聽力測試）**

### 6. 憂鬱症狀（Depressive Symptoms）

基於 PHQ-2（Patient Health Questionnaire-2）核心兩題：

| 評估項目 | 問題（過去兩週）               | 異常判定   |
| -------- | ------------------------------ | ---------- |
| 情緒低落 | 您是否常感到厭煩，或沒有希望？ | 回答「是」 |
| 失去興趣 | 是否減少很多的活動和興趣的事？ | 回答「是」 |

**異常 → 複評工具：GDS-15 量表**

## 初評異常後的附加評估

以上六大功能域**任一項異常**，需同時進行「用藥評估」及「社會照護與支持評估」。

### 用藥評估

| 題目                                                        | 異常結果 | 處置                 |
| ----------------------------------------------------------- | -------- | -------------------- |
| 每天使用的藥物是否 10 種（含）以上？（含中藥，中藥算 1 種） | 是       | 收案加入藥事照護計畫 |
| 服用的藥品中是否包含止痛藥、幫助睡眠的藥物？                | 是       | 提供用藥衛教         |
| 是否因服藥而發生平衡感改變、嗜睡、眩暈、低血壓或口乾舌燥？  | 是       | 提供用藥衛教         |

### 社會照護與支持評估

| 題目                                               | 異常結果 | 處置                 |
| -------------------------------------------------- | -------- | -------------------- |
| 在屋內行動、上廁所、穿衣、洗澡、整潔、進食有困難？ | 是       | 轉介長期照顧管理中心 |
| 居住地點困難或財務困難？                           | 是       | 轉介社工師或社福資源 |
| 感覺孤獨或缺乏社交活動？                           | 是       | 轉介社區據點         |

## 複評工具對照表

| 異常域   | 複評工具                    | 說明                                                                                      |
| -------- | --------------------------- | ----------------------------------------------------------------------------------------- |
| 認知功能 | **BHT 或 AD8 量表**         | 任一認知題異常 → 選用 BHT（Blessed 定向記憶集中測驗）或 AD8（8 題家屬版）；AD8 由家屬回答 |
| 行動功能 | **SPPB 量表**               | 簡易體能狀況量表（平衡、步速、起身測試），評估下肢功能與跌倒風險                          |
| 營養不良 | **MNA-SF 量表**             | 迷你營養評估簡式版（6 題），< 12 分為有營養風險                                           |
| 視力障礙 | **眼科轉介 / WHO 視力量表** | 有困難 → 眼科；懷疑但自述無困難 → WHO Tumbling E 視力量表                                 |
| 聽力障礙 | **醫療院所聽力檢測**        | 轉介接受純音聽力測試（35 dB 以下為通過標準）                                              |
| 憂鬱症狀 | **GDS-15 量表**             | 老年抑鬱量表 15 題版；≥ 5 分提示抑鬱，≥ 11 分為重度                                       |

## ICOPE 的核心概念：內在能力（Intrinsic Capacity）

WHO ICOPE 框架以「內在能力」（Intrinsic Capacity, IC）作為老年健康的核心指標，取代傳統以疾病診斷為中心的評估方式。

**內在能力的五大維度：**

| 維度          | 對應 ICOPE 評估域  | 說明                         |
| ------------- | ------------------ | ---------------------------- |
| 認知能力      | 認知功能           | 記憶、定向、執行功能         |
| 心理能力      | 憂鬱症狀           | 情緒調節、心理韌性           |
| 感官能力      | 視力障礙、聽力障礙 | 視覺、聽覺功能               |
| 活力/運動能力 | 行動功能、營養不良 | 肌力、平衡、營養狀態         |
| 社會環境      | 社會照護評估       | 社交支持、居住環境、社區參與 |

## 健康老化的預防策略

### 認知功能維護

- 持續學習新事物（語言、樂器、棋藝）
- 社交互動（參加社區活動、志工服務）
- 控制心血管危險因子（高血壓、糖尿病、高血脂）
- 地中海飲食型態

### 行動能力維持

- 多模態運動：每週 ≥ 150 分鐘中等強度有氧運動
- 阻力訓練：每週 2 次，預防肌少症（Sarcopenia）
- 太極拳、平衡訓練：降低跌倒風險
- 改善居家環境（防滑墊、扶手、充足照明）

### 營養支持

- 每日蛋白質攝取 ≥ 1.2 g/kg（高齡者）
- 維持 BMI 22–26 kg/m²
- 補充維生素 D（預防肌少症與跌倒）
- 定期牙科檢查（咀嚼功能影響進食）

### 視力保健

- 定期眼科檢查（每年一次）
- 控制糖尿病與高血壓（預防視網膜病變）
- 白內障及早手術評估

### 聽力保護

- 避免長時間暴露於噪音環境
- 定期聽力檢查
- 助聽器適當使用（聽損 ≥ 35 dB 時）
- 聽力損失與認知退化的關聯：每增加 10 dB 的聽損，失智症風險增加約 2 倍

### 情緒健康

- 維持規律社交活動
- 睡眠品質管理（慢性失眠的非藥物治療）
- 喪親哀傷支持
- 必要時進行 GDS-15 評估並轉介心理資源

## 篩查流程

```
初評（6 大功能域）
     ↓
任一異常？
   ├─ 否 → 所有項目正常，建議 6–12 個月後再次篩查
   └─ 是 → 同時進行：
            ├─ 用藥評估（3 題）
            ├─ 社會照護評估（3 題）
            └─ 各異常域複評工具評估
                  ├─ 認知異常 → BHT/AD8
                  ├─ 行動異常 → SPPB
                  ├─ 營養異常 → MNA-SF
                  ├─ 視力異常 → 眼科轉介/WHO視力量表
                  ├─ 聽力異常 → 聽力檢測
                  └─ 憂鬱異常 → GDS-15
                         ↓
                  制定個人化照護計畫
                         ↓
                  定期追蹤（每 3–6 個月）
```

## 注意事項

::: warning 初評為篩查工具，非診斷工具
ICOPE 初評僅為快速篩查，陽性結果（異常）需透過複評工具及專科評估確認。單一初評問題不能取代完整的臨床評估與診斷。
:::

::: warning 認知初評限制
初評認知題（記憶力、定向力）僅能辨識可能的認知問題，確診需使用 MMSE、MoCA 或 AD8 等標準化工具，並排除譫妄、藥物影響等可逆因素。
:::

::: info 台灣 ICOPE 推動現況
國民健康署自 112 年起，將 ICOPE 納入基層醫療的老年健康促進計畫，提供 65 歲以上長者免費功能評估。各縣市長照管理中心可協助轉介及後續複評安排。
:::

::: info 免責聲明
本計算器依據 WHO ICOPE 篩查工具及國健署 112 年評估量表建置，僅供臨床衛教參考使用，不能取代正式的醫療評估。初評發現異常者，請轉介至適當的醫療或長照資源進行後續評估與照護。
:::
