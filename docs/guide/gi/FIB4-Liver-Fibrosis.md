# FIB-4 肝纖維化指數 — 非侵入性肝纖維化評估

**FIB-4 Liver Fibrosis Index — Noninvasive Fibrosis Assessment**

## 概述

肝纖維化是慢性肝病進展至肝硬化的核心病理過程。準確評估纖維化分期對於治療決策、監測疾病進展及預測預後至關重要。FIB-4 指數結合四項常規臨床參數，提供簡便、廉價且有效的非侵入性纖維化篩查，已獲 EASL、AASLD 及 WHO 推薦作為第一線評估工具。

## 什麼是 FIB-4 指數？

### 公式與參數

```
FIB-4 = (Age × AST) / (Platelets × √ALT)
```

FIB-4 由 Sterling 等人於 2006 年在 HIV/HCV 共感染族群中開發，隨後在多個獨立族群（NAFLD、HBV、酒精性肝病）中驗證，是目前最廣泛應用的非侵入性肝纖維化指數之一。

**四個輸入參數的臨床意義：**

- **年齡**：肝纖維化風險隨年齡增加；老年患者常有假性升高
- **AST**：肝細胞損傷標記，在肝纖維化及肝硬化中常持續升高
- **ALT**：急性肝細胞損傷的敏感指標；公式中取平方根以降低急性波動的影響
- **血小板**：肝硬化導致脾功能亢進 → 血小板破壞增加；門脈高壓 → 血小板生成下降；是肝硬化最早出現的血液學異常之一

### 診斷閾值

| FIB-4 值      | 風險分層 | 對應纖維化 | 臨床意義                              |
| ------------- | -------- | ---------- | ------------------------------------- |
| **< 1.30**    | 低風險   | F0–F1      | 晚期纖維化 NPV ~90%，可常規追蹤       |
| **1.30–2.67** | 不確定   | F1–F3      | 需進一步評估（彈性成像或血清標記）    |
| **> 2.67**    | 高風險   | F3–F4      | 晚期纖維化 PPV ~80%，建議肝臟專科評估 |

#### 年齡校正閾值（≥65 歲）

老年患者 AST/ALT 基礎值偏高，使用標準閾值會導致高達 50% 的假陽性率。EASL 2023 指引建議 65 歲以上患者採用：

| 年齡校正閾值 | 風險分層 |
| ------------ | -------- |
| FIB-4 < 2.0  | 低風險   |
| 2.0–4.0      | 不確定   |
| > 4.0        | 高風險   |

## 臨床應用場景

### NAFLD / MASLD（最常用情境）

EASL 2023「MASLD 臨床實踐指引」建議以 FIB-4 作為社區及非肝臟專科的第一線纖維化篩查工具，形成「兩步驟評估路徑」：

```
步驟 1：FIB-4（社區 / 初級醫療）
    ├── < 1.30 → 低風險，定期追蹤（1–2 年）
    ├── > 2.67 → 高風險，轉介肝臟專科
    └── 1.30–2.67 → 不確定，進入步驟 2

步驟 2：肝臟彈性成像（FibroScan, LSM）
    或血清標記（ELF, Pro-C3）
```

**NAFLD 患者追蹤頻率建議：**

- FIB-4 < 1.30：每 1–2 年複查，若代謝危險因子惡化提早複查
- FIB-4 1.30–2.67：每年複查，並安排彈性成像評估
- FIB-4 > 2.67：轉介肝臟專科，評估肝硬化併發症

### 慢性 C 型肝炎（HCV）— 原始驗證族群

Sterling 等人原始研究在 HIV/HCV 共感染族群中驗證（n=832）。FIB-4 在 HCV 族群中的準確度：

- AUC 0.765（區分 F3–F4 vs F0–F2）
- 優於 APRI（AUC 0.740）

DAA（直接作用抗病毒藥物）治療後 SVR 患者的纖維化可能逆轉，FIB-4 應在治療後 6–12 個月重新評估。

### 慢性 B 型肝炎（HBV）— 臺灣重要情境

HBV 相關肝纖維化評估的特殊考量：

- **免疫耐受期（HBeAg 陽性，ALT 正常）**：FIB-4 可能低估纖維化程度，建議結合彈性成像
- **免疫活化期（ALT 急性上升）**：FIB-4 因 ALT 急性升高可能假性降低，需待肝炎緩解後複查
- **抗病毒治療後**：長期核苷類似物治療可逆轉纖維化，建議治療 3–5 年後重新評估

### 酒精性肝病 — 需謹慎解讀

酒精性肝炎患者 AST/ALT 比值常 > 2（與粒線體 AST 釋放增加有關），導致 FIB-4 系統性高估纖維化程度。建議：

- 戒酒 4–6 週後重新評估（急性酒精性肝炎緩解後）
- 結合臨床評估及 GGT 趨勢
- 不確定區間患者應以彈性成像確認

## 診斷準確度與局限性

### 準確度數據

| 參數                    | 數值                     |
| ----------------------- | ------------------------ |
| AUC（F3–F4 vs F0–F2）   | 0.74–0.85（視族群而異）  |
| 低風險閾值（< 1.30）NPV | ~90%                     |
| 高風險閾值（> 2.67）PPV | ~80%                     |
| 不確定區間比例          | 約 35–40% 患者落於此區間 |

### 已知局限性

**高估纖維化的情境：**

- 酒精性肝病（AST 系統性升高）
- 非肝臟原因的 AST 升高（溶血、心肌梗塞、橫紋肌溶解症）
- 老年患者（使用標準閾值時）
- 急性肝炎急性期（ALT 急性上升，分母大 → FIB-4 假性降低）

**低估纖維化的情境：**

- HBV 免疫耐受期（ALT 正常）
- 血小板假性升高（脾切除術後、骨髓增生性疾病）

**結構性限制：**

- 無法區分炎症（壞死性炎症）與纖維化
- 對 F2 與 F3 的鑑別能力有限
- 連續監測時的生物學變異（建議以趨勢而非單點解讀）

## 第二層評估工具

當 FIB-4 落在不確定區間（1.30–2.67）時，建議進行第二層評估：

### 瞬態彈性成像（FibroScan / LSM）

| 適應症         | 肝硬度閾值                |
| -------------- | ------------------------- |
| 排除晚期纖維化 | LSM < 8 kPa               |
| 確認晚期纖維化 | LSM > 12–15 kPa（視病因） |
| 篩查肝硬化     | LSM > 20 kPa（高特異性）  |

**注意：**飯後、急性肝炎、右心衰竭、肥胖（BMI > 30）均可影響 LSM 準確度。建議空腹 2–3 小時後執行。

### 血清纖維化標記

- **ELF 評分（Enhanced Liver Fibrosis）**：含 HA、PIIINP、TIMP-1；EASL 認可，對 F3–F4 準確度優於 FIB-4
- **Pro-C3**：新生膠原蛋白合成標記，對活動性纖維化較敏感
- **APRI**（AST to Platelet Ratio Index）：免費、廣泛可用，但準確度略低於 FIB-4

### 影像學評估

- **MR-elastography（MRE）**：目前非侵入性纖維化評估的最高準確度方法（AUC > 0.90），但費用高、可及性低
- **超音波**：肝硬化形態學變化（肝葉比例改變、表面結節、脾腫大）提示晚期纖維化，但對早期纖維化不敏感

## 肝纖維化分期快速參考

### METAVIR 分期系統

| 分期   | 纖維化程度 | 描述                       |
| ------ | ---------- | -------------------------- |
| **F0** | 無纖維化   | 正常肝臟結構               |
| **F1** | 輕度       | 門脈周圍纖維化，無纖維間隔 |
| **F2** | 中度       | 少數纖維間隔形成           |
| **F3** | 重度       | 多數纖維間隔，無肝硬化     |
| **F4** | 肝硬化     | 假小葉形成                 |

FIB-4 主要目標是區分 **F0–F2**（無或輕中度纖維化）與 **F3–F4**（晚期纖維化 / 肝硬化）。

## 肝硬化監測建議

確診晚期纖維化或肝硬化的患者應進行：

| 監測項目               | 頻率                | 目的                                |
| ---------------------- | ------------------- | ----------------------------------- |
| 腹部超音波 + AFP       | 每 6 個月           | HCC 篩查（肝硬化患者年發生率 1–8%） |
| 胃鏡                   | 確診時、後每 1–3 年 | 食道靜脈曲張評估                    |
| CBC + 肝功能           | 每 3–6 個月         | 肝功能儲備監測                      |
| Child-Pugh / MELD 評分 | 依臨床需要          | 肝功能儲備及移植優先序評估          |

## 不同病因的 FIB-4 閾值比較

| 病因            | 低風險閾值     | 高風險閾值 | 特殊考量             |
| --------------- | -------------- | ---------- | -------------------- |
| NAFLD（一般）   | < 1.30         | > 2.67     | EASL 2023 標準       |
| NAFLD（≥65 歲） | < 2.0          | > 4.0      | 年齡校正             |
| HCV             | < 1.45         | > 3.25     | 部分指引略高於 NAFLD |
| HBV             | < 1.30         | > 2.67     | 免疫活化期需謹慎     |
| 酒精性肝病      | 不建議單獨使用 | —          | 系統性高估           |

## 注意事項

::: warning FIB-4 不適用的情境
急性肝炎（ALT > 5× ULN）、溶血性疾病（假性 AST 升高）、骨髓增生性疾病（血小板假性升高/降低）、脾切除術後患者，FIB-4 數值可能嚴重失真。這些情境應直接以影像學或切片評估，不應依賴 FIB-4 結果。
:::

::: warning 酒精性肝病的特殊提示
酒精性肝炎患者 AST 因粒線體損傷系統性升高（AST/ALT > 2），導致 FIB-4 公式分子項偏高，纖維化程度可能被高估 1–2 個分期。建議在戒酒後 4–6 週轉氨酶穩定後重新計算，或直接採用彈性成像評估。
:::

::: info 連續監測的意義
FIB-4 的臨床價值在於連續追蹤趨勢，而非單一時間點的絕對值。有效治療（如 DAA 清除 HCV、抗病毒治療控制 HBV、體重減輕改善 NAFLD）可使 FIB-4 在數月至數年內顯著下降，反映肝纖維化的逆轉。建議每次複查時與前次結果比較，並記錄趨勢方向。
:::

::: info 與其他非侵入性工具的整合
FIB-4 應作為第一步篩查，而非獨立診斷工具。在 EASL「兩步驟路徑」中，FIB-4 不確定區間的患者應進一步以 LSM（FibroScan）確認分期，這樣的整合策略可使約 90% 的患者免於肝臟切片，同時保持足夠的診斷準確度。
:::

::: info 免責聲明
本頁工具依據 EASL 2023、AASLD 2023 指引及 Sterling et al. Hepatology 2006 原始文獻設計，僅供臨床教育及輔助評估參考，**不能取代肝臟專科醫師的正式評估**。纖維化分期最終診斷需結合完整臨床病史、理學檢查、影像學及必要時的肝臟組織學評估。
:::
