# Centor Score 鏈球菌咽喉炎風險評估

**Modified Centor (McIsaac) Score · A 群鏈球菌咽喉炎風險**

<CentorCalculator />

## Centor / McIsaac 評分

**Centor Score**（Centor 1981）以 4 項臨床指標估計 A 群鏈球菌（GAS）咽喉炎的機率；**McIsaac 校正**（1998）加入年齡調整，擴大至兒童與成人。

| 項目     | 內容                          | 分數 |
| -------- | ----------------------------- | ---- |
| 扁桃體滲出物或腫脹 | 白色/黃色膿性分泌物或明顯腫脹 | +1 |
| 前頸淋巴結腫大且壓痛 | 頸前淋巴結腫大且按壓疼痛     | +1 |
| 發燒 > 38°C | 就診時或病史發燒               | +1 |
| 無咳嗽   | 無咳嗽（咳嗽提示病毒病因）     | +1 |
| 年齡 3–14 歲 | McIsaac 校正                  | +1 |
| 年齡 15–44 歲 | McIsaac 校正                  | 0  |
| 年齡 ≥ 45 歲 | McIsaac 校正                  | −1 |

### 分數意義與處置

| 總分 | 風險等級   | GAS 機率 | 意義與建議處置                     |
| ---- | ---------- | -------- | ---------------------------------- |
| ≤ 0  | 極低風險   | 1–2.5%   | 不建議培養或抗生素治療             |
| 1    | 低風險     | 5–10%    | 以症狀治療為主，不建議常規培養     |
| 2    | 中等風險   | 11–17%   | 考慮快速抗原檢測（RADT）或咽喉培養 |
| 3    | 中高風險   | 28–35%   | 建議 RADT 或培養，陽性者給予抗生素 |
| ≥ 4  | 高風險     | 51–53%   | 強烈考慮抗生素，可不等待培養結果   |

> **≤ 1 分**者 GAS 機率低，不需檢測或抗生素；**2–3 分**建議先以 RADT 或咽喉培養確認再決定用藥（避免抗生素濫用）；**≥ 4 分**機率過半，可考慮經驗性治療。Centor 的目的在於**減少不必要的抗生素處方**——多數急性咽炎為病毒性，即使 GAS 感染多為自限性，治療主要在於預防化膿性與非化膿性併發症。

## 注意事項

::: warning 風險分層非確診
Centor Score 估計 GAS 咽喉炎的**驗前機率**，不等於確診；中間分數應以 RADT 或培養佐證。
:::

::: info 免責聲明
本工具僅供臨床輔助與教學參考，抗生素處方應由醫師依完整臨床評估與在地抗藥性資料判定。
:::
