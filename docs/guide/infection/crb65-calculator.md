# CRB-65 肺炎嚴重度評估

**CRB-65 Community-Acquired Pneumonia Severity Score**

<Crb65Calculator />

## CRB-65 評分

**CRB-65**（Lim 2003）為社區型肺炎（CAP）的床邊嚴重度分層工具，**不需檢驗數據**，由 4 項臨床指標組成，每項 +1，總分 0–4。

| 項目                    | 判定標準                              |
| ----------------------- | ------------------------------------- |
| C — Confusion（意識混亂） | 新發生的定向障礙（時間、地點、人物）  |
| R — Respiratory（呼吸）   | 呼吸速率 ≥ 30 次 / 分鐘               |
| B — Blood pressure（血壓）| 收縮壓 < 90 mmHg 或舒張壓 ≤ 60 mmHg  |
| 65 — Age（年齡）          | 年齡 ≥ 65 歲                          |

### 分數意義與處置

| 總分 | 風險等級   | 30 天死亡率 | 意義與建議處置                       |
| ---- | ---------- | ----------- | ------------------------------------ |
| 0    | 低風險     | < 1%        | 可考慮門診治療，密切追蹤             |
| 1    | 低至中風險 | 1–3%        | 考慮短期住院觀察或密切門診追蹤       |
| 2    | 中至高風險 | 3–15%       | 建議住院治療與監測                   |
| 3–4  | 高風險     | > 15%       | 建議緊急住院，評估加護病房照護       |

> **CRB-65 ≥ 1 分**即建議考慮住院；**≥ 2 分**死亡率明顯上升，應積極住院處置。CRB-65 因不需血液尿素氮（BUN），特別適合門急診與社區端快速分層；院內若能取得 BUN，可改用資訊更完整的 CURB-65。此工具評估「死亡風險」，不能取代整體臨床判斷與病因處理。

## 注意事項

::: warning 分層工具非診斷
CRB-65 用於估計肺炎嚴重度與死亡風險以輔助住院決策，不能取代對病因、共病與社會因素的整體評估。
:::

::: info 免責聲明
本工具僅供臨床輔助與教學參考，最終處置應由醫師依完整臨床情境判定。
:::
