# CRB-65 肺炎嚴重度評估

**CRB-65 Pneumonia Severity Assessment**

## 概述

**CRB-65** 由 Lim 等人於 2003 年發表於《胸腔》期刊（_Thorax_），是從 CURB-65 衍生的簡化版社區型肺炎（CAP）嚴重度評估工具。CRB-65 去除了需要血液檢查的尿素氮（BUN/Urea）項目，僅保留 4 項臨床參數，特別適合**基層診所、急診快速分診**，以及無法立即進行血液檢查的場合。

## 評分說明

CRB-65 共 4 項指標，每項符合計 **1 分**，總分 0–4 分：

| 指標           | 英文             | 定義                                 | 分數 |
| -------------- | ---------------- | ------------------------------------ | ---- |
| **C** 意識混亂 | Confusion        | 新發生的定向障礙（時間、地點、人物） | +1   |
| **R** 呼吸急促 | Respiratory Rate | 靜息呼吸頻率 ≥ 30 次/分              | +1   |
| **B** 低血壓   | Blood Pressure   | 收縮壓 < 90 mmHg 或舒張壓 ≤ 60 mmHg  | +1   |
| **65** 年齡    | Age              | ≥ 65 歲                              | +1   |

## 臨床切截點

| CRB-65 分數 | 風險等級   | 預估 30 天死亡率 | 建議處置               |
| ----------- | ---------- | ---------------- | ---------------------- |
| 0           | 低風險     | < 1%             | 門診口服抗生素治療     |
| 1           | 低至中風險 | 1–3%             | 考慮住院或密切門診追蹤 |
| 2           | 中至高風險 | 3–15%            | 建議住院治療           |
| 3–4         | 高風險     | > 15%            | 緊急住院，評估 ICU     |

> **關鍵切截值：CRB-65 ≥ 2** 建議住院，**≥ 3** 需考慮 ICU 或積極支持治療。

## 各項指標的臨床意義

### C — 意識混亂（Confusion）

新發生的定向力障礙是肺炎嚴重度的重要指標，反映腦部灌流不足或全身性低氧狀態。評估時應排除慢性認知症造成的基礎混亂，重點在於**急性發作**的狀態改變。

### R — 呼吸急促（Respiratory Rate ≥ 30/min）

靜息狀態下呼吸頻率 ≥ 30 次/分，提示肺部通氣嚴重受損，為需要住院的強力指標。臨床上常被低估，需在患者安靜狀態下計數 1 分鐘。

### B — 低血壓（Blood Pressure）

收縮壓 < 90 mmHg 或舒張壓 ≤ 60 mmHg，反映循環系統受損，可能提示敗血性休克前期。任一條件符合即計分。

### 65 — 年齡（Age ≥ 65 歲）

年齡是肺炎死亡率最強的獨立預測因子之一。65 歲以上患者的免疫反應減弱、合併症多，對感染的代償能力顯著下降。

## CRB-65 vs CURB-65

|                | CRB-65                 | CURB-65                                    |
| -------------- | ---------------------- | ------------------------------------------ |
| **項目數**     | 4 項                   | 5 項                                       |
| **額外項目**   | —                      | 血液尿素氮（BUN > 7 mmol/L 或 > 19 mg/dL） |
| **適用場合**   | 基層診所、急診快速分診 | 住院評估、有完整血液檢查結果               |
| **血液檢查**   | 不需要                 | 需要                                       |
| **預測準確度** | 略低                   | 略高                                       |
| **操作便利性** | 高                     | 中                                         |

> 在無法即時取得血液檢查結果的情況下，CRB-65 是優先選擇。若有完整實驗室數據，CURB-65 或 PSI（Pneumonia Severity Index）提供更精確的風險分層。

## 社區型肺炎的常見病原體

| 分類       | 病原體                               | 特徵                         |
| ---------- | ------------------------------------ | ---------------------------- |
| 典型肺炎   | _Streptococcus pneumoniae_（最常見） | 急性發病、高燒、膿痰         |
| 典型肺炎   | _Haemophilus influenzae_             | 慢性阻塞性肺病患者常見       |
| 非典型肺炎 | _Mycoplasma pneumoniae_              | 年輕患者、乾咳、緩慢進展     |
| 非典型肺炎 | _Chlamydia pneumoniae_               | 老年患者，常與其他病原體混合 |
| 非典型肺炎 | _Legionella pneumophila_             | 重症，低鈉血症，空調水塔暴露 |
| 病毒性     | SARS-CoV-2、流感病毒                 | 可引起重症肺炎，需病毒學確認 |

## 治療建議

### 門診治療（CRB-65 = 0）

- 健康成人（無合併症）：Amoxicillin 口服 5–7 天
- 有合併症或近期使用抗生素：Amoxicillin-Clavulanate 或 Respiratory Fluoroquinolone
- 懷疑非典型病原體：Macrolide（Azithromycin、Clarithromycin）

### 住院治療（CRB-65 ≥ 2）

- 靜脈 Beta-lactam（Ampicillin-sulbactam 或 Ceftriaxone）+ Macrolide
- 或單用 Respiratory Fluoroquinolone（Levofloxacin、Moxifloxacin）
- 懷疑 MRSA 或 _Pseudomonas_：需調整覆蓋範圍

### ICU 治療（CRB-65 ≥ 3）

- Beta-lactam + Macrolide **或** Beta-lactam + Respiratory Fluoroquinolone
- 評估機械通氣或高流量氧療需求
- 考慮早期使用靜脈皮質類固醇（嚴重 CAP）

## 注意事項

::: warning CRB-65 不能取代臨床整體判斷
CRB-65 是輔助決策工具，預測的是統計風險，不能機械性地套用於個別患者。低分患者（0–1 分）若有其他危險因素（免疫低下、多重合併症、社會支持不足、無法口服藥物），仍應考慮住院。高分患者若整體狀況良好，也不一定需要 ICU。
:::

::: warning 意識混亂的評估需排除慢性因素
「C」（意識混亂）的判斷需確認為**急性發作**的定向力障礙，而非失智症或慢性腦部疾病造成的基礎認知障礙。建議詢問家屬或照護者，確認患者目前狀態是否為近期改變。
:::

::: warning 免疫低下患者需提高警覺
CRB-65 是針對一般族群設計的工具，對免疫低下患者（HIV、化療後、長期使用免疫抑制劑、器官移植）的風險可能低估。此族群即使 CRB-65 分數低，也應積極評估並降低住院門檻。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不能取代完整病史、身體檢查及影像學評估。社區型肺炎的診斷及治療決策應由合格醫療人員根據患者個別狀況綜合判斷。
:::
