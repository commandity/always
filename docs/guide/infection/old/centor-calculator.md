# Centor Score 鏈球菌咽喉炎風險評估

**Centor Score Streptococcal Pharyngitis Risk Assessment**

## 概述

**Centor Score** 由 Robert Centor 於 1981 年首次發表，是評估**急性咽喉炎患者感染 A 群 β 溶血性鏈球菌（GAS，_Streptococcus pyogenes_）機率**的臨床決策工具。1998 年，Warren McIsaac 在原版 4 項指標基礎上加入**年齡修正**，形成目前廣泛使用的修正版（Modified Centor / McIsaac Score），總分範圍 −1 至 +5 分。

## 評分說明

### 臨床指標（各 +1 分）

| 指標                 | 英文                                     | 定義                             |
| -------------------- | ---------------------------------------- | -------------------------------- |
| 扁桃體滲出物或腫脹   | Tonsillar Exudate or Swelling            | 扁桃體可見膿性分泌物，或明顯腫脹 |
| 前頸淋巴結腫大且壓痛 | Tender Anterior Cervical Lymphadenopathy | 頸前淋巴結觸痛腫大               |
| 發燒（> 38°C）       | Temperature > 38°C                       | 就診時體溫或病史記錄             |
| 無咳嗽               | Absence of Cough                         | 咳嗽的存在提示病毒性病因         |

### 年齡修正（McIsaac 1998）

| 年齡     | 修正分數 |
| -------- | -------- |
| 3–14 歲  | +1       |
| 15–44 歲 | 0        |
| ≥ 45 歲  | −1       |

> 「無咳嗽」是唯一的**反向指標**——咳嗽的存在降低了細菌性咽喉炎的可能性，故無咳嗽才計分。

## 臨床切截點

| Centor 分數 | GAS 感染機率 | 建議處置                         |
| ----------- | ------------ | -------------------------------- |
| ≤ 0         | 1–2.5%       | 不建議培養或抗生素，症狀治療     |
| 1           | 5–10%        | 不建議常規培養，症狀治療         |
| 2           | 11–17%       | 考慮快速抗原檢測（RADT）         |
| 3           | 28–35%       | 建議 RADT 或培養，陽性給予抗生素 |
| 4–5         | 51–53%       | 強烈考慮抗生素，可不等待培養     |

> **關鍵切截值：Centor ≥ 2** 建議進行快速抗原檢測（RADT）；**≥ 4** 時 GAS 機率超過 50%，可考慮直接給予抗生素治療。

## 各評估指標的臨床意義

### 扁桃體滲出物

扁桃體表面的白色或黃色膿性分泌物是細菌感染的重要徵象，但並非絕對特異——病毒性單核球增多症（EBV）也可出現類似表現。評估時需在良好光線及壓舌板輔助下仔細檢查。

### 前頸淋巴結腫大且壓痛

頸前淋巴結（胸鎖乳突肌前方）的觸痛腫大反映區域性淋巴引流反應，是 GAS 咽喉炎的重要徵象。需注意**壓痛**是必要條件，單純腫大（無壓痛）特異性較低。

### 發燒

GAS 咽喉炎通常急性發病、高燒（常超過 38.5°C）。若發燒不明顯或病程緩慢，更傾向病毒性病因。

### 無咳嗽

咳嗽提示病毒性上呼吸道感染，而非細菌性咽喉炎。若患者有明顯咳嗽，GAS 感染機率大幅降低。這是評分中唯一「缺乏某症狀」才計分的項目。

### 年齡的重要性

GAS 咽喉炎在 **5–15 歲**兒童中最常見；成人（尤其 45 歲以上）罹患細菌性咽喉炎的機率顯著較低，病毒性病因更常見。年齡修正有效提升了成人族群的評分特異性。

## 診斷工具比較

### 快速抗原檢測（RADT）

- 靈敏度：70–90%
- 特異度：95–99%
- 報告時間：5–10 分鐘
- **適用情境**：Centor 2–3 分，用於確認或排除 GAS

### 咽喉培養

- 靈敏度：90–95%（金標準）
- 特異度：99%
- 報告時間：24–48 小時
- **適用情境**：RADT 陰性但臨床高度懷疑，或用於確認診斷

### 使用建議

| Centor 分數 | 推薦檢測策略                       |
| ----------- | ---------------------------------- |
| ≤ 1         | 不建議 RADT 或培養                 |
| 2–3         | RADT（陰性時考慮培養）             |
| ≥ 4         | 可直接治療，或 RADT 確認後立即治療 |

## GAS 咽喉炎的治療

### 首選抗生素

**Penicillin** 仍是 GAS 咽喉炎的首選治療，GAS 對 Penicillin 至今**無抗藥性紀錄**：

| 藥物                            | 劑量                     | 療程  |
| ------------------------------- | ------------------------ | ----- |
| Penicillin V（口服）            | 成人 500 mg bid          | 10 天 |
| Amoxicillin（口服）             | 成人 500 mg tid 或 1g qd | 10 天 |
| Benzathine Penicillin G（肌注） | 120 萬單位（成人）       | 單次  |

> 療程 **10 天**是清除 GAS 和預防風濕性心臟病的最低標準，不可提前停藥。

### 青黴素過敏患者

| 藥物                   | 說明                                      |
| ---------------------- | ----------------------------------------- |
| Azithromycin（Z-pack） | 5 天療程，但 GAS 抗藥率上升中（約 5–10%） |
| Clarithromycin         | Macrolide 類，療程 10 天                  |
| Clindamycin            | 嚴重過敏者，療程 10 天                    |
| Cephalosporins         | 非嚴重過敏者可用（約 1% 交叉過敏率）      |

### 治療目的

1. **縮短病程**：抗生素可縮短症狀約 1–3 天
2. **防止傳播**：用藥 24 小時後傳染性大幅降低
3. **預防化膿性併發症**：扁桃體周圍膿瘍、咽後膿瘍
4. **預防非化膿性併發症**：急性風濕熱、風濕性心臟病（Penicillin 療程 10 天可幾乎完全預防）

## GAS 咽喉炎的並發症

### 化膿性並發症

| 併發症         | 說明                   |
| -------------- | ---------------------- |
| 扁桃體周圍膿瘍 | 最常見，需手術引流     |
| 咽後膿瘍       | 深頸部感染，需緊急處置 |
| 中耳炎         | 尤其兒童常見           |
| 鼻竇炎         | 繼發感染               |

### 非化膿性並發症

| 併發症                         | 說明                                                |
| ------------------------------ | --------------------------------------------------- |
| 急性風濕熱                     | 抗生素治療可預防，開發中國家仍常見                  |
| 風濕性心臟病                   | 反覆 GAS 感染導致，Penicillin 預防最重要            |
| 鏈球菌感染後腎絲球腎炎（PSGN） | 特定血清型，抗生素治療無法完全預防                  |
| PANDAS                         | 兒童強迫症與抽動症，與 GAS 感染相關（機制仍有爭議） |

## 注意事項

::: warning Centor Score 不適用於 3 歲以下兒童
GAS 咽喉炎在 3 歲以下幼兒極為少見，此年齡層的咽喉炎幾乎全為病毒性。Centor / McIsaac 評分系統僅適用於 **3 歲以上**患者。
:::

::: warning 排除傳染性單核球增多症
傳染性單核球增多症（EBV 感染）的臨床表現可與 GAS 咽喉炎高度重疊（發燒、扁桃體滲出物、頸淋巴結腫大），Centor 分數可能偏高。給予 Amoxicillin 可引起特徵性全身性皮疹。評估時需考慮 EBV 的可能性，必要時進行 Monospot 或 EBV 抗體檢測。
:::

::: warning 抗生素過度使用的問題
全球抗生素抗藥性日趨嚴重。Centor ≤ 1 分患者幾乎不需要抗生素——症狀治療（止痛、退燒、補充水分）即可有效緩解病毒性咽喉炎。過度使用抗生素不僅無益，還會增加抗藥菌的風險。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不能取代完整病史及身體檢查。咽喉炎的診斷及治療決策應由合格醫療人員根據患者個別狀況綜合判斷。快速抗原檢測或咽喉培養仍是確認 GAS 感染的重要工具。
:::
