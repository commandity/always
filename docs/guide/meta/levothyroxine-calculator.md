# Levothyroxine 劑量計算器

**Levothyroxine Dosage Calculator**

## 概述

**Levothyroxine**（合成 L-甲狀腺素，LT4）是**原發性甲狀腺功能低下症**的標準替代治療藥物。成人體重基礎劑量公式為 **1.6 mcg/kg/day**（全量替代），但實際起始劑量需根據臨床情境、年齡、心臟功能及甲狀腺殘存功能調整。本計算器提供四種臨床情境下的體重基礎劑量建議，並自動推薦最接近的市售錠劑規格。

## 劑量係數說明

| 臨床情境          | 劑量係數（mcg/kg/day） | 適用對象                                           |
| ----------------- | ---------------------- | -------------------------------------------------- |
| 完全替代          | 1.6                    | 全甲狀腺切除術後、嚴重原發性甲狀腺功能低下         |
| 部分替代          | 1.0–1.2                | 橋本氏甲狀腺炎、輕中度原發性甲低（甲狀腺殘存功能） |
| 亞臨床甲低        | 0.5–0.75               | TSH 輕度升高（< 10 mIU/L）而 FT4 正常              |
| 老年 / 高心臟風險 | 0.8–1.0（低劑量起始）  | ≥ 65 歲或合併心臟疾病，從 12.5–25 mcg/day 起始     |

> **計算公式**：起始劑量（mcg/day）= 體重（kg）× 劑量係數（mcg/kg/day）
> 計算結果應調整至最接近的市售錠劑規格（誤差 ≤ 15 mcg 視為可接受）。

## 市售 Levothyroxine 錠劑規格

市售 Levothyroxine 有以下標準強度（mcg）：

**25 · 50 · 75 · 88 · 100 · 112 · 125 · 137 · 150 · 175 · 200 · 300**

> 若計算劑量介於標準規格之間，可採用以下策略：
>
> - **兩錠組合**：如每日 112 mcg = 75 mcg + 37.5 mcg（半錠 75 mcg）
> - **錠劑切割**：部分錠劑可對半切割
> - **每週變化劑量**：如週一至五 100 mcg，週六日 125 mcg（週平均 ≈ 107 mcg/day）
> - Levothyroxine 半衰期約 **7 天**，每日小幅波動對整體血中濃度影響輕微

## 臨床情境詳細說明

### 完全替代（1.6 mcg/kg/day）

適用於**全甲狀腺切除術後**或嚴重原發性甲狀腺功能低下（TSH 顯著升高、無甲狀腺功能殘留）。此為標準成人全劑量替代。

| 體重（kg） | 計算劑量（mcg/day） | 建議錠劑（mcg） |
| ---------- | ------------------- | --------------- |
| 50         | 80                  | 88              |
| 60         | 96                  | 100             |
| 70         | 112                 | 112             |
| 80         | 128                 | 125 或 137      |
| 90         | 144                 | 137 或 150      |
| 100        | 160                 | 150 或 175      |

### 部分替代（1.0–1.2 mcg/kg/day）

適用於甲狀腺仍有部分功能殘存的患者，如**橋本氏甲狀腺炎早中期**。起始劑量可從 50–75 mcg/day 開始，依 TSH 反應漸進調整。

### 亞臨床甲狀腺功能低下（0.5–0.75 mcg/kg/day）

**TSH 升高但 FT4 正常**。是否治療需個別化評估：

| 是否治療     | 建議                               |
| ------------ | ---------------------------------- |
| **建議治療** | TSH > 10 mIU/L、有症狀、妊娠或備孕 |
| **考慮治療** | TSH 5–10 mIU/L 且有症狀            |
| **觀察追蹤** | TSH < 10 mIU/L 且無症狀、老年患者  |

### 老年患者 / 高心臟風險（起始 12.5–25 mcg/day）

老年患者或合併冠狀動脈疾病、心律不整者，**甲狀腺素過量可誘發心房顫動、心絞痛加重**，需「低劑量起始、緩慢增加」：

- 起始：12.5–25 mcg/day
- 每 4–8 週增加 12.5–25 mcg
- 目標 TSH：1–4 mIU/L（不必追求與年輕人相同）

## 服藥指示

### 服藥時機

| 方式                 | 說明                                                |
| -------------------- | --------------------------------------------------- |
| **早晨空腹（首選）** | 起床後立即服藥，以清水送服，30–60 分鐘後再進食      |
| **睡前服藥（替代）** | 需在最後一餐後 3–4 小時服用，適合無法早晨空腹服藥者 |

### 藥物交互作用

以下物質會**減少 Levothyroxine 吸收**，需與 LT4 間隔至少 **4 小時**：

| 物質                        | 間隔時間  |
| --------------------------- | --------- |
| 鈣劑（含碳酸鈣、檸檬酸鈣）  | ≥ 4 小時  |
| 鐵劑（含綜合維他命中的鐵）  | ≥ 4 小時  |
| 制酸劑（含鋁、鎂）          | ≥ 4 小時  |
| Cholestyramine / Colestipol | ≥ 4 小時  |
| Sucralfate                  | ≥ 4 小時  |
| 咖啡（包括含咖啡因飲料）    | ≥ 60 分鐘 |

### 漏服處理

- **漏服一天**：次日服雙倍劑量（因半衰期長，偶發漏服影響輕微）
- **連續漏服**：盡快恢復正常服藥，不建議累積補服

## TSH 監測與劑量調整

### 初始治療追蹤

| 時程              | 建議                 |
| ----------------- | -------------------- |
| 開始治療後 4–8 週 | 第一次 TSH 追蹤      |
| TSH 達目標後      | 每 6–12 個月追蹤     |
| 劑量調整幅度      | 每次增減 12.5–25 mcg |

### 目標 TSH 範圍

| 族群                 | 目標 TSH（mIU/L）   |
| -------------------- | ------------------- |
| 一般成人             | 0.5–2.5             |
| 老年患者（> 65 歲）  | 1–4                 |
| 妊娠（第一孕期）     | < 2.5               |
| 妊娠（第二、三孕期） | < 3.0               |
| 甲狀腺癌術後（高危） | 0.1–0.5（輕度抑制） |
| 甲狀腺癌術後（低危） | 0.5–2.0             |

## 需警覺的情況

### 劑量需求異常高

若患者所需劑量顯著超過體重計算值（> 200 mcg/day 且體重 < 80 kg），需考慮：

| 可能原因             | 評估方式                       |
| -------------------- | ------------------------------ | --------------------- |
| **吸收不良**         | 乳糜瀉、萎縮性胃炎、短腸症候群 | 空腹血中 LT4 濃度測定 |
| **服藥順從性不佳**   | 不規則服藥                     | 加強衛教、考慮監測    |
| **藥物交互作用**     | 上述干擾吸收的藥物             | 詳細藥物史            |
| **假性甲狀腺素抵抗** | 少見                           | 轉介內分泌科          |

### 甲狀腺素過量症狀

| 症狀               | 嚴重程度   |
| ------------------ | ---------- |
| 心跳加速、心悸     | 早期警示   |
| 手抖、焦慮、失眠   | 常見       |
| 體重減輕、過度流汗 | 常見       |
| 心房顫動           | 需立即處理 |
| 骨質疏鬆（長期）   | 慢性風險   |

## 注意事項

::: warning 靜脈注射劑量需減少 30%
若需改用靜脈注射 Levothyroxine（如術前、嚴重吸收不良），靜脈劑量應比口服劑量**減少約 30%**，因靜脈給藥的生體可用率（bioavailability）接近 100%，遠高於口服的 70–80%。
:::

::: warning 品牌藥與學名藥不宜頻繁更換
雖然 Levothyroxine 品牌藥（Synthroid、Euthyrox 等）與學名藥療效相當，但不同廠牌間可能存在微小的生體可用率差異，**不建議頻繁更換廠牌**，尤其是 TSH 控制精確要求高的患者（甲狀腺癌、妊娠）。更換廠牌後應在 4–8 週後重新追蹤 TSH。
:::

::: warning 妊娠期間需求顯著增加
懷孕期間 Levothyroxine 需求量可增加 **25–50%**，尤其在第一孕期最為明顯。確認懷孕後應立即增加劑量（通常每週增加 2 天劑量），並每 4 週追蹤 TSH，維持在妊娠期目標範圍。
:::

::: info 免責聲明
本計算器提供體重基礎的 Levothyroxine 起始劑量估算，僅供臨床參考。實際劑量應根據 TSH 監測結果、臨床症狀及個別患者因素由醫師調整。複雜或難治性甲狀腺功能低下建議轉介內分泌科醫師。
:::
