# Calorie Requirements Table 成人每日熱量需求參考表

**Calorie Requirements Table — Adult Daily Energy Needs Reference**

## 概述

本工具依據 **USDA 膳食指南 2020–2025（Dietary Guidelines for Americans）附錄 2** 及美國醫學院（IOM）膳食參考攝取量（DRI）中的 EER 方程式，提供按**年齡、性別與身體活動程度**分層的成人每日熱量需求參考值，並同步計算**體重管理**（維持 / 每週減 0.5 kg / 每週減 1.0 kg）的熱量目標。

| 活動等級                   | 描述                                                                |
| -------------------------- | ------------------------------------------------------------------- |

| 中度活動 Moderately Active | 日常活動加上相當於每日步行 2.4–4.8 km（時速 4.8–6.4 km/hr）的活動量 |

## 減重熱量目標計算

### 基本原則

體重管理基於能量平衡：

```
熱量攝入 − 熱量消耗 = 能量儲存（± 體重）
```

每減少 **7,700 kcal** 的累積熱量缺口，約等於減少 **1 公斤**體重。

| 減重目標 | 每日熱量赤字 | 預期減重速率（每週） |
| -------- | ------------ | -------------------- |
| 維持體重 | 0 kcal       | 0 kg                 |
| 緩和減重 | −500 kcal    | 約 0.5 kg            |
| 積極減重 | −1,000 kcal  | 約 1.0 kg            |

### 安全最低熱量下限

| 性別 | 最低熱量（kcal/day） | 來源                  |
| ---- | -------------------- | --------------------- |
| 女性 | **1,200**            | AND 2014 體重管理指引 |
| 男性 | **1,500**            | AND 2014 體重管理指引 |

> 低於上述下限可能導致蛋白質流失（肌肉消耗）、微量營養素缺乏、基礎代謝率下降及電解質失衡。

## 為何熱量需求隨年齡下降

### 1. 基礎代謝率（BMR）下降

30 歲之後，BMR 每十年約下降 **5–10%**，主要原因是：

- **肌少症（Sarcopenia）**：從 30 歲起，每年流失約 0.5–1% 的骨骼肌
- **激素變化**：睪固酮（男性）和雌激素（女性）分泌減少，降低肌肉合成效率
- **細胞代謝減慢**：粒線體功能和細胞更新率降低

### 2. 身體組成變化

年齡增長導致**脂肪比例增加、肌肉量減少**，即使體重不變，代謝活躍組織的絕對量也在減少。

### 3. 身體活動量降低

社會角色轉變、健康狀況限制及心理因素，使許多成人隨年齡減少了日常身體活動量，進一步降低總能量消耗。

## 男女熱量需求差異的原因

在相同年齡和活動程度下，男性的每日熱量需求通常比女性**高 400–600 kcal**，主要原因包括：

| 因素         | 說明                                                            |
| ------------ | --------------------------------------------------------------- |
| 體型較大     | 男性平均身高和體重較高，維持基本生理功能的能量需求也更多        |
| 肌肉量較高   | 肌肉組織的靜息代謝率約為脂肪組織的 3 倍，男性因此有更高的 BMR   |
| 激素差異     | 睪固酮促進蛋白質合成及肌肉維持，相對提升代謝率                  |
| 參考體型差異 | USDA 男性參考體型（178 cm）比女性（163 cm）高，熱量估算本已較高 |

- 產後前 6 個月：**+500 kcal/day**
- 6 個月後：**+400 kcal/day**

### 疾病與恢復期

| 狀況                   | 熱量調整                          |
| ---------------------- | --------------------------------- |
| 一般術後恢復           | BEE × 1.0–1.1                     |
| 感染/發燒              | 體溫每升高 1°C，需求增加約 10–13% |
| 慢性阻塞性肺病（COPD） | 可能需要 120–150% 的正常需求      |
| 腫瘤惡病質             | BEE × 1.3–1.5                     |
| 嚴重燒傷               | BEE × 1.5–2.0                     |

## 體重管理的實用策略

### 飲食方面

- **漸進式減量**：每日減少 300–500 kcal，避免劇烈限制導致代謝適應
- **蛋白質充足**：每公斤體重 1.2–1.6 g 的蛋白質，有助於保留肌肉量
- **食物選擇**：優先選擇高體積、低熱量密度的食物（蔬菜、水果、全穀類）
- **定時進食**：規律的進食時間有助於調節食慾激素（瘦素、飢餓素）

### 身體活動方面

- **結合有氧與阻力訓練**：有氧運動消耗熱量，阻力訓練維持肌肉量（避免 BMR 下降）
- **增加 NEAT**：非運動性身體活動（NEAT，如站立、走路購物、爬樓梯）對長期體重管理貢獻顯著
- **美國 HHS 建議**：每週至少 150 分鐘中等強度有氧活動（或 75 分鐘高強度），加上每週 2 次肌力訓練

### 行為改變

- **自我監控**：記錄飲食日記（手機 App）可使減重效果提升 40–50%
- **設定具體目標**：SMART 目標（Specific, Measurable, Achievable, Relevant, Time-bound）
- **睡眠充足**：睡眠不足增加食慾（飢餓素上升），並降低自制力

## 此表格與個人化 EER 計算的差異

| 特性     | 本表格（USDA DGA）       | 個人化 EER 計算                      |
| -------- | ------------------------ | ------------------------------------ |
| 輸入資料 | 年齡 + 性別 + 活動程度   | 年齡 + 性別 + 身高 + 體重 + 活動程度 |
| 適用對象 | 一般成人（接近參考體型） | 各種體型成人                         |
| 準確性   | 人群平均估算             | 個人化估算                           |
| 使用場景 | 快速參考、衛教使用       | 臨床評估、精確計算                   |
| 體型適應 | 固定參考體型（無法調整） | 完全個人化                           |

> 若個人身高體重與 USDA 參考體型差距較大（如肥胖、矮個子或高個子），建議使用個人化 EER 計算器。

## 注意事項

::: warning 參考體型限制
本表格數值使用 USDA 固定參考體型（男性 178 cm、女性 163 cm）推算，適用於體型接近參考值的一般成人。若個人身高體重差異顯著，應使用個人化 EER 方程式計算實際需求。
:::

::: warning 最低熱量安全下限
每日熱量攝取不應低於女性 1,200 kcal 或男性 1,500 kcal。過度限制熱量可能導致：肌肉流失、基礎代謝率顯著下降、微量營養素缺乏、骨質流失及電解質失衡等代謝併發症。
:::

::: warning 特殊族群不適用
本表格不適用於孕婦、哺乳婦女、嚴重疾病患者、兒童及青少年。這些族群應由醫師或臨床營養師進行個別化評估，以確保滿足額外的生理需求。
:::

::: info 免責聲明
本計算器提供的熱量目標為一般參考值。實際的體重管理計畫，尤其是有慢性病（糖尿病、高血壓、腎臟病等）的患者，應在醫師或註冊營養師（RD）的指導下進行，以確保安全和有效性。
:::
