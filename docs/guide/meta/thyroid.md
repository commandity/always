## Levothyroxine 劑量計算

**Levothyroxine**（合成 L-甲狀腺素，LT4）是原發性甲狀腺功能低下症的標準替代治療。起始劑量以體重為基礎，並依臨床情境選擇劑量係數：

```
起始劑量（mcg/day）= 體重（kg）× 劑量係數（mcg/kg/day）
```

### 劑量係數解讀

| 臨床情境          | 係數（mcg/kg/day）  | 意義與適用對象                                       |
| ----------------- | ------------------- | ---------------------------------------------------- |
| 完全替代          | 1.6                 | 全甲狀腺切除術後、嚴重原發性甲低——腺體無殘存功能     |
| 部分替代          | 1.0–1.2             | 橋本氏甲狀腺炎、輕中度甲低——腺體仍有部分功能         |
| 亞臨床甲低        | 0.5–0.75            | TSH 輕度升高（< 10 mIU/L）且 FT4 正常                |
| 老年／高心臟風險  | 0.8–1.0（低劑量起始）| ≥ 65 歲或冠心病患者，由 12.5–25 mcg/day 緩慢上調     |

> **為什麼老年與心臟病患者要低劑量起始**：甲狀腺素會增加心肌耗氧量，全量替代可能誘發心絞痛或心律不整；建議每 4–6 週依 TSH 調整一次。計算結果應對應至最接近的市售錠劑規格（25・50・75・88・100・112・125・137・150・175・200・300 mcg，誤差 ≤ 15 mcg 可接受），不足規格可用兩錠組合、切半或每週變化劑量策略達成。

### 治療目標與追蹤

| 指標           | 目標                        |
| -------------- | --------------------------- |
| TSH（一般）    | 0.4–4.0 mIU/L               |
| TSH（甲癌術後）| 依風險分層抑制至更低        |
| 追蹤時機       | 起始或調整劑量後 6–8 週複驗 |

## 注意事項

::: warning 空腹服藥
Levothyroxine 應於早餐前 30–60 分鐘空腹服用；鈣片、鐵劑、制酸劑會顯著影響吸收，需間隔至少 4 小時。
:::

::: info 免責聲明
本計算器僅供臨床輔助參考，實際劑量需由醫師依 TSH／FT4 追蹤結果個別化調整。
:::
