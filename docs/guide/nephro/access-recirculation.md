# Access Recirculation 血管通路再循環率

**Access Recirculation Rate**

## 概述

**血管通路再循環**（Access Recirculation）是指透析後的靜脈端血液（已清除尿素的血液）在回流至體循環之前，再次被動脈端抽入透析器，導致透析效率下降的現象。再循環率 ≥ 10% 即有臨床意義。

## 計算公式

```
Recirculation [%] = ( Systemic BUN − Inlet BUN ) / ( Systemic BUN − Outlet BUN ) × 100
```

| 變項         | 採樣位置                | 說明               |
| ------------ | ----------------------- | ------------------ |
| Systemic BUN | 非通路側手臂靜脈        | 代表體循環真實 BUN |
| Inlet BUN    | 動脈端（arterial line） | 透析器入口血液     |
| Outlet BUN   | 靜脈端（venous line）   | 透析器出口血液     |

**解讀：** 若 Inlet BUN 與 Systemic BUN 相同，代表無再循環。若 Inlet BUN 較低（接近 Outlet BUN），代表已清除的血液被重新抽入，再循環率高。

## 臨床參考值

| 再循環率 | 意義               | 建議行動           |
| -------- | ------------------ | ------------------ |
| < 5%     | 正常               | 定期追蹤           |
| 5–10%    | 輕度               | 密切監測，評估原因 |
| 10–20%   | 顯著（有臨床意義） | 通路超音波評估     |
| ≥ 20%    | 嚴重               | 立即介入評估       |

## 常見再循環原因

**針頭位置問題（最常見）：**

- 動靜脈針距離 < 5 cm
- 針頭插入方向與血流相同（順流穿刺）
- 建議針頭距離 ≥ 5 cm，且逆血流方向穿刺

**通路結構異常：**

- 靜脈狹窄（venous stenosis）：引起靜脈高壓，使透析後血液逆流
- 動靜脈瘻管血栓部分阻塞
- 人工血管（graft）彎折或扭曲

**血流量設定過高：**

- 若設定血流量（Qb）> 通路自然血流量（Qa）× 1/3，再循環機率大增
- Qa > 3 × Qb 可避免顯著再循環

**中心靜脈導管（CVC）位置不良：**

- 雙腔導管的動靜脈端相互影響
- 導管尖端未在右心房入口處

## 最精確的測量方法

| 方法                     | 準確度 | 說明                                   |
| ------------------------ | ------ | -------------------------------------- |
| **Transonic 血流稀釋法** | 最高   | 超音波稀釋技術，同時測量通路流量（Qa） |
| 葡萄糖輸注稀釋法         | 高     | 注入葡萄糖後測量稀釋程度               |
| BUN 法（本計算器）       | 中等   | 受抽血時間和技術影響                   |
| 尿素動力學模型法         | 中等   | 依賴多次 BUN 測量                      |

## 通路流量（Qa）監測

再循環常與通路流量降低並存。建議每 3–6 個月測量 Qa：

| Qa                 | 意義                 |
| ------------------ | -------------------- |
| AVF > 600 mL/min   | 正常                 |
| AVF 400–600 mL/min | 警戒，密切追蹤       |
| AVF < 400 mL/min   | 介入評估（血管攝影） |
| Graft > 700 mL/min | 正常                 |
| Graft < 600 mL/min | 狹窄風險高           |

::: warning BUN 法測量再循環受操作技術影響較大
抽血時間點、針頭位置、透析機暫停時機均可影響 BUN 法的準確性。建議在透析穩定進行至少 30 分鐘後抽取 Systemic BUN，並確保三個血液樣本在同一時間點採集，以減少誤差。
:::

::: warning 再循環率高不等於通路狹窄，需鑑別原因
高再循環率最常見的原因是**針頭位置錯誤**（順流穿刺或距離過近），而非通路狹窄。在進行昂貴的血管攝影或介入治療前，應先確認穿刺技術，並以 Transonic 血流稀釋法確認 Qa 是否下降。
:::

::: warning 中心靜脈導管（CVC）的再循環特性不同
CVC 的再循環機制與 AVF/Graft 不同，主要源自導管尖端位置不佳或管腔設計問題。CVC 患者發現高再循環率時，應優先評估導管位置（胸部 X 光確認）和嘗試交換動靜脈管腔後重新測量。
:::

本計算器僅供透析通路功能初步評估參考，不能取代完整的通路評估、Transonic 血流測量及必要的血管攝影檢查。透析通路的監測與介入決策應由腎臟科醫師及相關專科醫師根據患者完整臨床狀況判斷。
