# Bicarbonate Deficit 碳酸氫鈉缺乏量

**Bicarbonate Deficit**

## 概述

**碳酸氫鈉缺乏量計算**用於**代謝性酸中毒**患者的碳酸氫鈉（NaHCO₃）補充量估算，協助臨床醫師安全、適量地糾正酸鹼失衡。

## 計算公式

```
HCO₃⁻ Deficit (mEq) = 0.5 × Weight (kg) × ( HCO₃⁻ desired − HCO₃⁻ observed )
```

**係數 0.5** 代表 HCO₃⁻ 的分佈容積（約體重的 50%），實際上酸中毒時因骨骼緩衝作用，分佈容積可能更大（0.4–0.6）。

## 代謝性酸中毒嚴重度

| HCO₃⁻（mmol/L） | 嚴重度 | 估算 pH   | 臨床表現                  |
| --------------- | ------ | --------- | ------------------------- |
| 18–22           | 輕度   | 7.30–7.35 | 輕度過度換氣              |
| 10–17           | 中度   | 7.10–7.30 | Kussmaul 呼吸（深大呼吸） |
| < 10            | 重度   | < 7.10    | 心律不整、血流動力學不穩  |

## 補充原則

**首次僅給予計算量的 50%（half correction rule）：**

1. 計算總缺乏量
2. 首次靜脈給予 **50% 量**（約 3–4 小時輸注）
3. 複查血氣及電解質
4. 根據新的血氣結果再決定是否繼續補充
5. **目標 HCO₃** 通常設在 **18–20 mmol/L**，而非完全正常化

## 代謝性酸中毒分類（AG 分析）

**高陰離子間隙（High AG）代謝性酸中毒：**
記憶口訣 MUDPILES：甲醇（M）、尿毒症（U）、DKA（D）、丙二醇（P）、異煙鹼/鐵（I）、乳酸中毒（L）、乙二醇（E）、水楊酸（S）

**正常陰離子間隙（Hyperchloremic）代謝性酸中毒：**
記憶口訣 HARDUP：高氯性輸液（H）、腎小管性酸中毒（A/R）、腹瀉（D）、輸尿管分流（U）、胰臟瘻管（P）

## 何時積極給予 NaHCO₃？

| 情境                       | 建議                               |
| -------------------------- | ---------------------------------- |
| pH < 7.1                   | 考慮積極給予                       |
| HCO₃ < 10 + 血流動力學不穩 | 積極治療                           |
| DKA                        | **不建議**常規給予，胰島素治療為主 |
| 乳酸中毒                   | 爭議性，治療根本病因為主           |
| 腎小管性酸中毒（RTA）      | 口服 NaHCO₃ 長期維持               |
| 透析患者低 HCO₃            | 透析液調整或口服補充               |

::: warning 過度糾正的多重風險
首次僅給予計算量的 50%，避免：**代謝性鹼中毒**（HCO₃ 過快升高）、**低鉀血症**（鹼化使 K⁺ 進入細胞）、**呼吸代償失調**（CO₂ 清除延遲使顱內 pH 先降後升）、**高鈉血症**（8.4% NaHCO₃ 每 mL 含 1 mEq Na⁺）。補充後必須重新評估血氣及電解質。
:::

::: warning DKA 和乳酸中毒不建議常規使用 NaHCO₃
DKA 的根本治療是胰島素和補液，補充 NaHCO₃ 可能延遲酮體清除、加重低鉀血症，且 pH > 7.0 時幾乎無益。乳酸中毒的根本治療是改善組織灌流，NaHCO₃ 在 pH ≥ 7.1 時通常不建議使用，需優先處理根本病因。
:::

::: warning 分佈容積係數（0.5）是估算值，個體差異大
嚴重酸中毒時骨骼系統參與緩衝，HCO₃⁻ 的實際分佈容積可達 0.4–0.8 × 體重。計算結果僅為初始劑量的參考起點，臨床上必須根據血氣動態調整，不可機械性按公式給足計算量。
:::

::: info 免責聲明
本計算器僅供代謝性酸中毒初步補充量估算參考，不能取代完整的酸鹼評估與臨床判斷。碳酸氫鈉補充的適應症、劑量及速率應由具執照的醫師根據患者完整血氣、電解質及臨床狀況綜合判斷。
:::
