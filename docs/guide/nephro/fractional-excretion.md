# Fractional Excretion 分餾排泄率

**Fractional Excretion**

## 概述

**分餾排泄率（Fractional Excretion, FE）**是評估腎臟對特定物質重吸收能力的重要指標，最常用於急性腎損傷（AKI）的病因鑑別，特別是區分**腎前性**與**腎性（固有性）AKI**。

## 計算公式

```
FE(X) [%] = ( U_X × Se-Cr ) / ( P_X × U-Cr ) × 100
```

| 變項  | 說明                                  |
| ----- | ------------------------------------- |
| U_X   | 尿液中物質 X 的濃度（mEq/L 或 mg/dL） |
| Se-Cr | 血清肌酸酐（mg/dL）                   |
| P_X   | 血漿中物質 X 的濃度                   |
| U-Cr  | 尿液肌酸酐（mg/dL）                   |

肌酸酐作為分母的意義：消除尿量變化的影響，反映**真實腎臟清除比例**。

## FENa 解讀（急性腎損傷鑑別）

| FENa     | 診斷       | 常見病因                                      | 腎小管完整性       |
| -------- | ---------- | --------------------------------------------- | ------------------ |
| **< 1%** | 腎前性 AKI | 低血容量、心衰竭、肝硬化（肝腎症候群）、NSAID | 完整（正常重吸收） |
| **1–2%** | 不確定     | 需結合臨床                                    | —                  |
| **> 2%** | 腎性 AKI   | 急性腎小管壞死（ATN）、急性腎炎               | 受損（無法重吸收） |

## FEUrea（利尿劑使用中的替代指標）

當患者正在使用 **Loop Diuretics 或 Thiazide** 時，FENa 會因藥物抑制 Na 重吸收而**偽性升高**，此時改用 FEUrea：

```
FE_Urea [%] = ( U_Urea × Se-Cr ) / ( P_Urea × U-Cr ) × 100
```

| FEUrea    | 解讀            |
| --------- | --------------- |
| **< 35%** | 腎前性 AKI      |
| **> 50%** | 腎性 AKI（ATN） |

## 其他電解質的分餾排泄率

| 電解質           | 正常 FE   | 臨床應用                                           |
| ---------------- | --------- | -------------------------------------------------- |
| **FEK（鉀）**    | 5–15%     | 評估腎臟鉀排泄，鑑別低鉀原因（腸胃道 vs 腎臟流失） |
| **FECl（氯）**   | 類似 FENa | 輔助酸鹼診斷                                       |
| **FEMg（鎂）**   | < 3%      | 低鎂血症病因分析；> 3–4% = 腎臟流失                |
| **FEPhos（磷）** | < 5%      | 低磷血症鑑別；FTR（磷小管再吸收率）= 1−FEPhos      |

## 臨床案例解析

**案例：** 65 歲患者因嘔吐 3 天就診，血清 Cr 升至 2.5 mg/dL（基礎 0.9）。尿液分析：U-Na 8 mEq/L、U-Cr 120 mg/dL、Se-Na 142、Se-Cr 2.5。

計算：FENa = (8 × 2.5) / (142 × 120) × 100 = **0.12%**

→ FENa < 1%：**腎前性 AKI**（脫水引起），補液為主要治療。

::: warning FENa 在使用利尿劑的患者中失去鑑別效力
Loop Diuretics 可使 FENa 偽性升高至 > 2%，即使是腎前性 AKI 亦然。使用利尿劑的患者應改用 FEUrea（< 35% 提示腎前性），或等待利尿劑效果消退後再測。
:::

::: warning FENa < 1% 不代表一定是腎前性 AKI
以下情況 FENa < 1% 但非腎前性：造影劑腎病（早期）、橫紋肌溶解（肌紅蛋白尿）、急性腎絲球腎炎（AGN）、早期尿道阻塞。需結合臨床病史、尿液沉渣及影像學排除這些診斷。
:::

::: warning 少尿時單次尿液樣本的局限性
FENa 僅反映採樣當下的腎臟功能，少尿期的尿液濃度可能極度波動。建議在體液狀態穩定後採樣，且同時採集血液和尿液樣本（避免時間差），以確保計算結果的代表性。
:::

::: info 免責聲明
本計算器僅供急性腎損傷病因初步鑑別參考，不能取代完整的腎功能評估、尿液沉渣檢查及影像學評估。AKI 的診斷與治療決策應由具執照的醫師根據患者完整臨床狀況判斷。
:::
