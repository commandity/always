# H₂O Deficit 自由水缺乏量

**H₂O Deficit Free Water Deficit**

## 概述

**自由水缺乏量**（Free Water Deficit, FWD）計算用於**高血鈉症**（Hypernatremia）的補液治療，估算需要補充多少純水才能將血鈉恢復至正常範圍（140 mEq/L）。

## 計算公式

```
FWD (L) = TBW × ( Na / 140 − 1 )
```

```
TBW = LBM (kg) × 0.6  （男性）
TBW = LBM (kg) × 0.5  （女性）
```

| 變項 | 說明                         |
| ---- | ---------------------------- |
| TBW  | 總體水量（Total Body Water） |
| LBM  | 去脂體重（Lean Body Mass）   |
| Na   | 實測血清鈉（mEq/L）          |
| 140  | 目標血鈉（mEq/L）            |

## 高血鈉症分類與病因

| 類型 | 血鈉          | 常見病因                          |
| ---- | ------------- | --------------------------------- |
| 輕度 | 145–150 mEq/L | 液體攝入不足、輕度脫水            |
| 中度 | 150–160 mEq/L | 尿崩症、發燒、腹瀉                |
| 重度 | > 160 mEq/L   | 中樞性/腎源性尿崩症、高張鹽水輸注 |

**依據體積狀態分類：**

| 體積狀態           | 尿鈉       | 常見病因                       |
| ------------------ | ---------- | ------------------------------ |
| 低血容量（最常見） | < 20 mEq/L | 腎外流失（出汗、嘔吐、腹瀉）   |
| 低血容量           | > 20 mEq/L | 腎性流失（利尿劑、滲透性利尿） |
| 正常血容量         | 低張尿     | 中樞性或腎源性尿崩症           |
| 高血容量           | > 20 mEq/L | 高張鹽水、原發性醛固酮增多症   |

## 糾正速率原則

| 高血鈉類型           | 發病速度 | 建議糾正速率   | 時間            |
| -------------------- | -------- | -------------- | --------------- |
| 急性（< 48 小時）    | 快速     | ≤ 1 mEq/L/hr   | 可在 24h 內完成 |
| 慢性（> 48h 或不明） | 緩慢     | ≤ 0.5 mEq/L/hr | 48–72h 以上     |

**實際補液計算範例：**
FWD = 3 L → 建議在 48h 內緩慢補充 → 輸注速率 ≈ 62 mL/hr D5W

## 補液選擇

| 液體             | 自由水含量 | 適用情境                           |
| ---------------- | ---------- | ---------------------------------- |
| D5W（5% 葡萄糖） | 幾乎 100%  | 純自由水補充首選                   |
| 0.45% NaCl       | 約 50%     | 同時需補充部分 Na                  |
| 0.9% NaCl        | 0%（等張） | 低血容量合併高血鈉（先恢復血容量） |
| 口服水分         | 100%       | 輕度、意識清楚                     |

## 監測

- 每 2–4 小時複查血清鈉
- 目標：每小時血鈉下降 ≤ 0.5 mEq/L（慢性）
- 同時評估尿量、尿滲透壓（排除持續性水分流失）

::: danger 過快糾正高血鈉症可導致腦水腫
慢性高血鈉（> 48 小時）時，腦細胞已透過增加有機滲透物質（如 myoinositol、牛磺酸）來適應高滲狀態。過快補充自由水會使細胞外滲透壓驟降，水分快速湧入腦細胞，引發腦水腫，表現為意識改變、癲癇，嚴重時可致死。糾正速率不得超過 0.5 mEq/L/hr。
:::

::: warning FWD 公式僅計算靜態缺乏量，未包含持續流失
FWD 公式計算的是「恢復至正常血鈉所需的水量」，但不包括患者仍持續的不顯性失水（發燒、尿崩症、腹瀉等）。若有持續性水分流失，實際補液量需在 FWD 基礎上加計每日維持需水量和持續失水量，並根據每 2–4 小時的血鈉監測動態調整。
:::

::: warning 低血容量合併高血鈉時，需先補充等張液
若患者同時有嚴重低血容量（血壓低、心跳快），應先以 0.9% NaCl 恢復血液動力學穩定，再逐步轉換為低張液補充自由水。過早使用低張液在低血容量患者可能加重循環不穩定。
:::

::: info 免責聲明
本計算器僅供高血鈉症初步補液量估算參考，不能取代完整的電解質評估與臨床監測。高血鈉症的補液治療應由具執照的醫師根據患者完整臨床狀況、持續失水速率及每 2–4 小時血鈉監測結果動態調整。
:::
