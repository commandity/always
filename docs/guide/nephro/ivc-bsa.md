# IVC/BSA 比值

**IVC/BSA Ratio**

## 概述

**IVC/BSA（下腔靜脈直徑/體表面積）比值**是評估血液透析患者體積狀態（volume status）和乾體重（dry weight）的超音波輔助工具，能客觀反映體內水分過載或不足的程度。

## 計算公式

**體表面積（BSA）— Mosteller 公式：**

```
BSA (m²) = √( Height (cm) × Weight (kg) / 3600 )
```

**IVC/BSA 比值：**

```
IVC/BSA Ratio = IVC 直徑 (mm) / BSA (m²)
```

**IVC 塌陷指數（Collapsibility Index, CI）：**

```
CI (%) = ( IVC_呼氣 − IVC_吸氣 ) / IVC_呼氣 × 100
```

- CI > 50%：提示低血容量或前負荷不足
- CI < 15%：提示體積過載

## 參考範圍

| IVC/BSA（mm/m²） | 體積狀態       | 臨床建議                 |
| ---------------- | -------------- | ------------------------ |
| < 8              | 低血容量       | 補充體液、降低超濾量     |
| **8–11.5**       | **正常乾體重** | 維持現有設定             |
| 11.5–14          | 輕度過載       | 考慮增加超濾             |
| > 14             | 明顯過載       | 積極超濾、降低乾體重目標 |

## 超音波測量技術

- 測量位置：**劍突下 2–3 cm 處**，矢狀切面
- 呼吸時相：**平靜呼氣末**（排除呼吸影響）
- 測量方法：前後徑（AP diameter）
- 建議測量 3 次取平均值

## 臨床應用

| 應用場景        | 說明                                     |
| --------------- | ---------------------------------------- |
| 乾體重調整      | 每月定期超音波評估，動態調整透析超濾目標 |
| 透析後評估      | 確認乾體重是否達標                       |
| 低血壓發作評估  | 鑑別低血容量 vs 心因性低血壓             |
| 心衰竭合併 ESRD | 體積管理特別困難，超音波輔助尤為重要     |

::: warning 右心衰竭患者的 IVC 結果不可靠
右心衰竭（右心房壓升高）會導致 IVC 持續擴張，即使患者實際上並非體積過載，IVC/BSA 比值也可能偏高。此類患者需結合臨床症狀（頸靜脈怒張、肝腫大、下肢水腫）和超音波心臟功能評估，不可單憑 IVC/BSA 決定超濾量。
:::

::: warning 測量技術和呼吸配合影響結果重現性
IVC 直徑隨呼吸週期大幅波動，測量時相（呼氣末 vs 吸氣末）不同可差距達 50%。建議固定在**平靜呼氣末**測量，由同一操作者執行，每次取 3 次平均值，以確保不同時間點之間的可比性。
:::

::: warning IVC/BSA 為輔助工具，需結合其他體積評估指標
IVC/BSA 應與臨床症狀（症狀性低血壓、水腫程度、體重變化）、生物指標（BNP/NT-proBNP）及患者主觀感受（疲勞感、痙攣）綜合判斷，不可作為單一決策依據。
:::

::: info 免責聲明
本計算器僅供透析患者體積狀態初步評估參考，不能取代完整的臨床評估與超音波心臟功能檢查。乾體重的設定與調整應由腎臟科醫師根據患者完整臨床狀況及個別因素綜合判斷。
:::
