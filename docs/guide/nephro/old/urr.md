# Urea Reduction Ratio 尿素減少率

**Urea Reduction Ratio**

## 概述

**尿素減少率**（Urea Reduction Ratio, URR）是評估血液透析充分度（hemodialysis adequacy）最簡便的指標，僅需透析前後的血尿素氮（BUN）即可計算，不需透析時間、體重或超濾量。

## 計算公式

```
URR [%] = ( Pre-BUN − Post-BUN ) / Pre-BUN × 100
```

## KDOQI 充分度標準

| URR        | 透析充分度                 | 臨床建議             |
| ---------- | -------------------------- | -------------------- |
| ≥ 70%      | 優良                       | 維持現有透析處方     |
| **65–70%** | **達標（KDOQI 最低標準）** | 可接受，持續監測     |
| 55–65%     | 邊緣不足                   | 延長時間或增加血流量 |
| < 55%      | 嚴重不足                   | 立即重新評估透析處方 |

## URR 的限制

URR 是 Kt/V 的簡化近似值，有以下限制：

1. **不計入超濾效應**：超濾使血液濃縮，透析後 BUN 相對升高，低估實際清除率
2. **不反映溶質分佈容積變化**：不同體型患者比較困難
3. **適用於穩定狀態患者**：殘餘腎功能顯著者不準確

**URR 與 Kt/V 的近似關係：**

| Kt/V | 對應 URR（近似） |
| ---- | ---------------- |
| 1.0  | ≈ 58%            |
| 1.2  | ≈ 65%            |
| 1.4  | ≈ 72%            |

## 提升 URR 的方法

| 方法                   | 效果                                |
| ---------------------- | ----------------------------------- |
| 延長透析時間（最有效） | 每 30 分鐘 ≈ +3–5% URR              |
| 增加血流量（Qb）       | 受通路流量限制                      |
| 增加透析液流量（Qd）   | 效果有限（> 500 mL/min 邊際效益低） |
| 改善通路功能           | 降低再循環，提升有效清除率          |
| 高通量透析膜           | 改善中分子清除，URR 影響小          |

## 透析頻率與方案

| 透析方式       | 每週次數        | URR 目標          |
| -------------- | --------------- | ----------------- |
| 標準血液透析   | 3 次/週         | ≥ 65%             |
| 每日短時透析   | 5–6 次/週       | 個別化            |
| 夜間透析       | 3 次/週（長時） | 通常 > 75%        |
| 腹膜透析（PD） | 持續性          | Weekly Kt/V ≥ 1.7 |

::: warning URR 不計超濾，可能高估透析不足的嚴重程度
超濾量大的患者（透析中脫水 ≥ 3 L），超濾本身會濃縮血液中的 BUN，使透析後 BUN 升高，導致 URR 計算值偏低。此情況下建議同時計算 Kt/V（含超濾修正項），以更準確反映真實透析劑量。
:::

::: warning URR 僅評估尿素清除，無法反映整體透析毒素負荷
URR 以尿素作為代理指標，但中分子毒素（β₂-microglobulin、PTH 片段）、蛋白結合毒素（硫酸吲哚酚）的清除不反映在 URR 中。患者即使 URR 達標，仍可能有中大分子毒素蓄積相關症狀，此時需考慮高通量透析或血液透析濾過。
:::

::: warning URR 每月監測一次，不穩定患者需更頻繁
通路功能下降（Qa 降低）、體重大幅變化（影響 V）或透析時間縮短（患者提早結束）均可使 URR 驟降。若發現 URR 較基礎值下降 > 10%，應立即評估通路功能（Transonic 測量）和透析記錄，而非直接調整透析處方。
:::

::: info 免責聲明
本計算器僅供透析充分度初步評估參考，不能取代完整的透析處方評估。URR 未達標時的處方調整應由腎臟科醫師根據患者個別狀況（體型、通路功能、殘餘腎功能、透析時間記錄）綜合判斷。建議與 Kt/V 搭配使用以提高評估準確性。
:::
