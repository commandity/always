# Migraine 偏頭痛篩查與診斷標準

**Migraine Screening & Diagnostic Criteria**

## 概述

**偏頭痛**（Migraine）是全球第二大致殘神經疾病，影響全球約 15% 人口，女性盛行率（18–20%）為男性（6–8%）的三倍。本頁整合 **ID Migraine 篩查工具**（初級照護快速篩查）及 **ICHD-3 正式診斷標準**（無先兆偏頭痛 1.1 / 有先兆偏頭痛 1.2），協助臨床分級診斷。

## ID Migraine 篩查流程

```
初級照護就診（頭痛主訴）
              ↓
     排除繼發性頭痛（SNOOP）
   S: 全身性症狀/疾病
   N: 神經學症狀
   O: 突發性（Thunder clap）
   O: 老年新發（> 50 歲）
   P: 體位性或用力性頭痛
              ↓
      ID Migraine 3 題篩查
   ├─ 噁心（Nausea）
   ├─ 畏光（Photophobia）
   └─ 失能（Disability ≥ 1 天）
              ↓
  ≥ 2/3 陽性？
   ├─ 是 → 偏頭痛篩查陽性（PPV 93%）
   │        → 進行 ICHD-3 完整診斷
   └─ 否 → 考慮緊張型頭痛或其他診斷
              ↓
     ICHD-3 有無先兆分類
   ├─ 無先兆偏頭痛（1.1）→ ≥5 次 + A+B+C+D+E 全達標
   └─ 有先兆偏頭痛（1.2）→ ≥2 次 + A+B+C+D 全達標
```

## ID Migraine 篩查工具

### 三題評估項目

| 題目                | 評估內容                                   | 陽性定義 |
| ------------------- | ------------------------------------------ | -------- |
| 噁心（Nausea）      | 過去 3 個月頭痛時有噁心或胃部不適？        | 是       |
| 畏光（Photophobia） | 過去 3 個月頭痛時光線令您困擾？            | 是       |
| 失能（Disability）  | 過去 3 個月頭痛使您至少 1 天無法正常活動？ | 是       |

**判讀：≥ 2/3 項陽性 → 偏頭痛篩查陽性**

### 診斷效能

| 指標                  | 數值（95% CI） |
| --------------------- | -------------- |
| 敏感性（Sensitivity） | 81%（77–85%）  |
| 特異性（Specificity） | 75%（64–84%）  |
| 陽性預測值（PPV）     | 93%            |
| 三題全陽性            | PPV ≈ 98%      |

> ID Migraine 由 Lipton RB 等人於 2003 年在初級照護環境中驗證（n=563），是目前最廣泛使用的偏頭痛初級篩查工具。

## ICHD-3 無先兆偏頭痛診斷標準（1.1）

需同時符合 A–E 所有標準：

| 標準  | 條件                                               |
| ----- | -------------------------------------------------- |
| **A** | 至少 **5 次**符合 B–D 的頭痛發作                   |
| **B** | 頭痛持續 **4–72 小時**（未治療或治療無效）         |
| **C** | 頭痛特徵 **≥ 2/4**：單側、搏動性、中重度、活動加重 |
| **D** | 發作期 **≥ 1 項**：噁心/嘔吐、畏光＋畏聲           |
| **E** | 無法以其他 ICHD-3 診斷更好解釋                     |

### C 標準詳解（疼痛特徵，≥2/4）

| 特徵                      | 說明                             |
| ------------------------- | -------------------------------- |
| 單側（Unilateral）        | 頭痛局限於一側，可換側           |
| 搏動性（Pulsating）       | 跳動或脈搏感，非壓迫性           |
| 中重度（Moderate/Severe） | 影響日常活動                     |
| 活動加重（Aggravation）   | 行走、爬樓梯等日常活動使頭痛加重 |

## ICHD-3 有先兆偏頭痛診斷標準（1.2）

需同時符合 A–D 所有標準：

| 標準  | 條件                                                            |
| ----- | --------------------------------------------------------------- |
| **A** | 至少 **2 次**符合 B–C 的發作                                    |
| **B** | ≥ **1 種**完全可逆的先兆症狀（視覺/感覺/語言/運動/腦幹/視網膜） |
| **C** | ≥ **3/6** 先兆特徵（見下表）                                    |
| **D** | 無法以其他 ICHD-3 診斷更好解釋（特別排除 TIA）                  |

### C 標準詳解（先兆特徵，≥3/6）

| 特徵 | 說明                                    |
| ---- | --------------------------------------- |
| C1   | 至少 1 種先兆逐漸擴展 ≥ 5 分鐘          |
| C2   | ≥ 2 種先兆症狀連續出現                  |
| C3   | 每種先兆症狀持續 5–60 分鐘              |
| C4   | 至少 1 種先兆為單側                     |
| C5   | 至少 1 種先兆為陽性症狀（閃光、針刺感） |
| C6   | 先兆伴隨或在 60 分鐘內出現頭痛          |

### 先兆類型

| 類型                    | 臨床表現                                               |
| ----------------------- | ------------------------------------------------------ |
| 視覺先兆（最常見，99%） | 閃光、鋸齒形弧線（fortification spectrum）、暗點、模糊 |
| 感覺先兆                | 單側麻木、針刺感，從手向臉部擴散                       |
| 語言先兆                | 失語、找詞困難、構音障礙                               |
| 運動先兆                | 單側無力（偏癱型偏頭痛，CACNA1A/ATP1A2 基因）          |
| 腦幹先兆                | 暈眩、複視、共濟失調、耳鳴（基底動脈型）               |
| 視網膜先兆              | 單眼視力症狀（需排除視網膜剝離、TIA）                  |

## 偏頭痛治療策略

### 急性期治療

| 藥物類別                     | 代表藥物                                 | 適應症         |
| ---------------------------- | ---------------------------------------- | -------------- |
| NSAIDs                       | Ibuprofen 400–800 mg, Naproxen 500 mg    | 輕至中度       |
| Triptans（5-HT1B/1D 促效劑） | Sumatriptan 50–100 mg, Rizatriptan 10 mg | 中至重度，首選 |
| Gepants（CGRP 拮抗劑）       | Rimegepant 75 mg, Ubrogepant 50 mg       | Triptan 不耐受 |
| Ditans（5-HT1F 促效劑）      | Lasmiditan 50–200 mg                     | 心血管禁忌者   |
| 止吐藥                       | Metoclopramide 10 mg                     | 合併噁心嘔吐   |

> Triptan 禁忌：未控制高血壓、缺血性心臟病、腦中風病史、Prinzmetal 心絞痛、有先兆偏頭痛（相對禁忌，因腦中風風險）。

### 預防性治療（月 ≥ 4 天偏頭痛者）

| 藥物類別      | 代表藥物                             | 降頻效果               |
| ------------- | ------------------------------------ | ---------------------- |
| β 阻斷劑      | Propranolol 40–120 mg/d, Metoprolol  | 50% 降頻率             |
| 抗癲癇藥      | Topiramate 25–100 mg/d, Valproate    | 50% 降頻率             |
| 抗憂鬱藥      | Amitriptyline 10–75 mg/d             | 合併睡眠障礙           |
| CGRP 單株抗體 | Erenumab, Fremanezumab, Galcanezumab | 慢性偏頭痛首選         |
| 肉毒桿菌毒素  | OnabotulinumtoxinA（BOTOX）          | 慢性偏頭痛（≥15天/月） |

## 注意事項

::: warning 繼發性頭痛紅旗徵兆
以下情況需立即評估排除繼發性頭痛：雷擊性頭痛（突然最嚴重頭痛）、首次在 50 歲後出現的頭痛、頭痛伴發燒/頸項強直、神經學症狀（除典型先兆外）、頭痛隨體位、用力、Valsalva 改變、近期外傷史、HIV 或免疫抑制患者。
:::

::: warning 有先兆偏頭痛與腦中風風險
有先兆偏頭痛（尤其女性 + 口服避孕藥 + 吸菸）缺血性中風風險增加 2–4 倍。含雌激素避孕藥應謹慎使用；確診有先兆偏頭痛的女性偏頭痛患者，建議考慮純孕激素避孕法。
:::

::: info Probable Migraine
若患者符合所有 ICHD-3 偏頭痛標準，但僅有 1 項未達標，可診斷為「可能偏頭痛（Probable Migraine）」，建議頭痛日誌追蹤 1–3 個月後重新評估。
:::

::: info 台灣偏頭痛流行病學
台灣偏頭痛盛行率約 9.1%（女性 14.3%，男性 4.3%）。每年因偏頭痛造成的工作失能天數平均達 3.8 天/人，是台灣最重要的致殘神經疾病之一。
:::

::: info 免責聲明
本計算器依據 ID Migraine 量表及 ICHD-3 診斷標準設計，僅供臨床教育及輔助評估參考，不能取代神經科或頭痛專科正式評估。偏頭痛診斷需結合完整病史、神經學檢查及必要的影像學排除繼發性病因。
:::
