# ACUTE VERTIGO 急性眩暈鑑別診斷

**Acute Vertigo Differential Diagnosis**

## 概述

眩暈是急診與神經科最常見、也最具挑戰性的主訴之一。約 3–5% 的急性眩暈患者為後循環腦幹或小腦中風，而前庭神經炎（AUVP）、BPPV、前庭性偏頭痛與梅尼爾氏症合計佔 80% 以上。正確的鑑別診斷流程可避免誤診與漏診兩大錯誤。

下方整合工具包含五個步驟，從排除中風開始，逐步確立周圍性診斷。

## 如何使用工具

### 五步驟鑑別流程

此工具設計為臨床決策輔助，建議依以下順序操作：

**Step 0 · 眩暈分流** — 根據症狀持續時間和誘發模式，將眩暈分為三類：急性持續性（AVS）、位置性誘發、反覆發作性。初始分流決定後續應使用哪個評估路徑。

**Step 1 · HINTS 排除中風** — 對所有急性持續性眩暈（AVS）患者，必須先完成 HINTS 三步驟眼動檢查，排除後循環缺血性中風。任何一項 HINTS 陽性（正常 HIT、方向改變眼振、斜偏存在）即應立即啟動中風評估流程，不得假設為良性前庭疾患。

**Step 2 · BPPV 決策樹** — 針對位置性眩暈，依序確認症狀特徵（潛伏期、持續時間、疲勞性），再進行 Dix-Hallpike 測試及眼振類型判讀，確認半規管位置後執行對應的復位治療。

**Step 3 · AUVP / 前庭神經炎** — HINTS 三項均周圍性後，進一步以 Bárány Society 2022 的五項標準（A：持續眩暈；B：水平扭轉性眼振；C：VOR 不對稱；D：無中樞體徵；E：排除其他）確立急性單側前庭病變診斷。

**Step 4 · VM / Ménière's** — 對反覆發作性眩暈，依據偏頭痛特徵（偏頭痛病史、畏光畏聲、視覺先兆）和耳部症狀（低頻聽損、耳鳴、耳悶）分別評估前庭性偏頭痛或梅尼爾氏症。

## HINTS 眼動檢查詳解

HINTS 是針對急性前庭症候群（AVS）最有效的床邊排中風工具，其敏感性（94–100%）甚至優於發病 24 小時內的早期 DWI-MRI（72–83%）。

### 為什麼「正常 HIT」是危險信號？

這是 HINTS 最重要、也最反直覺的概念。在 AVS 情境下：

- **前庭神經炎**（周圍性）：前庭功能受損 → VOR 功能下降 → 頭部快轉後眼球滑移 → 需要補正掃視（**異常 HIT**）
- **腦幹/小腦中風**（中樞性）：VOR 路徑通常完整 → 頭部快轉後眼球自然跟隨 → 無需補正掃視（**正常 HIT** = 危險信號）

換句話說：「正常的頭部衝擊試驗」在 AVS 情境中反而意味著更高的中風風險。

### 三步驟判讀速查

| 步驟  | 檢查                     | 周圍性（良性）     | 中樞性（中風警訊）     |
| ----- | ------------------------ | ------------------ | ---------------------- |
| **H** | 頭部衝擊試驗（HIT）      | 異常（有補正掃視） | **正常（無補正掃視）** |
| **I** | 眼振方向                 | 單一方向固定快相   | **方向改變**           |
| **N** | 斜偏測試（Test of Skew） | 無垂直補正         | **斜偏存在**           |

> **任一項中樞性** → HINTS 陽性 → 立即神經科評估 + 急診 MRI（含 DWI）

### HINTS Plus

HINTS 三步驟均為周圍性，但出現**新出現單側聽力損失**時，須考慮前下小腦動脈（AICA）梗塞（HINTS Plus 陽性），仍建議影像學評估。

## BPPV 診斷詳解

BPPV 是最常見的眩暈原因，佔眩暈門診約 20–30%，且有高達 90% 的治癒率（Epley 復位術）。關鍵在於確認半規管位置。

### 三種半規管類型比較

| 類型                 | 盛行率 | Dix-Hallpike 眼振                   | 首選治療                      |
| -------------------- | ------ | ----------------------------------- | ----------------------------- |
| **後半規管（PC）**   | 85–95% | 向上跳動＋旋轉性（上極朝患側）      | Epley 復位術，單次成功率 ~90% |
| **水平半規管（HC）** | 5–15%  | 純水平眼振（需 Roll Test 鑑別亞型） | Lempert 360° 或 Gufoni 法     |
| **前半規管（AC）**   | < 1%   | 向下跳動＋旋轉性                    | 特殊復位；需排除中樞病變      |

### Roll Test 判讀（水平半規管）

水平眼振的方向性決定亞型與患側：地向性（geotropic）眼振的**較強側**為患側（管石症）；背地性（apogeotropic）眼振的**較弱側**為患側（壺腹石症）。

## 急性單側前庭病變（AUVP）/ 前庭神經炎

AUVP 是急性持續性眩暈最常見的良性病因，估計發病率為每 10 萬人年 3.5 例。Bárány Society 2022 診斷標準將其分為確定性（需滿足 A–E 全部五項）與可能性（4/5 達標）兩個層次。

### 五項診斷標準

**A. 急性持續性眩暈（≥24h，非位置性）** — 靜止時仍有持續性旋轉或非旋轉性眩暈，伴步態不穩及噁心嘔吐。

**B. 自發性水平扭轉性眼振（方向固定）** — 快相朝健側（遠離患側），去除視覺固定後增強。純垂直或純扭轉性眼振是中樞警訊，不符合 AUVP。

**C. VOR 不對稱（患側受損）** — 頭部衝擊試驗（HIT）顯示患側補正掃視，或 caloric 測試顯示患側半規管麻痺（CP ≥25%），或 vHIT 增益不對稱。

**D. 排除中樞體徵** — 無凝視誘發眼振（GEN）、無顯著斜偏（skew deviation）、無急性聽力損失（若有則診斷迷路炎而非 AUVP）。

**E. 排除其他診斷** — 不能以 BPPV、梅尼爾氏症、VM 或中樞病變更好解釋。

### 治療原則

| 時期             | 治療                                   | 目的             |
| ---------------- | -------------------------------------- | ---------------- |
| 急性期（1–3 天） | Methylprednisolone 100mg/d 遞減 3 週   | 加速前庭功能恢復 |
| 急性期           | 前庭抑制劑（Dimenhydrinate 等，≤3 天） | 緩解急性噁心眩暈 |
| 恢復期           | 前庭復健運動（凝視穩定、步態訓練）     | 促進中樞代償     |

前庭抑制劑不應超過 3 天，過度使用會抑制中樞代償機制，延長恢復時間。

## 前庭性偏頭痛

前庭性偏頭痛（Vestibular Migraine, VM）估計盛行率為 1–3%，是繼 BPPV 後第二常見的反覆性眩暈原因，但因頭痛不一定在每次前庭發作時出現，臨床誤診率偏高。

### Bárány Society / IHS 2022 診斷標準

診斷需同時符合四項：

- **A**：≥5 次前庭發作，每次持續 5 分鐘至 72 小時，強度中度或重度
- **B**：現有或曾有符合 ICHD-3 的有先兆或無先兆偏頭痛病史
- **C**：≥50% 的前庭發作期間，同時出現至少 1 項偏頭痛特徵（頭痛含 ≥2 偏頭痛特性；畏光且畏聲；視覺先兆）
- **D**：排除其他前庭疾患或 ICHD-3 診斷

僅符合 3/4 項則診斷為「可能前庭性偏頭痛（Probable VM）」。

### VM vs Ménière's 鑑別要點

| 特徵         | 前庭性偏頭痛（VM）           | 梅尼爾氏症（MD） |
| ------------ | ---------------------------- | ---------------- |
| 頭痛         | 常見（可能在不同發作中出現） | 少見             |
| 低頻聽損     | 無永久性聽損                 | ✓ 患側低頻 SNHL  |
| 耳鳴         | 偶爾                         | ✓ 同側，波動性   |
| 耳悶脹       | 少見                         | ✓ 常見           |
| 畏光 / 畏聲  | ✓ 常見                       | 少見             |
| 發作時間     | 5 min – 72 h                 | 20 min – 12 h    |
| 偏頭痛家族史 | 常見                         | 不常見           |

## 梅尼爾氏症

梅尼爾氏症（Ménière's Disease, MD）由內淋巴水腫引起，特徵為反覆的三聯症：陣發性眩暈 + 患耳低頻感音性聽損 + 耳鳴/耳悶。Bárány Society 2015 聯合診斷標準區分確定（Definite）與可能（Probable）兩個層級。

### 確定 MD vs 可能 MD

| 標準      | 確定梅尼爾氏症                  | 可能梅尼爾氏症             |
| --------- | ------------------------------- | -------------------------- |
| A（眩暈） | ≥2 次自發眩暈，20 min–12 h      | ≥2 次前庭發作，20 min–24 h |
| B（聽力） | **聽力圖記錄低頻 SNHL（必要）** | 無需聽力圖                 |
| C（耳部） | 波動性聽力/耳鳴/耳悶（≥1 項）   | 波動性耳部症狀             |
| D（排除） | 排除其他診斷                    | 排除其他診斷               |

確定 MD 的核心要求是**聽力圖文件**：純音聽力圖記錄患耳低至中頻（250–2000 Hz）PTA ≥30 dB HL，這是區分兩個診斷層級的關鍵。

### 治療階梯

低鹽飲食（< 1500 mg/day）+ 利尿劑為基礎治療，反覆發作者考慮鼓室注射 dexamethasone，保守治療無效時再考慮 gentamicin 鼓室注射（化學性迷路切除術）。確定 MD 診斷前需以 MRI 排除聽神經瘤及排除自體免疫性內耳病。

## 注意事項

::: warning HINTS 使用限制
HINTS 需由受過訓練的臨床人員執行，判讀需整合三項結果而非單一項目。橋腦梗塞因累及前庭核可偽裝為異常 HIT（假周圍性），是 HINTS 最危險的假陰性來源。HINTS 不能作為排除中風的唯一依據，需結合完整神經學評估及危險因子評估。
:::

::: warning 下跳性眼振警示
無論是 Dix-Hallpike 誘發或自發性**向下跳動性眼振（downbeat nystagmus）**，必須優先排除中樞病變：小腦扁桃體疝出（Arnold-Chiari）、脊髓小腦萎縮症（SCA）、小腦梗塞、副腫瘤症候群。前半規管 BPPV（AC-BPPV）極罕見（< 1%），應為排除性診斷。
:::

::: warning 梅尼爾氏症繼發性病因
確診前需排除：聽神經瘤（acoustic neuroma，MRI DWI+FLAIR 建議）、自體免疫性內耳病（AIED）、梅毒性迷路炎（血清 VDRL/FTA-ABS）、繼發性內淋巴水腫。雙側梅尼爾氏症（30–40%）應特別排除 AIED。
:::

::: info BPPV 復發與老年族群
BPPV 五年累積復發率約 50%。老年人（尤其骨質疏鬆、維生素 D 缺乏者）復發率更高，應追蹤補充。BPPV 在 65 歲以上是最常見眩暈原因（約 40%），但老年眩暈常為多重病因，需同時考慮前庭功能老化及藥物性前庭毒性。
:::

::: info 前庭性偏頭痛的識別
VM 常被誤診的關鍵原因：頭痛不一定每次發作都出現，部分患者甚至以純眩暈為唯一症狀；發作間期神經學檢查完全正常；與梅尼爾氏症的發作時間窗有重疊（20 min–12 h 亦可見於 VM）。完整偏頭痛病史及發作日誌是鑑別的核心。
:::

::: info 免責聲明
本頁工具依據 Bárány Society 及 IHS 最新診斷標準設計，僅供臨床教育及輔助評估參考，**不能取代神經科、耳鼻喉科或神經耳科正式評估**。急性眩暈鑑別診斷需整合完整病史、神經耳科學檢查及必要的聽力學、影像學評估。
:::
