# CMAI 激動情緒行為量表

**CMAI Cohen-Mansfield Agitation Inventory**

## 概述

**CMAI**（Cohen-Mansfield Agitation Inventory）由 Jiska Cohen-Mansfield 於 1986 年發展，是目前全球**最廣泛使用的失智症激動行為評估工具**。CMAI 共 29 個行為項目，涵蓋身體攻擊性（PA）、身體非攻擊性（PNA）、語言激動（V）三個面向，由照護者依據患者**過去兩週**的行為頻率評分，總分 29–203 分。CMAI 具有優異的信度與效度，廣泛應用於機構照護、藥物臨床試驗及非藥物干預的療效評估。

## 評分說明

每題依行為發生頻率評分，共 7 個等級：

| 分數 | 頻率描述     | 說明               |
| ---- | ------------ | ------------------ |
| 1    | 從未         | 過去兩週內從未發生 |
| 2    | 少於每週一次 | 偶發，極少出現     |
| 3    | 每週一至兩次 | 每週偶發           |
| 4    | 每週數次     | 每週多次出現       |
| 5    | 每天一至兩次 | 幾乎每天出現       |
| 6    | 每天數次     | 每天多次出現       |
| 7    | 每小時數次   | 持續反覆出現       |

**總分範圍：29–203 分**（基準值 29 = 所有行為從未發生）

> **臨床顯著性標準**：任何單項行為評分 **≥ 4**（每週數次以上），或總分明顯高於基準值 29 分，均提示臨床顯著激動。

## 三個面向的臨床意義

| 面向                | 英文                    | 題數 | 行為特徵                                           | 常見誘因                         |
| ------------------- | ----------------------- | ---- | -------------------------------------------------- | -------------------------------- |
| 身體攻擊性（PA）    | Physical Aggressive     | 11   | 打人、踢人、咬人、推人、抓撓、吐口水等直接攻擊行為 | 疼痛、個人照護介入、環境過度刺激 |
| 身體非攻擊性（PNA） | Physical Non-Aggressive | 10   | 遊蕩、穿脫衣、重複動作、囤積、普遍不安             | 無聊、感覺剝奪、睡眠障礙         |
| 語言激動（V）       | Verbal Agitation        | 8    | 尖叫、咒罵、重複問題、抱怨、要求關注               | 焦慮、憂鬱、未被滿足的需求       |

## 29 個評估項目

### 身體攻擊性（PA）——11 項

| 題號 | 行為               | 臨床意義                             |
| ---- | ------------------ | ------------------------------------ |
| Q1   | 打人（包括打自己） | 最常見的攻擊行為，常在個人照護時發生 |
| Q2   | 踢人               | 常見於床上照護或換尿布時             |
| Q3   | 抓住他人           | 可能反映對安全感的需求               |
| Q4   | 推人               | 常在照護者接近時出現                 |
| Q5   | 投擲物品           | 挫折或憤怒的表達                     |
| Q6   | 咬人               | 較嚴重的攻擊行為，需評估口腔疼痛     |
| Q7   | 抓撓他人           | 常見於個人照護抗拒                   |
| Q8   | 吐口水             | 可能為拒絕或表達不適                 |
| Q9   | 傷害自己或他人     | 需立即評估安全風險                   |
| Q10  | 撕毀物品或破壞財產 | 挫折或認知混亂的行為表現             |
| Q11  | 不恰當的身體性接觸 | 需評估用藥及環境因素                 |

### 身體非攻擊性（PNA）——10 項

| 題號 | 行為                   | 臨床意義                     |
| ---- | ---------------------- | ---------------------------- |
| Q12  | 來回踱步或漫無目的遊蕩 | 最常見的激動行為，需防止走失 |
| Q13  | 穿著不當或脫衣         | 可能反映對溫度、舒適度的反應 |
| Q14  | 試圖前往不同地方       | 可能反映對家的渴望或定向障礙 |
| Q15  | 故意跌倒               | 需評估憂鬱及尋求關注的動機   |
| Q16  | 食用或飲用不當物質     | 需確保環境安全               |
| Q17  | 不當處理物品           | 常見於嚴重認知退化患者       |
| Q18  | 藏匿物品               | 可能源於不安全感或偏執       |
| Q19  | 囤積物品               | 常見行為，通常無安全疑慮     |
| Q20  | 重複性舉動             | 反映強迫性或神經性行為模式   |
| Q21  | 普遍性煩躁不安         | 常為多種誘因的非特異性表現   |

### 語言激動（V）——8 項

| 題號 | 行為               | 臨床意義                       |
| ---- | ------------------ | ------------------------------ |
| Q22  | 尖叫               | 常反映疼痛、恐懼或嚴重焦慮     |
| Q23  | 言語性性挑逗       | 需評估用藥及額葉功能           |
| Q24  | 咒罵或言語攻擊     | 常在照護介入時出現             |
| Q25  | 重複句子或問題     | 反映記憶障礙及焦慮             |
| Q26  | 發出奇怪的聲音     | 可能為疼痛或不適的替代表達     |
| Q27  | 抱怨               | 需評估是否有真實身體不適       |
| Q28  | 消極態度           | 可能反映憂鬱或習得性無助       |
| Q29  | 持續無理由要求關注 | 常見於高焦慮或安全感不足的患者 |

## 激動行為的常見誘因

激動行為往往是患者**未被滿足需求**的表達，系統性評估誘因是非藥物干預的基礎：

| 類別     | 常見誘因                                                    |
| -------- | ----------------------------------------------------------- |
| 身體因素 | 疼痛、感染（UTI、肺炎）、便秘、尿瀦留、藥物副作用、睡眠障礙 |
| 環境因素 | 過度噪音、溫度不適、光線不足、陌生環境、照護者更換          |
| 心理因素 | 憂鬱、焦慮、恐懼、無聊、孤獨感                              |
| 照護因素 | 個人照護介入（洗澡、換尿布）、不熟悉的照護者                |
| 認知因素 | 定向障礙、幻覺、妄想（被偷竊、被遺棄）                      |

## 非藥物干預策略

非藥物干預為激動行為的**首選治療**，應在考慮藥物前優先嘗試：

| 策略                       | 適用行為           | 證據等級 |
| -------------------------- | ------------------ | -------- |
| 音樂治療                   | 語言激動、普遍不安 | 高       |
| 感覺刺激（芳香療法、按摩） | 身體非攻擊性       | 中       |
| 懷舊治療                   | 重複問題、焦慮     | 中       |
| 以人為本的照護（PCC）      | 照護抗拒、攻擊行為 | 高       |
| 光照治療                   | 睡眠障礙相關激動   | 中       |
| 運動與活動計劃             | 遊蕩、普遍不安     | 中       |
| 環境改造                   | 遊蕩、定向障礙     | 中       |
| 行為分析（ABC 模式）       | 所有激動類型       | 高       |

## 藥物治療

非藥物干預無效或行為造成安全風險時，考慮藥物治療：

| 藥物類別           | 代表藥物                | 適用情況               |
| ------------------ | ----------------------- | ---------------------- |
| 非典型抗精神病藥   | Risperidone、Quetiapine | 攻擊性、妄想相關激動   |
| 抗憂鬱藥（SSRI）   | Sertraline、Citalopram  | 憂鬱相關激動、語言激動 |
| 苯二氮平類         | Lorazepam（短期）       | 急性激動、程序前鎮靜   |
| 抗癲癇藥           | Valproic acid           | 衝動性攻擊行為         |
| 乙醯膽鹼酯酶抑制劑 | Donepezil               | 輕中度失智合併激動     |

> **警告**：抗精神病藥在失智症患者中的使用與中風及死亡率增加相關（FDA 黑框警告）。應以最低有效劑量短期使用，並定期重新評估必要性。

## 注意事項

::: warning CMAI 由照護者填寫，需熟悉患者
CMAI 的準確性依賴照護者對患者過去兩週行為的直接觀察。應由最熟悉患者日常行為的主要照護者（家屬、護理師）填寫，而非偶爾接觸的工作人員。不同照護者的評分可能存在差異，建議固定評估者以確保追蹤的一致性。
:::

::: warning 攻擊行為需優先評估安全
PA 面向任何單項評分 ≥ 4，特別是打人、咬人、傷害自己或他人，需立即評估照護者及患者的安全風險，並制訂安全照護計畫。非藥物干預應與安全防護措施同步實施。
:::

::: warning 激動行為可能反映未被治療的疼痛
失智症晚期患者常無法以語言表達疼痛，激動行為（尤其是尖叫、攻擊）可能是疼痛的唯一表現。在啟動任何激動行為干預前，應系統性評估疼痛（如使用 PAINAD 量表），並嘗試止痛治療。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不具診斷效力。激動行為的評估與治療應由神經科、精神科、老年醫學科醫師或具備失智症照護專業的團隊根據完整評估綜合判斷。
:::
