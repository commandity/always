## FAST 功能性評估分期

**Functional Assessment Staging**（Reisberg）以「功能」界定失智進程，分 1–7 期（6、7 期再分亞期），與心智年齡及 MMSE 對應。

| 分期 | 分類         | 功能特徵                         | 對應 MMSE |
| ---- | ------------ | -------------------------------- | --------- |
| 1–2  | 正常／主觀抱怨 | 無客觀缺損                     | 28–30     |
| 3    | 輕度認知障礙 | 複雜任務受干擾（工作、規劃）     | 24–28     |
| 4    | 輕度失智     | IADL 受影響（繳費、烹飪）        | 19–20     |
| 5    | 中度失智     | 需協助選擇衣物                   | 15        |
| 6a–e | 中重度失智   | 穿衣、盥洗、如廁、大小便失禁     | 5–9       |
| 7a–f | 重度失智     | 語言喪失至無法行走、坐立、微笑   | 0         |

> **FAST 7c（無法行走）是安寧緩和照護轉介的常用門檻**；FAST 適合追蹤中重度以後的進程（此時 MMSE 已地板效應）。

## MMSE 簡易心智狀態

30 分制，涵蓋定向（10）、即時記憶（3）、注意計算（5）、延遲記憶（3）、語言與視覺建構（9）。

| 總分  | 認知程度     | 意義                                   |
| ----- | ------------ | -------------------------------------- |
| 24–30 | 正常範圍     | 需依教育程度校正（低教育切截值較低）   |
| 18–23 | 輕度障礙     | 建議完整神經心理評估                   |
| 10–17 | 中度障礙     | 明顯功能影響，安排失智症候群評估       |
| < 10  | 重度障礙     | 重度失智；照護需求評估優先             |

> 台灣常用教育校正切截：未受教育 < 16、國小 < 21、國中以上 < 24 視為異常。MMSE 對早期／高教育者不敏感，懷疑 MCI 時建議改用 MoCA。

## CDR 臨床失智評量表

六領域（記憶、定向、判斷、社區、家居、自我照顧）各評 0–3，經規則合成整體 CDR。

| CDR | 嚴重度     | 意義                     |
| --- | ---------- | ------------------------ |
| 0   | 正常       | 無失智                   |
| 0.5 | 可疑／極輕 | MCI 或極早期失智         |
| 1   | 輕度       | 日常生活可獨立但需提醒   |
| 2   | 中度       | 需相當程度協助           |
| 3   | 重度       | 完全依賴照護             |

> **CDR 0.5 是身心障礙鑑定與藥物給付常引用的分界**；CDR 以「與病前相比」評分，需照護者提供可靠病史。

## GDS 整體退化量表

**Global Deterioration Scale**（Reisberg）1–7 期，與 FAST 對應但以「認知」描述為主：1–2 正常／主觀抱怨、3 = MCI、4–7 = 輕→重度失智。

> GDS ≥ 4 即進入失智範圍；與 FAST 搭配可同時掌握認知與功能兩軸。

## NPI 神經精神症狀量表

12 項精神行為症狀（妄想、幻覺、激動、憂鬱、焦慮、欣快、冷漠、脫抑制、易怒、異常動作、夜間行為、食慾改變），每項分數 = 頻率（1–4）× 嚴重度（1–3），總分 0–144；另評照顧者困擾（0–5）。

> **單項 ≥ 4 分視為臨床顯著**，是行為介入或藥物治療的評估門檻；總分變化用於追蹤 BPSD 治療成效。

## CMAI 激動情緒行為量表

29 項激動行為，各以頻率 1（從未）–7（每小時數次）計分，總分 29–203，分四因子：身體攻擊、身體非攻擊、語言攻擊、語言非攻擊。

> 總分 **≥ 39** 一般視為有臨床意義的激動行為；因子分析可指引介入方向（如遊走 vs 攻擊行為的不同處理策略）。

## 注意事項

::: warning 工具分工
MMSE 篩檢認知、CDR/GDS/FAST 分期、NPI/CMAI 評估行為精神症狀；完整失智評估應合併使用並佐以病史、實驗室與影像檢查。
:::

::: info 免責聲明
本計算器僅供臨床評分記錄與教育參考，失智症診斷需由神經科或精神科醫師綜合判斷。
:::
