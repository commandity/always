# FAST 功能性評估分期

**FAST Functional Assessment Staging**

## 概述

**FAST**（Functional Assessment Staging Test）由 Barry Reisberg 於 1984 年發展，是目前**最廣泛使用且最具科學文獻支持的阿茲海默症功能退化分期工具**。FAST 共 16 個分期（7 個主要分期，部分含子分期），依序描述從正常老化至重度失智症的功能退化軌跡，常與整體退化量表（GDS）合併使用，並廣泛應用於臨床評估、安寧療護資格判定及治療計畫制訂。

## 評分說明

FAST 共 **16 個分期**，患者的 FAST 分期應為目前功能狀態最符合的描述：

| ---- | ---------------- | ------------------------------- | -------------- | -------- | ----- |
| 1    | 正常老化         | 無任何功能缺損                  | —              | 成人     | 29–30 |
| 2    | 可能輕度認知障礙 | 主觀功能缺損（自覺記憶退步）    | —              | —        | 28–29 |
| 3    | 輕度認知障礙     | 客觀功能缺損干擾最複雜任務      | 84             | 12+      | 24–28 |
| 4    | 輕度失智症       | IADL 受影響（繳費、烹飪、外出） | 24             | 8–12     | 19–20 |

## FAST 四大階段

### 第一階段：正常至輕度認知障礙（FAST 1–3）

功能尚可獨立，但可能有主觀記憶抱怨（FAST 2）或複雜任務受影響（FAST 3）。FAST 3 相當於輕度認知障礙（MCI），雖不符合失智症診斷，但為阿茲海默症的高風險族群。

### 第二階段：輕度至中度失智症（FAST 4–5）

工具性日常生活活動（IADL）逐漸受損。FAST 4 患者無法獨立管理財務、烹飪或外出旅行；FAST 5 患者需要協助選擇適當衣物，但仍可自行穿衣。

### 第三階段：中重度失智症（FAST 6a–6e）

基本日常生活活動（ADL）依序受損，順序為：穿衣 → 洗澡 → 如廁 → 尿失禁 → 糞便失禁。此階段的依序退化是 FAST 的核心診斷特徵，任何跳躍均提示其他病因。

### 第四階段：重度失智症（FAST 7a–7f）

言語功能依序喪失（7a → 7b），接著喪失行走能力（7c）、坐立能力（7d）、微笑能力（7e），最後無法抬頭（7f）。此階段相當於嬰兒期前的發育逆轉。

## FAST 的核心原則：依序退化

FAST 量表最重要的臨床應用原則是：**阿茲海默症導致的功能退化嚴格依序發生，不會跳過分期。**

| 情況                        | 臨床意義                         |
| --------------------------- | -------------------------------- |
| 功能退化按序出現            | 符合阿茲海默症進展               |
| 跳過某分期（如 6a 跳至 6b） | 需考慮其他病因                   |
| 退化速度異常快              | 需評估合併症或用藥影響           |
| 功能突然惡化                | 需排除感染、代謝異常、藥物副作用 |

## FAST 與其他評估工具比較

|              | FAST                     | MMSE                       | CDR        | GDS              |
| ------------ | ------------------------ | -------------------------- | ---------- | ---------------- |
| **評估面向** | 功能退化                 | 認知功能                   | 認知+功能  | 整體退化         |
| **適用範圍** | 全程（1–7f）             | 輕至中度（受地板效應限制） | 輕至重度   | 全程（1–7）      |
| **特色**     | 依序退化，鑑別診斷       | 快速篩查，廣泛使用         | 多面向評估 | 與 FAST 高度對應 |
| **安寧療護** | ✓ FAST 7c 為常用資格標準 | —                          | —          | ✓ GDS 7          |

## 安寧療護的 FAST 資格標準

美國 Medicare 的失智症安寧療護資格通常要求：

1. **FAST ≥ 7c**（無法行走）
2. 合併至少一項醫療合併症（如吸入性肺炎、尿路感染、褥瘡、進食困難）
3. 生命預期 ≤ 6 個月

> FAST 7c 以上的患者通常需要全天候照護，無法獨立完成任何 ADL，建議討論安寧療護選項及預立醫療決定（AD）。

## 注意事項

::: warning FAST 為阿茲海默症特異性工具
FAST 是針對**阿茲海默症**設計的分期工具，對其他類型失智症（路易體失智、額顳葉失智、血管性失智）的分期準確性有限。若患者的退化模式不符合 FAST 的依序原則，需重新評估診斷。
:::

::: warning 跳過分期需積極鑑別
若患者跳過 FAST 分期（如仍能選擇衣物但已有糞便失禁），可能原因包括：診斷錯誤、合併其他失智症、急性醫療問題（感染、電解質異常）、藥物副作用，或照護環境改變。需積極評估。
:::

::: warning 預期持續時間僅供參考
FAST 各分期的預期持續時間為統計平均值，個別患者差異極大。疾病進展速度受遺傳、合併症、教育程度、照護品質及藥物治療等多重因素影響。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不具診斷效力。失智症的診斷及治療計畫應由神經科、精神科或老年醫學科醫師根據完整病史、神經心理評估及相關檢查綜合判斷。
:::
