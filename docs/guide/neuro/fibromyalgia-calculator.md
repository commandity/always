# SS 症狀嚴重度量表

**SS Symptom Severity Scale**

## 概述

**SS**（Symptom Severity Scale）是 2010 年美國風濕病學會（ACR）修訂版纖維肌痛診斷標準的核心組成之一，由 Wolfe 等人發表。SS 量表評估三項核心症狀（疲倦感、醒來仍感疲憊、認知功能問題）及整體身體症狀的嚴重程度，與**廣泛疼痛指數**（WPI，Widespread Pain Index）合併使用，構成不需壓痛點檢查的現代化纖維肌痛診斷標準。

## 評分說明

SS 量表共 4 項，總分 **0–12 分**：

| 項目           | 評分範圍 | 說明                                   |
| -------------- | -------- | -------------------------------------- |
| 疲倦感         | 0–3      | 0=沒有問題，1=輕微，2=中度，3=嚴重     |
| 醒來時仍感疲憊 | 0–3      | 0=沒有問題，1=輕微，2=中度，3=嚴重     |
| 認知功能問題   | 0–3      | 記憶力、專注力困難（腦霧）；0–3 分     |
| 整體身體症狀   | 0–3      | 0=沒有症狀，1=少數，2=相當數量，3=很多 |

> **整體身體症狀**包括：頭痛、腹痛、腸躁症、疲勞、記憶問題、肌肉無力、胸痛、頭暈、失眠、憂鬱、便秘、腹瀉、噁心、口乾、性慾減退、耳鳴等。

## 臨床切截點

| SS 總分 | 嚴重程度 | 臨床建議                             |
| ------- | -------- | ------------------------------------ |
| 0–3     | 輕微     | 症狀輕微，可觀察追蹤                 |
| 4–6     | 中等     | 中等程度，建議就醫評估               |
| 7–9     | 較重     | 症狀較重，建議積極治療               |
| 10–12   | 嚴重     | 嚴重症狀，建議風濕科或疼痛科完整評估 |

> **診斷門檻：SS ≥ 5**（需配合 WPI 評估，見下方診斷標準）。

## 2010 ACR 纖維肌痛診斷標準

纖維肌痛的診斷需同時滿足以下條件：

| 條件         | 標準                     |
| ------------ | ------------------------ |
| **方案一**   | WPI ≥ 7 **且** SS ≥ 5    |
| **方案二**   | WPI 3–6 **且** SS ≥ 9    |
| **持續時間** | 症狀持續 ≥ 3 個月        |
| **排除標準** | 無其他疾病可完全解釋症狀 |

> **WPI（廣泛疼痛指數）**：評估過去一週內 19 個身體部位的疼痛，每個部位計 1 分，總分 0–19 分。WPI 工具需另外完成。

## 各症狀的臨床意義

| 症狀         | 臨床意義                                                  |
| ------------ | --------------------------------------------------------- |
| 疲倦感       | 纖維肌痛最常見症狀之一，與睡眠障礙及中樞敏感化密切相關    |
| 醒來仍感疲憊 | 非恢復性睡眠（Non-restorative Sleep）是纖維肌痛的核心特徵 |
| 認知功能問題 | 「纖維霧」（Fibro Fog）——注意力、記憶力及思考清晰度下降   |
| 整體身體症狀 | 反映全身多系統的功能性症狀負擔，與中樞敏感化程度相關      |

## 纖維肌痛的病理機制

纖維肌痛被認為是一種**中樞敏感化症候群（Central Sensitization Syndrome）**：

- **中樞敏感化**：脊髓與大腦對疼痛訊號的放大處理，導致對正常刺激產生過度疼痛反應
- **下行抑制失調**：內生性鎮痛機制（如血清素、去甲腎上腺素路徑）功能減弱
- **神經炎症**：小膠質細胞活化，促進中樞神經系統的慢性低度炎症
- **自律神經失調**：心率變異性降低，交感神經過度活躍

## 治療建議

### 藥物治療

| 藥物類別     | 代表藥物                            | FDA 核准          |
| ------------ | ----------------------------------- | ----------------- |
| SNRI         | Duloxetine（Cymbalta）、Milnacipran | ✓                 |
| 抗癲癇藥     | Pregabalin（Lyrica）、Gabapentin    | ✓（Pregabalin）   |
| 三環抗憂鬱藥 | Amitriptyline（低劑量）             | 常用但非 FDA 核准 |
| 肌肉鬆弛劑   | Cyclobenzaprine                     | 常用於睡眠改善    |

### 非藥物治療（證據等級最高）

- **有氧運動**：游泳、快走、騎腳踏車，每週 3 次以上，循序漸進
- **認知行為治療（CBT）**：改善疼痛應對策略與睡眠
- **多學科疼痛計劃**：整合藥物、物理治療、心理治療的綜合方案
- **睡眠衛教**：規律作息、限制咖啡因、改善睡眠品質

## 注意事項

::: warning SS 量表需與 WPI 合併使用才能診斷
SS 量表單獨分數不足以診斷纖維肌痛，必須配合廣泛疼痛指數（WPI）評估，並由具備相關專業的醫師綜合判斷。纖維肌痛的診斷同時需排除其他可解釋症狀的疾病（如甲狀腺功能低下、類風濕性關節炎、系統性紅斑狼瘡）。
:::

::: warning 纖維肌痛不是排除診斷
2010 年後的 ACR 標準明確指出，纖維肌痛可與其他疾病（如類風濕性關節炎）共存，不應作為排除診斷使用。高 SS 分數合併其他疾病症狀者，需同時處理兩者。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不具診斷效力。纖維肌痛的診斷及治療應由風濕科、疼痛科或家庭醫學科醫師根據完整病史、身體檢查及相關評估綜合判斷。
:::
