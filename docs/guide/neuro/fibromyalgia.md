# 纖維肌痛評估工具

**Fibromyalgia Assessment · WPI & SS（ACR 2016）**

<FibromyalgiaAssessment />

## WPI 廣泛疼痛指數

**Widespread Pain Index**：勾選過去一週疼痛的身體部位（左右側各 7 區＋軀幹 5 區），總分 0–19。

| WPI  | 評估結果 | 意義                             |
| ---- | -------- | -------------------------------- |
| 0–2  | 局部疼痛 | 不符合廣泛性疼痛                 |
| 3–6  | 多區疼痛 | 搭配 SS ≥ 9 可符合診斷           |
| ≥ 7  | 廣泛疼痛 | 搭配 SS ≥ 5 即符合診斷           |

## SS 症狀嚴重度量表

**Symptom Severity Scale**：疲倦、睡醒仍累、認知症狀（腦霧）各 0–3 分，加上整體身體症狀 0–3，總分 0–12。

| SS 總分 | 嚴重度 | 意義                       |
| ------- | ------ | -------------------------- |
| 0–3     | 輕微   | 症狀負擔低                 |
| 4–8     | 中度   | 顯著疲勞／認知症狀         |
| 9–12    | 重度   | 高症狀負擔，需整體治療計畫 |

### ACR 2016 診斷標準

符合以下三項即可診斷纖維肌痛：

1. **WPI ≥ 7 且 SS ≥ 5**，或 **WPI 4–6 且 SS ≥ 9**
2. 症狀持續 **≥ 3 個月**
3. 至少 4/5 個區域有疼痛（廣泛性），且診斷不受其他疾病排除

> 兩個分頁的分數會共同決定是否達標——WPI 高而 SS 低（或反之）都可能不符合；治療追蹤常用 **FS 總分（WPI + SS，0–31）**，下降 ≥ 20% 視為有意義改善。

## 注意事項

::: warning 診斷為臨床綜合判斷
ACR 2016 為問卷型標準，仍需排除甲狀腺功能低下、發炎性關節炎、肌病變等模仿疾病。
:::

::: info 免責聲明
本計算器僅供臨床輔助參考，纖維肌痛診斷與治療請諮詢風濕免疫科或疼痛科。
:::
