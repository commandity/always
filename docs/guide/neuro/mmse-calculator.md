# MMSE 簡易心智狀態評估

**MMSE Mini-Mental State Examination**

## 概述

**簡易心智狀態評估**（Mini-Mental State Examination，MMSE）由 Folstein 等人於 1975 年發表，是全球臨床應用最廣泛的認知功能篩檢工具。MMSE 透過一系列標準化問題，在 5–10 分鐘內快速評估患者的整體認知功能，適用於失智症篩檢、認知障礙分期及病程追蹤。

## 評估領域

MMSE 共 **30 題**，涵蓋五大認知領域：

| 領域           | 題數 | 滿分 | 評估內容                                                                   |
| -------------- | ---- | ---- | -------------------------------------------------------------------------- |
| 定向力         | 10   | 10   | 時間定向（年、季、月、日、星期）與地點定向（國家、縣市、鄉鎮、地點、樓層） |
| 即時記憶       | 3    | 3    | 立即複述三個不相關的詞彙                                                   |
| 注意力與計算力 | 5    | 5    | 連續減 7（100→93→86→79→72→65）或逐字拼 WORLD                               |
| 延遲記憶       | 3    | 3    | 回憶先前複述的三個詞彙                                                     |
| 語言能力       | 9    | 9    | 命名、複述、三步驟指令、閱讀理解、自發書寫、圖形臨摹                       |

## 評分標準

每題以 **正確（1分）** 或 **錯誤／無法回答（0分）** 計分，總分 30 分。

| MMSE 總分 | 認知程度 | 臨床意義                                     |
| --------- | -------- | -------------------------------------------- |
| 27–30     | 正常     | 認知功能在正常範圍內                         |
| 21–26     | 輕度障礙 | 輕度認知功能缺損，建議進一步評估             |
| 10–20     | 中度障礙 | 中度認知障礙，日常生活受到影響，建議積極就醫 |
| 0–9       | 重度障礙 | 重度認知障礙，需全面性照護介入               |

> **注意**：MMSE 分數受**教育程度**顯著影響。低教育程度者（如未受正規教育）可能因語言、書寫或計算等題目而分數偏低，建議參考教育調整後的切截分數。

## 各領域說明

### 定向力（10分）

分為 **時間定向**（5分）與 **地點定向**（5分）兩部分。時間定向詢問年份、季節、月份、日期及星期；地點定向詢問國家、縣市、鄉鎮、所在地點及樓層。

### 即時記憶（3分）

清楚說出三個不相關的詞彙（如「蘋果、桌子、國旗」），請患者立即重複，每答對一個給 1 分。此三個詞彙稍後用於測試延遲記憶。

### 注意力與計算力（5分）

請患者從 100 開始連續減 7，共減 5 次（100、93、86、79、72、65）。每次正確減 7 給 1 分，若前一次計算有誤但繼續減 7 正確，仍可給分。若患者無法計算，可改以逐字倒念「WORLD」替代。

### 延遲記憶（3分）

在完成注意力測試後，請患者回憶先前複述的三個詞彙，每答對一個給 1 分。

### 語言能力（9分）

- **命名（2分）**：指出手錶和鉛筆，請患者說出名稱
- **複述（1分）**：複述「非也，和，但是」（或相應語言的版本）
- **三步驟指令（3分）**：「請拿起這張紙，將它對折，然後放到地上」
- **閱讀理解（1分）**：出示「請閉上眼睛」的字卡，患者需依指示執行
- **自發書寫（1分）**：自行寫出一個完整、有意義的句子
- **圖形臨摹（1分）**：臨摹兩個相交的五邊形，需保持各五個角及正確交疊

## 臨床應用

MMSE 廣泛應用於以下情境：

- **失智症篩檢**：初級醫療場域的快速認知評估
- **疾病分期**：配合 CDR、GDS 等量表進行失智嚴重度評估
- **病程追蹤**：定期施測監測認知功能變化（通常每 6–12 個月）
- **藥物療效評估**：膽鹼酯酶抑制劑等治療前後的功能比較
- **術前評估**：高齡患者手術前認知基線評估

## 注意事項

::: warning 教育程度影響分數
MMSE 分數受教育年限顯著影響。建議參考以下教育調整切截點：

- 未受教育者：≥ 17 分為正常
- 小學程度：≥ 20 分為正常
- 國中以上：≥ 24 分為正常

低分不一定代表失智，高分也不能完全排除早期失智症，需結合臨床判斷。
:::

::: warning MMSE 對早期失智症敏感度有限
MMSE 對**輕度認知障礙（MCI）**及**早期阿茲海默症**的偵測敏感度較低。若臨床高度懷疑但 MMSE 正常，建議搭配 MoCA（蒙特利爾認知評估）或其他更敏感的神經心理測驗進行完整評估。
:::

::: info 免責聲明
本計算器僅供臨床評分記錄與參考，不具診斷效力，不能取代完整的神經心理評估。若患者出現認知功能下降的跡象，請諮詢神經內科、精神科或失智症專科醫療人員進行完整評估與診斷。
:::
