## UPDRS 巴金森症狀衡量表

**Unified Parkinson's Disease Rating Scale** 分四部份：

| 部份 | 內容             | 題數 | 最高分 |
| ---- | ---------------- | ---- | ------ |
| I    | 智能、行為、情緒 | 4    | 16     |
| II   | 日常生活能力     | 13   | 52     |
| III  | 動作檢查         | 14＋ | 56     |
| IV   | 治療併發症       | 8    | 23     |

### 分數解讀

| 總分    | 嚴重度 | 意義與建議處置                             |
| ------- | ------ | ------------------------------------------ |
| < 32    | 輕度   | 症狀輕微；配合運動治療與藥物微調           |
| 32–58   | 中度   | 功能受影響；優化藥物時程，評估動作波動     |
| ≥ 59    | 重度   | 顯著失能；評估進階治療（DBS、幫浦治療）    |

> **Part III（動作檢查）是療效追蹤核心**：藥物 on/off 狀態下分別施測可量化動作波動；MCID 約 Part III 下降 5 分。Part IV 專門捕捉異動症與 off 期問題，是調整左旋多巴策略的依據。

## 注意事項

::: warning 施測狀態需註記
UPDRS 分數受服藥時間影響大，記錄時應註明 on/off 狀態與距上次服藥時間。
:::

::: info 免責聲明
本計算器僅供臨床評分記錄與教育參考，巴金森病治療調整需由神經科醫師決定。
:::
