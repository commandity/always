# NIHSS 中風量表

**NIHSS National Institutes of Health Stroke Scale**

## 概述

**美國國立衛生研究院中風量表**（National Institutes of Health Stroke Scale，NIHSS）由 Brott 等人於 1989 年發表，是目前全球急性腦中風臨床照護與研究中最廣泛使用的**神經功能缺損評估工具**。NIHSS 透過 15 個標準化評估項目，在 5–10 分鐘內量化中風造成的神經功能損傷程度，提供客觀、可重複的嚴重度評分，作為急性期治療決策、病程監測與預後評估的重要依據。

## 評估項目

NIHSS 共 **15 個評估項目**，涵蓋意識、眼部、運動、感覺及語言等多個神經系統面向：

| 項目 | 名稱         | 滿分     |
| ---- | ------------ | -------- |
| 1a   | 意識水平     | 3        |
| 1b   | 意識問答     | 2        |
| 1c   | 意識指令     | 2        |
| 2    | 凝視         | 2        |
| 3    | 視野         | 3        |
| 4    | 面部麻痺     | 3        |
| 5    | 左側上肢運動 | 4（+UN） |
| 6    | 右側上肢運動 | 4（+UN） |
| 7    | 左側下肢運動 | 4（+UN） |
| 8    | 右側下肢運動 | 4（+UN） |
| 9    | 肢體協調     | 2        |
| 10   | 感覺         | 2        |
| 11   | 語言         | 3        |
| 12   | 構音障礙     | 2（+UN） |
| 13   | 忽視         | 2        |

> 標記「+UN」的項目，若因截肢或關節融合等客觀原因無法評估，標記為 UN，不納入總分計算。

## 嚴重度分級

NIHSS 總分範圍為 **0–42 分**，分數越高代表神經功能缺損越嚴重：

| 總分  | 嚴重程度   | 臨床意義                       |
| ----- | ---------- | ------------------------------ |
| 0     | 正常       | 無神經功能缺損                 |
| 1–4   | 輕微中風   | 輕微神經功能缺損，預後通常良好 |
| 5–15  | 中度中風   | 中度缺損，建議積極評估與治療   |
| 16–20 | 中重度中風 | 中重度缺損，需密切監測         |
| 21–42 | 重度中風   | 重度缺損，預後較差             |

## 評分原則

- **記錄所見，而非推測**：依據患者當下的實際表現評分，不猜測其「可能的能力」
- **記錄第一反應**，而非最佳反應（第 11 項「語言」除外，語言取最佳表現）
- **不給予提示或協助**：讓患者自行完成指令，避免引導
- **UN（不計分）**：若因截肢、關節融合等客觀原因無法評估特定項目，標記 UN，不納入總分
- **評分時機**：急性期盡快完成基線評分；入院後 24 小時、出院前及定期追蹤時重複評估

## 臨床應用

### 急性期治療決策

- **靜脈溶栓（IV rt-PA / Alteplase）**：NIHSS ≥ 4 且症狀發作 **4.5 小時內**，無禁忌症者可考慮給予
- **靜脈溶栓（Tenecteplase）**：NIHSS 4–25 分，發作 4.5 小時內的輕至中度中風
- **血管內取栓（Mechanical Thrombectomy）**：NIHSS ≥ 6 合併影像確認的**大血管阻塞（LVO）**，發作 24 小時內（依影像結果決定）

### 嚴重度分級與照護規劃

- NIHSS ≥ 16：通常需入住加護病房（Stroke ICU）密切監測
- NIHSS 5–15：一般住院或中風病房
- NIHSS 1–4：部分患者可考慮觀察後出院，但仍需密切追蹤

### 預後評估

- **90 天良好預後**（mRS 0–2）的預測因子之一
- 溶栓後 24 小時 NIHSS 改善 ≥ 4 分，通常預示良好治療反應
- 基線 NIHSS 每增加 1 分，90 天良好預後機率下降約 17%

### 病程監測

- 定期評分可早期偵測**神經功能惡化**（NIHSS 增加 ≥ 2 分定義為惡化）
- 客觀記錄病程變化，利於跨團隊溝通

## 各項目評分重點

### 意識（1a–1c）

- **1a 意識水平**：觀察清醒程度，若需強力刺激才有反應評 2 分，完全無反應評 3 分
- **1b 意識問答**：詢問「現在幾月？」及「你幾歲？」，需精確回答才給分
- **1c 意識指令**：請患者「張眼、閉眼」及「握拳、張開」，以實際執行為準

### 運動功能（5–8）

- 上肢：手臂伸直上抬 90°（坐位）或 45°（臥位），維持 10 秒
- 下肢：平躺抬腿 30°，維持 5 秒
- 分數 1 = 可維持但不足時間；2 = 抵抗重力但無法維持；3 = 無法抵抗重力；4 = 完全無動作

### 語言（11）

- 評估命名、閱讀及描述能力，取**最佳表現**計分（與其他項目不同）
- 插管患者以書寫方式評估

## 注意事項

::: warning 後循環中風可能被低估
NIHSS 對**前循環**（尤其左側半球）中風的評估較為完整，但對**後循環**中風（腦幹、小腦）的嚴重度容易**低估**。眩暈、複視、共濟失調、吞嚥困難、Horner 氏症候群等後循環症狀在量表中的權重較低，分數可能偏低但實際臨床嚴重度不容忽視。後循環中風應配合神經影像（尤其 MRI-DWI）與完整神經學檢查綜合判斷。
:::

::: warning 訓練認證建議
NIHSS 的評分者間信度（inter-rater reliability）在受過訓練者中較高。美國心臟協會（AHA）建議定期接受 NIHSS 認證訓練，以確保評分的一致性與準確性，尤其在多中心研究或跨科團隊照護的情境下。
:::

::: info 免責聲明
本計算器僅供臨床評分記錄與參考，不能取代完整神經學評估與臨床判斷。所有治療決策（包括溶栓及取栓）應由合格神經科醫師根據患者具體情況、影像結果及禁忌症評估後做出。
:::
