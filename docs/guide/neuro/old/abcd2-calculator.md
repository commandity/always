# ABCD-2 TIA 風險評估

**ABCD-2 TIA Risk Assessment**

## 概述

**ABCD-2**（Age, Blood pressure, Clinical features, Duration, Diabetes）由 Johnston 等人於 2007 年發表於《刺胳針》（_The Lancet_），整合英國與美國多個世代研究的臨床資料，是目前臨床上最廣泛使用的**短暫性腦缺血發作（TIA）後早期腦中風風險預測工具**。其設計目的是協助臨床醫師在急診或門診快速識別需要緊急評估與住院處置的高風險 TIA 患者。

## 評分說明

ABCD-2 共 5 個項目，每項依臨床表現給予不同分數，總分最高 **7 分**：

| 項目         | 英文              | 條件                                 | 分數 |
| ------------ | ----------------- | ------------------------------------ | ---- |
| 年齡         | Age               | ≥ 60 歲                              | +1   |
| 血壓         | Blood Pressure    | 收縮壓 ≥ 140 mmHg 或舒張壓 ≥ 90 mmHg | +1   |
| 臨床症狀     | Clinical Features | 語言障礙但無偏癱                     | +1   |
|              |                   | 單側無力（上肢、下肢或顏面）         | +2   |
| 發作持續時間 | Duration          | 10–59 分鐘                           | +1   |
|              |                   | ≥ 60 分鐘                            | +2   |
| 糖尿病       | Diabetes Mellitus | 有糖尿病病史                         | +1   |

> 血壓以**就診時測量值**為準，而非過去病史。臨床症狀若同時出現語言障礙與單側無力，以**較高分**（+2）計算。

## 臨床切截點

| ABCD-2 分數 | 風險等級 | 2 天內中風風險 | 建議處置                             |
| ----------- | -------- | -------------- | ------------------------------------ |
| 0–3         | 低風險   | 約 1%          | 可考慮門診追蹤，7 天內完成神經影像   |
| 4–5         | 中風險   | 約 4%          | 建議 24 小時內完成評估，考慮住院觀察 |
| 6–7         | 高風險   | 約 8%          | 建議緊急住院，積極評估與介入         |

> **關鍵切截值：ABCD-2 ≥ 4** 為中至高風險，應積極住院評估並考慮雙重抗血小板治療（DAPT）。即使低分（0–3 分）患者，若有心源性病因或大血管狹窄，仍需謹慎處置。

## TIA 後的時間風險

TIA 是腦中風的**重要警示訊號**，發作後的早期風險極高：

| 時間窗    | 累積腦中風風險      |
| --------- | ------------------- |
| 24 小時內 | 3–10%（風險最高峰） |
| 7 天內    | 約 10–15%           |
| 30 天內   | 約 10–18%           |
| 90 天內   | 約 10–20%           |

> ABCD-2 ≥ 4 的患者應視為高度緊急，需盡速安排神經影像（MRI-DWI 優於 CT）、心臟評估（12 導程心電圖、心臟超音波）及血管檢查（頸動脈超音波或 CTA）。

## 各評分項目的臨床意義

| 項目               | 臨床意義                                           |
| ------------------ | -------------------------------------------------- |
| 年齡 ≥ 60 歲       | 老年患者血管硬化程度高，中風風險本身較高           |
| 血壓升高           | 就診時高血壓反映急性血管壓力，是中風的直接危險因子 |
| 單側無力（+2）     | 運動皮質或內囊受累，提示大血管或穿通動脈病變       |
| 語言障礙（+1）     | 優勢半球皮質受累，心源性栓塞較常見                 |
| 持續時間 ≥ 60 分鐘 | 症狀越長，腦部缺血損傷越嚴重，中風風險越高         |
| 糖尿病             | 微血管與大血管病變的獨立危險因子，增加再發風險     |

## 藥物治療建議

### ABCD-2 ≥ 4 分（中至高風險）

- 建議啟動**雙重抗血小板治療**（DAPT）
- 常見方案：Aspirin + Clopidogrel，連續使用 **21 天**後改為單一抗血小板藥物
- 依據 POINT 試驗（2018）及 CHANCE 試驗（2013）結果

### 所有 TIA 患者

- 盡早啟動 **Statin** 治療（目標 LDL < 70 mg/dL）
- 積極控制高血壓、糖尿病等危險因子
- 若為心源性栓塞（如心房顫動），改以**抗凝血藥物**為主，不用 DAPT

## ABCD-2 的限制

ABCD-2 是輔助決策工具，以下情況需特別謹慎：

| 情況           | 注意事項                                      |
| -------------- | --------------------------------------------- |
| 腦幹或小腦 TIA | 症狀可能不典型，ABCD-2 可能低估風險           |
| 心房顫動       | 本工具未納入心律評估，心源性 TIA 風險獨立且高 |
| MRI-DWI 陽性   | 若顯示急性梗塞病灶，無論分數高低均應積極處置  |
| 複發性 TIA     | 短期內多次發作風險極高，不應單靠分數決策      |
| 大血管狹窄     | 頸動脈或顱內血管重度狹窄者，低分亦需積極處置  |

## 注意事項

::: warning ABCD-2 不應單獨用於決定出院
研究顯示即使 ABCD-2 低分（0–3 分），部分患者仍有顯著中風風險，尤其是心源性病因或大血管狹窄者。美國神經學會（AAN）建議：所有 TIA 患者應在 **24–48 小時內**接受完整神經血管評估，不應僅以 ABCD-2 分數作為是否住院的唯一依據。
:::

::: warning 影像評估優先於評分工具
若 MRI-DWI 顯示急性腦梗塞病灶，即使臨床表現已緩解，仍應視為腦中風而非 TIA，需立即積極處置。神經影像結果應凌駕於任何臨床評分工具之上。
:::

::: info 免責聲明
本計算器僅供臨床參考，不能取代臨床判斷與完整神經學評估。所有治療決策應由合格醫療人員根據患者具體情況、影像結果及病史綜合做出。
:::
