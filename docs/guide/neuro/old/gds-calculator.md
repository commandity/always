# GDS 整體退化量表

**GDS Global Deterioration Scale**

## 概述

**整體退化量表**（Global Deterioration Scale，GDS）由 Barry Reisberg 等人於 1982 年在紐約大學發表，是目前臨床上應用最廣泛的**失智症整體分期工具**之一。GDS 不依賴單一認知測驗，而是透過對患者整體認知功能與日常生活能力的綜合評估，將退化程度分為七個階段，從認知正常到末期失智，提供直觀且易於溝通的病程描述。GDS 常與 CDR、MMSE 等工具合併使用，以獲得更全面的臨床圖像。

## 評分架構

GDS 共分為 **7 個階段**，數字越高代表認知與日常功能退化越嚴重：

| 階段 | 程度       | 對應失智嚴重度      | 核心特徵                               |
| ---- | ---------- | ------------------- | -------------------------------------- |
| 1    | 無認知退化 | 正常                | 無主觀或客觀記憶問題                   |
| 2    | 極輕度退化 | 主觀認知抱怨        | 偶有健忘，他人不易察覺                 |
| 3    | 輕度退化   | 輕度認知障礙（MCI） | 工作或社交能力輕微下降，家人開始察覺   |
| 4    | 中度退化   | 輕度失智            | 難以處理複雜事務，近期記憶受損         |
| 5    | 中重度退化 | 中度失智            | 日常生活需部分協助，忘記重要資訊       |
| 6    | 重度退化   | 中重度失智          | 需大量照護，出現明顯行為與人格改變     |
| 7    | 極重度退化 | 重度失智            | 語言、行走、吞嚥嚴重受損，完全依賴照護 |

## 各階段詳細說明

### 第一階段：無認知退化

認知功能完全正常，無任何主觀記憶抱怨，臨床訪談亦無異常發現。

### 第二階段：極輕度退化

患者主觀感覺記憶有些許退步，如忘記熟悉的人名或物品放置位置，但客觀測驗仍在正常範圍，他人不易察覺。此階段可能是正常老化，也可能是早期病理性退化的最初表現，需持續追蹤。

### 第三階段：輕度退化

開始出現可客觀測量的認知缺損，包括記憶力、學習新事物、空間定向及語言流暢度的輕微下降。工作表現可能受影響，家屬或密切相處者已能察覺。部分患者在此階段會被診斷為**輕度認知障礙（MCI）**，每年約 10–15% 的 MCI 患者會進展為失智症。

### 第四階段：中度退化（輕度阿茲海默症）

明顯記憶缺損，難以處理複雜日常事務（如財務管理、旅行規劃）。近期事件記憶受損，但對個人過去重要事件仍有部分記憶。患者可能出現迴避困難任務的傾向或防衛性否認。

### 第五階段：中重度退化（中度阿茲海默症）

患者在日常生活中需要旁人協助，無法回憶重要個人資訊（如電話、地址），但通常仍記得自己的姓名及重要家屬。定向感受損，但尚能自行如廁與進食。

### 第六階段：重度退化（中重度阿茲海默症）

需要大量日常生活協助，包括穿衣、盥洗、如廁。記憶嚴重退化，偶爾忘記配偶或主要照顧者的名字。常出現人格改變、睡眠障礙、遊走、妄想及失禁等問題。

### 第七階段：極重度退化（重度阿茲海默症）

認知與運動功能嚴重喪失。語言能力退化至僅能說出極少詞彙，最終完全喪失言語。無法自行行走或維持坐姿，進食需協助，通常有尿便失禁。此階段患者完全依賴全天候照護。

## GDS 與其他量表的對應關係

| GDS 階段 | CDR 分數 | MMSE 分數（參考） | 失智嚴重度     |
| -------- | -------- | ----------------- | -------------- |
| 1–2      | 0        | 28–30             | 正常至主觀抱怨 |
| 3        | 0.5      | 24–27             | 可疑 / MCI     |
| 4        | 1        | 18–23             | 輕度           |
| 5        | 2        | 10–17             | 中度           |
| 6        | 3        | 3–9               | 中重度         |
| 7        | 3        | 0–2               | 重度           |

> 各量表分數僅供參考對應，實際分數因患者個別差異而有所不同。

## 評估原則

- **綜合多方資訊**：結合患者本人訪談、家屬或照顧者觀察，以及臨床檢查
- **以近期功能為主**：聚焦於患者目前的實際日常表現，而非過去最佳狀態
- **排除可逆因素**：譫妄、急性感染、憂鬱、藥物副作用等可逆性因素應先排除
- **定期追蹤**：GDS 階段的進展速度因人而異，建議每 6–12 個月重新評估

## 臨床應用

- **失智症分期**：直觀描述病程進展，協助照護計劃制定
- **照護需求評估**：依分期決定居家、日照中心或機構照護的適切程度
- **病程溝通**：以具體階段描述協助家屬理解預後與照護重點
- **研究與試驗**：作為失智症介入研究的分層與結果評估工具
- **與 CDR、MMSE 合用**：提供多維度的認知功能評估

## 注意事項

::: warning 特殊族群評估可能受限
以下情況可能影響 GDS 評估的準確性，需由專業醫療人員審慎判斷：

- 失語症患者（語言障礙可能高估退化程度）
- 重度感官障礙（視力、聽力問題影響測試表現）
- 急性譫妄或感染期間（認知表現暫時性下降）
- 教育程度極低或文化背景差異顯著者
  :::

::: warning GDS 不等同失智症診斷
GDS 第 3 階段（輕度退化）僅代表認知功能輕微下降，並非失智症的確立診斷。失智症的正式診斷需符合特定臨床標準，並排除其他可逆性病因，應由專科醫師進行完整評估。
:::

::: info 免責聲明
本量表僅供初步篩檢與臨床參考，不具診斷效力，結果不能取代專業醫療評估。若有認知退化、記憶力下降或行為改變等情況，請諮詢神經內科、精神科或失智症專科醫療人員。
:::
