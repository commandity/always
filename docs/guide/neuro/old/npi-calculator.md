# NPI 神經精神症狀量表

**NPI Neuropsychiatric Inventory**

## 概述

**神經精神症狀量表**（Neuropsychiatric Inventory，NPI）由 Cummings 等人於 1994 年開發，是目前臨床上評估**失智症患者行為與精神症狀**（Behavioral and Psychological Symptoms of Dementia，BPSD）最廣泛使用的標準化工具。NPI 透過對主要照顧者的半結構式訪談進行評估，能夠量化症狀的**頻率**與**嚴重性**，反映患者在過去四週內的實際行為表現。

## 評估架構

NPI 共評估 **10 個神經精神症狀領域**：

| 項目               | 英文                    | 臨床表現                                   |
| ------------------ | ----------------------- | ------------------------------------------ |
| 妄想               | Delusions               | 對現實的錯誤信念，如認為有人偷竊、配偶不忠 |
| 幻覺               | Hallucinations          | 視幻覺最常見，也可有聽、嗅、觸覺幻覺       |
| 激動／攻擊行為     | Agitation / Aggression  | 言語或肢體攻擊、不合作、易激惹             |
| 憂鬱／情緒不佳     | Depression / Dysphoria  | 悲傷、哭泣、情緒低落、缺乏希望感           |
| 焦慮               | Anxiety                 | 分離焦慮、緊張、對未來過度擔憂             |
| 怡然自得／欣快感   | Elation / Euphoria      | 不恰當的愉悅、幼稚的開心或玩笑             |
| 冷漠／毫不在意     | Apathy / Indifference   | 對原有興趣失去動力、情緒表達減少           |
| 言行失控           | Disinhibition           | 衝動行為、不適切的言語或社交行為           |
| 暴躁易怒／情緒易變 | Irritability / Lability | 情緒起伏大、容易沮喪或發脾氣               |
| 異常動作（重複性） | Aberrant Motor Behavior | 重複性走動、翻找、把玩物品等動作           |

> NPI 亦有擴充版本（NPI-Q、NPI-NH）供快速篩檢或護理機構使用，本工具採用標準版評分方式。

## 評分方式

每個項目的評分分為兩個維度：

### 頻率（Frequency）

| 分數 | 描述 | 說明                 |
| ---- | ---- | -------------------- |
| 1    | 偶爾 | 每週少於一次         |
| 2    | 時常 | 每週約一次           |
| 3    | 經常 | 每週數次（但非每天） |
| 4    | 常常 | 每天或幾乎持續出現   |

### 嚴重性（Severity）

| 分數 | 描述 | 說明                           |
| ---- | ---- | ------------------------------ |
| 1    | 輕   | 輕微症狀，幾乎不造成困擾       |
| 2    | 中   | 中度症狀，對患者有顯著影響     |
| 3    | 重   | 嚴重症狀，明顯影響日常生活功能 |

### 計分公式

每個領域的分數 = **頻率 × 嚴重性**（最高 12 分）

**NPI 總分** = 各領域分數加總（最高 120 分）

## 嚴重度分級

| NPI 總分 | 嚴重程度 | 臨床建議                                  |
| -------- | -------- | ----------------------------------------- |
| 0        | 無症狀   | 無明顯行為精神症狀                        |
| 1–12     | 輕度     | 輕度 BPSD，持續觀察，考慮非藥物介入       |
| 13–36    | 中度     | 中度 BPSD，建議評估藥物或結構化非藥物介入 |
| ≥ 37     | 重度     | 重度 BPSD，建議積極治療與跨專業照護計劃   |

## 評估原則

- **由照顧者填答**：NPI 設計為訪談主要照顧者（家屬或長期照護人員），而非患者本人
- **評估時間範圍**：回顧過去**四週**內的行為表現
- **先判斷有無**：若照顧者確認有此症狀，再評估頻率與嚴重性；若無則跳過，該項計 0 分
- **以最差表現為準**：若症狀程度有變化，以最明顯的時期為準進行評分

## 臨床應用

NPI 在以下情境廣泛使用：

- **BPSD 嚴重度評估**：量化患者行為精神症狀，輔助診斷與分期
- **治療反應監測**：藥物（如 cholinesterase inhibitors、antipsychotics）或非藥物介入前後的比較
- **照顧者負擔評估**：部分版本（NPI-D）同時記錄照顧者的困擾程度
- **臨床試驗**：作為失智症介入研究的標準化結果指標
- **照護計劃制定**：依症狀類型與嚴重度，訂定個別化的行為管理策略

## 常見 BPSD 的非藥物介入方向

| 症狀           | 建議介入方式                       |
| -------------- | ---------------------------------- |
| 激動／攻擊     | 環境調整、音樂治療、避免觸發因子   |
| 憂鬱／冷漠     | 活動安排、社交刺激、回憶療法       |
| 妄想／幻覺     | 溝通技巧訓練、環境安全化、避免爭辯 |
| 遊走／重複動作 | 結構化活動、安全環境、感覺刺激     |
| 睡眠障礙       | 光照治療、日間活動增加、睡眠衛生   |

## 注意事項

::: warning 需由知情照顧者填答
NPI 的有效性高度依賴**知情照顧者的準確回報**。若照顧者與患者相處時間不足（每日少於 4 小時），評估結果可能低估實際症狀嚴重度。建議由最主要的日常照顧者進行訪談。
:::

::: warning 冷漠與憂鬱的鑑別
**冷漠**（Apathy）與**憂鬱**（Depression）在臨床上容易混淆，但治療方向不同。冷漠主要表現為動機與主動性降低，不一定伴隨悲傷情緒；憂鬱則以情緒低落為核心。NPI 將兩者分開評分，有助於區分。
:::

::: info 免責聲明
本計算器僅供臨床評分記錄與參考，不具診斷效力，亦不能取代完整的神經精神科評估。若患者出現明顯行為精神症狀影響日常生活，請諮詢神經內科、精神科或失智症專科醫療人員進行完整評估與治療規劃。
:::
