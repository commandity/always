## NIHSS 中風量表

**NIH Stroke Scale** 為急性中風嚴重度的國際標準量表，15 個項目（意識、凝視、視野、顏面、四肢肌力、感覺、語言、構音、忽略等），總分 0–42。

### NIHSS 分數解讀

| 總分  | 嚴重度   | 意義與建議處置                                   |
| ----- | -------- | ------------------------------------------------ |
| 0     | 無症狀   | 無神經學缺損                                     |
| 1–4   | 輕度中風 | 小中風；仍需完整病因評估與二級預防               |
| 5–15  | 中度中風 | 血栓溶解／取栓評估的核心族群                     |
| 16–20 | 中重度   | 密切監測，注意腦水腫與吞嚥安全                   |
| ≥ 21  | 重度中風 | 大範圍梗塞可能，評估加護照護與去顱骨減壓         |

> **NIHSS ≥ 6 常作為機械取栓的參考門檻**（合併大血管阻塞時）；分數需以「觀察到的表現」計分，不因失語或昏迷推測。連續追蹤時上升 ≥ 4 分視為神經學惡化。

## ABCD² TIA 風險評估

**ABCD² Score** 預估 TIA 後早期中風風險：年齡 ≥ 60（+1）、血壓 ≥ 140/90（+1）、單側無力（+2）／語言障礙無偏癱（+1）、持續 ≥ 60 分（+2）／10–59 分（+1）、糖尿病（+1），總分 0–7。

### ABCD² 分數解讀

| 總分 | 風險分層 | 2 日內中風風險 | 建議處置                         |
| ---- | -------- | -------------- | -------------------------------- |
| 0–3  | 低風險   | 約 1.0%        | 儘速門診評估（48 小時內）        |
| 4–5  | 中風險   | 約 4.1%        | 建議住院或快速通道 TIA 評估      |
| 6–7  | 高風險   | 約 8.1%        | 住院觀察，積極影像與病因學檢查   |

> **ABCD² ≥ 4 為住院評估的常用門檻**。近年指引強調：不論分數高低，TIA 均應在 24–48 小時內完成血管影像與心律評估——低分不能排除大血管狹窄或心房顫動。

## 注意事項

::: warning NIHSS 需訓練後施測
NIHSS 有標準化施測手冊與認證訓練，未受訓者計分變異大；本工具供記錄與教學參考。
:::

:::
