# UPDRS 巴金森症狀衡量表

**UPDRS Unified Parkinson's Disease Rating Scale**

## 概述

**統一巴金森病評量表**（Unified Parkinson's Disease Rating Scale，UPDRS）由 Fahn、Marsden、Goldstein 與 Calne 於 1987 年共同開發，是目前全球臨床與研究上最廣泛使用的巴金森病嚴重度評估工具。UPDRS 整合了認知、行為、日常功能與動作表現四大面向，能夠全面反映疾病對患者生活的影響，並追蹤病程變化及治療反應。

## 量表架構

UPDRS 共分為**四個部份**，涵蓋 39 個評估項目，總分最高 **147 分**：

| 部份     | 內容             | 題數                | 最高分 |
| -------- | ---------------- | ------------------- | ------ |
| 第一部份 | 智能、行為、情緒 | 4                   | 16     |
| 第二部份 | 日常生活能力     | 13                  | 52     |
| 第三部份 | 動作能力之檢查   | 14 項（含雙側細項） | 56     |
| 第四部份 | 治療之併發症     | 8                   | 23     |

> 第三部份由**臨床醫師**進行直接觀察與檢查評分，其餘部份可根據患者及照顧者的訪談資料進行評估。

## 各部份說明

### 第一部份：智能、行為、情緒

評估患者的認知功能、思想混亂程度、情緒狀態（包括沮喪傾向）及動機與主動性，反映疾病對神經精神面向的影響。

### 第二部份：日常生活能力

包含語言溝通、吞嚥、進食、穿衣、衛生清潔、翻身能力、走路、凍僵現象及感覺異常等日常功能，評估患者獨立生活的程度。

### 第三部份：動作能力之檢查

為 UPDRS 中最重要的客觀評估部份，涵蓋：

- **靜止型顫抖**：靜止狀態下的不自主震顫（臉部、四肢）
- **動作型／姿勢型顫抖**：動作中或維持姿勢時的顫抖
- **僵硬**：關節被動活動時的阻力（頸部、四肢）
- **動作遲緩**：手指打拍、手掌握合、前臂迴旋、腳部靈敏度
- **步態與姿勢**：起立、步態、姿勢平穩度、全身動作遲緩

### 第四部份：治療之併發症

評估長期使用多巴胺藥物（levodopa）後可能出現的副作用：

- **（A）異動症（Dyskinesias）**：藥物引起的不自主動作，評估持續時間、殘障程度及疼痛程度
- **（B）臨床藥波動（Clinical Fluctuations）**：包括可預測或不可預測的「無效期（Off period）」及其佔清醒時段的比例

## 嚴重度分級

UPDRS 總分與臨床嚴重度的對應關係（參考指引）：

| UPDRS 總分 | 嚴重程度 | 臨床描述                           |
| ---------- | -------- | ---------------------------------- |
| 0          | 正常     | 無巴金森症狀                       |
| 1–20       | 輕度障礙 | 症狀輕微，日常生活大致不受影響     |
| 21–50      | 中度障礙 | 明顯功能受限，部分活動需要協助     |
| 51–80      | 重度障礙 | 嚴重功能喪失，日常生活高度依賴他人 |
| > 80       | 極重度   | 幾乎完全依賴，行動嚴重受限         |

> 注意：UPDRS 各部份分數同樣具有獨立臨床意義，尤其**第三部份**分數常用於評估藥物療效與手術結果（如深部腦刺激術 DBS）。

## 臨床應用

UPDRS 在以下情境廣泛使用：

- **疾病嚴重度分期**：配合 Hoehn & Yahr 分期量表，全面評估病程
- **藥物療效評估**：比較服藥前（off state）與藥效高峰期（on state）的分數差異
- **手術適應症評估**：深部腦刺激術（DBS）術前評估的重要依據
- **病程追蹤**：定期施測，監測疾病進展速度
- **臨床試驗**：作為巴金森病研究的標準化評估工具
- **照護需求規劃**：依各部份分數評估患者所需的支持程度

## On / Off 狀態評估

第三部份（動作能力）建議分別在以下兩個狀態下施測：

- **Off state（無效期）**：距上次服藥至少 12 小時後，反映藥物基礎效果
- **On state（藥效期）**：藥效最高峰時，反映最佳運動功能

兩次分數的差異（**UPDRS Part III off − on**）可作為評估患者對多巴胺藥物反應程度的指標，通常改善幅度 ≥ 30% 視為有良好藥物反應。

## 注意事項

::: warning 第三部份須由臨床醫師執行
UPDRS 第三部份為**客觀動作檢查**，需由受訓的神經科或相關專科醫師親自執行，包括觀察靜止型顫抖、被動關節活動測試僵硬程度、以及姿勢平穩度測試（後拉測試）。本計算器提供評分記錄輔助，不能取代實際臨床檢查。
:::

::: warning 第四部份須根據實際服藥狀況評估
治療併發症的評估需結合患者的藥物使用史，包括 levodopa 的劑量、給藥頻率及症狀日誌。建議在門診追蹤時同步記錄患者的藥物日記（medication diary）。
:::

::: info 免責聲明
本計算器僅供臨床評分記錄與參考，不具診斷效力，亦不能取代完整的神經學評估。若患者出現巴金森病相關症狀（如靜止型顫抖、動作遲緩、肌肉僵硬、步態異常），請諮詢神經內科或動作障礙（Movement Disorder）專科醫師進行完整評估與診斷。
:::
