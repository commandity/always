# WPI 廣泛疼痛指數

**WPI Widespread Pain Index**

## 概述

**WPI**（Widespread Pain Index，廣泛疼痛指數）是 2010 年美國風濕病學會（ACR）修訂版纖維肌痛診斷標準的核心組成之一，由 Wolfe 等人發表。WPI 評估患者在過去一週內於 19 個特定身體部位的疼痛分布，與**症狀嚴重度量表**（SS Scale）合併使用，構成現代化的纖維肌痛診斷工具，取代舊版需要壓痛點檢查的 1990 年標準。

## 評分說明

WPI 共 **19 個身體部位**，分布於左側、右側及軀幹三個區域。每個部位若在**過去一週內**曾有疼痛，計 **1 分**，總分 **0–19 分**：

| 區域 | 部位                                             | 部位數 |
| ---- | ------------------------------------------------ | ------ |
| 左側 | 肩胛帶、上臂、前臂、髖部／臀部、顎部、大腿、小腿 | 7      |
| 右側 | 肩胛帶、上臂、前臂、髖部／臀部、顎部、大腿、小腿 | 7      |
| 軀幹 | 頸部、胸部、腹部、上背、下背                     | 5      |

> 疼痛的定義為**任何疼痛或酸痛感**，不限程度。即使是輕微的酸痛，只要在該部位出現，即計 1 分。

## 臨床切截點

| WPI 分數 | 評估結果 | 臨床意義                                   |
| -------- | -------- | ------------------------------------------ |
| 0–2      | 局部疼痛 | 疼痛分布局限，不符合廣泛性標準             |
| 3–6      | 中等廣泛 | 需配合 SS ≥ 9 才符合纖維肌痛診斷           |
| ≥ 7      | 廣泛疼痛 | 配合 SS ≥ 5 符合 2010 ACR 纖維肌痛診斷標準 |

> **關鍵切截值：WPI ≥ 7** 為廣泛疼痛的診斷門檻。需同時配合 SS 量表評估（見下方診斷標準）。

## 2010 ACR 纖維肌痛診斷標準

WPI 需與 SS 量表合併使用，同時滿足以下所有條件：

| 條件         | 標準                     |
| ------------ | ------------------------ |
| **方案一**   | WPI ≥ 7 **且** SS ≥ 5    |
| **方案二**   | WPI 3–6 **且** SS ≥ 9    |
| **持續時間** | 症狀持續 ≥ 3 個月        |
| **排除標準** | 無其他疾病可完全解釋症狀 |

> 纖維肌痛可與其他疾病**共存**（如類風濕性關節炎、系統性紅斑狼瘡），2010 年標準明確指出不應作為排除診斷使用。

## WPI 與舊版 1990 ACR 標準比較

|              | 1990 ACR 標準                 | 2010 ACR 標準（WPI + SS）     |
| ------------ | ----------------------------- | ----------------------------- |
| **評估方式** | 醫師壓痛點檢查（18 個壓痛點） | 患者自填問卷                  |
| **診斷門檻** | ≥ 11/18 個壓痛點陽性          | WPI + SS 分數組合             |
| **性別偏差** | 女性偏高（壓痛點敏感度差異）  | 較少性別偏差                  |
| **重複性**   | 受檢查者主觀影響              | 標準化，可重複施測            |
| **症狀涵蓋** | 僅疼痛                        | 疼痛 + 疲倦 + 認知 + 全身症狀 |
| **適用場合** | 需醫師操作                    | 門診、研究、自我監測均可      |

## 纖維肌痛的廣泛疼痛特徵

纖維肌痛的疼痛具有以下典型特徵，有助於與其他疾病鑑別：

| 特徵     | 說明                                              |
| -------- | ------------------------------------------------- |
| 廣泛性   | 同時涉及身體多個部位，常為雙側                    |
| 遊走性   | 疼痛部位可隨時間變化                              |
| 變動性   | 疼痛強度隨壓力、睡眠、天氣波動                    |
| 異常疼痛 | 對正常刺激（如輕觸）產生過度疼痛反應（Allodynia） |
| 痛覺過敏 | 對疼痛刺激反應增強（Hyperalgesia）                |
| 合併症狀 | 常伴隨疲倦、睡眠障礙、腦霧、情緒問題              |

## 鑑別診斷

WPI 高分（廣泛多部位疼痛）需排除以下疾病：

| 疾病                  | 鑑別要點                                    |
| --------------------- | ------------------------------------------- |
| 類風濕性關節炎（RA）  | 關節腫脹、晨僵 > 1 小時、RF / anti-CCP 陽性 |
| 系統性紅斑狼瘡（SLE） | 蝶形紅斑、ANA 陽性、多系統受累              |
| 多發性肌炎 / 皮肌炎   | 肌肉無力、CK 升高、EMG 異常                 |
| 甲狀腺功能低下        | TSH 升高、怕冷、體重增加                    |
| 維生素 D 缺乏         | 25-OH VitD 降低，補充後改善                 |
| 睡眠呼吸中止症        | 打鼾、日間嗜睡，PSG 確診                    |

## 注意事項

::: warning WPI 需與 SS 量表合併使用才能診斷
WPI 單獨分數無法診斷纖維肌痛，必須配合症狀嚴重度量表（SS Scale）合併評估。WPI ≥ 7 僅代表疼痛分布廣泛，診斷仍需由風濕科或疼痛科醫師根據完整病史及檢查確認。
:::

::: warning 需排除其他可解釋症狀的疾病
廣泛性疼痛可由多種疾病引起。進行 WPI 評估前，應先完成基本血液檢查（CBC、CMP、TSH、ESR、CRP、ANA）以排除其他病因。WPI 高分合併炎症指標升高時，需優先排除自體免疫疾病。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不具診斷效力。纖維肌痛的診斷及治療應由風濕科、疼痛科或家庭醫學科醫師根據完整病史、身體檢查及相關評估綜合判斷。
:::
