# 腕隧道症候群評估工具

**Carpal Tunnel Syndrome · Katz/D'Arcy 診斷模型 & Wainner CPR**

<CtsAssessment />

## 診斷模型（Katz・D'Arcy）

第一個分頁整合**雙變量模型**（Flick 徵 ＋ Katz 手圖）與**多變量模型**（年齡、夜間症狀、感覺分布等），在神經傳導檢查（NCS）前估算 CTS 的臨床機率。

| 模型結果       | 意義                                     |
| -------------- | ---------------------------------------- |
| 低機率         | 考慮其他病因（頸神經根、近端壓迫、多發神經病變） |
| 中機率         | 安排 NCS／超音波確認                     |
| 高機率         | 臨床即可高度懷疑；NCS 供嚴重度分級與術前佐證 |

> **Flick 徵**（甩手緩解）LR+ ≈ 21 為單一最強的病史線索；**Katz 手圖 classic/probable 型**特異度高。兩者皆陽性時檢後機率大幅上升。

## Wainner CPR

第二個分頁為 **Wainner 2005 臨床預測規則**，五項標準：Flick 徵、腕比值（wrist ratio）> 0.67、SSS 症狀嚴重度 > 1.9、正中神經感覺減退（拇指）、年齡 > 45 歲。

| 符合項數 | LR+  | 檢後機率（盛行率 34%） | 意義                     |
| -------- | ---- | ---------------------- | ------------------------ |
| ≥ 2      | 1.0  | ≈ 34%                  | 無鑑別力                 |
| ≥ 3      | 2.1  | ≈ 52%                  | 輕度提高                 |
| ≥ 4      | 4.6  | ≈ 70%                  | 中度提高，建議 NCS       |
| 5        | 18.3 | ≈ 90%                  | 高度可能，可逕行治療評估 |

> **兩個工具的分工**：Katz/D'Arcy 模型偏重病史與症狀分布，適合初診快速分流；Wainner CPR 納入理學檢查與量表，適合復健／疼痛門診的結構化評估。兩者結論一致時，臨床把握度最高。

## 注意事項

::: warning 不能取代電生理檢查
臨床決策樹用於估算檢前機率；手術前確診與嚴重度分級仍以 NCS/EMG 為準，並排除頸椎神經根病變等模仿疾病。
:::

::: info 免責聲明
本計算器僅供臨床輔助參考，治療決策請諮詢神經科、復健科或手外科醫師。
:::
