# 失能評估工具

**Disability Assessment · Oswestry Disability Index & Roland-Morris Questionnaire**

<DisabilityCalculator />

## Oswestry 失能指數（ODI）

10 個面向（疼痛強度、自我照顧、提舉、行走、坐、站、睡眠、性生活、社交、旅行），每題 0–5 分；總分換算為 0–100%：

```
ODI (%) = 得分總和 ÷ (作答題數 × 5) × 100
```

### ODI 分數解讀

| ODI     | 障礙程度 | 意義與建議處置                         |
| ------- | -------- | -------------------------------------- |
| 0–20%   | 輕度     | 多數活動可進行；自我管理、運動衛教     |
| 21–40%  | 中度     | 日常活動顯著受限；物理治療、疼痛管理   |
| 41–60%  | 重度     | 生活各面向受影響；多學科介入           |
| 61–80%  | 極重度   | 廣泛功能喪失；外科評估                 |
| 81–100% | 臥床級   | 完全依賴（或症狀誇大）；住院與心理評估 |

> **MCID 約 10–12 個百分點**（或基線值的 30%）——手術與復健文獻最常引用的療效標準。未作答題目（如性生活）按公式自動剔除分母，不影響效度。

## Roland-Morris 失能問卷（RMDQ）

24 題「今天是否因腰痛而…」是/否勾選，總分 0–24，聚焦身體功能，對**輕中度失能**變化最敏感。

### RMDQ 分數解讀

| 總分  | 障礙程度 | 意義與建議處置               |
| ----- | -------- | ---------------------------- |
| 0–4   | 輕度     | 自我管理、姿勢衛教、核心運動 |
| 5–8   | 中度     | 物理治療 4–6 週              |
| 9–14  | 中重度   | 多學科介入、心理社會評估     |
| 15–24 | 重度     | 積極多學科治療               |

> **兩工具的分工**：RMDQ 在輕中度族群的反應性較佳（適合基層與復健追蹤）；ODI 涵蓋面向較廣、天花板較高，適合重度失能與手術前後比較。RMDQ 的 MCID 約 **3.5–5 分**，同一患者追蹤請固定使用同一工具。

## 注意事項

::: warning 心理社會因素（黃旗）
失能分數持續偏高而客觀病灶輕微時，應評估恐懼迴避、災難化思考等黃旗因子，考慮認知行為取向的復健。
:::

::: info 免責聲明
本計算器僅供臨床評分記錄與教育參考，治療決策請諮詢復健科或疼痛科醫師。
:::
