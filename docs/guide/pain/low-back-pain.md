# Low Back Pain Evaluation 腰痛評估決策樹

**Low Back Pain Evaluation Clinical Decision Tree**

## 概述

**腰痛評估決策樹**（LBP TreeCalc）是依據臨床指引（NICE, ACOEM）設計的三步驟分類工具，協助臨床醫師快速將腰痛患者分類為**嚴重特異性病因**、**神經根性腰痛**或**非特異性腰痛**，指引後續評估與治療決策。

## 腰痛三分類模型

| 類別               | 比例      | 代表性病因                   | 處置原則                    |
| ------------------ | --------- | ---------------------------- | --------------------------- |
| **嚴重特異性病因** | 約 1–5%   | 骨折、腫瘤、感染、馬尾症候群 | 緊急影像學及轉介            |
| **神經根性腰痛**   | 約 5–10%  | 椎間盤突出、椎管狹窄         | 保守治療 4–6 週，必要時 MRI |
| **非特異性腰痛**   | 約 85–90% | 肌肉韌帶拉傷、功能性疼痛     | 積極自我管理，不鼓勵臥床    |

## 紅旗徵象（Red Flags）

| 紅旗                                 | 可疑診斷   | 緊急度        |
| ------------------------------------ | ---------- | ------------- |
| 大小便失禁、鞍區感覺喪失             | 馬尾症候群 | **外科急症**  |
| 有癌症病史 + 新發腰痛                | 脊椎轉移   | 立即影像學    |
| 持續發燒 + 腰痛 + 靜脈注射藥物使用史 | 硬膜外膿腫 | 立即評估      |
| 外傷後嚴重腰痛（骨質疏鬆患者）       | 椎體骨折   | X 光優先      |
| 首次腰痛年齡 < 20 或 > 55 歲         | 特殊病因   | 影像學排除    |
| 非機械性疼痛（休息時不緩解）         | 腫瘤或感染 | 血液 + 影像學 |

## 神經根性腰痛診斷標準

**L4 神經根：**

- 感覺：小腿內側
- 運動：脛前肌（踝背屈）
- 反射：膝反射↓

**L5 神經根（最常見）：**

- 感覺：足背、大拇趾
- 運動：拇趾伸肌（大拇趾背屈）
- 反射：無特定

**S1 神經根：**

- 感覺：足外側、小趾
- 運動：腓腸肌（足蹠屈）
- 反射：踝反射↓

## 黃旗心理社會因素

| 黃旗         | 評估工具                          | 慢性化風險 |
| ------------ | --------------------------------- | ---------- |
| 災難化思考   | PCS（Pain Catastrophizing Scale） | 高         |
| 恐懼迴避行為 | FABQ                              | 高         |
| 憂鬱/焦慮    | PHQ-9、GAD-7                      | 高         |
| 低工作滿意度 | 問診                              | 中         |

## 急性、亞急性、慢性腰痛處置比較

| 類型              | 時間 | 第一線治療             | 進一步評估       |
| ----------------- | ---- | ---------------------- | ---------------- |
| 急性（< 6 週）    | —    | 維持活動、NSAIDs       | 僅紅旗者需影像學 |
| 亞急性（6–12 週） | —    | 物理治療、認知行為治療 | 黃旗評估         |
| 慢性（> 12 週）   | —    | 多學科介入、運動治療   | 考慮介入性治療   |

::: warning 影像學的適當使用
無紅旗徵象的急性非特異性腰痛**不建議**常規影像學檢查。過早進行 MRI 可能發現無症狀的退化性變化，導致過度醫療化（over-medicalization）和不必要的焦慮，反而不利於患者恢復。
:::

::: warning 紅旗徵象需優先排除，不可延誤
發現任何一項紅旗徵象均需立即進一步評估。馬尾症候群（大小便失禁 + 鞍區麻木）為神經外科急症，需在數小時內完成 MRI 並外科評估，延誤可能造成永久性神經損傷。
:::

::: warning 黃旗因素是腰痛慢性化的主要預測因子
心理社會因素（災難化思考、恐懼迴避行為、憂鬱）比影像學發現更能預測慢性腰痛的發展。即使影像學正常，黃旗因素多者仍需積極介入，包括認知行為治療和疼痛教育。
:::

::: info 免責聲明
本工具僅供臨床分類參考，不能取代完整的病史詢問、理學檢查及必要的影像學評估。腰痛的診斷與治療決策應由具執照的醫師或治療師根據患者完整臨床狀況判斷。
:::
