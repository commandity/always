# 偏頭痛評估工具

**Migraine Screening & ICHD-3 Diagnostic Criteria**

<MigraineAssessment />

## ID Migraine 篩查

三題快速篩查（過去三個月）：頭痛伴噁心、畏光、活動受限（失能）。

| 陽性題數 | 結果         | 意義                                       |
| -------- | ------------ | ------------------------------------------ |
| 0–1      | 篩查陰性     | 偏頭痛可能性低，考慮其他頭痛類型           |
| ≥ 2      | 篩查陽性     | 偏頭痛可能性高（PPV 約 93%），進入 ICHD-3 確認 |

> 篩查前務必先以 **SNOOP 紅旗**排除繼發性頭痛（全身症狀、神經學異常、突發雷擊、50 歲後新發、型態改變）。

## ICHD-3 診斷標準

**無先兆偏頭痛（1.1）**：≥ 5 次發作；持續 4–72 小時；具備單側、搏動性、中重度、活動加劇中至少 2 項；伴噁心嘔吐或畏光畏聲。
**有先兆偏頭痛（1.2）**：≥ 2 次發作；可逆性先兆（視覺、感覺、語言等）持續 5–60 分鐘，隨後 60 分鐘內頭痛。

> 工具會依勾選項目自動比對各條標準的達成狀況；**每月頭痛 ≥ 15 天且其中 ≥ 8 天符合偏頭痛特徵、持續 3 個月**則屬慢性偏頭痛（1.3），為預防性治療的明確適應症。

## 注意事項

::: warning 紅旗症狀優先
任何 SNOOP 紅旗陽性應先安排影像或進一步檢查，不可直接套用偏頭痛診斷。
:::

本計算器僅供臨床輔助參考，頭痛診斷與治療請諮詢神經科醫師。
:::
