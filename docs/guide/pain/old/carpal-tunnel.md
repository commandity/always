# Carpal Tunnel Syndrome 腕隧道症候群臨床決策樹

**Carpal Tunnel Syndrome Clinical Decision Tree**

## 概述

**腕隧道症候群（Carpal Tunnel Syndrome, CTS）臨床決策樹**基於 Wainner 等人（2005）發表的**臨床預測規則**（Clinical Prediction Rule, CPR），使用特定臨床發現評估 CTS 診斷可能性，有助於在電生理檢查（NCS/EMG）前進行初步分類。

## 腕隧道症候群概述

CTS 是最常見的**周邊神經壓迫症候群**，正中神經在腕橫韌帶（Flexor Retinaculum）下方受到壓迫。

**盛行率：** 3–6%，女性：男性 ≈ 3：1
**好發年齡：** 40–60 歲
**危險因子：** 重複性手部動作、振動工具、懷孕、糖尿病、甲狀腺功能低下、類風濕性關節炎

## 典型症狀

**感覺分佈（正中神經支配範圍）：**

- 拇指、食指、中指的掌側
- 無名指橈側半
- **注意：** 手掌近端感覺通常正常（掌皮支不通過腕隧道）

**時間特性：**

- 夜間或清晨麻木、刺痛（常於睡眠中痛醒）
- 長時間使用手機、開車時加重
- 搖晃或甩動手部可暫時緩解（「flick sign」或 Shaking Sign）

**晚期表現：**

- 拇短展肌（APB）萎縮
- 大魚際肌萎縮
- 精細動作困難（扣鈕扣、持筷）

## Wainner 臨床預測規則詳解

### 二元變量法

兩項具最高陽性似然比的臨床發現：

| 發現                | +LR | 陰性時 −LR |
| ------------------- | --- | ---------- |
| 腕部比例指數 < 0.67 | 3.1 | 0.6        |
| Shaking Sign 陽性   | 2.5 | 0.4        |

### 多元變量法（5 項 CPR）

| 項目                          | +LR |
| ----------------------------- | --- |
| 症狀因手部動作加重            | 1.6 |
| 腕部比例指數 < 0.67           | 3.1 |
| CTS 問卷 ≥ 1.9 分             | 2.5 |
| 正中神經叩擊試驗（Tinel）陽性 | 2.0 |
| 年齡 > 45 歲                  | 1.8 |

**Wainner CPR 診斷準確度：**

| 陽性項目數 | 診斷可能性 |
| ---------- | ---------- |
| ≥ 4        | ≈ 83%      |
| 3          | ≈ 60%      |
| 2          | ≈ 40%      |
| ≤ 1        | ≈ 20%      |

## 常用臨床測試彙整

| 測試               | 敏感性 | 特異性 | 操作說明                            |
| ------------------ | ------ | ------ | ----------------------------------- |
| **Phalen 試驗**    | 75%    | 47%    | 雙腕完全屈曲 60 秒，誘發麻木 = 陽性 |
| **Tinel 叩擊試驗** | 60%    | 67%    | 叩擊腕橫韌帶，誘發放電感 = 陽性     |
| **腕部壓迫試驗**   | 64%    | 83%    | 壓迫腕橫韌帶 30 秒，誘發症狀 = 陽性 |
| **拇短展肌萎縮**   | 23%    | 98%    | 大魚際肌萎縮，特異性高但敏感性低    |
| **Shaking Sign**   | 37%    | 74%    | 搖晃手部可緩解症狀                  |

## 治療選擇

### 保守治療

| 治療                         | 適應症                 | 證據等級 |
| ---------------------------- | ---------------------- | -------- |
| **夜間腕部副木（中立位）**   | 輕至中度 CTS 首選      | I 級     |
| **類固醇局部注射**           | 保守治療無效或急性加重 | I 級     |
| **非類固醇消炎藥（NSAIDs）** | 急性期症狀控制         | II 級    |
| **運動治療（神經滑動）**     | 輕度 CTS               | II 級    |
| **工作人體工學調整**         | 職業性 CTS             | II 級    |

### 手術治療

**適應症：** 保守治療 3–6 個月無效、電生理確認中重度 CTS、大魚際肌萎縮（晚期）

| 術式         | 優點                 | 恢復時間 |
| ------------ | -------------------- | -------- |
| 開放式鬆解術 | 技術成熟，直視下操作 | 4–6 週   |
| 內視鏡鬆解術 | 切口小，恢復快       | 2–4 週   |

## 電生理確認（NCS/EMG）

| 發現                            | 意義         |
| ------------------------------- | ------------ |
| 正中神經感覺傳導速度↓（最敏感） | 早期病變     |
| 正中神經遠端運動潛伏期延長      | 中度以上     |
| 拇短展肌去神經電位              | 嚴重軸突損傷 |

**NCS 診斷標準（AANEM）：**

- 正中感覺傳導速度 < 50 m/s（參考對側）
- 正中感覺潛伏期 − 尺神經感覺潛伏期 > 0.5 ms

::: warning Wainner CPR 為輔助工具，不能單獨確診 CTS
臨床預測規則的陽性率（≈ 83% at ≥ 4 項）仍非確定性診斷。在考慮手術之前，電生理檢查（NCS/EMG）仍為建議的確診工具，尤其是症狀不典型、症狀較輕或考慮手術的患者。
:::

::: warning 需鑑別頸椎神經根病變（C6/C7）
CTS 與 C6/C7 頸椎神經根病變的症狀高度重疊（均可造成手指麻木）。鑑別要點：頸椎病變常伴有頸部疼痛和近端症狀，Spurling 試驗陽性；CTS 則麻木集中於正中神經分佈，夜間加重明顯，Phalen/Tinel 陽性。兩者可合併存在（Double Crush Syndrome）。
:::

::: warning 大魚際肌萎縮代表嚴重神經損傷，需緊急評估
出現拇短展肌（APB）萎縮代表正中神經已有顯著軸突損傷，此時保守治療效果有限，需優先安排電生理檢查並儘早外科評估，以防止神經功能進一步喪失。
:::

::: info 免責聲明
本工具僅供 CTS 診斷可能性的初步評估參考，不能取代完整的神經學檢查及電生理確認。CTS 的診斷與治療決策應由具執照的醫師根據患者完整臨床狀況判斷。
:::
