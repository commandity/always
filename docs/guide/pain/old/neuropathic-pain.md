# Neuropathic Pain Scale 神經性疼痛量表

**Neuropathic Pain Scale**

## 概述

**神經性疼痛量表**（Neuropathic Pain Scale, NPS）由 Galer & Jensen 於 1997 年發展，是第一個專門針對**神經性疼痛**（Neuropathic Pain）多維度感覺特性設計的評估工具，能量化不同類型感覺異常的嚴重程度，有助於選擇針對性治療。

## 量表結構

共 10 項，每項 0–10 分（0 = 無此特性，10 = 最嚴重），總分 0–100。

| 項目 | 疼痛特性              | 神經病理基礎                  |
| ---- | --------------------- | ----------------------------- |
| Q1   | 整體疼痛強度          | 傷害感受器整體激活程度        |
| Q2   | 疼痛不愉快程度        | 情感-動機維度                 |
| Q3   | 銳痛/刺痛感           | Aδ 纖維激活                   |
| Q4   | 熱燒灼感              | C 纖維激活（TRPV1 通道）      |
| Q5   | 冷痛感                | TRPM8 通道、A 纖維            |
| Q6   | 觸覺敏感（Allodynia） | 中樞敏感化                    |
| Q7   | 搔癢感                | 組織胺依賴型 C 纖維           |
| Q8   | 持續性疼痛            | 自發性放電、持續性 C 纖維激活 |
| Q9   | 陣發性疼痛            | 電壓門控鈉離子通道異常        |
| Q10  | 深層/表淺             | 疼痛空間特性                  |

## 嚴重度分類

| NPS 總分 | 嚴重度           |
| -------- | ---------------- |
| 0–20     | 輕度神經性疼痛   |
| 21–40    | 中度神經性疼痛   |
| 41–60    | 中重度神經性疼痛 |
| > 60     | 重度神經性疼痛   |

## 依 NPS 特徵選擇治療

| 主要特徵                   | 首選藥物                     | 輔助藥物                    |
| -------------------------- | ---------------------------- | --------------------------- |
| **燒灼痛（Q4 高）**        | SNRIs（Duloxetine 60–120mg） | TCAs                        |
| **刺痛/陣發（Q3、Q9 高）** | Gabapentin、Pregabalin       | Carbamazepine（三叉神經痛） |
| **觸覺敏感（Q6 高）**      | Lidocaine 5% 貼片            | Capsaicin 貼片、外用 NSAIDs |
| **冷痛（Q5 高）**          | TCAs（Amitriptyline）        | SNRI                        |
| **持續性廣泛痛（Q8 高）**  | TCAs、SNRIs                  | Tramadol                    |
| **睡眠受影響**             | TCAs（晚上給藥）             | Pregabalin（晚間劑量較大）  |

## NPS 在臨床試驗中的應用

NPS 能區分**特定感覺特徵對治療的反應**，例如：

- Capsaicin 對 Q4（燒灼）和 Q6（觸覺敏感）特別有效
- Gabapentin 對 Q9（陣發性）特別有效
- SNRIs 對 Q2（不愉快程度）和 Q4（燒灼）有效

## 神經性疼痛診斷標準（NeuPSIG 2016）

```
可能（Possible） ：病史及症狀符合神經性疼痛分佈
很可能（Probable）：+ 感覺測試異常（感覺喪失或過敏）
確定（Definite） ：+ 影像學或電生理確認神經損傷
```

::: warning NPS 是主觀自評工具，無法單獨確診神經性疼痛
NPS 反映患者的主觀感覺特性，高分並不等同於神經性疼痛診斷。中樞敏感化、纖維肌痛症或心理因素均可能造成多個項目高分。診斷神經性疼痛需結合明確的神經損傷病史、感覺測試及必要時的電生理或影像學確認。
:::

::: warning 不同神經性疼痛的感覺特徵可能重疊
NPS 特徵模式能提供治療方向的線索，但不同疾病的感覺特徵常有重疊。例如高 Q4（燒灼）和高 Q6（觸覺敏感）可見於多種神經性疼痛。建議將 NPS 結果作為治療選擇的輔助參考，而非替代完整的神經學評估。
:::

::: warning 治療反應評估需定期重複施測
NPS 的臨床價值在於**縱向追蹤**治療前後各感覺維度的變化，而非單次評估。建議在啟動治療前完成基礎評估，治療 4–8 週後再次施測，比較各項目分數的變化以評估療效。
:::

::: info 免責聲明
本工具僅供疼痛特性評估參考，不能取代完整的神經學檢查及必要的電生理或影像學評估。神經性疼痛的診斷與治療決策應由具執照的醫師根據患者完整臨床狀況判斷。
:::
