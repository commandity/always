# Roland-Morris 失能問卷

**Roland-Morris Disability Questionnaire**

## 概述

**Roland-Morris Disability Questionnaire**（RMDQ）由 Roland & Morris 於 1983 年發展，是全球使用最廣泛的腰痛功能障礙評估患者報告工具（Patient-Reported Outcome Measure, PROM）之一，共 24 題，患者自填，5 分鐘完成。

## 計分方式

每題為是/否選擇：患者勾選所有**今天**符合自身狀況的項目。

- **勾選（是）= 1 分**
- **未勾選（否）= 0 分**
- **總分範圍：0–24 分**（分數越高 = 障礙越嚴重）

## 嚴重度分類

| 分數      | 障礙程度            | 臨床建議                     |
| --------- | ------------------- | ---------------------------- |
| **0–4**   | 輕度（Minimal）     | 自我管理、姿勢衛教、核心運動 |
| **5–8**   | 中度（Moderate）    | 物理治療 4–6 週              |
| **9–14**  | 中重度（Severe）    | 多學科介入、心理評估         |
| **15–24** | 重度（Very Severe） | 積極多學科治療               |

## 最小臨床意義差異（MCID）

| 族群                 | MCID   |
| -------------------- | ------ |
| 急性腰痛（< 6 週）   | 2–3 分 |
| 慢性腰痛（> 3 個月） | 4–5 分 |
| 外科手術後           | 5 分   |

MCID 代表患者感受到明確改善所需的最小分數變化，低於此值的改變可能僅為測量誤差。

## 24 個問題的涵蓋面向

RMDQ 的 24 個項目改編自 Sickness Impact Profile（SIP），涵蓋：

| 面向     | 題數 | 代表項目                   |
| -------- | ---- | -------------------------- |
| 整體行動 | 6    | 走路速度、爬樓梯、行走距離 |
| 自我照護 | 4    | 穿衣、沐浴、穿鞋           |
| 坐立耐受 | 3    | 久坐、久站、需扶物起身     |
| 睡眠     | 1    | 睡眠受干擾                 |
| 休息     | 1    | 需頻繁躺下                 |
| 情緒     | 1    | 易怒、情緒不穩             |
| 家務     | 2    | 家務能力                   |
| 社交     | 1    | 整天待在家                 |

## RMDQ vs ODI 適用場景

| 特性                   | RMDQ   | ODI       |
| ---------------------- | ------ | --------- |
| **敏感度（輕度障礙）** | ✓ 較高 | 中等      |
| **區分度（重度障礙）** | 中等   | ✓ 較高    |
| **完成時間**           | 5 分鐘 | 5–10 分鐘 |
| **研究標準工具**       | ✓      | ✓         |
| **手術前後評估**       | ✓      | ✓         |

## 信效度

- **內部一致性**：Cronbach's α = 0.86–0.93
- **再測信度**：ICC = 0.83–0.91
- **構念效度**：與 VAS 疼痛、ODI 高度相關（r = 0.70–0.85）

::: warning RMDQ 分數易受非腰痛因素影響
RMDQ 完全依賴患者當天主觀感受，合併纖維肌痛症、憂鬱、睡眠障礙或其他慢性疼痛的患者可能使分數偏高。建議結合客觀功能評估（如步行測試、6 分鐘步行）來提高評估準確性。
:::

::: warning RMDQ 對重度障礙的區分度有限
RMDQ 在輕至中度障礙患者中敏感度高，但分數接近上限（> 18 分）時區分度下降。重度障礙或手術適應症評估時建議優先使用 ODI，其對高分段的分辨力更佳。
:::

::: warning 追蹤時請固定使用同一工具
RMDQ（0–24）和 ODI（0–100%）的分數範圍與切截值完全不同，在同一患者的縱向追蹤中應固定使用同一工具，避免混用造成判斷錯誤。
:::

::: info 免責聲明
本工具僅供臨床功能評估參考，不能取代完整的臨床評估與影像學檢查。腰痛的診斷與治療決策應由具執照的醫師或物理治療師根據患者完整臨床狀況判斷。
:::
