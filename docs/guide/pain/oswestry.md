# Oswestry 失能指數

**Oswestry Disability Index**

## 概述

**Oswestry Disability Index**（ODI）由 Fairbank 等人於 1980 年首次發表，2000 年修訂為 Version 2.1，是評估**腰痛功能障礙**最廣泛使用的患者報告工具，被稱為**腰痛研究的黃金標準**（gold standard），廣泛用於臨床評估、手術決策及治療效果追蹤。

## 計分方式

**10 個項目 × 6 個選項（0–5 分）= 最高 50 分**

```
ODI% = 原始分數 ÷ ( 5 × 完成項目數 ) × 100%
```

若 S8（性生活）跳過，以可作答的 9 項計算分母（= 45）。

## 嚴重度分類

| ODI%        | 障礙程度            | 臨床描述                       | 建議               |
| ----------- | ------------------- | ------------------------------ | ------------------ |
| **0–20%**   | 輕度（Minimal）     | 大多數活動可進行，無需特殊治療 | 自我管理、運動衛教 |
| **21–40%**  | 中度（Moderate）    | 日常活動顯著受限，坐立困難     | 物理治療、疼痛管理 |
| **41–60%**  | 重度（Severe）      | 影響生活各面向，詳細評估必要   | 多學科介入         |
| **61–80%**  | 極重度（Crippling） | 廣泛生活功能喪失               | 外科評估           |
| **81–100%** | 臥床（Bed-Bound）   | 完全依賴或誇大                 | 住院評估、心理評估 |

## 10 個評估面向的臨床意義

| 項目            | 面向         | 為何重要             |
| --------------- | ------------ | -------------------- |
| **S1 疼痛強度** | 主觀感受     | 區分不同鎮痛藥需求   |
| **S2 個人照護** | ADL 基本功能 | 是否需要居家照護     |
| **S3 提重物**   | 上肢功能     | 職業適應評估         |
| **S4 步行**     | 移動能力     | 評估行走輔具需求     |
| **S5 坐**       | 靜態耐受     | 辦公室工作能力       |
| **S6 站立**     | 靜態耐受     | 站立工作能力         |
| **S7 睡眠**     | 睡眠品質     | 疼痛對恢復的影響     |
| **S8 性生活**   | 生活品質     | 整體生活功能（可略） |
| **S9 社交生活** | 社會參與     | 心理社會功能         |
| **S10 旅行**    | 移動能力     | 廣義移動功能         |

## 最小臨床意義差異（MCID）

| 情境             | MCID       | 說明                 |
| ---------------- | ---------- | -------------------- |
| 急性腰痛治療     | 10% ODI    | 代表患者感受明確改善 |
| 慢性腰痛保守治療 | 12–15% ODI | —                    |
| 脊椎手術後評估   | 15% ODI    | 術後療效評估標準     |

## 手術決策參考

| 情境             | ODI 參考值         | 說明           |
| ---------------- | ------------------ | -------------- |
| 椎間盤手術考量   | > 40% + 神經根症狀 | 保守治療失敗後 |
| 強力手術指徵     | > 60% + 影像學確認 | —              |
| 一般保守治療優先 | < 40%              | —              |
| 術後成功定義     | 改善 ≥ 15% ODI     | —              |

## ODI vs RMDQ 選擇指南

| 臨床情境             | 推薦工具 | 原因                 |
| -------------------- | -------- | -------------------- |
| 輕至中度腰痛門診追蹤 | RMDQ     | 對輕度障礙敏感度高   |
| 手術前後評估         | ODI      | 重度障礙區分度佳     |
| 研究需要標準化比較   | ODI      | 更廣泛的跨研究資料庫 |
| 重複頻繁評估         | RMDQ     | 完成時間較短         |

## ODI 版本差異

| 版本             | 主要差異                          |
| ---------------- | --------------------------------- |
| ODI v1（1980）   | 原始版本，S8 問「性生活及社交」   |
| ODI v2（1989）   | S8 分拆為性生活（S8）和社交（S9） |
| ODI v2.1（2000） | 措辭修訂，為目前國際標準版本      |

本計算器採用 **ODI Version 1** 結構，S8 詢問性生活，S9 詢問社交生活。

## 信效度

- **內部一致性**：Cronbach's α = 0.71–0.87
- **再測信度**：ICC = 0.83–0.99
- **對應效度**：與 SF-36 身體功能維度高度相關（r = −0.70 至 −0.85）

::: warning ODI 完全依賴患者主觀自評，可能受非器質性因素影響
疼痛放大（pain catastrophizing）、纖維肌痛症合併症、憂鬱或工傷補償動機均可能使 ODI 分數偏高。分數在 81–100% 的患者需特別評估是否有非器質性腰痛（non-organic back pain）的 Waddell 徵象，必要時配合心理評估。
:::

::: warning ODI > 60% 應積極排除手術適應症，但不可單憑 ODI 決策
ODI > 60% 是考慮外科評估的閾值，但手術決策必須同時結合影像學確認（MRI/CT 顯示結構壓迫）、神經學症狀（肌力下降、反射異常）及保守治療失敗的紀錄。單憑高 ODI 分數不構成手術指徵。
:::

::: warning 追蹤時應固定使用同一 ODI 版本
ODI v1 和 v2.1 的 S8 題目設計不同，在同一患者的縱向追蹤中應固定使用同一版本，切勿混用。跨研究比較時也需注意版本差異對結果的影響。
:::

::: info 免責聲明
本計算器僅供臨床功能障礙評估參考，不能取代完整的臨床評估與影像學檢查。腰痛的診斷、手術適應症評估及治療決策應由具執照的醫師根據患者完整臨床狀況及個別因素綜合判斷。
:::
