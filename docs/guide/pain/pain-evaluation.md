## LBP 腰痛評估決策樹

第一個分頁依「紅旗 → 神經根症狀 → 非特異性」三層分流，對應腰痛的三大類病因：

| 類別           | 比例    | 代表病因                     | 處置原則                      |
| -------------- | ------- | ---------------------------- | ----------------------------- |
| 嚴重特異性病因 | 1–5%    | 骨折、腫瘤、感染、馬尾症候群 | 緊急影像與轉介                |
| 神經根性腰痛   | 5–10%   | 椎間盤突出、椎管狹窄         | 保守治療 4–6 週，必要時 MRI   |
| 非特異性腰痛   | 85–90%  | 肌肉韌帶拉傷、功能性疼痛     | 積極自我管理，不鼓勵臥床      |

> **紅旗優先**：大小便失禁＋鞍區麻木＝馬尾症候群（外科急症）；夜間痛＋體重減輕提示腫瘤；發燒＋注射藥物史提示感染。無紅旗、無神經根症狀者，**前 4–6 週不需影像檢查**——早期 MRI 與較差預後相關。

## 神經性疼痛量表（NPS）

第二個分頁為 10 題 0–10 評分，各題對應不同的神經病理機轉（銳痛＝Aδ 纖維、灼熱＝C 纖維/TRPV1、觸誘發痛＝中樞敏感化、深部/表淺痛區分等）。

| 結果模式               | 意義                                       |
| ---------------------- | ------------------------------------------ |
| 灼熱、電擊、觸誘發痛高分 | 神經性疼痛特徵明顯，考慮抗癲癇藥/TCA/SNRI |
| 鈍痛、深部痛為主       | 傷害感受性疼痛可能性較高，NSAID 導向       |
| 混合模式               | 常見於慢性腰腿痛，需複合治療策略           |

> NPS 不是診斷工具，而是**疼痛表型分析**：各分項分數描繪疼痛「性質指紋」，用於選藥（neuropathic 成分 → gabapentinoids/duloxetine）與追蹤治療反應；總分下降 ≥ 20% 視為有意義改善。

## 注意事項

::: warning 紅旗症狀請立即就醫
本工具的分流建議不能取代急診評估；任何馬尾症候群徵象應於數小時內處理。
:::

::: info 免責聲明
本計算器僅供臨床輔助參考，疼痛診療請諮詢疼痛科、復健科或神經外科醫師。
:::
