## 體重選擇依據

兒科藥物劑量計算建議：

- **一般兒童**：使用實際體重（Actual Body Weight）
- **肥胖兒童**：建議使用理想體重（Ideal Body Weight, IBW）計算，避免過量
- **極端體重**（過輕或過重）：體重異常者為劑量誤差高風險族群，應特別審慎

## 臨床注意事項

::: warning 劑量計算
計算兒童藥物劑量時應考慮患者體重、年齡、肝腎功能等共病因素。早產兒、新生兒應特別審慎。
:::

::: warning 藥物交互作用
部分藥物如 Macrolides（紅黴素類）可能與其他藥物產生交互作用，應注意 QT 延長風險。
:::

::: info 免責聲明
本工具僅供臨床輔助與教學參考，實際劑量應由兒科醫師依個案情況判定。
:::
