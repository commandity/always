# PHQ-9 & GAD-7 自我評估量表

**PHQ-9 & GAD-7 Self-Assessment Scales**

## 概述

<p class="subtitle-text">憂鬱症與焦慮症篩檢工具</p>

**PHQ-9**（Patient Health Questionnaire-9，病人健康問卷）由 Kroenke 等人於 2001 年發表，**GAD-7**（Generalized Anxiety Disorder-7，廣泛性焦慮症量表）則由 Spitzer 等人於 2006 年發表。兩者均源自美國初級醫療場域，是目前全球臨床上最廣泛使用的**憂鬱症與焦慮症自我篩檢工具**，具備優異的信度與效度，且施測快速（各約 2–3 分鐘），適合整合於常規門診流程。

## PHQ-9 病人健康問卷

### 工具背景

PHQ-9 直接對應 DSM 的**重鬱症（Major Depressive Disorder）診斷準則**，9 個題目分別對應 9 項憂鬱症狀。患者回顧**過去兩週**內每項症狀出現的頻率，以 0–3 分計分，總分 0–27 分。

### PHQ-9 評分說明

| 分數  | 嚴重程度   | 臨床建議                     |
| ----- | ---------- | ---------------------------- |
| 0–4   | 無憂鬱傾向 | 繼續觀察，必要時定期追蹤     |
| 5–9   | 輕度憂鬱   | 心理衛教、壓力管理，持續追蹤 |
| 10–14 | 中度憂鬱   | 建議就醫評估，考慮心理治療   |
| 15–19 | 中重度憂鬱 | 積極治療，藥物合併心理治療   |
| 20–27 | 重度憂鬱   | 立即就醫，評估住院必要性     |

> **臨床閾值**：PHQ-9 ≥ 10 分對重鬱症的**敏感度約 88%，特異度約 88%**，是建議的臨床介入切截點。

### PHQ-9 各題對應症狀

| 題號 | 症狀             | DSM 對應              |
| ---- | ---------------- | --------------------- |
| Q1   | 失去樂趣或興趣   | 興趣喪失（Anhedonia） |
| Q2   | 心情低落或絕望   | 憂鬱情緒              |
| Q3   | 睡眠問題         | 失眠或嗜睡            |
| Q4   | 疲累或無活力     | 疲勞                  |
| Q5   | 食慾改變         | 食慾或體重變化        |
| Q6   | 自我否定或罪惡感 | 無價值感              |
| Q7   | 專注困難         | 注意力下降            |
| Q8   | 動作或言語變化   | 精神運動遲緩或激動    |
| Q9   | 自傷念頭         | 死亡或自殺意念        |

## GAD-7 廣泛性焦慮症量表

### 工具背景

GAD-7 設計用於篩檢**廣泛性焦慮症（Generalized Anxiety Disorder，GAD）**，同時對恐慌症、社交焦慮症及創傷後壓力症候群（PTSD）亦有一定的偵測能力。患者同樣回顧**過去兩週**內的症狀頻率，以 0–3 分計分，總分 0–21 分。

### GAD-7 評分說明

| 分數  | 嚴重程度   | 臨床建議                   |
| ----- | ---------- | -------------------------- |
| 0–4   | 無焦慮傾向 | 繼續觀察                   |
| 5–9   | 輕度焦慮   | 壓力管理、放鬆技巧         |
| 10–14 | 中度焦慮   | 建議就醫評估，考慮心理治療 |
| 15–21 | 重度焦慮   | 積極治療，藥物或心理治療   |

> **臨床閾值**：GAD-7 ≥ 10 分對廣泛性焦慮症的**敏感度約 89%，特異度約 82%**。

## PHQ-9 與 GAD-7 合併使用

憂鬱與焦慮在臨床上高度共病，合併施測可：

- 同時篩檢兩種最常見的情緒障礙
- 協助鑑別症狀主軸（以憂鬱為主？以焦慮為主？或混合型？）
- 追蹤治療反應：分數下降 50% 以上通常定義為治療有效（response），降至 0–4 分為緩解（remission）

## 評分選項說明

每題皆以下列頻率評分：

| 選項           | 分數 | 說明                 |
| -------------- | ---- | -------------------- |
| 完全沒有       | 0    | 過去兩週幾乎未出現   |
| 幾天           | 1    | 出現數天（少於一週） |
| 一半以上的天數 | 2    | 超過一週以上出現     |
| 幾乎每天       | 3    | 幾乎每天都有         |

## 臨床應用

- **初級醫療篩檢**：整合於家醫科、內科、婦產科等常規門診
- **治療前基線評估**：建立治療前的症狀嚴重度基準
- **療效監測**：每 2–4 週追蹤，客觀評估症狀改善程度
- **研究用途**：作為臨床試驗的標準化結果指標

## 注意事項

::: warning PHQ-9 第 9 題——自我傷害念頭
若 PHQ-9 第 9 題（有不如死掉或用某種方式傷害自己的念頭）分數 **> 0**，**無論 PHQ-9 總分高低**，均應立即進行自殺風險評估，並考慮：

- 直接詢問自殺意念、計劃與方式
- 移除可能的致命工具
- 視風險程度安排緊急轉介或住院

如有立即危險，請撥打：📞 安心專線 **1925**　📞 生命線 **1995**
:::

::: warning 篩檢工具的限制
PHQ-9 與 GAD-7 為**篩檢工具**，陽性結果需搭配完整臨床訪談才能確立診斷。假陽性（如甲狀腺疾病、慢性疼痛）與假陰性（症狀輕微或刻意低報）均可能存在。
:::

::: info 免責聲明
本量表僅供初步篩檢參考，不具診斷效力，結果不能取代專業醫療評估。若有需要，請諮詢身心科醫師或臨床心理師以獲得完整評估與治療建議。
:::
