# PHQ-9 & GAD-7 自我評估量表

**憂鬱與焦慮篩檢 · Patient Health Questionnaire & Generalized Anxiety Disorder**

<Phq9Gad7Calculator />

## PHQ-9 憂鬱症篩檢

**PHQ-9** 為 9 題憂鬱症狀自填量表，評估過去兩週的困擾程度（每題 0–3，總分 0–27），對應 DSM 憂鬱症九項核心症狀。

### 分數意義與處置

| 總分  | 分層     | 建議處置                         |
| ----- | -------- | -------------------------------- |
| 0–4   | 無 / 極輕微 | 無明顯憂鬱症狀                 |
| 5–9   | 輕度     | 自我觀察與生活調適               |
| 10–14 | 中度     | 建議專業諮詢，考慮治療           |
| 15–19 | 中重度   | 積極就醫，多需藥物合併心理治療   |
| 20–27 | 重度     | 盡速精神科協助與安全評估         |

> **第 9 題（自傷 / 輕生念頭）只要 > 0，即須立即進行安全評估**，不論總分高低。安心專線 1925、生命線 1995。PHQ-9 ≥ 10 對重鬱症的敏感度與特異度約 88%，是門診常用的追蹤指標。

## GAD-7 焦慮症篩檢

**GAD-7** 為 7 題廣泛性焦慮量表，評估過去兩週的焦慮困擾（每題 0–3，總分 0–21）。

### 分數意義與處置

| 總分  | 分層 | 建議處置                   |
| ----- | ---- | -------------------------- |
| 0–4   | 無 / 極輕微 | 無明顯焦慮症狀        |
| 5–9   | 輕度 | 放鬆與舒壓                 |
| 10–14 | 中度 | 建議專業諮詢               |
| 15–21 | 重度 | 積極就醫評估               |

> **GAD-7 ≥ 10** 為進一步評估廣泛性焦慮症的常用切點（敏感度 89%、特異度 82%），亦可篩檢恐慌症、社交焦慮與創傷後壓力症。

## 注意事項

::: warning 篩檢非診斷
兩量表為自填篩檢工具，不能單獨作為診斷；確診需由專業人員依完整會談與臨床評估判定。
:::
