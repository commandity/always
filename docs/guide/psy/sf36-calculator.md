# SF-36 健康狀況調查

**SF-36 Health Survey**

## 概述

**SF-36**（36-Item Short Form Health Survey）由 John Ware 等人於 1992 年在美國醫療結果研究（Medical Outcomes Study，MOS）計畫中發展，是目前全球應用最廣泛的**通用型健康相關生活品質**（Health-Related Quality of Life，HRQOL）評估工具。SF-36 已被翻譯成 170 種以上語言，在數千項臨床研究、藥物試驗及人口健康調查中使用，繁體中文版亦已完成信效度驗證。

## 量表架構

SF-36 共 **36 個問題**（含 1 題健康變化單一項目），涵蓋 8 個健康領域，並整合為兩個綜合指標：

### 身體健康综合指標（PCS — Physical Component Summary）

| 領域               | 英文                 | 題數 | 評估內容                                       |
| ------------------ | -------------------- | ---- | ---------------------------------------------- |
| 身體功能（PF）     | Physical Functioning | 10   | 各種體力活動的受限程度（從劇烈運動到自行洗澡） |
| 生理角色功能（RP） | Role–Physical        | 4    | 過去4週內因身體健康影響工作或日常活動的程度    |
| 身體疼痛（BP）     | Bodily Pain          | 2    | 疼痛的嚴重程度及對日常工作的干擾               |
| 整體健康（GH）     | General Health       | 5    | 對自身整體健康狀況的主觀評估                   |

### 心理健康綜合指標（MCS — Mental Component Summary）

| 領域               | 英文               | 題數 | 評估內容                           |
| ------------------ | ------------------ | ---- | ---------------------------------- |
| 活力（VT）         | Vitality           | 4    | 精力充沛與疲勞感的頻率             |
| 社會功能（SF）     | Social Functioning | 2    | 健康問題對社交活動的干擾程度       |
| 情緒角色功能（RE） | Role–Emotional     | 3    | 因情緒問題影響工作或日常活動的程度 |
| 心理健康（MH）     | Mental Health      | 5    | 心理困擾（焦慮、憂鬱）與心理幸福感 |

### 健康變化（Health Transition）

第 2 題為單一問題，評估受測者對自身健康與一年前相比的主觀感受，不納入 PCS 或 MCS 計算，作為獨立參考指標。

## 評分方式

SF-36 的原始分數需經過標準化轉換才能得出 0–100 分的領域分數：

1. **逆向計分**：部分題目需先反向轉換（高分 = 低健康狀況的題目需翻轉）
2. **原始分數加總**：各領域題目原始分數加總
3. **線性轉換為 0–100 分**：

   ```
   轉換後分數 = ( 原始分數 − 最低可能分數 ) ÷ ( 最高 − 最低可能分數 ) × 100
   ```

4. **PCS 與 MCS**：各取所屬 4 個領域分數的平均值

> **所有領域：分數越高代表健康狀況越好。**

## 分數解讀參考

SF-36 的分數解讀需參照常模（normative data）。台灣成年人的大致參考範圍：

| 領域分數 | 解讀參考                     |
| -------- | ---------------------------- |
| ≥ 75     | 健康狀況良好                 |
| 50–74    | 中等程度，與一般人口差異不大 |
| 25–49    | 功能受限，低於一般人口       |
| < 25     | 功能顯著受損                 |

> **注意**：SF-36 分數的解讀應與同年齡、同性別的標準常模比較，以正確評估健康狀況。單純的絕對分數不足以判斷健康狀態，需參照各國本土化的常模資料。

## 各領域詳細說明

### 身體功能（PF，Physical Functioning）

評估各種體力活動的受限程度，從最劇烈（跑步、搬重物）到最基本（自行穿衣洗澡），共 10 個活動項目。PF 是 SF-36 中題數最多的領域，對偵測身體功能障礙特別敏感。

### 生理角色功能（RP，Role–Physical）

詢問過去 4 週內是否因**身體健康**原因而減少工作量、無法完成預定工作、受限於工作種類或有執行困難。以「是／否」計分，反映身體狀況對角色功能的影響。

### 身體疼痛（BP，Bodily Pain）

評估疼痛的強度（無痛到非常嚴重）及疼痛對日常工作的干擾程度，分數越高代表疼痛越輕微、干擾越小。

### 整體健康（GH，General Health）

包含對自身目前健康狀況的整體評估，以及對健康的信念（如「我比別人容易生病」）。GH 兼具主觀感受與健康信念兩個面向。

### 活力（VT，Vitality）

評估精力充沛感與疲憊感的頻率，包含「充滿活力」「精力充沛」（正向）與「疲倦」（逆向）的交叉驗證，避免純粹正向或負向偏誤。

### 社會功能（SF，Social Functioning）

評估身體或情緒問題對社交活動（訪友、參與社群）的干擾程度與頻率，分數越高代表社交功能越完整。

### 情緒角色功能（RE，Role–Emotional）

詢問過去 4 週內是否因**情緒問題**（沮喪、焦慮）而減少工作時間、工作量下降或工作不夠謹慎。以「是／否」計分，是偵測情緒對角色功能影響的敏感指標。

### 心理健康（MH，Mental Health）

透過 5 個問題涵蓋焦慮（「非常緊張」）、憂鬱（「心情低落」「感到悲傷」）與心理幸福感（「感到平靜」「覺得快樂」），提供心理健康的全面評估。

## 臨床應用

SF-36 廣泛應用於以下情境：

- **臨床試驗**：藥物或介入治療前後的 HRQOL 比較
- **慢性病管理**：追蹤慢性疾病（糖尿病、心臟病、風濕性疾病）患者的健康變化
- **術後評估**：手術（如人工關節置換、脊椎手術）的功能恢復評估
- **公共衛生研究**：人口健康狀態調查與政策評估
- **心理衛生篩檢**：配合其他量表（如 PHQ-9）提供更完整的身心健康圖像
- **職業醫學**：工傷或職業病對整體功能的影響評估

## SF-36 vs SF-12

**SF-12** 是 SF-36 的簡化版本，僅含 12 題，施測時間約 2 分鐘，仍可計算 PCS 和 MCS 兩個綜合指標。適合在資源有限或需要快速施測的情境下使用，但無法提供 8 個獨立領域的分析。

## 注意事項

::: warning 評估時間窗為過去四週
SF-36 所有問題（除第 2 題健康比較）均以**過去四週**為評估基準。若患者在此期間有急性事件（手術、急性感染、住院），分數可能無法代表其慢性基線狀態。
:::

::: warning 分數解讀需參照常模
SF-36 的絕對分數本身意義有限，應與**本土化常模**（台灣成年人正常值）或**疾病特異性常模**比較。研究顯示台灣成年人 PCS 平均約 48–52 分，MCS 平均約 50–54 分（依年齡層有所不同）。
:::
