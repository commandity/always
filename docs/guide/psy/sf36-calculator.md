# SF-36 健康狀況調查

**36-Item Short Form Health Survey · 通用型健康相關生活品質**

<Sf36Calculator />

## SF-36 量表

**SF-36**（Ware 1992）為全球應用最廣的通用型健康相關生活品質（HRQOL）量表，以 36 題評估過去四週的健康感受，涵蓋 8 個領域，整合為身體（PCS）與心理（MCS）兩大綜合指標；各領域轉換為 **0–100 分（分數愈高愈好）**。

| 綜合指標 | 領域                                         |
| -------- | -------------------------------------------- |
| 身體 PCS | 身體功能（PF）、身體角色（RP）、身體疼痛（BP）、一般健康（GH） |
| 心理 MCS | 活力（VT）、社交功能（SF）、情緒角色（RE）、心理健康（MH） |

### 分數意義

| 領域分數 | 分層 | 意義                             |
| -------- | ---- | -------------------------------- |
| ≥ 80     | 良好 | 該面向健康感受佳、少受限         |
| 50–79    | 中等 | 中度困擾或功能受限               |
| 25–49    | 偏低 | 明顯受限，影響日常               |
| < 25     | 不佳 | 嚴重受限，需關注與介入           |

> **SF-36 為描述性剖面（profile），非單一總分**——8 個領域各自解讀，再看 PCS / MCS 兩綜合分。分數愈高代表健康狀態愈好。適合追蹤慢性病、術後或介入前後的生活品質變化，並可與常模（如 50 分為一般人群平均）比較；單次分數的臨床意義有限，趨勢變化更具參考價值。

## 注意事項

::: warning 通用量表非疾病專屬
SF-36 反映整體健康感受，非特定疾病的診斷或嚴重度工具；解讀須結合病史與臨床脈絡，並留意填答者的理解與作答完整度。
:::
