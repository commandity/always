# 僵直性脊椎炎疾病活動評估

**Ankylosing Spondylitis Disease Activity · BASDAI**

<AnkylosingSpondylitisCalculator />

## BASDAI 疾病活動指數

**Bath Ankylosing Spondylitis Disease Activity Index**（Garrett 1994）為 6 題患者自填 VAS 問卷（各 0–10）：疲勞、脊椎疼痛、關節腫痛、整體不適、晨僵嚴重度與持續時間。

```
BASDAI = ( Q1 + Q2 + Q3 + Q4 + (Q5 + Q6) / 2 ) / 5
```

Q5、Q6 先取平均，使晨僵兩面向的合計權重等同其他單項。

### BASDAI 分數解讀

| BASDAI | 疾病活動度 | 意義與建議處置                             |
| ------ | ---------- | ------------------------------------------ |
| < 2    | 良好控制   | 維持現有治療與運動計畫                     |
| 2–4    | 輕至中度   | 密切監測，優化 NSAID 與復健                |
| ≥ 4    | 高活動度   | 疾病控制不佳——生物製劑（TNFi/IL-17i）評估門檻 |
| ≥ 6    | 極高活動度 | 積極介入，優先評估生物製劑                 |

> **BASDAI ≥ 4 是國際共識的「活動性疾病」切截值**，也是多數健保與指引啟動生物製劑的必要條件之一（通常需合併 NSAID 治療失敗）。追蹤時下降 ≥ 2 分（或 50%）視為 BASDAI50 治療反應。

## 注意事項

::: warning 主觀量表的限制
BASDAI 為純患者自評，受情緒與共病（纖維肌痛）影響；客觀發炎證據建議搭配 CRP／ASDAS 與影像學評估。
:::
