# 風濕免疫自我評估

**Autoimmune Disease Symptom Self-Screening**

<RheumaScreenCalculator />

## 工具說明

本篩檢問卷依症狀群（關節、皮膚、全身性、乾燥、血管等面向）進行自我評估，比對常見自體免疫疾病的典型表現（類風濕關節炎、紅斑性狼瘡、僵直性脊椎炎、乾燥症、血管炎等），提示最值得優先評估的方向。

### 結果解讀

| 篩檢結果     | 意義與建議                                             |
| ------------ | ------------------------------------------------------ |
| 無明顯符合   | 症狀不符合常見自體免疫疾病模式，持續觀察即可           |
| 部分症狀符合 | 建議記錄症狀變化（時間、誘因、部位），門診時提供醫師   |
| 高度符合     | 建議風濕免疫科就診，安排血清學檢查（ANA、RF、HLA-B27 等）|

> **發炎性 vs 機械性關節症狀的鑑別關鍵**：發炎性表現為晨僵 > 30 分鐘、休息後加重、活動後改善、好發 < 40 歲；機械性則相反。此區分決定是否需要風濕免疫科評估。

## 注意事項

::: warning 篩檢不能取代診斷
自體免疫疾病診斷需結合血清學、影像學與臨床分類標準（ACR/EULAR），本工具僅協助整理症狀線索。
:::
