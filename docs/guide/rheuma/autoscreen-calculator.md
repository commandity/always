## 本工具涵蓋的五種疾病

### 類風濕性關節炎（RA）

類風濕性關節炎是最常見的炎症性關節疾病，以**對稱性多關節炎**為主要表現，好發於雙手、雙腕等小關節。台灣盛行率約 0.4%，女性發病率是男性的 3 倍，好發年齡為 40–60 歲。

**核心症狀：**

- 雙手近端指間關節（PIP）和掌指關節（MCP）腫脹疼痛
- 晨間僵硬超過 1 小時（炎症活動的重要指標）
- 對稱性關節受累
- 長期未治療可能導致關節破壞和變形

**診斷關鍵：** 2010 ACR/EULAR 分類標準（關節數目、血清學、急性期反應物、症狀持續時間）

**血液檢查：** RF（類風濕因子）、Anti-CCP（抗環瓜氨酸抗體，特異性高達 98%）

### 僵直性脊椎炎（AS）/ 中軸型脊椎關節炎

僵直性脊椎炎是**中軸型脊椎關節炎**（axSpA）的代表性疾病，以骶髂關節炎和脊椎炎症為主要病變，好發於 40 歲以下年輕男性，HLA-B27 陽性率高達 90%。

**核心症狀：**

- 炎症性腰背痛：夜間痛、休息後加重、活動後改善
- 晨間脊椎僵硬超過 30 分鐘
- 臀部深層交替性疼痛（骶髂關節炎）
- 附著點炎（足底、跟腱、肋骨軟骨連接處）
- 葡萄膜炎（急性眼睛紅痛）

**炎症性背痛 vs 機械性背痛：**

| 特徵       | 炎症性背痛（AS） | 機械性背痛 |
| ---------- | ---------------- | ---------- |
| 發作年齡   | < 40 歲          | 任何年齡   |
| 起病方式   | 緩慢隱匿         | 急性或慢性 |
| 晨間僵硬   | > 30 分鐘        | < 30 分鐘  |
| 休息時疼痛 | 加重             | 改善       |
| 活動後     | 改善             | 加重或不變 |

**診斷關鍵：** ASAS 標準（影像學骶髂關節炎或 HLA-B27 陽性 + 典型症狀）

### 全身性紅斑狼瘡（SLE）

SLE 是最複雜的自體免疫疾病之一，可累及**全身多個器官系統**。台灣盛行率約每 10 萬人 70 例，女性發病率是男性的 9 倍，好發年齡為 15–45 歲（育齡期女性）。

**核心症狀（2019 ACR/EULAR 分類標準）：**

| 系統 | 症狀                                   |
| ---- | -------------------------------------- |
| 皮膚 | 蝶形紅斑、光敏感、圓盤狀紅斑、口腔潰瘍 |
| 關節 | 非侵蝕性關節炎、晨僵                   |
| 血液 | 溶血性貧血、白血球低下、血小板低下     |
| 腎臟 | 狼瘡性腎炎（蛋白尿、血尿）             |
| 神經 | 癲癇、精神症狀、認知障礙               |
| 漿膜 | 胸膜炎、心包炎                         |
| 血管 | 雷諾現象、抗磷脂抗體症候群             |

**診斷關鍵：** ANA（抗核抗體，敏感性高）、anti-dsDNA（特異性高）、抗 Sm 抗體

**妊娠注意：** SLE 對懷孕影響重大，孕前需評估疾病穩定度並調整藥物。

### 修格蘭氏症候群（Sjögren's Syndrome，SS）

修格蘭氏症候群是以**外分泌腺（淚腺、唾液腺）自體免疫**破壞為主的疾病，以眼乾和口乾為核心症狀，可為原發性（primary SS）或繼發於其他自體免疫疾病。台灣盛行率約 0.1–0.5%，女性佔 90%。

**核心症狀：**

- 乾眼症：眼乾、灼熱、異物感、畏光（淚液分泌不足）
- 口乾：需頻繁飲水、進食乾食困難、蛀牙快速增加
- 腮腺腫大
- 雷諾現象（30–40%）
- 關節痛、疲勞

**診斷評估：**

- Schirmer 試驗（淚液分泌測試）
- 唇腺切片（唾液腺病理）
- 血清：Anti-SSA（Ro）陽性率 70–80%、Anti-SSB（La）

**繼發性 SS：** 約 30% RA 患者、20% SLE 患者合併繼發性修格蘭氏症候群。

### 乾癬性關節炎（PsA）

乾癬性關節炎是**乾癬（牛皮癬）相關的炎症性關節炎**，約 20–30% 的乾癬患者會發展為關節炎，通常在皮膚症狀出現後 10 年內發病。

**五種臨床表型：**

| 表型                  | 特點                     |
| --------------------- | ------------------------ |
| 非對稱性寡關節炎      | 最常見，≤ 4 個關節       |
| 對稱性多關節炎        | 類似 RA，但 RF 陰性      |
| 遠端指間關節（DIP）炎 | 乾癬性特有，常伴指甲病變 |
| 毀損性關節炎          | 最嚴重，骨質溶解         |
| 中軸型（脊椎炎）      | 類似 AS，骶髂關節炎      |

**乾癬性指炎（香腸指）：** 整根手指或腳趾腫脹如香腸狀，是 PsA 的特徵性表現。

**診斷：** CASPAR 標準（乾癬病史/皮損、指甲病變、RF 陰性、指炎、附著點炎、影像學骨增生）

## 自體免疫疾病的共同機制

自體免疫疾病的根本原因是**免疫耐受性（immune tolerance）破壞**——免疫系統對自身組織產生攻擊。

```
正常免疫 → 識別「自己」vs「外來」→ 只攻擊外來抗原
自體免疫 → 識別失誤 → 產生自身抗體 → 攻擊自身組織
```

**促發因素：**

- **基因**：HLA 基因（AS 的 HLA-B27、RA 的 HLA-DRB1）
- **環境**：感染（分子擬態）、紫外線（SLE）、吸菸（RA 抗 CCP 陽性）
- **荷爾蒙**：雌激素促進自體免疫（解釋女性高發病率）
- **腸道菌叢失調**：近年研究顯示與 AS、RA 發病相關

## 常用免疫學血液檢查

| 檢驗項目        | 縮寫       | 相關疾病                   | 臨床意義                         |
| --------------- | ---------- | -------------------------- | -------------------------------- |
| 抗核抗體        | ANA        | SLE（敏感性 98%）、SS、PsA | 篩查用，陽性需進一步確認         |
| 抗雙鏈 DNA 抗體 | anti-dsDNA | SLE（特異性高）            | 與狼瘡腎炎活動度相關             |
| 類風濕因子      | RF         | RA（約 70%陽性）、SS       | 非特異性，老年人可假陽性         |
| 抗環瓜氨酸抗體  | Anti-CCP   | RA（特異性 98%）           | 早於症狀出現數年，預測侵蝕性病程 |
| 抗 SSA/Ro 抗體  | Anti-SSA   | SS（70–80%）、SLE          | 可通過胎盤，影響胎兒             |
| 抗 SSB/La 抗體  | Anti-SSB   | SS（40–60%）、SLE          | —                                |
| HLA-B27         | —          | AS（90%陽性）、PsA         | 遺傳標誌，非診斷標準             |
| C 反應蛋白      | CRP        | 全身炎症指標               | 快速反映發炎活動度               |
| 紅血球沉降速率  | ESR        | 全身炎症指標               | 受非炎症因素影響較多             |
| 補體            | C3、C4     | SLE                        | 補體低下反映疾病活動             |

## 何時應立即就診？

以下情況請**盡快至急診或門診**，不要等待：

::: danger 緊急就診指標

- 急性眼睛紅痛、畏光、視力模糊（可能為葡萄膜炎，需 24 小時內就診）
- 深呼吸時劇烈胸痛（可能為心包炎或胸膜炎）
- 出現大量蛋白尿或血尿（可能為狼瘡性腎炎）
- 新出現的肌肉無力，特別是上肢抬高或下蹲困難
- 持續高燒（> 38.5°C）伴有關節腫痛
- 突發性關節劇烈紅腫熱痛（需排除感染性關節炎）
  :::

## 就診準備清單

前往風濕免疫科就診時，建議準備：

**症狀記錄：**

- 症狀開始時間和演變過程
- 哪些關節受影響（可用手機拍照記錄腫脹的關節）
- 晨間僵硬的持續時間
- 疼痛對日常活動的影響（可使用本工具的評估結果）

**個人資料：**

- 家族自體免疫疾病史
- 目前使用的所有藥物（含保健品）
- 女性需說明月經週期和生育計畫
- 最近 3–6 個月的血液檢查報告（若有）

**可能被要求的檢查：**

- 基本血液檢查（CBC、CRP、ESR）
- 免疫學全套（ANA panel、RF、Anti-CCP）
- 尿液常規（排除腎臟受累）
- 影像學（X 光、MRI 或超音波）

## 自體免疫疾病的治療原則

### 治療目標（Treat-to-Target）

現代風濕免疫學的核心概念是**達標治療（T2T）**——設定明確的疾病緩解或低活動度目標，並定期評估是否達標，若未達標則及時調整治療。

### 藥物治療層次

| 層次 | 藥物類別       | 代表藥物                                | 適用疾病       |
| ---- | -------------- | --------------------------------------- | -------------- |
| 一線 | NSAIDs         | Naproxen、Celecoxib                     | AS、PsA        |
| 一線 | 傳統合成 DMARD | Methotrexate（MTX）、Hydroxychloroquine | RA、SLE、PsA   |
| 升階 | 生物製劑       | 抗 TNF-α、抗 IL-17、抗 IL-6             | RA、AS、PsA    |
| 升階 | JAK 抑制劑     | Tofacitinib、Upadacitinib、Baricitinib  | RA、PsA、AS    |
| 特定 | Belimumab      | —                                       | SLE            |
| 輔助 | 糖皮質激素     | Prednisolone                            | 急性期橋接治療 |

### 非藥物介入

- **規律運動**：游泳、瑜珈、太極拳——維持關節活動度和肌力
- **物理治療**：特別對 AS 的脊椎伸展和姿勢訓練
- **防曬**：SLE 患者必須嚴格防曬
- **戒菸**：吸菸加重 RA 和 AS，並降低生物製劑療效
- **口腔衛生**：SS 患者需特別加強牙齒護理
