## FRAX 骨折風險評估

**FRAX**（WHO Fracture Risk Assessment Tool）以年齡、性別、BMI、七項臨床危險因子（既往骨折、父母髖部骨折、吸菸、糖皮質激素、類風濕關節炎、次發性骨質疏鬆、飲酒），加上可選的股骨頸 T-score，估算 **10 年骨折機率**：

- **MOF**：主要骨質疏鬆性骨折（脊椎、髖部、前臂、肱骨）
- **Hip**：髖部骨折

### 風險解讀與治療門檻

| 10 年風險                 | 分級   | 意義與建議處置                        |
| ------------------------- | ------ | ------------------------------------- |
| MOF ≥ 20% **或** Hip ≥ 3% | 高風險 | 建議啟動抗骨鬆藥物治療；符合 DXA 給付 |
| MOF 10–20% 或 Hip 1–3%    | 中風險 | 補充鈣＋維生素 D，安排 BMD 追蹤       |
| MOF < 10% 且 Hip < 1%     | 低風險 | 生活方式介入（負重運動、防跌），定期追蹤 |

> **符合任一高風險切截值即建議治療**。FRAX 的最大優勢是無 BMD 也能估算——適合基層初篩；已知 T-score ≤ −2.5 者 FRAX 可能低估風險，且工具不納入跌倒史、骨折次數與腰椎 BMD，臨床判斷應據此上調。糖皮質激素 ≥ 7.5 mg/day 者實際風險亦高於計算值。

## 注意事項

::: warning 適用範圍
FRAX 適用於 40–90 歲未接受抗骨鬆治療者；已治療中的患者其結果僅供參考。
:::

::: info 免責聲明
本計算器僅供臨床輔助參考，藥物治療決策需由醫師綜合 BMD、跌倒風險及共病評估。
:::
