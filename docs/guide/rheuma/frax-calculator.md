# FRAX 骨折風險評估工具

**FRAX Fracture Risk Assessment Tool**

## 概述

**FRAX**（Fracture Risk Assessment Tool）由世界衛生組織（WHO）骨折風險評估工作小組開發，整合多項臨床危險因子，計算 **40–90 歲成人未來 10 年**發生重大骨質疏鬆性骨折及髖部骨折的機率。FRAX 已針對全球 70 個以上國家建立各自的人口統計模型，是目前**全球使用最廣泛的骨折風險預測工具**，廣泛應用於骨質疏鬆症篩查、治療決策及資源分配。

## 評估項目說明

### 基本資料

| 項目               | 說明                             | 臨床重要性                                      |
| ------------------ | -------------------------------- | ----------------------------------------------- |
| **年齡**           | 40–90 歲（超出範圍按邊界值計算） | 骨折風險隨年齡呈指數增長，65 歲以上風險顯著上升 |
| **性別**           | 男性或女性                       | 女性骨折風險整體高於男性，尤其停經後            |
| **體重**           | 公斤                             | 低體重（BMI < 20）是重要危險因子                |
| **身高**           | 公分                             | 與體重共同計算 BMI                              |
| **股骨頸 T-score** | DXA 測量（可選填）               | 加入 BMD 可顯著提升預測準確度                   |

### 七項臨床危險因子

| 危險因子             | 定義                                                                             | 風險強度 |
| -------------------- | -------------------------------------------------------------------------------- | -------- |
| **既往骨折史**       | 40歲後自發性或輕微外傷骨折（含影像學椎體骨折）                                   | 強       |
| **父母髖部骨折史**   | 父親或母親曾發生髖部骨折                                                         | 中       |
| **目前吸菸**         | 現在有吸菸習慣                                                                   | 中       |
| **糖皮質激素使用**   | 目前或曾連續使用 > 3 個月，劑量 ≥ Prednisolone 5mg/日                            | 強       |
| **類風濕性關節炎**   | 確診 RA（非骨關節炎），需臨床或實驗室證據                                        | 中       |
| **續發性骨質疏鬆症** | 第1型糖尿病、成骨不全、未治療甲亢、性腺低下、早停經（<45歲）、慢性肝病、吸收不良 | 中       |
| **酗酒**             | 每日飲酒 ≥ 3 單位（1單位 ≈ 標準啤酒285ml / 烈酒30ml / 紅酒120ml）                | 中       |

## FRAX 計算公式

FRAX 使用 **Poisson 迴歸模型**，整合年齡、BMI（以 log 轉換）及各危險因子的加權係數，計算 10 年累積骨折機率：

```
P(10yr) = 1 − e^[ −H₀ × e^( Σ βᵢXᵢ ) ]
```

其中 $H_0$ 為基準 10 年骨折風險（依國家/性別不同），$\beta_i$ 為各危險因子的迴歸係數，$X_i$ 為各危險因子的值（連續或二元）。

**輸出兩個指標：**

- **主要骨質疏鬆性骨折（MOF）**：脊椎、前臂、髖部、肱骨骨折的 10 年機率
- **髖部骨折（Hip Fracture）**：僅計算髖部骨折的 10 年機率

## 臨床切截值與治療建議

依據台灣骨鬆學會（2022）及國際骨質疏鬆基金會（IOF）的建議：

| 指標                   | 切截值    | 臨床建議                        |
| ---------------------- | --------- | ------------------------------- |
| 主要骨折（MOF）        | **≥ 20%** | 建議啟動藥物治療                |
| 髖部骨折（Hip）        | **≥ 3%**  | 建議啟動藥物治療                |
| MOF 10–20% 或 Hip 1–3% | 中風險    | 補充鈣質/維生素D，考慮 BMD 追蹤 |
| MOF < 10% 且 Hip < 1%  | 低風險    | 生活方式介入，定期追蹤          |

> **符合任一切截值**（MOF ≥ 20% 或 Hip ≥ 3%）即建議考慮藥物治療，並可申請 DXA 健保給付。

## 骨密度（BMD）的角色

FRAX 可在**無 BMD 數據**時仍提供有臨床意義的風險估算，這是其最大優勢之一。

| 情境                       | FRAX 使用方式                       |
| -------------------------- | ----------------------------------- |
| 無 BMD                     | 僅用臨床因子計算，作為初步篩查      |
| 有股骨頸 T-score           | 加入 BMD 計算，準確度顯著提升       |
| T-score ≤ −2.5（骨質疏鬆） | FRAX 可能低估風險，建議臨床判斷調整 |

### T-score 分類（WHO 標準，NHANES III 參考）

| T-score           | 分類                     | 說明                 |
| ----------------- | ------------------------ | -------------------- |
| > −1.0            | 正常                     | 骨密度在正常範圍     |
| −1.0 至 −2.5      | 骨質減少（Osteopenia）   | 需注意危險因子並追蹤 |
| ≤ −2.5            | 骨質疏鬆（Osteoporosis） | 需積極介入           |
| ≤ −2.5 + 脆性骨折 | 嚴重骨質疏鬆             | 需立即啟動藥物治療   |

## 骨質疏鬆症的治療選項

### 非藥物介入（所有風險等級適用）

| 措施     | 目標劑量                  | 說明                                       |
| -------- | ------------------------- | ------------------------------------------ |
| 鈣質補充 | 1000–1200 mg/天（含飲食） | 乳製品、豆腐、深綠色蔬菜                   |
| 維生素 D | 800–2000 IU/天            | 日曬 + 補充劑，目標血清 25(OH)D ≥ 30 ng/mL |
| 負重運動 | 每週 ≥ 3 次               | 走路、太極拳、阻力訓練                     |
| 跌倒預防 | —                         | 居家環境改善、平衡訓練、視力矯正           |
| 戒菸戒酒 | —                         | 直接降低骨折危險因子                       |

### 藥物治療（高風險患者）

| 藥物類別              | 代表藥物                                  | 機制                | 適用                       |
| --------------------- | ----------------------------------------- | ------------------- | -------------------------- |
| **雙磷酸鹽**          | Alendronate、Risedronate、Zoledronic acid | 抑制破骨細胞        | 首選，脊椎及髖部骨折均有效 |
| **RANK-L 抑制劑**     | Denosumab（Prolia）                       | 抑制破骨細胞分化    | 腎功能不全者尤適           |
| **副甲狀腺素**        | Teriparatide（Forteo）、Abaloparatide     | 促進骨形成          | 重度骨質疏鬆、多發骨折     |
| **Sclerostin 抑制劑** | Romosozumab（Evenity）                    | 促骨形成 + 抑骨吸收 | 極高骨折風險               |
| **SERM**              | Raloxifene                                | 選擇性雌激素受體    | 停經後女性，脊椎骨折       |

> **台灣健保給付條件**：T-score ≤ −2.5 且 FRAX 達切截值，或曾有脆性骨折，可申請 DXA 及藥物給付。

## FRAX 的限制

| 限制                     | 說明                                                         |
| ------------------------ | ------------------------------------------------------------ |
| **危險因子為二元變數**   | 骨折史、吸菸等僅記錄是否，不計量多寡，可能低估高暴露者的風險 |
| **不含所有骨密度部位**   | 僅使用股骨頸 T-score，不含腰椎 BMD                           |
| **未涵蓋跌倒風險**       | 跌倒是骨折重要機制，但 FRAX 未直接納入                       |
| **以白人族群建立**       | 各國需重新校正，台灣使用中國大陸人口統計，可能存在誤差       |
| **不適用兒童及90歲以上** | 僅適用 40–90 歲                                              |
| **忽略劑量效應**         | 吸菸量、飲酒量、糖皮質激素劑量均以平均值計算                 |
| **治療中不適用**         | FRAX 為未治療患者設計，不應用於已接受骨鬆藥物者              |

## 注意事項

::: warning 骨關節炎不計入類風濕性關節炎
「類風濕性關節炎」項目**必須是確診的 RA**，需要臨床檢查（關節腫脹、晨僵）或實驗室證據（RF、Anti-CCP 陽性）。骨關節炎（退化性關節炎）實際上對骨密度有輕微保護作用，**不可勾選此項**。
:::

::: warning 糖皮質激素效應可能被低估
FRAX 的糖皮質激素項目假設為「平均」使用量（Prednisolone 7.5mg/日）。若患者使用**高劑量**（≥ Prednisolone 30mg/日）或**長期使用**，實際骨折風險可能高於 FRAX 計算值，建議臨床醫師主動調高風險判斷並考慮積極治療。
:::

::: warning FRAX 不適用於已接受骨鬆藥物治療者
FRAX 是為**未治療患者**設計的骨折風險篩查工具，不應用於評估已接受雙磷酸鹽、Denosumab 等藥物治療患者的骨折風險。治療中患者應以 DXA 追蹤骨密度變化及臨床骨折發生情況評估療效。
:::

::: info 免責聲明
本計算器依據 WHO FRAX 演算法及台灣/中國人口統計模型建立，結果僅供臨床決策參考，不能取代完整的骨密度檢查及骨科/風濕科醫師的臨床判斷。骨質疏鬆症的診斷及治療應由合格醫療人員根據患者完整病史與檢查結果綜合決定。若需精確的 FRAX 計算，建議使用 WHO 官方平台：[www.sheffield.ac.uk/FRAX](https://www.sheffield.ac.uk/FRAX)。
:::
