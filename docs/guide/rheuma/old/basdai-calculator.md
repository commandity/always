# BASDAI 疾病活動指數

**BASDAI Bath Ankylosing Spondylitis Disease Activity Index**

## 概述

**BASDAI**（Bath Ankylosing Spondylitis Disease Activity Index）由 Garrett 等人於 1994 年發表，是評估**僵直性脊椎炎（AS）及中軸型脊椎關節炎（axSpA）疾病活動度**的標準工具。BASDAI 包含 6 個患者自評問題，涵蓋疲勞、脊椎疼痛、關節症狀、整體不適及晨間僵硬，分數範圍 0–10 分，完全由患者自行填寫，約需 2 分鐘完成。

## 計算公式

```
BASDAI = ( Q1 + Q2 + Q3 + Q4 + [(Q5 + Q6) ÷ 2] ) ÷ 5
```

| 部分     | 問題 | 內容                                | 計分 |
| -------- | ---- | ----------------------------------- | ---- |
| 全身症狀 | Q1   | 疲勞程度                            | 0–10 |
| 全身症狀 | Q2   | 脊椎疼痛                            | 0–10 |
| 全身症狀 | Q3   | 關節腫脹或壓痛                      | 0–10 |
| 全身症狀 | Q4   | 整體不適感                          | 0–10 |
| 晨間僵硬 | Q5   | 晨間僵硬嚴重程度                    | 0–10 |
| 晨間僵硬 | Q6   | 晨間僵硬持續時間（0=無，10=≥2小時） | 0–10 |

> **Q5 與 Q6 先取平均**，再與 Q1–Q4 合計後除以 5，使晨間僵硬的兩個面向（嚴重程度與持續時間）合計權重等同於其他單項問題。

## 疾病活動度分類

| BASDAI 分數 | 疾病活動度                | 臨床意義                         |
| ----------- | ------------------------- | -------------------------------- |
| **< 2**     | 良好控制（Controlled）    | 疾病活動性很低，維持現有治療方案 |
| **2–4**     | 輕至中度（Low-Moderate）  | 需密切監測，考慮治療優化         |
| **≥ 4**     | 高活動度（High Activity） | 疾病控制不佳，建議調整治療方案   |
| **≥ 6**     | 極高活動度（Very High）   | 需積極介入，優先評估生物製劑     |

> **關鍵切截值：BASDAI ≥ 4** 是調整治療的重要閾值，也是評估生物製劑適應症的核心標準之一。

## 六個評估項目詳解

### Q1 — 疲勞程度（Fatigue）

AS 患者的疲勞往往被低估，但研究顯示它是影響生活品質最重要的因素之一。疲勞不僅反映疾病炎症活動，也與睡眠障礙、情緒狀態及功能受限相關。0 = 完全無疲勞，10 = 嚴重到無法正常生活。

### Q2 — 脊椎疼痛（Spinal Pain）

評估頸椎、胸椎、腰椎的整體疼痛程度。AS 的脊椎疼痛典型表現為**夜間痛和靜止痛**，活動後改善，區別於機械性腰痛。0 = 無疼痛，10 = 難以忍受的劇烈疼痛。

### Q3 — 關節腫脹或壓痛（Peripheral Joint Involvement）

評估**脊椎以外**的周邊關節（髖部、膝蓋、肩膀、踝關節等）的腫脹和壓痛程度。約 30–40% 的 AS 患者有周邊關節炎，髖關節受累預後較差，需特別關注。

### Q4 — 整體不適感（Overall Discomfort）

患者對整體健康狀況和疾病造成不適感的主觀評估，涵蓋所有因 AS 引起的身體不舒服感受。

### Q5 — 晨間僵硬嚴重程度（Morning Stiffness Severity）

評估早晨起床時脊椎僵硬的嚴重程度。**晨間僵硬是 AS 的典型症狀**，通常在起床後最嚴重，活動後逐漸改善。嚴重的晨間僵硬常使患者難以起床或完成早晨例行活動。

### Q6 — 晨間僵硬持續時間（Morning Stiffness Duration）

0 = 無晨間僵硬（或極短暫）；10 = 僵硬持續 **≥ 2 小時**。依據 ASAS 定義，晨間僵硬 ≥ 30 分鐘是炎症性腰背痛的標準之一，而 AS 的晨間僵硬通常更長。

> **Q5 + Q6 設計原理**：將晨間僵硬分為嚴重程度（主觀感受）和持續時間（客觀測量）兩個維度，取平均後可更全面反映晨間僵硬的整體負擔，同時避免單一問題的過度加權。

## 臨床應用

### 生物製劑適應症評估

BASDAI ≥ 4 是國際指引（ASAS/EULAR）中評估生物製劑（TNF 抑制劑、IL-17 抑制劑）適應症的核心標準之一。需同時符合：

| 條件                   | 說明                                                 |
| ---------------------- | ---------------------------------------------------- |
| **BASDAI ≥ 4**         | 高疾病活動度（需兩次評估確認）                       |
| **NSAIDs 治療失敗**    | 充分使用 ≥ 2 種 NSAIDs 至少 4 週，療效不佳或無法耐受 |
| **影像學或血清學支持** | 骶髂關節炎（X 光或 MRI）或 HLA-B27 陽性              |
| **專科醫師評估**       | 由風濕科或相關專科醫師確認診斷                       |

### 治療目標

| 目標                   | BASDAI 切截值         |
| ---------------------- | --------------------- |
| 理想目標（接近緩解）   | < 2                   |
| 可接受目標（良好控制） | < 4                   |
| 需要調整治療           | ≥ 4                   |
| 生物製劑評估門檻       | ≥ 4（需配合其他條件） |

### 追蹤頻率建議

- **活動期**（BASDAI ≥ 4）：每 4–6 週評估
- **穩定期**（BASDAI < 4）：每 3–6 個月評估
- **生物製劑治療後**：12–16 週評估治療反應（目標降幅 ≥ 2 分或 50%）
- **MCID**（最小臨床意義差異）：約 1.1 分

## BASDAI 與其他 AS 評估工具

| 工具       | 全名                           | 評估面向   | 特點                         |
| ---------- | ------------------------------ | ---------- | ---------------------------- |
| **BASDAI** | Bath AS Disease Activity Index | 疾病活動度 | 患者自填，6 題，臨床標準工具 |
| **ASDAS**  | AS Disease Activity Score      | 疾病活動度 | 需 CRP 或 ESR，更客觀        |
| **BASFI**  | Bath AS Functional Index       | 身體功能   | 10 題，評估日常功能          |
| **BASMI**  | Bath AS Metrology Index        | 脊椎活動度 | 5 項身體測量，需醫師執行     |
| **BASRI**  | Bath AS Radiology Index        | 影像學損傷 | X 光評分，追蹤結構破壞       |

> BASDAI 和 ASDAS 是最常並用的活動度評估工具。ASDAS 整合了客觀發炎指標（CRP 或 ESR），在偵測治療反應方面更靈敏，EULAR 2022 指引推薦優先使用 ASDAS，但 BASDAI 仍廣泛用於臨床實務和患者自我監測。

## AS 的藥物治療概覽

### 非甾體抗炎藥（NSAIDs）——首選

NSAIDs 是 AS 的一線治療，對炎症性脊椎疼痛和晨間僵硬效果最佳：

| 藥物                   | 特點                                     |
| ---------------------- | ---------------------------------------- |
| Indomethacin（消炎痛） | 傳統首選，效果強但胃腸副作用較多         |
| Diclofenac（服他寧）   | 常用，胃腸耐受性較好                     |
| Celecoxib（希樂葆）    | COX-2 選擇性，胃腸副作用少，心血管需評估 |
| Etoricoxib（萬克適）   | 強效，每日一次，需注意心血管風險         |

### 生物製劑（bDMARDs）

適用於 BASDAI ≥ 4 且 NSAIDs 治療失敗者：

| 類別      | 代表藥物                                        | 靶點      |
| --------- | ----------------------------------------------- | --------- |
| 抗 TNF-α  | Adalimumab、Etanercept、Golimumab、Certolizumab | TNF-α     |
| 抗 IL-17A | Secukinumab、Ixekizumab                         | IL-17A    |
| 抗 IL-23  | Risankizumab                                    | IL-23 p19 |

### JAK 抑制劑（tsDMARDs）

| 藥物         | 靶點   | 適應症                        |
| ------------ | ------ | ----------------------------- |
| Tofacitinib  | JAK1/3 | r-axSpA（有影像學骶髂關節炎） |
| Upadacitinib | JAK1   | r-axSpA 及 nr-axSpA           |
| Filgotinib   | JAK1   | 歐洲已核准                    |

## AS 的關節外表現

AS 是一種系統性疾病，除脊椎和關節外，常有以下關節外表現：

| 部位 | 表現                                 | 發生率 |
| ---- | ------------------------------------ | ------ |
| 眼睛 | 急性前葡萄膜炎（虹膜炎）             | 25–40% |
| 皮膚 | 乾癬（牛皮癬）                       | 10–25% |
| 腸道 | 炎症性腸病（克隆氏症、潰瘍性結腸炎） | 5–10%  |
| 心臟 | 主動脈炎、傳導異常                   | < 5%   |
| 肺臟 | 上葉纖維化（晚期）                   | 罕見   |
| 腎臟 | IgA 腎病（繼發性澱粉樣變性）         | 罕見   |

> 發生急性眼睛紅痛畏光（葡萄膜炎）時需緊急至眼科就診，BASDAI 評估的關節症狀不包含葡萄膜炎，但後者是 AS 疾病負擔的重要組成。

## 注意事項

::: warning BASDAI 是主觀評估，可能受其他因素影響
BASDAI 完全依賴患者主觀感受。疼痛敏感化（central sensitization）、纖維肌痛症合併症、憂鬱或睡眠障礙均可能使分數偏高，而實際的炎症活動並未增加。建議同時測量客觀發炎指標（CRP、ESR）或使用 ASDAS 進行補充評估。
:::

::: warning BASDAI ≥ 4 是生物製劑的必要條件之一，但非唯一條件
BASDAI ≥ 4 達到生物製劑評估門檻，但還需結合影像學骶髂關節炎或 HLA-B27 陽性、NSAIDs 治療失敗（充分使用 ≥ 2 種 NSAIDs），以及排除禁忌症（活動性感染、結核病、心衰竭、脫髓鞘疾病等）後，方可開始生物製劑治療。
:::

::: warning 追蹤時應固定使用 BASDAI 或 ASDAS，不可混用
BASDAI 和 ASDAS 的分數範圍和切截值不同，在同一患者的縱向追蹤中應固定使用同一工具，切勿在不同次評估中混用，以確保治療反應評估的一致性。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不能取代完整的臨床評估、影像學及實驗室檢查。AS 及 axSpA 的診斷和治療決策應由風濕科醫師根據患者完整的臨床狀況及個別因素綜合判斷。
:::
