# RAPID3 快速評估指數

**RAPID3 Routine Assessment of Patient Index Data**

## 概述

**RAPID3**（Routine Assessment of Patient Index Data 3）由 Theodore Pincus 於 2008 年提出，是類風濕性關節炎（RA）疾病活動度評估工具中**唯一完全由患者自行填寫**的評分系統。RAPID3 包含三個部分：身體功能評估（MDHAQ）、疼痛程度及整體健康狀態，總分 0–30 分，預計完成時間僅需 **3 分鐘**，是門診快速追蹤 RA 疾病活動度的理想工具。

## 計算公式

**RAPID3 = MDHAQ 換算分（0–10）+ 疼痛（0–10）+ 整體健康（0–10）**

最高分 = 30 分

| 部分                | 內容                           | 計分方式              | 範圍 |
| ------------------- | ------------------------------ | --------------------- | ---- |
| **Part 1** MDHAQ    | 10 項身體功能活動題（0/1/2/3） | 原始分合計（0–30）÷ 3 | 0–10 |
| **Part 2** 疼痛     | 過去一週關節炎疼痛程度 VAS     | 患者自評 0–10         | 0–10 |
| **Part 3** 整體健康 | 過去一週整體健康狀態 VAS       | 患者自評 0–10         | 0–10 |

## 疾病活動度分類

| RAPID3 分數 | 疾病活動度                      | 臨床意義                     |
| ----------- | ------------------------------- | ---------------------------- |
| **≤ 3**     | 接近緩解（Near Remission）      | 疾病活動性很低，症狀控制良好 |
| **3.1–6**   | 輕度嚴重度（Low Severity）      | 輕度活動，需持續監測         |
| **6.1–12**  | 中度嚴重度（Moderate Severity） | 中度活動，可能需要調整治療   |
| **> 12**    | 高度嚴重度（High Severity）     | 高度活動，需要積極治療       |

## 第一部分：MDHAQ 身體功能評估

MDHAQ（Modified Health Assessment Questionnaire）源自 Stanford HAQ，由 10 個日常活動題目組成。每題依困難程度評分：

| 選項     | 說明                   | 分數 |
| -------- | ---------------------- | ---- |
| 沒有困難 | Without any difficulty | 0    |
| 有些困難 | With some difficulty   | 1    |
| 很大困難 | With much difficulty   | 2    |
| 無法進行 | Unable to do           | 3    |

**10 個評估活動：**

| 題號 | 活動                 | 臨床意義                 |
| ---- | -------------------- | ------------------------ |
| 1    | 穿衣（含鞋帶和鈕扣） | 精細動作，手部小關節功能 |
| 2    | 起床和下床           | 軀幹及下肢功能           |
| 3    | 將滿杯水舉到嘴邊     | 上肢肌力及肩肘功能       |
| 4    | 在平地上走路         | 下肢及步態功能           |
| 5    | 清洗和擦乾全身       | 全身關節活動度           |
| 6    | 彎腰從地板撿起衣物   | 脊椎及髖關節功能         |
| 7    | 開關水龍頭           | 手腕及握力功能           |
| 8    | 上下車               | 下肢及軀幹協調           |
| 9    | 步行 2 英里或 3 公里 | 耐力及整體活動能力       |
| 10   | 參加娛樂活動和運動   | 社會參與及整體功能       |

> **計分換算**：10 題原始分合計（0–30）**除以 3** 得到換算分（0–10），使三個部分的分數範圍一致，均為 0–10。

## 第二部分：疼痛程度評估

患者使用 0–10 視覺類比量表（VAS）自評**過去一週因關節炎引起的疼痛程度**：

- **0 分**：完全無疼痛
- **10 分**：難以忍受的最嚴重疼痛

## 第三部分：整體健康狀態評估

患者使用 0–10 VAS 評估**過去一週的整體健康狀況**，涵蓋疾病、身體、情緒和社交各方面：

- **0 分**：非常好
- **10 分**：非常差

## RAPID3 與其他評估工具比較

| 比較項目           | RAPID3    | CDAI        | DAS28-ESR   | DAS28-CRP   |
| ------------------ | --------- | ----------- | ----------- | ----------- |
| **填寫者**         | 患者自填  | 醫師 + 患者 | 醫師 + 患者 | 醫師 + 患者 |
| **關節檢查**       | ❌ 不需要 | ✓ 需要      | ✓ 需要      | ✓ 需要      |
| **實驗室**         | ❌ 不需要 | ❌ 不需要   | ✓ ESR       | ✓ CRP       |
| **完成時間**       | ≈ 3 分鐘  | ≈ 5 分鐘    | ≈ 10 分鐘   | ≈ 10 分鐘   |
| **分數範圍**       | 0–30      | 0–76        | 0–9.4       | 0–9.4       |
| **緩解切截值**     | ≤ 3       | ≤ 2.8       | < 2.6       | < 2.6       |
| **門診候診室完成** | ✓ 可以    | ✗           | ✗           | ✗           |
| **電子問卷適合性** | ✓ 極佳    | 中          | 差          | 差          |
| **EULAR 認可**     | 部分      | ✓           | ✓           | ✓           |

## RAPID3 的核心優勢：患者自填，候診室完成

RAPID3 最大的臨床優勢是可在**患者等候就診期間**完成，醫師無需額外花時間進行關節檢查或等待血液報告即可掌握疾病活動狀況：

| 情境         | RAPID3 的優勢                        |
| ------------ | ------------------------------------ |
| 繁忙門診     | 患者候診時完成，不佔診間時間         |
| 電子健康系統 | 可整合入 EMR/HIS，自動計算並顯示趨勢 |
| 遠距醫療     | 可於線上視訊前先由患者填寫           |
| 大規模追蹤   | 成本低，可高頻率追蹤（建議每月一次） |
| 患者教育     | 填寫過程本身即幫助患者了解自身病情   |

## RAPID3 與 DAS28 的近似換算

以下換算公式僅供參考（Pincus 2011），**不可取代直接計算**：

**DAS28 ≈ 0.302 × RAPID3 + 1.671**

| RAPID3 | 近似 DAS28 | 活動度對應   |
| ------ | ---------- | ------------ |
| ≤ 3    | ≈ 2.6      | 接近緩解     |
| 6      | ≈ 3.5      | 低至中活動度 |
| 12     | ≈ 5.3      | 中至高活動度 |
| 20     | ≈ 7.7      | 高活動度     |
| 30     | ≈ 10.7     | 極高活動度   |

## 使用建議

### 追蹤頻率

- **建議每月至少一次**，尤其在治療調整後
- 每次就診前完成，有助於醫師快速了解治療反應
- 建議固定在相同時間填寫（如晨起）以提高一致性

### 結果解讀原則

- **RAPID3 分數下降** → 治療有效，疾病活動度改善
- **RAPID3 分數上升** → 需要評估治療方案，考慮調整
- **分數穩定但偏高** → 可能需要積極介入
- **MCID（最小臨床意義差異）**：約 6 分的變化具有臨床意義

## 治療目標

依據 RAPID3 的 Treat-to-Target 原則：

| 目標                 | RAPID3 切截值     |
| -------------------- | ----------------- |
| 首要目標（接近緩解） | ≤ 3               |
| 替代目標（輕度）     | ≤ 6               |
| 需要調整治療         | > 6，未達輕度目標 |
| 需要積極介入         | > 12              |

## 注意事項

::: warning RAPID3 依賴患者主觀感受，可能受非關節炎因素影響
RAPID3 完全基於患者自評，疼痛感受和功能限制可能受到**合併纖維肌痛症、憂鬱、焦慮、睡眠障礙或其他慢性病**的影響，導致分數偏高但並非 RA 本身活動度上升。臨床上需結合客觀發炎指標（ESR/CRP）和關節檢查來綜合判斷。
:::

::: warning RAPID3 不含關節計數，無法偵測亞臨床滑膜炎
由於 RAPID3 不進行實際關節檢查，無法偵測患者主觀感覺不明顯但影像學（超音波或 MRI）可見的**亞臨床滑膜炎**，可能在部分患者中低估實際的關節損傷進展風險。高風險患者（如早期 RA、血清陽性）建議定期配合 DAS28 或影像學評估。
:::

::: warning 不同評估工具間的切截值不可混用
RAPID3（≤ 3 緩解）、CDAI（≤ 2.8 緩解）、DAS28（< 2.6 緩解）使用不同的切截值和分數範圍，在同一患者的縱向追蹤中應**固定使用同一工具**，不可混用切截值來判斷疾病活動度的變化。
:::

::: info 免責聲明
本計算器僅供臨床評估參考，不能取代完整的臨床評估、關節檢查及必要的實驗室和影像學檢查。RA 的治療決策應由風濕科醫師根據患者完整的臨床狀況及個別因素綜合判斷。建議每月評估一次，並將結果帶至回診時與醫師討論。
:::
