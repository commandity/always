# 類風濕關節炎疾病活動評估

**Rheumatoid Arthritis Disease Activity · RAPID3 & DAS28 & CDAI**

<RheumatoidArthritisCalculator />

## RAPID3 快速評估指數

**Routine Assessment of Patient Index Data 3**（Pincus）為純患者自填指數，不需關節數或抽血：MDHAQ 十項身體功能（0–30 ÷ 3）＋疼痛 VAS（0–10）＋整體健康 VAS（0–10），總分 0–30。

| RAPID3 分數 | 疾病活動度 | 意義與建議處置               |
| ----------- | ---------- | ---------------------------- |
| ≤ 3         | 接近緩解   | 症狀控制良好，維持現行治療   |
| 3.1–6       | 輕度       | 持續監測                     |
| 6.1–12      | 中度       | 考慮調整治療                 |
| > 12        | 高度       | 需積極治療，與醫師討論強化方案 |

> RAPID3 與 DAS28/CDAI 高度相關，卻只需 1–2 分鐘，適合門診常規追蹤與遠距照護；其限制是無關節客觀評估，纖維肌痛等非發炎性疼痛會使分數偏高。

## DAS28 疾病活動指數

**Disease Activity Score 28** 結合觸痛關節數（TJC28）、腫脹關節數（SJC28）、發炎指標（ESR 或 CRP）與患者整體評估（GH 0–100）：

| DAS28 分數 | 疾病活動度 | 意義與建議處置               |
| ---------- | ---------- | ---------------------------- |
| < 2.6      | 緩解       | 治療目標達成（T2T 首要目標） |
| 2.6–3.2    | 低度活動   | 接近緩解，密切追蹤           |
| 3.2–5.1    | 中度活動   | 考慮強化治療方案             |
| > 5.1      | 高度活動   | 積極調整治療，評估生物製劑   |

> **DAS28 < 2.6 是 Treat-to-Target 策略的首要目標**；無法達成時以 < 3.2（低度活動）為替代目標。ESR 版與 CRP 版分數不完全互換——ESR 受貧血、高齡影響較大，CRP 反映急性發炎較即時。

## CDAI 臨床疾病活動指數

**Clinical Disease Activity Index** 不需抽血，於診間即可完成：TJC28 ＋ SJC28 ＋患者整體（0–10）＋醫師整體（0–10），總分 0–76。

| CDAI 分數 | 疾病活動度 | 意義與建議處置               |
| --------- | ---------- | ---------------------------- |
| ≤ 2.8     | 緩解       | 治療目標達成，維持現有方案   |
| > 2.8–10  | 低度活動   | 接近緩解，維持並密切追蹤     |
| > 10–22   | 中度活動   | 需考慮強化治療方案           |
| > 22      | 高度活動   | 積極調整治療，評估生物製劑   |

> **三工具分工**：RAPID3 適合患者自填快速追蹤；CDAI 適合診間即時決策（免抽血）；DAS28 是臨床試驗與健保給付評估的標準指標。同一患者建議固定使用同一工具追蹤趨勢。

## 注意事項

::: warning 關節評估需一致性
TJC28/SJC28 應由受訓者以一致手法評估；DAS28 與 CDAI 皆不含足踝關節，足部症狀明顯者需另行記錄。
:::

::: info 免責聲明
本計算器僅供臨床評分記錄與教育參考，治療調整需由風濕免疫科醫師決定。
:::
