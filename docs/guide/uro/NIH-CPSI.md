# NIH-CPSI 慢性攝護腺炎症狀評分

**NIH-CPSI Chronic Prostatitis Symptom Score**

## 概述

**NIH-CPSI**（NIH Chronic Prostatitis Symptom Index，美國國家衛生研究院慢性攝護腺炎症狀評分）是評估慢性攝護腺炎／慢性骨盆疼痛症候群（CP/CPPS）的標準工具。由 Litwin 等人於 1999 年研發，為慢性攝護腺炎協作研究網絡（CPCRN）的核心結果指標。共 9 題，分為三大域，總分 0–43 分，廣泛應用於 CP/CPPS 的診斷、嚴重度分類及治療反應評估。

## 三大域評估架構

NIH-CPSI 涵蓋慢性攝護腺炎最重要的三個面向，各域可獨立解讀也可合計總分：

| 域別                  | 題目  | 計分範圍    | 評估內容                                                           |
| --------------------- | ----- | ----------- | ------------------------------------------------------------------ |
| **疼痛域（Pain）**    | Q1–Q4 | 0–21 分     | 疼痛部位（Q1）、射精疼痛（Q2）、疼痛頻率（Q3）、疼痛程度 NRS（Q4） |
| **排尿域（Urinary）** | Q5–Q6 | 0–10 分     | 排尿不盡感（Q5）、排尿頻率（Q6）                                   |
| **生活品質域（QoL）** | Q7–Q9 | 0–12 分     | 日常活動影響（Q7）、症狀佔據思緒（Q8）、整體生活品質（Q9）         |
| **NIH-CPSI 總分**     | Q1–Q9 | **0–43 分** | 三域合計                                                           |

## 各題計分詳解

### 疼痛域（Pain Domain, Q1–Q4）

**Q1 疼痛部位**（可複選，每個部位 1 分，最高 4 分）：

- 會陰部（肛門與陰囊之間）
- 睪丸
- 陰莖前端（非排尿時）
- 恥骨或膀胱區域

**Q2 射精疼痛**：有 = 1 分，無 = 0 分

**Q3 疼痛頻率**（0–4 分）：

| 分數 | 頻率描述 |
| ---- | -------- |
| 0    | 從來沒有 |
| 1    | 很少     |
| 2    | 有時候   |
| 3    | 常常     |
| 4    | 總是     |

**Q4 疼痛程度 NRS**（0–10 分）：0 = 完全不痛，10 = 能想像最嚴重的疼痛

> 疼痛域滿分 21 分 = Q1（0–4）+ Q2（0–1）+ Q3（0–4）+ Q4（0–10）+ Q2（0–1）

### 排尿域（Urinary Domain, Q5–Q6）

Q5 與 Q6 各以 6 點頻率量表（0–5 分）評估：0 = 完全沒有，5 = 幾乎每次

**Q5** 排尿後膀胱未完全排空感　**Q6** 排尿後 2 小時內又需再次排尿

> 排尿域反映 CP/CPPS 患者的 LUTS 成分，與 IPSS 有部分重疊，但在 CP/CPPS 患者中排尿域對生活品質的貢獻（r = 0.320）遠低於疼痛域（r = 0.678）。

### 生活品質域（QoL Domain, Q7–Q9）

**Q7 日常活動影響**（0–3 分）：0 = 完全沒有，3 = 很多

**Q8 症狀佔據思緒**（0–3 分）：0 = 完全沒有，3 = 很多

**Q9 整體生活品質**（0–6 分）：0 = 非常滿意（Delighted），6 = 非常難過（Terrible）

## NIH-CPSI 總分解釋

| 總分         | 嚴重程度 | 臨床建議                                                               |
| ------------ | -------- | ---------------------------------------------------------------------- |
| **0–14 分**  | ✅ 輕度  | 生活調整、NSAIDs 止痛、追蹤觀察                                        |
| **15–29 分** | ⚠️ 中度  | 藥物治療（α 阻斷劑、抗生素療程評估）、進一步泌尿科評估                 |
| **≥ 30 分**  | 🔴 重度  | 積極多模態介入（藥物＋物理治療＋心理支持），考慮轉介疼痛科或心身醫學科 |

### 疼痛域嚴重度分級（Pain Domain, 0–21 分）

| 疼痛域分數 | 疼痛嚴重度 |
| ---------- | ---------- |
| 0–7        | 輕度疼痛   |
| 8–13       | 中度疼痛   |
| 14–21      | 重度疼痛   |

> 疼痛域是預測整體 CP/CPPS 生活品質最重要的指標，優先針對疼痛介入通常可獲得最大的生活品質改善。

### 治療有效的最小改變量（MCID）

- **總分下降 ≥ 6 分**：臨床有意義的改善
- **疼痛域下降 ≥ 25%**：疼痛治療有效指標
- **用於臨床試驗**：通常以疼痛域 ≥ 50% 改善作為主要療效指標

## CP/CPPS 的 NIH 分類

NIH-CPSI 適用於以下 CP/CPPS 分類中的 III 型（最常見臨床類型）：

```
NIH 慢性攝護腺炎分類
├─ I 型：急性細菌性攝護腺炎（Acute bacterial prostatitis）
├─ II 型：慢性細菌性攝護腺炎（Chronic bacterial prostatitis）
├─ III 型：慢性攝護腺炎／慢性骨盆疼痛症候群（CP/CPPS）← NIH-CPSI 主要適用
│    ├─ IIIa 型：發炎型（尿液、攝護腺液含白血球）
│    └─ IIIb 型：非發炎型
└─ IV 型：無症狀性發炎型攝護腺炎（組織學或精液發現）
```

## 疼痛的深入評估

CP/CPPS 疼痛具有多樣性，Q1 的部位分布提供重要臨床線索：

| 疼痛部位 | 盛行率（n=1563） | 臨床意義                     |
| -------- | ---------------- | ---------------------------- |
| 會陰部   | 63%              | 最常見，與神經敏化相關       |
| 睪丸     | 58%              | 可能反映輸精管或副睪來源     |
| 射精疼痛 | 45%              | 影響性功能，降低生活品質明顯 |
| 排尿疼痛 | 43%              | 與膀胱頸功能障礙相關         |
| 恥骨區域 | 42%              | 可能合併間質性膀胱炎         |
| 陰莖前端 | 32%              | 常見於 IIIb 型               |

> 歐洲患者（vs 北美）有更多疼痛部位與更高疼痛評分（p<0.001），顯示疾病表現具地理差異。

## 治療策略

### 依 NIH-CPSI 分數選擇治療

#### NIH-CPSI 0–14 分（輕度）：保守治療

| 介入方式 | 說明                                               |
| -------- | -------------------------------------------------- |
| 生活調整 | 避免長時間坐姿、減少辛辣食物及咖啡因、規律有氧運動 |
| NSAIDs   | Ibuprofen 或 Celecoxib，短期止痛                   |
| 坐浴     | 溫水坐浴每日 2–3 次，改善骨盆血流                  |
| 追蹤觀察 | 3 個月後重新評估 NIH-CPSI                          |

#### NIH-CPSI 15–29 分（中度）：藥物治療 + 評估

**α 阻斷劑**（初治或症狀 < 1 年患者優先）：

- Tamsulosin 0.4 mg/d 或 Alfuzosin 10 mg/d
- 有效率約 40–60%，通常 4–6 週起效

**抗生素療程**（僅適用於 IIIa 型或初次發作）：

- Fluoroquinolone（ciprofloxacin 或 levofloxacin）4–6 週
- 長期反覆抗生素使用無充分證據支持

**植物萃取物**（輔助選項）：

- Quercetin、Pollen extract（Cernilton）有部分 RCT 支持

#### NIH-CPSI ≥ 30 分（重度）：多模態治療

- **物理治療**：骨盆底肌肉鬆弛訓練、肌筋膜放鬆，對 IIIb 型效果較佳
- **心理支持**：認知行為治療（CBT），改善疼痛災難化（pain catastrophizing）
- **神經調節**：薦骨神經調節（Sacral neuromodulation）用於難治型
- **疼痛科轉介**：慢性疼痛管理（神經阻斷、低劑量 amitriptyline）
- **5α 還原酶抑制劑**：攝護腺體積 > 30 mL 時合併使用

### UPOINT 表現型分類介入

CP/CPPS 的現代治療採 **UPOINT** 個人化治療架構，依症狀表現型選擇介入：

| UPOINT 類別     | 評估面向                   | 介入建議                 |
| --------------- | -------------------------- | ------------------------ |
| U（泌尿症狀）   | NIH-CPSI 排尿域            | α 阻斷劑                 |
| P（心理）       | 焦慮、憂鬱篩查             | CBT、抗憂鬱藥            |
| O（器官特異性） | 攝護腺壓痛、前列腺液白血球 | 抗生素、5-ARI            |
| I（感染）       | 尿液及攝護腺液培養         | 抗生素                   |
| N（神經）       | 疼痛放射、神經敏化         | 神經調節劑（gabapentin） |
| T（肌肉壓痛）   | 骨盆底肌肉壓痛             | 物理治療                 |

## 與 IPSS 的比較

| 特性           | IPSS               | NIH-CPSI              |
| -------------- | ------------------ | --------------------- |
| 主要適應症     | BPH 相關 LUTS      | CP/CPPS（NIH III 型） |
| 題數           | 7 + 1（QoL）       | 9（三大域）           |
| 疼痛評估       | 無                 | 核心（占最大比重）    |
| 總分範圍       | 0–35（+QoL 0–6）   | 0–43                  |
| 最重要預測因子 | IPSS 總分 + Q8 QoL | 疼痛域（Pain Domain） |

> 若患者同時有 BPH 相關症狀與骨盆疼痛，兩種量表可同時使用，但解讀時需注意症狀重疊。

## 注意事項

::: warning 診斷前提
NIH-CPSI 是症狀評估工具，不是診斷工具。使用前應排除急性細菌性攝護腺炎（NIH I 型）、慢性細菌性攝護腺炎（NIH II 型）、攝護腺癌、膀胱癌及間質性膀胱炎等器質性疾病。
:::

::: warning 疼痛域的核心地位
研究顯示疼痛域與生活品質的相關性（r = 0.678）遠高於排尿域（r = 0.320）。疼痛程度（r = 0.627）和疼痛頻率（r = 0.594）是最重要的生活品質預測因子。臨床評估應優先聚焦疼痛管理。
:::

::: info UPOINT 個人化治療
CP/CPPS 是高度異質性疾病，同樣的 NIH-CPSI 分數可能反映完全不同的病理機制。UPOINT 架構允許針對每位患者的優勢症狀表現型設計個人化治療，療效優於「一刀切」治療策略。
:::

::: info 重複評估的重要性
CP/CPPS 病程長且症狀波動，建議每 3 個月重新評估 NIH-CPSI。治療後總分下降 ≥ 6 分或疼痛域改善 ≥ 25% 視為有臨床意義的進步，可用於指引是否調整治療方案。
:::

::: info 免責聲明
本計算器依據 NIH-CPSI 量表設計，僅供臨床教育及輔助評估參考，不能取代泌尿科或疼痛科正式評估。NIH-CPSI 由 Litwin MS 等人於 1999 年開發，已翻譯為多種語言並在全球廣泛驗證。
:::
