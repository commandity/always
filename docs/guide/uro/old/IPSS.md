# IPSS 國際攝護腺症狀評分表

**IPSS International Prostate Symptom Score**

## 概述

**IPSS**（International Prostate Symptom Score，國際攝護腺症狀評分表）是評估男性下泌尿道症狀（LUTS）嚴重度的標準工具。由美國泌尿科學會（AUA）於 1992 年研發（原名 AUA-7），後經 WHO 納入第 8 題生活品質評估，成為目前全球最廣泛使用的攝護腺相關症狀評估量表。共 8 題，前 7 題計入總分（0–35 分），第 8 題（生活品質）單獨評估（0–6 分）。

## 評估項目與計分

IPSS 包含 **7 題症狀評估**（每題 0–5 分）與 **1 題生活品質評估**（0–6 分，不計入總分）。

### 症狀題目（Q1–Q7）

各題依過去**一個月**的頻率作答（0 = 完全沒有，5 = 幾乎每次），Q7 夜尿則計次數（0 = 不需起床，5 = ≥5 次）：

| 題號 | 症狀類型 | 評估內容                    | 計分範圍 |
| ---- | -------- | --------------------------- | -------- |
| Q1   | 儲尿症狀 | 排尿後仍有膀胱未排空感覺    | 0–5      |
| Q2   | 儲尿症狀 | 排尿後 2 小時內又需再次排尿 | 0–5      |
| Q3   | 排尿症狀 | 排尿時尿流斷斷續續          | 0–5      |
| Q4   | 儲尿症狀 | 難以延遲排尿（尿急感）      | 0–5      |
| Q5   | 排尿症狀 | 排尿時尿流無力              | 0–5      |
| Q6   | 排尿症狀 | 需要用力才能開始排尿        | 0–5      |
| Q7   | 儲尿症狀 | 夜間起床排尿次數            | 0–5      |

> **症狀域分類**：儲尿症狀（Q1、Q2、Q4、Q7）反映膀胱過動與逼尿肌不穩定；排尿症狀（Q3、Q5、Q6）反映膀胱出口阻塞與尿道阻力增加。

### 生活品質評估（Q8）

**Q8**：「如果您必須在餘生中都保持現在的泌尿狀況，您的感受如何？」
0 = 非常滿意（Delighted）→ 6 = 非常難過（Terrible）

Q8 **不計入 IPSS 總分**，但為治療決策最重要的依據之一。即使 IPSS 分數偏低，若 Q8 ≥ 4，仍建議積極介入治療。

## IPSS 總分解釋

| 總分         | 嚴重程度            | 臨床建議                                                                 |
| ------------ | ------------------- | ------------------------------------------------------------------------ |
| **0–7 分**   | ✅ 輕度（Mild）     | 觀察等待（Watchful waiting）、生活調整（飲水控制、避免咖啡因）、定期追蹤 |
| **8–19 分**  | ⚠️ 中度（Moderate） | 藥物治療（α 阻斷劑或 5α 還原酶抑制劑），評估尿流速及殘尿量               |
| **20–35 分** | 🔴 重度（Severe）   | 泌尿科轉介，評估手術治療指標（TURP 或微創手術）                          |

> IPSS 總分與尿流速、殘尿量及攝護腺體積的相關性有限，不能單獨作為治療選擇依據，需結合 Q8 生活品質評分與客觀檢查結果。

## 評估流程

```
初診患者主訴下泌尿道症狀（LUTS）
              ↓
        IPSS 量表評估
（7 題症狀 + Q8 生活品質）
              ↓
      基本評估（依 AUA 指引）
   ├─ 病史詢問（排尿日誌、夜尿頻率）
   ├─ 身體檢查（肛門指診 DRE）
   ├─ 尿液分析（排除感染或血尿）
   └─ 血清 PSA（若懷疑攝護腺癌）
              ↓
      依嚴重度選擇介入
   ├─ IPSS 0–7：觀察等待 + 生活調整
   ├─ IPSS 8–19：藥物治療
   │    ├─ α 阻斷劑（tamsulosin、silodosin、alfuzosin）
   │    └─ 5α 還原酶抑制劑（finasteride、dutasteride，適合攝護腺 >30g）
   └─ IPSS 20–35：評估手術
        ├─ TURP（黃金標準）
        ├─ 綠光雷射氣化術（PVP）
        └─ 微創手術（UroLift、水蒸氣消融 Rezum）
              ↓
      追蹤（3–6 個月後重新評估 IPSS）
```

## 兩大症狀域詳解

### 儲尿症狀（Storage Symptoms，Q1、Q2、Q4、Q7）

儲尿症狀主要反映**膀胱逼尿肌過動**（OAB）或**膀胱容量下降**：

| 症狀           | 機制                           | 臨床意義                  |
| -------------- | ------------------------------ | ------------------------- |
| 頻尿（Q2）     | 膀胱容量有效降低               | 常見於 BPH 繼發性膀胱變化 |
| 尿急感（Q4）   | 逼尿肌不自主收縮               | 預測手術效果較差          |
| 夜尿（Q7）     | 夜間尿液產生增加或膀胱容量下降 | 與生活品質最相關          |
| 排尿不盡（Q1） | 殘尿增加                       | 可能增加泌尿道感染風險    |

### 排尿症狀（Voiding Symptoms，Q3、Q5、Q6）

排尿症狀反映**膀胱出口阻塞**（BOO）：

| 症狀             | 機制                     | 臨床意義                      |
| ---------------- | ------------------------ | ----------------------------- |
| 尿流無力（Q5）   | 尿道阻力增加             | 與 Qmax 相關性最強            |
| 間歇性排尿（Q3） | 逼尿肌收縮力不足合併阻塞 | 可見於長期 BPH 造成逼尿肌受損 |
| 排尿費力（Q6）   | 膀胱頸或尿道阻塞         | 術前評估重要指標              |

## 藥物治療指引

### α 阻斷劑（Alpha-blockers）

適用於 IPSS 8–19 分，症狀快速改善首選：

| 藥物       | 劑量     | 特點                     |
| ---------- | -------- | ------------------------ |
| Tamsulosin | 0.4 mg/d | 選擇性高，姿勢性低血壓少 |
| Silodosin  | 8 mg/d   | 高選擇性，射精障礙較多   |
| Alfuzosin  | 10 mg/d  | 較少影響血壓             |
| Doxazosin  | 4–8 mg/d | 合併高血壓患者優先       |

> α 阻斷劑通常在 1–2 週內改善症狀，平均可降低 IPSS 4–6 分。

### 5α 還原酶抑制劑（5-ARIs）

適用於攝護腺體積 > 30 mL 或 PSA > 1.5 ng/mL：

| 藥物        | 劑量     | 特點                             |
| ----------- | -------- | -------------------------------- |
| Finasteride | 5 mg/d   | 縮小攝護腺 15–20%，需 6 個月起效 |
| Dutasteride | 0.5 mg/d | 同時抑制 Type 1 & 2，效果略佳    |

> 5-ARIs 可將 PSA 降低約 50%，評估 PSA 時需 ×2 校正。

### 合併治療（Combination Therapy）

IPSS ≥ 13 分且攝護腺體積 > 30 mL → α 阻斷劑 + 5-ARI 合併療法（MTOPS / CombAT 研究支持）。

## 手術治療指標

以下情形建議直接泌尿科轉介評估手術：

| 絕對適應症                 | 相對適應症                           |
| -------------------------- | ------------------------------------ |
| 急性尿滯留（AUR）          | IPSS 重度（20–35）且藥物治療無效     |
| 反覆尿滯留                 | Q8 ≥ 4 且患者強烈希望手術            |
| 反覆血尿（排除其他原因）   | Qmax < 10 mL/s 合併症狀              |
| 腎功能受損（梗阻性腎病變） | 大量殘尿（PVR > 300 mL）合併反覆感染 |
| 膀胱結石                   | 攝護腺體積極大（> 80–100 mL）        |

## 追蹤與療效評估

- **觀察等待**：每年重新評估 IPSS
- **藥物治療**：3 個月後評估症狀改善（IPSS 下降 ≥ 25–30% 視為有效）
- **手術後**：3 個月及 12 個月分別重新評估 IPSS
- **有意義的最小改變量（MCID）**：IPSS 下降 3 分（感知有改善）

> 夜尿（Q7）對生活品質影響最大，若夜尿改善不理想，需評估是否合併夜間多尿症（Nocturnal polyuria）。

## 注意事項

::: warning IPSS 的局限性
IPSS 為症狀評估工具，不能診斷 BPH，亦不能確認膀胱出口阻塞。症狀與客觀指標（尿流速、殘尿量、攝護腺體積）相關性有限。女性亦可使用 IPSS 評估 LUTS，但解釋需考量不同的解剖生理。
:::

::: warning Q8 生活品質評分的重要性
Q8 是治療決策的核心依據。即使 IPSS 屬輕度（0–7），若 Q8 ≥ 4（不滿意以上），表示症狀已嚴重影響生活品質，應積極討論治療選項，不能僅依據 IPSS 分數決定是否治療。
:::

::: info 與 NIH-CPSI 的區別
IPSS 主要評估 BPH 相關的 LUTS，著重排尿症狀與膀胱出口阻塞；NIH-CPSI 則專門評估慢性攝護腺炎／慢性骨盆疼痛症候群，特別強調疼痛域。兩者不可互換使用。
:::

::: info 台灣臨床應用
IPSS 已納入台灣泌尿科學會（TUA）BPH 臨床指引，建議於初診時常規評估，並於每次追蹤時重新填寫以監測疾病進展與治療反應。
:::

::: info 免責聲明
本計算器依據 IPSS 量表設計，僅供臨床教育及輔助評估參考，不能取代泌尿科正式評估。IPSS 由美國泌尿科學會（AUA）開發，WHO 採納為全球標準。
:::
