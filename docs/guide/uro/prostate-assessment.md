## IPSS 國際攝護腺症狀評分

**International Prostate Symptom Score（IPSS）** 為 7 題排尿症狀量表（各 0–5 分，總分 0–35）加 1 題生活品質評估（0–6，不計入總分），用於評估良性攝護腺肥大（BPH）下尿路症狀嚴重度。

| 項目                 | 分類     |
| -------------------- | -------- |
| 排空不全感、頻尿、間歇、急迫、滴尿感 | 儲尿 / 排尿症狀 |
| 尿流細弱、需用力     | 排尿症狀 |
| 夜尿                 | 儲尿症狀（0–5 對應次數） |

### 分數意義與處置

| 總分  | 嚴重度 | 意義與建議處置                                   |
| ----- | ------ | ------------------------------------------------ |
| 0–7   | 輕度   | 症狀輕微，觀察追蹤、生活型態調整                 |
| 8–19  | 中度   | 建議藥物治療（α 阻斷劑、5-ARI），追蹤反應        |
| 20–35 | 重度   | 症狀顯著，評估藥物或手術（TURP 等）              |

> **IPSS 主要反映症狀負擔**，與攝護腺體積、尿流速並非完全相關。第 8 題「生活品質」（0–6）是重要的治療決策依據——即使總分中度，若病人對症狀高度困擾（QoL ≥ 4），也應積極治療。IPSS 常用於追蹤治療前後的症狀變化。

## NIH-CPSI 慢性攝護腺炎症狀評分

**NIH Chronic Prostatitis Symptom Index（NIH-CPSI）** 為 9 題量表，分三個面向評估慢性攝護腺炎 / 慢性骨盆疼痛症候群（CP/CPPS）：

| 面向             | 題目    | 分數範圍 |
| ---------------- | ------- | -------- |
| 疼痛（Pain）     | Q1–Q4   | 0–21     |
| 排尿（Urinary）  | Q5–Q6   | 0–10     |
| 生活品質（QoL）  | Q7–Q9   | 0–12     |

### 分數意義與處置

| 疼痛分數 | 嚴重度 | 意義                                     |
| -------- | ------ | ---------------------------------------- |
| 0–7      | 輕度   | 疼痛負擔輕微                             |
| 8–14     | 中度   | 疼痛明顯，影響生活                       |
| 15–21    | 重度   | 疼痛嚴重，需積極多模式治療               |

> **NIH-CPSI 的疼痛面向（Q1–Q4）最具鑑別力**，其中 Q4 為 0–10 疼痛強度評分。總分（0–43）用於追蹤治療反應：**分數下降 ≥ 6 分**通常視為有臨床意義的改善。CP/CPPS 治療常採 UPOINT 表型導向的多模式策略；症狀量表不能取代排除感染、癌症等其他病因的評估。

## 注意事項

::: warning 症狀量表非診斷
IPSS 與 NIH-CPSI 量化症狀嚴重度與生活品質影響，不能單獨作為診斷；需結合病史、理學檢查與必要檢驗。
:::

::: info 免責聲明
本工具僅供臨床輔助與教學參考，最終診斷與治療應由醫師依完整評估判定。
:::
