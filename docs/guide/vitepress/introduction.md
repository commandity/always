# Introduction

Welcome to **MyDocs** — a documentation site built with VitePress.

## What is VitePress?

VitePress is a static site generator designed specifically for documentation. It takes your Markdown files and turns them into a fast, beautiful site with:

- Dark / light mode toggle (top-right corner)
- Collapsible sidebar for navigation
- Full-text search with `⌘K` shortcut
- Top navigation bar with links to main sections
- Mobile-responsive layout

## How this site is structured

```
docs/

```
