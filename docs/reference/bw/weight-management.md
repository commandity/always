# 減重指引
**Body Weight Management**

## 1. 評估

### 1.1 BMI 分級

| 分級                     | BMI (kg/m²) | 亞洲人建議 |
| ------------------------ | ----------- | ---------- |
| **體重過輕**             | < 18.5      | < 18.5     |
| **正常體重**             | 18.5–24.9   | 18.5–23.9  |
| **體重過重**             | 25.0–29.9   | 24.0–27.9  |
| **肥胖等級 I**           | 30.0–34.9   | ≥ 28.0     |
| **肥胖等級 II**          | 35.0–39.9   | ≥ 35.0     |
| **肥胖等級 III（重度）** | ≥ 40.0      | ≥ 40.0     |

> 亞洲人（含台灣）切點：BMI ≥ 27 為肥胖，≥ 28 為重度肥胖（部分指引）

### 1.2 腰圍與中心型肥胖

| 性別     | 切點（一般族群） | 切點（亞洲人/台灣） |
| -------- | ---------------- | ------------------- |
| **男性** | ≥ 102 cm         | ≥ 90 cm             |
| **女性** | ≥ 88 cm          | ≥ 80 cm             |

| 指標         | 計算方式    | 異常切點              |
| ------------ | ----------- | --------------------- |
| **腰臀比**   | 腰圍 / 臀圍 | 男 > 0.90 / 女 > 0.85 |
| **腰身高比** | 腰圍 / 身高 | ≥ 0.5（不分性別）     |

### 1.3 共病篩檢

| 系統         | 篩檢項目                                      | 工具          |
| ------------ | --------------------------------------------- | ------------- |
| **代謝**     | 空腹血糖、HbA1c、血脂、尿酸                   | 抽血          |
| **心血管**   | 血壓、靜態心電圖、Framingham/PREVENT 風險評估 | 抽血 + 問診   |
| **肝臟**     | AST/ALT、FIB-4、NAFLD 評分（FibroScan/US）    | 抽血 + 超音波 |
| **呼吸**     | STOP-BANG、Epworth 嗜睡量表、PSG（必要時）    | 問卷/睡眠檢查 |
| **内分泌**   | TSH、皮質醇（必要時排除繼發性肥胖）           | 抽血          |
| **骨骼肌肉** | 骨密度（BMI 極端或骨折風險者）                | DEXA          |
| **精神**     | Binge Eating Scale (BES)、PHQ-9、EQ-5D        | 問卷          |
| **營養**     | 營養不良篩檢（vitamin D、B12、鐵、葉酸）      | 抽血          |

### 1.4 腹部超音波適應症

- ALT > 1.5x ULN 或有代謝症候群者建議肝臟超音波
- FIB-4 > 1.3 考慮 FibroScan 進一步評估纖維化

### 1.5 繼發性肥胖排除

| 病因                 | 特徵                                                              | 確認方式                            |
| -------------------- | ----------------------------------------------------------------- | ----------------------------------- |
| **甲狀腺功能低下**   | 疲倦、怕冷、便秘、皮膚乾燥                                        | TSH ↑                               |
| **庫欣氏症候群**     | 滿月臉、水牛肩、紫紋、近端肌無力                                  | 24hr urine cortisol / overnight DST |
| **多囊性卵巢症候群** | 月經不規則、多毛、痤瘡                                            | 經超音波 + 睾固酮                   |
| **下視丘性肥胖**     | 腦傷/手術後、失明                                                 | 病史                                |
| **藥物性**           | 類固醇、thiazolidinediones、insulin、SSRI、mirtazapine、valproate | 用藥史                              |

## 2. 治療目標

### 2.1 體重減輕目標

| 目標             | 體重減輕幅度 | 臨床意義                         |
| ---------------- | ------------ | -------------------------------- |
| **最低有效目標** | ≥ 5% TBWL    | 改善血糖、血脂、血壓             |
| **有意義目標**   | ≥ 10% TBWL   | 顯著改善 NAFLD、OSA、關節症狀    |
| **最佳目標**     | ≥ 15% TBWL   | 可媲美代謝手術部分效果           |
| **藥物最高達成** | ≥ 20% TBWL   | Tirzepatide (Zepbound) 可達 ~21% |

> TBWL = Total Body Weight Loss

### 2.2 治療目標分層

| 初始 BMI                | 建議減重目標        | 首選治療                                    |
| ----------------------- | ------------------- | ------------------------------------------- |
| **27–29.9**（合併共病） | ≥ 5–10%             | 生活方式調整 ± 口服藥物                     |
| **30–34.9**             | ≥ 10%               | 口服藥物或 GLP-1 RA                         |
| **35–39.9**             | ≥ 10–15%            | GLP-1 RA 或 tirzepatide；若共病嚴重考慮手術 |
| **≥ 40**                | ≥ 15–20% 或代謝手術 | 代謝手術（首選）或高強度藥物治療            |

### 2.3 血糖共病目標

| 指標      | 目標                       | 備註                          |
| --------- | -------------------------- | ----------------------------- |
| **HbA1c** | < 7.0%（個體化）           | 有低血糖風險者可放寬至 < 8.0% |
| **體重**  | 與非糖尿病患者相同減重目標 | 使用具減重效果之降血糖藥      |
| **血壓**  | < 130/80 mmHg              |                               |
| **LDL**   | < 100 mg/dL（ASCVD：< 70） |                               |

## 3. 治療原則

### 3.1 治療階梯

| 階段        | BMI               | 生活方式  | 口服藥物                                  | 注射藥物                         | 代謝手術      |
| ----------- | ----------------- | --------- | ----------------------------------------- | -------------------------------- | ------------- |
| **第 1 階** | 25–27（合併共病） | ✓（必須） | 考慮                                      | —                                | —             |
| **第 2 階** | 27–29.9           | ✓         | ✓（Orlistat/Phentermine/Contrave/Qsymia） | 考慮                             | —             |
| **第 3 階** | 30–34.9           | ✓         | ✓                                         | ✓（GLP-1 RA 或 Tirzepatide）     | 考慮          |
| **第 4 階** | 35–39.9           | ✓         | ✓                                         | ✓（高劑量 GLP-1 RA/Tirzepatide） | ✓（合併共病） |
| **第 5 階** | ≥ 40              | ✓         | ✓                                         | ✓                                | ✓（首選）     |

### 3.2 核心治療原則

| 原則                 | 說明                                                             |
| -------------------- | ---------------------------------------------------------------- |
| **長期治療慢性病**   | 肥胖是慢性疾病，需長期治療（類似糖尿病/高血壓），非短期介入      |
| **生活方式為基礎**   | 飲食 + 運動 + 行為改變為所有治療的基石（不可或缺）               |
| **個體化治療**       | 根據 BMI、共病、偏好、經濟、藥物耐受性選擇治療                   |
| **多模式聯合**       | 單一治療不足時，應升階或聯合治療（藥物 + 行為 + 手術可並行）     |
| **藥物不應停**       | 體重達標後應持續用藥，停藥後體重通常會回彈（代謝適應）           |
| **以共病改善為導向** | 不只看體重數字，更重要的是代謝、心血管、關節、呼吸、生活品質改善 |
| **營養師介入**       | 所有減重治療都應搭配專業營養諮詢                                 |

### 3.3 飲食介入

| 策略             | 熱量赤字           | 適用場景                              |
| ---------------- | ------------------ | ------------------------------------- |
| **低熱量飲食**   | 1200–1500 kcal/day | 一般減重                              |
| **極低熱量飲食** | < 800 kcal/day     | BMI ≥ 30 + 共病需快速減重；需醫療監督 |
| **間歇性斷食**   | 16:8 或 5:2        | 部分患者可行                          |
| **地中海飲食**   | 減少飽和脂肪       | 代謝症候群、NAFLD                     |
| **低碳水化合物** | 碳水 < 130g/day    | 糖尿病、胰島素阻抗                    |

### 3.4 運動建議

| 類型             | 頻率               | 效果                   |
| ---------------- | ------------------ | ---------------------- |
| **中等強度有氧** | ≥ 150 min/week     | 心肺功能、體脂減少     |
| **高強度間歇**   | 75 min/week        | 更高效率減脂           |
| **阻力訓練**     | 2–3 sessions/week  | 維持肌肉量、提升代謝率 |
| **日常活動**     | 每日 8000–10000 步 | 降低久坐風險           |

## 9. 代謝手術

### 9.1 手術適應症

| 準則            | 適應症                                            |
| --------------- | ------------------------------------------------- |
| **NIH（一般）** | BMI ≥ 40 或 BMI ≥ 35 + 共病                       |
| **亞洲人/台灣** | BMI ≥ 37 或 BMI ≥ 32 + 共病                       |
| **糖尿病患者**  | BMI ≥ 30（或亞洲人 ≥ 27.5）— 代謝手術可改善血糖   |
| **青少年**      | BMI ≥ 120% 95th percentile + 嚴重共病（個案評估） |

### 9.2 手術方式比較

| 手術                     | 機制                      | 體重減輕 | 糖尿病緩解率 | 常見併發症                                |
| ------------------------ | ------------------------- | -------- | ------------ | ----------------------------------------- |
| **Roux-en-Y 胃繞道手術** | 限制 + 吸收不良           | 25–35%   | 60–80%       | 吻合口瘻管、邊緣性潰瘍、dumping、營養缺乏 |
| **袖狀胃切除術**         | 限制 + 荷爾蒙（ghrelin↓） | 20–30%   | 50–60%       | 胃食道逆流、狹窄、瘻管                    |
| **可調整式胃束帶術**     | 純限制                    | 15–20%   | 30–40%       | 滑脫、侵蝕、胃食道逆流、需定期調整        |
| **膽胰分流術**           | 限制 + 吸收不良（最大）   | 30–40%   | 80–95%       | 營養嚴重缺乏、腹瀉、蛋白質不足            |

### 9.3 術後追蹤

| 時間            | 追蹤項目                                  |
| --------------- | ----------------------------------------- |
| **術後 2 週**   | 傷口、進食耐受、嘔吐                      |
| **術後 1 個月** | 體重、營養狀態                            |
| **術後 3 個月** | 體重、Vitamin B12、鐵、鈣、Vitamin D、PTH |
| **術後 6 個月** | 全面代謝評估                              |
| **每年**        | 體重、營養、骨密度、甲狀腺、賀爾蒙        |

## 10. 監測與追蹤

### 10.1 藥物治療監測

| 時間點          | 追蹤項目                                                                             |
| --------------- | ------------------------------------------------------------------------------------ |
| **用藥前**      | BMI、腰圍、血壓、抽血（HbA1c、lipids、LFTs、TSH、Creatinine）、心電圖（phentermine） |
| **用藥後 4 週** | 體重、副作用耐受性、血壓（phentermine）                                              |
| **每 3 個月**   | 體重、BMI、腰圍、血壓、HbA1c（糖尿病）、LFTs                                         |
| **每 6 個月**   | 全面代謝評估（lipids、uric acid、LFTs、creatinine）                                  |
| **每年**        | 完整健檢 + 營養評估（vitamin D、B12、iron）                                          |

### 10.2 藥物調整原則

| 情境                   | 處理                                                |
| ---------------------- | --------------------------------------------------- |
| **3 個月內體重無變化** | 確認飲食順從性 → 滴定至最高劑量 → 考慮換藥或加藥    |
| **達到目標體重**       | 維持目前劑量，不減量                                |
| **無法耐受副作用**     | 滴定回較低劑量 → 重新緩慢滴定 → 若仍不耐受則換藥    |
| **體重回彈**           | 確認用藥順從性 → 升階治療（如 oral → injectable）   |
| **GLP-1 RA 失敗**      | 考慮 Tirzepatide（若未使用）→ 聯合口服藥 → 手術評估 |

### 10.3 停藥後體重管理

| 策略             | 說明                               |
| ---------------- | ---------------------------------- |
| **持續用藥**     | 首選策略，停藥後幾乎所有患者會回彈 |
| **緩慢減量**     | 非標準建議，但可考慮逐漸減量過渡   |
| **加強生活行為** | 停藥後需更積極飲食/運動維持        |
| **重新用藥**     | 體重回彈 ≥ 50% 時考慮重新啟動藥物  |
| **代謝手術**     | 長期需求者考慮手術作為永久解決方案 |
