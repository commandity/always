# 減重完整指引

**Comprehensive Weight Management Guide**

## 概述

減重治療涵蓋評估、目標設定、生活型態介入、口服與注射藥物、代謝手術及長期監測。肥胖為慢性疾病，需長期治療（類似糖尿病/高血壓），非短期介入。本文整理 2025–2026 年最新臨床證據，聚焦臺灣可用藥物。

> 2025 臺灣共識：BMI ≥ 27 為肥胖，≥ 28 為重度肥胖。有 ASCVD / HF / CKD 者，除減重治療外，應同時考慮 SGLT2i 或 GLP-1 RA 器官保護。

## 減重藥物發展史

減少熱量攝入最直接的方法就是減少食物攝取，或降低食慾。減重藥物的發展可追溯至 19 世紀末：

| 年代     | 藥物/事件                 | 說明                                                             | 結局           |
| -------- | ------------------------- | ---------------------------------------------------------------- | -------------- |
| **1885** | 麻黃鹼（ephedrine）       | 日本化學家長井長義純化，具有交感神經興奮作用，副作用包括食慾下降 | 歷史用途       |
| **1938** | 安非他命（amphetamine）   | Lesses & Myerson 於 NEJM 發表可有效降低食慾及體重                | 濫用、成癮問題 |
| **1973** | Fenfluramine 上市         | 透過調節血清素（5-HT）抑制食慾                                   | 減重效果不彰   |
| **1992** | Fen-phen 臨床試驗         | Fenfluramine + Phentermine 複方效果優於單方                      | —              |
| **1994** | Fen-phen（芬芬）上市      | 複方減重藥上市                                                   | —              |
| **1997** | Fen-phen 下市             | 數百例出現瓣膜性心臟病、肺高壓，甚至死亡                         | **下市**       |
| **1997** | Sibutramine（諾美婷）上市 | 透過調節血清素抑制食慾                                           | —              |
| **2010** | Sibutramine 下架          | 大型試驗發現增加心肌梗塞與腦中風風險                             | **下架**       |
| **2012** | Lorcaserin（沛麗婷）上市  | 選擇性 5-HT2C 促效劑                                             | —              |
| **2020** | Lorcaserin 下架           | 可能增加癌症發生率、血清素症候群風險                             | **下架**       |

> **歷史教訓**：早期減重藥物多為交感神經興奮劑或血清素調節劑，因嚴重心血管毒性、濫用成癮、甚至致癌等問題相繼下市。目前尚無一個既安全又有效的減重藥物，皆需更多大型臨床試驗以證實長期安全性。

## 治療目標

### 體重減輕目標

| 目標             | 體重減輕幅度 | 臨床意義                         |
| ---------------- | ------------ | -------------------------------- |
| **最低有效目標** | ≥ 5% TBWL    | 改善血糖、血脂、血壓             |
| **有意義目標**   | ≥ 10% TBWL   | 顯著改善 NAFLD、OSA、關節症狀    |
| **最佳目標**     | ≥ 15% TBWL   | 可媲美代謝手術部分效果           |
| **藥物最高達成** | ≥ 20% TBWL   | Tirzepatide (Zepbound) 可達 ~21% |

> TBWL = Total Body Weight Loss

### 治療目標分層

| 初始 BMI                | 建議減重目標        | 首選治療                                    |
| ----------------------- | ------------------- | ------------------------------------------- |
| **27–29.9**（合併共病） | ≥ 5–10%             | 生活方式調整 ± 口服藥物                     |
| **30–34.9**             | ≥ 10%               | 口服藥物或 GLP-1 RA                         |
| **35–39.9**             | ≥ 10–15%            | GLP-1 RA 或 tirzepatide；若共病嚴重考慮手術 |
| **≥ 40**                | ≥ 15–20% 或代謝手術 | 代謝手術（首選）或高強度藥物治療            |

### 血糖共病目標

| 指標      | 目標                       | 備註                          |
| --------- | -------------------------- | ----------------------------- |
| **HbA1c** | < 7.0%（個體化）           | 有低血糖風險者可放寬至 < 8.0% |
| **體重**  | 與非糖尿病患者相同減重目標 | 使用具減重效果之降血糖藥      |
| **血壓**  | < 130/80 mmHg              |                               |
| **LDL**   | < 100 mg/dL（ASCVD：< 70） |                               |

## 治療原則

### 治療階梯

| 階段        | BMI               | 生活方式  | 口服藥物           | 注射藥物                         | 代謝手術      |
| ----------- | ----------------- | --------- | ------------------ | -------------------------------- | ------------- |
| **第 1 階** | 25–27（合併共病） | ✓（必須） | 考慮               | —                                | —             |
| **第 2 階** | 27–29.9           | ✓         | ✓（Orlistat 惠纖） | 考慮                             | —             |
| **第 3 階** | 30–34.9           | ✓         | ✓                  | ✓（GLP-1 RA 或 Tirzepatide）     | 考慮          |

| **第 5 階** | ≥ 40              | ✓         | ✓                  | ✓                                | ✓（首選）     |

### 核心治療原則

| 原則                 | 說明                                                             |
| -------------------- | ---------------------------------------------------------------- |
| **長期治療慢性病**   | 肥胖是慢性疾病，需長期治療（類似糖尿病/高血壓），非短期介入      |
| **生活方式為基礎**   | 飲食 + 運動 + 行為改變為所有治療的基石（不可或缺）               |
| **個體化治療**       | 根據 BMI、共病、偏好、經濟、藥物耐受性選擇治療                   |
| **多模式聯合**       | 單一治療不足時，應升階或聯合治療（藥物 + 行為 + 手術可並行）     |
| **藥物不應停**       | 體重達標後應持續用藥，停藥後體重通常會回彈（代謝適應）           |
| **以共病改善為導向** | 不只看體重數字，更重要的是代謝、心血管、關節、呼吸、生活品質改善 |
| **營養師介入**       | 所有減重治療都應搭配專業營養諮詢                                 |

### 飲食介入

| 策略             | 熱量赤字           | 適用場景                              |
| ---------------- | ------------------ | ------------------------------------- |
| **低熱量飲食**   | 1200–1500 kcal/day | 一般減重                              |
| **極低熱量飲食** | < 800 kcal/day     | BMI ≥ 30 + 共病需快速減重；需醫療監督 |
| **間歇性斷食**   | 16:8 或 5:2        | 部分患者可行                          |
| **地中海飲食**   | 減少飽和脂肪       | 代謝症候群、NAFLD                     |
| **低碳水化合物** | 碳水 < 130g/day    | 糖尿病、胰島素阻抗                    |

### 運動建議

| 類型             | 頻率               | 效果                   |
| ---------------- | ------------------ | ---------------------- |
| **中等強度有氧** | ≥ 150 min/week     | 心肺功能、體脂減少     |
| **高強度間歇**   | 75 min/week        | 更高效率減脂           |
| **阻力訓練**     | 2–3 sessions/week  | 維持肌肉量、提升代謝率 |
| **日常活動**     | 每日 8000–10000 步 | 降低久坐風險           |

## 口服藥物

### Orlistat（Xenical 惠纖 / Alli 艾麗）

| 項目       | Orlistat 120 mg（Xenical 惠纖）            | Orlistat 60 mg（Alli 艾麗 OTC） |
| ---------- | ------------------------------------------ | ------------------------------- |
| 作用機轉   | 脂肪酶抑制劑，↓ 腸道脂肪吸收               | 同，低劑量                      |
| 劑量       | 每日 3 次（隨餐）                          | 每日 3 次（隨餐）               |
| 體重減輕   | ~5%                                        | ~3–5%                           |
| 常見副作用 | 油便、排氣帶油、大便失禁、脂溶性維生素缺乏 | 同上但較輕                      |
| 禁忌症     | 膽汁鬱積、吸收不良症候群                   | 同上                            |

> **Orlistat 定位**：唯一 OTC 減重藥物，無全身吸收，不影響心血管。主要限制為 GI 副作用（油便）及脂溶性維生素缺乏。應補充 fat-soluble vitamins（A, D, E, K）。

### Phentermine（Duromine 芬特明）

| 項目       | Phentermine HCl                         |
| ---------- | --------------------------------------- |
| 作用機轉   | 交感神經促效劑（安非他命類），↓ 食慾    |
| 劑量       | 每日 1 次（15–37.5 mg）                 |
| 體重減輕   | 5–7%                                    |
| 常見副作用 | 心悸、血壓↑、失眠、口乾、焦慮           |
| 禁忌症     | 心血管疾病、甲亢、青光眼、MAO inhibitor |
| 適應症 BMI | 僅短期（≤ 12 週）                       |
| 臺灣可取得 | ✗（未核可）                             |

> ⚠️ **Phentermine 臺灣未核可**：結構上類似安非他命（amphetamine-like agent），在腦中會增加中樞性正腎上腺素（NE）及多巴胺（Dopamine）釋放，在達到抑制食慾的同時，可能也會有強烈的交感神經興奮作用導致致命心血管毒性，以及未知程度的精神病症障礙。臺灣食藥署未核可上市。

## 代謝手術

### 手術適應症

| 準則                  | 適應症                                            |
| --------------------- | ------------------------------------------------- |
| **臺灣（2022 準則）** | BMI ≥ 37 或 BMI ≥ 32 + 共病                       |
| **糖尿病患者**        | BMI ≥ 30（或亞洲人 ≥ 27.5）— 代謝手術可改善血糖   |
| **青少年**            | BMI ≥ 120% 95th percentile + 嚴重共病（個案評估） |

### 手術方式比較

| 手術                     | 機制                      | 體重減輕 | 糖尿病緩解率 | 常見併發症                                |
| ------------------------ | ------------------------- | -------- | ------------ | ----------------------------------------- |
| **Roux-en-Y 胃繞道手術** | 限制 + 吸收不良           | 25–35%   | 60–80%       | 吻合口瘻管、邊緣性潰瘍、dumping、營養缺乏 |
| **袖狀胃切除術**         | 限制 + 荷爾蒙（ghrelin↓） | 20–30%   | 50–60%       | 胃食道逆流、狹窄、瘻管                    |
| **可調整式胃束帶術**     | 純限制                    | 15–20%   | 30–40%       | 滑脫、侵蝕、胃食道逆流、需定期調整        |
| **膽胰分流術**           | 限制 + 吸收不良（最大）   | 30–40%   | 80–95%       | 營養嚴重缺乏、腹瀉、蛋白質不足            |

### 術後追蹤

| 時間            | 追蹤項目                                  |
| --------------- | ----------------------------------------- |
| **術後 2 週**   | 傷口、進食耐受、嘔吐                      |
| **術後 1 個月** | 體重、營養狀態                            |
| **術後 3 個月** | 體重、Vitamin B12、鐵、鈣、Vitamin D、PTH |
| **術後 6 個月** | 全面代謝評估                              |
| **每年**        | 體重、營養、骨密度、甲狀腺、賀爾蒙        |

## 監測與追蹤

### 藥物治療監測

| 時間點          | 追蹤項目                                                                             |
| --------------- | ------------------------------------------------------------------------------------ |
| **用藥前**      | BMI、腰圍、血壓、抽血（HbA1c、lipids、LFTs、TSH、Creatinine）、心電圖（phentermine） |
| **用藥後 4 週** | 體重、副作用耐受性、血壓（phentermine）                                              |
| **每 3 個月**   | 體重、BMI、腰圍、血壓、HbA1c（糖尿病）、LFTs                                         |
| **每 6 個月**   | 全面代謝評估（lipids、uric acid、LFTs、creatinine）                                  |
| **每年**        | 完整健檢 + 營養評估（vitamin D、B12、iron）                                          |

### 藥物微調原則

| 情境                   | 處理                                                |
| ---------------------- | --------------------------------------------------- |
| **3 個月內體重無變化** | 確認飲食順從性 → 微調至最高劑量 → 考慮換藥或加藥    |
| **達到目標體重**       | 維持目前劑量，不減量                                |
| **無法耐受副作用**     | 微調回較低劑量 → 重新緩慢微調 → 若仍不耐受則換藥    |
| **體重回彈**           | 確認用藥順從性 → 升階治療（如 oral → injectable）   |
| **GLP-1 RA 失敗**      | 考慮 Tirzepatide（若未使用）→ 聯合口服藥 → 手術評估 |

### 停藥後體重管理

| 策略             | 說明                               |
| ---------------- | ---------------------------------- |
| **持續用藥**     | 首選策略，停藥後幾乎所有患者會回彈 |
| **緩慢減量**     | 非標準建議，但可考慮逐漸減量過渡   |
| **加強生活行為** | 停藥後需更積極飲食/運動維持        |
| **重新用藥**     | 體重回彈 ≥ 50% 時考慮重新啟動藥物  |
| **代謝手術**     | 長期需求者考慮手術作為永久解決方案 |
