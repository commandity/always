# 氣喘 **Asthma**

## 1. 總論與流行病學

### 1.1 定義

氣喘是一種存在於兒童及成人的**慢性呼吸道疾病**，呼吸道因長期發炎而導致反覆出現氣道阻塞（airway obstruction）的臨床症狀，症狀的表現和嚴重度常因人及因時而異。氣喘為**異質性很大**的疾病，主要特徵為氣道的慢性發炎，具有兩項典型特徵：

1. **具有呼吸道症狀之病史**：如呼吸時出現喘鳴（wheezing）、呼吸短促、胸悶及咳嗽等症狀，其嚴重度隨時間變化。
2. **出現吐氣的氣流受阻**，且此一氣流受阻具有**可變性**。

### 1.2 流行病學數字

- 2017 年全球氣喘盛行率約 **3.57%**；WHO 估計 2019 年全世界約有 **2.62 億人**罹患氣喘。
- Asthma affects approximately **334 million people worldwide**（不分年齡種族之別，但各國流行病學不同）。
- **台灣成人**：2000–2011 年健保資料庫顯示，>18 歲成人氣喘盛行率由 2000 年的 **7.57%** 上升至 2011 年 **11.53%**。（另一統計：2000–2007 年健保資料庫約 **11.9%**）
- **台灣兒童及青少年（<20 歲）**：盛行率約 **15.7%**。
- ISAAC 研究：全世界約 **14%** 孩童可能有氣喘相關症狀。
- GINA 2017：高盛行率地區——拉丁美洲、澳洲、北美、南非；低盛行率地區——印度、亞太、環地中海東部、北東歐；非洲盛行率通常偏低。

- **107 年度全民健保醫療統計年報**：0–14 歲氣喘病人門住診（含急診）約**28 萬人**（男 16 萬 6 千、女 11 萬 4 千）。

### 1.3 氣喘的分類

**傳統三分類：**

1. **外因性／過敏性氣喘**：多數兒童氣喘屬此類；可定義出明確外界過敏原；異位性體質（atopic），多有家族史。
2. **內因性氣喘**：多數成年型氣喘；找不到過敏原；常因乾冷空氣、劇烈運動、強烈氣味誘發；多無家族史。
3. **職業性氣喘**：因工作接觸化學藥劑、動物毛髮／排泄物、廢氣等物質引起。

**氣喘表現型（phenotypes）：**

| 表現型                              | 特徵                                                                        |
| ----------------------------------- | --------------------------------------------------------------------------- |
| 過敏性氣喘（allergic asthma）       | 通常幼年發作，伴隨濕疹、過敏性鼻炎或食物/藥物過敏；對 ICS 治療反應良好      |
| 非過敏性氣喘（non-allergic asthma） | 無過敏症；痰液以嗜中性球為主或發炎細胞少；對 ICS 反應不佳                   |
| 晚發性氣喘（late-onset asthma）     | 成年後（特別女性）首次發作，通常非過敏性；需較高 ICS 劑量或對類固醇反應不佳 |
| 氣喘伴隨固定氣流限制                | 長期氣喘病人發展出固定氣流限制，與呼吸道壁重塑有關，類似 COPD               |
| 氣喘伴隨肥胖                        | 呼吸道症狀明顯但嗜酸性球發炎現象少                                          |

**其他特殊分型：** 阿斯匹靈敏感型、過敏性肺支氣管黴菌病（ABPM）、成人過敏型、氣喘預測指標陽性之學齡前哮喘、重度晚發高嗜酸性球型、運動誘發型。

### 1.4 氣喘誘發因子

- 病毒或氣道感染
- 過敏原（塵蟎、花粉、蟑螂、動物毛髮、黴菌）
- 吸菸及空氣汙染
- 運動或壓力、情緒變化
- 氣溫變化
- 藥物（β-blockers、Aspirin、NSAIDs 類）

## 2. 致病機轉與病理生理

### 2.1 基因與環境因子

- 過敏體質具遺傳性：父母一方過敏體質，子女 **50%** 機會遺傳；雙親皆有則 **80%** 機會。
- 台灣已辨識與氣喘感受度相關基因：ADAM33、DPP10、PHF11、SET domain bifurcated 2、GPRA、SPINK5。

### 2.2 過敏性氣喘之免疫機轉

- 外因性／過敏性氣喘與**Th2 細胞**及**IgE**媒介之免疫失調有關；大部分氣喘屬 IgE 媒介且 Th2 主導型，但也有非 Th2 主導型氣喘。
- 病理上：黏膜細胞過度增生且有免疫細胞浸潤；CD4 陽性 T 淋巴球、嗜酸性球、肥大細胞為三種最常見免疫細胞。
- Th2 分泌大量 IL-4、IL-13、IL-5 等介質；Treg 具氣管保護作用，可抑制 Th2 發炎反應。

**Th1 vs Th2：**

|      | Th1                                              | Th2                                                                            |
| ---- | ------------------------------------------------ | ------------------------------------------------------------------------------ |
| 功能 | 對抗細胞內黴漿菌；與自體免疫、延遲性過敏反應相關 | 對抗寄生蟲；與異位性體質、氣喘、過敏性鼻炎相關；分泌 IL-4、IL-13 促進 IgE 合成 |

**肥大細胞角色**：接觸過敏原後，IgE 附著肥大細胞表面（FcεRI），再次接觸過敏原時大量釋放組織胺、細胞激素、白三烯素、PGD2 等發炎物質，導致急性氣管收縮。

**清潔假說（Hygiene Hypothesis）**：工業化程度高環境成長之兒童發生過敏性疾病機率較高（台北市兒童氣喘盛行率為綠島 3 倍、金門 2 倍）。過於清潔的環境減少對 Th1 的刺激，使免疫系統偏向 Th2 分化，強化日後過敏反應（學界尚無定論）。

### 2.3 非異位性氣喘機轉

- Protease allergen 造成 IL-33、TSLP 等細胞激素上升，活化嗜鹼性球和 ILC2，分泌 Th2 相關細胞激素。
- 嗜鹼性球釋放 IL-4、IL-33 活化 ILC2，釋放 CCL11 吸引嗜酸性球聚集；活化的 ILC2 分泌 IL-5、IL-13 造成嗜酸性球增多與黏液分泌。

### 2.4 氣道病理變化五大機轉

1. **氣管平滑肌增生**：受 PDGF、endothelin-1 等生長因子刺激；平滑肌質量增加包含肥大與增生變化，主要出現在中央型氣道。
2. **血管反應**（通透性增加、滲漏）：血液流量增加致氣管狹窄；慢性免疫反應促成血管新生（VEGF 介導），發炎使血管通透性增加造成氣管水腫。
3. **黏液增多**：形成黏液塞（mucous plug）；IL-13 可能與過度分泌相關；杯狀細胞增生、黏膜下腺體增生。
4. **氣管重塑（airway remodeling）**：上皮脫落、氣道平滑肌肥大增生、杯狀細胞及黏膜下腺體增生、支氣管血管新生。
5. **氣管過度反應性（airway hyperresponsiveness）**。

**上皮下纖維化與基底膜增厚**：膠原蛋白 I/III/V 沉積造成皮下層增厚；正常人基底膜（LR）厚度約 4–5 µm，氣喘病人約**7–23 µm**；TGF-β 濃度與纖維母細胞數量、纖維化及氣喘嚴重度相關。

### 2.5 臨床症狀與病理生理之對應

- **氣喘常見呼吸症狀**：喘鳴（wheezing）、呼吸困難（shortness of breath）、胸悶（chest tightness）、咳嗽（cough）。
- 呼氣困難原因：支氣管狹窄、氣道壁增厚、氣道黏液分泌增加（均與過敏反應及氣道慢性發炎相關）。
- 氣流受限主要表現：**FEV1/FVC 比例降低**、**PEF 下降**；周邊氣管可能提早關閉，導致肺部過度充氣及滯留（air-trapping）。

### 2.6 氣喘與 COPD 鑑別（病理生理角度）

| 特徵         | 氣喘                                              | COPD（肺阻塞）                                   |
| ------------ | ------------------------------------------------- | ------------------------------------------------ |
| 發病年齡     | 20 歲前                                           | 40 歲後                                          |
| 症狀特點     | 隨時間變化，夜間/凌晨較重，因運動情緒外在刺激誘發 | 治療後症狀仍存在，伴運動性呼吸困難；慢性咳嗽有痰 |
| 肺功能       | 呼氣氣流受阻程度不一                              | 持續性呼氣氣流受阻（FEV1/FVC <0.7）              |
| 緩解期肺功能 | 正常                                              | 不正常                                           |
| 病程         | 可能自發性改善，支氣管擴張劑/ICS 反應佳           | 症狀緩慢惡化，短效支氣管擴張劑療效有限           |
| 胸部 X 光    | 正常                                              | 嚴重肺部過度充氣                                 |

**氣喘肺阻塞重疊（ACO, asthma-COPD overlap）**：若病患符合單一疾病 3 項以上特徵則診斷為該疾病；若兩疾病特徵數量相近，應考慮 ACO。

## 3. 臨床診斷

### 3.1 成人與青少年氣喘診斷標準（GINA 核心表格）

臨床表徵包含呼吸症狀病史（哮鳴、呼吸困難、胸悶、咳嗽）以及**可變化的呼氣氣流受阻**，兩者皆隨時間變化。

**（1）多變化呼吸道症狀的病史**

- 一般多種呼吸系統症狀（成人單獨性咳嗽較少由氣喘引起）
- 症狀隨時間變化，晚上或醒來時往往加劇
- 通常由運動、大笑、過敏原、冷空氣引起；病毒感染常誘發或使症狀惡化

**（2）確定的可變化呼氣氣流受阻**

| 檢測項目                           | 判讀標準                                                                                                                                                                                                                |
| ---------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| FEV1/FVC 降低確認阻塞              | 診斷過程至少一次確認 FEV1/FVC 減少（成人通常 >0.75–0.80，孩童通常 >0.9）                                                                                                                                                |
| 陽性支氣管擴張劑可逆性試驗         | 成人：FEV1 增加 >12% 且 >200mL（增加 >15% 且 >400mL 更佳確認）；孩童：FEV1 增加 12%。吸入 200–400mcg salbutamol，10–15 分鐘後測量。試驗前停藥使結果更準確：SABA ≥4 小時、每日兩次 LABA ≥24 小時、每日一次 LABA ≥36 小時 |
| 2 週內 PEF 過度變化                | 平均每日晝夜 PEF 變異性 >10%（每日兩次 PEF：日最高減日最低/最高最低日平均值，超過一週平均）                                                                                                                             |
| 4 週抗發炎治療後肺功能顯著增加     | 治療 4 週後 FEV1 增加 >12% 且 >200mL 或 PEF >20%（需排除呼吸道感染）                                                                                                                                                    |
| 陽性運動激發試驗                   | 基準 FEV1 降低 >10% 且 >200mL                                                                                                                                                                                           |
| 陽性支氣管激發試驗（通常僅成人）   | 標準劑量乙醯甲膽鹼或組織胺使 FEV1 降低 ≥20%；或標準化過度換氣、高滲鹽水、mannitol 試驗使 FEV1 降低 ≥15%                                                                                                                 |
| 不同次 FEV1 差異變化大（不太可靠） | 不同次 FEV1 變化 >12% 且 >200mL（呼吸道感染除外）                                                                                                                                                                       |

> 註：以上測試可於症狀發生或清晨重複；PEF 應每次使用同一儀器（不同儀器可能差異達 20%）；嚴重惡化或病毒感染期間可逆性可能喪失；**五歲以下幼童及老人應再做進一步檢查才能確診**。

### 3.2 已接受維持藥物病人之氣喘診斷確認

| 現況                                       | 處置方式                                                                                                                 |
| ------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------ |
| 多變呼吸道症狀＋可變呼氣氣流受阻           | 氣喘診斷確認，評估嚴重度並檢視維持藥物                                                                                   |
| 多變呼吸道症狀，但無可變呼氣氣流受阻       | FEV1 >70% 預估值：降階並 2–4 週後重評估，考慮激發試驗；FEV1 <70% 預估值：升階治療 3 個月後重評估，無反應則回復治療並轉診 |
| 極少呼吸道症狀、正常肺功能、無可變氣流受阻 | 考慮降階；症狀惡化肺功能下降則確認氣喘並升階回最低有效劑量；降階後仍無症狀及惡化，考慮停藥並密切追蹤至少 12 個月         |
| 持續呼吸困難和呼氣氣流受阻                 | 升階治療 3 個月並重評估；無改善則回復治療並轉診；需考慮 ACO                                                              |

### 3.3 診斷輔助檢查

**支氣管激發試驗（Bronchial Provocation Test）**

- 最常用藥物：吸入乙醯甲膽鹼（methacholine）；對氣喘診斷具中度敏感性但特異性有限。
- **PC20 定義**：給予 methacholine 致 FEV1 下降 20% 時之給藥濃度。
- 未使用 ICS 病人：**激發試驗陰性可排除氣喘**；陽性結果不代表一定有氣喘（過敏性鼻炎、COPD、囊性纖維化、支氣管肺發育不良亦可能陽性）。
- GRADE（GP 級）：未用 ICS 病人，methacholine 試驗陰性有助排除氣喘；陽性不表示一定有氣喘。

**呼氣一氧化氮濃度（FeNO）**

- 較專一反映嗜酸性球引發的氣道發炎。

| 濃度分類 | 兒童（12 歲以下） | 成人（12 歲以上） |
| -------- | ----------------- | ----------------- |
| 低濃度   | <20 ppb           | <25 ppb           |
| 中間濃度 | 20–35 ppb         | 25–50 ppb         |
| 高濃度   | >35 ppb           | >50 ppb           |

- 目前**沒有足夠證據**用 FeNO 確定或排除氣喘診斷；低 FeNO 無法排除氣喘。
- GRADE（1B，強建議，證據等級中）：高 FeNO（通常 ≥50 ppb）可作為氣喘診斷參考，特別在嚴重型嗜酸性第二型氣喘可預測未來急性發作風險。
- 特定族群（如兒童）使用 FeNO 指引治療可顯著降低急性發作次數（OR 0.67，95% CI 0.51–0.90，平均降低 0.27 次/年，證據等級 A）；懷孕氣喘病人 FeNO 監測可減少急性發作。

**動脈血氣體分析**：主要用於長期嚴重度觀察；正常值 PaO2 80–100 mmHg、PaCO2 35–45 mmHg；急性惡化常見高鹼血症。

**其他檢查**：血液嗜酸性球/IgE 檢測、痰液嗜酸性球檢查（Charcot-Leyden 結晶、Curschmann 螺旋、Creola 體）、胸部 X 光（通常正常，可見肺部過度充氣）、過敏原測試（皮膚測試/專一性 IgE）。

**氣喘確診後追蹤檢查**：每 3–6 個月定期追蹤肺功能，視臨床需要包括血液或痰中嗜酸性白血球、IgE 檢測、過敏原測試、FeNO 檢測、胸部 X 光。

### 3.5 特殊族群之氣喘臨床診斷重點

| 族群                  | 診斷重點                                                                            |
| --------------------- | ----------------------------------------------------------------------------------- |
| 咳嗽變異型氣喘        | 慢性咳嗽伴氣道過度反應；肺功能變異性證據十分重要                                    |
| 職業性/工作加重型氣喘 | 成年後發病；詢問工作接觸史、下班後症狀是否改善；客觀標準確診並轉診專科              |
| 運動員                | 依肺功能檢測（多為支氣管激發測試）；排除鼻炎、咽喉症狀、心臟疾病、過度訓練          |
| 孕婦／計畫懷孕        | 詢問是否患有氣喘                                                                    |
| 老年人                | 易低估（誤認正常老化）或高估（心血管疾病誤判）；吸菸或生質燃料暴露者需考慮 COPD/ACO |
| 吸菸者/已戒菸者       | 氣喘與 COPD 可能同時存在（ACO）；病史與症狀模式協助鑑別                             |
| 肥胖者                | 呼吸道症狀可能被誤認為氣喘；需客觀測量氣流受阻變異性確認                            |
| 正在使用控制型藥物者  | 25–35% 基層診斷病人無法確認患有氣喘；無法確認應轉介專家                             |
| AERD 病人             | 服用 aspirin/NSAIDs 後急性惡化史提示 AERD；阿斯匹靈測試為目前診斷黃金標準           |

## 4. 氣喘控制評估與監測 Control Assessment & Monitoring {#ch4}

### 4.1 評估病人的三大面向（GINA 架構）

1. **氣喘控制狀況**——評估症狀控制情形和危險因子
2. **治療相關情況**——治療階段、吸入器技巧、遵囑性、書面氣喘行動計畫、病人態度與目標
3. **共病症**——鼻炎、鼻竇炎、胃食道逆流、肥胖、OSA、憂鬱焦慮等

### 4.2 症狀控制評估（過去 4 週）

**成人／12 歲以上 GINA 評估表（4 項）：**

1. 每週是否出現超過兩次日間氣喘症狀？
2. 是否因氣喘於夜間醒來？
3. 是否因症狀需使用超過每週兩次的緩解型藥物（排除運動前預防性使用）？
4. 是否因氣喘使活動力受限？

| 出現項目數 | 控制程度 |
| ---------- | -------- |
| 無         | 控制良好 |
| 1–2 項     | 部分控制 |
| 3–4 項     | 控制不良 |

**6-11 歲兒童版**與**5 歲以下兒童版**評估表內容相近但用詞略調整（詳見[第 11 章](#ch11)）。

### 4.3 未來危險因子評估（急性惡化、固定氣流受限、藥物副作用）

**氣喘不佳結果（急性惡化）危險因子：**

- 藥物：SABA 使用過量（**每月 >1 支 200-dose 罐，死亡風險升高**）；ICS 未開立/未使用/使用不當；不適當吸入器技巧
- 共病症：肥胖、慢性鼻竇炎、GERD、確認的食物過敏、懷孕
- 暴露：吸菸、致敏性過敏原、空氣污染
- 病人背景：重大心理或社經問題
- 肺功能：低 FEV1（尤其 <預估值 60%）；高度可逆性支氣管擴張試驗
- 第二型發炎反應指標：痰或血液嗜酸性球過多、較高 FeNO（成人過敏性氣喘且使用 ICS 者）

**其他主要獨立危險（急性發作）因子：** 曾因氣喘惡化插管或送 ICU；過去 1 年內 ≥1 次嚴重急性發作。

**發展出固定呼吸氣流受阻的危險因子：** 早產、低出生體重、嬰兒期體重增加較高、慢性痰液過度分泌；曾嚴重急性發作缺乏 ICS 治療；吸菸或有毒化學物質、職業暴露；較低第一次 FEV1、痰液或血液嗜酸性白血球增多。

**藥物副作用危險因子：** 全身性——頻繁 OCS、長期高劑量或強效 ICS、同時 P450 抑制劑；局部——高劑量或強效 ICS、吸入器使用不良。

> 重要觀念：**即使症狀控制良好，只要存在 1 個以上上述危險因子，急性惡化風險即提高**，評估不應僅看症狀。

### 4.5 PEF 監測

- 每天早晚各吹三次記錄最高值；適用平時症狀不明顯的病人。
- **PEFR 變異度計算**：(24 小時內最高值 − 最低值) ÷ 平均值 × 100%
- 使用 ICS 後個人最佳 PEF 約於**2 週後**達到、日夜變異度約**3 個月**達最低點。
- **三區燈號分級：**

| 燈號 | 範圍（個人最佳值%） | 意義         |
| ---- | ------------------- | ------------ |
| 綠燈 | ≥80%                | 治療良好     |
| 黃燈 | 50–80%              | 可能即將惡化 |
| 紅燈 | <50%                | 正急性發作   |

- 長期 PEF 監測建議僅用於**嚴重氣喘**或對氣流受阻感知不佳（易發生 near-fatal 氣喘）病人。

### 4.6 肺功能監測

- FEV1 較 PEF 可信度高；FEV1 偏低是未來急性惡化的重要獨立因子。
- 治療前及開始控制型藥物**3–6 個月後**測量 FEV1 以記錄個人最佳肺功能，之後定期追蹤（一般每 1–2 年一次，高風險/嚴重氣喘增加頻率）。
- 確診且規則用藥病人，檢查前無需暫停控制或緩解藥物。

### 4.7 追蹤頻率總表

| 情境       | 頻率          |
| ---------- | ------------- |
| 治療初期   | 1–3 個月一次  |
| 治療穩定後 | 3–12 個月一次 |
| 懷孕期間   | 4–6 週一次    |
| 急性惡化後 | 一週內追蹤    |

### 5.2 SABA 單獨使用之安全性議題（重要觀念）

**自 2019 年起，GINA 不再建議單獨使用 SABA 治療氣喘**，所有成人及青少年氣喘患者都應接受含 ICS 之控制型藥物治療。

- 理由：單獨 SABA 可能增加急性惡化風險並降低肺功能，規律頻繁使用會使過敏反應與呼吸道發炎增加。
- SABA 使用過量（**一年 ≥3 瓶**）與嚴重急性惡化風險增加有關；**一年 ≥12 瓶**可能提高氣喘相關死亡風險。
- Track 1 治療策略：各階梯緩解藥物建議「需要時使用 ICS-SABA」取代單獨 SABA。

### 5.3 成人及青少年氣喘階梯式治療（GINA 2022，5 階）

治療策略核心：評估病情 → 調整治療藥物 → 檢視治療反應，循環進行。

**升階治療：**

- 持續性升階（至少 2–3 個月）：先評估吸入器技巧、遵囑性、可修正危險因子、共病症，再考慮升階
- 短期升階（1–2 週）：病毒感染或過敏原暴露時，依醫囑或書面計畫短期增加 ICS 劑量
- 每日調整：以 budesonide-formoterol 或 beclomethasone-formoterol 作維持/緩解治療者，可依症狀自行調整劑數

**降階治療（氣喘持續 3 個月控制良好且肺功能穩定時）：**

- 選擇適當時機（無呼吸道感染、未出遊、未懷孕）
- 3 個月內調降 ICS 劑量 25–50%
- 除非為確認診斷需要，否則**不可完全停用 ICS**
- 記錄病人狀態，提供書面氣喘治療計畫，安排回診

### 5.5 各類藥物作用機轉、療效與副作用

**ICS（吸入型類固醇）**

- 副作用：口咽念珠菌感染（藥水漱口）、發聲障礙／咳嗽（用 spacer 改善）；長期高劑量：瘀青、腎上腺抑制、骨密度下降、結核風險上升（budesonide <400mcg/日通常不致全身性影響）。
- 病患自述副作用發生率常高於醫師估計，顯示醫病認知落差。

**ICS/LABA 併用**

- 全身性副作用比單用口服 β 促效劑少；LABA 可安全與 ICS 併用，但**LABA 不應取代 ICS 或 OCS 單獨使用**。
- 副作用：頭痛、痙攣；全身性：心血管刺激、骨骼肌震顫、低血鉀。
- 實證：Budesonide/Formoterol 維持及緩解雙效療法（SMART/MART，如 STEAM、STEP、AHEAD、STAY、COMPASS、SMILE 等研究）可顯著降低重度急性發作風險（RR 0.58–0.65）。
- FF/VI（Salford Lung Study 真實世界 RCT）：較其他 ICS/LABA 多幫助 25% 病患改善氣喘控制（OR 1.95, 95% CI 1.60–2.38）。

**LTRA（白三烯素受體拮抗劑）**

- 成人療效較 ICS 差；兒童則無明顯差異。適合不能/不願用 ICS 患者（ICS 副作用不耐受、合併過敏性鼻炎）。
- 代表藥物：**Montelukast（欣流）**——FDA 已要求標示嚴重心理健康副作用風險警示。
- Zileuton 與肝毒性有關。

**Theophylline（茶鹼）**

- 緩釋型對氣喘只有微弱療效，但副作用常見，高劑量可能危及生命。

**LAMA（Tiotropium）附加治療（Step 5）**

- PrimoTinA-asthma 試驗：可降低嚴重氣喘發作風險，改善 FEV1；ICS+LABA+LAMA 三合一可降低急性惡化。
- 台灣 GRADE 建議 1A（強建議，證據等級高）：建議使用 Tiotropium 減少嚴重氣喘急性惡化頻率。
- 安全性與安慰劑組無顯著差異。

**OCS 長期治療**

- 低劑量 OCS 加成治療用於吸入器技巧及第 4 階治療遵囑性佳但控制仍不佳的成人（須先排除其他原因）。
- 使用 ≥3 個月應評估骨質疏鬆風險並給予預防性療法。

  - T2 high 再分為：嚴重過敏性氣喘、嚴重嗜酸性白血球性氣喘
- **非第二型發炎（T2 low）**：嚴重嗜中性白血球性氣喘、嚴重寡顆粒球性氣喘

| 多重吸入器使用          | 盡量統一裝置型式，避免混淆 |

### 5.9 吸入器使用技巧訓練（4C 模式）

1. **Choose（選擇）**：依藥物、生理障礙、操作技術、費用選擇最適合吸入器
2. **Check（檢查）**：每次回診觀察病人操作、核對檢查表（研究顯示**約 80% 病人未正確使用吸入器**）
3. **Correct（修正）**：示範正確方式，必要時要求重複操作 2–3 次
4. **Confirm（確認）**：直到連續三次操作正確為止

### 5.10 藥物遵囑性

- **約五成病人未遵循醫囑使用控制型藥物**。
- 不遵從原因分類：
  - **藥物因素**：操作困難、使用次數多、多種不同形式吸入器
  - **非故意因素**：認知不足、誤解醫囑、忘記、藥物不在身邊、費用
  - **故意因素**：自認不需治療、否認有氣喘、副作用恐懼、文化因素、費用
- 提升自我照護能力可減少：住院（RR 0.50）、用藥不良反應處置（RR 0.75）、非預期就診（RR≈0.68）。

## 6. 非藥物治療 Non-pharmacological Treatment {#ch6}

### 6.1 核心非藥物治療項目

1. **建議戒菸**：強烈建議並提供戒菸資源；氣喘病童家長/照護者應避免同室或同車吸菸
2. **規律體能活動**：增進心肺體適能；運動可能誘發支氣管收縮，建議暖身、運動前使用 SABA 或 ICS-formoterol
3. **鑑別職業性氣喘**：詢問工作史，及早移除致敏根源
4. **注意 NSAIDs（含 Aspirin）**：開立前詢問氣喘病史
5. **接種流感疫苗**：建議中重度氣喘病人每年接種

### 6.2 戒菸實證效益

- 戒菸 6 週後：痰液嗜中性球比例下降（-29.1%）、巨噬細胞比例上升（+21.6%）
- FEV1、FVC、PEF 隨戒菸週數增加而改善；氣喘控制評分改善（6 週：-1.2 分）；呼氣一氧化碳持續下降

### 6.3 過敏原/刺激原迴避（GINA 附證據等級）

| 建議                                                                           | 證據等級       |
| ------------------------------------------------------------------------------ | -------------- |
| 開立 NSAID/Aspirin 前應詢問氣喘病史                                            | A              |
| 減少居家濕氣、黴菌可降低氣喘症狀                                               | A              |
| 一般**不建議**以避開過敏原作為主要控制策略（過於複雜、花費高、難判斷獲益族群） | A（不建議）／D |
| 使用低油煙烹飪方式，減少室內刺激原                                             | B              |

### 6.4 職業性氣喘認定基準（台灣勞動部）

1. 特定工作開始後才發生氣喘或明顯惡化
2. 醫學診斷肺功能或相關試驗證實可逆性呼吸道阻塞
3. 有職業性作業環境暴露史
4. 合理排除其他常見阻塞性肺部疾病

（約 **15%** 氣喘病人之氣喘原因與工作場所有關；常見誘發物：TDI、TMA、白金化合物、甲醛、鎳等）

## 7. 慢性照護與病人自我管理 Chronic Care & Self-Management {#ch7}

### 7.1 慢性照護四大核心策略

控制症狀、緩解持續發炎、降低急性發作風險、維持肺功能。長期追蹤需涵蓋：確立診斷、嚴重度與表現型評估、治療效果評估（用藥狀況/技巧、處置計畫、共病症）。

### 7.2 書面氣喘行動計畫（Asthma Action Plan）

**內容架構：**

1. 紀錄臨床症狀（如 ACT）及自我監測方式
2. 紀錄每日預防性藥物種類劑量、緩解型藥物名稱劑量
3. PEF 監測三區分級（<50% 惡化中／50–80% 可能惡化／>80% 治療良好）
4. 詳細治療計畫：何時加強預防治療、何時調升 ICS 或加 OCS、何時緊急送醫、誘發因子迴避
5. 定期回診追蹤安排

**成效**：病人帶著書面計畫及 PEF 計出院者，預後優於未帶者。

### 7.3 氣喘控制不佳的處置流程（五步驟）

1. 觀察吸入器使用＋討論順從性障礙（核對檢查表，重複確認至三次）
2. 確認氣喘診斷（無法證實變異性氣流限制可考慮 ICS 減半，2–3 週後複查；確認有行動計畫；考慮激發測試）
3. 去除可修正危險因子＋評估共病症（吸菸、β-blocker/NSAIDs、過敏原；鼻炎、肥胖、GERD、OSA、憂鬱焦慮）
4. 考慮治療升階（醫病共享決策 SDM）
5. 轉介專科或嚴重氣喘門診（3–6 個月高劑量 ICS/LABA 仍未控制，或危險因子持續存在）

### 7.4 共病症處置摘要

| 共病症            | 處置重點                                                                                                            |
| ----------------- | ------------------------------------------------------------------------------------------------------------------- |
| 肥胖              | 呼吸道發炎以嗜中性球為主，ICS 反應較差；減重 5–10% 即可改善控制；IgE 高可考慮 Omalizumab；合併 OSA 者 CPAP 有助改善 |
| 過敏性鼻炎/鼻竇炎 | 控制好可減少住院及急診；鼻內類固醇為首選 ± 口服抗組織胺；嚴重氣喘可考慮 Anti-IgE/Anti-IL-5                          |
| GERD              | β-agonist 及 theophylline 可能使括約肌鬆弛致惡化；PPI 可改善晨間 PEF 與夜間症狀，但未證實減少急性發作               |
| 憂鬱/焦慮         | 高品質研究稀少；Cochrane 分析對 CBT、心理教育、放鬆治療、生物回饋效果未證實                                         |

### 7.5 藥物順從性提升技巧

**確認順從性之詢問方式：** 過去一個月每週約幾天使用吸入藥物？每天吸幾次？每次吸幾口？早上或晚上較不易忘記？

**改善措施：** 選擇最簡單裝置與方案、書面資料、每次回診複查技巧、以同理心探討不遵囑原因。

### 7.6 危險因子管控（急性惡化）具體措施

| 危險因子             | 控制措施                                                                                           |
| -------------------- | -------------------------------------------------------------------------------------------------- |
| 過敏原/吸菸/運動誘發 | 避免室內外過敏原、戒菸、運動前用藥                                                                 |
| 空氣汙染/氣溫變化    | 氣溫變化與氣喘急診/門診具相關性（分年齡層）                                                        |
| 情緒因素             | 為常見誘發危險因子之一                                                                             |
| Aspirin/NSAIDs       | AERD 盛行率約成人氣喘病人 7%；誘發氣喘常於服藥後 1–2 小時內；建議改用 COX-2 抑制劑或 acetaminophen |
| β-blocker            | 儘量避免，或選用高選擇性 β1 阻斷劑（高劑量仍有誘發氣喘風險）                                       |

## 8. 急性發作處置 Acute Exacerbation Management {#ch8}

### 8.1 定義與誘發因子

急性發作為症狀和肺功能較平常**急性或亞急性變差**，症狀進行性加劇且肺功能持續下降，通常需額外治療。誘發因子：病毒上呼吸道感染、過敏原、食物過敏、空氣污染、季節/氣候變化、運動情緒、藥物（Aspirin/NSAIDs）、控制型藥物順從性不良。

### 8.2 氣喘猝死高風險族群特徵

1. 曾有需插管和機械通氣的近乎致命氣喘病史
2. 過去一年內因氣喘住院或急診
3. 目前使用或最近停用 OCS
4. 目前未使用 ICS
5. 過度使用 SABA（尤其每月 >1 罐）
6. 對 ICS 藥物順從性差
7. 對（或缺乏）書面氣喘行動計畫的順從性差
8. 有精神病史或社會心理問題
9. 對食物過敏
10. 合併症：肺炎、糖尿病、心律不整

### 8.3 GINA 成人急性發作嚴重度分級

| 項目       | 輕微至中度       | 嚴重發作       | 危及生命                  |
| ---------- | ---------------- | -------------- | ------------------------- |
| 說話       | 能說句子或片段   | 只能說單字     | 無法說話                  |
| 姿勢       | 能平躺但偏好坐姿 | 坐姿前傾       | 無法坐起                  |
| 情緒       | 不焦躁           | 焦躁           | 嗜睡或意識不清            |
| 呼吸次數   | 增快             | 通常 >30 次/分 | 可能變慢，呼吸音減弱/消失 |
| 呼吸輔助肌 | 未使用           | 使用           | 異常胸腹動作              |
| 脈搏       | 100–120 次/分    | >120 次/分     | 可能變慢                  |
| 血氧飽和度 | 90–95%           | <90%           | <90%（或 PaO2 <60mmHg）   |
| PEF        | >50%             | <50%           | 無法測量                  |

> 危及生命之發作出現呼吸喘促、昏睡意識不清、聽診無呼吸音，代表嚴重支氣管痙攣，**建議立即轉送加護病房**。

### 8.4 居家自我照護（輕度至中度，PEF 降低 >20% 且持續 >2 天）

- 提升緩解型藥物用量；增加控制型藥物用量；必要時添加 OCS
- **SABA 居家使用**：第一小時每 20 分鐘 4–10 puffs；之後每 3–4 小時 4–10 puffs
- **SMART 策略（ICS/formoterol）**：家中惡化早期可將 budesonide/formoterol 或 beclometasone/formoterol 提升至**4 倍**作緩解藥物（GRADE 2B）
- **ICS 單方 4 倍劑量**：可能減少 OCS 需求、就醫及住院機會（GRADE 2B；RCT AHR 0.81, 95% CI 0.71–0.92, p=0.002）
- **居家啟動 OCS 時機**：增量 2–3 天未反應、迅速惡化或 PEF/FEV1 <個人最佳值 60%、突然嚴重惡化病史 → 每日 40–50mg 持續 5–7 天
- 若持續惡化或呼吸困難，須儘速就醫

### 8.5 基層/急診共通處置

**病史評估**：發作時間誘因、症狀嚴重度、過敏症狀、死亡危險因子、目前用藥遵囑性。

**身體檢查**：發作嚴重度、是否合併過敏性休克/肺炎/氣胸、鑑別診斷（心臟衰竭、上呼吸道功能不全、吸入異物、肺栓塞）。

**藥物治療：**

| 藥物                | 劑量/使用方式                                                                                                                      |
| ------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| 氧氣治療            | 面罩 24%（約 1L/min）維持血氧飽和度 94–98%（成人）或 93–95%（控制型低流量氧氣）                                                    |
| 吸入型 SABA         | 每 20 分鐘 2–6 puffs（吸入輔助器）或 2.5mg（霧化劑），持續 1 小時後再評估；症狀持續每小時 2–3 puffs；住院 3–4 小時內可達 >10 puffs |
| 全身性類固醇        | 口服 prednisolone 起始 1–2mg/kg，成人最多每天 50mg，或每日 30–50mg 療程 5–7 天；或分次 hydrocortisone 200–300mg/日                 |
| Ipratropium bromide | 中重度發作，每 20 分鐘 2 puffs（80mcg）或 2.5mg 霧化，持續 1 小時；可與 SABA 併用提升效果、降低住院率                              |
| 抗生素              | 不建議常規使用，除非有肺部感染強力證據                                                                                             |

**不建議使用之藥物：**

- 氨茶鹼/茶鹼——療效安全性不如 SABA，不應用於急性處置，常致噁心嘔吐，靜脈注射有嚴重副作用風險
- 硫酸鎂——不建議常規使用；適用初步治療無效、FEV1 25–30% 預估值、或 1 小時治療後 FEV1 仍 <60% 者：**2 克單一劑量緩慢靜注 20 分鐘**
- 口服白三烯受體拮抗劑——證據不足，不建議
- 腎上腺素——僅用於過敏性休克及血管性水腫，不建議常規用於一般氣喘發作
- 鎮靜劑——會抑制呼吸，宜審慎使用

### 8.6 客觀評估與住院/加護病房標準

- 密集監測 PEF 或 FEV1、脈衝式血氧計監測 SpO2（維持 93–95% 以上）
- 動脈血氣（非必要但可輔助）：**PaO2 <60mmHg 或 PaCO2 >45mmHg**表示呼吸衰竭，建議住加護病房
- 胸部 X 光評估肺炎/肺扁塌/氣胸/縱膈氣腫
- **立即轉送加護病房情況**：呼吸喘促、昏睡意識不清、聽診無呼吸音（嚴重支氣管痙攣）

### 8.7 出院準備與追蹤

- **OCS 療程**：成人至少 5–7 天（prednisolone 等效每天 1mg/kg，最多 50mg/天）；順從性不良風險者可考慮肌肉注射型類固醇
- **緩解型藥物**：症狀與客觀改善後改回視需要使用；ipratropium 可儘快停用
- **ICS**：未曾開立者應於出院前開始；已開立者一般升階並持續 2–4 週
- 找出並矯治該次急性發作危險因子
- 檢視吸入器使用技巧、PEF meter 使用技巧
- **出院時或儘快提供書面氣喘處置計畫**
- **追蹤約診安排在出院後 2–7 天內**

## 9. 嚴重型氣喘 Severe Asthma {#ch9}

### 9.1 定義（2022 GINA）

過去一年需 GINA 第 4–5 階治療（高劑量 ICS 及 LABA 或 LTRA/茶鹼）或 **≥50% 時間需全身性類固醇** 才能控制，或仍未獲控制者。**盛行率**最佳估計約 **4%**（真正盛行率尚不明確）。

### 9.2 診斷嚴重型氣喘須先確認三項

1. 是否有診斷錯誤
2. 是否有良好遵囑性及正確吸入器技巧
3. 共病症已獲得適當治療

### 9.3 生物標記區分發炎型態

見[5.7 節](#sec5-7)：約 70% 為第二型發炎（T2 high），再分嚴重過敏性氣喘與嚴重嗜酸性白血球性氣喘；非第二型（T2 low）分嚴重嗜中性白血球性氣喘與嚴重寡顆粒球性氣喘。

### 9.4 嚴重氣喘治療原則

- 非生物製劑藥物優化後，依發炎型態選擇合適生物製劑（詳見[5.6 節](#sec5-6)）
- 若條件允許應轉介胸腔專科醫師，評估是否需併用其他藥物和/或痰誘導治療
- GINA 2022 建議依 10 步驟流程進行治療決策（評估表型、生物標記、共病症 → 選擇初始生物製劑治療）

## 10. 特殊族群與特殊情況 Special Populations & Situations {#ch10}

### 10.1 懷孕

- 盛行率 3–12%；約 1/3 病人孕期症狀惡化，**第二孕期最常惡化**。
- **常見誤解**：42% 誤認 OCS 致畸胎、12% 誤認 ICS 致畸胎、5% 誤認 SABA 致畸胎——**實際上目前證據顯示氣喘用藥皆無致畸胎性**。
- 孕期常見自行停藥/減量（5–13 週：SABA↓13%、ICS↓23%、類固醇救援↓54%；38% 完全未用藥）。

**孕期氣喘控制不佳之併發症風險：**

| 併發症           | RR/OR（95% CI）      |
| ---------------- | -------------------- |
| 子癲前症         | RR 1.54（1.32–1.81） |
| 畸胎             | OR 1.48（1.04–2.09） |
| 新生兒體重過輕   | RR 1.46（1.22–1.75） |
| 早產             | RR 1.41（1.23–1.62） |
| 流產             | OR 1.41（1.33–1.49） |
| 週產期死亡率     | RR 1.25（1.05–1.50） |
| 胎兒小於妊娠年齡 | RR 1.21（1.14–1.31） |

**治療原則**：與一般原則一致，首選 ICS；急性惡化積極給予吸入型 SABA、氧氣、全身性類固醇（避免胎兒缺氧）；直到生產前均應維持 ICS 治療，不應隨意停藥；生產時因過度換氣致支氣管收縮可用吸入型 SABA（需監測低血糖）；每月密集監測肺功能；接種流感疫苗；GRADE（GP）：控制不良時考慮 ICS+LABA 併用。

### 10.2 肥胖

- 肥胖成人氣喘風險為正常體重 1.5–3.5 倍；肥胖孩童 1.1–3.5 倍。
- BMI 每增加 10 kg/m²，FEV1/FVC 改變 3%；BMI 每增加 1 kg/m²，嚴重程度分數改變 0.183。
- 呼吸道發炎以嗜中性球為主，對 ICS 治療反應可能較差（BMI ≥25 病人 PBMC 對 dexamethasone 誘發 MKP-1 表現量降低）。
- GRADE（GP）：建議減重，**體重減輕 5–10%** 較能使氣喘易於控制、改善生活品質；IgE 高且有過敏原反應可考慮加 Omalizumab；合併 OSA 者 CPAP 有助改善。

### 10.3 老年人

- 診斷常被低估（誤認呼吸困難為正常老化）或高估（心血管疾病誤判）；有吸菸或生質燃料暴露史需懷疑 COPD 或 ACO。
- 肺功能退化機轉：胸壁僵硬、呼吸肌功能減弱、彈性回縮力及氣道重塑功能喪失、嗜中性球發炎反應變化。
- 共病症盛行率：糖尿病 16.3%、心血管疾病 21.5%、骨質疏鬆/關節炎 85.5%、失智症風險比 1.56、關節炎/骨質疏鬆風險比 1.4。
- 用藥注意：青光眼病人避免 β2 促效劑眼藥水；維持最低類固醇劑量降低白內障/骨質疏鬆風險；避免同時處方三種以上吸入劑。

### 10.4 鼻炎與鼻竇炎

- 共病症發生率 10–40%；80% 氣喘病人曾有鼻腔症狀；慢性鼻竇炎併鼻息肉與氣喘共病率約 60%。
- 機轉：鼻－氣管反射、鼻涕倒流、發炎介質全身循環擴散、局部超級抗原誘發、嗜酸性球/Th2 細胞遷移。
- 治療：鼻內類固醇為首選（ARIA 證據等級 B 可治療氣喘、B 可減少惡化）；依嚴重度加抗組織胺/LTRA；嚴重氣喘可考慮 Anti-IgE/Anti-IL-5。

### 10.5 胃食道逆流（GERD）

- 共病率 17–74%；GERD 未控制惡化氣喘症狀（迷走神經反射、微量逆流吸入）；氣喘藥物（β2-agonist、theophylline）可能加重逆流（惡性循環）。
- PPI 治療：可緩解症狀改善晨間 PEF，但**未證實**能改善急性惡化。GRADE（GP）：建議藥物控制 GERD 症狀以改善氣喘控制。

### 10.6 心血管疾病

- 心臟專一性 β-blocker 長期使用**不會**增加中重度氣喘急性發作；非專一型則**明顯增加**風險（劑量相關，應從最小劑量開始）。
- 吸入型 SABA 可能增加心衰竭病人急性惡化及死亡風險；LABA/LAMA 未觀察到心血管不良事件增加。

### 10.7 焦慮與憂鬱

- 共病率 4–17%；恐慌發作易與氣喘症狀混淆；與控制不佳、順從性差、生活品質惡化相關。處置：CBT、心理教育、放鬆治療、生物回饋（證據支持有限）。

### 10.8 阿斯匹靈加重性呼吸道疾病（AERD）

- 機轉：Aspirin 抑制 COX-1 → PGE2 減少 → 肥大細胞活化（經 CRTH2/TP 受體）釋放組織胺、tryptase、LTC4；涉及 IL-33/TSLP/ST2/TSLPR 路徑；嗜酸性球、嗜鹼性球、中性球、血小板參與白三烯活化，致氣管平滑肌收縮。
- 臨床表徵：流鼻水、鼻塞、結膜炎、頭頸部潮紅、嚴重支氣管痙攣甚至休克意識喪失。
- 處置：避免 aspirin/NSAIDs，改用 COX-2 抑制劑或 acetaminophen；首選 ICS 或口服類固醇，可考慮 LTRA。
- 成人氣喘病人約 **7%** 患有 AERD；服藥後**1–2 小時**內誘發。

### 10.9 食物過敏與全身性過敏反應

- 食物過敏原引發氣喘比例 <2% 但常致命；最常見：花生、木本堅果。
- 症狀：上呼吸道（鼻塞癢流鼻水打噴嚏、咽喉水腫聲音沙啞）、下呼吸道（咳嗽胸悶呼吸困難喘鳴）。
- **第一線治療：Epinephrine**；輔助：支氣管擴張劑、H1 抗組織胺、類固醇、氧氣、輸液、升壓劑、升糖素、Atropine。

### 10.10 手術考量

- 手術全期嚴重支氣管痙攣發生率**2%**，多發生於麻醉誘導前。
- 術前準備：至少提前 2 個月戒菸；考慮口服 methylprednisolone 40mg×5 天；過去 6 個月內全身性類固醇使用 ≥2 週者，手術期間應給予 hydrocortisone 避免腎上腺危象。
- 用藥選擇：避免易致組織胺釋出藥物（Atracurium、Morphine、Vancomycin、Protamine sulphate）；優先選擇 Cisatracurium、Rocuronium、Propofol、Ketamine、Lidocaine、Sevoflurane、Isoflurane。
- Neostigmine 具膽鹼作用，可能提高支氣管痙攣風險，須留意。

### 10.11 COVID-19 與氣喘

- 氣喘不增加感染風險，感染後嚴重度及預後也不比無氣喘者差。
- 疫情期間應繼續各項治療特別是 ICS，突然停用可能致嚴重惡化。
- 感染者建議以 MDI 取代霧化器（減少病毒傳播）；肺量計/PEF 檢查非急迫則延後。
- 接種疫苗安全；嚴重氣喘接受生物製劑者應避免與疫苗同天施打，以利釐清副作用肇因。

### 10.12 運動誘發型支氣管收縮（EIB）

- 對氣喘病人而言，運動是支氣管收縮**最常見誘發因子**。
- 診斷：運動測試後 FEV1 變化 ≥10–15% 確診。
- 處置：運動前 15–20 分鐘使用 SABA（Evidence A）或 ICS-formoterol as needed（Evidence B）；暖身運動（Evidence A）；寒冷天氣戴口罩；症狀持續或每天需 SABA 者，加用每日 ICS 或 LTRA。
- GRADE（1B，強建議，證據等級中）：建議運動前使用短效吸入性 β2 促效劑。

### 10.13 運動員用藥禁忌（世界運動禁藥管制規範）

| 藥物                         | 規範                                                                                                                                            |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| β2 促效劑                    | 原則禁用，例外：吸入式 salbutamol（24 小時上限 1600mcg，每 12 小時 ≤800mcg）、formoterol（24 小時上限 54mcg）、salmeterol（24 小時上限 200mcg） |
| 類固醇                       | 列入監測計畫，需治療用途豁免（TUE）核可                                                                                                         |
| LTRA、Anti-IgE、Theophylline | 一般氣喘常用但運動員禁用                                                                                                                        |

### 10.14 職業性氣喘

- 約 **15%** 氣喘病人與工作場所相關；常見致敏物：TDI、TMA、白金化合物、甲醛、鎳等。
- 診斷步驟：注意症狀週末/假日改善 → 暴露史與 SDS 回顧 → 支氣管擴張劑前後肺量測定/methacholine 測試 → 工作中 1–2 週及離職後 2–4 週連續監測 → 必要時專一性吸入測試。
- 及早移除職業暴露並開始治療可提高復原機會（Evidence A）。

### 10.15 交通相關空氣汙染（TRAP）

- 造成呼吸道氧化損傷、發炎、氣道重塑，並激發敏感體質；與氣喘新發個案增加、控制困難、急性發作增加有關。
- 對策：持續使用 ICS、避免高污染天候出門運動或通勤。

## 11. 兒童氣喘專節 Pediatric Asthma {#ch11}

### 11.1 年齡分層

| 族群       | 年齡     | 備註                                         |
| ---------- | -------- | -------------------------------------------- |
| 學齡前兒童 | ≤5 歲    | 症狀有時步入學齡後自主消失；治療方式獨立討論 |
| 學齡兒童   | 6–11 歲  | 本節重點                                     |
| 青少年     | 12–18 歲 | 診斷標準與成人同                             |

### 11.2 病理生理特點

呼吸道阻塞、對誘發因子（運動、過敏、病毒感染）過度反應及慢性發炎，產生間歇性症狀反覆發作。

- **氣道過度敏感**：內外因性刺激導致過度支氣管收縮
- **反覆性氣道阻滯**：急性支氣管收縮、氣道壁腫脹、慢性黏液栓塞、氣道壁變形

> 兒童氣喘臨床症狀多變且非特異性，病理特徵往往無法常規評估。

### 11.3 6–11 歲兒童臨床診斷

**診斷流程**：典型症狀？→ 病史/檢查支持診斷？→（不確定時）支氣管擴張劑可逆性測試/PEF →（仍不確定）ICS 經驗療法 1–3 個月評估。

**呼氣限制判讀標準：**

- FEV1/FVC 比值下降，兒童標準 **>0.90**
- 支氣管擴張劑可逆性：FEV1 增加 **>12%**（predicted 值）
- 日間 PEF 變化幅度 **>13%**
- 抗發炎治療 4 週後 FEV1 增加 **>12%**（predicted 值，排除感染）

**症狀控制評估表（過去 4 週）：** 日間症狀 >2 次/週？夜間醒來？SABA 使用 >2 次/週？活動受限？→ 0 項良好控制／1–2 項部分控制／3–4 項控制不良。

**危險因子**：與成人相近（詳見 4.3 節），另強調 SABA 使用過量（>1×200 劑量/瓶/月死亡風險升高）、ICS 使用不足、肥胖/慢性鼻竇炎/GERD/食物過敏、吸菸暴露、FEV1 <60% 預測值、血中嗜酸性球增多、FeNO 升高。

### 11.4 6–11 歲階梯式治療（5 階）

| 治療階 | 首選控制型藥物                                                     | 其他選擇                                   |
| ------ | ------------------------------------------------------------------ | ------------------------------------------ |
| 第一階 | 使用 SABA 時同時使用低劑量 ICS                                     | 考慮每日低劑量 ICS                         |
| 第二階 | 每日低劑量 ICS                                                     | LTRA，或 SABA 時併用低劑量 ICS             |
| 第三階 | 低劑量 ICS-LABA 或中劑量 ICS；或極低劑量 ICS-formoterol 維持及緩解 | 低劑量 ICS+LTRA                            |
| 第四階 | 中劑量 ICS-LABA，或低劑量 ICS-formoterol 維持及緩解；轉介專家      | 添加 tiotropium 或 LTRA                    |
| 第五階 | 轉介氣喘表型評估 ± 高劑量 ICS-LABA 或 Anti-IgE/Anti-IL4R           | 添加 anti-IL5 或低劑量 OCS（需考慮副作用） |

**低劑量 ICS 適應症**：症狀持續 >2 個月、每月因氣喘夜醒 >1 次、症狀伴任一惡化危險因子。

### 11.6 兒童身高追蹤（重要注意事項）

- 至少每年檢測 1 次身高：控制不佳氣喘會影響成長；ICS 治療頭兩年成長速率會變慢但不持續惡化；長期 ICS 治療與常人成年身高差異僅**0.7%**。
- 成長速率減緩應考慮：氣喘控制不佳、頻繁使用口服類固醇、營養不良。

### 11.7 5 歲以下兒童臨床診斷

反覆性哮喘常見且多與上呼吸道病毒感染相關，確診不易。

**症狀型態光譜（偏病毒感染 vs 偏氣喘）：**

| 面向           | 比較少是氣喘 | 大多數是氣喘                                   |
| -------------- | ------------ | ---------------------------------------------- |
| 感染時症狀天數 | <10 天       | >10 天                                         |
| 年發作次數     | 2–3 次       | ≥3 次或嚴重發作/夜間惡化                       |
| 發作間隔症狀   | 無           | 玩耍大笑時偶發咳嗽喘鳴                         |
| 其他           | —            | 過敏性體質、異位性皮膚炎、食物過敏、氣喘家族史 |

**改良式氣喘預測指標（mAPI，3 歲以下參考）**：過去一年 ≥4 次喘鳴惡化（≥1 次經醫師確認），且符合 ≥1 項主要條件或 ≥2 項次要條件。

| 主要條件（≥1 項）                  | 次要條件（≥2 項）        |
| ---------------------------------- | ------------------------ |
| 雙親氣喘病史                       | 食物過敏（牛奶蛋花生等） |
| 醫師確診異位性皮膚炎               | 與感冒無關喘鳴           |
| 空氣過敏原敏感（塵蟎貓狗黴菌花草） | 血中嗜酸性球 ≥4%         |

判讀：陽性 → 76% 學齡後氣喘風險；陰性 → 95% 機率學齡後不再發氣喘。

**輔助檢測**：診斷性治療（SABA 或低劑量 ICS，2–3 個月）、過敏性體質測試（滿 3 歲後氣喘兒童通常併有過敏體質）、胸部 X 光、肺功能（5 歲幼童經適當指引或可行）、FeNO（較大兒童適用）。

**喘鳴發展為持續性氣喘的危險因子**：發病年齡（2 歲為分段點，越早消失機率越高）、性別（男童學齡易罹患但青春期易消失，女童相反）、發作嚴重度頻率、異位性體質相關疾病、家族史（母系更強）、肺功能異常。

**鑑別診斷**：慢性上呼吸道咳嗽症候群、吸入異物（突發單側喘鳴）、支氣管擴張症、原發性纖毛運動障礙、先天性心臟病、支氣管肺發育不全、囊狀纖維化症。

### 11.8 5 歲以下症狀控制評估表

過去 4 週：日間症狀 >1 次/週（持續數分鐘）？活動受限？緩解藥物使用 >1 次/週（排除運動前）？夜間咳嗽或醒來？→ 0 項良好控制／1–2 項部分控制／3–4 項控制不良。

**惡化危險因子**：控制不佳、過去 1 年 ≥1 次嚴重惡化、季節（尤其秋季）、暴露（二手菸、空污、室內過敏原 + 病毒感染）、家庭心理社經問題。**固定氣流受限危險因子**：嚴重氣喘多次住院、細支氣管炎病史。

### 11.9 5 歲以下階梯式治療（4 階）

| 治療階 | 首選控制型藥物                | 其他選擇                          |
| ------ | ----------------------------- | --------------------------------- |
| 第一階 | 有需要時 SABA                 | 病毒感染時短期使用 ICS            |
| 第二階 | 每日低劑量 ICS                | 每日 LTRA，或呼吸道症狀時短期 ICS |
| 第三階 | 雙倍「低劑量」ICS（即中劑量） | 低劑量 ICS+LTRA，考慮轉介專科     |
| 第四階 | 持續控制型藥物 & 轉介專科評估 | 添加 LTRA 或增加 ICS 頻率         |

### 11.12 5 歲以下急性惡化處置

**嚴重程度分級：**

| 症狀                 | 輕度      | 重度*                                |
| -------------------- | --------- | ------------------------------------ |
| 意識                 | 無改變    | 情緒焦躁、意識混亂或昏睡             |
| 血氧飽和度（治療前） | >95%      | <92%                                 |
| 說話                 | 完整句子  | 僅隻字片語                           |
| 脈搏                 | ≤100 bpm  | >180bpm（0-3 歲）／>150bpm（4-5 歲） |
| 呼吸速率             | ≤40 次/分 | >40 次/分                            |
| 中心性紫紺           | 未出現    | 容易發生                             |
| 喘鳴強度             | 多變性    | 可能聽不到呼吸音                     |

*出現任一即為重度

**輕中度處置**：每 20 分鐘 2 puffs salbutamol 100mcg（或 2.5mg 霧化），持續 1 小時；氧氣治療維持 94–98%；考慮加 ipratropium 1–2puffs；觀察 1–2 小時。惡化：每 20 分鐘 6 puffs、prednisolone 起始 2mg/kg（<2 歲最高 20mg；2–5 歲最高 30mg）。

**重度/危及生命**：立即轉送加護病房；轉送期間每 20 分鐘 6 puffs salbutamol（或 2.5mg 霧化）、氧氣 94–98%、prednisolone 2mg/kg、每 20 分鐘 ipratropium bromide 160mcg（或 250mcg 霧化）持續 1 小時。

**送醫指標（任一即應送急診）**：呼吸停止或即將停止、無法說話喝水、發紺、呼吸 >40 次/分、SpO2 <92%、聽診無呼吸音、對 6puffs SABA 治療 1–2 小時無反應、持續呼吸過速、社會環境因素無法妥善照護。

**正常呼吸速率參考（判定呼吸過速）**：0–2 月齡 <60 次/分；2–12 月齡 <50 次/分；1–5 歲 <40 次/分。

**出院追蹤**：1–2 天內追蹤，prednisone 療程 ≥3–5 天；1–2 個月後回診。

### 11.13 兒童氣喘初級預防

- 基因與環境交互作用為主要成因；胎兒與新生兒期為關鍵。
- 現行建議：懷孕及新生兒期避免二手菸；鼓勵自然產與母乳哺育；孕婦維他命 D 缺乏可考慮補充；1 歲前盡量避免廣效抗生素；留意交通空氣污染。

### 11.14 5 歲以下高度懷疑氣喘之症狀組合

頻繁喘鳴 >1 次/月、運動後誘發咳嗽喘鳴、無感染時夜咳、喘鳴無季節性、症狀持續 >3 歲。
