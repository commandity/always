# 慢性肺阻塞

**Chronic Obstructive Pulmonary Disease**

## 1. 總論：疾病衝擊與給付改善方案

### 1.1 COPD 定義（2023 GOLD 修正版）

> 肺阻塞（COPD）是一種**異質性肺部疾病**，其特點是由於氣道異常（支氣管炎、小支氣管炎）和／或肺泡異常（肺氣腫）引起的**慢性呼吸症狀**（呼吸困難、咳嗽、痰液、急性發作），導致**持續性、通常是進展性的氣流阻塞**。

**核心病理生理路徑：**

- 小呼吸道疾病：呼吸道發炎 → 呼吸道纖維化 → 呼吸道阻塞導致阻力上升
- 肺實質破壞：肺部回彈力降低＋肺泡失去對小呼吸道的貼附支撐
- 兩者共同導致 → **呼氣氣流受阻**

### 1.2 台灣疾病衝擊數字

| 指標       | 數值                                |
| ---------- | ----------------------------------- |
| 盛行率     | 6~10%（高盛行率）                   |
| 診斷率     | 33%（低診斷率）                     |
| 死亡率     | 22%（高死亡率，急性惡化出院後一年） |
| 治療遵循率 | 20%（低遵循率）                     |
| 共病率     | 50%（高共病率）                     |

- 住院死亡率：**4%**
- 加護病房住院率：**10%**
- 呼吸器使用率：**8.3%**
- 出院後**一年死亡率 22%**

**死因排名：**

- WHO 2019：COPD 為**全球十大死因第 3 名**，每年奪走約 300 萬人生命
- 台灣 109 年：慢性下呼吸道疾病為**十大死因第 8 名**，每年死亡逾 5,000 人，COPD 占近 5,000 人
- 2021 年台灣慢性下呼吸道疾病粗死亡率 26.6/10 萬人口

**全球盛行率（2019 系統性回顧，Lancet Respir Med 2022）：** 30–79 歲人口，依 GOLD 定義 10.3%（95% CI 8.2–12.8），全球約 3.919 億人；台灣健保資料庫盛行率 2.48%（明顯低估），電話問卷調查約 6.1–9.5%。

**罹病危險因子（Odds Ratio）：** 男性 OR 2.1；現行吸菸 OR 3.2、曾吸菸 OR 2.3；BMI<18.5 OR 2.2；biomass 燃料暴露 OR 1.4；職業/環境煙塵暴露 OR 1.4。

### 1.3 經濟與社會負擔

- 台灣健保資料庫（2010 年）：COPD 病人年平均醫療費用約新台幣 **9 萬 1 千餘元**；依嚴重度分層：中等嚴重度 3.8–14.9 萬元，嚴重 COPD 達 **28.8 萬元**。
- 2004–2010 趨勢：每位病人門診費用增加 29.3%，但總醫療費用因住院及 ICU 入院次數下降而減少 12.4%。
- 全球 DALY（失能調整生命年）損失排名預計 2030 年爬升至第 7 名；台灣 2018 年 COPD 造成 70 歲以下生命年數損失約 9.4 年，排名第十。

- **初級預防**：避免吸菸暴露、幼童呼吸感染預防、室內外空污與職場暴露控制、新生兒營養
- **二三級預防**：戒菸、肺部復健、降低有毒微粒暴露、早期檢測
- **跨機構分工**：國健署（戒菸與二手菸識能、臨床指引）、健保署（給付改善方案）、胸腔科暨重症醫學會（自在呼吸健康網線上支援系統）

## 2. 致病機轉、危險因子與流行病學

### 2.1 致病機轉

呼吸道因受到有害氣體或微粒（如吸菸菸霧）的**慢性刺激**，導致**異常發炎反應**，機轉包括：

1. 氧化壓力的產生
2. 肺部**蛋白酶調控失衡**（protease-antiprotease imbalance）
3. 發炎細胞的出現
4. 發炎介質
5. 氣管旁間質纖維化

病理變化主要發生於**呼吸道、肺實質、肺部血管**三大部位；反覆發炎反應造成的傷害與修復導致呼吸道結構改變（remodeling）。

### 2.2 影響發生與惡化進展的危險因子

1. 基因（如 α1-抗胰蛋白酶缺乏症）
2. 年齡與性別
3. 肺部生長與發育（是否達正常肺功能峰值）
4. 暴露於有害微粒：吸菸煙霧、職場塵埃或化學物質
5. 社經地位
6. 氣喘／支氣管過度反應
7. 慢性支氣管炎
8. 感染

### 2.3 常見共病症（COPD 為全身性系統疾病）

心血管疾病（高血壓、缺血性心臟病、心衰竭、心律不整）、代謝症候群（糖尿病、肥胖）、睡眠呼吸中止症、骨質疏鬆、焦慮憂鬱、感染症、肺癌、支氣管擴張症。台灣本土研究：COPD 病人較高機會併發癌症、躁鬱症；合併糖尿病者死亡率明顯提高。

### 2.4 病態生理與症狀對應

| 病態生理機轉                                   | 對應症狀   |
| ---------------------------------------------- | ---------- |
| 呼氣氣流受阻與空氣滯積                         | 呼吸喘促   |
| 黏液過度分泌                                   | 咳嗽、有痰 |
| 氣體交換異常、肺血管高壓、全身性表現、急性惡化 | 依個體而異 |

**身體檢查發現**：吐氣費力、噘嘴式吐氣、吐氣期喘鳴聲（呼氣氣流受阻相關）；胸廓前後徑增加/桶狀胸（肺氣腫相關）；聽診痰音 rhonchi（慢性支氣管炎相關）。**無這些徵象不代表沒有 COPD**。

### 2.5 影像學檢查角色

- **胸部 X 光**：評估肺部過度充氣、過度透亮、血管紋路減少、氣胸，及其他心肺共病；**無法用於確立 COPD 診斷**。
- **胸部 CT**：評估肺氣腫、氣體滯積、小氣道異常，及外科肺減容積手術前分佈範圍；不建議常規檢查，不用於確立診斷。
- GRADE（2A，弱建議，證據等級高）：確診後可用 CT 定量指標區分亞型，評估惡化、肺移植/手術前分布、共病及疾病進展。

### 2.6 2023 GOLD 新增診斷概念

| 概念                           | 定義                                                                                                                         |
| ------------------------------ | ---------------------------------------------------------------------------------------------------------------------------- |
| **Early COPD**（早期肺阻塞）   | 僅建議用於基礎研究，討論疾病生物學早期致病機轉                                                                               |
| **Mild COPD**（輕度肺阻塞）    | 「輕度氣流阻塞」≠「早期疾病」；僅描述肺量計測量嚴重程度；約 50% 病人因 FEV1 加速下降形成 COPD，另 50% 與肺部生長發育異常有關 |
| **Young COPD**（年輕型肺阻塞） | 20–50 歲因肺功能下降診斷 COPD；與早期未達肺功能峰值或早期下降有關；危險因子含家族史、5 歲前住院等早期生活事件                |
| **Pre-COPD**（肺阻塞前期）     | 任何年齡，未用支氣管擴張劑時 FEV1/FVC≥0.7，但已有呼吸症狀或結構/功能異常（肺氣腫、過度膨脹、擴散能力降低、FEV1 快速下降）    |
| **PRISm**                      | 支氣管擴張劑後 FEV1/FVC≥0.7 且 FEV1<80% 預測值；盛行率 7.1–20.3%，增加全因死亡率；可能進展為正常/阻塞性/限制性               |
| **Etiotype**（病因類型）       | 2023 GOLD 新增，將 COPD 病因擴展至非吸菸相關類型                                                                             |

## 3. 臨床診斷與嚴重度評估

### 3.1 診斷三要素

症狀（咳嗽、有痰、呼吸喘促）＋ 危險因子暴露（吸菸、有害氣體微粒）＋ **需要肺功能檢查證實呼氣氣流受阻**（吸入支氣管擴張劑後 **FEV1/FVC<0.7**）

**GRADE（1A，強建議，證據等級高）**：建議使用支氣管擴張試驗後 FEV1/FVC 比值來診斷肺阻塞。

**常用支氣管擴張劑試驗方案：**

| 藥物                                  | 劑量     | 測量時機                 |
| ------------------------------------- | -------- | ------------------------ |
| 乙二型交感神經刺激劑（如 salbutamol） | 400 微克 | 吸入後 10–15 分鐘測 FEV1 |
| 抗膽鹼藥物                            | 160 微克 | 吸入後 30–45 分鐘測 FEV1 |
| 二者併用                              | —        | 吸入後 30–45 分鐘測 FEV1 |

**FEV1 可逆程度**：≥12% 且 ≥200mL 提升，**但 GRADE（1B，強建議，證據等級中）：不應用可逆程度排除 COPD 診斷**，也不應用於預測長期治療反應。

### 3.2 篩檢建議

- GRADE（2B，弱建議，證據等級中）：**不建議**對無症狀成人使用篩檢問卷或篩檢性肺量計。
- 已有慢性咳嗽、咳痰、呼吸困難、喘鳴等症狀者，及具 α1-抗胰蛋白酶缺乏症家族史者，應立即安排肺功能及影像學檢查。

### 3.3 共病症對診斷之影響

**GRADE（1B，強建議，證據等級中）**：合併心衰竭、肺纖維化或嚴重肥胖等共病時，COPD 診斷應根據**臨床症狀、身體檢查、影像學與其他心肺功能參數之綜合判斷**。（嚴重肥胖病患 BMI 增加時 FEV1/FVC 也會增加，故 COPD 診斷與盛行率可能被低估）

### 3.4 GOLD 肺功能嚴重度分級（1–4 級）

適用對象：FEV1/FVC<0.7 之病人，依吸入支氣管擴張劑**後**FEV1% 預測值分級：

| GOLD 分級 | 嚴重度 | FEV1（% 預測值） |
| --------- | ------ | ---------------- |
| GOLD 1    | 輕度   | ≥80%             |
| GOLD 2    | 中度   | 50%≤FEV1<80%     |
| GOLD 3    | 重度   | 30%≤FEV1<50%     |
| GOLD 4    | 極重度 | <30%             |

**結論**：肺功能越差，急性惡化及死亡風險越高；但**肺功能與症狀關聯性不佳**——肺功能可評估「風險」，不是評估「症狀」的好工具。

### 3.5 疾病嚴重度整合評估工具

**評估架構**：GOLD 分級（肺功能）＋mMRC/CAT（症狀）＋急性惡化風險 → ABE 分組

- **CAT**（COPD Assessment Test）：評估咳嗽、咳痰、喘、活力、情緒、睡眠等多面向症狀，與 SGRQ 有良好關聯性且更方便使用。
- **mMRC**（Modified Medical Research Council）：主要評估呼吸喘促程度。

**GRADE（1A，強建議，證據等級強）**：頻繁惡化表現型（**每年 ≥2 次惡化**）應列入評估項目。過去急性發作頻率是未來急性發作頻率的最佳預測因子（**優於單獨使用肺功能**）。

### 3.6 GOLD 整合分組系統

**急性惡化風險評估：**

- 過去 1 年惡化 **≥2 次，或曾住院** → 高風險
- 過去 1 年惡化 **≤1 次**（未住院） → 低風險

**症狀評估：**

- mMRC 0–1 分、CAT<10 分 → 低症狀
- mMRC≥2 分、CAT≥10 分 → 高症狀

**2017/2019 GOLD ABCD 分組：**

|            | 低症狀 | 高症狀 |
| ---------- | ------ | ------ |
| 低惡化風險 | A      | B      |
| 高惡化風險 | C      | D      |

**2023 GOLD 更新為 ABE 分組**（C、D 合併為單一「E」組，不再依症狀細分高風險族群）：

- **A 組**：低惡化風險＋低症狀
- **B 組**：低惡化風險＋高症狀
- **E 組**：高惡化風險（過去 1 年 ≥2 次惡化或 ≥1 次因惡化住院），**不論症狀嚴重度**

**完整評估流程**：①肺功能確認診斷（FEV1/FVC<0.7）→②評估氣道阻塞程度（GOLD1-4）→③評估症狀/急性惡化危險（ABE 分組）

### 3.7 氣喘與 COPD 鑑別診斷

| 疾病特徵     | 氣喘                                              | COPD                                               |
| ------------ | ------------------------------------------------- | -------------------------------------------------- |
| 發病年齡     | 20 歲以前                                         | 40 歲以後                                          |
| 症狀特點     | 短時間變化大，夜間/凌晨較重，運動/情緒/過敏原誘發 | 治療後症狀持續存在，運動性呼吸困難，咳痰非誘發因素 |
| 肺功能       | 呼氣氣流受阻程度有變化                            | 持續性呼氣氣流受阻（FEV1/FVC<0.7）                 |
| 緩解期肺功能 | 正常                                              | 不正常                                             |
| 病史/家族史  | 曾診斷氣喘、過敏性疾病家族史                      | 曾診斷 COPD/慢性支氣管炎/肺氣腫、大量接觸危險因子  |
| 病程         | 可能季節性變化，對支擴劑/ICS 反應佳               | 逐年緩慢惡化，短效支擴劑效果有限                   |
| 胸部 X 光    | 正常                                              | 嚴重肺部過度充氣                                   |

**判讀原則**：具其中一種疾病 3 項以上特徵建議診斷該疾病；特徵數量相近需考慮 **ACO（氣喘肺阻塞重疊）**。

**ACO 特點**：較年輕、以女性為主、不吸菸者較多；FEV1 下降速度比單純氣喘快；伴心血管共病時住院風險較高。

**2016 西班牙 ACO 診斷標準：**

| 主要標準                          | 次要標準                    |
| --------------------------------- | --------------------------- |
| 過去氣喘病史                      | IgE>100 IU                  |
| 對 salbutamol 反應 >15% 且 >400mL | 特異體質過敏症病史          |
| —                                 | 二次測試反應 >12% 且 >200mL |
| —                                 | 周邊血液嗜酸性球 >5%        |

診斷準則：符合 1 個主要標準或 2 個次要標準 → 診斷 ACO。GRADE（1C，強建議，證據等級低）：符合 ACO 診斷者，建議以支氣管擴張劑試驗、嗜酸性球和 IgE 結果進一步判斷。

### 3.8 鑑別診斷檢查

- **血液嗜酸性球數**：COPD 治療（是否用 ICS）及預後的重要 biomarker（詳見 [5.4 節](#sec5-4)）
- 支氣管鏡、痰液培養等依臨床需要安排

## 4. 肺功能檢查詳解

### 4.1 基本概念與三項指標

肺量計是評估呼氣氣流受阻**重現性最高、最客觀**的檢查方式，配合病史症狀是診斷 COPD **最準確可靠**的方法。

| 指標     | 定義                                                                                                                 |
| -------- | -------------------------------------------------------------------------------------------------------------------- |
| FVC      | 用力呼氣肺活量，一口氣盡全力呼出的氣體總量                                                                           |
| FEV1     | 用力呼氣一秒量，用力呼氣第 1 秒的呼氣量                                                                              |
| FEV1/FVC | 正常值 >0.7；<0.7 可能為呼吸道阻塞                                                                                   |
| FEV6     | 6 秒用力呼氣量，可替代 FVC，重現性較優，患者較省力；FEV1/FEV6<0.73 約等同 FEV1/FVC<0.7（敏感性 94.4%，專一性 93.3%） |

### 4.2 支氣管擴張劑可逆性測試

- **可逆性定義**：FEV1 較基礎值提升 **≥12% 且 ≥200mL**；若 ≥400mL（或 >12%）為顯著增加，應考慮氣喘或併發氣喘（GINA 2017）。
- **COPD 病人使用支擴劑後 FEV1 可能增加，不能因此排除 COPD 診斷**；COPD 病人 FEV1/FVC 仍 <0.7。
- **檢查前準備**：短效藥物檢查前 4–6 小時不可使用；長效支擴劑前 12 小時不可用；超長效 LABA 前 24 小時不可用；檢查前 1 小時不可吸菸。
- FVC、FEV1 取**三次有效曲線最大值**，三次誤差不超過 5% 或 100mL。

### 4.3 判讀標準

**異常判斷標準：**

- FEV1<80% 預測值
- FVC<80% 預測值或 FEV6<82% 預測值
- FEV1/FVC<0.7 或 FEV1/FEV6<0.73

**阻塞性 vs 侷限性（限制性）判讀：**

| 型態           | FEV1        | FVC/FEV6    | FEV1/FVC                  |
| -------------- | ----------- | ----------- | ------------------------- |
| 阻塞性         | <80% 預測值 | —           | <0.7（或 FEV1/FEV6<0.73） |
| 侷限性（可能） | <80% 預測值 | <80% 預測值 | 正常（>0.7）              |

侷限性結果僅顯示呼吸困難可能不是 COPD，需進一步 TLC、DLCO 確認。

**流速-容積曲線特徵**：阻塞性肺病尖峰呼氣流速降低且曲線下凹；侷限性肺病曲線形狀正常但整體容積下降。

### 4.4 檢查操作要點

- 儀器每日校正（3L 注射器）；記錄年齡性別身高及最後用藥時間
- 患者姿勢：坐正、頭略抬高、雙腿不交叉
- 操作：捏鼻 → 深吸飽氣 → 閉氣含吹嘴 → **盡量快速用力呼氣**直到完全吐出（重度 COPD 可能需 15 秒）
- 至少重複 3 次，最佳兩次結果相差應在 **100mL 或 5% 以內**；單次檢查不超過 8 次（含練習）
- 75 歲以上患者預測正常值較不準確；呼吸道阻塞嚴重者，緩慢吐氣可能取得較可靠結果

### 4.5 肺功能檢查禁忌症

| 禁忌症                                            | 等級           |
| ------------------------------------------------- | -------------- |
| 近期心肌梗塞、升主動脈動脈瘤、肺栓塞、心絞痛      | 絕對／相對     |
| 胸腹部手術、腦眼耳 ENT 手術、氣胸、咳血、急性腹瀉 | 相對           |
| 嚴重高血壓（>200/120mmHg）                        | 檢查前應測血壓 |
| 意識不清或失智、患者不適、感染控制問題            | 視情況暫緩     |

### 4.6 常見操作錯誤與品質判讀

**操作錯誤清單**：吸氣不足、吐氣不盡全力、操作中咳嗽、額外呼吸、嘴唇未緊含吹嘴、開始吐氣延遲、太早結束吐氣、部分經鼻呼氣。

| 錯誤類型      | 容積-時間曲線     | 流速-容積曲線 |
| ------------- | ----------------- | ------------- |
| 起始用力不足  | 緩慢攀升          | 尖峰圓形/平坦 |
| 咳嗽          | 曲線下凹          | 曲線下凹      |
| 額外呼吸      | 縮影額外曲線      | 額外呼吸痕跡  |
| 聲門關閉/閉氣 | 突然變平          | 垂直下降      |
| 太早結束呼氣  | 平坦時間不足 1 秒 | —             |

> 提醒：可逆性試驗**無法預測 LABA/LAMA 相對於 ICS/LABA 的治療反應**，不建議根據可逆性試驗結果決定治療策略。

## 5. 藥物治療

### 5.1 治療目標

減緩/預防症狀、改善 FEV1、提升生活品質、降低急性惡化風險與死亡率。

### 5.2 藥物分類

| 給藥途徑  | 藥物                                                                                       |
| --------- | ------------------------------------------------------------------------------------------ |
| 吸入型    | SABA、SAMA、LABA、LAMA、ICS（及各種複方）                                                  |
| 口服/靜脈 | 茶鹼、口服類固醇、PDE4 抑制劑（roflumilast）、大環內酯類抗生素、口服化痰藥、口服 β-blocker |

### 5.3 支氣管擴張劑

**短效（SABA/SAMA）**：改善症狀的主要藥物，常規使用可減緩預防症狀（Evidence A）；合併使用效果優於單獨使用（Evidence A）。

**LAMA vs LABA（16 篇 RCT 統合分析）**：肺功能、症狀、生活品質無顯著差異；**預防急性惡化及安全性 LAMA 優於 LABA**。

**GRADE（1A，強建議，證據等級高）**：單一支擴劑 LAMA 與 LABA 均可考慮；急性惡化風險較高者建議優先選擇 LAMA。

**雙重長效支擴劑（LABA+LAMA）常見複方：**

| 成分                       | 商品名            |
| -------------------------- | ----------------- |
| Glycopyrronium-Indacaterol | Ultibro（昂帝博） |
| Umeclidinium-Vilanterol    | Anoro（安肺樂）   |
| Tiotropium-Olodaterol      | Spiolto（適倍樂） |
| Glycopyrronium-Formoterol  | Bevespi（必肺宜） |

**療效**：複方在 FEV1、TDI score、SGRQ 均優於單獨使用；tiotropium-olodaterol 較單用 tiotropium 降低 7% 中重度急性惡化。

**GRADE（1A，強建議，證據等級高）**：症狀較輕、低惡化風險（Group A）優先用單一 LAMA 或 LABA 起始，控制不佳可改複方；症狀較嚴重者（Group B）優先用固定劑量複方。

### 5.4 ICS

**劑量分級（每日 μg）：**

| 藥物                   | 低劑量  | 中劑量   | 高劑量    |
| ---------------------- | ------- | -------- | --------- |
| Beclomethasone         | 200–500 | 500–1000 | 1000–2000 |
| Budesonide             | 200–400 | 400–800  | 800–1600  |
| Fluticasone propionate | 100–250 | 250–500  | 500–1000  |
| Fluticasone furoate    | 100     | —        | 200       |

- **單獨使用 ICS**：無法改善 FEV1 長期下降，無法降低死亡率；**不建議單獨使用**。
- **ICS+LABA vs LABA+LAMA**：LABA+LAMA 在肺功能與健康狀況改善上明顯優於 ICS+LABA；ICS+LABA 肺炎風險明顯較高。

**三合一治療（ICS+LABA+LAMA）**：改善肺功能並防止急性惡化。**IMPACT 研究**：三合一較 LABA+LAMA 降低一年死亡率 **28%**；**ETHOS 研究**：降低 **49%**。

**血液嗜酸性球數作為 ICS 治療決策 biomarker（2019 GOLD 切點）：**

| 嗜酸性球數           | ICS 使用建議                       |
| -------------------- | ---------------------------------- |
| **≥300 cells/µL**    | ICS 適應症（起始或後續追蹤治療）   |
| **100–300 cells/µL** | 效果呈連續性關係，需綜合考量惡化史 |
| **<100 cells/µL**    | 不建議使用 ICS                     |

**GRADE（1B，強建議，證據等級強）**：高惡化風險穩定期病人，LABA+LAMA 預防惡化效果優於 ICS+LABA；但嗜酸性球愈高者對 ICS+LABA 反應可能愈好。

**ICS 使用時機建議：**

| 建議使用                                                             | 不建議使用                 |
| -------------------------------------------------------------------- | -------------------------- |
| 有氣喘病史或 ACO                                                     | 反覆肺炎發生               |
| 高風險（每年 ≥2 次惡化或曾住院）且嗜酸性球 ≥300 者，初始治療即可考慮 | 血液嗜酸性球 <100 cells/µL |
| 已用 1–2 種長效支擴劑仍惡化，血中嗜酸性球 ≥100（後續治療）           | 具肺結核感染病史           |

**ICS 副作用**：口咽念珠菌感染、聲音沙啞；長期高劑量增加肺炎及肺結核風險，風險與劑量、時間、年齡、吸菸狀況、BMI、肺功能相關。

**ICS 停用議題**：WISDOM Trial（2,485 人）——停用 ICS 不增加惡化風險，但 FEV1 一年後相差 43mL（肺功能惡化）。GRADE（1B，強建議，證據等級中）：LABA+LAMA+ICS 穩定病人若嗜酸性球 **<300**，可考慮在適當評估後停用 ICS，但需密切監控肺功能與惡化風險。

### 5.5 OCS

- 長期使用（一年以上）**不應常規使用**：每日 prednisolone>5mg 顯著增加脊椎骨折風險（OR 2.31）。GRADE（1C，強建議，證據等級低）。

### 5.6 Theophylline

- 低濃度（>5μg/ml）抗發炎；高濃度（>10μg/ml）支氣管擴張；更高（>20-25μg/ml）明顯副作用。需注意藥物交互作用。
- GRADE（2B，弱建議，證據等級中）：已用吸入型支擴劑但肺功能仍不佳，可用**最低有效劑量**口服茶鹼附加治療。

### 5.7 Oral PDE4i（Roflumilast）

- 抑制 cAMP 分解，抑制嗜中性球和嗜酸性球添補；每日 **roflumilast 500μg**：慢性支氣管炎表現型 FEV1、惡化頻率、生活品質皆改善。
- 副作用：腹瀉、噁心、頭痛、嘔吐、體重減輕；可能增加憂鬱焦慮失眠。
- GRADE：嚴重/非常嚴重障礙者可用（2B）；慢性支氣管炎伴生活品質不佳可用（2B）；慢性支氣管炎伴頻繁惡化**建議使用**（1A，強建議，證據等級強）。

### 5.8 Macrolide

- **azithromycin 250mg** 或 **erythromycin 200-750mg** 或 **clarithromycin 500mg**，使用 3–36 個月：降低惡化比率及頻率，改善生活品質。
- 副作用：胃腸道反應、聽力減弱（azithromycin）、抗藥性增加、QT 延長。
- GRADE（1A，強建議，證據等級強）：已用 ICS+LABA+LAMA 仍惡化者，可用大環內酯類（erythromycin 或 azithromycin）附加治療。

### 5.9 Mucolytics

GRADE（2B，弱建議，證據等級中）：未用 ICS 病人可用口服抗氧化型化痰藥（NAC、carbocysteine）降低惡化風險；Erdosteine 可有效減少輕度惡化風險。

### 5.10 口服 β-blocker

- 穩定期 COPD **並非**心臟選擇性 β1-blocker 使用禁忌。GRADE（1C，強建議，證據等級低）：合併心血管疾病可用心臟選擇性 β1-blocker 改善預後，惟應注意肺功能。

### 5.11 GOLD ABE 分組治療流程

**起始治療：**

| 分組                                 | 起始治療                                          |
| ------------------------------------ | ------------------------------------------------- |
| Group E（≥2 次中度惡化或 ≥1 次住院） | LABA+LAMA；血中嗜酸性球 ≥300 可考慮 LABA+LAMA+ICS |
| Group A（0–1 次未住院）              | 單一支擴劑                                        |
| Group B（0–1 次未住院，高症狀）      | LABA+LAMA                                         |

**後續治療調整（依 Dyspnea 軸與 Exacerbations 軸）：**

- Dyspnea 軸：LABA 或 LAMA → 症狀持續 → LABA+LAMA
- Exacerbations 軸：LABA 或 LAMA → 持續惡化 → LABA+LAMA → 持續惡化 → LABA+LAMA+ICS（依表現型可加 Roflumilast［FEV1<50% 且慢性支氣管炎表現型］或 Azithromycin［優先用於前吸菸者］）
- **考慮降階 ICS**：若發生肺炎或明顯副作用；嗜酸性球 ≥300 者降階較可能伴隨惡化，需審慎評估

**E 族群治療要點**：LABA+ICS **不建議**作初始治療；如有 ICS 適應症建議三合一治療（優於 LABA+ICS）。

## 6. 吸入性裝置衛教

### 6.1 裝置選擇基本原則

**藥物面**：評估症狀與惡化次數判斷 GOLD 分組（A/B/E），決定藥物種類。
**裝置面**：決定 pMDI 或 DPI；pMDI 需評估是否搭配 spacer；DPI 需依吸氣能力選擇。

### 6.2 台灣常見吸入型藥物

| 分類       | 藥物                                                                                                         |
| ---------- | ------------------------------------------------------------------------------------------------------------ |
| SABA       | 泛得林 Ventolin（Salbutamol）、備勞喘 Berotec（Fenoterol）                                                   |
| SABA+SAMA  | 冠喘衛 Combivent、備喘全 Berodual                                                                            |
| Ultra-LABA | 昂舒 Onbrez（Indacaterol）、適維樂 Striverdi（Olodaterol）                                                   |
| LAMA       | 英克賜 Incruse（Umeclidinium）、吸補力 Seebri（Glycopyrronium）、適喘樂 Spiriva（Tiotropium）                |
| LABA+LAMA  | 安肺樂 Anoro、適倍樂 Spiolto、昂帝博 Ultibro、必肺宜 Bevespi                                                 |
| ICS        | 保衛康 Alvesco、帝舒滿 Duasma、輔舒酮 Flixotide、可滅喘 Pulmicort                                            |
| ICS+LABA   | 潤娃 Relvar、使肺泰 Seretide、吸必擴 Symbicort、肺舒坦 Foster、呼特康 Flutiform、愛克喘 Atectura（僅氣喘用） |
| 三合一     | 肺樂喜 Trelegy、喘寶 Trimbow、艾能舒 Enerzair（僅氣喘用）、必肺暢 Breztri                                    |

### 6.3 裝置選擇決策表

| 口手協調 | 吸氣流速  | 建議裝置                 |
| -------- | --------- | ------------------------ |
| 好       | ≥30 L/min | pMDI、DPI、霧化器        |
| 好       | ≤30 L/min | pMDI、霧化器             |
| 差       | ≥30 L/min | pMDI+spacer、DPI、霧化器 |
| 差       | ≤30 L/min | pMDI+spacer、霧化器      |

### 6.4 各類裝置操作要點

**pMDI（壓力定量吸入器）**：搖動 4-5 次 → 打開吸嘴 →（超過 3 天未用需排空）→ 接延伸管 → 吐氣 → 按壓同時緩吸 4-5 秒 → 閉氣 5-10 秒 → 兩劑間隔 30-60 秒 → ICS 須漱口。

**DPI 裝置比較：**

| 裝置             | 操作要點                                                                      | 閉氣時間 |
| ---------------- | ----------------------------------------------------------------------------- | -------- |
| Accuhaler 準納   | 推開至卡住 → 扳上藥扳手至「嗒」聲 → 快速用力吸 → 關閉                         | 10 秒    |
| Ellipta 易利達   | 開 → 吸（緩而長）→ 關；口訣「開 → 吸 → 關」                                   | 3-4 秒   |
| Breezhaler 微風  | 裝入膠囊 → 蓋上 → 兩鈕壓到底刺破 → 穩定吸（聽膠囊滾動聲）→ 可重複吸至藥粉吸完 | 10 秒    |
| Turbuhaler 都保  | 保持直立 → 紅色底盤右轉到底再左轉到「喀」聲 → 快速用力吸                      | 5-10 秒  |
| NEXThaler 耐舒樂 | 開蓋（即完成上藥）→ 深吸至「喀」聲，持續 1-2 秒 → 關蓋（此時劑量窗格才減少）  | 5-10 秒  |

**Respimat（SMI 緩釋型氣霧吸入器）**：首次需裝填藥罐並排空試噴三次；使用時右轉半圈至「卡塔」聲 → 吐氣 → 按黑鈕同時緩吸 → 閉氣 5-10 秒；每日一次共吸兩劑，間隔 30-60 秒。

**霧化器（Nebulizer）**：藥物以生理食鹽水稀釋至 4-6mL；驅動氣體流量 6-10 L/min；面罩連接；平常潮氣容積慢速呼吸（30 L/min）。

### 6.5 常見使用錯誤重點

- pMDI：瓶底未朝上、未充分振搖、按壓力道不足、手口協調差（建議配合 spacer）
- Accuhaler：未完成上藥步驟、吸嘴朝下藥物散落、對吸嘴吐氣
- Ellipta：打開未吸而損失劑量、手指擋通氣孔
- Breezhaler：無「喀」聲及甜味（未吸到藥粉）、直接吞服膠囊
- Turbuhaler：未保持直立、初次未排空、吸力不足未察覺
- Respimat：未完成藥罐充填、未初次排空三次、先開蓋再轉底座、吸太快

### 6.6 吸藥輔助器（Spacer）

**優點**：增加藥物肺部濃度、減少嘴破聲音沙啞、減少口腔咽部藥物沉澱及念珠菌感染、減少全身副作用。**使用時機**：pMDI 使用時無法達到手口協調者。

**清潔**：約 3 個月一次，中性清潔劑常溫水浸泡 15-30 分鐘，勿刷洗、勿高低溫消毒、不可共用、勿水龍頭直沖面罩。

### 6.7 遵囑性評估

約 **80% 病人未正確使用吸入器**；改善策略：4C 模式（Choose 選擇、Check 檢查、Correct 修正、Confirm 確認）；7 步驟衛教流程（示範 → 重複確認直到熟練 → 提供書面資料）。

**含類固醇成分吸入藥物，吸藥後皆須漱口**；**兩劑間隔至少 30-60 秒**（適用多數需吸兩劑之裝置）。

## 7. 非藥物治療與肺部復原

### 7.1 核心非藥物治療項目

無菸環境、加強處方藥物依從性、正確吸入器使用技術、促進身體活動、接種疫苗、轉診肺部復原。

### 7.2 戒菸

- 戒菸是**最能改變 COPD 自然病史**的策略，有效投入可達 25% 長期戒菸成功率。
- **尼古丁替代治療（NRT）**：口香糖、吸入劑、鼻噴劑、皮膚貼片、舌下錠、口含錠，皆優於安慰劑。
- **藥物**：Varenicline、Bupropion、Nortriptyline——僅作**支持性介入**，不建議單獨使用。
- GRADE（1A）：所有吸菸 COPD 病人，無論嚴重度，強烈建議戒菸。

### 7.3 疫苗接種

**建議接種**：流感疫苗、肺炎鏈球菌疫苗、百日咳疫苗、COVID-19 疫苗、帶狀疱疹疫苗（**50 歲以上建議**）。

**肺炎鏈球菌疫苗接種順序（65 歲以上，依過往接種史）：**

| 過往接種史          | 接續建議                          |
| ------------------- | --------------------------------- |
| 從未接種            | PCV13→PPV23                       |
| 65 歲後曾接種 PPV23 | 接種 PCV13                        |
| 65 歲前曾接種 PPV23 | 接種 PCV13（與 PPV23 間隔 ≥1 年） |
| 曾接種 PCV13        | 接種 PPV23（間隔 ≥1 年）          |

PCV13 與 PPV23 間隔 ≥1 年；兩劑 PPV23 間隔 ≥5 年。

### 7.4 運動訓練與肺部復原（Pulmonary Rehabilitation）

**定義**：詳細評估後依個別狀況擬定的全面性介入，包括運動訓練、衛教與健康行為促進。

**執行族群（2023 GOLD ABE）**：建議族群為 **B、E 組**；輕度 COPD 無明顯症狀不強烈建議。

**執行時間**：持續 **6-8 週**，**一週二次**監督式運動訓練。

**效益（依證據等級）：**

| 效益                          | 證據等級 |
| ----------------------------- | -------- |
| 減少喘、增加運動容量耐受性    | A        |
| 增加生活品質                  | A        |
| 下降住院次數天數              | A        |
| 下降焦慮憂鬱                  | A        |
| 增加急性發作後復原程度        | A        |
| 增加長效支擴劑作用            | B        |
| 降低急性發作後 4 週內再住院率 | B        |
| 增加存活率                    | B        |

**運動訓練分類**：耐力訓練、阻力/肌力訓練、上肢運動、呼吸/吸氣肌肉訓練、全身振動訓練（WBV）；恆定負荷或間歇訓練併合力量訓練效果較佳。

**評估工具**：6 分鐘行走測試（6MWT）、漸增/耐力式往返走路測試（ISWT/ESWT）、CPET 心肺功能運動測試；健康狀況問卷 CAT/CRQ/SGRQ；心理評估 HADS、PRIME-MD。

### 7.5 呼吸訓練技巧

**噘嘴式呼吸（Pursed Lip Breathing）**：鼻吸氣默念 1、2 → 噘嘴吐氣默念 1234；**吐氣時間約吸氣 2 倍以上**。

**瑜珈調息法**：吐氣約 5 秒 → 屏息 2-3 秒 → 吸氣約 5 秒。

**柔軟訓練**：慢拉慢放不憋氣，維持 10-30 秒/次，3-5 次/組。

**胸腔物理治療（CPT）**：主動呼吸技巧（ACBT）、自發性引流、姿位引流（每日 3-4 次，飯前 30 分-1 小時或飯後 1-1.5 小時執行，每次 10-15 分鐘）、高頻胸腔振動。

**省力技巧**：訓練兩週後日常活動能量消耗與代謝當量顯著下降，血氧濃度上升。

### 7.6 營養評估

- COPD 住院病人 **30-60% 營養不良**；高達 50% 體重低於理想體重 90%。
- 機轉：促發炎細胞因子上升、leptin 上升抑制食慾；負氮平衡致肌肉量功能下降。
- 營養支持可改善肌肉量、BMI、運動表現，但**無法改善肺功能**；住院病人補充蛋白質可降低死亡率。

### 7.7 氧氣治療

**長期氧氣治療適應症（2023 GOLD）：**

| 條件                                           | 切點                       |
| ---------------------------------------------- | -------------------------- |
| 條件一（3 週內兩次）                           | PaO2≤55mmHg 或 SaO2<88%    |
| 條件二（合併肺高壓/心衰竭/紅血球增生 Hct>55%） | PaO2 55-60mmHg 或 SaO2 88% |

- 調整目標：SaO2≥90%；每 2-3 個月評估是否繼續氧氣治療。
- 供氧系統：氧氣濃縮機（居家常用）、壓縮氧氣鋼瓶（滿筒 2200psi）、液態氧氣系統（輕巧攜帶）。

### 7.8 呼吸器與介入性治療

- **NIV**：偶爾用於病情穩定但嚴重、白天明顯高碳酸血症且近期住院者，證據尚無法支持或反駁。
- **CPAP**：支持用於 COPD 合併 OSA 病人。
- **介入支氣管鏡/外科手術**：適用難治型肺氣腫或顯著過度充氣；肺減容積術（EBV、LVRC、VA 熱消融）、大肺泡切除術（bullectomy）、肺移植。

### 7.9 緩和療護與安寧照護

**緩和醫療分層**：初級緩和醫療（原團隊提供症狀舒適）、次級緩和醫療（專家團隊處理複雜需求）。

**介入時機**：嚴重肺功能退化、每年惡化住院 ≥2 次、曾因惡化致呼吸衰竭、使用 NIPPV、居家氧氣、嚴重共病、症狀未改善、日常功能嚴重退化。

## 8. 急性發作處置

### 8.1 定義

病人以**呼吸困難和/或咳嗽、咳痰**為特徵的症狀，在 **<14 天內**急性惡化，超過平日常態變化，導致藥物治療改變。

**GRADE（1B，強建議，證據等級中）**：診斷依賴症狀急性變化（呼吸困難加重、痰量增加、痰變濃稠），但**需排除**心衰竭、肺栓塞、ACS、氣胸、肺炎、肺塌陷。

### 8.2 Anthonisen 診斷標準（1987）

**主要症狀**：呼吸困難加重、痰液量增加、痰液變膿樣
**次要症狀**：喉嚨痛、感冒症狀、無其他原因發燒、咳嗽、喘鳴

**診斷標準**：≥2 項主要症狀惡化（持續 ≥2 天），或 ≥1 項主要症狀 +≥1 項次要症狀（持續 ≥2 天）。

（臨床分型：3 項主要症狀 =Type I 最重度，抗生素效益最明確；2 項主要 =Type II；1 項主要 +1 次要 =Type III）

### 8.3 嚴重度分級

| 嚴重度 | 定義                                                       |
| ------ | ---------------------------------------------------------- |
| 輕度   | 調整平常用藥（增加 SABA）即可改善，不需全身性類固醇/抗生素 |
| 中度   | 需全身性類固醇及/或抗生素才能改善，不需住院                |
| 重度   | 需住院治療；急診處置超過 **24 小時**視為住院               |

### 8.4 急性惡化成因

約 2/3 為呼吸道感染及空氣污染所致；其餘 1/3 原因難以確定。**常反覆嚴重惡化高風險特徵**：合併氣喘（Adj OR 4.02）、使用家庭氧療（Adj OR 9.39）、肺功能較差（FEV1, Adj OR 0.16）。

**既往急性惡化史是預測未來惡化最強因子**（1 年內 ≥2 次 vs 0 次，OR 5.72，95% CI 4.47-7.31）。

### 8.5 急性惡化的檢驗檢查（GRADE 建議）

| 建議                                          | 等級                     |
| --------------------------------------------- | ------------------------ |
| 全血球計數（嗜酸性球作類固醇治療參考）        | 2B                       |
| 血液生化（電解質、血糖）                      | 2C                       |
| 動脈血氣分析或血氧監測（氧氣/呼吸器治療參考） | 1C                       |
| 痰液細菌培養（懷疑感染時）                    | 2C                       |
| 心電圖（排除心臟病）                          | 1C                       |
| 胸腔 X 光（排除其他胸部疾病）                 | 1C                       |
| 急性期肺功能檢查                              | **不建議常規執行**（1C） |
| 生物標記作診斷依據                            | **不建議使用**（2C）     |

### 8.6 藥物治療

**支氣管擴張劑**：短效 SABA+/-抗膽鹼藥物為初始治療；盡快開始長效支擴劑維持治療；甲基黃嘌呤**不建議使用**。

**全身性類固醇**：

- **REDUCE 試驗**：5 天療程不亞於 14 天。**GOLD 2023 建議：每日總量 40mg prednisolone，為期 5 天**。
- GRADE（1B，強建議，證據等級中）：使用全身性類固醇可縮短恢復時間、改善肺功能與低血氧，降低早期復發、治療失敗及長期住院風險。

**抗生素**：治療約 5 天。

| 症狀組合                                   | 建議           | 等級 |
| ------------------------------------------ | -------------- | ---- |
| 呼吸困難 + 痰量增加 + 膿痰增加（三項並存） | 建議使用抗生素 | 1B   |
| 膿痰增加 +（呼吸困難或痰量增加，其一）     | 建議使用抗生素 | 1C   |
| Procalcitonin 可作為抗生素使用指標         | 可考慮         | 2B   |

### 8.7 氧氣治療

- 目標飽和度 **88-92%**；Venturi 面罩比鼻導管更準確可控。
- GRADE（1B）：血氧飽和度 <88% 建議使用氧氣治療。
- **經鼻高流量氧氣（HFNT）**：可達 60 L/min，可改善高碳酸血症但對死亡率無顯著影響（GRADE 2B）；2022 歐洲胸腔醫學會指引：急性高碳酸血症呼吸衰竭者應**先嘗試 NIV**再考慮 HFNT。
- 無低血氧者使用氧氣可能致二氧化碳滯留及嚴重呼吸性酸中毒。

### 8.8 住院與加護病房收治標準

**考慮住院**：休息時呼吸困難突然惡化、呼吸頻率高、SpO2 降低、意識模糊嗜睡、新發紫紺/周邊水腫、初始治療無反應、嚴重共病、家庭支持不足。

**急性呼吸衰竭判定**：呼吸速率 **>30 次/分**及/或使用呼吸輔助肌。

**住加護病房**：初始治療反應不佳之嚴重呼吸困難、精神狀態變化、**PaO2<5.3 kPa（40mmHg）** 和/或 **呼吸性酸中毒 pH<7.25**持續或惡化、需侵襲性正壓呼吸器、血液動力學不穩定需升壓劑。

### 8.9 呼吸器支持

**NIV 為 COPD 急性呼吸衰竭建議之初始通氣模式**，可降低死亡率和插管率。

**NIV 適應症（符合任一）**：呼吸性酸中毒（**PaCO2≥6.0kPa/45mmHg 且 pH<7.35**）、嚴重呼吸困難伴呼吸肌疲勞、儘管氧氣治療仍持續低氧血症。

**NIV 禁忌症**：心跳停止、血液動力學不穩定、高嗆咳風險、意識不佳無法配合、近期顏面手術、痰液過多過黏。

> 病情改善能承受**至少 4 小時**自主呼吸，NIV 可直接停止，無需 weaning 過渡期。

**侵襲性正壓呼吸器適應症**：無法忍受/NIV 失敗、呼吸心跳停止、意識改變或躁動、大量異物吸入或持續嘔吐、持續無法清除分泌物、嚴重血流動力不穩定、嚴重心律不整。

### 8.10 住院期間肺復原

- **高強度運動訓練可能增加死亡率**；但肌肉阻力訓練及衛教安全且有效。
- **早期（4 週內）** 肺復原可降低死亡率、住院天數、再入院次數，效果維持 ≥12 個月。
- 美國資料：住院後 **90 天內**開始肺復原，1 年內死亡風險顯著減少 **37%**。
- GRADE（1B，強建議，證據等級中）：早期肺復原能降低再入院率、減少呼吸困難、改善運動能力及生活品質。
- 出院後肺復原建議於 **48 小時內**啟動，效果優於住院當下立即介入（GRADE 2B）。

### 8.11 出院準則與追蹤

**出院條件**：可用 LABA/LAMA±ICS；SABA 使用頻率超過 4 小時以上；行走能力良好；已穩定無症狀 **12-24 小時**；ABG 值穩定 **12-24 小時**；病人/照護者了解用藥方式；已安排追蹤。

**追蹤評估項目**：適應環境能力、測量 FEV1、吸入器技巧、藥物瞭解、是否需長期氧氣/居家噴霧、CAT/mMRC 分數、共病症狀態。

## 9. 共病照護

### 9.1 總論

合併症在任何嚴重度 COPD 都很常見；**合併症存在不應改變 COPD 治療原則**，COPD 與合併症皆應按通常標準治療；多重病症照護應**簡化治療、減少多重用藥**。

### 9.2 心血管相關共病

**心臟衰竭**：COPD 病人心衰竭盛行率 **20-70%**，年發生率 3-4%；40% 高碳酸血症呼吸衰竭 COPD 病人有左心室功能不全證據。β1-blocker 改善心衰生存率，建議在有心血管適應症下使用**選擇性**β1-blocker。

**COPD 病人心臟衰竭診斷（2020 台灣心胸共病共識手冊）**：

- 胸腔專科視角：端坐呼吸/PND/水腫、≥2 個冠心病危險因子、不相稱呼吸困難 → BNP>100pg/mL 或 NT-proBNP>300pg/mL → 心臟超音波
- 用藥：ACEi、ARB、ARNI、心臟選擇性 β-blocker（bisoprolol、metoprolol、nebivolol）、MRA；竇性心律偏快可用 Ivabradine；**non-DHP 類 CCB（verapamil、diltiazem）不建議用於收縮性心衰**

**缺血性心臟病**：急性發作住院者 90 天內出現急性心肌梗塞/中風/顱內出血死亡率較高；異常 troponins 增加短期及長期死亡風險。

**心律不整**：心房顫動常見且與較低 FEV1 有關；**短效 β2 促效劑和茶鹼酸鹽應謹慎使用**（可能引起心房顫動）；長效 β2 促效劑、抗膽鹼藥物、ICS 安全性較高。

### 9.3 肺癌篩檢

- 肺癌與**肺氣腫**的關聯性比與氣流受限嚴重度更高。
- **USPSTF 2021 建議**：**50-80 歲**、**20 包年**吸菸史、目前吸菸或過去 **15 年內**戒菸者，**每年低劑量 CT（LDCT）篩檢**。GOLD 建議符合條件 COPD 病人每年 LDCT 篩檢。

### 9.4 其他共病

| 共病              | 重點                                                                                                       |
| ----------------- | ---------------------------------------------------------------------------------------------------------- |
| 支氣管擴張症      | COPD 病人盛行率 20-69%（平均 54.3%）；合併者急性惡化更頻繁、死亡率更高；**ICS 可能不適用**（細菌聚生風險） |
| OSA               | 同時患 COPD+OSA 預後更差；正壓通氣可減少住院急診及惡化                                                     |
| 糖尿病/代謝症候群 | COPD 病人代謝症候群盛行率 >30%；全身性類固醇與血糖代謝障礙相關；**重度 COPD 須維持 BMI≥21 kg/m²**          |
| GERD              | COPD 惡化**獨立風險因素**；胃酸抑制劑預防價值仍有爭議                                                      |
| 骨質疏鬆          | 與肺氣腫、低 BMI 相關；**全身性類固醇顯著增加風險，應盡可能避免為 AECOPD 重複使用**                        |
| 焦慮憂鬱          | COPD 病人**自殺風險為無 COPD 者 1.9 倍**；肺復健可能有正向影響；治療原則同一般病人                         |

| 牙周炎            | COPD 急診次數增加與牙周炎風險相關；牙周治療可能與肺功能下降減慢有關                                        |

## 10. 監測與追蹤／健保給付規範

### 10.1 監測四大面向

疾病進程及併發症、藥物或非藥物治療、急性惡化病史、共病症。

### 10.2 建議追蹤評估頻率（依 GOLD 嚴重度分層）

| 項目       | GOLD 1-3                                                                                | GOLD 4                                          |
| ---------- | --------------------------------------------------------------------------------------- | ----------------------------------------------- |
| 肺功能評估 | 至少每年一次                                                                            | 至少每半年一次                                  |
| 臨床評估   | 吸菸/戒菸意願、症狀控制、惡化風險、共病、吸入器技巧、遵囑性、副作用、營養、是否需肺復原 | 同左，另加：是否需氧氣治療、是否需 NIPPV 等介入 |
| 測量       | BMI、mMRC 或 CAT                                                                        | 同左，另加 SpO2                                 |

### 10.3 自我處置計畫

**良好自我處置計畫組成**：症狀監控、解決問題能力（急性惡化辨識處理）、戒菸協助、藥物遵囑性、身體活動、營養狀況。成效：提升生活品質、改善呼吸困難、減少住院；**對肺功能、運動能力及死亡率無明顯改變**。

### 10.4 收案與給付規範重點問答

- **收案條件**：90 天內同院所主診斷 COPD（ICD10 J41-J44），需有肺功能紀錄，至少就醫 2 次（**不限門診或住院**）；轉介病人不受 90 天限制。
- **P6015C 肺復原及呼吸訓練評估費**：600 點/次，限**第一類院所**申報，限**每半年一次，一年至多兩次**，需專業人員親自實施並記錄於病歷；病人拒絕執行則勾選「無」，無法申報。
- **17006C 支氣管擴張試驗**：基層診所可申報（原 17006B，106 年 5 月起修訂）。
- **上下轉**：轉出院所需於 VPN 登錄結案，轉入院所登錄新收案；因轉診重新收案免受 90 天兩次就醫限制。
- 年度病人追蹤率 <12% 經輔導 6 個月未改善，暫停資格半年（個案數 <10 人者除外）。

### 10.5 案例學習重點（教材範例摘要）

**案例一（初診斷、輕度病程）**：68 歲男性、45 年菸齡（一天兩包）、GOLD3 但 Group A（無惡化史、CAT9、mMRC1）；初始 LABA 治療 6 個月效果有限後改 LABA+LAMA，後續穩定追蹤 2 年（每三個月回診），已成功戒菸並規則運動，肺功能穩定於 FEV1 44-48%。

**案例二（急性惡化升階治療、最終上轉整合照護）**：59 歲男性、COPD 病史 20 年、LAMA+LABA 規則使用中，因肺炎併 AECOPD 住院（SpO2 86%，需 NIPPV、IV methylprednisolone 40mg/day、IV moxifloxacin 400mg/day），出院後升階為 LABA+LAMA+ICS；後因血中嗜酸性球降至 80/µL 且惡化風險降低嘗試停用 ICS；三年後因心肌梗塞住院、肺功能持續下降（FEV1 36%）、未能戒菸、營養不足，轉至第一類院所接受完整評估（含 6MWT、DLCO）、再次加上 ICS 成三合一治療、建議 PDE-4 抑制劑或 Macrolide，並轉介戒菸門診、營養師、精神科（疑似憂鬱症）；病人簽立 DNR。

## 11. 病患衛教溝通技巧

### 11.1 影響醫囑遵從度的三大構面

- **患者面**：健康觀念、認知能力、自我效能、共病症、心理特性、疾病覺悟
- **治療面**：給藥途徑、用藥時程、多重用藥、副作用
- **社會面**：醫病關係、社會支持、藥物取得、吸入器訓練、追蹤

### 11.2 溝通誠實原則

- 需明確告知患者：**目前研究並未證實任一藥物能改變肺功能的長期衰退**。
- **唯一被證實能有效減緩肺功能衰退速度的介入是戒菸**——衛教溝通應以此為核心誠實告知重點，避免不切實際的藥物期待。

### 11.3 每次回診綜合性 COPD 諮詢檢核表

治療回顧（吸入器技巧、自主管理計畫）、評估問題嚴重度（肺量計、SpO2、BMI）、評估症狀（CAT、mMRC）、急性惡化（抗生素/OCS 療程數、住院次數）、吸菸狀態、回顧目標（戒菸、體能活動）、接種疫苗、整體回顧（憂鬱焦慮、社交狀況）、評估轉介需求（肺復原、緩和療護、骨鬆評估、心血管評估）、下次回診時間。

### 11.4 吸入器衛教溝通實證

- 數十年來（1965-2014）**僅不到 40% 患者能正確使用吸入劑**，未見明顯改善。
- 大型研究（n=3,112）：操作**無錯誤組**中重度惡化率 32.1%，**關鍵錯誤組**達 38.5%；曾獲示範者無錯誤組 90% vs 關鍵錯誤組 83.6%；遵囑性不佳比例無錯誤組 4.4% vs 關鍵錯誤組 12.6%。
- **7 步驟確認流程**：病人先示範 → 醫護示範修正 → 病人再示範 → 必要時反覆 → 即使病人自述會用仍要求示範確認（teach-back 模式）→ 提供書面資料並註記錯誤步驟 → 強調輔助器效益。

### 11.5 提升用藥遵囑性

- 長效吸入劑 1 年後僅 **21-27%** 病人持續使用初始治療；3 年後僅剩 17%。
- **遵囑性與死亡率關聯**：未遵從醫囑（用藥 ≤80%）3 年死亡率 **26.4%** vs 遵從醫囑（>80%）**11.3%**——可作為溝通時強調重點。
- 妨礙因素：欠缺疾病藥物知識、醫病信賴不足、**多重用藥**（COPD 患者平均使用 6 種藥物）、無法立即緩解症狀的藥物遵囑性較低。

### 11.6 戒菸衛教：5A 模式（GOLD 2017）

1. **詢問（Ask）**：每次回診確認吸菸狀態
2. **建議（Advise）**：明確堅定個別化勸導戒菸
3. **評估（Assess）**：詢問戒菸意願（如「往後 30 天內是否願意戒菸」）
4. **協助（Assist）**：擬訂戒菸計畫、提供諮詢、社會支持、藥物治療、輔助資料
5. **安排（Arrange）**：預約追蹤（門診或電話）

**「肺部年齡」溝通技巧**（Parkes G, et al. BMJ 2008，N=561）：告知「肺部年齡」（而非僅原始 FEV1 數據）組，12 個月戒菸率 **13.6%** vs FEV1 組 **6.4%**（差異 7.2%，p=0.005，NNT=14）——將肺功能數據轉譯為易懂、有情緒衝擊的概念可提升戒菸動機。

### 11.7 自主管理訓練實證效益（Cochrane 2014）

| 結果指標               | 相對療效         |
| ---------------------- | ---------------- |
| SGRQ 總分              | MD −3.51（改善） |
| 因呼吸道問題住院 ≥1 次 | OR 0.57          |
| 因所有原因住院 ≥1 次   | OR 0.60          |
| mMRC 分數              | MD −0.83（改善） |

**個人化急性惡化行動計畫應包含**：如何辨識急性惡化、應採用之治療及療程長度、緊急聯絡方式、建議就醫評估症狀緩解。

### 11.8 跨團隊照護

醫師、護理師、物理治療師、職能治療師、社工、運動專家、呼吸治療師、心理師、營養師——衛教溝通為多專業團隊共同分工，非僅醫師單一角色責任。

## 12. 菸害防制

### 12.1 吸菸與 COPD

- 吸菸者罹患 COPD 風險為非吸菸者 **6.3 倍**；COPD 死亡中 **45%** 可歸因於菸品使用。
- 菸煙含超過 **7,000 種**化學物質，**93 種**致癌，**15 種**為第一級致癌物。
- 二手菸每年造成全球 **57 萬幼童死亡**；5 歲以下幼童死亡第一名為空污及二手菸引發的呼吸道感染。

### 12.2 電子煙與加熱菸危害

- 美國 CDC 通報電子煙嚴重肺部疾病達 **1,888 人**，死亡 **37 人**（108.10.29 統計）。
- **急性外因性油脂性肺炎（AELP）**：吸電子煙含大麻油或濃縮物者可見兩側玻璃粉及樹芽狀浸潤。
- **丁二酮**成分可引發「爆米花肺」（阻塞性細支氣管炎）。
- 電子煙僅取代吸菸，非戒除尼古丁，長期使用致成癮，呼吸道副作用及氣喘比率反而增加。
- 電子煙含重金屬：鉻（上呼吸道癌）、鉛（神經腎臟生殖系統）、錳（神經系統）、鎳（過敏肺癌鼻竇癌）。
- 大型研究（N=21,618）：電子煙使用與 COPD、肺氣腫、慢性支氣管炎、氣喘等呼吸道疾病風險增加獨立相關。

### 12.3 2023 菸害防制法修法重點

**六大重點**：①禁止電子菸、嚴管加熱菸及載具　②禁止促銷廣告　③禁止郵購電子購物販賣　④禁止營業場所免費提供菸品　⑤未滿 20 歲不得吸菸　⑥菸品警示圖文面積不得小於 **50%**。

電子煙製造進口販賣可處**新台幣 1,000 萬至 5,000 萬元罰鍰**。

### 12.4 戒菸服務體系

- 依 WHO 菸草控制框架公約（FCTC）第 14 條；經費來自菸品健康福利捐。
- **每年補助 2 個療程**，每次療程最多補助 **8 週藥費或 8 次戒菸衛教**；部分負擔 20%、最高 200 元；住院急診亦可戒菸；個案追蹤 3 個月及 6 個月。
- **免付費戒菸諮詢專線：0800-636363**（週一至週六 9:00-21:00）。
