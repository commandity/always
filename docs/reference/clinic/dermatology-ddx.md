## 六大步驟總覽

| Step  | 步驟                                | 內容                                |
| ----- | ----------------------------------- | ----------------------------------- |
| **1** | Identify Primary Lesion（原發病灶） | 確認最早出現的單一病灶形態          |
| **2** | Assess Distribution（分布排列）     | 評估位置、排列、範圍                |
| **3** | Evaluate Symptoms（症狀特質）       | 癢 vs 痛 vs 無症狀                  |
| **4** | Time Course（時間病程）             | 急性（小時∼天）vs 慢性（週∼年）     |
| **5** | Diagnostic Tools（輔助檢查）        | 皮膚鏡、KOH、伍氏燈、切片、貼膚測試 |
| **6** | Systemic Correlation（全身關聯）    | 發燒、關節痛、黏膜受累              |

## 原發病灶分類

### 平坦病灶

| 類型       | 尺寸  | 常見鑑別診斷                                       |
| ---------- | ----- | -------------------------------------------------- |
| **Macule** | ≤1 cm | Vitiligo, freckles, lentigo, pityriasis versicolor |
| **Patch**  | >1 cm | Melasma, café-au-lait spots, vitiligo, CTCL patch  |

**Hypopigmented**: Pityriasis versicolor, Vitiligo, Post-inflammatory hypopigmentation, Tuberous sclerosis (ash-leaf)

**Hyperpigmented**: Melasma, Freckles (ephelides), Café-au-lait spots (NF1), Lentigo, Post-inflammatory hyperpigmentation

**Erythematous**: Drug eruption (morbilliform), Viral exanthem, Early cutaneous T-cell lymphoma, Pityriasis rosea (herald patch)

### 隆起實質病灶

| 類型       | 尺寸                | 常見鑑別診斷                                   |
| ---------- | ------------------- | ---------------------------------------------- |
| **Papule** | ≤0.5 cm             | Acne, lichen planus, warts, folliculitis       |
| **Plaque** | >0.5 cm flat-topped | Psoriasis, eczema, lichen simplex              |
| **Nodule** | 0.5–2 cm deep       | Erythema nodosum, dermatofibroma, neurofibroma |
| **Tumor**  | >2 cm               | BCC, SCC, melanoma, lymphoma                   |

**Papulosquamous** (papules + scale):

- **Psoriasis**: Silvery scale, extensor surfaces, Auspitz sign, nail pitting
- **Lichen planus**: Purple, polygonal, planar; Wickham striae; oral lesions
- **Pityriasis rosea**: Herald patch, Christmas tree distribution on trunk
- **Seborrheic dermatitis**: Greasy yellow scale, scalp/eyebrows/nasolabial
- **Secondary syphilis**: Copper macules on palms/soles, lymphadenopathy
- **Cutaneous T-cell lymphoma**: Chronic, pruritic, often misdiagnosed as eczema
- **Drug eruption**: Temporal link to new medication

**Nodular**:

- Erythema nodosum (pretibial, tender, ± systemic)
- Dermatofibroma (dimple sign)
- Neurofibroma (NF1)
- Kaposi sarcoma (HHV-8, HIV-associated)

### 液體填充病灶

| 類型        | 內容物               | 常見鑑別診斷                                     |
| ----------- | -------------------- | ------------------------------------------------ |
| **Vesicle** | Clear fluid, ≤0.5 cm | HSV, HZV, varicella, dyshidrotic eczema          |
| **Bulla**   | Clear fluid, >0.5 cm | Pemphigus vulgaris, bullous pemphigoid, SJS/TEN  |
| **Pustule** | Pus                  | Folliculitis, acne, pustular psoriasis, impetigo |

**Vesicular**:

- **Herpes simplex**: Grouped vesicles on erythematous base, painful
- **Herpes zoster**: Dermatomal, prodromal pain, unilateral
- **Varicella (chickenpox)**: "Dewdrop on rose petal," centrifugal distribution
- **Dyshidrotic eczema (pompholyx)**: Sides of fingers/toes, vesicles
- **Contact dermatitis**: Linear/geometric pattern, exposure history

**Bullous**:

- **Pemphigus vulgaris**: Flaccid bullae, +Nikolsky sign, mucosal involvement
- **Bullous pemphigoid**: Tense bullae, flexural, elderly, –Nikolsky sign
- **SJS / TEN**: Widespread blistering, mucosal erosions, drug-induced emergency
- **Erythema multiforme**: Target lesions, often post-infectious (HSV)

**Pustular**:

- **Folliculitis**: Hair follicle centered, ± Staphylococcus
- **Acne vulgaris**: Comedones + inflammatory papules/pustules
- **Pustular psoriasis**: Sterile pustules, ± generalized (von Zumbusch)
- **Impetigo**: Honey-colored crust, contagious, children

### 水腫性病灶

| 病灶           | 特徵                                  | 鑑別診斷                                                     |
| -------------- | ------------------------------------- | ------------------------------------------------------------ |
| **Wheal**      | Transient (<24h), edematous, pruritic | Urticaria (acute/chronic), physical urticaria, dermographism |
| **Angioedema** | Deep dermis/subcutis, non-pitting     | Allergic, ACEi-induced, hereditary angioedema                |

**Eczematous / Dermatitis** (also considered in this category):

- **Atopic dermatitis**: Flexural, early onset, personal/family atopy
- **Allergic contact dermatitis**: Delayed type IV hypersensitivity
- **Irritant contact dermatitis**: Direct chemical injury
- **Nummular eczema**: Coin-shaped plaques on extremities
- **Stasis dermatitis**: Lower legs, venous insufficiency, edema
- **Lichen simplex chronicus**: Localized lichenification from scratching

### 續發性病灶

| 類型                | 定義                                  | 鑑別診斷                                                        |
| ------------------- | ------------------------------------- | --------------------------------------------------------------- |
| **Scale**           | Flaky stratum corneum                 | Psoriasis, seborrheic dermatitis, ichthyosis, actinic keratosis |
| **Crust**           | Dried exudate (serum/pus/blood)       | Impetigo, eczema, herpes                                        |
| **Erosion**         | Partial epidermal loss                | Intertrigo, candidiasis, pemphigus                              |
| **Ulcer**           | Full epidermal ± dermal loss          | Venous/arterial/pressure/diabetic ulcer, malignancy             |
| **Fissure**         | Linear crack                          | Athlete's foot, hand eczema, angular cheilitis                  |
| **Lichenification** | Thickened skin + accentuated markings | Chronic atopic dermatitis                                       |

## 輔助工具速查

| Tool                     | Use                                                 | Positive Finding                                                            |
| ------------------------ | --------------------------------------------------- | --------------------------------------------------------------------------- |
| **Dermoscopy**           | Differentiate benign vs malignant pigmented lesions | Pigment network, blue-white veil, regression structures                     |
| **KOH prep (10-20%)**    | Fungal infection                                    | Hyphae (dermatophytes), pseudohyphae (Candida), spores (Pityriasis)         |
| **Wood's lamp (365 nm)** | Bacterial/fungal infection, pigmentary disorders    | Coral red (erythrasma), yellow-green (tinea capitis), blue-white (vitiligo) |
| **Skin biopsy**          | Gold standard for uncertain diagnosis               | H&E histopathology, immunofluorescence (lupus, pemphigus)                   |
| **Patch test**           | Allergic contact dermatitis                         | Positive reaction at 48-96h                                                 |
| **Tzanck smear**         | Herpes virus infection                              | Multinucleated giant cells                                                  |
