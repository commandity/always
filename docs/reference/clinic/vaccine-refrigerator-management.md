# 疫苗專用冰箱管理原則

**Vaccine Refrigerator Management Guidelines**

> **【疫苗事用冰箱管理原則】**
>
> 1. 需指定專人負責管理疫苗，每次使用後均確認冰箱門需緊閉。
> 2. 使用高低溫度計，維持冰箱內溫度冷藏需在攝氏2-8°C之間，洽凍需在攝氏-10°C以下。
> 3. 每天早、晚定時查核並確實記錄述箱溫度。
> 4. 使用疫苗原則一先進先出（將新進疫苗放置於冰箱右邊或後面，使用時需由左邊或前面開始拿取）。
> 5. 各種疫苗需填寫標示牌（含疫苗名稱、劑型、有效期限、批號、廠牌等）並排列整齊，且疫苗間應留空隙以利空氣流通，以免影響疫苗效價。廠牌除國光為中文顯示外其餘一律填寫英文。
> 6. 平時冷凍室應放置至少5個以上冰寶，並視冰箱冷藏室底層空間大小放置多個有蓋之廣口瓶或水寶，以備停電時提供適切保冷。
> 7. 勿將疫苗或高低溫度計放置於冰箱門，（因冰箱門散熱快，溫度不穩）。
> 8. 活化疫苗 ex：麻疹德國麻疹腮腺炎疫苗（MMR）、水痘疫苗（Var）等疫苗對高溫較敏感（比較怕熱），故需放置於冰箱冷藏室內溫度最低之層
> 9. 不活化疫苗 ex：B 型肝炎疫苗（HBV）、B 型肝炎免疫球蛋白疫苗（HBIG）、流感疫苗（FIu）、肺炎疫苗（PPV） 等疫苗，對低溫較敏感（比較怕冷），故需放置於冰箱冷藏室內溫度較高之層。
> 10. 疫苗貯存應有專用冰箱，不可將食物、飲料或疫苗以外之藥物、檢體等與疫苗一起存放。
> 11. 無發電機設備或不斷電系統蓄電量小於3小時之合約醫療院所，若遇台電通知停電時間超過3小時，則請聯繫衛生所主辦人，並將疫苗途至衛生所存放，以維疫苗品質。
> 12. 疫苗冰箱若發生溫度異常，無法立即處理，請立即與轄區衛生所預注主辦人連絡，並將疫苗送至衛生所存放，以維疫苗品質。
> 13. 冰箱要離牆壁或冰箱與冰箱間距至少10公分，以利熱空氣散熱。
> 14. 冰箱使用之電源插座為專用插座，冰箱之插頭應用適當的謬布妥善固定於插座上，以防脫落。

## 1. 溫度管理 Temperature Management

| 項目         | 要求                                   |
| ------------ | -------------------------------------- |
| **冷藏溫度** | 攝氏 2–8°C（35.5–46.4°F）              |
| **冷凍溫度** | 攝氏 -10°C 以下                        |
| **記錄頻率** | 每天早、晚各查核一次，確實記錄冰箱溫度 |
| **設備**     | 使用經認證之高低溫度計                 |

## 2. 疫苗存放原則 Storage Principles

### 2.1 專用冰箱 Dedicated Refrigerator

- 疫苗貯存應有**專用冰箱**，**不可**將食物、飲料或疫苗以外之藥物、檢體等與疫苗一起存放。

### 2.2 先進先出 First-In-First-Out (FIFO)

- 新進疫苗放置於冰箱**右邊或後面**
- 使用時由**左邊或前面**開始拿取

### 2.3 標示與排列 Labeling & Arrangement

| 項目           | 要求                                 |
| -------------- | ------------------------------------ |
| **標示牌內容** | 疫苗名稱、劑型、有效期限、批號、廠牌 |
| **廠牌語言**   | 國光為中文顯示，其餘一律填寫英文     |
| **排列方式**   | 排列整齊，疫苗間應留空隙以利空氣流通 |
| **目的**       | 避免影響疫苗效價                     |

### 2.4 溫度分層存放 Temperature Zoning

| 疫苗類型       | 存放位置                         | 說明                 | 範例                                                                |
| -------------- | -------------------------------- | -------------------- | ------------------------------------------------------------------- |
| **活化疫苗**   | 冰箱冷藏室**最底層**（溫度最低） | 對高溫較敏感（怕熱） | MMR（麻疹德國麻疹腮腺炎）、Var（水痘）                              |
| **不活化疫苗** | 冰箱冷藏室**較上層**（溫度較高） | 對低溫較敏感（怕冷） | HBV（B型肝炎）、HBIG（B型肝炎免疫球蛋白）、Flu（流感）、PPV（肺炎） |

## 3. 冰箱環境要求 Environmental Requirements

| 項目           | 要求                                                             |
| -------------- | ---------------------------------------------------------------- |
| **離牆距離**   | 冰箱需離牆壁至少 **10 公分**，以利熱空氣散熱                     |
| **冰箱間距**   | 冰箱與冰箱間距至少 **10 公分**                                   |
| **專用插座**   | 冰箱使用之電源插座為**專用插座**                                 |
| **插頭固定**   | 冰箱之插頭應用適當之膠布妥善固定於插座上，以防脫落               |
| **溫度計位置** | **勿**將疫苗或高低溫度計放置於冰箱門（因冰箱門散熱快，溫度不穩） |

## 4. 停電應變措施 Power Outage Response

### 4.1 冷凍室保冷準備

| 項目            | 要求                                               |
| --------------- | -------------------------------------------------- |
| **冰寶**        | 冷凍室應放置至少 **5 個以上**冰寶                  |
| **廣口瓶/水寶** | 視冰箱冷藏室底層空間大小放置多個有蓋之廣口瓶或水寶 |
| **目的**        | 備停電時提供適切保冷                               |

### 4.2 停電超過 3 小時

| 條件                                      | 處理方式                                 |
| ----------------------------------------- | ---------------------------------------- |
| 無發電機設備 或 不斷電系統蓄電量 < 3 小時 | 若遇台電通知停電時間超過 3 小時          |
| **處理方式**                              | 聯繫衛生所主辦人，並將疫苗送至衛生所存放 |
| **目的**                                  | 維護疫苗品質                             |

## 5. 溫度異常處理 Temperature Abomaly Response

- 疫苗冰箱若發生溫度異常，**無法立即處理**時：
  1. **立即**與轄區衛生所預注主辦人連絡
  2. 將疫苗送至衛生所存放
  3. 維護疫苗品質

## 6. 管理責任 Management Responsibility

- 需指定**專人負責**管理疫苗
- 每次使用後均確認冰箱門需**緊閉**
