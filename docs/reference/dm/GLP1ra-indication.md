# GLP-1 受體促效劑適應症 **GLP-1 RA Indications**

## 概述

GLP-1 RA 透過活化 GLP-1 受體，促進胰島素分泌、抑制升糖素、延緩胃排空、抑制食慾。適應症已從 T2D 擴展至體重管理、心血管保護、腎臟保護、MASH 及 OSA。

- 2025 年 WHO 指引建議 GLP-1 RA 合併行為介入用於非妊娠成人肥胖症長期治療
- 所有 GLP-1 RA 均建議作為飲食與運動的輔助治療

## 第 2 型糖尿病

| 藥物                                  | 核准年份 | 適用年齡 | 劑量                                          |
| ------------------------------------- | -------- | -------- | --------------------------------------------- |
| **Exenatide**（Byetta / Bydureon）    | 2005     | 成人     | 5 → 10 mcg SC 每日兩次；Bydureon 2 mg SC 每週 |
| **Liraglutide**（Victoza）            | 2010     | ≥ 10 歲  | 0.6 → 1.2 → 最高 1.8 mg/日 SC                 |
| **Dulaglutide**（Trulicity 易週糖）   | 2014     | ≥ 10 歲  | 0.75 → 1.5 → 最高 4.5 mg/週 SC                |
| **Semaglutide SC**（Ozempic 胰妥讚）  | 2017     | 成人     | 0.25 → 0.5 → 1 → 最高 2 mg/週 SC              |
| **Semaglutide PO**（Rybelsus 瑞倍適） | 2019     | 成人     | R1: 3 → 7 → 14 mg/日；R2: 1.5 → 4 → 9 mg/日   |
| **Tirzepatide**（Mounjaro 猛健樂）    | 2022     | 成人     | 2.5 → 5 → 最高 15 mg/週 SC                    |

> Exenatide（Byetta/Bydureon）已於 2024 年停產。Rybelsus R1/R2 配方不可依毫克互換。

## 體重管理

| 藥物                              | 核准年份 | 適用對象                                         | 劑量                                  |
| --------------------------------- | -------- | ------------------------------------------------ | ------------------------------------- |
| **Liraglutide**（Saxenda 善纖達） | 2014     | ≥ 12 歲（體重 > 60 kg）；成人（BMI ≥ 27 + 共病） | 0.6 → 3.0 mg/日 SC（每週遞增 0.6 mg） |
| **Semaglutide**（Wegovy 週纖達）  | 2021     | ≥ 12 歲；成人（BMI ≥ 27 + 共病）                 | 0.25 → 0.5 → 1 → 1.7 → 2.4 mg/週 SC   |
| **Tirzepatide**（Zepbound）       | 2023     | 成人（BMI ≥ 27 + 共病）                          | 2.5 → 5 → 10 → 15 mg/週 SC            |

> Saxenda 無法耐受 3 mg 的兒科患者可用 2.4 mg。Wegovy 無法耐受者可以 1.7 mg 維持。Teva 已上市 liraglutide 減重學名藥。

## 心血管風險降低

| 藥物                                  | 核准年份 | 適應症                            | 劑量                     |
| ------------------------------------- | -------- | --------------------------------- | ------------------------ |
| **Liraglutide**（Victoza）            | 2017     | T2D + 已確診 CVD                  | 0.6 → 1.2 → 1.8 mg/日 SC |
| **Dulaglutide**（Trulicity）          | 2020     | T2D + CVD 或多項危險因子          | 0.75 → 1.5 mg/週 SC      |
| **Semaglutide SC**（Ozempic 胰妥讚）  | 2020     | T2D + 已確診 CVD                  | 0.25 → 0.5 → 1 mg/週 SC  |
| **Semaglutide SC**（Wegovy 週纖達）   | 2024     | 肥胖/體重過重 + 已確診 CVD        | 2.4 mg/週 SC             |
| **Semaglutide PO**（Rybelsus 瑞倍適） | 2025     | T2D + 高 CVD 風險（無已確診 CVD） | R1: 3 → 7 → 14 mg/日     |

> Wegovy 為首個專為肥胖/體重過重成人降低 MACE 核准的藥物。Rybelsus 為首個獲批 CV 適應症的口服 GLP-1 RA（SOUL 試驗）。

## 注意事項

::: warning 仿製 GLP-1 RA 安全疑慮
FDA 對未經核准的仿製（compounded）GLP-1 RA 表達嚴重關切，這類產品未經安全性、有效性及品質審查。臨床醫師應確保患者使用經合法核准的正規藥品。
:::

::: warning Rybelsus 兩種配方不可互換
口服 semaglutide（Rybelsus 瑞倍適）有 R1 和 R2 兩種配方，因賦形劑不同導致生物利用率差異，**不可依毫克數互換使用**。心血管適應症核准基於 R1 配方（SOUL 試驗）。
:::

::: info 臺灣上市現況
臺灣 TFDA 已核准：胰妥讚（Ozempic）、瑞倍適（Rybelsus）、善纖達（Saxenda）、易週糖（Trulicity）、猛健樂（Mounjaro，2025 上市）。Wegovy、Zepbound 臺灣未核准。各藥物臺灣核准適應症可能與美國 FDA 有差異，請參閱 TFDA 最新核准資訊及健保給付規定。
:::
