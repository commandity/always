# GLP-1 RA 選擇指引 **How to Choose the Right GLP-1 Agonist**

## 概述

GLP-1 RA 透過活化 GLP-1 受體，促進胰島素分泌、抑制升糖素、延緩胃排空、抑制食慾。適應症已從 T2D 擴展至體重管理、心血管保護、腎臟保護、MASH 及 OSA。

- **GLP-1**：腸泌素荷爾蒙，高血糖下刺激胰島素分泌，抑制食慾、減少攝食
- **GIP**：另一腸泌素，與 GLP-1 協同調節血糖與體重
- **T2D 患者**腸泌素效應減弱，藥理性刺激 GLP-1 可顯著改善血糖控制
- 2025 年 WHO 指引建議 GLP-1 RA 合併行為介入用於非妊娠成人肥胖症長期治療

## 注意事項

::: warning 適應症外使用
並非所有 GLP-1 RA 均核准用於體重管理。臺灣可用藥物中，僅 Saxenda 善纖達及 Mounjaro 猛健樂核准減重適應症。Ozempic 減重為 off-label，應告知患者。
:::

::: warning 仿製 GLP-1 RA 安全疑慮
FDA 對未經核准的仿製（compounded）GLP-1 RA 表達嚴重關切，這類產品未經安全性、有效性及品質審查。臨床醫師應確保患者使用經合法核准的正規藥品。
:::

::: warning Rybelsus 兩種配方不可互換
口服 semaglutide（Rybelsus 瑞倍適）有 R1 和 R2 兩種配方，因賦形劑不同導致生物利用率差異，**不可依毫克數互換使用**。
:::

::: info 臺灣上市現況
臺灣 TFDA 已核准：胰妥讚（Ozempic）、瑞倍適（Rybelsus）、善纖達（Saxenda）、易週糖（Trulicity）、猛健樂（Mounjaro，2025 上市）。Wegovy、Zepbound 臺灣未核准。各藥物臺灣核准適應症可能與美國 FDA 有差異，請參閱 TFDA 最新核准資訊及健保給付規定。
:::
