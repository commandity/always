# 停用 GLP-1 RA 後的體重維持

**Weight Maintenance After GLP-1 RA Cessation**

## 概述

AACE 最新指引以「併發症為中心」照護模式，GLP-1 RA 為肥胖症首選藥物。然而多項研究證實，停藥後體重回升及心臟代謝改善逆轉幾乎不可避免，目前仍缺乏停藥後維持體重的充分實證指引。

## 停藥後體重回升：臨床試驗證據

| 試驗                                                                       | 藥物                      | 停藥後變化                                                                     |
| -------------------------------------------------------------------------- | ------------------------- | ------------------------------------------------------------------------------ |
| **STEP 1 延伸**（Wilding et al., 2022）Semaglutide 2.4 mg/週 × 68 週後停藥 | Semaglutide               | 回升已減體重之 2/3；淨變化 −5.6%（vs 安慰劑 −0.1%）                            |
| **SURMOUNT-4**（Aronne et al., JAMA 2024）Tirzepatide 36 週後停藥轉安慰劑  | Tirzepatide               | 回升已減體重之 14%                                                             |
| **SRMA**（Berg et al., Obesity Rev 2025）8 項 RCT，n=2372，BMI ≥27         | Liraglutide               | +2.20 kg（95% CI 1.69–2.70, P<0.00001）                                        |
| **SRMA**（Berg et al., Obesity Rev 2025）8 項 RCT，n=2372，BMI ≥27         | Semaglutide / Tirzepatide | +9.69 kg（95% CI 5.78–13.60, P<0.00001）                                       |
| **NMR**（Budini et al., 2026）6 項 RCT，n=3236，追蹤至停藥後 104 週        | GLP-1 RA（整體）          | 1 年回升已減體重之 60%；估計平台期回升 75.3%（95% CI 68.9–81.6），半衰期 23 週 |

> SRMA = Systematic Review and Meta-Analysis；NMR = Nonlinear Meta-Regression；RCT = Randomized Controlled Trial

## 生物機制：為何體重必然回升

AACE 2025 指引（Nadolsky 等人）指出**脂肪量受生物機制保護**。停藥後：

- 食慾與飽足感調節效應消失
- 代謝適應加劇：飢餓感↑、靜止代謝率↓、總能量消耗↓
- 回升體重以**脂肪量**為主，進一步削弱代謝健康

> 肥胖症本身為慢性、進行性、易復發疾病，GLP-1 RA 應視為長期治療藥物。

## 停藥後維持策略：現有證據

| 策略                                  | 研究                                                                                                        | 結果                                                                                                               |
| ------------------------------------- | ----------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| **低劑量 semaglutide + 生活型態強化** | TAILGATE 真實世界研究（Gudbergsen et al., ECO 2024）n=353 開始減量，240 完成，85 有停藥後 26 週數據         | 停藥後 26 週平均體重僅增加 1.5%；46 人重新啟用 semaglutide（平均回升 1.3%）                                        |
| **監督式運動 + liraglutide**          | S-LiTE 後續分析（Jensen et al., EClinicalMedicine 2024）RCT，n=109 追蹤至停藥後 1 年（week 104），BMI 32–43 | 合併運動+liraglutide 停藥後體重 −5.1 kg（95% CI −10.0 至 −0.2, P=0.040），體脂 −2.3%（P=0.026）vs 單用 liraglutide |
| **運動計畫**                          | S-LiTE 附屬研究（Holt et al., Obesity 2025）n=195，52 週運動 vs 常規活動                                    | 運動組晚期餐後 GLP-1 反應↑37%（組內），較常規活動組↑25%（95% CI 3–51%, P=0.02）；liraglutide 組無此效果            |
| **轉換口服抗肥胖藥**                  | 真實世界前瞻性研究（Paddu et al., Obesity 2024）n=40 轉換，基線 BMI 36.4，GLP-1 RA 12 個月後                | metformin 80%、topiramate 32.5%、bupropion 32.5%；12 個月時平均減重 18.3%，24 個月維持減重 25.5%                   |
| **劑量降階**（每週→每兩週）           | 回顧性圖表審查（Biermann et al., ObesityWeek 2025）n=31，semaglutide 或 tirzepatide 轉每兩週                | 32 週追蹤：額外減重 1.3 kg（−2%, P<0.05）；代謝症候群盛行率維持 39%（vs 治療前 79%）；5 人因體重回升改回每週       |

## 實務建議

| 項目             | 建議                                                         |
| ---------------- | ------------------------------------------------------------ |
| **停藥方式**     | 逐步減量（gradual tapering）優於突然停藥                     |
| **同步介入**     | 強化飲食控制、運動計畫、行為支持、頻繁體重監測               |
| **口服藥物過渡** | 可考慮 metformin、topiramate、bupropion（尚無充分實證）      |
| **蛋白質攝取**   | 0.8–1.6 g/kg/day 或 80–120 g/day（專家共識，尚無前瞻性試驗） |
| **阻力訓練**     | 所有患者建議合併，老年患者尤為重要                           |

## 注意事項

::: warning 停藥後體重回升風險高
STEP 1 延伸試驗、SURMOUNT-4 一致顯示停用 GLP-1 RA 後體重回升幾乎不可避免。治療前應向患者充分說明，定位為長期治療藥物。
:::

::: warning 肌肉流失與蛋白質攝取
GLP-1 RA 治療期間體重減輕含相當比例肌肉流失，老年患者尤顯著。建議合併阻力訓練及充足蛋白質攝取（0.8–1.6 g/kg/day），惟此建議來自專家共識，尚缺乏前瞻性隨機試驗支持。
:::

::: info 逐步減量優於突然停藥
現有有限證據傾向支持逐步減量並同步強化生活型態介入。劑量降階（延長注射間隔）作為維持策略數據尚屬初步，需更大規模試驗確認。
:::
