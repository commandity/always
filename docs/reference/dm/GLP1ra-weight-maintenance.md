# 停用 GLP-1 RA 後的體重維持

**Weight Maintenance After GLP-1 RA Cessation**

## 概述

AACE 最新指引以「併發症為中心」照護模式，GLP-1 RA 為肥胖症首選藥物。然而多項研究證實，停藥後體重回升及心臟代謝改善逆轉幾乎不可避免，目前仍缺乏停藥後維持體重的充分實證指引。

## 停藥後體重回升：臨床試驗證據

| 試驗                                                                               | 藥物                      | 停藥後變化                                                                     |
| ---------------------------------------------------------------------------------- | ------------------------- | ------------------------------------------------------------------------------ |
| **STEP 1 延伸**（Wilding et al., 2022）Semaglutide 2.4 mg/週 × 68 週後停藥         | Semaglutide               | 回升已減體重之 2/3；淨變化 −5.6%（vs 安慰劑 −0.1%）                            |
| **SURMOUNT-4**（Aronne et al., JAMA 2024）Tirzepatide 36 週後停藥轉安慰劑          | Tirzepatide               | 回升已減體重之 14%                                                             |
| **系統性回顧與統合分析**（Berg et al., Obesity Rev 2025）8 項 RCT，n=2372，BMI ≥27 | Liraglutide               | +2.20 kg（95% CI 1.69–2.70, P<0.00001）                                        |
| **系統性回顧與統合分析**（Berg et al., Obesity Rev 2025）8 項 RCT，n=2372，BMI ≥27 | Semaglutide / Tirzepatide | +9.69 kg（95% CI 5.78–13.60, P<0.00001）                                       |
| **非線性統合回歸**（Budini et al., 2026）6 項 RCT，n=3236，追蹤至停藥後 104 週     | GLP-1 RA（整體）          | 1 年回升已減體重之 60%；估計平台期回升 75.3%（95% CI 68.9–81.6），半衰期 23 週 |

## 生物機制：為何體重必然回升

AACE 2025 指引（Nadolsky 等人）指出**脂肪量受生物機制保護**。停藥後：

- 食慾與飽足感調節效應消失
- 代謝適應加劇：飢餓感↑、靜止代謝率↓、總能量消耗↓
- 回升體重以**脂肪量**為主，進一步削弱代謝健康

> 肥胖症本身為慢性、進行性、易復發疾病，GLP-1 RA 應視為長期治療藥物。

## 停藥後維持策略：現有證據

| 策略                                  | 研究                                     | 結果                                                                           |
| ------------------------------------- | ---------------------------------------- | ------------------------------------------------------------------------------ |
| **低劑量 semaglutide + 生活型態強化** | 2024 歐洲肥胖大會（n=353，240 完成減量） | 停藥後 26 週平均體重僅增加 1.5%                                                |
| **監督式運動 + liraglutide**          | Jensen 等人                              | 停藥 1 年後回升明顯少於單用 liraglutide（−5.1 kg, P=0.040）                    |
| **運動計畫**                          | Holt 等人                                | 運動組晚期餐後 GLP-1 反應↑25%，有助防止食慾增加                                |
| **轉換口服抗肥胖藥**                  | Paddu 等人                               | GLP-1 RA 12 個月後改用 metformin / topiramate / bupropion，維持體重減輕達 2 年 |
| **劑量降階**（每週→每兩週）           | Obesity Week 2025（小型研究）            | 初步顯示有助維持體重及代謝改善，待大規模驗證                                   |

## 實務建議

| 項目             | 建議                                                         |
| ---------------- | ------------------------------------------------------------ |
| **停藥方式**     | 逐步減量（gradual tapering）優於突然停藥                     |
| **同步介入**     | 強化飲食控制、運動計畫、行為支持、頻繁體重監測               |
| **口服藥物過渡** | 可考慮 metformin、topiramate、bupropion（尚無充分實證）      |
| **蛋白質攝取**   | 0.8–1.6 g/kg/day 或 80–120 g/day（專家共識，尚無前瞻性試驗） |
| **阻力訓練**     | 所有患者建議合併，老年患者尤為重要                           |

## 注意事項

::: warning 停藥後體重回升風險高
STEP 1 延伸試驗、SURMOUNT-4 一致顯示停用 GLP-1 RA 後體重回升幾乎不可避免。治療前應向患者充分說明，定位為長期治療藥物。
:::

::: warning 肌肉流失與蛋白質攝取
GLP-1 RA 治療期間體重減輕含相當比例肌肉流失，老年患者尤顯著。建議合併阻力訓練及充足蛋白質攝取（0.8–1.6 g/kg/day），惟此建議來自專家共識，尚缺乏前瞻性隨機試驗支持。
:::

::: info 逐步減量優於突然停藥
現有有限證據傾向支持逐步減量並同步強化生活型態介入。劑量降階（延長注射間隔）作為維持策略數據尚屬初步，需更大規模試驗確認。
:::
