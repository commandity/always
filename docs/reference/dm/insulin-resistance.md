# 胰島素阻抗的臨床表現與皮膚表徵

**Insulin Resistance: Clinical & Cutaneous Manifestations**

---

胰島素阻抗（insulin resistance, IR）是指細胞對胰島素的反應降低，身體需要分泌更多胰島素才能維持正常血糖的代償狀態。在胰島素阻抗的早期階段，空腹血糖與糖化血色素可能仍屬正常，因此臨床辨識依賴對身體表徵、生物標記及代謝症候群元件的綜合判斷。

早期辨識胰島素阻抗至關重要——在 β 細胞功能尚未衰竭前介入生活型態調整，可有效預防或延緩第 2 型糖尿病及心血管疾病的發生。

## 代謝症候群：胰島素阻抗的臨床框架

胰島素阻抗是代謝症候群的核心病理機制。根據我國衛生福利部國民健康署及國際共識，代謝症候群的診斷需符合以下五項中至少三項：

| 指標                        | 異常切點                             |
| --------------------------- | ------------------------------------ |
| 腹部肥胖（腰圍）            | 男性 ≥ 90 cm、女性 ≥ 80 cm           |
| 空腹血糖                    | ≥ 100 mg/dL 或已使用降血糖藥物       |
| 血壓                        | 收縮壓 ≥ 130 mmHg 或舒張壓 ≥ 85 mmHg |
| 三酸甘油酯（TG）            | ≥ 150 mg/dL                          |
| 高密度脂蛋白膽固醇（HDL-C） | 男性 < 40 mg/dL、女性 < 50 mg/dL     |

### TG/HDL 比值：簡便有效的胰島素阻抗指標

TG/HDL 比值源自標準空腹血脂報告，是臨床上最經濟的胰島素阻抗替代標記。胰島素阻抗時肝臟 VLDL 合成過盛、TG 清除減速，同時 HDL 代謝加速、顆粒縮小。

**判讀切點（mg/dL 為單位）：**

| TG/HDL 比值 | 臨床意義       |
| ----------- | -------------- |
| < 2.0       | 理想           |
| 2.0–3.5     | 疑似胰島素阻抗 |
| > 3.5       | 顯著胰島素阻抗 |

- 性別差異：女性切點約低 10–15%（因基礎 HDL 較高）
- 種族差異：非裔族群 baseline TG 較低，切點約 2.0–3.0 即有意義；東亞族群 2.5–3.0 即建議進一步評估
- TG/HDL 為間接指標，確定診斷需搭配空腹胰島素 + 血糖計算 HOMA-IR

### TyG 指數（Triglyceride-Glucose Index）

TyG 指數為另一項簡便指標，敏感度與特異度均優於 TG/HDL：

TyG = ln\[TG (mg/dL) × 空腹血糖 (mg/dL) ÷ 2\]

- **TyG ≥ 8.65**：提示胰島素阻抗
- **TyG 8.0–8.64**：邊緣，需搭配臨床表徵綜合判斷
- 文獻回顧顯示其診斷胰島素阻抗的 AUC 約 0.82–0.88，優於 TG/HDL（AUC ~ 0.75）

### 空腹胰島素（Fasting Insulin）

單獨的空腹胰島素值對臨床判斷亦極有價值：

- **> 15 μU/mL**：明顯異常，高度提示胰島素阻抗
- **10–15 μU/mL**：臨界值，需配合其他指標
- **< 10 μU/mL**：理想

值得注意的是，空腹胰島素受檢驗方法影響，不同實驗室之間變異大，建議在同一實驗室穩定追蹤。

## 皮膚表徵：胰島素阻抗的視覺線索

胰島素對皮膚有多重生理作用，包括促進角質細胞與纖維母細胞增生。當循環中胰島素濃度升高（高胰島素血症），這些作用被過度活化，產生了可供臨床辨識的皮膚變化。許多患者在進展至糖尿病前，皮膚已先呈現了胰島素阻抗的訊號。

### 黑色棘皮症（Acanthosis Nigricans）

黑色棘皮症是胰島素阻抗最具特異性的皮膚表徵。

- **外觀**：對稱性、天鵝絨狀（velvety）、色素沉著的斑塊
- **分佈位置**：頸部（最常見）、腋下、腹股溝、指關節背面、肘部、膝部
- **觸感**：皮膚增厚，觸摸時有粗糙、乳頭狀突起感
- **病理機制**：高濃度胰島素結合 IGF-1 受體，刺激角質細胞與纖維母細胞增生

![Acanthosis Nigricans](/reference/skin/skin-manifestations/Acanthosis-nigricans.jpg)
_Acanthosis Nigricans — 頸後灰棕色天鵝絨樣斑塊_

臨床注意事項：

- 黑色棘皮症的嚴重程度（範圍及厚度）通常與胰島素阻抗的嚴重度相關
- 在兒童及青少年族群中，此表徵尤其重要——兒童肥胖合併黑色棘皮症幾乎可認定存在顯著的胰島素阻抗
- 未必與肥胖同時存在；體型苗條但出現黑色棘皮症者，仍應評估胰島素阻抗及相關內分泌疾病
- 部分藥物（如全身性皮質類固醇、口服避孕藥、菸鹼酸）可能誘發類似的皮膚變化，鑑別診斷時須一併考量

### 皮膚贅瘤／軟纖維瘤（Skin Tags, Acrochordons）

- 多發性、膚色或褐色、有蒂的軟性小突起
- 好發於頸部、腋下、腹股溝等皮膚皺摺處
- 與胰島素阻抗的關聯強烈：多項研究顯示，多發性皮膚贅瘤（> 5–10 顆）患者出現胰島素阻抗的風險顯著升高
- 數量與 HOMA-IR 呈正相關
- 可能與 IGF-1 受體過度活化及胰島素的促增生作用有關

![Acrochordons](/reference/skin/skin-manifestations/Acrochordons.jpg)
_Acrochordons（Skin Tags）— 頸部軟性有柄贅生物_

### 毛囊角化症（Keratosis Pilaris）

- 毛囊口的角質過度堆積，形成多發性、細小、膚色或淡紅色的毛囊性丘疹
- 好發於上臂外側、大腿前側、臀部，有時也見於面頰部
- **病理意義**：胰島素阻抗可能影響角質細胞的正常分化與脫屑，加劇毛囊口的角質堆積
- 雖在一般族群中亦相當常見，但若合併其他胰島素阻抗表徵（如黑色棘皮症、肥胖），應提高警覺
- 治療以保濕及角質溶解劑為主，但根本改善有賴胰島素敏感度的提升

![Keratosis Pilaris](/reference/skin/skin-manifestations/Keratosis-pilaris.jpg)
_Keratosis Pilaris — 上臂後側毛囊角化丘疹_

### 其他相關皮膚表現

| 皮膚變化                                    | 說明                                                                       |
| ------------------------------------------- | -------------------------------------------------------------------------- |
| **多毛症（Hirsutism）**                     | 女性在雄激素依賴區域出現毛髮過度生長，反映胰島素阻抗刺激卵巢產生過多雄激素 |
| **痤瘡（Acne）**                            | 胰島素阻抗透過 IGF-1 及 mTOR 路徑促進皮脂分泌與角質堆積                    |
| **黑色表皮（Dirty Neck 外觀）**             | 清潔後仍無法去除的灰褐色污垢樣外觀，實為黑色棘皮症的一種表現形式           |
| **脛前色素斑（Pretibial Pigmented Spots）** | 糖尿病前期即可見的小腿前側萎縮性色素斑，與微血管病變有關                   |

## 生殖內分泌表現

### 多囊性卵巢症候群（PCOS）

胰島素阻抗在 PCOS 的病理生理中扮演關鍵角色：

- 約 50–70% 的 PCOS 患者存在胰島素阻抗
- 高胰島素血症直接刺激卵巢的 theca 細胞產生過多雄激素
- 同時抑制肝臟的性荷爾蒙結合球蛋白（SHBG）合成，使游離睪固酮濃度進一步升高
- 臨床表現：月經稀發或無月經、不孕、多毛症、痤瘡、雄性禿
- 不論是否肥胖，PCOS 患者均應接受胰島素阻抗評估

## 臨床綜合評估流程

### 高危險族群篩檢

以下族群應考慮進行胰島素阻抗評估：

- 過重或肥胖（BMI ≥ 24 kg/m²）
- 腰圍超標（男性 ≥ 90 cm、女性 ≥ 80 cm）
- 一等親糖尿病家族史
- 高血壓、血脂異常、心血管疾病史
- 妊娠糖尿病史或生產巨嬰（出生體重 > 4000 g）
- 多囊性卵巢症候群
- 黑色棘皮症或其他相關皮膚表徵
- 非酒精性脂肪肝疾病（NAFLD / MASLD）
- 特定族群：亞洲人在較低 BMI 即出現胰島素阻抗，應特別留意

### 評估工具

| 工具                           | 公式                                                | 切點                                        | 備註                             |
| ------------------------------ | --------------------------------------------------- | ------------------------------------------- | -------------------------------- |
| **HOMA-IR**                    | 空腹胰島素 (μU/mL) × 空腹血糖 (mmol/L) ÷ 22.5       | ≥ 2.5（東亞族群建議 ≥ 2.0）                 | 黃金標準，需空腹胰島素檢驗       |
| **TyG 指數**                   | ln\[TG (mg/dL) × 空腹血糖 (mg/dL) ÷ 2\]             | ≥ 8.65                                      | AUC 0.82–0.88，優於 TG/HDL       |
| **TG/HDL 比值**                | TG (mg/dL) ÷ HDL (mg/dL)                            | < 2.0 理想／2.0–3.5 疑似／> 3.5 顯著 IR     | 最經濟，適合基層篩檢             |
| **空腹胰島素**                 | 單一檢驗值                                          | > 15 μU/mL 異常／10–15 臨界                 | 實驗室間變異大，宜同院追蹤       |
| **QUICKI**                     | 1 ÷ [log(空腹胰島素) + log(空腹血糖)]               | ≤ 0.33                                      | 適用於胰島素極端值               |
| **McAuley Index**              | exp\[2.63 − 0.28 × ln(空腹胰島素) − 0.31 × ln(TG)\] | < 5.8                                       | 較少臨床使用                     |
| **口服葡萄糖耐受試驗（OGTT）** | 75 g 葡萄糖負荷                                     | 2h ≥ 140 mg/dL 葡萄糖耐受不良／≥ 200 糖尿病 | 同時評估 glucose 及 insulin 反應 |

### 臨床處置要點

一旦辨識出胰島素阻抗，應立即啟動生活型態介入：

1. **體重減輕**：減輕 5–10% 體重即可顯著改善胰島素敏感度
2. **飲食調整**：減少精緻碳水化合物及添加糖，增加膳食纖維（每日 ≥ 25 g）
3. **身體活動**：每週至少 150 分鐘中等強度有氧運動，合併阻力訓練效果更佳
4. **藥物治療**：
   - Metformin 為首選藥物，可改善肝臟及周邊組織胰島素敏感度
   - 若合併糖尿病或減重需求，可考慮 GLP-1 RA 或 SGLT2i
   - PCOS 合併胰島素阻抗者，metformin 可改善排卵功能及代謝指標

## 注意事項

::: warning 胰島素阻抗的異質性
胰島素阻抗的表現具有個體差異——並非所有患者都會出現典型的皮膚表徵或血脂異常。缺乏黑色棘皮症或 TG/HDL 升高，不代表可以排除胰島素阻抗的存在。臨床判斷應綜合家族史、體型、代謝指標及身體檢查發現，而非依賴單一標記。
:::

::: warning 皮膚表徵的鑑別診斷
黑色棘皮症雖高度提示胰島素阻抗，但仍需排除其他可能原因，包括：惡性腫瘤（尤其是胃癌引起的惡性黑色棘皮症）、內分泌疾病（庫欣氏症候群、肢端肥大症）、藥物誘發（皮質類固醇、菸鹼酸、胰島素增敏劑）、自體免疫疾病及遺傳性因素。若成人患者短期內出現範圍廣泛且進展迅速的黑色棘皮症，應警覺潛在的內臟惡性腫瘤。
:::

::: info 追蹤與監測
已辨識出胰島素阻抗的患者應定期追蹤：每 3–6 個月測量空腹血糖及糖化血色素（HbA1c），每年評估血脂及肝腎功能。若合併 PCOS，另需追蹤月經週期及排卵狀況。體重及腰圍變化是評估介入成效最簡便的臨床指標。
:::

::: info 免責聲明
本文僅供醫療專業人員教育參考，**不構成個別患者的治療建議**。胰島素阻抗的診斷與處置應依個別患者狀況及最新臨床指引決定。
:::
