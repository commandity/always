# Abdominal Ultrasound Anatomy

**腹部超音波解剖學**

## 1. Liver

### Normal Anatomy & Segmental Division

| Lobe           | Segments                                                                                     |
| -------------- | -------------------------------------------------------------------------------------------- |
| **Caudate**    | Segment 1 (independent portal triads + direct venous drainage to IVC)                        |
| **Left lobe**  | 2 (lateral superior), 3 (lateral inferior), 4a (medial superior), 4b (medial inferior)       |
| **Right lobe** | 5 (anterior inferior), 6 (posterior inferior), 7 (posterior superior), 8 (anterior superior) |

**Key landmark**: Plane through GB fossa + middle hepatic vein divides right and left lobes.

### Normal US Measurements & Flow

| Parameter                          | Normal Value                      |
| ---------------------------------- | --------------------------------- |
| Portal vein diameter               | < 13 mm                           |
| Portal vein velocity               | 13–55 cm/s (hepatopetal)          |
| Hepatic artery PSV (proper)        | 40–80 cm/s (< 100 cm/s)           |
| Hepatic artery RI                  | 0.5–0.8 (increases post-prandial) |
| CBD proximal (inner-to-inner wall) | < 5 mm                            |

### Blood Supply (Unique Dual Afferent)

| Vessel             | Contribution | Key Feature                                                                                  |
| ------------------ | ------------ | -------------------------------------------------------------------------------------------- |
| **Portal vein**    | 75–80%       | Carries nutrients from gut + hepatotrophic hormones (40% more O₂ than systemic venous blood) |
| **Hepatic artery** | 20–25%       | Biliary tree more dependent on arterial supply than liver                                    |

### Vascular Anatomy Pearls – Doppler

| Vessel                     | Waveform                                                   | Key Feature                                                                            |
| -------------------------- | ---------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| **Portal vein**            | Undulating (varies with cardiac/respiratory activity)      | Thick reflective walls; runs in transverse plane                                       |
| **Hepatic veins**          | Triphasic (A: atrial contraction, S: systole, D: diastole) | No reflective walls (large sinusoids); trace to IVC                                    |
| **Hepatic artery**         | Low-resistance, continuous forward diastole                | Classic anatomy in 75%; replaced RHA from SMA (11%); replaced LHA from L gastric (10%) |
| **Absent flow / reversal** | Portal vein                                                | Suggests portal hypertension                                                           |

### US Echogenicity Pattern

| Structure               | Normal Echogenicity                                                    |
| ----------------------- | ---------------------------------------------------------------------- |
| Liver capsule (Glisson) | Echogenic, well-defined borders                                        |
| Liver parenchyma        | Mid-level, slightly hyperechoic vs. renal cortex; isoechoic vs. spleen |
| Portal triad walls      | Thicker, more reflective than hepatic veins (fibromuscular walls)      |
| Hepatic veins           | Echolucent, no reflective wall                                         |
| Intrahepatic bile ducts | Not visible peripherally (normal)                                      |

### Imaging Approach

| Lobe                          | Best Window                                                                                  |
| ----------------------------- | -------------------------------------------------------------------------------------------- |
| **Left lobe**                 | Subcostal, full inspiration                                                                  |
| **Right lobe**                | Intercostal (better resolution, less bowel gas); subcostal with cranial/rightward angulation |
| **Liver capsule/superficial** | High-frequency linear (7–12 MHz)                                                             |

### Clinical Pearls

- **First-line for elevated LFTs**: steatosis, cirrhosis, hepatitis, biliary dilatation all well seen
- **Shear wave elastography** → characterizes liver fibrosis with high accuracy
- **Metastases**: colon, pancreas, stomach → portal vein; breast, lung → hematogenous
- **HCC screening**: US q6 months in at-risk patients (cirrhosis, chronic HBV)

## 2. Biliary System

### Normal Anatomy & US Measurements

| Structure                | Normal Value                                          |
| ------------------------ | ----------------------------------------------------- |
| Gallbladder length       | 7–10 cm                                               |
| Gallbladder capacity     | ≤ 50 mL                                               |
| CBD/CHD diameter         | < 6–7 mm (no prior biliary disease)                   |
| Intrahepatic duct        | 1st–2nd order < 2 mm or < 40% of adjacent portal vein |
| CBD post-cholecystectomy | < 8–9 mm (1–2 mm increase expected)                   |
| CBD in elderly           | < 8.5 mm (loss of tissue elasticity)                  |

### Key Anatomic Points

- **CBD forms** from union of cystic duct + CHD in free edge of lesser omentum
- **CBD course**: Descends posterior/medial to duodenum, lies on dorsal pancreatic head → joins pancreatic duct → ampulla of Vater
- **Sphincters**: Boyden (distal CBD) + Oddi (hepatopancreatic segment) → prevent bile entry into duodenum, force storage in GB
- **Cystic duct**: 3–4 cm; spiral folds of Heister regulate bile flow
- **GB fundus**: Peritonealized, mobile; **body**: Contacts liver, duodenum, transverse colon; **neck**: Narrowed, joins CD

### Imaging Approach

- **Patient fasting**: ≥ 4 hrs (ideally 8–12 hrs overnight)
- **Best views**: Full suspended inspiration, left lateral oblique position
- **Harmonic imaging**: Improves contrast between ducts and tissues
- **Gallstone maneuvers**: Left lateral decubitus to demonstrate mobility; set focal zone at shadowing level

### Pitfalls

| Pitfall                    | Mimic                    | Solution                                     |
| -------------------------- | ------------------------ | -------------------------------------------- |
| GB neck folding            | Dilated CHD/CBD          | Scan in full inspiration; real-time tracking |
| Heister valves / bowel gas | Gallstones               | Reposition patient                           |
| Gastric antral contents    | Milk of calcium / stones | Check peristalsis; oral water                |
| Gas in adjacent duodenum   | Choledocholithiasis      | Decubitus views; water                       |
| Biliary gas                | Choledocholithiasis      | Look for reverberation artifact              |

### Clinical Pearls

- **Direct GB venous drainage** into liver (bypasses portal vein) → GB area spared from steatosis
- **GB carcinoma nodal mets** to peripancreatic nodes may mimic primary pancreatic tumor
- **Obstructive jaundice**: US differentiates obstruction from parenchymal disease; determines level and cause
- **CBD obstruction** most commonly from: distal bile duct stones, pancreatic head carcinoma, cholangiocarcinoma
- **Cystic duct/bile duct anomalies** → surgical injury risk at cholecystectomy

## 3. Spleen

### Normal Anatomy & US Measurements

| Parameter                 | Normal Value |
| ------------------------- | ------------ |
| Length                    | ≤ 12 cm      |
| Thickness (hilum)         | ≤ 4 cm       |
| Width                     | ≤ 7 cm       |
| Splenic index (L × T × W) | 120–480 cm³  |
| Splenic artery PSV        | 25–45 cm/s   |
| Splenic artery diameter   | 4–8 mm       |

### Key Anatomic Points

- **Largest lymphatic organ**; functions: lymphocyte production, blood filtration, RBC/platelet removal, blood reservoir
- **Gastrosplenic ligament (GSL)**: Left anterior margin of lesser sac; carries short gastric + left gastroepiploic vessels
- **Splenorenal ligament (SRL)**: Left posterior margin of lesser sac; carries splenic artery/vein
- **Splenic vein**: Runs along dorsal pancreas; joins SMV → portal vein; receives IMV

### Imaging Approach

- **Best window**: 10th–11th intercostal space, left midaxillary line, end-expiration, right decubitus
- **Spleen is poorly accessed** from posterior (lung base), anterior, or subcostal (stomach/colon)
- **Splenic vein**: Assess patency and flow direction at hilum and midline

### Clinical Pearls

| Finding                             | Significance                                                                                       |
| ----------------------------------- | -------------------------------------------------------------------------------------------------- |
| **Accessory spleen (splenunculus)** | 10–30%; near hilum; can enlarge post-splenectomy, mimic mass; intrapancreatic ≤ 3 cm from tail tip |
| **Wandering spleen**                | Long mesentery; anywhere in abdomen; risk of torsion                                               |
| **Asplenia/polysplenia**            | Heterotaxy syndromes; situs ambiguus; associated with cardiovascular anomalies, malrotation        |
| **Splenosis**                       | Peritoneal implantation post-trauma; can mimic polysplenia                                         |
| **Portal hypertension**             | Splenic vein diameter increase <20% on inspiration (normal = 50–100% increase)                     |

## 4. Pancreas

### Normal Anatomy & US Measurements

| Parameter                    | Normal Value                                 |
| ---------------------------- | -------------------------------------------- |
| Parenchymal width (sagittal) | < 3 cm                                       |
| Pancreatic duct diameter     | < 3 mm                                       |
| Echogenicity                 | Homogeneous, isoechoic/hyperechoic vs. liver |

### Anatomic Divisions & Relations

| Division | Key Relations                                                                                                                       |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| **Head** | C-loop of duodenum; uncinate process posterior to SMV; bile duct along posterior surface; joins pancreatic duct at ampulla of Vater |
| **Neck** | Thinnest part; anterior to SMA/SMV; SMV + splenic vein → portal vein behind neck                                                    |
| **Body** | Splenic vein in groove on posterior surface; anterior = back wall of lesser sac                                                     |
| **Tail** | Within splenorenal ligament at splenic hilum                                                                                        |

### Imaging Pearls

- **Hard to visualize** due to overlying bowel gas — best with graded compression, fasting >6–8 hrs, oral water
- **Transverse epigastric** approach preferred
- **Pancreatic duct** becomes more prominent with age (still normal < 3 mm)
- **Parenchyma**: Young males may have very echogenic pancreas; uncinate process may be echo-poor in 25% (lack of fat)
- **EUS / CEUS** useful for masses, tissue sampling, vascular evaluation
- **Annular pancreas**: Parenchyma surrounds 2nd part of duodenum

### Clinical Pearls

| Condition                       | US Findings                                                                       |
| ------------------------------- | --------------------------------------------------------------------------------- |
| **Acute pancreatitis**          | Enlarged hypoechoic pancreas, peripancreatic fluid, collections/pseudocysts       |
| **Chronic pancreatitis**        | Atrophy, hypoechoic, dilated duct, calcifications, ± pseudocyst, ± pseudoaneurysm |
| **Pancreatic ductal carcinoma** | Double duct sign (dilated CBD + pancreatic duct)                                  |

## 5. Kidneys

### Normal Anatomy & US Measurements

| Parameter                  | Normal Value                 |
| -------------------------- | ---------------------------- |
| Bipolar length             | 9–14 cm                      |
| Width                      | ~5 cm                        |
| Cortical thickness         | ≥7 mm                        |
| Parenchymal thickness      | ≥15 mm                       |
| Pelvis AP diameter (adult) | < 10 mm                      |
| Renal artery PSV           | 75–125 cm/s (abnormal > 200) |
| Resistive index (RI)       | < 0.7 (normal)               |
| Renal vein velocity        | 18–33 cm/s                   |

### US Echogenicity Pattern

| Structure                 | Echogenicity                                  |
| ------------------------- | --------------------------------------------- |
| Renal capsule             | Well-defined echogenic line                   |
| Renal cortex              | Less echogenic than liver/spleen              |
| Medullary pyramids        | Less echogenic than cortex                    |
| Corticomedullary junction | Normally well-defined                         |
| Renal sinus               | Echogenic (fat + vessels + collecting system) |

### Vascular Anatomy Pearls

- **Renal arteries**: Single in ~2/3; duplicated or accessory in ~1/3; arise at L1–L2
- **Left renal vein**: Crosses between aorta and SMA (anterior); retroaortic or circumaortic variations exist
- **Doppler**: Low-resistance waveform; continuous diastolic flow; PSV 75–125 cm/s; RI < 0.7
- **Accessory vessels**: Must be accounted for in surgical planning; best seen on CTA/MRA

### Physiologic Pelvicalyceal Dilatation

- Seen in pregnancy (up to 75% of right kidneys at 20 weeks), resolves within 48 hrs postpartum
- Post-micturition images help exclude obstruction

### Imaging Approach

| Side                     | Best Approach                                                                   |
| ------------------------ | ------------------------------------------------------------------------------- |
| **Right kidney**         | Subcostal/intercostal with liver as window; raise right side                    |
| **Left kidney**          | Posterolateral with spleen as window; raise left side; pillow under right flank |
| **Renal artery origins** | Midline anterior approach                                                       |

### Key Embryologic Anomalies

- Renal ectopia, duplication, fetal lobulations, column of Bertin (lobar dysmorphism)
- VATER association: Vertebral, anorectal, tracheoesophageal, radial, renal

## 6. Bowel (GI Tract)

### Wall Layers by US (5 concentric layers, alternating echogenicity)

| Layer           | Sonographic Appearance | Histologic Correlate     |
| --------------- | ---------------------- | ------------------------ |
| 1st (outermost) | Echogenic              | Serosa                   |
| 2nd             | Hypoechoic             | Muscularis propria       |
| 3rd             | Echogenic              | Submucosa                |
| 4th             | Hypoechoic             | Muscularis mucosa        |
| 5th (innermost) | Echogenic              | Mucosa + lumen interface |

### Normal Measurements

| Segment             | Normal Wall Thickness  | Diameter                       |
| ------------------- | ---------------------- | ------------------------------ |
| Small bowel         | < 3–4 mm               | Jejunum ≤ 3 cm; Ileum 2.5–3 cm |
| Colon               | < 3–4 mm (distended)   | —                              |
| Appendix (normal)   | —                      | 4.5–7 mm outer diameter        |
| Stomach (distended) | ≤ 5 mm                 | —                              |
| Gastric rugae       | Visible when collapsed | —                              |

### Organ-Specific Key Points

| Organ               | Key Anatomy & US Pearls                                                                                                            |
| ------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| **Esophagus**       | Poorly seen on transabdominal US (rib cage, lung air); GEJ visible on epigastric view (between liver and aorta)                    |
| **Stomach**         | Best seen with oral water contrast; fundus/body/antrum identifiable; pylorus appears as hypoechoic thick structure                 |
| **Duodenum**        | 1st–2nd parts best seen; duodenal diverticula (~10%, air near pancreatic head)                                                     |
| **Jejunum**         | LUQ, stepladder pattern; feathery appearance from valvulae conniventes (more abundant than in ileum)                               |
| **Ileum**           | Smooth targetoid appearance; fewer folds; terminal ileum in RLQ, enters cecum medially at ~90° angle                               |
| **Peyer patches**   | Asymmetric hypoechoic deep mucosal thickening; common in children/young adults; can be marked in viral infections                  |
| **Ileocecal valve** | Silt-like (most common), dome, or fatty lips morphology                                                                            |
| **Colon**           | Retrograde sonographic approach recommended; haustral pattern + gas shadowing common; ascending/descending colon = retroperitoneal |
| **Appendix**        | Blind-ended, compressible; normal ≤ 7 mm; lumen may be collapsed or fluid-filled without inflammation                              |
| **Anus**            | Internal sphincter (hypoechoic, 2–4 mm), external sphincter (echogenic, 10–13 mm)                                                  |

### Embryology – Gut Rotation

1. **6th–8th week**: Midgut herniates into umbilical cord
2. **270° counterclockwise rotation** upon return
3. **Final position**: Small bowel central, colon peripheral (picture frame)
4. Ascending/descending colon fuse with posterior wall → retroperitoneal
5. Mesentery positions obliquely: duodenojejunal junction (L2 left) → right SI joint

## 7. Abdominal Lymph Nodes

| Node Group                      | Drainage Territory                                                                   |
| ------------------------------- | ------------------------------------------------------------------------------------ |
| **Preaortic (celiac, SM, IM)**  | Alimentary tract, liver, spleen, pancreas → intestinal trunks → cisterna chyli       |
| **Lateral aortic (paraaortic)** | Kidneys, adrenals, ureter, posterior abdominal wall, gonads, uterus/tubes            |
| **Retroaortic**                 | Posterior abdominal wall                                                             |
| **External iliac**              | Primary from inguinal nodes → common iliac                                           |
| **Internal iliac**              | Pelvic viscera, deep perineum, gluteal → common iliac                                |
| **Superficial inguinal**        | Superficial lower extremity, abdominal wall, perineum → deep inguinal/external iliac |
| **Deep inguinal**               | Medial femoral vein; receives superficial inguinal + popliteal → external iliac      |

### Normal Size Thresholds by Location

| Location               | Short-Axis Upper Limit |
| ---------------------- | ---------------------- |
| Retrocrural            | < 6 mm                 |
| Hepatogastric ligament | < 8 mm                 |
| Porta hepatis          | < 7 mm                 |
| Mesenteric             | < 5 mm                 |
| General (elsewhere)    | < 10 mm                |

### Key Imaging Points

- **CT** is gold standard for nodal staging; PET/CT or MRI for select cancers
- **US** normal nodes: elliptical, echogenic fatty hilum, uniform hypoechoic cortex; rarely detected in normal adults
- **Abnormal nodes**: loss of fatty hilum, central necrosis, cystic change, calcification, irregular margins, location in tumor drainage field
- **Lymphoma**: multiple enlarged hypoechoic/anechoic nodes
- **Metastatic**: more echogenic and heterogeneous than lymphomatous
- **Reactive/infectious**: nonspecific; necrotic centers suggest mycobacterial infection
- Normal-sized nodes can still harbor malignancy

## 8. Peritoneal Spaces and Structures

| Structure                                      | Key Points                                                                                                                                                             |
| ---------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Greater sac vs. Lesser sac (omental bursa)** | Lesser sac communicates via epiploic foramen (of Winslow); bounded anteriorly by caudate lobe, stomach, greater omentum; posteriorly by pancreas, left adrenal, kidney |
| **Supramesocolic space**                       | Divided by falciform ligament into right (subphrenic, subhepatic, lesser sac) and left (perihepatic, subphrenic: anterior perigastric, posterior perisplenic)          |
| **Inframesocolic compartment**                 | Right/left inframesocolic spaces, paracolic gutters, pelvic cavity — **most dependent part** in both erect and supine positions                                        |
| **Peritoneum**                                 | Parietal (lines wall) vs. visceral (lines organs); thin serous membrane of squamous mesothelium                                                                        |
| **Mesentery**                                  | Double-layer peritoneum enclosing organ; root of small bowel mesentery ~15 cm, from L2 left → right iliac fossa; contains SMA, nerves, lymphatics                      |
| **Greater omentum**                            | 4-layer fold from greater curvature; mobile barrier against infection/tumor spread; contains fat and abundant lymph nodes                                              |
| **Lesser omentum**                             | Hepatogastric + hepatoduodenal ligaments; contains CBD, hepatic/gastric vessels, portal vein                                                                           |
| **Ligaments**                                  | Double-layered peritoneal folds (e.g., splenorenal, falciform); connect viscera or viscera to wall; contain vessels or fetal remnants                                  |
| **Peritoneal recesses**                        | Morison pouch (hepatorenal), pouch of Douglas (rectouterine); key dependent fluid collection sites                                                                     |

### Imaging Recommendations

| Parameter      | Details                                                                                                                      |
| -------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| **Transducer** | 2–5 MHz for abdominal survey; 8–15 MHz high-frequency linear for anterior wall/parietal peritoneum                           |
| **Position**   | Supine + decubitus to differentiate free vs. loculated fluid                                                                 |
| **FAST exam**  | Scans RUQ (Morison pouch), LUQ (perisplenic), pelvis (Douglas pouch); widely used in trauma and undifferentiated hypotension |
| **Pitfall**    | Acute intraperitoneal bleeding is anechoic — may be confused with ascites                                                    |

## 9. Abdominal Wall

### Muscle Layers (Anterior)

| Muscle                    | Key Feature                                                            |
| ------------------------- | ---------------------------------------------------------------------- |
| **External oblique**      | Largest, most superficial; origin: ribs 5–12                           |
| **Internal oblique**      | Runs at right angles to external oblique                               |
| **Transversus abdominis** | Innermost flat muscle                                                  |
| **Rectus abdominis**      | Strap-like; in rectus sheath with superior/inferior epigastric vessels |

### Landmarks

| Structure                | Definition                                                                                     |
| ------------------------ | ---------------------------------------------------------------------------------------------- |
| **Linea alba**           | Fibrous raphe xiphoid → pubis; central attachment for abdominal muscles                        |
| **Linea semilunaris**    | Vertical band at lateral edge of rectus sheath                                                 |
| **Rectus sheath**        | Formed by aponeuroses of oblique + transversus muscles                                         |
| **Transversalis fascia** | Deep to muscles; lines entire abdominal wall; separated from peritoneum by extraperitoneal fat |
| **Inguinal ligament**    | Inferior edge of external oblique aponeurosis                                                  |

### Muscles (Posterior)

| Muscle                 | Origin                             | Insertion                      | Action                                      |
| ---------------------- | ---------------------------------- | ------------------------------ | ------------------------------------------- |
| **Psoas major**        | T12–L5 transverse processes/bodies | Lesser trochanter of femur     | Flexes thigh, lateral trunk bending         |
| **Iliacus**            | Iliac fossa                        | Lesser trochanter (with psoas) | Iliopsoas flexes thigh                      |
| **Quadratus lumborum** | Iliac crest + lumbar TP            | 12th rib                       | Stabilizes thorax/pelvis during respiration |
| **Erector spinae**     | Sacrum, ilium, lumbar spines       | Ribs + vertebrae               | Extends vertebral column                    |

### Imaging Approach

| Parameter      | Details                                                               |
| -------------- | --------------------------------------------------------------------- |
| **Transducer** | 5–18 MHz linear (anterior wall, paraspinal); 3–5 MHz (posterior wall) |
| **Position**   | Supine; Valsalva + standing to elicit hernias; prone for paraspinal   |
| **Comparison** | Always compare with contralateral side                                |

## Summary of Key Normal Measurements

| Structure                | Measurement              | Cutoff                                     |
| ------------------------ | ------------------------ | ------------------------------------------ |
| **Small bowel wall**     | < 3–4 mm                 | > 4 mm abnormal                            |
| **Colonic wall**         | < 3–4 mm                 | > 4 mm abnormal                            |
| **Appendix**             | ≤ 7 mm OD                | > 7 mm suggests inflammation               |
| **GB wall**              | ≤ 3 mm (fasting)         | > 3–4 mm abnormal                          |
| **CBD**                  | < 6–7 mm                 | > 7 mm dilated                             |
| **Pancreatic duct**      | < 3 mm                   | > 3 mm dilated                             |
| **Kidney length**        | 9–14 cm                  | < 9 cm small; asymmetry > 1 cm significant |
| **Renal pelvis AP**      | < 10 mm                  | > 10 mm suggests obstruction               |
| **Renal RI**             | < 0.7                    | > 0.7 abnormal                             |
| **Spleen length**        | ≤ 12 cm                  | > 12 cm splenomegaly                       |
| **Portal vein diameter** | < 13 mm                  | > 13 mm suggests portal hypertension       |
| **Lymph node (general)** | < 10 mm short axis       | Varies by location                         |
| **Liver span**           | 15–20 cm (midclavicular) | —                                          |
