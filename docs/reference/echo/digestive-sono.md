# Digestive System Sonography — Diagnosis & Echo Findings

**消化系統超音波 — 疾病診斷與回音特徵**

## 概述

The digestive system encompasses the GI tract from the oral cavity to the anus, along with accessory organs — the liver, gallbladder, biliary tree, and pancreas. Abdominal ultrasound is the first-line imaging modality due to its wide availability, real-time capability, and lack of ionizing radiation, complemented by CT, MRI, and EUS for higher-resolution evaluation.

This document provides a structured overview by organ system, covering clinical context, diagnostic criteria, and sonographic findings (B-mode, color Doppler, spectral Doppler, and CEUS where relevant).

## 1. Esophagus

The esophagus is poorly visualized by conventional transabdominal US due to air interference; the gastroesophageal junction (GEJ) and distal esophagus are accessible from a subcostal/epigastric window.

### 1.1 GERD & Hiatal Hernia

| Parameter                          | Findings                                                                                                |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------- |
| **GERD**                           | GEJ mucosal fold thickening >3 mm on high-res US; dynamic fluid reflux occasionally seen (pediatric)    |
| **Hiatal hernia (sliding)**        | Posterior mediastinal echogenic mass continuous with stomach, reducible; gastric fundus above diaphragm |
| **Hiatal hernia (paraesophageal)** | Herniated segment shows peristalsis; ± fluid/food content within intrathoracic stomach                  |

### 1.2 Esophageal Carcinoma

| Parameter        | Findings                                                       |
| ---------------- | -------------------------------------------------------------- |
| **Primary mass** | Distal esophageal/GEJ: hypoechoic irregular mural thickening   |
| **LN**           | Celiac axis, lesser curvature: enlarged round hypoechoic nodes |
| **Metastases**   | Liver: hypoechoic or heterogeneous focal lesions               |

## 2. Stomach & Duodenum

Best evaluated with oral water contrast. Normal wall ≤5 mm (distended). Five layers visible: mucosa, muscularis mucosae, submucosa, muscularis propria, serosa.

### 2.1 Gastritis

| Parameter   | Findings                                                            |
| ----------- | ------------------------------------------------------------------- |
| **Wall**    | Diffuse or focal thickening (>5 mm), preserved layered architecture |
| **Doppler** | Increased mural vascularity                                         |
| **Rugae**   | Thickened folds (antrum/fundus)                                     |

### 2.2 Peptic Ulcer Disease (PUD)

| Parameter       | Findings                                                                                                          |
| --------------- | ----------------------------------------------------------------------------------------------------------------- |
| **Direct**      | Focal mucosal defect with surrounding hypoechoic edema; crater with air (hyperechoic + shadow)                    |
| **Perforation** | Free intraperitoneal air (hyperechoic foci, dirty shadowing, non-gravity-dependent); free subhepatic/pelvic fluid |

### 2.3 Gastric Carcinoma

| Parameter            | Findings                                                                                                 |
| -------------------- | -------------------------------------------------------------------------------------------------------- |
| **Morphology**       | Focal or circumferential hypoechoic mural thickening, **loss of wall stratification**; pseudokidney sign |
| **Linitis plastica** | Diffuse rigid thickening, reduced distensibility                                                         |
| **LN**               | Perigastric, celiac, retropancreatic                                                                     |
| **Mets**             | Liver (hypoechoic), omental cake, ascites                                                                |

### 2.4 Gastric Outlet Obstruction

| Parameter        | Findings                                                                             |
| ---------------- | ------------------------------------------------------------------------------------ |
| **Stomach**      | Markedly dilated, fluid-filled, extending to pelvis; active proximal peristalsis     |
| **Pyloric mass** | Carcinoma; or hypertrophic pyloric stenosis (infant: muscle >3-4 mm, channel >17 mm) |

### 2.5 GIST (Gastrointestinal Stromal Tumor)

| Parameter              | Findings                                                                                      |
| ---------------------- | --------------------------------------------------------------------------------------------- |
| **Morphology**         | Hypoechoic mass from muscularis propria, well-defined, round/lobulated; exophytic most common |
| **Internal**           | Small: homogeneous; Large: ± cystic degeneration, necrosis, calcification                     |
| **Doppler**            | Hypervascular                                                                                 |
| **Malignant features** | >5 cm, lobulated, necrosis, invasion, liver mets                                              |

### 2.6 Duodenal Ulcer (Indirect Signs)

| Parameter       | Findings                                                                              |
| --------------- | ------------------------------------------------------------------------------------- |
| **Indirect**    | Bulb wall thickening, hypoechoic edema, bulb deformity (scarring), pyloric thickening |
| **Doppler**     | Increased flow at ulcer base                                                          |
| **Perforation** | Free air, localized fluid, omental thickening                                         |

## 3. Small Intestine

Normal wall ≤3 mm. Valvulae conniventes visible as transverse folds.

### 3.1 Crohn's Disease

| Parameter         | Findings                                                                                                           |
| ----------------- | ------------------------------------------------------------------------------------------------------------------ |
| **Wall**          | Hypoechoic **transmural thickening** (≥4 mm), most commonly terminal ileum; loss of stratification (active)        |
| **Doppler**       | Increased mural vascularity (active inflammation)                                                                  |
| **Mesentery**     | Fat hyperechogenicity (creeping fat / fat wrapping)                                                                |
| **LN**            | Enlarged hypoechoic mesenteric nodes                                                                               |
| **Complications** | Fistula (linear hypoechoic channel, Doppler flow), abscess (complex collection), stricture (pre-stenotic dilation) |
| **Distribution**  | Skip lesions                                                                                                       |

### 3.2 Celiac Disease

| Parameter   | Findings                                                                          |
| ----------- | --------------------------------------------------------------------------------- |
| **Bowel**   | Dilated fluid-filled loops, reduced peristalsis; increased duodenal/jejunal folds |
| **LN**      | Mesenteric lymphadenopathy (sandwich sign)                                        |
| **Spleen**  | Atrophy (late)                                                                    |
| **Ascites** | Rare, refractory disease                                                          |

### 3.3 Small Bowel Obstruction (SBO)

| Parameter       | Ileus (Paralytic)       | Mechanical Obstruction                                                    |
| --------------- | ----------------------- | ------------------------------------------------------------------------- |
| **Dilation**    | Diffuse (small + large) | Focal proximal to obstruction                                             |
| **Peristalsis** | Absent/weak             | Hyperactive (early → to-and-fro / whirling), antiperistalsis; late absent |
| **Wall**        | Normal/thin             | Early normal, late edematous thickening                                   |
| **Valvulae**    | Normal                  | Edematous → **keyboard sign** / spring sign                               |
| **Doppler**     | Normal flow             | Ischemia → decreased/absent                                               |
| **Ascites**     | Common (small)          | ±; strangulation → fluid                                                  |
| **Content**     | Gas-predominant         | **Fluid-predominant**                                                     |
| **Note**        |                         | Seek **transition point** + cause (adhesions, hernia, tumor)              |

### 3.4 Meckel's Diverticulum

| Parameter         | Findings                                                                               |
| ----------------- | -------------------------------------------------------------------------------------- |
| **Morphology**    | Blind-ending tubular structure from antimesenteric ileum; wall layers like small bowel |
| **Complications** | Diverticulitis (fat stranding, fluid), intussusception (lead point)                    |

## 4. Large Intestine

Normal wall ≤4 mm. Haustra and teniae identify colon.

### 4.1 Appendicitis

| Parameter               | Findings                                                                                                             |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------- |
| **Outer diameter**      | **≥7 mm** (outer-to-outer), non-compressible, blind-ending                                                           |
| **Wall**                | Thickened (>3 mm), blurred stratification                                                                            |
| **Cross-section**       | **Target sign**: central hypoechoic (edematous submucosa) + outer hyperechoic rim                                    |
| **Appendicolith**       | Hyperechoic focus with shadow (≈30%)                                                                                 |
| **Periappendiceal fat** | Increased echogenicity (fat stranding)                                                                               |
| **Doppler**             | Hyperemic wall (gangrenous: decreased flow)                                                                          |
| **Complications**       | Perforation (non-visualized appendix, abscess, free fluid, mesenteric LN); phlegmon (ill-defined heterogeneous mass) |
| **Tenderness**          | Sonographic McBurney sign                                                                                            |
| **Note**                | Operator-dependent; higher sensitivity in children; CT second-line                                                   |

### 4.2 Diverticular Disease

| Parameter      | Diverticulosis                                                  | Diverticulitis                                                        |
| -------------- | --------------------------------------------------------------- | --------------------------------------------------------------------- |
| **Morphology** | Small outpouchings, hyperechoic foci with dirty shadowing (gas) | Hypoechoic wall thickening; hyperechoic pericolonic fat (stranding)   |
| **Abscess**    | -                                                               | Pericolic complex collection                                          |
| **Doppler**    | Normal                                                          | Increased bowel wall + pericolonic flow                               |
| **Location**   | Sigmoid most common (Western); right colon (Asian)              | Same                                                                  |
| **DDx**        |                                                                 | Colon cancer: more focal, rigid, no intense pericolonic fat stranding |

### 4.3 Colorectal Carcinoma

| Parameter      | Findings                                                                                  |
| -------------- | ----------------------------------------------------------------------------------------- |
| **Morphology** | Focal hypoechoic mural thickening, **loss of stratification**; pseudokidney / target sign |
| **Lumen**      | Narrowing with proximal dilation                                                          |
| **LN**         | Pericolic lymphadenopathy                                                                 |
| **Liver mets** | Hypoechoic (common), hyperechoic (mucinous), cystic, ± halo; essential to assess          |

### 4.4 Ulcerative Colitis

| Parameter        | Findings                                                                                                       |
| ---------------- | -------------------------------------------------------------------------------------------------------------- |
| **Wall**         | Continuous circumferential hypoechoic thickening from rectum; preserved stratification (submucosal thickening) |
| **Doppler**      | Increased mural flow (active)                                                                                  |
| **Chronic**      | Loss of haustra (lead-pipe colon); decreased stratification (severe)                                           |
| **Complication** | Toxic megacolon: >6 cm transverse, air-fluid levels, free fluid                                                |

### 4.5 Intussusception

| Parameter        | Findings                                                                           |
| ---------------- | ---------------------------------------------------------------------------------- |
| **Transverse**   | **Target/doughnut sign**: concentric hyperechoic + hypoechoic rings                |
| **Longitudinal** | **Pseudokidney / sandwich sign**: hypoechoic walls + central hyperechoic mesentery |
| **Doppler**      | Flow → hydrostatic reduction possible; No flow → ischemia, surgery                 |
| **Location**     | Right abdomen (ileocolic) most common                                              |
| **Children**     | Idiopathic (>90%); **Adults**: seek lead point (tumor, polyp, Meckel's)            |

### 4.6 Mesenteric Ischemia / Infarction

| Parameter             | Findings                                                                                        |
| --------------------- | ----------------------------------------------------------------------------------------------- |
| **Bowel wall**        | Early: hypoechoic thickening (edema); Late: thinning, intramural gas (pneumatosis intestinalis) |
| **Peristalsis**       | **Absent** (observe 2-3 min)                                                                    |
| **Mesentery**         | Increased echogenicity (fat stranding)                                                          |
| **Doppler**           | **No bowel wall flow** — critical sign                                                          |
| **Portal venous gas** | Punctate hyperechoic foci in peripheral liver — grave prognosis                                 |
| **Ascites**           | ±                                                                                               |
| **Clinical**          | Pain out of proportion; high-risk (AF, atherosclerosis, hypotension)                            |

## 5. Liver

Normal: homogeneous, medium echogenicity (similar to or slightly > renal cortex).

### 5.1 Diffuse Liver Disease

#### 5.1.1 Fatty Liver (Hepatic Steatosis / MASLD)

| Parameter                 | Findings                                                                                                                                                                                       |
| ------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Echogenicity**          | Diffusely **increased** (bright liver), > renal cortex and spleen                                                                                                                              |
| **Texture**               | Homogeneous fine echoes; positive hepatorenal echo contrast                                                                                                                                    |
| **Vascular blurring**     | Reduced/absent intrahepatic vessel wall definition                                                                                                                                             |
| **Posterior attenuation** | Severe: posterior attenuation, poor diaphragmatic visualization                                                                                                                                |
| **Grading**               | **Mild**: subtle increase, hepatorenal contrast (+), vessels clear; **Moderate**: moderate increase, vascular blurring; **Severe**: marked increase, posterior attenuation, diaphragm obscured |
| **Note**                  | May be focal (focal fatty sparing / focal fatty infiltration), near gallbladder fossa and left portal vein                                                                                     |

#### 5.1.2 Cirrhosis

| Parameter                | Findings                                                                                                     |
| ------------------------ | ------------------------------------------------------------------------------------------------------------ |
| **Echogenicity**         | Coarse heterogeneous, nodular surface                                                                        |
| **Lobe ratio**           | Right lobe atrophy; caudate + left lateral hypertrophy (CL/RL >0.65)                                         |
| **Regenerative nodules** | Multiple ill-defined isoechoic nodules, ≤1-2 cm, no flow                                                     |
| **Hepatic veins**        | Loss of phasicity, narrowed, tortuous                                                                        |
| **Portal HTN**           | PV >13 mm, splenomegaly, ascites, recanalized umbilical vein, collaterals (coronary vein, splenorenal shunt) |
| **Doppler**              | PV velocity <15 cm/s, hepatofugal flow; hepatic artery RI >0.7                                               |

#### 5.1.3 Acute Hepatitis

| Parameter        | Findings                                                                                                        |
| ---------------- | --------------------------------------------------------------------------------------------------------------- |
| **Echogenicity** | Diffusely **hypoechoic** or normal, ± mottled                                                                   |
| **GB wall**      | Marked thickening (≥5 mm), double-wall sign (edema)                                                             |
| **Liver size**   | Mild-moderate hepatomegaly, blunt edge                                                                          |
| **Vessels**      | Bright periportal tracking — **"starry sky"** pattern (hyperechoic portal tracts against hypoechoic background) |
| **Portal LN**    | ± Hilar lymphadenopathy                                                                                         |
| **Note**         | Low US sensitivity for mild hepatitis; relies on clinical + lab                                                 |

#### 5.1.4 VOD / SOS

| Parameter         | Findings                                     |
| ----------------- | -------------------------------------------- |
| **Echogenicity**  | Heterogeneous mosaic pattern                 |
| **Hepatic veins** | Marked narrowing/absent, monophasic waveform |
| **Portal vein**   | Hepatofugal flow                             |
| **GB wall**       | Marked edema                                 |
| **Ascites**       | Refractory massive                           |
| **Context**       | Post-HSCT, chemotherapy (cyclophosphamide)   |

### 5.2 Focal Liver Lesions

#### 5.2.1 Simple Hepatic Cyst

| Parameter        | Findings                                                                                       |
| ---------------- | ---------------------------------------------------------------------------------------------- |
| **Morphology**   | Round/oval, **smooth well-defined** borders                                                    |
| **Echogenicity** | **Anechoic**, posterior acoustic enhancement, lateral edge shadowing                           |
| **Wall**         | Thin (<1 mm), imperceptible                                                                    |
| **Internal**     | No septations, no solid components, no calcification                                           |
| **Doppler**      | No flow                                                                                        |
| **Caution**      | Septations, wall thickening, irregularity, solid components → complex cyst (cystadenoma, mets) |

#### 5.2.2 Hemangioma

| Parameter        | Findings                                                                         |
| ---------------- | -------------------------------------------------------------------------------- |
| **Morphology**   | Round/oval, **well-defined**; most common benign liver tumor                     |
| **Echogenicity** | Classic: **homogeneous hyperechoic** with posterior enhancement                  |
| **Size**         | Mostly <3 cm; giant (>5 cm): mixed echo                                          |
| **Doppler**      | Usually no detectable flow (low-flow); giant: ± punctate flow                    |
| **CEUS**         | **Nodular peripheral enhancement**, centripetal fill-in, persistent (no washout) |
| **DDx**          | Hyperechoic metastases (RCC, NET): irregular borders, washout on CEUS            |

#### 5.2.3 Hepatocellular Carcinoma (HCC)

| Parameter                | Findings                                                                                                                     |
| ------------------------ | ---------------------------------------------------------------------------------------------------------------------------- |
| **Background**           | 80% in cirrhosis                                                                                                             |
| **Morphology**           | Round/lobulated, well-defined or blurred; ± tumor capsule                                                                    |
| **Small (<2 cm)**        | **Hypoechoic**, homogeneous                                                                                                  |
| **Medium (2-5 cm)**      | **Mosaic pattern** (hemorrhage, necrosis, fatty change)                                                                      |
| **Large (>5 cm)**        | Markedly heterogeneous, ± calcification, necrosis                                                                            |
| **Halo**                 | **Hypoechoic halo** (peripheral sonolucent zone) — specific                                                                  |
| **Doppler**              | **Hypervascular**: intratumoral pulsatile arterial flow; **basket pattern** (peripheral vessels); low RI (<0.5); AV shunting |
| **Portal vein invasion** | Tumor thrombus in main PV or branches — critical; arterial flow within thrombus on Doppler                                   |
| **CEUS**                 | **Arterial hyperenhancement**, portal/delayed **washout** — diagnostic key                                                   |
| **Note**                 | Screening: US + AFP q6mo in cirrhosis                                                                                        |

#### 5.2.4 Liver Metastases

| Parameter               | Findings                                                                      |
| ----------------------- | ----------------------------------------------------------------------------- |
| **Number**              | Usually **multiple**; solitary less common (e.g., colon)                      |
| **Hypoechoic**          | Most common (lung, breast, gastric, pancreatic, lymphoma)                     |
| **Target / bull's-eye** | Central hyperechoic necrosis + peripheral hypoechoic halo                     |
| **Hyperechoic**         | RCC, NET, thyroid, melanoma, choriocarcinoma                                  |
| **Cystic**              | Ovarian, mucinous colon, sarcoma (or post-treatment)                          |
| **Calcified**           | Mucinous colon, gastric, osteosarcoma, melanoma                               |
| **Doppler**             | Some peripheral hypervascularity                                              |
| **Features**            | Multiple, variable size, **irregular margins** (vs hemangioma's sharp border) |

#### 5.2.5 Liver Abscess

| Parameter        | Pyogenic                                                 | Amebic                                   | Fungal (Microabscesses)     |
| ---------------- | -------------------------------------------------------- | ---------------------------------------- | --------------------------- |
| **Number**       | Multiple (may be single)                                 | **Solitary**                             | Multiple small              |
| **Location**     | Variable                                                 | **Right lobe**, near diaphragm           | Diffuse                     |
| **Echogenicity** | Mixed → anechoic (liquefaction); ± gas (dirty shadowing) | Homogeneous hypoechoic ("anchovy paste") | Target / bull's-eye lesions |
| **Wall**         | Thick irregular; edematous halo                          | Well-defined                             | Poorly defined              |
| **Doppler**      | Hyperemic rim, no central flow                           | Peripheral hyperemia                     | No central flow             |
| **CEUS**         | Rim enhancement, no central enhancement                  | Same                                     | Same                        |
| **Context**      | Immunocompetent; biliary source                          | Endemic area                             | Immunocompromised           |

#### 5.2.6 Focal Nodular Hyperplasia (FNH)

| Parameter        | Findings                                                                                                   |
| ---------------- | ---------------------------------------------------------------------------------------------------------- |
| **Echogenicity** | **Isoechoic** or mildly hypoechoic; may be invisible on B-mode                                             |
| **Central scar** | Classic central **stellate scar** (hyper- or hypoechoic), 25-50%                                           |
| **Doppler**      | **Spoke-wheel pattern**: central artery radiating outward, low RI                                          |
| **CEUS**         | Rapid homogeneous arterial enhancement, no washout (or delayed isoenhancement); central scar non-enhancing |

#### 5.2.7 Hepatocellular Adenoma

| Parameter        | Findings                                                                                 |
| ---------------- | ---------------------------------------------------------------------------------------- |
| **Background**   | Young women (OCP), steroids, metabolic disease                                           |
| **Echogenicity** | Mostly **hyperechoic** (fat/glycogen), well-defined; ± hemorrhage → complex cystic areas |
| **Doppler**      | Mild peripheral flow; no spoke-wheel                                                     |
| **CEUS**         | Arterial enhancement, no washout (key DDx from HCC); β-catenin type: malignant potential |

#### 5.2.8 Intrahepatic Cholangiocarcinoma (ICC)

| Parameter              | Findings                                                                                                           |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------ |
| **Morphology**         | Irregular; types: **mass-forming**, **periductal infiltrating**, **intraductal**                                   |
| **Echogenicity**       | Heterogeneous **hypoechoic**, ill-defined                                                                          |
| **Bile duct dilation** | Peripheral ducts distal to mass; satellite nodules; capsular retraction                                            |
| **Calcification**      | ± (≈20%)                                                                                                           |
| **Doppler**            | Peripheral flow, scant central (hyalinized stroma)                                                                 |
| **CEUS**               | Arterial rim enhancement, delayed **progressive central enhancement** (fibrous stroma) — distinct from HCC washout |

#### 5.2.9 Nodular Regenerative Hyperplasia (NRH)

| Parameter        | Findings                                                     |
| ---------------- | ------------------------------------------------------------ |
| **Background**   | Non-cirrhotic; autoimmune, myeloproliferative, chemotherapy  |
| **Morphology**   | Multiple micronodules (1-3 mm), diffuse nodular parenchyma   |
| **Echogenicity** | Mostly **isoechoic**, ill-defined                            |
| **DDx**          | US difficult vs cirrhosis; needs pathology (reticulin stain) |

### 5.3 Focal Lesions — Feature Summary

| Lesion         | Echogenicity   | Margin        | Halo      | Doppler                | CEUS Arterial        | CEUS Delayed           |
| -------------- | -------------- | ------------- | --------- | ---------------------- | -------------------- | ---------------------- |
| **Cyst**       | Anechoic       | Sharp, smooth | -         | No flow                | No enhancement       | No enhancement         |
| **Hemangioma** | Hyperechoic    | Sharp         | -         | No flow                | Nodular peripheral   | Centripetal fill-in    |
| **HCC**        | Hypo-/mosaic   | Variable      | +/ halo   | Hypervascular arterial | **Hyperenhancement** | **Washout**            |
| **Metastasis** | Variable       | Irregular     | +/ target | Variable               | Rim / heterogeneous  | **Washout**            |
| **Abscess**    | Mixed/anechoic | Thick wall    | +/ edema  | Rim hyperemia          | Rim enhancement      | No central enhancement |
| **FNH**        | Iso-/hypo      | Lobulated     | -         | Spoke-wheel            | **Hyperenhancement** | Iso-/hyperenhancement  |
| **Adenoma**    | Iso-/hyper     | Well-defined  | -         | Mild peripheral        | **Hyperenhancement** | Isoenhancement         |

## 6. Gallbladder & Bile Ducts

Fasting ≥6 hr. Normal wall ≤3 mm, CBD ≤6 mm (post-cholecystectomy/elderly up to 8-10 mm).

### 6.1 Cholelithiasis

| Parameter                | Findings                                                                                                             |
| ------------------------ | -------------------------------------------------------------------------------------------------------------------- |
| **Morphology**           | **Echogenic intraluminal focus** with **clean acoustic shadowing** (sensitivity ≈95%)                                |
| **Mobility**             | **Gravity-dependent** — key DDx from polyps                                                                          |
| **Small stones (<3 mm)** | May not shadow; use harmonic imaging                                                                                 |
| **Sludge**               | Hypoechoic sediment, no shadow, may shift                                                                            |
| **WES triad**            | **W**all (hyperechoic GB wall) + **E**cho (stone surface) + **S**hadow (posterior acoustic shadow) — stone-filled GB |
| **Tumefactive sludge**   | Nodular/mobile mass-like echoes; **no Doppler flow** (vs polyp/carcinoma)                                            |
| **Doppler**              | No flow (vs polyp)                                                                                                   |
| **Note**                 | Impacted stone at neck → hydrops, no mobility                                                                        |

### 6.2 Biliary Sludge

| Parameter        | Findings                                                                          |
| ---------------- | --------------------------------------------------------------------------------- |
| **Echogenicity** | Low-level gravity-dependent echoes, no shadowing                                  |
| **Behavior**     | Fluid-debris level; slow movement with repositioning                              |
| **Tumefactive**  | Nodular mobile mass; **no vascularity on Doppler** (key DDx from polyp/carcinoma) |

### 6.3 Cholecystitis

#### 6.3.1 Acute Cholecystitis

| Parameter                 | Findings                                                                                                                                                                                                                         |
| ------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **GB wall**               | Diffuse **thickening** (≥4 mm), **stratified / double-wall sign**: inner hypoechoic (edema) + outer hyperechoic                                                                                                                  |
| **Distension**            | >5 cm transverse, >10 cm long                                                                                                                                                                                                    |
| **Stones**                | Present in >90%; impacted stone at neck                                                                                                                                                                                          |
| **Murphy sign**           | Probe compression → **pain** (sonographic Murphy sign), sensitivity ≈75%                                                                                                                                                         |
| **Pericholecystic fat**   | Increased echogenicity (fat stranding)                                                                                                                                                                                           |
| **Pericholecystic fluid** | ±                                                                                                                                                                                                                                |
| **Doppler**               | Hyperemic wall                                                                                                                                                                                                                   |
| **Complications**         | **Gangrenous**: loss of wall definition, intraluminal membranes; **Emphysematous**: intramural/intraluminal gas (hyperechoic + dirty shadowing), DM, male, emergent; **Perforation**: focal wall defect, pericholecystic abscess |

#### 6.3.2 Chronic Cholecystitis

| Parameter        | Findings                                                                       |
| ---------------- | ------------------------------------------------------------------------------ |
| **GB wall**      | Mild thickening (2-4 mm), hyperechoic, **homogeneous** (no edematous layering) |
| **GB size**      | Usually **contracted**, reduced lumen                                          |
| **Stones**       | Almost always present                                                          |
| **Porcelain GB** | Extensive curvilinear wall calcification with posterior shadow                 |

### 6.4 Adenomyomatosis

| Parameter           | Findings                                                                                 |
| ------------------- | ---------------------------------------------------------------------------------------- |
| **Morphology**      | Focal/diffuse wall thickening with intramural cystic spaces (Rokitansky-Aschoff sinuses) |
| **Comet-tail sign** | Ring-down artifact from cholesterol crystals — **diagnostic**                            |
| **Types**           | **Focal** (fundal most common), **segmental** (hourglass), **diffuse**                   |

### 6.5 Gallbladder Polyp

| Parameter      | Findings                                                     |
| -------------- | ------------------------------------------------------------ |
| **Morphology** | Hyperechoic/isoechoic nodule, **non-mobile** (fixed to wall) |
| **<10 mm**     | Mostly cholesterol polyp, multiple, pedunculated, no shadow  |
| **>10 mm**     | Rule out adenomatous polyp or adenocarcinoma                 |
| **Doppler**    | Prominent flow → suspect malignancy                          |

### 6.6 Gallbladder Carcinoma

| Parameter        | Findings                                                                                                                                                                                   |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Morphology**   | Three patterns: ① **Mass replacing GB** (most common) — irregular, heterogeneous; ② **Focal/diffuse irregular wall thickening** (>3 mm); ③ **Polypoid mass** (>1 cm, broad base, vascular) |
| **Echogenicity** | Heterogeneous hypoechoic, ± calcification (stones)                                                                                                                                         |
| **Invasion**     | Direct liver (segment IV/V), hilar LN, bile duct obstruction                                                                                                                               |
| **Doppler**      | Hypervascular, elevated RI                                                                                                                                                                 |
| **Note**         | >80% with gallstones; early US diagnosis difficult                                                                                                                                         |

### 6.7 Bile Duct Dilation & Obstruction

| Parameter     | Normal      | Dilation                                                         |
| ------------- | ----------- | ---------------------------------------------------------------- |
| **CHD**       | <5 mm       | ≥6 mm (post-choice ≤8 mm)                                        |
| **CBD**       | <6-7 mm     | ≥7 mm; elderly/post-choice ≤10 mm                                |
| **IHD**       | Not visible | Visible → **double-barrel shotgun sign / parallel channel sign** |
| **Duct wall** | Thin        | >1.5 mm → sclerosing cholangitis, cholangiocarcinoma             |

| Obstruction level             | GB      | CHD     | CBD                | IHD        | Pancreatic duct                |
| ----------------------------- | ------- | ------- | ------------------ | ---------- | ------------------------------ |
| **Hilar**                     | Normal  | Dilated | Normal             | Dilated    | Normal                         |
| **Mid-CBD**                   | Dilated | Dilated | Dilated (proximal) | Dilated    | Normal                         |
| **Distal CBD (ampulla)**      | Dilated | Dilated | Dilated            | Dilated    | Dilated **(double duct sign)** |
| **Intrahepatic (unilateral)** | Normal  | Normal  | Normal             | Unilateral | Normal                         |

### 6.8 Choledocholithiasis

| Parameter       | Findings                                              |
| --------------- | ----------------------------------------------------- |
| **Morphology**  | Echogenic focus in dilated CBD, with shadow           |
| **Location**    | Distal CBD most common — obscured by bowel gas        |
| **Sensitivity** | 40-60% (limited); MRCP/EUS preferred for confirmation |
| **Doppler**     | No flow (vs cholangiocarcinoma)                       |

### 6.9 Cholangiocarcinoma

| Parameter      | Hilar (Klatskin, 60-70%)                                              | Intrahepatic (5-10%)                                     | Distal (20-30%)                  |
| -------------- | --------------------------------------------------------------------- | -------------------------------------------------------- | -------------------------------- |
| **Morphology** | Dilated IHD without visible mass; hyperechoic stricture at confluence | Peripheral mass ± satellite nodules; capsular retraction | Dilated CBD with abrupt tapering |
| **Duct wall**  | Thickened                                                             | -                                                        | Periductal soft tissue           |
| **Doppler**    | Hypovascular                                                          | Hypovascular                                             | Hypovascular                     |
| **Note**       | DDx: Mirizzi syndrome (GB neck stone compressing CHD)                 |                                                          |                                  |

### 6.10 Acute Cholangitis

| Parameter     | Findings                                                                                         |
| ------------- | ------------------------------------------------------------------------------------------------ |
| **Duct wall** | Thickened (≥2 mm), hyperechoic                                                                   |
| **Dilation**  | Mild-moderate                                                                                    |
| **Cause**     | CBD stones often found                                                                           |
| **Doppler**   | Increased wall flow                                                                              |
| **Clinical**  | **Charcot triad**: fever, jaundice, RUQ pain (+ Reynolds pentad: shock, AMS → acute suppurative) |

## 7. Pancreas

Normal: head <3 cm, body <2.5 cm, tail <2.5 cm. Duct ≤2 mm. Echogenicity isoechoic or slightly > liver.

### 7.1 Acute Pancreatitis

| Parameter        | Findings                                                                               |
| ---------------- | -------------------------------------------------------------------------------------- |
| **Causes**       | Gallstones (most common), alcohol, hypertriglyceridemia, medications, ERCP, idiopathic |
| **Size**         | Diffuse/focal **enlargement**                                                          |
| **Echogenicity** | Diffuse **hypoechoic** (edema); heterogeneous (necrosis, hemorrhage)                   |
| **Margins**      | Indistinct; peripancreatic fat stranding                                               |
| **Main duct**    | ± dilated or not visualized                                                            |
| **Fluid**        | Left anterior pararenal space, lesser sac                                              |
| **Necrosis**     | Focal anechoic (liquefactive) or hyperechoic (hemorrhage); **CEUS**: no enhancement    |
| **Doppler**      | Reduced parenchymal flow (edema); peripheral inflammatory hyperemia                    |
| **US role**      | Limited by gas (CT superior for severity); useful for pseudocyst follow-up             |

**Complications:**

- **Pseudocyst**: Rounded, well-defined, anechoic, posterior enhancement; ≥4 wk after onset; no internal flow
- **Walled-off necrosis**: Complex, heterogeneous, thick-walled; mixed solid-liquid
- **Abscess**: Complex with internal gas (hyperechoic + dirty shadowing)
- **Vascular**: Splenic vein thrombosis (absent Doppler flow); pseudoaneurysm (pulsatile, yin-yang sign on color Doppler)

### 7.2 Chronic Pancreatitis

| Parameter         | Findings                                                                          |
| ----------------- | --------------------------------------------------------------------------------- |
| **Echogenicity**  | **Coarse heterogeneous** (fibrosis) or diffusely increased                        |
| **Size**          | Atrophy or normal; ± focal inflammatory mass                                      |
| **Main duct**     | **Dilated** (>3 mm), **irregular, beaded / chain-of-lakes**                       |
| **Calcification** | **Hyperechoic foci with shadowing** in parenchyma and duct (pancreatic lithiasis) |
| **Pseudocyst**    | ±                                                                                 |
| **CBD**           | Inflammatory stricture may narrow distal CBD                                      |
| **DDx**           | Inflammatory mass vs cancer: US difficult → EUS-FNA                               |

### 7.3 Pancreatic Ductal Adenocarcinoma (PDAC)

| Parameter              | Findings                                                                            |
| ---------------------- | ----------------------------------------------------------------------------------- |
| **Location**           | **Head** 60-70%, body 15-20%, tail 5-10%                                            |
| **Morphology**         | Ill-defined irregular **hypoechoic mass**                                           |
| **Internal**           | Homogeneous (small) → heterogeneous (large: necrosis, calcification)                |
| **Double duct sign**   | Simultaneous CBD + pancreatic duct dilation (head mass)                             |
| **Pancreatic atrophy** | Distal to obstruction                                                               |
| **GB**                 | Courvoisier's law: painless jaundice + palpable GB                                  |
| **Doppler**            | **Hypovascular** (fibrous stroma)                                                   |
| **Vascular invasion**  | SMV/PV compression, stenosis, encasement, occlusion — **unresectability criterion** |
| **LN**                 | Celiac, peripancreatic, para-aortic                                                 |
| **Mets**               | Liver (hypoechoic), peritoneal implants, ascites                                    |

### 7.4 Pancreatic Neuroendocrine Tumor (PNET)

| Parameter          | Findings                                                                            |
| ------------------ | ----------------------------------------------------------------------------------- |
| **Morphology**     | Well-defined, round homogeneous **hypoechoic** mass                                 |
| **Size**           | <2 cm: hypoechoic; >2 cm: mixed (cystic degeneration, calcification ≈20%)           |
| **Doppler**        | **Hypervascular** — key DDx from hypovascular PDAC                                  |
| **Functional**     | Insulinoma (small, head/body), Gastrinoma (head/duodenal wall), Glucagonoma, VIPoma |
| **Non-functional** | Asymptomatic, large at detection; worse prognosis                                   |
| **Note**           | ≈10% multiple (MEN-1)                                                               |

### 7.5 Cystic Pancreatic Neoplasms

| Type                               | Echo Features                                                      | Key Points                                       |
| ---------------------------------- | ------------------------------------------------------------------ | ------------------------------------------------ |
| **SCA** (Serous Cystadenoma)       | Honeycomb of microcysts, central scar ± calcification, hyperechoic | Head, female > male; benign                      |
| **MCN** (Mucinous Cystic Neoplasm) | Macrocystic, thick wall, septations, ± mural nodules               | Tail, young women; **malignant potential**       |
| **IPMN — Main duct**               | Diffuse/segmental duct dilation >5 mm, mural nodules               | High malignant potential; communicates with duct |
| **IPMN — Branch duct**             | Grape-like clusters, communicates with duct                        | Lower risk                                       |
| **SPN** (Solid Pseudopapillary)    | Mixed solid-cystic, encapsulated, hemorrhagic                      | Young women; low-grade malignant; good prognosis |

## 8. Spleen

Normal: <12 cm craniocaudal; homogeneous, medium-high echogenicity (similar to liver).

### 8.1 Splenomegaly

| Parameter           | Normal                 | Splenomegaly                                       |
| ------------------- | ---------------------- | -------------------------------------------------- |
| **Length**          | <11-12 cm              | >12 cm                                             |
| **Thickness**       | <4-5 cm                | >5 cm                                              |
| **Volume**          | <300 mL                | >400-500 mL                                        |
| **Inferior margin** | Pointed                | **Rounded**                                        |
| **Echogenicity**    | Homogeneous, isoechoic | Homogeneous (early) → heterogeneous (infiltrative) |

**Causes**: Portal HTN (most common), infection (EBV, malaria), hematologic (CML, lymphoma, MM), infiltrative (sarcoidosis, Gaucher), autoimmune (SLE, RA/Felty)

### 8.2 Splenic Infarction

| Parameter      | Findings                                                                  |
| -------------- | ------------------------------------------------------------------------- |
| **Morphology** | **Wedge-shaped**, peripheral, apex toward hilum                           |
| **Acute**      | Ill-defined hypoechoic                                                    |
| **Chronic**    | Hyperechoic (fibrosis), volume loss, ± cystic/calcified                   |
| **Doppler**    | No flow in affected area                                                  |
| **Causes**     | Cardiogenic embolism (AF, IE), sickle cell, vasculitis, post-embolization |

### 8.3 Splenic Cyst

| Parameter        | True Cyst                                  | Pseudocyst                        | Parasitic (Hydatid)                                |
| ---------------- | ------------------------------------------ | --------------------------------- | -------------------------------------------------- |
| **Echogenicity** | Anechoic, thin wall, posterior enhancement | Slightly thicker, internal debris | Multiloculated, daughter cysts, wall calcification |
| **Doppler**      | No flow                                    | No flow                           | No flow                                            |
| **Note**         | Benign, common                             | Post-traumatic/infarct            | Endemic area                                       |

### 8.4 Splenic Abscess

| Parameter      | Findings                                                          |
| -------------- | ----------------------------------------------------------------- |
| **Morphology** | Single/multiple hypoechoic to anechoic lesions, irregular margins |
| **Internal**   | ± Gas (hyperechoic + dirty shadowing)                             |
| **Doppler**    | Peripheral hyperemia, no central flow                             |
| **Clinical**   | Fever, LUQ pain; endocarditis, immunocompromised, DM              |

### 8.5 Splenic Lymphoma / Metastases

| Parameter        | Findings                                                                                        |
| ---------------- | ----------------------------------------------------------------------------------------------- |
| **Patterns**     | Diffuse homogeneous enlargement (no discrete nodules); milial pattern; solitary hypoechoic mass |
| **Echogenicity** | Hypoechoic (lymphoma); target sign (mets)                                                       |
| **Doppler**      | Lymphoma: hypovascular; melanoma mets: hypervascular                                            |
| **Note**         | Diffuse infiltration easily missed on US; CT/PET better                                         |

### 8.6 Splenic Trauma

| Parameter                | Findings                                               |
| ------------------------ | ------------------------------------------------------ |
| **FAST exam**            | Free fluid in splenorenal recess                       |
| **Laceration**           | Irregular hypoechoic/heterogeneous cleft               |
| **Subcapsular hematoma** | Crescentic avascular collection compressing parenchyma |
| **Parenchymal injury**   | Heterogeneous echotexture                              |

## 9. Peritoneal Cavity

### 9.1 Ascites

| Parameter         | Transudate                  | Exudate                             | Hemorrhagic                                     | Chylous                                 |
| ----------------- | --------------------------- | ----------------------------------- | ----------------------------------------------- | --------------------------------------- |
| **Echogenicity**  | **Anechoic**, free-flowing  | ± Debris, septations, loculated     | Homogeneous hypoechoic (blood) or mixed (clots) | Homogeneous hypoechoic, may mimic solid |
| **Doppler**       | No flow                     | No flow                             | No flow                                         | No flow                                 |
| **Common causes** | Portal HTN, hypoalbuminemia | Infection, malignancy, pancreatitis | Trauma, ruptured aneurysm                       | Lymphatic obstruction                   |

| Volume       | Locations                                                    |
| ------------ | ------------------------------------------------------------ |
| **Minimal**  | Morison's pouch (hepatorenal) only                           |
| **Moderate** | Pelvis + hepatorenal recess                                  |
| **Large**    | Entire cavity + floating bowel loops; check for omental cake |

### 9.2 Peritoneal Carcinomatosis

| Parameter              | Findings                                                                                     |
| ---------------------- | -------------------------------------------------------------------------------------------- |
| **Omental cake**       | Omental thickening as **hypoechoic layered sheet** (>5 mm) between abdominal wall and bowel  |
| **Peritoneal nodules** | Irregular hypoechoic nodules or plaque-like thickening (>3 mm) on peritoneal surfaces        |
| **Locations**          | Douglas pouch, mesenteric root, liver surface (sub-Glisson), splenic hilum, right subphrenic |
| **Ascites**            | Commonly present (complex, septated)                                                         |
| **Doppler**            | Increased flow in omental/peritoneal disease                                                 |
| **Primaries**          | Ovarian, gastric, colon, pancreatic                                                          |
| **Note**               | US moderate sensitivity; CT/PET more sensitive                                               |

### 9.3 Pseudomyxoma Peritonei

| Parameter        | Findings                                                                          |
| ---------------- | --------------------------------------------------------------------------------- |
| **Morphology**   | Multiple cystic/septated collections; **scalloping** of liver and spleen surfaces |
| **Echogenicity** | Hypoechoic mucin, ± punctate debris                                               |
| **Doppler**      | No flow                                                                           |
| **Origin**       | Ruptured appendiceal mucinous cystadenoma (most common); ovarian mucinous tumors  |
| **Treatment**    | Cytoreductive surgery + HIPEC                                                     |

### 9.4 Intra-abdominal Abscess

| Parameter      | Findings                                                                      |
| -------------- | ----------------------------------------------------------------------------- |
| **Morphology** | Round/oval or irregular hypoechoic to anechoic area, **thick irregular wall** |
| **Internal**   | ± Gas (hyperechoic + dirty shadowing), debris, septations                     |
| **Locations**  | Subhepatic, subphrenic, pelvic (Douglas pouch), intermesenteric               |
| **Doppler**    | Peripheral **hyperemic rim**, no central flow                                 |
| **Clinical**   | Post-op fever, pain, elevated WBC; US/CT-guided drainage                      |

### 9.5 Peritonitis

| Parameter          | Findings                                  |
| ------------------ | ----------------------------------------- |
| **Fluid**          | Free fluid (may be echogenic with debris) |
| **Bowel**          | Ileus: dilated, non-peristaltic loops     |
| **Peritoneum**     | Thickening and hyperemia                  |
| **Abdominal wall** | Edema                                     |

## 10. Abdominal Vasculature

### 10.1 Abdominal Aortic Aneurysm (AAA)

| Parameter      | Findings                                                                              |
| -------------- | ------------------------------------------------------------------------------------- |
| **Definition** | Max aortic diameter **≥3 cm**                                                         |
| **Morphology** | Fusiform or saccular                                                                  |
| **Wall**       | Calcification (hyperechoic + shadow); mural thrombus (hypoechoic intraluminal lining) |
| **Doppler**    | **Turbulent flow**; no flow in mural thrombus; **yin-yang sign** if pseudoaneurysm    |
| **Rupture**    | Retroperitoneal hypoechoic hematoma, irregular contour                                |
| **Screening**  | One-time US: men >65 with smoking history                                             |
| **Note**       | Measure **outer-to-outer**                                                            |

### 10.2 Aortic Dissection (Abdominal Extension)

| Parameter               | Findings                                                                  |
| ----------------------- | ------------------------------------------------------------------------- |
| **Intimal flap**        | Thin linear hyperechoic structure oscillating with cardiac cycle          |
| **True vs false lumen** | True: compressed; False: larger; color Doppler differentiates             |
| **Entry tear**          | Intimal discontinuity proximally                                          |
| **Doppler**             | True: high-velocity arterial; False: low-velocity, delayed, ± thrombus    |
| **Note**                | Most originate in thoracic aorta; US shows only extension; CT aortography |

## 11. Summary of Key Ultrasound Signs

| Sign                                         | Description                                                        | Disease                           |
| -------------------------------------------- | ------------------------------------------------------------------ | --------------------------------- |
| **Starry sky**                               | Bright periportal tracking on hypoechoic background                | Acute hepatitis                   |
| **Double-barrel shotgun / parallel channel** | Dilated bile duct parallel to portal vein                          | Biliary obstruction               |
| **WES triad**                                | Wall-Echo-Shadow in gallbladder                                    | Stone-filled GB                   |
| **Sonographic Murphy sign**                  | Maximal tenderness at GB under transducer                          | Acute cholecystitis               |
| **Pseudokidney / target sign**               | Hypoechoic rim + echogenic center                                  | Carcinoma, intussusception        |
| **Double duct sign**                         | CBD + pancreatic duct both dilated                                 | Pancreatic head mass              |
| **FAST free fluid**                          | Anechoic fluid in Morison's pouch / splenorenal recess             | Trauma, hemoperitoneum            |
| **Spoke-wheel Doppler**                      | Radiating arteries from central vessel                             | Focal nodular hyperplasia         |
| **Basket pattern Doppler**                   | Peripheral arterial vessels around nodule                          | HCC                               |
| **Yin-yang sign (Doppler)**                  | Bidirectional turbulent flow                                       | Pseudoaneurysm                    |
| **Omental cake**                             | Echogenic thickened omentum                                        | Peritoneal carcinomatosis         |
| **Portal venous gas**                        | Punctate hyperechoic foci in peripheral portal vessels             | Bowel ischemia (critical)         |
| **Keyboard sign**                            | Transverse mucosal folds across bowel lumen (valvulae conniventes) | Small bowel obstruction           |
| **Cavernous transformation**                 | Network of collaterals replacing portal vein                       | Portal vein thrombosis            |
| **Comet-tail artifact**                      | Ring-down from cholesterol crystals in GB wall                     | Adenomyomatosis                   |
| **Halo sign (liver)**                        | Hypoechoic rim around focal nodule                                 | HCC (specific)                    |
| **Target / bull's-eye (liver)**              | Central hyperechoic + peripheral hypoechoic rim                    | Liver metastasis / fungal abscess |
