## 標準檢查流程

### 探頭與擺位

- **線性探頭（Linear probe）**：7–15 MHz，高頻率解析表淺結構
- **擺位**：患者平躺，頸部輕度伸展（枕頭墊肩），必要時側頭檢查外側區
- **彩色都卜勒（Color Doppler）**：評估實質及結節血流

### 掃描範圍

1. **甲狀腺本體**：雙葉（縱切＋橫切）＋峽部
2. **頸部淋巴結**：Level I–VII（以甲狀腺癌患者為重點）
3. **周圍軟組織**：氣管、食道、頸動鞘、唾液腺（必要時）

## 正常參考值

| 參數                  | 正常範圍                                                  |
| --------------------- | --------------------------------------------------------- |
| **Right lobe**        | L 4.0–6.0 × W 1.5–2.0 × H 1.0–2.0 cm                      |
| **Left lobe**         | L 4.0–6.0 × W 1.5–2.0 × H 1.0–2.0 cm                      |
| **Isthmus thickness** | <0.5 cm                                                   |
| **Total volume**      | 男 <19 mL, 女 <14 mL（每葉體積 = L × W × H × 0.524）      |
| **Echogenicity**      | 高於或等於周圍肌肉（sternothyroid / sternocleidomastoid） |

## 結節報告

| 項目              | 描述選項                                                                     |
| ----------------- | ---------------------------------------------------------------------------- |
| **Location**      | 右葉／左葉／峽部，上極／中極／下極                                           |
| **Size**          | 三個徑向（L × W × H），記錄最大徑                                            |
| **Shape**         | Oval (wider-than-tall) / Round / Taller-than-wide / Irregular                |
| **Margin**        | Smooth / Ill-defined / Lobulated / Irregular / Extrathyroidal extension      |
| **Echogenicity**  | Isoechoic / Hypoechoic / Hyperechoic / Marked hypoechoic / Mixed             |
| **Composition**   | Solid / Cystic / Mixed (solid or cystic predominant) / Spongiform            |
| **Calcification** | None / Microcalcifications / Macrocalcifications / Peripheral rim / Punctate |
| **Halo**          | None / Thin (<2mm) / Thick (≥2mm) / Irregular                                |
| **Vascularity**   | None / Peripheral / Central / Mixed                                          |
| **TI-RADS**       | TR1–TR5                                                                      |

## TI-RADS 分級參考

| 分級    | 類別                  | 惡性風險 | 建議             |
| ------- | --------------------- | -------- | ---------------- |
| **TR1** | Benign                | <2%      | 無須 FNA         |
| **TR2** | Not suspicious        | <2%      | 無須 FNA         |
| **TR3** | Mildly suspicious     | <5%      | ≥2.5 cm 考慮 FNA |
| **TR4** | Moderately suspicious | 5–20%    | ≥1.5 cm 考慮 FNA |
| **TR5** | Highly suspicious     | >20%     | ≥1 cm 考慮 FNA   |

## 甲狀腺炎超音波特徵回顧

| Type                  | Echogenicity        | Texture                     | Vascularity                       | Special signs                     |
| --------------------- | ------------------- | --------------------------- | --------------------------------- | --------------------------------- |
| **Hashimoto's**       | Markedly hypoechoic | Heterogeneous, micronodular | Normal–slightly ↑                 | Fibrotic bands, pseudonodules     |
| **De Quervain's**     | Hypoechoic (patchy) | Patchy/geographic           | Normal–↓                          | Ill-defined hypoechoic areas      |
| **Graves'**           | Normal–hypoechoic   | Homogeneous                 | **Markedly ↑ (thyroid inferno)**  | Diffuse enlargement               |
| **Acute suppurative** | Hypoechoic          | Heterogeneous               | Peripheral ↑, central ↓ (abscess) | Abscess cavity, soft tissue edema |
| **Riedel's**          | Hypoechoic          | Structure effaced           | ↓                                 | Invasive, tracheal compression    |

## 注意事項

::: warning 本工具為報告模板參考
此模板提供結構化指引，不應取代完整的超音波檢查與臨床判斷。最終報告需由執行檢查之醫師根據實際影像所見判斷並簽署。
:::

::: info TI-RADS 需完整計分
TI-RADS 分級應依據 ACR TI-RADS 計分系統：組成、回音度、形狀、邊緣、鈣化 5 大項加總後對應分級。本模板提供快速選擇，但仍建議完整計分確認。
:::
