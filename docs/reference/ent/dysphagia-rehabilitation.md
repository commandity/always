# 吞嚥復健 — 吞嚥障礙之評估與治療

**吞嚥障礙之評估與治療 Dysphagia Rehabilitation — Assessment & Management of Swallowing Disorders**

## 概論 Overview

吞嚥障礙（dysphagia）指食物或液體從口腔到胃部運輸過程中任一階段發生困難，可分為**口咽性（oropharyngeal）**及**食道性（esophageal）**。口咽性吞嚥障礙最為常見，與吸入性肺炎、營養不良、脫水、生活品質下降及死亡率增加密切相關。吞嚥復健整合代償策略、行為治療、神經肌肉訓練及手術介入，目標為恢復安全且足夠的經口進食。

## 吞嚥四期 Phases of Swallowing

| 期別                 | 時間       | 關鍵事件                                                  | 神經控制                                 |
| -------------------- | ---------- | --------------------------------------------------------- | ---------------------------------------- |
| **Oral preparatory** | 自主       | 唇閉合→咀嚼→食團形成（bolus formation）                   | CN V (trigeminal), CN VII (facial)       |
| **Oral propulsive**  | 自主       | 舌頭將食團向後推送至口咽→軟顎上抬                         | CN XII (hypoglossal), CN V, CN VII       |
| **Pharyngeal**       | **反射性** | 軟顎關閉→喉部上抬＋前移→聲帶閉合＋會厭反折→UES鬆弛→咽收縮 | CN IX, CN X, medullary swallowing center |
| **Esophageal**       | 反射性     | 食道蠕動→LES鬆弛→食團進入胃部                             | CN X, enteric nervous system             |

## 病因分類 Etiology

| 類別                     | 常見原因                                                                                                                      |
| ------------------------ | ----------------------------------------------------------------------------------------------------------------------------- |
| **Neurogenic**           | Stroke（brainstem/bilateral hemispheric）、Parkinson's disease、dementia、ALS、MS、head injury、cerebral palsy                |
| **Structural**           | 頭頸癌術後（glossectomy、laryngectomy、pharyngectomy）、放療後 fibrosis、cricopharyngeal bar、Zenker's diverticulum、頸椎骨刺 |
| **Neuromuscular**        | Myasthenia gravis、polymyositis/dermatomyositis、muscular dystrophy、sarcopenia                                               |
| **Iatrogenic**           | 插管後聲帶損傷、甲狀腺／頸椎術後喉返神經損傷、鎮靜藥物、抗精神病藥物（EPS）                                                   |
| **Aging (presbyphagia)** | 吞嚥肌肉萎縮、感覺閾值上升、hyolaryngeal elevation 下降、UES compliance↓                                                      |
| **Psychogenic**          | Globus hystericus、phagophobia、functional dysphagia                                                                          |

## 臨床評估 Clinical Evaluation

### 床邊篩檢 Bedside Screening

| 工具                                   | 說明                                              | 備註                          |
| -------------------------------------- | ------------------------------------------------- | ----------------------------- |
| **Water swallow test (3oz / 90mL)**    | 患者連續飲水 90mL，觀察嗆咳、咳嗽、聲音變化       | 敏感度高但特異度有限          |
| **Modified Evan's Blue Dye Test**      | 氣切患者，藍色染劑加入食物→觀察抽吸分泌物是否染色 | 僅用於氣切患者                |
| **Yale Swallow Protocol**              | 簡短問卷＋意識評估＋口服水測試                    | 標準化篩檢，適用 acute stroke |
| **Eating Assessment Tool-10 (EAT-10)** | 10 題自評問卷（0–40）                             | ≥3 → 需進一步評估             |

### Clinical Swallow Evaluation（臨床吞嚥評估，CSE）

- **Oral motor exam**：唇閉合力量、舌頭活動度（前伸、側推、上抬）、軟顎上抬、喉部上抬幅度
- **Trial swallows**：不同濃度（thin liquid, nectar, honey, pudding）及不同體積（3mL, 5mL, 10mL, 杯飲）
- **Signs of aspiration**：吞嚥前／中／後咳嗽、choking、聲音潮濕（wet voice）、血氧飽和度↓≥3%、呼吸窘迫
- **Cervical auscultation**：吞嚥聲音異常（clicking, gurgling）

### 儀器評估 Instrumental Assessment

| 檢查                                                     | 原理                           | 優點                                                         | 限制                                                                     |
| -------------------------------------------------------- | ------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------------------ |
| **VFSS（Videofluoroscopic Swallow Study）**              | X 光即時動態錄影，側面＋前後位 | 完整觀察 oral→pharyngeal→esophageal 三期、精確測量、客觀參數 | 輻射暴露、需放射科、限定性姿勢                                           |
| **FEES（Flexible Endoscopic Evaluation of Swallowing）** | 經鼻內視鏡直視咽部及喉部       | 無輻射、床邊執行、可評估感覺（FEESST）、長期追蹤             | 無法觀察 oral phase 及 esophageal phase、吞嚥白化（white-out）期間無法看 |
| **HRM（High-Resolution Manometry）**                     | 記錄咽部收縮壓力及 UES 鬆弛    | 量化咽收縮力、UES 鬆弛比例、pharyngeal contractile integral  | 無法直接觀察 aspiration、需專業判讀                                      |
| **Pulse oximetry**                                       | 連續血氧監測                   | 簡便、輔助篩檢                                               | 敏感度不理想，不可單獨用來診斷                                           |

### 透視吞嚥關鍵參數 VFSS Key Parameters

| 參數                                   | 正常值                           | 異常意義                                       |
| -------------------------------------- | -------------------------------- | ---------------------------------------------- |
| **Oral transit time (OTT)**            | <1.5s                            | → oral dysphagia（舌頭推進無力）               |
| **Pharyngeal transit time (PTT)**      | <1.0s                            | → pharyngeal dysphagia（咽收縮延遲）           |
| **Pharyngeal delay time (PDT)**        | <0.2s                            | → swallow reflex 啟動延遲（aspiration 高風險） |
| **Hyolaryngeal excursion**             | ≥1.5 cm（女性），≥2.0 cm（男性） | ↓→ airway protection 不足，UES 開啟受限        |
| **Pharyngeal constriction ratio**      | >0.9                             | ↓→ pharyngeal residue                          |
| **UES opening width**                  | ≥0.5 cm（液體），≥0.8 cm（固體） | ↓→ cricopharyngeal dysfunction                 |
| **Penetration-Aspiration Scale (PAS)** | 1（正常）–8（嚴重吸入）          | ≥6 → aspiration                                |

### Penetration-Aspiration Scale (PAS)

| 分數  | 定義                                           |
| ----- | ---------------------------------------------- |
| **1** | 食物/液體未進入氣道                            |
| **2** | 進入氣道，停留在聲帶以上，且被有效清除         |
| **3** | 進入氣道，停留在聲帶以上，未被清除             |
| **4** | 進入氣道，接觸聲帶，且被有效清除               |
| **5** | 進入氣道，接觸聲帶，未被清除                   |
| **6** | 進入氣道，通過聲帶，且被有效清除至聲門上或咽部 |
| **7** | 進入氣道，通過聲帶，儘管嘗試清除仍未清除       |
| **8** | 進入氣道，通過聲帶，無清除嘗試                 |

## 治療策略 Treatment Approaches

### 代償策略—立即改善安全 Compensatory Strategies

| 策略                       | 方法                                                               | 適應症                                                      |
| -------------------------- | ------------------------------------------------------------------ | ----------------------------------------------------------- |
| **Postural modifications** | Chin tuck（低頭）— 保護 airway、擴大 vallecular space              | Pharyngeal delay + aspiration                               |
|                            | Head rotation（轉向患側）— 增加 UES 開啟                           | Unilateral pharyngeal weakness、cricopharyngeal dysfunction |
|                            | Head tilt（側傾健側）— 引導食團至健側                              | Unilateral oral/pharyngeal weakness                         |
|                            | Effortful swallow（用力吞）— 增加咽收縮力                          | Pharyngeal residue、UES 開啟不足                            |
|                            | Supraglottic swallow（閉氣吞）— 吞前閉氣→吞後立即咳嗽              | Reduced laryngeal closure                                   |
|                            | Super-supraglottic swallow（用力閉氣吞）— 更用力閉氣＋下壓會厭     | Reduced airway protection                                   |
| **Diet modification**      | 增稠液體（nectar/honey/pudding thick）— 降低流速，增加咽部感覺刺激 | Thin liquid aspiration                                      |
|                            | 食物質地調整（IDDSI 分級）— 從 puree 到 soft solid                 | Oral dysphagia、chewing 困難                                |
| **Volume/rate control**    | 小口（3–5mL）、慢速（間隔≥5s）、交替固液                           | Reduced pharyngeal clearance                                |
| **Bolus modification**     | 調整食團溫度（冰水刺激）、酸味（增加感覺輸入）                     | Delayed swallow reflex                                      |

### IDDSI Framework（International Dysphagia Diet Standardisation Initiative）

| 分級  | 名稱              | 描述                   |
| ----- | ----------------- | ---------------------- |
| **0** | Thin              | 如水般流動             |
| **1** | Slightly thick    | 稍稠（nectar-like）    |
| **2** | Mildly thick      | 中度稠（honey-like）   |
| **3** | Moderately thick  | 高度稠（pudding-like） |
| **4** | Extremely thick   | 極稠／泥狀             |
| **5** | Minced & moist    | 絞碎潮濕               |
| **6** | Soft & bite-sized | 柔軟一口大小           |
| **7** | Regular           | 一般質地               |

### 復健訓練 Rehabilitation Exercises

#### 口腔期運動 Oral Phase Exercises

| 訓練                       | 方法                               | 目標                                          |
| -------------------------- | ---------------------------------- | --------------------------------------------- |
| **Lip closure**            | 抿壓舌板、撅嘴微笑交替、straw 阻力 | 改善唇閉合→減少 drooling 及 anterior spillage |
| **Tongue range of motion** | 舌前伸／後縮／側推／上顎上抬       | 改善 bolus propulsion 及控制                  |
| **Tongue resistance**      | 舌頂壓舌板（前方、側方、上方）     | 增強舌頭力量                                  |
| **Chewing**                | 咀嚼橡皮管或特殊咀嚼器             | 改善咀嚼效能                                  |
| **Bolus hold**             | 口腔含食團不流入咽（hold command） | 改善 oral containment                         |

#### 咽期運動 Pharyngeal Phase Exercises

| 訓練                                              | 方法                                                       | 目標                                           |
| ------------------------------------------------- | ---------------------------------------------------------- | ---------------------------------------------- |
| **Mendelsohn maneuver**                           | 吞嚥時喉部上抬至最高點後維持 2–3s 再放鬆                   | 延長 UES 開啟時間、改善 hyolaryngeal elevation |
| **Effortful swallow**                             | 用力吞嚥（「把口水吞到腳底」）                             | 增強咽收縮力、減少 pharyngeal residue          |
| **Masako maneuver (tongue hold)**                 | 舌頭輕咬在前齒間後吞嚥                                     | 增強咽縮肌（pharyngeal constrictor）收縮       |
| **Shaker exercise (head lift)**                   | 躺臥，抬頭看腳趾（不聳肩），維持 30–60s，重複 3 次         | 增強 hyolaryngeal elevation、UES 開啟          |
| **Chin tuck against resistance (CTAR)**           | 坐姿，下巴用力內收抵抗手部阻力 30s                         | 替代 Shaker 訓練（較不傷頸椎）                 |
| **Expiratory muscle strength training (EMST)**    | 使用 EMST 裝置（PEP threshold trainer），每日 25 回 × 5 組 | 增強呼氣肌群 + hyolaryngeal complex 活化       |
| **Respiratory-swallow coordination**              | 練習吞嚥後立即呼氣→咳嗽 secures airway                     | 改善 aspiration 患者的喉部保護                 |
| **Semi-occluded vocal tract (SOVT) with swallow** | 使用 straw 或 Lax Vox 在水中吹泡後吞嚥                     | 促進喉部上抬及閉合協調                         |

#### 感覺刺激 Sensory Stimulation Techniques

| 技術                                            | 方法                                            | 適應症                         |
| ----------------------------------------------- | ----------------------------------------------- | ------------------------------ |
| **Thermal-tactile stimulation (TTS)**           | 冰喉鏡刺激 anterior faucial arch → 立即誘發吞嚥 | Delayed swallow reflex         |
| **Sour bolus**                                  | 檸檬味／酸味刺激（增加感覺輸入）                | Reduced pharyngeal sensation   |
| **Carbonated liquids**                          | 碳酸飲料增加三叉神經感覺輸入                    | Neurogenic dysphagia           |
| **Taste-enhanced bolus**                        | 明顯味道（如鹽、甜、苦）強化                    | Dementia 伴 dysphagia          |
| **Neuromuscular electrical stimulation (NMES)** | 表面電極置於 hyolaryngeal區域，感應性收縮刺激   | Reduced hyolaryngeal elevation |

### 手術及程序介入 Surgical & Procedural Interventions

| 術式                                  | 適應症                                                | 說明                                                                  |
| ------------------------------------- | ----------------------------------------------------- | --------------------------------------------------------------------- |
| **Cricopharyngeal myotomy**           | 孤立性 UES 弛緩不能／CP bar                           | 內視鏡（CO₂ laser / stapler）或開放手術                               |
| **Dilatation**                        | Cricopharyngeal stricture、radiation-induced stenosis | 球囊擴張（balloon dilation）或 Maloney 探條；漸進式增徑               |
| **Injection laryngoplasty**           | UVFP 伴 aspiration                                    | HA / fat / calcium hydroxylapatite 注射至患側聲帶                     |
| **Medialization laryngoplasty**       | UVFP 伴持續性 aspiration                              | Thyroplasty type I（Silastic 或 Gore-Tex）                            |
| **Laryngeal closure (partial/total)** | 嚴重 aspiration 且保守治療無效                        | Epiglottic flap、narrowed laryngeal inlet、laryngotracheal separation |
| **Tracheostomy with cuff**            | 嚴重 aspiration 需保護氣道                            | 提供密封氣道，但無法預防 supraglottic aspiration                      |
| **Laryngectomy (total)**              | 不可逆的致命性 aspiration                             | 永久性咽喉分離，終止 aspiration                                       |
| **PEG / PEJ（經皮胃造口）**           | 無法安全經口進食達營養需求                            | 長期腸道營養支持                                                      |

## 常見疾病與治療方案 Common Conditions & Specific Protocols

### 1. 中風後吞嚥障礙 Post-Stroke Dysphagia

- **發生率**：急性中風約 37–78%，其中 50% 在 6 個月內恢復
- **危險因子**：brainstem stroke（尤其是 medullary lateral syndrome / Wallenberg syndrome）、bilateral hemispheric stroke、NIHSS >10、年齡 >70
- **關鍵評估**：VFSS 或 FEES 確認 aspiration（silent aspiration 約 40%）
- **治療**：
  - Acute phase：NPO + compensatory strategies → 每日床邊篩檢
  - Recovery phase：NMES + 吞嚥訓練（Shaker / Masako / effortful swallow）→ 漸進式飲食升級（puree → soft solid → regular）
  - 若 6 個月未恢復 → 長期代償策略 ± PEG
- **預後**：post-stroke dysphagia 與 aspiration pneumonia（RR 3.2）、死亡率（OR 2.4）顯著相關

### 2. 帕金森氏症吞嚥障礙 Parkinson's Disease Dysphagia

- **發生率**：約 80%（disease duration-dependent），可發生於任何 Hoehn & Yahr stage
- **特徵**：舌頭震顫、重複性吞嚥運動減少（hypokinetic swallow）、pharyngeal residue、piecemeal deglutition、aspiration（>50% silent）
- **治療**：
  - **EMST**（Expiratory muscle strength training）— Level I evidence：改善 hyolaryngeal elevation、減少 aspiration
  - **Lee Silverman Voice Treatment (LSVT)** — 間接改善吞嚥（聲帶閉合＋呼吸支持）
  - **NMES** — 部分研究顯示有效
  - **Medication timing**：調整 levodopa timing → 在 on-phase 進食
  - **深腦刺激術（DBS）**：對吞嚥效果不一致，部分患者可能有負面影響
- **Monitor**：定期 FEES（因 silent aspiration 比例高）+ 營養評估 + 肺炎預防

### 3. 頭頸癌吞嚥障礙 Head & Neck Cancer Dysphagia

- **成因**：腫瘤本身（結構破壞）、手術（切除範圍）、放療（纖維化、唾液減少、trismus）、化療（黏膜炎）
- **特色**：multi-factorial — oral phase（舌缺損、trismus）+ pharyngeal phase（咽縮肌 fibrosis、laryngeal edema）+ UES（fibrosis stenosis）
- **Pre-treatment baseline**：VFSS + 吞嚥評估 — 非常重要（提供術／放療前參考基準）
- **治療**：
  - **Prophylactic swallowing exercises**：放療前及放療期間進行維持性吞嚥練習（可減少 fibrosis）
  - **Post-surgery**：依切除範圍設計客製化復健方案（如 glossectomy → 舌頭重建訓練）
  - **Post-RT**：hyaluronic acid injection for trismus、balloon dilatation for stricture、NMES
  - **IDDSI diet progression**：依功能逐步升級
- **長期追蹤**：dysphagia 可持續數年（RT 後 fibrosis 持續進展—late effect）

### 4. 老年吞嚥變化 Presbyphagia

- **生理改變**：肌肉萎縮（sarcopenia）、感覺閾值↑、hyolaryngeal excursion↓、UES compliance↓、唾液分泌↓
- **特徵**：吞嚥速度變慢、pharyngeal residue 比例↑、silent aspiration 風險↑
- **治療**：
  - **CTAR / Shaker** — 維持 hyolaryngeal elevation
  - **EMST** — 改善 cough 強度及 airway protection
  - **Resistance training** — 舌頭阻力訓練（IOPI device）
  - 營養補充（蛋白質 + vitamin D）— 改善 sarcopenia
  - 環境調整：充足時間、減少 distractions

### 5. 環咽肌功能障礙 Cricopharyngeal Dysfunction

- **生理**：UES（upper esophageal sphincter）由 cricopharyngeus 肌肉形成，平時保持張力性收縮，吞嚥時完全鬆弛至少 0.5–1.0s
- **異常形式**：
  - **不鬆弛（achalasia / CP bar）**：UES opening ↓→ pharyngeal residue ↑→ aspiration
  - **提早閉合（premature closure）**：類似不鬆弛表現
  - **鬆弛不全（incomplete relaxation）**：高解析度壓力計可見 UES residual pressure↑
- **治療**：
  - 行為：Mendelsohn maneuver、Masako maneuver、Shaker / CTAR
  - 擴張：balloon dilatation（逐步增加直徑至 18–20mm）
  - 注射：BTX-A（肉毒桿菌素）注射至 cricopharyngeus（100U Botox / 50U Dysport），效果持續 3–6 個月
  - 手術：Cricopharyngeal myotomy（經口或開放式）

### 6. 失智症相關吞嚥障礙 Dementia-Related Dysphagia

- **特徵**：progressively worsening、agnosia（無法辨識食物）、apraxia（無法執行吞嚥動作）、oral phase 受損明顯、late stage → pharyngeal phase 受損
- **治療原則**：
  - 早期：環境安靜、減少 distractions、手口提示（hand-over-hand feeding）
  - 中期：食物質地調整（IDDSI）、姿勢代償、訓練照護者餵食技巧
  - 晚期：comfort feeding、PEG 討論（倫理考量→需權衡生活品質）
  - 避免：強迫進食、behavioral dysphagia 惡化

## 療效評估 Outcome Measures

| 工具                                                    | 頻率        | 目標                                                    |
| ------------------------------------------------------- | ----------- | ------------------------------------------------------- |
| **PAS (VFSS/FEES)**                                     | 每 4–12 週  | 降低吸入分數至 ≤3                                       |
| **FOIS (Functional Oral Intake Scale)**                 | 每次追蹤    | 提升進食等級（1–7）                                     |
| **EAT-10**                                              | 每次追蹤    | 總分降低（minimal clinically important difference: ≥3） |
| **MBSImP (Modified Barium Swallow Impairment Profile)** | 治療前後    | 逐項 impairment reduction                               |
| **SWAL-QOL**                                            | 治療前後    | 生活品質改善                                            |
| **體重 + BMI**                                          | 每月        | 維持或增加體重                                          |
| **肺炎發生率**                                          | 每 3–6 個月 | aspiration pneumonia 減少                               |

### Functional Oral Intake Scale (FOIS)

| 等級  | 定義                                         |
| ----- | -------------------------------------------- |
| **1** | NPO（完全無法經口進食）                      |
| **2** | 管灌為主＋極少量液體或食物嘗試               |
| **3** | 管灌＋穩定經口進食（單一質地）               |
| **4** | 完全經口進食（單一質地）                     |
| **5** | 完全經口進食（多種質地，但需特殊準備）       |
| **6** | 完全經口進食（無特殊準備，但需避免特定食物） |
| **7** | 完全經口進食（無限制）                       |

## 特殊族群 Special Populations

| 族群                                | 考量                                                                                                                                                |
| ----------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Pediatric**                       | 先天結構異常（cleft palate、laryngomalacia）、neurologic（CP、genetic syndromes）、口腔感覺防禦（sensory-based feeding disorder）、需成長與發展評估 |
| **Critical care / Post-extubation** | 插管後喉部損傷、ICU-acquired weakness、delirium、呼吸吞嚥協調失調；拔管後 24h 內篩檢                                                                |
| **Progressive neurologic**          | ALS、MS、Huntington's、PSP → 需前瞻性計畫：early→compensatory→alternative feeding                                                                   |
| **Dementia**                        | 人權倫理議題：comfort feeding vs PEG、避免 physical restraint、尊重患者意願                                                                         |
| **Post-laryngectomy**               | Total laryngectomy → 咽部重建後可能仍有 aspiration（neoglottic incompetence）、需 TEP 復健                                                          |
