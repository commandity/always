# 甲狀腺腫瘤

**Thyroid Tumors**

## 概述

甲狀腺結節極為常見（超音波盛行率 19–68%），其中惡性僅佔 5–15%。臨床評估核心在於**精確鑑別良惡性**，避免不必要手術。主要分類依**細胞起源**與**分化程度**：

| 分類                                   | 起源細胞                 | 典型特徵                   |
| -------------------------------------- | ------------------------ | -------------------------- |
| **Benign: Follicular Adenoma**         | 濾泡細胞（分化）         | 有包膜、無浸潤             |
| **Papillary Thyroid Carcinoma (PTC)**  | 濾泡細胞（分化）         | 最常見惡性、淋巴轉移       |
| **Follicular Thyroid Carcinoma (FTC)** | 濾泡細胞（分化）         | 包膜／血管浸潤、血行轉移   |
| **Hürthle Cell Carcinoma (HCC)**       | 嗜酸性濾泡細胞（分化）   | FTC 亞型、RAI 吸收差       |
| **Poorly Differentiated (PDTC)**       | 濾泡細胞（去分化中間型） | 侵襲性中等                 |
| **Anaplastic Thyroid Carcinoma (ATC)** | 濾泡細胞（去分化）       | 極速生長、預後極差         |
| **Medullary Thyroid Carcinoma (MTC)**  | C 細胞（旁濾泡細胞）     | 降鈣素↑、常為遺傳性        |
| **Thyroid Lymphoma**                   | 淋巴球（B 細胞為主）     | 常合併 Hashimoto's         |
| **Metastatic Tumors**                  | 他處惡性轉移             | 腎細胞癌、肺癌、乳癌最常見 |

::: tip 判讀要點
本流程用於**細胞學無法定性**時，以血清／分子標記輔助鑑別：**Calcitonin ↑↑** → **MTC**（C 細胞來源，查 CEA、RET）。Calcitonin 正常者以 **Anti-TPO** 分流：**陽性**且 Hashimoto's 背景合併快速腫大 → 疑 **甲狀腺淋巴瘤**（LDH↑、假囊狀超音波、流式細胞術確診）；**陰性**者查 **BRAF V600E**：**陽性 → PTC**、**陰性 → FTC／FA**（濾泡型病灶良惡性須靠**術後病理**判斷包膜／血管侵犯，分子檢測無法區分）。
:::

## 1. 濾泡性腺瘤
**Follicular Adenoma**

### 簡介

最常見的**甲狀腺良性腫瘤**，具有完整包膜包覆的濾泡細胞增生，**無包膜或血管浸潤**。好發於**30–50 歲女性**，女：男 ≈ 3:1。依組織學分為正常濾泡型（normofollicular）、大濾泡型（macrofollicular）、小濾泡型（microfollicular）及實心型（solid/trabecular）。

**臨床重要性**：細胞學無法與濾泡癌區分（兩者均為「濾泡性病變」），**需手術切除後病理確認有無包膜／血管浸潤**，是良惡性鑑別的金標準。

### 臨床表現

- 通常**無症狀**，於常規檢查或自我觸診發現
- 甲狀腺單一結節，境界清楚，質地柔軟至稍硬
- 甲狀腺功能通常正常（極少數為 autonomous adenoma → 甲狀腺毒症）
- 少數大腺瘤可造成吞嚥不適或頸部壓迫感

### 檢查

- 結節境界清楚、表面光滑、質地均質
- **隨吞嚥上下移動**（與甲狀腺組織相連）
- 無淋巴結腫大
- 若為自律性腺瘤（toxic adenoma）：甲狀腺毒症徵象（心跳加速、手抖）

### 實驗室檢查

| 項目                     | 結果         |
| ------------------------ | ------------ |
| **TSH**                  | 正常（多數） |
| **Free T4 / T3**         | 正常         |
| **Anti-TPO / Anti-Tg**   | 陰性         |
| **Thyroglobulin (Tg)**   | 正常∼輕微↑   |
| **降鈣素（Calcitonin）** | 正常         |

> 若 TSH ↓ → 考慮自律性腺瘤（toxic adenoma）→ 甲狀腺核醫掃描：hot nodule（熱結節）

### 影像學特徵

- **結節特徵**：
  - **形狀**：圓形或橢圓形（wider-than-tall）
  - **邊界**：清楚，有完整**薄包膜（halo sign）** — 低回音暈環，代表包膜及周圍血管
  - **回音**：等回音至高回音（相對甲狀腺實質）
  - **質地**：均質（homogeneous）
  - **鈣化**：粗鈣化（coarse / eggshell calcification）> 微鈣化（若有微鈣化則警惕）
  - **彩色都卜勒**：周邊血流（perinodular flow）為主，中心血流少 — 形成**rim vascularity（環狀血流）**
  - **超音波彈性**：柔軟（soft/low stiffness）
- **TI-RADS 分類**：TR1–TR3（多數為 TR2–TR3）

**Halo sign**（低回音暈）是良性腺瘤的重要特徵，但並非100%特異性（約 10–15% 的濾泡癌亦有 halo）。

### 細針穿刺細胞學 FNAC

- **Bethesda 分類**：最常見 **Bethesda III**（AUS/FLUS）或 **Bethesda IV**（FN/SFN）
- **Cellularity**：高度
- **Cytology**：
  - **濾泡細胞**排列成微濾泡（microfollicles）、小梁狀（trabecula）或片狀
  - **膠質（colloid）**：稀少或無（≠ benign nodular goiter 的大量膠質）
  - 無核溝（nuclear groove）、無核內假包涵體（intranuclear pseudoinclusion）
  - 無乳突狀叢集
  - 細胞大小均一，核仁不明顯
- **Key limitation**：FNA **無法區分** Follicular Adenoma vs Follicular Carcinoma
  - 診斷需術後病理：有無包膜浸潤（capsular invasion）或血管浸潤（vascular invasion）
- **Molecular markers**（輔助 Bethesda III/IV）：
  - **Afirma GEC / Afirma GSC**、**ThyroSeq v3**：基因表現分類，提高良惡性鑑別準確度
  - **BRAF V600E**（若陽性 → 幾乎確定 PTC）；**RAS 突變**（可見於 FTC 或 FA）

### 治療

1. **活躍監測**（active surveillance）：若 FNA 為 Bethesda II 且臨床低風險
2. **手術切除**：
   - 若 FNA 為 Bethesda III/IV → 腺葉切除（hemithyroidectomy）+ 術中冷凍切片
   - 術後病理確認包膜無浸潤 → 確診 adenoma → 無需進一步治療
3. **Toxic adenoma**：放射碘（RAI）治療或手術

## 2. 乳突狀甲狀腺癌
**Papillary Thyroid Carcinoma，PTC**

### 簡介

**最常見的甲狀腺惡性腫瘤**（約 80–85%），起源於濾泡細胞，分化良好。好發於**30–50 歲女性**（女：男 ≈ 3:1），與**游離輻射暴露**（尤其童年期）高度相關。分子特徵：**BRAF V600E 突變**（約 40–60%）、**RET/PTC 重排**（10–20%）、**TERT 啟動子突變**（與侵襲性相關）。

**預後極佳**：10 年存活率 >95%（低風險者），但少數侵襲性亞型（高細胞型、彌漫硬化型）預後較差。

### 主要亞型

| 亞型                               | 特徵                                           | 預後      |
| ---------------------------------- | ---------------------------------------------- | --------- |
| **Classical PTC**                  | 最常見，乳突結構為主                           | 良好      |
| **Follicular variant (FVPTC)**     | 濾泡結構但具 PTC 核特徵；包裹型（NIFTP）良性化 | 良好∼中等 |
| **Tall cell variant**              | 細胞高度≥2倍寬；BRAF 突變率高                  | 較差      |
| **Diffuse sclerosing variant**     | 瀰漫浸潤、大量鈣化、淋巴血管浸潤               | 較差      |
| **Hobnail variant**                | 細胞呈釘狀（hobnail），去分化傾向              | 差        |
| **Microcarcinoma**                 | ≤1 cm；多數惰性，部分可主動監測                | 極佳      |
| **Papillary microcarcinoma (PMC)** | ≤1 cm；偶發性 vs 臨床顯著型（淋巴轉移、多發）  | 極佳      |

### 臨床表現

- 多數**無症狀**，超音波偶發發現
- 甲狀腺單一結節（solid），質地偏硬
- **頸部淋巴結腫大**（cystic lymph node → 高度懷疑 PTC 轉移）
- 聲音沙啞（喉返神經侵犯）、吞嚥困難（氣管侵犯）→ 局部侵犯徵象
- 甲狀腺功能通常**正常**

### 檢查

- 結節**質地較硬**（firm to hard），境界可不清楚
- 結節固定（adhesion to surrounding tissue）→ 侵犯周圍結構
- **頸部淋巴結**：可觸及，堅硬、無痛（Level VI、II–IV 最常見）
- 若喉返神經受侵：聲帶麻痺 → 聲音沙啞

### 實驗室檢查

| 項目                     | 結果                                |
| ------------------------ | ----------------------------------- |
| **TSH**                  | 正常（重要：高 TSH → 促進腫瘤生長） |
| **Free T4 / T3**         | 正常                                |
| **Thyroglobulin (Tg)**   | 可能↑，但非特異性（術後追蹤用）     |
| **Anti-Tg antibody**     | 若陽性，干擾 Tg 測量（需注意）      |
| **降鈣素（Calcitonin）** | 正常（用於排除 MTC）                |
| **Anti-TPO**             | 可能陽性（若合併 Hashimoto's）      |

> **術後追蹤**：刺激性 Tg（sTg）+ 頸部超音波為主要工具；Tg >1 ng/mL（TSH 抑制狀態）提示殘留或復發

### 影像學特徵

PTC 超音波特徵常可與 TI-RADS 高評分結合記憶：

- **回音**：**低回音（hypoechoic）**，比頸部肌肉更低 → 高度懷疑惡性
- **形狀**：**taller-than-wide（垂直生長）**— 縱橫比（A/T ratio）> 1 → 高度惡性
- **邊界**：**不規則（irregular）**、**毛刺狀（spiculated / microlobulated margins）**
- **鈣化**：**微鈣化（microcalcifications）** — 直徑 <1 mm，點狀強回音無聲影（**最特異性惡性特徵**，特異性 >85%）
- **周邊強化回音（rim calcification with disruption）** → 懷疑包膜侵犯
- **彩色都卜勒**：**中心血流增加（central vascularity）** > 周邊血流
- **超音波彈性（elastography）**：**高硬度（stiff）**
- **Cystic change**：部分 PTC 有囊性成分（cystic nodule with solid mural component → 警惕 PTC）

**特別特徵 — 淋巴結超音波**：

| 特徵            | 轉移性淋巴結（PTC）  | 正常淋巴結      |
| --------------- | -------------------- | --------------- |
| **形狀**        | 圓形（L/S ratio <2） | 橢圓形          |
| **門（hilum）** | 消失                 | 存在（高回音）  |
| **內部回音**    | **微鈣化、囊性變**   | 均質            |
| **血流**        | 周邊血流為主         | 門血流（hilar） |
| **微鈣化**      | 強烈提示 PTC 轉移    | 無              |

|              | 粗鈣化           | 1     |
|              | 周邊（蛋殼）鈣化 | 2     |
|              | **微鈣化**       | **3** |

| TI-RADS 等級 | 總分 | 惡性風險    | FNA 建議        |
| ------------ | ---- | ----------- | --------------- |
| **TR1**      | 0    | 良性        | 無需 FNA        |
| **TR2**      | 2    | 未定（低）  | 無需 FNA        |
| **TR3**      | 3    | 低（<5%）   | ≥2.5 cm → FNA   |
| **TR4**      | 4–6  | 中（5–20%） | ≥1.5 cm → FNA   |
| **TR5**      | ≥7   | 高（>20%）  | **≥1 cm → FNA** |

### 細針穿刺細胞學

- **Bethesda 分類**：**Bethesda V**（惡性可疑）或 **Bethesda VI**（惡性）
- **Cellularity**：高度
- **Cytology — 核特徵（PTC 診斷核心）**：
  - ⭐ **核溝（nuclear grooves）** — 縱向折疊，形成咖啡豆外觀（coffee-bean appearance）
  - ⭐ **核內假包涵體（intranuclear cytoplasmic pseudoinclusions）** — 核內圓形透明空泡
  - **核增大（nuclear enlargement）**，不規則核膜
  - **核染色質稀疏（powdery chromatin）** — 毛玻璃核（ground-glass nuclei）
  - **核重疊（nuclear crowding / overlapping）**
- **架構特徵**：
  - 乳突狀叢集（papillary clusters with fibrovascular cores）
  - 砂粒體（psammoma bodies）— 同心圓狀鈣化，**高度特異性**（約 40–50% PTC 可見）
- **背景（Background）**：多核巨細胞（multinucleated giant cells）、泡沫狀組織球
- **分子輔助**：BRAF V600E 免疫染色或 PCR（FNA 標本可做）

### 腫瘤分期（AJCC 8th，2018）

| Stage   | 條件（<55 歲）     | 條件（≥55 歲）                |
| ------- | ------------------ | ----------------------------- |
| **I**   | 任何 T、任何 N、M0 | T1–T2、N0/NX、M0              |
| **II**  | 任何 T、任何 N、M1 | T1–T2 + N1；或 T3、任何 N、M0 |
| **III** | —                  | T4a、任何 N、M0               |
| **IVA** | —                  | T4b、任何 N、M0               |
| **IVB** | —                  | 任何 T、任何 N、M1            |

> **<55 歲**：無 Stage III/IV（除非遠端轉移）→ 預後考量年齡是關鍵因子

### 治療

1. **手術**（第一線）：
   - **甲狀腺全切除（total thyroidectomy）**：腫瘤 >4 cm、雙葉病灶、腺外浸潤、淋巴轉移、遠端轉移
   - **腺葉切除（hemithyroidectomy）**：單一腫瘤 <4 cm、無淋巴轉移、低風險（ATA low risk）
   - **Central neck dissection（Level VI）**：臨床淋巴轉移者；預防性切除爭議較多
2. **放射碘治療（RAI，I-131）**：
   - 全切後殘留甲狀腺去除（remnant ablation）
   - 適應症：中高風險者（淋巴轉移、腺外浸潤、遠端轉移）
   - 低風險微癌通常**不需** RAI
3. **TSH 抑制治療**：LT4 將 TSH 維持低值（依風險分層）
   - 高風險：TSH <0.1 mIU/L
   - 低風險：TSH 0.5–2.0 mIU/L
4. **體外放射線（EBRT）**：腺外浸潤無法完全切除時
5. **標靶治療**（RAI-refractory 進行性疾病）：
   - **Lenvatinib**（VEGFR 抑制劑）、**Sorafenib** — 一線
   - **BRAF 抑制劑**（dabrafenib + trametinib）：BRAF V600E 突變者

### 預後（ATA 風險分層）

| 風險             | 定義                                              | 10 年復發率 |
| ---------------- | ------------------------------------------------- | ----------- |
| **Low**          | 無腺外浸潤、無血管浸潤、無轉移、R0 切除、良性亞型 | <5%         |
| **Intermediate** | 輕微腺外浸潤、微血管浸潤、cN1、侵襲性亞型         | 5–20%       |
| **High**         | 顯著腺外浸潤、不完全切除、遠端轉移、高 Tg         | >20%        |

## 3. 濾泡狀甲狀腺癌
**Follicular Thyroid Carcinoma，FTC**

### 簡介

第二常見甲狀腺惡性腫瘤（約 10–15%），起源於濾泡細胞。與 PTC 最大不同：**不經淋巴道，而以血行轉移（hematogenous metastasis）為主**，好發於肺、骨、肝。分子特徵：**RAS 突變**（NRAS, HRAS, KRAS；約 40–50%）、**PAX8-PPARγ 融合**（約 30–35%）。好發於**碘缺乏地區**及**年長女性（>50 歲）**。

**分類**：

- **Minimally invasive FTC**：僅包膜浸潤（無或輕微血管浸潤）→ 預後接近良性
- **Widely invasive FTC**：廣泛血管浸潤（≥4 個血管）、鏡外浸潤 → 預後差

### 臨床表現

- 類似 Follicular Adenoma — 臨床上幾乎無法區分
- 單一質地較硬結節，境界清楚（有包膜感）
- 甲狀腺功能正常
- **骨轉移**可為首發表現：背痛、病理性骨折、高鈣血症（罕見）
- **肺轉移**：偶發於影像學

### 實驗室檢查

| 項目                   | 結果             |
| ---------------------- | ---------------- |
| **TSH**                | 正常             |
| **Free T4 / T3**       | 正常             |
| **Thyroglobulin (Tg)** | 可能↑（不特異）  |
| **Anti-TPO**           | 通常陰性         |
| **降鈣素**             | 正常（排除 MTC） |
| **ALP / Ca²⁺**         | 骨轉移時↑        |

### 影像學特徵

- **相對 PTC 惡性特徵較少**，常被低估：
  - **回音**：等回音至低回音（較 PTC 接近等回音）
  - **形狀**：wider-than-tall（多為球形）
  - **邊界**：清楚（厚包膜可見）— 難以與 FA 鑑別
  - **鈣化**：粗鈣化（coarse）> 微鈣化
  - **Halo sign**：可見（但較 FA 更厚、不規則）— **厚而不規則的 halo 提示浸潤**
  - **彩色都卜勒**：中心血流（較 FA 更豐富）
- **TI-RADS**：TR3–TR4（常被低估為 TR3）
- **核醫掃描（Tc-99m scintigraphy）**：cold nodule（冷結節）—與 FA 同樣為冷結節

> ⚠️ **FTC 超音波表現接近良性，唯術後病理可確診（包膜/血管浸潤）**

### 細針穿刺細胞學

- **Bethesda 分類**：**Bethesda IV**（FN/SFN）— 無法與 FA 區分
- **Cytology**：
  - 微濾泡（microfollicles）排列為主
  - 膠質稀少
  - 細胞形態與 FA 幾乎相同
  - **缺乏 PTC 核特徵**（無核溝、無假包涵體）
- **Molecular testing**（Bethesda III/IV 者建議）：
  - **RAS 突變** → 惡性可能增加（但亦見於 FA）
  - **PAX8-PPARγ 融合** → 高度提示 FTC
  - **ThyroSeq v3**：基因 panel，sensitivity ~94%, specificity ~82%
- **診斷金標準**：手術切除標本，確認**包膜浸潤**或**血管浸潤**

### 治療

1. **手術**：甲狀腺全切除（全切術為首選，minimally invasive 者可考慮腺葉切除）
2. **RAI（I-131）**：FTC 通常保留較佳 RAI 攝取能力 → RAI ablation 為標準
3. **TSH 抑制**：LT4
4. **骨轉移**：骨科手術固定、雙磷酸鹽、EBRT
5. **廣泛浸潤型進行性 FTC**：Lenvatinib、Sorafenib

## 4. 嗜酸性細胞癌
**Hürthle Cell Carcinoma，HCC**

### 簡介

WHO 2022 已將其獨立分類（非 FTC 亞型），舊稱 Hürthle cell variant of FTC。起源於**嗜酸性（oncocytic）濾泡細胞**，因粒線體堆積而形成大量嗜酸性顆粒狀細胞質。**定義**：>75% 嗜酸性細胞組成，有包膜或血管浸潤。

**特點**：

- **RAI 攝取極低**（粒線體取代甲狀腺功能組件）→ RAI 治療效果差
- **淋巴轉移**比 FTC 更常見（類似 PTC）
- **雙側性**較 FTC 多

### 影像學特徵

- 等回音至高回音（較其他惡性腫瘤回音高）
- 邊界：清楚，厚包膜
- 彩色都卜勒：豐富血流（較 FA 更顯著）
- TI-RADS：TR3–TR4

### 細針穿刺細胞學

- **Bethesda IV**（Hürthle cell neoplasm）
- **Cytology**：
  - **嗜酸性顆粒狀細胞質（oxyphilic / oncocytic cytoplasm）**，豐富
  - 核大、**核仁顯著（prominent nucleolus）**
  - 低黏附性（discohesive）
  - 膠質稀少
  - **難以與 Hürthle cell adenoma（良性）區分** → 需手術
- 注意：Hashimoto's 之 Hürthle cells 為反應性，結合臨床脈絡鑑別

## 5. 未分化甲狀腺癌
**Anaplastic Thyroid Carcinoma，ATC**

### 簡介

最致命的甲狀腺惡性腫瘤，佔所有甲狀腺癌 1–2%，但佔甲狀腺癌死亡約 50%。幾乎**全部為 Stage IV A–C**（WHO 定義）。好發於**>60 歲**（平均 65–70 歲）。常由 PTC 或 FTC **去分化**演變而來（可見 TP53 突變、TERT 啟動子突變疊加）。

**中位存活期僅 3–5 個月**，積極多模式治療（手術＋放化療）可延長部分患者存活。

### 臨床表現

- **頸部巨大腫塊快速生長**（數週至數月）
- **壓迫三聯徵**：吞嚥困難（dysphagia）、聲音沙啞（hoarseness）、呼吸困難（dyspnea）/ 喘鳴（stridor）
- 皮膚侵犯：皮膚紅腫、潰瘍
- 發燒、體重快速減輕、全身虛弱
- 遠端轉移症狀（肺轉移→咳嗽；骨轉移→骨痛）

### 檢查

- **甲狀腺巨大、石頭樣硬、固定不動**（adherent to trachea/esophagus）
- 皮膚靜脈怒張（頸靜脈阻塞）
- 聲帶麻痺（喉鏡確認）
- 頸部廣泛淋巴結腫大、固定

### 實驗室檢查

| 項目             | 結果                 |
| ---------------- | -------------------- |
| **TSH**          | 通常正常（晚期可低） |
| **Free T4 / T3** | 通常正常             |
| **Tg**           | 可能↑，但非特異      |
| **ESR / CRP**    | 顯著↑↑（類似感染）   |
| **WBC**          | 可能↑（副腫瘤效應）  |
| **LDH / ALP**    | 可能↑（遠端轉移）    |
| **降鈣素**       | 正常                 |

### 影像學特徵

- **巨大低回音腫塊**，境界**極不規則**，浸潤甲狀腺外組織
- **邊界完全消失**（甲狀腺正常解剖結構破壞）
- **廣泛浸潤**：氣管、食道、頸靜脈、頸動脈鞘
- **壞死（necrosis）**：大面積無回音或混合回音（腫瘤中心壞死）
- **鈣化**：粗鈣化、不規則（前驅 PTC/FTC 遺留）
- **彩色都卜勒**：周邊血流豐富，中心壞死區無血流
- **淋巴結**：廣泛、固定、融合
- **CT/MRI**：評估氣管受壓程度、手術可行性

### 細針穿刺細胞學

- **Bethesda VI**（惡性）
- **Indication**：快速取樣，通常 core needle biopsy 更優（組織量足）
- **Cellularity**：高，但背景壞死多
- **Cytology — hallmark（三種細胞型態）**：
  1. **多形性巨細胞（pleomorphic giant cells）** — 多核，形態怪異，大量核分裂
  2. **梭形細胞（spindle cells）** — 肉瘤樣
  3. **上皮樣細胞（epithelioid cells）** — 較小，片狀排列
  - **大量壞死背景（tumor necrosis）**
  - 無 PTC 核特徵（無核溝、無假包涵體）
- **IHC（免疫組化）**：
  - **CK（cytokeratin）陽性**（確認上皮起源，排除肉瘤）
  - **Thyroglobulin 陰性**（去分化）
  - **TTF-1 陰性**（去分化）
  - **TP53 陽性**（常見）
  - **BRAF V600E**（約 30% 陽性 → 標靶治療機會）
- **鑑別**：Anaplastic carcinoma vs Acute suppurative thyroiditis（FNA：前者惡性細胞；後者中性球）

### 治療

1. **多模式治療（multidisciplinary）**：
   - 若局部可切除（Stage IVA/IVB 部分）→ 積極手術切除 + 術後放化療
   - 若無法切除 → 同步化放療（chemoradiation）
2. **化療**：Doxorubicin + Cisplatin；或 Paclitaxel
3. **標靶治療（BRAF V600E 陽性，約 30%）**：
   - **Dabrafenib + Trametinib**（BRAF + MEK 抑制劑）→ 顯著提升反應率
   - FDA 已核准（2018）
4. **免疫治療**：Pembrolizumab（PD-L1 高表現者）
5. **氣管插管 / 氣切**：處理急性呼吸道阻塞
6. **緩和治療（palliative）**：大多數患者

## 6. 甲狀腺髓質癌
**Medullary Thyroid Carcinoma，MTC**

### 簡介

起源於**C 細胞（旁濾泡細胞）**，分泌**降鈣素（calcitonin）**，是 MTC 的腫瘤標誌物。約佔甲狀腺惡性腫瘤 3–5%。

**遺傳型（25–30%）**：由 **RET 原致癌基因種系突變**引起：

- **MEN2A**（最常見）：MTC + 嗜鉻細胞瘤 + 副甲狀腺機能亢進
- **MEN2B**：MTC + 嗜鉻細胞瘤 + 黏膜神經瘤 + Marfanoid habitus
- **Familial MTC (FMTC)**：僅 MTC，無其他內分泌瘤

**散發型（70–75%）**：體細胞 RET 突變（約 40–50%）或 RAS 突變。

### 臨床表現

- **頸部結節**（甲狀腺中上 1/3，C 細胞集中區域）
- **腹瀉**（降鈣素、VIP 相關）— 常見於轉移性 MTC
- **顏面潮紅（flushing）**（降鈣素相關）
- **MEN2A**：腎上腺嗜鉻細胞瘤（高血壓發作、心悸）；副甲狀腺腺瘤（高鈣血症）
- **MEN2B**：黏膜神經瘤（嘴唇、舌頭）、Marfanoid 體型、習慣性便秘

> ⭐ **術前必須排除嗜鉻細胞瘤**（feochromocytoma）— 若先切甲狀腺麻醉誘導可引發高血壓危象

### 檢查

- 甲狀腺中段結節，質地堅硬
- 頸部淋巴結腫大（Level VI、II–V）
- 黏膜神經瘤（MEN2B）：嘴唇厚、舌頭表面結節

### 實驗室檢查

| 項目                                   | 結果                      |
| -------------------------------------- | ------------------------- |
| **降鈣素（Calcitonin）**               | **↑↑（顯著升高）**        |
| **CEA（癌胚抗原）**                    | **↑**（進展期）           |
| **TSH / Free T4**                      | 通常正常                  |
| **Anti-TPO / Anti-Tg**                 | 陰性                      |
| **Thyroglobulin**                      | 正常（C 細胞不產生 Tg）   |
| **24h 尿液兒茶酚胺（Catecholamines）** | 排除嗜鉻細胞瘤            |
| **血鈣 / PTH**                         | 排除副甲狀腺腺瘤（MEN2A） |
| **RET 基因種系突變**                   | 陽性（遺傳型）            |

> **Calcitonin 倍增時間（DT）**：<6 個月 → 高度侵襲性，預後差；>24 個月 → 惰性

### 影像學特徵

- **甲狀腺中段**（isthmus 附近）結節（C 細胞好發部位）
- **回音**：**低回音至等回音**
- **鈣化**：**粗顆粒狀鈣化**（dystrophic calcification，澱粉樣蛋白沉積引起）— 不同於 PTC 的微鈣化
  - 形成典型的**沙粒狀但較粗大（coarse stippled）鈣化**
- **邊界**：不規則，無明顯 halo
- **彩色都卜勒**：中心血流豐富
- **淋巴結**：雙側 Level VI 轉移（MTC 淋巴轉移率高，約 50%）
- **全身掃描**：¹⁸F-DOPA PET/CT、¹¹¹In-octreotide scan（評估遠端轉移）

### 細針穿刺細胞學

- **Bethesda V–VI**
- **Cellularity**：高度
- **Cytology — hallmark**：
  - **細胞形態多樣**：圓形、梭形、多角形（polymorphic）
  - **「鹽和胡椒」核染色（salt-and-pepper chromatin）** — 神經內分泌腫瘤典型特徵
  - 細胞質顆粒狀（密集嗜酸性顆粒）
  - 低黏附性（discohesive）
  - **澱粉樣蛋白（amyloid）**：細胞間粉紅色無定型物質，剛果紅染色（Congo red）陽性
  - 無 PTC 核特徵
- **IHC（確診）**：
  - **Calcitonin 陽性** ⭐（高度特異性）
  - **CEA 陽性**
  - **Chromogranin A / Synaptophysin 陽性**（神經內分泌標誌）
  - **Thyroglobulin 陰性**（與 PTC/FTC 鑑別）
  - **TTF-1 陽性**

### 治療

1. **手術**（唯一根治方法）：
   - **甲狀腺全切除 + 預防性 central neck dissection（Level VI）**
   - 術前必先排除嗜鉻細胞瘤（若 MEN2 相關 → 先切除腎上腺腫瘤）
2. **RAI**：**無效**（C 細胞不攝取碘）
3. **預防性甲狀腺切除**：RET 突變攜帶者依突變類型（ATA risk level A–D）決定手術時機
4. **進行性/轉移性 MTC**標靶治療：
   - **Vandetanib**（RET + VEGFR 抑制劑）— FDA 核准一線
   - **Cabozantinib**（RET + MET + VEGFR 抑制劑）— FDA 核准
   - **Selpercatinib / Pralsetinib**（高選擇性 RET 抑制劑）— 新一代，副作用更少
5. **術後追蹤**：降鈣素 + CEA，每 6 個月
6. **MEN 家族篩查**：基因型陽性者建議全家 RET 基因檢測

## 7. 甲狀腺淋巴瘤
**Thyroid Lymphoma**

### 簡介

原發性甲狀腺淋巴瘤罕見（占甲狀腺惡性腫瘤 <5%），但為重要鑑別診斷。**幾乎所有（>95%）均有 Hashimoto's 甲狀腺炎背景**，Hashimoto's 患者發生甲狀腺淋巴瘤的風險比一般人高 **67–80 倍**。

**病理分型**：

- **MALT lymphoma（最常見，約 60–70%）**：結外邊緣區 B 細胞淋巴瘤；惰性，預後良好
- **Diffuse Large B-Cell Lymphoma（DLBCL，約 20–30%）**：侵襲性，需積極化療
- **Mixed MALT + DLBCL（約 10%）**

好發於**60–70 歲女性**（女：男 ≈ 3–4:1）。

### 臨床表現
> Hashimoto's + 快速腫大

- **在既存 Hashimoto's 甲狀腺腫基礎上，結節或甲狀腺迅速腫大**（數週內）
- 壓迫症狀：聲音沙啞、吞嚥困難、呼吸困難（可發展為急症）
- 發燒、盜汗、體重減輕（B 症狀）— DLBCL 較常見
- 甲狀腺功能：多為**甲狀腺低下**（Hashimoto's 背景）

### 實驗室檢查

| 項目                 | 結果                           |
| -------------------- | ------------------------------ |
| **TSH / T4**         | 甲狀腺低下（Hashimoto's 背景） |
| **Anti-TPO**         | **顯著↑**（Hashimoto's 背景）  |
| **LDH**              | **↑↑**（DLBCL 尤其顯著）       |
| **β₂-microglobulin** | ↑（淋巴瘤活性標誌）            |
| **血球分類**         | 可能正常（原發性為主）         |
| **Tg / Calcitonin**  | 正常                           |

### 影像學特徵

- **Hashimoto's 背景下，新出現快速生長低回音腫塊**
- **回音**：**極度均質低回音（markedly hypoechoic）**— 幾乎無回音（pseudo-anechoic），有時**"pseudocystic" 外觀**
- **邊界**：相對清楚（MALT）至不規則浸潤（DLBCL）
- **血流**：豐富（DLBCL）至中等（MALT）
- **後方增強（posterior acoustic enhancement）**：由於均質低回音，超音波後方增強（類囊性）→ 為淋巴瘤特徵
- **無鈣化**（鑑別 PTC）
- **鄰近組織浸潤**（DLBCL）
- **CT**：評估縱膈、腹腔淋巴結（分期）

### 細針穿刺細胞學

- **Bethesda V–VI**（惡性），但 **FNA 常不足**，需 Core Needle Biopsy 或 Open Biopsy
- **Cytology**：
  - 大量**淋巴球**，多形性，可見大型淋巴母細胞（DLBCL）或小型成熟淋巴球（MALT）
  - **Hashimoto's** 背景（Hürthle cells + 淋巴球）
  - 無明顯惡性上皮細胞
- **確診必須**：
  - **Flow cytometry**（FNA 標本送流式細胞）：**clonal B-cell population**（CD19+, CD20+, kappa/lambda light chain restriction）
  - **Core biopsy + IHC**：CD20, CD3, Ki-67, BCL-2, BCL-6, MUM-1（GCB vs non-GCB DLBCL）
  - **MALT marker**：CD43, BCL-10

### 治療

1. **MALT**：
   - **放射線治療（Involved-field radiation therapy，IFRT）**：局部病灶（Stage IE）→ 優先
   - **化療 + Rituximab（R-CHOP）**：擴散病灶（Stage IIE+）
2. **DLBCL**：
   - **R-CHOP（Rituximab + 環磷醯胺 + Doxorubicin + Vincristine + Prednisolone）** × 6–8 週期
   - 放療（局部殘留病灶）
3. **緊急氣道處理**：嚴重壓迫時需立即介入（激素、氣切）

## 8. 甲狀腺轉移癌
**Metastatic Tumors to Thyroid**

### 簡介

雖罕見（屍體解剖約 2–4%，臨床診斷更少），但重要的鑑別診斷。**最常轉移至甲狀腺的原發癌**：

| 原發癌              | 特徵                         |
| ------------------- | ---------------------------- |
| **腎細胞癌（RCC）** | 最常見，富血管，甲狀腺高血流 |
| **肺癌**            | 常為多發性結節               |
| **乳癌**            | 多見於進展期                 |
| **大腸癌**          | 通常多發                     |
| **黑色素瘤**        | 可單發或多發                 |

### 影像學特徵

- **多發性結節**（多於原發甲狀腺癌的單發）
- **RCC 轉移**：**高度血管化（hypervascularity）**，類似 Graves' 血流，TI-RADS 偏高
- **邊界**：可清楚或不規則
- **無甲狀腺原發癌的典型微鈣化**
- **合併其他部位轉移**：臨床重要線索

### 細針穿刺細胞學

- IHC 區分原發 vs 轉移至關重要：
  - **RCC 轉移**：PAX8+, CD10+, RCC marker+, Tg−, TTF-1−
  - **肺腺癌轉移**：TTF-1+（但甲狀腺 PTC 也可 TTF-1+）, Napsin A+, Tg−
  - **乳癌轉移**：GATA3+, ER/PR+, Tg−
  - **黑色素瘤轉移**：S100+, HMB-45+, Melan-A+, Tg−

## 惡性特徵綜合比較

| US 特徵             | 良性提示                    | 惡性提示                         | 最特異性惡性特徵          |
| ------------------- | --------------------------- | -------------------------------- | ------------------------- |
| **回音**            | 高回音、等回音              | **低回音（< 頸肌）**             | 極低回音 → ATC/淋巴瘤     |
| **形狀**            | Wider-than-tall（橫生長）   | **Taller-than-wide（縱生長）**   | PTC 的 A/T > 1            |
| **邊界**            | 平滑、清楚、完整 halo       | **不規則、毛刺狀、腺外侵犯**     | Spiculated margins        |
| **鈣化**            | 粗鈣化、蛋殼鈣化（完整）    | **微鈣化（<1 mm 點狀強回音）**   | 微鈣化（PTC 特異性最高）  |
| **血流（Doppler）** | 周邊血流（rim vascularity） | **中心血流豐富（central flow）** | 不單獨使用診斷            |
| **後方聲影**        | 粗鈣化後方聲影（良性可見）  | 微鈣化無聲影（點狀強回音）       | 無聲影微鈣化 → 惡性       |
| **後方增強**        | 囊性腫塊                    | 淋巴瘤（pseudocystic）           | 實質腫塊後方增強 → 淋巴瘤 |
| **數量**            | 多發（多結節甲狀腺腫）      | 單發（不一定）                   | 多發可為轉移癌            |
| **淋巴結**          | 橢圓形、门（hilum）存在     | **圓形、囊性微鈣化**             | 囊性 LN → PTC 轉移        |

## 治療總結

| 腫瘤類型   | 首選手術               | 輔助治療                          | Key DON'Ts                      |
| ---------- | ---------------------- | --------------------------------- | ------------------------------- |
| **FA**     | 半切 + 術後病理        | 無（若確認良性）                  | 勿僅依 FNA 排除惡性（FTC 可能） |
| **PTC**    | 全切 ± Level VI 廓清   | RAI + TSH 抑制                    | 低風險微癌勿過度治療            |
| **FTC**    | 全切                   | RAI + TSH 抑制                    | 勿以 FNA 直接診斷（需病理）     |
| **HCC**    | 全切 ± 廓清            | RAI 效果差；TSH 抑制              | 勿依賴 RAI 治療                 |
| **ATC**    | 若可切除則手術（緊急） | 同步化放療；Dabrafenib（BRAF+）   | 勿延誤 BRAF 基因檢測            |
| **MTC**    | 全切 + Level VI 廓清   | Vandetanib/Cabozantinib（進展性） | 術前必排除嗜鉻細胞瘤            |
| **淋巴瘤** | 僅緊急氣道處理         | R-CHOP（DLBCL）；放療（MALT）     | 勿行廣泛甲狀腺切除              |

## 注意事項

::: warning 術前必排除嗜鉻細胞瘤（MTC / MEN2）
任何診斷或懷疑 MTC 的患者，**術前必須以 24 小時尿液兒茶酚胺或血漿游離甲氧腎上腺素（metanephrines）排除嗜鉻細胞瘤**。在嗜鉻細胞瘤未處理前進行甲狀腺手術，麻醉誘導可能引發致命的高血壓危象。
:::

::: warning FNA 無法區分濾泡性腺瘤 vs 濾泡癌
細針穿刺細胞學對 Bethesda III/IV 病灶**無法鑑別良惡性**，因包膜浸潤與血管浸潤需在完整切除標本上才能評估。臨床應善用分子檢測（ThyroSeq v3、Afirma GSC）輔助決策。
:::

::: info TI-RADS 降低不必要 FNA
ACR TI-RADS（2017）結合結節組成、回音、形狀、邊緣及強回音灶五大特徵評分，依分級（TR1–TR5）及結節大小決定是否 FNA，可減少約 30–40% 的不必要穿刺。
:::

::: warning Hashimoto's + 快速腫大 = 排除淋巴瘤
橋本氏甲狀腺炎患者若出現**短期內（數週）快速增大的甲狀腺腫塊**，必須積極排除甲狀腺淋巴瘤（LDH、超音波、Core Biopsy + Flow Cytometry）。延誤診斷可導致氣道阻塞。
:::

::: info NIFTP — 不再是癌症
**Noninvasive Follicular Thyroid neoplasm with Papillary-like nuclear features（NIFTP）** — 原被歸類為包裹型濾泡變異 PTC，2016 年起重新分類為**良性腫瘤**（不再稱 carcinoma），無需 RAI，復發率極低。NIFTP 的去癌化顯著降低了 PTC 的「過度診斷」問題。
:::
