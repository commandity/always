# 甲狀腺炎
**Thyroiditis**

## 概述

甲狀腺炎泛指甲狀腺的發炎性疾病，病因涵蓋感染、自體免疫、藥物、放射線及不明原因。臨床上依**病程長短**與**病理特徵**分為四大類：

| 分類                                       | 病程                | 典型病因               |
| ------------------------------------------ | ------------------- | ---------------------- |
| **Acute (Suppurative)**                    | 急性（數天）        | 細菌感染               |
| **Subacute (De Quervain's)**               | 亞急性（數週∼數月） | 病毒感染後發炎反應     |
| **Chronic (Hashimoto's)**                  | 慢性（數月∼數年）   | 自體免疫               |
| **Other (Riedel's, Silent, Drug-induced)** | 視類型而定          | 纖維化／自體免疫／藥物 |

::: tip 判讀要點
甲狀腺毒症（TSH↓）以 **RAIU** 區分：**RAIU 低** 為破壞／釋放型（甲狀腺炎、藥物、外源性甲狀腺素），preformed 荷爾蒙外漏，**禁用抗甲狀腺藥**；**RAIU 高／正常** 為真性合成過度（Graves'、毒性腺瘤／MNG）。TSH↑ 多為甲狀腺炎進入**晚期破壞／纖維化**（Hashimoto's、Riedel's）。

**fT3/fT4 ratio**（RAIU 無法取得時的輔助鑑別）：Graves' disease → **> 0.4**（T3 優先分泌）；Destructive thyroiditis → **< 0.3**（T4 為主釋放）。
:::

## 1. 急性化膿性甲狀腺炎 Acute Suppurative Thyroiditis

### 簡介 Introduction

少見但危急的甲狀腺細菌感染，通常由鄰近結構（咽喉、頸部）直接擴散或經血行感染。**最常見病原**：_Staphylococcus aureus_, _Streptococcus pyogenes_, _Streptococcus pneumoniae_。好發於兒童（常合併**梨狀窩瘻管**— piriform sinus fistula）及免疫不全成人。

### 臨床表現 Clinical Presentation

- 急性發作：頸部前方紅、腫、熱、痛
- 吞嚥痛（odynophagia）、頸部壓痛
- 發燒、寒顫、全身不適
- 甲狀腺功能通常**正常**（少數一過性輕微甲狀腺毒症）

### 身體檢查 Examination

- 甲狀腺單側腫大、劇烈壓痛、皮膚紅熱
- 侷限性波動感（膿瘍形成時）
- 頸部活動受限、頸部淋巴結腫大
- 咽喉鏡可能可見梨狀窩水腫

### 實驗室檢查 Laboratory

| 項目                   | 結果           |
| ---------------------- | -------------- |
| **TSH**                | 正常           |
| **Free T4 / T3**       | 正常           |
| **ESR / CRP**          | 顯著↑↑         |
| **WBC**                | ↑ 嗜中性球為主 |
| **Anti-TPO / Anti-Tg** | 正常           |

### 影像學特徵 Imaging

- **Thyroid ultrasound**（第一線）：
  - 單葉腫大、不均質低回音區域
  - 液化壞死／膿瘍形成：不規則無回音區
  - 彩色都卜勒：周邊充血、中心無血流（膿瘍）
  - 頸部軟組織水腫
- **CT with contrast**：膿瘍範圍評估、確認梨狀窩瘻管（兒童）

### 細針穿刺細胞學 FNAC

- **Indication**：診斷確立＋病原鑑定（非鑑別惡性）
- **Cellularity**：高（膿瘍液）
- **Cytology**：
  - **Neutrophils**（嗜中性球）為主 — 大量，退變性
  - **Necrotic debris**（壞死碎片）
  - **Histiocytes**（組織球）
  - 無上皮細胞或少量退變濾泡上皮
- **Gram stain + culture**：確定病原及藥物敏感性
- **Key role**：區分 acute suppurative vs 快速生長的 anaplastic carcinoma（後者細胞學可見明顯惡性細胞）

### 治療 Treatment

1. **Empiric antibiotics**：覆蓋 MSSA/MRSA + 鏈球菌（如 clindamycin + cefazolin 或 vancomycin）
2. **Adjust by culture**：目標治療 2–4 週
3. **Surgical drainage**：若有膿瘍形成（US 或 CT 確認）
4. **Supportive care**：退燒、止痛、保持呼吸道通暢

## 2. Subacute Granulomatous Thyroiditis（De Quervain's Thyroiditis）

### 簡介 Introduction

也稱 De Quervain's thyroiditis，為最常見的**疼痛性甲狀腺炎**。被認為是由病毒感染（Coxsackie, mumps, influenza, adenovirus, EBV）後的發炎反應引起，而非直接感染。好發於**30–50 歲女性**，夏季／秋季較多，常在前驅上呼吸道感染後 2–4 週發生。

### 臨床表現 Clinical Presentation

- **頸部疼痛**：單側或雙側，可放射至下頷、耳部、牙齒
- 甲狀腺腫大、質地堅硬、觸痛明顯
- **甲狀腺毒症期**（初期 4–8 週）：心悸、手抖、體重減輕、怕熱、焦慮
- 之後可能經歷**甲狀腺機能正常期** → **甲狀腺低下期**（數週∼數月）→ 多數恢復正常
- 疼痛於吞嚥、轉頭、按壓時加劇

### 檢查 Examination

- 甲狀腺瀰漫性或結節狀腫大，質地堅硬（hard）
- 明顯壓痛（marked tenderness）
- 甲狀腺上方的皮膚可能輕微紅（但無急性細菌感染的紅熱）
- 無頸部淋巴結腫大（vs acute suppurative）
- 觸診時疼痛往往非常顯著

### 實驗室檢查 Laboratory

| Phase                   | TSH  | Free T4 | T3   | ESR           | Anti-TPO |
| ----------------------- | ---- | ------- | ---- | ------------- | -------- |
| **Thyrotoxic** (1–2 mo) | ↓↓   | ↑↑      | ↑    | ↑↑ (通常 >50) | 正常／低 |
| **Euthyroid** (2–4 wk)  | 正常 | 正常    | 正常 | ↑             | 正常     |
| **Hypothyroid** (weeks) | ↑↑   | ↓       | ↓    | 正常∼輕↑      | 正常     |

- **ESR** （常 >50 mm/h，可達 100+）/ **CRP** 顯著升高
- **Anti-TPO / Anti-Tg** 通常陰性或低效價
- **RAIU**（放射碘攝取率）：極低（接近 0）

**Phase explanation — natural 3-phase course of De Quervain's thyroiditis:**

| Phase           | What's happening                                                   | Interpretation                                                                                                                                      |
| --------------- | ------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Thyrotoxic**  | Inflammation destroys follicles → preformed T4/T3 leaks into blood | TSH↓↓ FT4↑↑ T3↑ — **hormone release, not overproduction**. ESR↑↑ reflects active inflammation. **DO NOT use antithyroid drugs (methimazole, PTU)**. |
| **Euthyroid**   | Stored hormone depleted, inflammation subsiding                    | Labs transiently normalize. ESR still ↑ but coming down.                                                                                            |
| **Hypothyroid** | Follicular damage exhausted → insufficient production              | TSH↑↑ FT4↓↓. Most recover fully in 3–6 months; ~5–10% need permanent LT4.                                                                           |

> This 3-phase pattern is **unique to destructive thyroiditis** (De Quervain's, silent, postpartum) and contrasts with Graves' disease where hyperthyroidism persists without spontaneous resolution.

### 影像學特徵 Imaging

- **Thyroid ultrasound**：
  - **關鍵特徵**：**ill-defined hypoechoic areas**（邊界不清的低回音區）
  - 病灶呈 **patchy / geographic**（地圖狀、斑塊狀）分布
  - 低回音區可單側或雙側，無「結節」感（vs nodular goiter）
  - 彩色都卜勒：**血流正常至減少**（與 toxic 腺瘤的高血流相反）
  - 病程後期可能出現微小鈣化（vs 惡性腫瘤）
- **RAIU scan（碘-123 或 Technetium-99m）**：攝取率極低（<1%），為**診斷金標準**
  - 注意：甲狀腺毒症期 RAIU 極低 vs Graves' disease RAIU 升高

### 細針穿刺細胞學 FNAC

- **Indication**：不典型表現（無痛、單側結節）或治療無反應時排除 malignancy
- **Cellularity**：中度
- **Cytology — hallmark**：
  - **Granulomas**（肉芽腫）— 上皮樣組織球聚集，常含 multinucleated giant cells
  - **Colloid** 稀少或消失
  - **Granulation tissue**（肉芽組織）
  - **Lymphocytes** 及 **histiocytes** 混合浸潤
  - 無惡性細胞
- **Background**：發炎性、顆粒狀
- **Pitfall**：巨細胞可能被誤認為 Hürthle cells（oncocytes）→ 注意臨床關聯

### 診斷標準 Diagnostic Criteria — 可記憶為「De Quervain's Triad」

1. Painful nodular thyroid + marked tenderness
2. Elevated ESR / CRP
3. Low RAIU

### 治療 Treatment

1. **NSAIDs**（第一線）：ibuprofen 600–800 mg q6–8h 或 naproxen 500 mg BID，通常 2–4 週
2. **Corticosteroids**（若 NSAID 無效或疼痛嚴重）：
   - Prednisolone 30–40 mg/day，每 1–2 週減量
   - 總療程 6–8 週 — **勿太快停藥**（復發率高）
   - 症狀改善通常在 24–48h 內
3. **β-blocker**（控制心悸）：propranolol 10–40 mg q6–8h 或 atenolol
4. **勿給予抗甲狀腺藥物**（methimazole、PTU）— 發炎造成的甲狀腺毒症是破壞性釋放，不是合成過多
5. **Thyroxine**：僅在甲狀腺低下期持續 >3–6 個月時補充
6. **定期追蹤 TSH**：每 4–6 週，直到恢復正常

### 預後 Prognosis

- 90% 患者在 2–6 個月內完全恢復正常甲狀腺功能
- 約 5–10% 發展為永久性甲狀腺低下
- 少數復發（約 2–4%）

## 3. 橋本氏甲狀腺炎 Hashimoto's Thyroiditis — Chronic Lymphocytic Thyroiditis

### 簡介 Introduction

最常見的甲狀腺炎，也是**甲狀腺低下最常見的原因**（在碘充足地區）。自體免疫疾病：**Anti-TPO（甲狀腺過氧化酶抗體）** 與 **Anti-Tg（甲狀腺球蛋白抗體）** 攻擊甲狀腺組織。好發於 **30–50 歲女性**（女：男 ≈ 7–10:1），有家族聚集性，常合併其他自體免疫疾病（SLE、RA、Sjögren's、Addison's、T1DM、Celiac）。

### 臨床表現 Clinical Presentation

- 最常見表現：**無症狀甲狀腺腫大**（routine check 或 US 意外發現）
- **緩慢進行性的甲狀腺功能減退**：疲倦、不耐冷、便秘、體重增加、皮膚乾燥、落髮、記憶力減退、月經失調
- **Hashimoto's encephalopathy**（罕見但重要併發症）：癲癇、認知障礙、肌躍症
- 甲狀腺質地柔韌（firm / rubbery），表面**結節狀**（pseudonodular）
- 部分患者初期有短暫甲狀腺毒症 — **Hashitoxicosis**（抗體破壞釋放 T4/T3）

### 檢查 Examination

- 甲狀腺**瀰漫性腫大**，對稱或非對稱
- 質地：**firm / rubbery**（橡皮樣堅實）
- 表面：**不規則、結節狀**（pseudonodules）
- 無壓痛（vs subacute / acute）
- 晚期可能萎縮（atrophic form → 甲狀腺縮小）

### 實驗室檢查 Laboratory

| Stage                | TSH  | Free T4 | Anti-TPO | Anti-Tg    | Timeline                                        |
| -------------------- | ---- | ------- | -------- | ---------- | ----------------------------------------------- |
| **Euthyroid**        | 正常 | 正常    | ↑↑ (95%) | ↑ (60–80%) | 早期（數年）— 抗體陽性但功能正常                |
| **Subclinical hypo** | ↑    | 正常    | ↑↑       | ↑          | 中期（數月∼數年）— TSH 先升高                   |
| **Overt hypo**       | ↑↑   | ↓       | ↑↑       | ↑          | 晚期（數年∼數十年）— 需 LT4 治療                |
| **Hashitoxicosis**   | ↓    | ↑       | ↑        | ↑          | 任何時期可能發生（數天∼數週）— 抗體破壞釋放激素 |

- **Anti-TPO**：陽性率最高（>95%），**最敏感指標**
- **Anti-Tg**：陽性率 60–80%
- **TSH**是疾病進展的最佳監測指標

### 影像學特徵 Imaging

- **Thyroid ultrasound**：
  - **實質回音度顯著降低**（markedly hypoechoic）— 典型特徵
  - Heterogeneous echotexture（不均質回音）
  - **Hypoechoic micronodules / pseudonodules**（1–6 mm，邊界模糊）
  - **Fibrotic septations**（纖維化條狀高回音）
  - **Hyperechoic bands** — 纖維化產生的線狀高回音（類肝硬化樣 pattern）
  - 彩色都卜勒：**血流正常至輕微增加**（與 Graves' 的顯著高血流不同）
  - 晚期：體積萎縮，實質更不均質
  - **注意**：Hashimoto's 患者發生**甲狀腺淋巴瘤**及**乳突癌 (PTC)** 的風險略增

### 細針穿刺細胞學 FNAC

- **Indication**：評估可疑結節（排除 PTC / lymphoma）、不典型表現
- **Cellularity**：中∼高
- **Cytology — hallmark**：
  - **Lymphocytes**（淋巴球）— 大量，成熟型，可見 germinal center fragments
  - **Hürthle cells (oncocytes)** — 嗜酸性顆粒狀細胞質，核大、核仁明顯
  - **Plasma cells**（漿細胞）
  - 濾泡上皮細胞呈退變性、大小不一
- **Bethesda category**：最常見 **Bethesda II**（良性）— consistent with lymphocytic thyroiditis
- **Key stains**：
  - 若懷疑 lymphoma → 需 **flow cytometry**（CD19, CD20, kappa/lambda light chain restriction）
  - **CK19 / HBME-1 / Galectin-3**（若懷疑 PTC 合併）
- **Pitfall**：Hürthle cell 豐富時需鑑別 Hürthle cell neoplasm（Bethesda III/IV）

| Grade   | 發現                                                  |
| ------- | ----------------------------------------------------- |
| **I**   | Mild heterogeneity, normal echogenicity               |
| **II**  | Moderate heterogeneity, mild hypoechogenicity         |
| **III** | Marked hypoechogenicity, micronodules, fibrotic bands |
| **IV**  | Atrophic, end-stage                                   |

### 治療 Treatment

1. **Levothyroxine (LT4) 補充**：適用於 overt hypothyroidism（TSH ↑ + FT4 ↓）
2. **Subclinical hypothyroidism**（TSH ↑, FT4 正常）：
   - TSH >10 mIU/L → 治療
   - TSH 4.5–10 mIU/L → 若 TPOAb (+)、懷孕、有症狀、年輕、高膽固醇 → 考慮治療
3. **初始劑量**：1.6 mcg/kg/day（成人）; 年長者或心臟病患者從 12.5–25 mcg/day 起始
4. **目標**：TSH 0.5–2.5 mIU/L（一般成人）; 0.3–2.0（懷孕）
5. **監測**：開始後 6–8 週追蹤 TSH，調整後 6 週再追蹤

### 相關疾病 Associated Conditions

- **Thyroid lymphoma**：當快速增大、壓迫症狀 → US + FNA + flow cytometry
- **Papillary thyroid carcinoma**：Hashimoto's 患者風險稍增，US 追蹤
- **Other autoimmune diseases**：SLE, RA, T1DM, pernicious anemia, Addison's

## 4. 纖維性甲狀腺炎 Riedel's Thyroiditis

### 簡介 Introduction

極罕見的慢性甲狀腺炎（佔 <0.1%），特徵為甲狀腺實質被**緻密纖維組織取代**，浸潤至周圍結構。可能是系統性 IgG4-related disease 的一部分。**發病年齡 30–60 歲**，女多於男。

### 臨床特徵 Clinical

- 甲狀腺**石頭樣堅硬（stony hard）**、無痛、固定
- 明顯壓迫症狀：吞嚥困難、聲音沙啞、呼吸困難、喘鳴
- 甲狀腺功能：多為正常（早期）→ 最終低下
- 甲狀腺與周圍組織（氣管、頸部肌肉）沾黏固定

### 實驗室檢查 Laboratory

- Anti-TPO / Anti-Tg：可能陽性（低效價）
- ESR：正常∼輕微升高
- IgG4：可能升高

### 影像學特徵 Imaging

- **Thyroid ultrasound**：
  - **Hypoechoic mass**（低回音團塊）
  - 邊界不清、浸潤至周圍組織
  - 氣管受壓狹窄
  - 甲狀腺正常結構消失
- **CT**：甲狀腺低密度、軟組織浸潤、氣管受壓

### 細針穿刺細胞學 FNAC

- **Cellularity**：**通常 insufficient**（纖維化 → 細胞量極少）— FNA 診斷價值有限
- **Cytology**：
  - 少量**梭形細胞（spindle cells/fibroblasts）**
  - **Collagen bundles**（膠原纖維束）
  - 無或稀少濾泡上皮
  - 無淋巴球浸潤（vs Hashimoto's）
- **Core needle biopsy (CNB)** 或 **Open biopsy** 為診斷金標準
  - 病理：緻密纖維組織取代甲狀腺實質，storiform pattern，IgG4+ plasma cells 浸潤
- **Pitfall**：需與 anaplastic carcinoma、lymphoma（sclerosing variant）、fibrosarcoma 區分

### 治療 Treatment

1. **Corticosteroids**：prednisolone 40–60 mg/day → 逐漸減量（部分患者反應好）
2. **Tamoxifen**：與 steroids 合併或單用
3. **Surgery**：嚴重壓迫症狀（氣管狹窄）→ 峽部切除（isthmusectomy）或甲狀腺切除
4. **LT4**：若發展為甲狀腺低下

## 5. 無痛甲狀腺炎 ＆ 妊娠後甲狀腺炎 Silent (Painless) Thyroiditis & Postpartum Thyroiditis

### 簡介 Introduction

**Silent thyroiditis**（無痛性甲狀腺炎）：與 Hashimoto's 同屬自體免疫，但臨床表現為**無痛**甲狀腺毒症，與 De Quervain's 類似有 3 期病程（toxic → euthyroid → hypothyroid）。

**Postpartum thyroiditis**：產後 2–6 個月發生，本質上是 silent thyroiditis 的產後形式。

### 臨床特徵 Clinical

- 無痛性甲狀腺腫大（mild to moderate）
- **甲狀腺毒症期**（4–8 週）：心悸、怕熱、體重減輕
- 之後可能過渡到甲狀腺低下期（3–6 個月），多數恢復正常
- Postpartum：母體產後免疫回彈（rebound immune activation）

### 實驗室檢查 Laboratory

- **Anti-TPO**：**通常陽性**（**>90%** postpartum）
- **ESR**：正常∼輕微升高（vs De Quervain's 顯著升高）
- **RAIU**：顯著降低（與 De Quervain's 相似）

### 影像學特徵 Imaging

- **Thyroid ultrasound**：
  - 甲狀腺體積：正常至輕度增大
  - 實質：**輕度至中度不均質**
  - 低回音區 patchy distribution（但通常無 De Quervain's 明顯）
  - 無結節（vs Hashimoto's 的 pseudonodules）
  - 彩色都卜勒：血流正常至減少

### 細針穿刺細胞學 FNAC

- **Indication**：通常不須 FNA（臨床＋RAIU 即可診斷）；用於表現不典型時排除其他病因
- **Cellularity**：低∼中度
- **Cytology**：
  - **Lymphocytes** — 輕∼中度浸潤（較 Hashimoto's 少）
  - **Hurthle cells** — 少見或無（vs Hashimoto's）
  - 濾泡上皮細胞大小不一，退變性改變（degenerative changes）
  - **多形核白血球（PMN）** — 少見
  - 無 granulomas（vs De Quervain's）
  - 無惡性細胞
- **Key feature**：細胞學與 Hashimoto's 類似但**程度較輕**，且臨床上有典型 thyrotoxicosis → hypothyroidism 雙相病程

### 治療 Treatment

- **甲狀腺毒症期**：β-blocker（propranolol）— 勿使用抗甲狀腺藥物
- **甲狀腺低下期**：LT4（若症狀持續 >3–6 個月）
- **產後**：可哺乳（LT4 安全），需監測嬰兒甲狀腺功能

## 6. 藥物性甲狀腺炎 Drug-Induced Thyroiditis

| 藥物                                                             | 機轉                                | 表現                         |
| ---------------------------------------------------------------- | ----------------------------------- | ---------------------------- |
| **Amiodarone**                                                   | Type 1 / Type 2 / 直接毒性          | 甲狀腺毒症或低下（詳見下表） |
| **Lithium**                                                      | 抑制甲狀腺荷爾蒙釋放                | 甲狀腺低下                   |
| **Interferon-α**                                                 | 誘發自體免疫                        | 類似 silent thyroiditis      |
| **Tyrosine kinase inhibitors (sunitinib, sorafenib)**            | 抑制甲狀腺過氧化酶                  | 甲狀腺低下                   |
| **Checkpoint inhibitors (pembrolizumab, nivolumab, ipilimumab)** | Immune-related adverse event (irAE) | 甲狀腺炎（常為永久性低下）   |
| **Aminoglutethimide**                                            | 抑制碘有機化                        | 甲狀腺低下                   |

### Amiodarone-Induced Thyroiditis (AIT)

Amiodarone 為 class III 抗心律不整藥，每一分子含 37% 碘（200 mg = 75 mg 碘），半衰期長達 50–100 天。甲狀腺副作用最常見且最複雜，需區分 3 種型態：

| Feature                                       | Type 1 Thyrotoxicosis                                                 | Type 2 Thyrotoxicosis                                                     | Hypothyroidism                                    |
| --------------------------------------------- | --------------------------------------------------------------------- | ------------------------------------------------------------------------- | ------------------------------------------------- |
| **Mechanism**                                 | 碘過量（iodine excess）→ Jod-Basedow phenomenon                       | 直接毒性引起破壞性發炎（destructive thyroiditis）                         | 碘過量抑制甲狀腺荷爾蒙合成（Wolff-Chaikoff 效應） |
| **Risk factors**                              | 碘缺乏地區、既存結節性甲狀腺腫／Graves' disease                       | 無特定風險因子                                                            | 碘充足地區、既存 Hashimoto's                      |
| **Thyroid antibodies (Anti-TPO)**             | 常陽性（約 60–80%）                                                   | 通常陰性                                                                  | 常陽性                                            |
| **Thyroid function**                          | Thyrotoxicosis                                                        | Thyrotoxicosis                                                            | Hypothyroidism                                    |
| **24h RAIU**                                  | 碘充足區 ↓；碘缺乏區 正常或 ↑                                         | **<5%**（顯著降低）                                                       | 通常↓                                             |
| **Color Doppler US**                          | **Hypervascularity**（血流增加，類似 Graves'）                        | **Reduced blood flow**（血流減少）                                        | Variable（血流可正常或減少）                      |
| **Serum IL-6**                                | 正常∼輕微↑                                                            | **顯著↑**                                                                 | —                                                 |
| **Response to thionamides**                   | **Methimazole/PTU → 有效**（中等至大劑量）                            | 無效                                                                      | —                                                 |
| **Response to corticosteroids**               | 部分有效（合併使用）                                                  | **顯著有效**（prednisolone 40–60 mg/day）                                 | —                                                 |
| **Response to KClO₄ (potassium perchlorate)** | 有效（減少甲狀腺碘攝取）                                              | 無效                                                                      | —                                                 |
| **Preferred therapy**                         | Antithyroid drugs ± 碘造影劑（iopanoic acid）± KClO₄ → 最終甲狀腺切除 | **High-dose corticosteroids**（prednisolone 40–60 mg/day）6–12 週逐漸減量 | LT4 補充                                          |

**Clinical approach to AIT with thyrotoxicosis：**

1. **Color Doppler US** 是最實用的第一線區分工具（hypervascular → Type 1; hypovascular → Type 2）
2. 若無法確認 → 可嘗試治療性診斷（therapeutic trial）：prednisolone 40 mg/day 2 週後，若 T3/T4 顯著下降 → Type 2
3. **Mixed form** 約 20% 患者兩型共存 → 需合併治療（antithyroid drugs + corticosteroids）
4. **勿停用 amiodarone**：除非心律問題可控且替代藥物可行（半衰期長，停藥後效應持續數月）

### 細針穿刺細胞學 FNAC

- **Indication**：甲狀腺快速增大、有壓迫症狀、或 US 發現局部結節時排除惡性
- **Cellularity**：視型態而異
- **Type 1**：
  - 類似 Graves' disease 之細胞學變化：濾泡上皮增生，細胞質 vacoules（scalloping）
  - 少量淋巴球浸潤
- **Type 2**：
  - 濾泡上皮退變性改變、空泡化
  - **Lymphocytes + histiocytes** 浸潤
  - 無 granulomas
  - 無增生性變化
- **Hypothyroidism form**：
  - 類似 Hashimoto's：濾泡萎縮、纖維化、淋巴球浸潤
- **Pitfall**：無特定細胞學標誌可區分 Type 1 vs Type 2 — 需仰賴臨床、US Doppler、IL-6

## 注意事項 Precautions

::: warning 甲狀腺毒症者 RAIU 鑑別至關重要
甲狀腺毒症（TSH ↓, fT4 ↑）時，**RAIU 是區分 Graves' disease（攝取↑）與破壞性甲狀腺炎（攝取↓）的關鍵檢查**。錯誤診斷可能導致不當使用抗甲狀腺藥物。

**輔助指標 — fT3/fT4 比值**：Graves'（**>0.4**，T3 優先分泌） vs 破壞性甲狀腺炎（**<0.3**，T4 儲存釋放為主）。比值愈高愈傾向 Graves'，可用於 RAIU 不可得或不適用時的初步鑑別。
:::

::: info 超音波不能取代 RAIU
雖然 US 可提供重要線索，但 De Quervain's 與 Silent thyroiditis 的**確診仍需 RAIU scan** — 兩者 US 表現可能相似，但臨床處置不同。
:::

::: warning Hashimoto's 與甲狀腺惡性腫瘤
Hashimoto's 患者發生 **Papillary thyroid carcinoma**及**甲狀腺淋巴瘤**的風險略增。若 US 發現可疑結節（低回音、微鈣化、邊界不規則），應進行 FNA。
:::
