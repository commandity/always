# 前庭復健 — 前庭功能障礙之評估與治療

**Vestibular Rehabilitation — Assessment & Management of Vestibular Disorders**

## 概論 Overview

前庭系統負責感知頭部位置與運動，一旦功能受損便可能引發**眩暈、步態不穩、視覺震盪**等症狀。前庭復健（Vestibular Rehabilitation Therapy, VRT）是一套以運動為基礎的物理治療方法，透過重複性、漸進性的刺激訓練，促進**中樞代償（central compensation）**，改善患者的平衡與生活品質。

## 作用機制

| 機制                      | 說明                                                 |
| ------------------------- | ---------------------------------------------------- |
| **習慣化（Habituation）** | 反覆暴露誘發眩暈的刺激，使症狀反應逐漸減弱           |
| **適應（Adaptation）**    | 增益再校正（gain recalibration）使前庭眼反射趨向正常 |
| **替代（Substitution）**  | 以視覺或本體感覺補償前庭功能缺損                     |

## 原地踏步轉身訓練

**適應症**：慢性單側前庭功能低下、前庭神經炎恢復期、老年人平衡訓練
**目標**：增強前庭—脊髓反射整合，改善動態平衡與步態穩定性

### 執行方法

1. 雙腳與肩同寬，站立於平坦地面
2. 雙手自然擺動，**原地踏步**（每秒約 1 步）
3. 踏步同時，以**小步伐原地轉身**，方向如下圖所示
4. 目視前方固定點（視覺固視），或視需要閉眼以增加難度
5. 每回合：**右轉 5 圈 → 稍事休息 → 左轉 5 圈**

> ⚠️ 首次執行請於牆壁或扶手旁進行，確保安全。如有明顯眩暈請立即停止。

### 訓練劑量建議

| 項目         | 說明                                       |
| ------------ | ------------------------------------------ |
| **頻率**     | 每日 1–2 次                                |
| **組數**     | 右轉 5 圈 + 左轉 5 圈 為 1 組，每次 2–3 組 |
| **踏步節奏** | 約每秒 1 步（節拍器可輔助）                |
| **進階方式** | 閉眼、泡棉墊上、雙手抱胸（減少上肢輔助）   |
| **療程**     | 通常建議持續 4–8 週                        |

### 漸進式升級 Level Up

當患者能穩定完成基本訓練（右轉 + 左轉各 5 圈，無明顯眩暈或不穩），可按以下層級逐步增加難度：

| 層級                 | 動作調整                                  | 挑戰機制                                       |
| -------------------- | ----------------------------------------- | ---------------------------------------------- |
| **1 — 基礎**         | 睜眼，雙臂自然擺動                        | 熟悉動作模式                                   |
| **2 — 減少上肢輔助** | 雙手抱胸或交握於腹前                      | 減少上肢平衡代償，增加核心穩定需求             |
| **3 — 閉眼**         | 閉眼執行踏步轉身                          | 移除視覺代償，強迫前庭—本體感覺主導            |
| **4 — 不穩定平面**   | 於泡棉墊／平衡墊上執行（先睜眼→再閉眼）   | 增加本體感覺干擾，提升前庭脊髓整合             |
| **5 — 增加圈數**     | 單方向增至 10 圈                          | 延續性旋轉刺激，挑戰中樞代償持久度             |
| **6 — 變化節奏**     | 快步（0.5 秒／步）↔ 慢步（2 秒／步）交替 | 適應不同角加速度，提升增益調整靈活性           |
| **7 — 頭部同步運動** | 踏步轉身同時左右擺頭或上下點頭            | 同時挑戰前庭眼反射（VOR）與前庭脊髓反射（VSR） |
| **8 — 雙重任務**     | 踏步轉身同時計算倒數或持杯盛水            | 增加認知負荷，模擬日常複雜環境                 |

> **原則**：每 1–2 週晉升一層級，前提為當前層級可連續 2 次訓練無明顯不適。如有眩暈加劇或跌倒風險，退回前一層級。

## 其他前庭復健方法

### 1. Cawthorne–Cooksey 運動

最早期的系統化前庭復健方案，由英國耳科醫師 Cawthorne 與物理治療師 Cooksey 於 1940 年代提出。從**眼睛運動**（固視、追蹤）→ **頭部運動**（前後、左右）→ **坐姿軀幹運動** → **站立動態平衡** → **行走活動** 循序漸進。

**適應症**：慢性前庭功能低下、老年跌倒預防

### 2. 前庭眼反射（VOR）增益訓練

患者注視固定目標，同時**主動左右或上下旋轉頭部**，要求視覺影像保持穩定不晃動。

- **靜態 VOR 訓練**：頭動但視線不移，注視卡片上文字
- **動態 VOR × 1 訓練**：頭與目標反向移動，速度一致
- **VOR × 2 訓練**：頭部移動時目標同向移動（進階）

**適應症**：前庭眼反射增益低下、振動幻視（oscillopsia）

### 3. Epley 耳石復位術 Canalith Repositioning Procedure

針對**後半規管良性陣發性眩暈（BPPV）**的首選處置。透過一系列頭部擺位，使游離耳石（canalith）從後半規管滾落至橢圓囊。

**步驟摘要**：

1. 坐位，頭轉向患側 45°
2. 快速躺下，頭仍轉向患側（Hallpike 位置）
3. 頭轉向對側 90°
4. 身體翻向對側，鼻尖朝下
5. 坐起

每個位置維持 30–60 秒，等待眼震消失。單次成功率約 **80–90%**。

### 4. Semont 耳石解脫術 Liberatory Maneuver

另一種後半規管 BPPV 的復位法，動作較快速。

1. 坐於床緣，頭轉向健側 45°
2. 快速倒向**患側**（頭朝下，鼻朝天花板）
3. 維持 2–3 分鐘
4. 快速翻身至**對側**（頭朝下，鼻朝地板）
5. 緩慢回正

### 5. 平衡板（Balance Board）訓練

以不穩定平面訓練**本體感覺—前庭整合**，常見工具包括：

- **海綿泡棉墊**（Foam Pad）
- **BOSU 球**
- **Wobble Board**

訓練模式：睜眼→閉眼 × 單腳→雙腳 × 靜態→動態頭部運動

### 6. 虛擬實境前庭復健 VR-based VRT

近年新興方法，利用 VR 頭戴裝置提供**可控制的視覺衝突**環境，同時結合動作感應追蹤，優點為：

- 可精準控制視覺流（optic flow）強度
- 自動記錄頭部運動頻率與幅度
- 遊戲化設計提升依從性

目前實證多屬小型試驗，尚未取代傳統 VRT，但前景可期。

## 禁忌與注意事項

| 注意事項         | 說明                                                     |
| ---------------- | -------------------------------------------------------- |
| **急性期眩暈**   | 急性前庭神經炎發病 72 小時內，優先藥物控制，不宜積極運動 |
| **中樞性眩暈**   | HINTS 陽性、小腦／腦幹病灶，應先排除中風再評估復健       |
| **心血管不穩定** | 嚴重姿位性低血壓、心律不整應調整運動強度                 |
| **骨科限制**     | 頸椎不穩定（RA、創傷）應避免快速頭部運動                 |
| **跌倒風險**     | 高齡、步態不穩患者，訓練時應有輔助人員或扶持設備         |

## 療效與實證

- 系統性回顧（Cochrane 2015 更新）顯示，前庭復健對**單側前庭功能低下**具中至高強度實證，能有效改善眩暈、平衡及生活品質
- 早期介入（症狀出現後 2–4 週內）效果優於延遲介入
- 個別化方案（tailored VRT）優於通用性運動，建議由物理治療師或前庭專科評估後訂定
