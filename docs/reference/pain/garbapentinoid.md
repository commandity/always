# Gabapentin 與 Pregabalin 聯合用藥 **Gabapentin & Pregabalin Combination**

## 概述

神經病理性疼痛的治療常面臨療效不足與副作用難以耐受的兩難困境。Gabapentinoids——包括 gabapentin（GBP）與 pregabalin（PGB）——是目前第一線用藥之一。然而，當單一藥物療效不佳時，臨床醫師是否可以考慮兩者聯合使用？本文回顧現有文獻，提供實證依據與臨床注意事項。

## 藥理機制

Gabapentin 與 pregabalin 同屬 gabapentinoids，作用機制相同：與脊髓背根神經節初級傳入神經元上的 **α2δ 亞單位**結合，抑制鈣離子內流，進而減少麩胺酸（glutamate）、P 物質（substance P）、正腎上腺素、多巴胺、血清素及 CGRP 等神經傳導物質的釋放，達到鎮痛效果。

值得注意的是，兩藥物與 α2δ 亞單位的結合在急性暴露（3–6 小時）對鈣電流的影響甚小，慢性暴露（17–20 小時）後才顯著抑制鈣電流。這一藥效動力學特性解釋了為何兩者均需**定時規律給藥**而非按需使用的臨床建議。

### 藥物動力學

| 特性           | gabapentin（GBP）                       | pregabalin（PGB）                |
| -------------- | --------------------------------------- | -------------------------------- |
| α2δ 結合親和力 | 基準                                    | **高 2–10 倍**                   |
| 藥動學         | **非線性**（高劑量吸收率下降）          | **線性**（劑量與血中濃度成正比） |
| 口服生物利用率 | 900 mg：60%；1200 mg：47%；3600 mg：34% | 較一致且可預測                   |
| 排泄途徑       | 腎臟（與腎功能成正比）                  | 腎臟（與腎功能成正比）           |

Gabapentin 的非線性動力學意味著高劑量時吸收效率顯著降低，限制了其有效劑量範圍；而 pregabalin 的線性動力學使劑量調整更為可預期。兩者均主要經腎臟排泄，**腎功能損害患者需謹慎調整劑量**。

## 適應症

| 適應症                        | gabapentin | pregabalin |
| ----------------------------- | ---------- | ---------- |
| **帶狀疱疹後神經痛（PHN）**   | FDA 核准   | FDA 核准   |
| **糖尿病周圍神經病變（DPN）** | 仿單外使用 | FDA 核准   |

| 焦慮症、失眠、偏頭痛預防      | 仿單外使用 | 仿單外使用 |

## 現有證據

### 動物研究基礎

Gabapentin 與 pregabalin 的協同鎮痛效應最早在動物模型中被發現：在大鼠熱痛閾值實驗中，以 1:1 或 10:1（GBP:PGB）比例給藥時，觀察到協同鎮痛效果，這為後續臨床研究提供了理論基礎。

### 臨床病例報告與回顧性研究

目前人體研究數量有限，以下為主要文獻摘要：

**案例一：腎功能損害合併轉移性腎癌患者（Senderovich & Jeyapragasan, 2018）**

一位 76 歲男性，患有轉移性腎癌並伴有腎功能損害（CrCl ≈ 15 mL/min），接受 pregabalin 150 mg 每日兩次治療但鎮痛效果不佳，且因跌倒風險無法增加劑量。加入 gabapentin 100 mg 每晚一次後，pregabalin 減量至 150 mg 每日一次，患者疼痛明顯改善。後續將 pregabalin 逐步減量至 25 mg/day，同時 gabapentin 增加至 100 mg 每日兩次，未觀察到明顯不良反應。

**案例二：放射性神經痛患者系列（Chow et al, 2023）**

兩位成人門診放射性神經痛患者，以 **24:1（1800 mg:75 mg）** GBP:PGB 比例治療。52 歲健康患者達到疼痛數值評量表（NRS）20% 降低且無副作用；88 歲腎功能損害患者則因多重干擾因素導致結果難以解讀。

**案例三：帶狀疱疹後神經痛回顧性研究（Tan et al, 2024）**

15 名 PHN 患者（均對單一藥物反應不足），以 pregabalin 150 mg/day 合併 gabapentin 300 mg/day 起始，逐漸調整至最終 **6:1（900 mg:150 mg）** 比例。14 天聯合治療後，NRS 疼痛評分從基線 4.40 顯著降至 2.40（P = 0.001）。僅 1 名患者出現頭暈但未停藥。**限制**：排除 CrCl ≤ 60 mL/min 患者，結果可能無法推廣至腎功能損害族群。

**案例四：糖尿病周圍神經病變的負面案例（Ghayur, 2021）**

一位 57 歲 DPN 患者在 gabapentin 原有療程中加入 pregabalin 50 mg（比例 6:1，300 mg:50 mg），出現疲勞、嗜睡與頭暈。停用 pregabalin 並將 gabapentin 緩慢增量至 200 mg 每日三次後，疼痛改善且不良反應消失。此案例提示聯合用藥在 DPN 族群中的安全性尚存疑慮，且腎功能資料未記錄為重要缺陷。

## 各適應症劑量

### Gabapentin 單藥劑量

- **帶狀疱疹後神經痛：** 1800 mg/day 與 2400 mg/day 的鎮痛效果相當（基線改善 32% vs 34%），但 1800 mg/day 的不良反應較少，是較佳的目標劑量。
- **糖尿病周圍神經病變：** 最高可微調至 3600 mg/day，顯著改善每日疼痛評分（基線 6.4 → 終點 3.9，vs 安慰劑 6.5 → 5.1，P < 0.001）；惟僅 67% 患者可耐受目標劑量，主要限制為頭暈、嗜睡及頭痛。
- **術後疼痛管理注意：** 回顧性研究顯示 gabapentin > 1050 mg 時嚴重不良事件發生率數字上增加，雖未達統計顯著，臨床仍應謹慎。

### Pregabalin 單藥劑量

- **帶狀疱疹後神經痛：** 彈性劑量（150–600 mg/day，平均 396 mg）的 ≥30% 疼痛改善率優於固定劑量 300 mg/day（70% vs 58%，P < 0.001）。
- **糖尿病周圍神經病變：** 600 mg/day 達到最高療效（≥30% 及 ≥50% 改善率）及最低的「治療所需人數（NNT）」，為 DPN 的最佳目標劑量。

## 不良反應與安全考量

### 常見副作用

Gabapentin 與 pregabalin 的共同副作用包括：

- 頭暈（dizziness）
- 嗜睡（drowsiness）／疲勞（fatigue）
- 共濟失調（ataxia）
- 周邊水腫（2–8%，多見於下肢）

### 高風險情境

下列情況需特別謹慎，副作用更為明顯：

- **腎功能損害：** 兩藥均以腎臟排泄，CrCl 降低時需大幅減量
- **高齡患者（≥65 歲）：** 跌倒風險、腎功能下降及藥物代謝改變使副作用風險倍增
- **合併中樞神經系統抑制劑：** 尤其與**鴉片類藥物（opioids）**合用時，呼吸抑制風險顯著上升
- **物質使用障礙病史：** 此族群出現欣快感、嗜睡、戒斷症狀等藥物濫用相關不良事件的風險更高
- **兩者聯合使用：** 相加的中樞抑制效果可能加劇上述所有不良反應

### 精神健康風險

雖然數據尚存爭議，部分回顧性研究顯示 gabapentinoids 可能增加自殺行為風險，臨床開立時應常規詢問精神科病史。

## 臨床建議摘要

### 一般臨床醫師

1. **先優化單一藥物**：在考慮聯合用藥前，應將單一 gabapentinoids 微調至其最低有效劑量，同時將副作用降至最低
2. **避免未達療效即換藥**：充分評估目前藥物的微調空間，確認已達足夠治療劑量
3. **謹慎評估腎功能**：開立前及追蹤中定期評估 CrCl，腎功能損害患者起始劑量應更低

### 疼痛科專科醫師

有豐富疼痛管理經驗的臨床醫師，可在以下條件下考慮 gabapentin/pregabalin 聯合治療：

- 難治性神經痛（放射性神經痛或帶狀疱疹後神經痛）對單一藥物反應不足
- 充分評估患者年齡、腎功能及合併用藥
- 採用文獻支持的比例範圍：**GBP:PGB = 6:1 至 24:1**
- 從低劑量起始，緩慢微調，密切監測副作用

### 目前尚不建議聯合治療

- 糖尿病周圍神經病變（DPN）——現有研究極為有限
- 纖維肌痛症
- 脊髓損傷相關神經痛
- 上述情境尚待進一步研究確立療效與安全比例

## 注意事項

::: warning 呼吸抑制風險
Gabapentinoids 與鴉片類藥物合用時，呼吸抑制風險顯著增加。此藥物組合應僅在明確臨床適應症下謹慎使用，並定期重新評估其必要性，尤其是高齡及腎功能損害患者。
:::

::: warning DPN 患者缺乏充分實證
目前聯合用藥的臨床研究幾乎均針對放射性神經痛（radiculopathy）及帶狀疱疹後神經痛（PHN），尚無針對糖尿病周圍神經病變（DPN）的高品質研究。臨床上對 DPN 患者使用此組合應極為謹慎，並視為個案考量。
:::

::: info 「最佳比例」尚未確立
現有文獻描述的 GBP:PGB 比例範圍從 6:1 至 24:1，但尚無隨機對照試驗確立最適比例。不同比例的選擇應依據患者腎功能、耐受性及疼痛特性個別化決定，不能套用固定公式。
:::
