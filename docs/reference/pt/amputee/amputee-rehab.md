# 截肢復健

**Amputee Rehabilitation**

## 常見截肢原因

| 原因                                | 說明                                       |
| :---------------------------------- | :----------------------------------------- |
| **血管疾病 Vascular disease**       | 糖尿病足、周邊動脈疾病，是最常見的截肢原因 |
| **外傷 Trauma**                     | 交通事故、工業意外、戰傷                   |
| **腫瘤 Tumor**                      | 骨肉瘤、軟組織肉瘤                         |
| **感染 Infection**                  | 壞死性筋膜炎、慢性骨髓炎                   |
| **先天異常 Congenital abnormality** | 先天性肢體缺失                             |

## 截肢層級與功能預後

截肢位置越高，功能影響越大，義肢使用率越低。

<details>
<summary>下肢截肢 Lower Limb</summary>

| 層級                              | 截肢位置        | 說明                                                    |
| :-------------------------------- | :-------------- | :------------------------------------------------------ |
| **經踝截肢 Transtibial (TT)**     | 膝下 Below-knee | 保留膝關節，約 80% 可恢復社區步行                       |
| **經膝截肢 Knee disarticulation** | 膝關節離斷      | 保留股骨長度，承重面積佳                                |
| **經股截肢 Transfemoral (TF)**    | 膝上 Above-knee | 失去膝關節，能量消耗增加 60-100%，約 50% 可恢復社區步行 |
| **半骨盆截肢 Hemi-pelvectomy**    | 半側骨盆移除    | 挑戰性高，輪椅為主要移動方式                            |

</details>

<details>
<summary>上肢截肢 Upper Limb</summary>

| 層級                                    | 截肢位置         | 說明                     |
| :-------------------------------------- | :--------------- | :----------------------- |
| **經腕截肢 Transradial (TR)**           | 前臂 Below-elbow | 保留肘關節與旋前旋後功能 |
| **經肱截肢 Transhumeral (TH)**          | 上臂 Above-elbow | 失去肘關節與前臂功能     |
| **肩關節離斷 Shoulder disarticulation** | 肩關節離斷       | 上肢功能幾乎全失         |

</details>

## 術後照護（0-4 週）

::: warning 術後重點
此階段重點：傷口保護優先，預防併發症。
:::

<details>
<summary>殘肢照護 Residual Limb Care</summary>

- **皮膚照護 Skin integrity**：每日檢查殘肢皮膚，留意紅疹、潰瘍、繭
- **殘肢塑型 Residual limb shaping**：使用彈性繃帶或矽膠套筒促進錐形塑型
- **關節活動度 ROM maintenance**：每日執行所有保留關節的全範圍活動
- **水腫控制 Edema management**：加壓包紮、抬高、早期活動

</details>

<details>
<summary>去敏感化訓練 Desensitization</summary>

殘肢感覺過敏（hypersensitivity）是常見問題，影響義肢穿戴舒適度。常用方法：

- **震動刺激 Vibration**：不同頻率震動器接觸殘肢，每天 3-5 次，每次 5 分鐘
- **輕觸訓練 Light touch**：棉花、毛刷、不同材質交替，每天 2-3 次
- **加壓按摩 Pressure massage**：由近端向遠端的漸進式壓力，每天 2 次，每次 10 分鐘
- **敲擊 Tapping**：殘肢表面輕柔敲擊，每天 3 次

</details>

<details>
<summary>水腫管理 Edema Management</summary>

- **加壓繃帶 Compression bandaging**：術後初期使用多層彈性繃帶系統
- **矽膠套筒 Silicone liner**：慢性水腫時使用定制加壓矯具
- **抬高 Elevation**：急性腫脹時殘肢高於心臟水平

</details>

<details>
<summary>攣縮預防 Contracture Prevention</summary>

不同截肢類型需注意的攣縮部位：

- **經股截肢 TF**：注意髖屈曲，需俯臥位、髖伸展運動
- **經踝截肢 TT**：注意膝屈曲，需膝伸展運動
- **經肱截肢 TH**：注意肩屈曲不足，需肩部全關節活動
- **經腕截肢 TR**：注意肘屈曲，需肘伸展與旋前旋後練習

</details>

<details>
<summary>疼痛管理 Pain Management</summary>

- **殘肢痛 Residual limb pain**：NSAIDs 藥物、冷療、電刺激
- **幻肢痛 Phantom limb pain**：見下方幻肢痛管理章節
- **神經痛 Neuropathic pain**：gabapentin、pregabalin 藥物調整、TENS

</details>

<details>
<summary>心理支持 Psychological Support</summary>

截肢是重大身體形象改變，可能引發哀傷、抑鬱、焦慮。應：

- 以同理心傾聽，允許哀傷過程
- 提供實際功能資訊
- 必要時轉介心理衛生團隊
- 鼓勵家屬參與復健
- 連結支持團體

</details>

## 義肢準備期（4-8 週）

<details>
<summary>治療目標 Treatment Goals</summary>

- 強化殘肢與核心肌群
- 開始站立平衡訓練
- 提升日常生活活動能力
- 為義肢穿戴做準備

</details>

<details>
<summary>重點訓練 Key Exercises</summary>

- **殘肢強化 Residual limb strengthening**：開放鏈與封閉鏈肌力訓練
- **核心穩定 Core stability**：平板支撐、死蟲式、鳥狗式
- **坐姿平衡 Sitting balance**：靜態到動態，雙手釋放到軀幹偏移
- **站立訓練 Standing training**：平行桿內站立、重心轉移
- **轉位訓練 Transfer training**：床到輪椅、輪椅到馬桶
- **日常生活活動 ADL training**：穿衣、如廁、衛浴

</details>

<details>
<summary>進階肌力訓練 Advanced Strengthening</summary>

**經股截肢者 TF 特有強化重點**：

| 肌群                       | 訓練動作           | 目的       |
| :------------------------- | :----------------- | :--------- |
| **股四頭肌 Quadriceps**    | 直膝抬腿、腿部推蹬 | 膝關節穩定 |
| **臀大肌 Gluteus maximus** | 橋式、俯臥伸髖     | 髖伸展力量 |
| **臀中肌 Gluteus medius**  | 側躺外展、站立外展 | 步態穩定   |

**經踝截肢者 TT 特有強化重點**：

| 肌群                   | 訓練動作   | 目的         |
| :--------------------- | :--------- | :----------- |
| **腘繩肌 Hamstrings**  | 俯臥屈膝   | 殘肢控制     |
| **內收肌群 Adductors** | 彈力帶內收 | 義肢對線控制 |

</details>

## 義肢訓練期（8-16 週）

::: warning 穿戴注意
初次穿戴義肢應由義肢技師與治療師共同監督。穿戴時間由短漸增（最初 30 分鐘→逐步增加至全天）。
:::

<details>
<summary>治療目標 Treatment Goals</summary>

- 學習義肢穿脫技術
- 達成安全、有效的義肢步行
- 建立步態對稱性

</details>

<details>
<summary>穿脫訓練 Donning & Doffing</summary>

| 義肢類型                          | 穿戴方法                        | 注意事項               |
| :-------------------------------- | :------------------------------ | :--------------------- |
| **吸著式懸掛 Suction suspension** | 殘肢塗滑石粉→穿襪→套入→釋放繃帶 | 確保無皺褶             |
| **矽膠套筒+鎖 Pin lock**          | 矽膠套筒穿著→鎖入管柱           | 確實接合鎖入機制       |
| **肌電義肢 Myoelectric**          | 穿戴→電極校準                   | 皮膚清潔、電極接觸良好 |

</details>

<details>
<summary>步態訓練進階 Gait Training Progression</summary>

**階段一：平行桿內**

- 重心轉移（前後、左右）
- 單腳站立
- 側向移動
- 前後移動
- 轉向

**階段二：助行器/拐杖**

- 四腳拐/助行器步行
- 平地步行（10-50 公尺）
- 轉向與轉彎
- 漸進減少輔具

**階段三：獨立步行**

- 無輔具步行（50-100 公尺）
- 不平坦地面
- 樓梯
- 功能性步行（購物、戶外）

</details>

<details>
<summary>常見步態偏差矯正 Common Gait Deviations</summary>

| 偏差                                    | 可能原因                 | 處理方式                   |
| :-------------------------------------- | :----------------------- | :------------------------- |
| **側向壓步 Trendelenburg**              | 臀中肌無力或義肢對線不良 | 強化臀中肌、調整對線       |
| **腳趾外旋 Toe-out**                    | 繫鞍位不良或股骨殘肢過短 | 繫鞍調整                   |
| **膝屈曲不穩 Knee flexion instability** | 膝屈曲力矩過大           | 繫鞍角度調整、股四頭肌強化 |
| **軀幹側屈 Lateral trunk lean**         | 義肢過長或臀中肌無力     | 義肢長度修正、強化臀中肌   |

</details>

## 社區回歸（>16 週）

<details>
<summary>治療目標 Treatment Goals</summary>

- 高階移動能力：快速步行、長距離
- 樓梯與斜坡安全上下
- 不平坦地面與多種表面適應
- 職業與運動回歸
- 獨立社區參與

</details>

<details>
<summary>進階訓練 Advanced Training</summary>

- **快速步行 Fast walking**：步速 1.0-1.4 m/s
- **長距離耐力 Long-distance endurance**：10-30 分鐘持續步行
- **樓梯訓練 Stair training**：扶手使用、交替步伐
- **斜坡訓練 Ramp training**：上坡/下坡、不同坡度
- **障礙物跨越 Obstacle negotiation**：跨步、繞行、抬腿
- **地面起立 Floor rising**：從地板安全站起

</details>

<details>
<summary>職業回歸 Vocational Return</summary>

| 回歸類型                             | 介入重點                   |
| :----------------------------------- | :------------------------- |
| **久坐職業 Sedentary work**          | 坐姿人體工學、義肢適應     |
| **輕度體力職業 Light physical work** | 站立耐力、功能性訓練       |
| **重度體力職業 Heavy physical work** | 高強度肌力、平衡、安全策略 |

</details>

## 幻肢痛管理

幻肢感（phantom sensation）是指截肢者感受到已不存在之肢體的感覺，發生率高達 90% 以上。幻肢痛（phantom limb pain）則是發生在幻肢部位的疼痛，約 50-80% 的截肢者會經歷。

<details>
<summary>機制 Mechanisms</summary>

- **周邊機制 Peripheral**：神經瘤形成、異位放電
- **脊髓機制 Spinal**：背角重塑、失傳入性增強
- **皮質機制 Cortical**：皮質重塑、感覺運動地圖失序

</details>

<details>
<summary>治療策略 Treatment Strategies</summary>

| 策略                                    | 說明                                         | 證據等級 |
| :-------------------------------------- | :------------------------------------------- | :------- |
| **鏡像治療 Mirror therapy**             | 使用鏡子創造健側肢體運動的視覺錯覺           | 強       |
| **TENS**                                | 經皮電神經刺激，調控疼痛門控                 | 中       |
| **漸進式運動想像 Graded motor imagery** | 左右辨識→運動想像→鏡像治療                   | 中       |
| **虛擬實境 VR therapy**                 | 沉浸式虛擬環境中的肢體運動訓練               | 中       |
| **藥物治療 Pharmacotherapy**            | gabapentin、pregabalin、duloxetine、tramadol | 中-強    |

</details>

<details>
<summary>鏡像治療方案 Mirror Therapy Protocol</summary>

鏡像治療是處理幻肢痛的首選非藥物介入：

1. **設定**：坐於鏡箱前，殘肢與鏡面同側放置
2. **健側運動**：在鏡中看到健側肢體的反射影像
3. **執行動作**：彎曲、伸直、握拳、張開手指等
4. **專注想像**：想像幻肢正在執行相同動作
5. **頻率**：每次 15-20 分鐘，每日 2-3 次
6. **療程**：至少 4-12 週

</details>

## 特定訓練運動

<details>
<summary>平衡進階訓練 Balance Progression</summary>

| 階段        | 動作                       | 進階                        |
| :---------- | :------------------------- | :-------------------------- |
| **Level 1** | 雙腳站立、平行桿內靜態站立 | 閉眼、不穩定面              |
| **Level 2** | 單腳站立（健側）、重心轉移 | 時間延長、外力 Perturbation |
| **Level 3** | 義肢側站立、側向踏步       | 動態平衡挑戰                |
| **Level 4** | 步行中轉向、避障           | 快速反應                    |
| **Level 5** | 不平坦地面步行、斜坡行走   | 視覺剝奪                    |

</details>

<details>
<summary>踏步訓練 Stepping Exercises</summary>

| 訓練動作                        | 描述                   | 目的         |
| :------------------------------ | :--------------------- | :----------- |
| **前方踏步 Forward stepping**   | 義肢側或交替向前踏步   | 步態啟動     |
| **後方踏步 Backward stepping**  | 向後踏步以訓練後側鏈   | 後側肌力     |
| **側向踏步 Lateral stepping**   | 側踏步、交叉步         | 側向穩定     |
| **抬腿踏步 Step-ups**           | 低階梯（10-20 cm）上下 | 股四頭肌力量 |
| **跨越踏步 Crossover stepping** | 跨過小障礙物           | 抬腿能力     |

</details>

<details>
<summary>每日建議運動量 Daily Exercise Recommendations</summary>

| 運動類別                   | 頻率        | 時間           | 強度       |
| :------------------------- | :---------- | :------------- | :--------- |
| **關節活動度 ROM**         | 每日 2-3 回 | 每個方向 10 次 | 全域範圍   |
| **肌力訓練 Strengthening** | 每日 1-2 回 | 10-15 次/組    | 中至高度   |
| **平衡訓練 Balance**       | 每日 1-2 回 | 每次 30-60 秒  | 進階性挑戰 |
| **步行訓練 Gait training** | 每日 1 回   | 10-30 分鐘     | 功能性速度 |

</details>

## 義肢考量

<details>
<summary>下肢義肢組件 Lower Limb Components</summary>

| 組件                 | 低階       | 中階           | 高階                   |
| :------------------- | :--------- | :------------- | :--------------------- |
| **膝關節 Knee (TF)** | 單軸機械膝 | 多軸氣壓膝     | 微處理器控制膝         |
| **足部 Foot (TT)**   | 固定踝     | SACH 足/多軸足 | 儲能足                 |
| **套筒 Socket**      | 傳統接受腔 | 矽膠襯裡+吸著  | CAD/CAM 定制、真空懸掛 |

</details>

<details>
<summary>上肢義肢選配原則 Upper Limb Prosthetic Selection</summary>

- **截肢層級 Level**：近端截肢需更多組件，遠端截肢功能保留較多
- **慣用手 Dominant hand**：非慣用手截肢較易適應；慣用手截肢訓練量更大
- **年齡 Age**：年輕者偏好肌電/動態義肢；年長者偏好輕量簡易
- **職業 Occupation**：高體能需求者可能不使用義肢而依賴輔具
- **美觀需求 Cosmesis**：部分個案優先考量外觀
- **預算 Budget**：組件價格差異巨大，考慮更換與維護成本

</details>

## 注意事項

<details>
<summary>皮膚完整性 Skin Integrity</summary>

::: warning 皮膚警語
義肢穿戴者之殘肢皮膚損傷是最常見的併發症之一。任何皮膚發紅若持續 30 分鐘以上未消退，應立即脫除義肢並評估。
:::

- 穿戴前仔細檢查繃帶/套筒是否有皺褶
- 吸汗襪、保持皮膚清潔、使用吸濕粉
- 每日殘肢清洗、透氣時間
- 定期義肢對線評估
- 套筒內襯加墊覆蓋骨突處

</details>

<details>
<summary>對側肢體照護 Contralateral Limb Care</summary>

::: warning 對側風險
下肢截肢者的對側肢體承受更高負荷與跌倒風險。任何傷口或問題可能完全剝奪行走能力。
:::

- 義肢對線優化、輔具使用
- 定期足部檢查、適當鞋具
- 平衡訓練、環境改造、照明改善
- 體重管理、低衝擊運動

</details>

<details>
<summary>跌倒預防 Fall Prevention</summary>

- 訓練跌倒安全策略（前倒、側倒）
- 教導地面起立技巧
- 家庭環境評估（消除絆倒 hazard、改善照明）
- 穿戴穩固鞋具
- 避免光滑地面、潮濕環境

</details>
