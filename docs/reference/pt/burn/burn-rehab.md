# 燒傷復健

**Burn Rehabilitation**

<details open>

## 什麼是燒傷？

燒傷（burn）是由熱、電、化學物質或輻射線造成的組織損傷。燒傷復健的核心目標是**預防攣縮（contracture）、維持功能、促進傷口癒合**，並協助患者重返日常生活。

### 燒傷深度分類

| 分類                       | 特徵                       | 癒合時間              |
| -------------------------- | -------------------------- | --------------------- |
| 表淺 superficial           | 紅斑、疼痛、無水皰 blister | 5–7 天                |
| 部分厚度 partial thickness | 水皰、潮紅、劇痛           | 10–28 天              |
| 全層厚度 full thickness    | 乾燥、皮革狀、無痛覺       | 需手術植皮 skin graft |

### 燒傷面積評估

使用**九則法（rule of nines）**估算燒傷面積 TBSA：

- 頭頸 head & neck：9%
- 上肢 upper extremity：各 9%（共 18%）
- 軀幹前 anterior trunk：18%
- 軀幹後 posterior trunk：18%
- 下肢 lower extremity：各 18%（共 36%）
- 會陰 perineum：1%

## 第一期：急性期 （0-2 週）

### 定位

**Positioning**

- **抗攣縮體位 anti-contracture positioning**
- 頸部：中立或輕度伸展
- 腋窩：外展 90°、前屈 45°
- 肘部：伸直（掌側）或屈曲 90°（背側）
- 手部：腕伸 20–30°、掌指關節屈曲 70–90°、指間關節伸直、拇指外展
- 髖部：伸直、外展
- 膝部：伸直
- 踝部：背屈 90°（中立位）

### 水腫管理

**Edema Management**

- **抬高患肢 elevation** 高於心臟水平
- **加壓包紮 compression wrapping**
- **氣動壓力治療 IPC（intermittent pneumatic compression）**
- 早期床上翻身與體位變換

### 矯形器

**Splinting**

- **靜息夾板 resting splint**：維持功能位 position of function
- **動態夾板 dynamic splint**：提供漸進牽引
- 每天至少 22 小時配戴，僅在換藥與治療時取下

### 被動關節活動度訓練

**Passive ROM**

- 每日 2–3 次，每關節 5–10 次重複
- 避免造成傷口撕裂
- 在水療或換藥時進行

</details>

## 第二期：癒合期 （2–6 週）

### 主動關節活動度訓練

**Active ROM**

- 鼓勵患者主動活動所有未受傷關節
- 受傷關節進行**輕柔主動協助活動 active-assisted ROM**
- 功能性動作練習（抓握、抬手、步行）

### 疤痕管理

**Scar Management**

- **壓力衣 pressure garment**：23–25 mmHg 持續配戴
- **矽膠製品 silicone**：矽膠片 sheeting 或矽膠凝膠 gel，每日 12–23 小時
- **按摩 massage**：傷口癒合後開始，配合乳液，力度由輕至中

### 壓力治療

**Pressure Garment Therapy**

- 定製壓力衣需在**癒合後 2 週內**取得
- 持續配戴 **6–18 個月**
- 每 2–3 個月更換（因疤痕變薄）
- 合併夜間夾板維持 ROM

### 脫敏訓練

**Desensitization**

針對觸覺過敏 hypersensitivity：

1. 用不同質地物品輪流接觸患部
2. 從輕柔質地（棉花）漸進至粗糙質地（毛巾）
3. 每日數次，每次 10–15 分鐘

::: warning
脫敏訓練應在傷口完全癒合、無感染風險後開始。
:::

## 第三期：疤痕成熟期 (6 週 - 2 年)

### 疤痕重塑

**Scar Remodeling**

- 疤痕由增生期進入成熟期，顏色變淡、柔軟度增加
- 壓力衣與矽膠持續使用
- 疤痕可能持續改善至 18–24 個月

### 伸展訓練

**Stretching**

- 每日主動伸展所有受影響關節
- 結合**長時間低負荷牽引 sustained low-load stretch**
- 配合熱敷或溫水浸泡後進行

### 肌力強化

**Strengthening**

- 漸進式阻力訓練 progressive resistance training
- 從等長收縮 isometric 開始，逐步進展至等張 isotonic 與功能性動作
- 每週 3–5 次

### 功能性訓練

**Functional Training**

- ADLs（日常活動）練習：穿衣、進食、如廁、洗澡
- 工作模擬訓練
- 社會參與準備

</details>

## 第四期：重建與復健期

### 攣縮鬆解術

**Contracture Release**

- 若保守治療無法改善功能性攣縮
- 手術方式：植皮 skin graft、皮瓣 flap、組織擴張 tissue expansion
- 術後需密集復健，防止再次攣縮

### 持續疤痕管理

**Continued Scar Management**

- 壓力衣與矽膠至少使用至傷後 18–24 個月
- **激光治療 laser therapy**：脈衝染料激光 PDL 或分段激光 fractional laser
- 注射治療：類固醇注射 steroid injection、5-FU、博來霉素 bleomycin

</details>

## 疤痕管理

### 壓力治療

**Pressure Therapy**

| 項目                    | 說明                          |
| ----------------------- | ----------------------------- |
| 壓力範圍 pressure range | 15–40 mmHg（建議 24–30 mmHg） |
| 配戴時間 wear duration  | 每日 23 小時，持續 6–18 個月  |
| 類型 type               | 壓力衣、壓力套、壓力繃帶      |
| 機制 mechanism          | 降低膠原合成、促進膠原重排    |

### 矽膠

**Silicone**

- **矽膠片 silicone sheeting**：可重複使用，每日 12–23 小時
- **矽膠凝膠 silicone gel**：塗抹後形成薄膜
- 適應症：增生性疤痕 hypertrophic scar、蟹足腫 keloid

### 按摩

**Massage**

- 配合保濕乳液或油
- 力度：輕至中度壓力
- 方向：垂直於疤痕方向
- 頻率：每日 2–3 次

### 伸展

**Stretching**

- 主動伸展 active stretch
- 被動伸展 passive stretch
- 牽引夾板 traction splint 輔助

</details>

## 特定關節運動

### 頸部

- 屈曲 flexion、伸展 extension、側彎 lateral flexion、旋轉 rotation
- 壓力衣搭配頸部夾板
- 每日維持伸展位 ≥ 6 小時

### 腋窩

- 外展 abduction、前屈 flexion、外旋 external rotation
- 使用**腋下楔形夾板 axillary wedge splint**
- 棍棒輔助伸展 wand exercise

### 肘部

- 屈曲與伸展
- 旋前 pronation 與旋後 supination
- 配合前臂壓力衣

### 手部

- 掌指關節 MCP 屈曲、指間關節 IP 伸展
- 拇指外展與對掌 opposition
- 抓握與捏力訓練 grip and pinch exercise
- **爪形手預防 claw hand prevention**

### 膝部

- 屈曲與伸展
- 低姿坐位練習
- 下樓梯訓練

### 踝部

- 背屈 dorsiflexion 維持 90°
- **踮腳尖 tiptoe standing** 預防馬蹄足 equinus
- 使用夜間背屈夾板

</details>

## 矯形器

**Splinting**

### 抗攣縮體位夾板

**Anti-Contracture Positioning Splints**

| 關節         | 建議體位                                    |
| ------------ | ------------------------------------------- |
| 頸部         | 中立伸展 neutral extension                  |
| 肩部         | 外展 90° abduction、前屈 45° flexion        |
| 肘部         | 伸直（掌側 volar）或屈曲 90°（背側 dorsal） |
| 腕部         | 伸展 extension 20–30°                       |
| 掌指關節 MCP | 屈曲 flexion 70–90°                         |
| 指間關節 IP  | 伸直 extension                              |
| 拇指         | 外展 abduction、對掌 opposition             |
| 髖部         | 伸直 extension、外展 abduction              |
| 膝部         | 伸直 extension                              |
| 踝部         | 背屈 dorsiflexion 90°                       |

### 靜息夾板

**Resting Splints**

- 維持功能位 position of function
- 每天至少 22 小時配戴
- 定期評估與調整

### 動態夾板

**Dynamic Splints**

- 提供漸進、可調節的牽引力量
- 適用於已有攣縮的關節

## 水療 Hydrotherapy

水療利用水的浮力、溫度與阻力特性促進復健。

### 益處

- **浮力 buoyancy**：減少關節負荷，便於早期活動
- **溫熱效應 thermal effect**：緩解疼痛、軟化疤痕、促進血液循環
- **流體阻力 fluid resistance**：提供漸進式肌力訓練
- **心理放鬆 psychological relaxation**：減輕焦慮、改善情緒

### 參數

| 項目                   | 建議參數                   |
| ---------------------- | -------------------------- |
| 水溫 water temperature | 33–35°C（體感舒適溫度）    |
| 時間 duration          | 每次 20–45 分鐘            |
| 頻率 frequency         | 每週 3–5 次                |
| 深度 depth             | 依患者耐受度，通常胸部深度 |

::: warning
水療需在傷口完全癒合後開始。若仍有未癒合傷口，應使用防水敷料 waterproof dressing 保護，並避免長時間浸泡。
:::

</details>

## 疼痛管理

### 疼痛類型

| 類型                       | 特徵                             |
| -------------------------- | -------------------------------- |
| 急性疼痛 acute pain        | 劇烈、與組織損傷直接相關         |
| 慢性疼痛 chronic pain      | 持續性、可能為神經性 neuropathic |
| 操作性疼痛 procedural pain | 換藥、治療時由醫療操作引發       |

### 疼痛緩解

- **藥物治療 pharmacological**：鴉片類止痛藥 opioids、非鴉片類止痛藥、神經痛藥物（gabapentin、pregabalin）
- **非藥物治療 non-pharmacological**：分散注意力 distraction、虛擬實境 VR（virtual reality）、催眠 hypnosis
- **操作前止痛 pre-procedural**：換藥前 30 分鐘給予速效止痛藥或吸入性止痛藥 nitrous oxide

</details>

## 注意事項

### 傷口感染

::: warning
感染徵兆 signs of infection：紅腫加劇、滲出液增加或變色、發熱、異味。應立即通知醫療團隊。
:::

- 嚴格無菌換藥技術 sterile dressing technique
- 監測白血球計數與培養結果 WBC count and culture
- 適時使用抗生素 antibiotics

### 自體植皮照護

- 植皮後 5–7 天避免剪力與摩擦 shear and friction
- 供皮區 donor site 保持清潔乾燥
- 使用不沾黏敷料 non-adherent dressing
- 避免陽光直射供皮區 6–12 個月

### 異位性骨化

**Heterotopic Ossification (HO)**

- 發生率：燒傷面積 > 30% TBSA 患者約 1–3%
- 好發關節：肘部、肩部、髖部
- 預防：早期關節活動、消炎痛 indomethacin
- 治療：手術切除 excision ＋ 放射治療 radiation therapy

### 心理社會支持

- 燒傷倖存者**創傷後壓力症候群 PTSD（post-traumatic stress disorder）**發生率高達 30–45%
- 常見心理問題：憂鬱 depression、焦慮 anxiety、身體形象障礙 body image disturbance、社交退縮 social withdrawal
- 介入措施：
  - 團體治療 group therapy
  - 認知行為治療 CBT（cognitive behavioral therapy）
  - 家族支持教育 family support
  - 同儕支持 peer support
  - 職業復健 vocational rehabilitation

::: warning
心理支持應從急性期開始，貫穿整個復健過程。燒傷疤痕對外觀的影響可能造成嚴重的心理困擾。
:::

## 兒童燒傷注意事項

- **生長板 growth plate**：燒傷可能影響骨骼生長
- **皮膚面積比例**：兒童頭部佔比較大，需使用修正版 Lund-Browder 圖表
- **心理發展**：遊戲治療 play therapy 是兒童復健的重要手段
- **家長參與**：教育家長進行居家復健訓練

</details>

## 何時就醫

::: warning
出現以下情況應立即就醫：
:::

- 傷口紅腫加劇、滲出液增加或變色
- 發熱、異味
- 傷口癒合後關節活動受限 limited ROM
- 疤痕增生、影響外觀或功能
- 出現心理困擾症狀

</details>

## 日常照護要點

1. **傷口護理 wound care**：保持清潔乾燥，按時換藥 dressing change
2. **壓力衣配戴 pressure garment**：每日 23 小時，持續 6–18 個月
3. **矽膠使用 silicone**：每日 12–23 小時
4. **按摩 massage**：每日 2–3 次，配合乳液
5. **運動 exercise**：每日至少 2 次關節活動與伸展
6. **防曬 sun protection**：避免陽光直射傷口 6–12 個月

</details>
