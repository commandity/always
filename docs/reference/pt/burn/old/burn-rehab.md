# 燒傷復健 

**Burn Rehabilitation**

## 概述

燒傷（burn）是由熱、電、化學物質或輻射線造成的組織損傷。燒傷復健的核心目標是**預防攣縮、維持功能、促進傷口癒合**，並協助患者重返日常生活與社會參與。

### 燒傷病理生理

- **局部反應 local response**：凝固性壞死（coagulation necrosis）形成焦痂（eschar）
- **全身反應 systemic response**：大量細胞激素釋放（cytokine release），可導致代謝亢進、免疫抑制
- **液體移動 fluid shift**：燒傷後 24–48 小時內毛細血管通透性增加，造成組織水腫

### 燒傷深度分類 Depth Classification

| 分類                             | 累及層級           | 臨床特徵             | 癒合時間               |
| -------------------------------- | ------------------ | -------------------- | ---------------------- |
| 表淺 Superficial                 | 表皮 epidermis     | 紅斑、疼痛、無水皰   | 5–7 天                 |
| 表淺部分厚度 Superficial partial | 表皮＋真皮淺層     | 水皰、潮紅、劇痛     | 10–14 天               |
| 深層部分厚度 Deep partial        | 表皮＋真皮深層     | 水皰、蒼白、痛覺遲鈍 | 21–28 天（可能需植皮） |
| 全層厚度 Full thickness          | 全層皮膚及皮下組織 | 乾燥、皮革狀、無痛覺 | 需手術植皮             |

### 燒傷面積評估 Total Body Surface Area (TBSA)

TBSA 使用**九則法（rule of nines）**或 **Lund-Browder 圖表**計算。99% TBSA 以上燒傷之死亡率約 50%（Baux index）。

## 評估 Assessment

### 燒傷面積計算

#### 九則法 Rule of Nines

| 部位                      | 成人佔比  |
| ------------------------- | --------- |
| 頭頸 Head & Neck          | 9%        |
| 上肢 Each Upper Extremity | 9% (18%)  |
| 軀幹前 Anterior Trunk     | 18%       |
| 軀幹後 Posterior Trunk    | 18%       |
| 下肢 Each Lower Extremity | 18% (36%) |
| 會陰 Perineum             | 1%        |

#### Lund-Browder 圖表

適用於兒童及不規則燒傷，依年齡調整頭部與下肢佔比。

### Baux Score

死亡風險估計 = **年齡 + TBSA%**

> 若合併吸入性損傷（inhalation injury），再加 17 分。

### 關節活動度測量 ROM Measurement

- 使用**量角器（goniometer）**測量攣縮角度
- 記錄**主動（AROM）**與**被動（PROM）**ROM
- 與健側或正常值比較
- 每次換藥時同步評估

### 疤痕評估 Scar Assessment

| 工具  | 全名                                       | 評估項目                                                         |
| ----- | ------------------------------------------ | ---------------------------------------------------------------- |
| VSS   | Vancouver Scar Scale                       | 色澤、厚度、血管分布、柔軟度                                     |
| POSAS | Patient and Observer Scar Assessment Scale | 觀察者量表＋患者自評（疼痛、癢感、顏色、硬度、厚度、不規則外觀） |

::: warning
疤痕評估應在傷口完全癒合後開始，並定期追蹤至疤痕成熟期（約 12–18 個月）。
:::

## 第一期：急性期 Phase 1 (Acute, 0–2 weeks)

### 定位 Positioning

- **抗攣縮體位 anti-contracture positioning**
- 頸部：中立或輕度伸展
- 腋窩：外展 90°、前屈 45°
- 肘部：伸直（掌側）或屈曲 90°（背側）
- 手部：腕伸 20–30°、掌指關節屈曲 70–90°、指間關節伸直、拇指外展
- 髖部：伸直、外展
- 膝部：伸直
- 踝部：背屈 90°（中立位）

### 水腫管理 Edema Management

- **抬高患肢（elevation）**高於心臟水平
- **加壓包紮（compression wrapping）**
- **氣動壓力治療（intermittent pneumatic compression, IPC）**
- 早期床上翻身與體位變換

### 矯形器 Splinting

- **靜息夾板（resting splint）**：維持功能位
- **動態夾板（dynamic splint）**：提供漸進牽引
- 每天至少 22 小時配戴，僅在換藥與治療時取下

### 被動關節活動度訓練 Passive ROM

- 每日 2–3 次，每關節 5–10 次重複
- 避免造成傷口撕裂
- 在水療或換藥時進行

---

## 第二期：癒合期 Phase 2 (Healing, 2–6 weeks)

### 主動關節活動度訓練 Active ROM

- 鼓勵患者主動活動所有未受傷關節
- 受傷關節進行**輕柔主動協助活動（active-assisted ROM）**
- 功能性動作練習（抓握、抬手、步行）

### 疤痕管理 Scar Management

- **壓力衣（pressure garment）**：23–25 mmHg 持續配戴
- **矽膠製品（silicone）**：矽膠片或矽膠凝膠，每日 12–23 小時
- **按摩 massage**：傷口癒合後開始，配合乳液，力度由輕至中

### 壓力治療 Pressure Garment Therapy

- 定製壓力衣需在**癒合後 2 週內**取得
- 持續配戴 **6–18 個月**
- 每 2–3 個月更換（因疤痕變薄）
- 合併夜間夾板維持 ROM

### 脫敏訓練 Desensitization

針對觸覺過敏（hypersensitivity）：

1. 用不同質地物品輪流接觸患部
2. 從輕柔質地（棉花）漸進至粗糙質地（毛巾）
3. 每日數次，每次 10–15 分鐘

::: warning
脫敏訓練應在傷口完全癒合、無感染風險後開始。
:::

---

## 第三期：疤痕成熟期 Phase 3 (Scar Maturation, 6 weeks–2 years)

### 疤痕重塑 Scar Remodeling

- 疤痕由增生期進入成熟期，顏色變淡、柔軟度增加
- 壓力衣與矽膠持續使用
- 疤痕可能持續改善至 18–24 個月

### 伸展訓練 Stretching

- 每日主動伸展所有受影響關節
- 結合**長時間低負荷牽引（sustained low-load stretch）**
- 配合熱敷或溫水浸泡後進行

### 肌力強化 Strengthening

- 漸進式阻力訓練（progressive resistance training）
- 從等長收縮（isometric）開始，逐步進展至等張（isotonic）與功能性動作
- 每週 3–5 次

### 功能性訓練 Functional Training

- ADLs（日常活動）練習：穿衣、進食、如廁、洗澡
- 工作模擬訓練
- 社會參與準備

---

## 第四期：重建與復健期 Phase 4 (Reconstructive/Rehabilitation)

### 攣縮鬆解術 Contracture Release

- 若保守治療無法改善功能性攣縮
- 手術方式：植皮（skin graft）、皮瓣（flap）、組織擴張（tissue expansion）
- 術後需密集復健，防止再次攣縮

### 持續疤痕管理 Continued Scar Management

- 壓力衣與矽膠至少使用至傷後 18–24 個月
- 激光治療（laser therapy）：脈衝染料激光（PDL）或分段激光（fractional laser）
- 注射治療：類固醇注射、5-FU、博來霉素

---

## 疙痕管理治療模態 Scar Management Modalities

### 壓力治療 Pressure Therapy

| 項目     | 說明                          |
| -------- | ----------------------------- |
| 壓力範圍 | 15–40 mmHg（建議 24–30 mmHg） |
| 配戴時間 | 每日 23 小時，持續 6–18 個月  |
| 類型     | 壓力衣、壓力套、壓力繃帶      |
| 機制     | 降低膠原合成、促進膠原重排    |

### 矽膠 Silcone

- **矽膠片（silicone sheeting）**：可重複使用，每日 12–23 小時
- **矽膠凝膠（silicone gel）**：塗抹後形成薄膜
- 適應症：增生性疤痕（hypertrophic scar）、蟹足腫（keloid）

### 按摩 Massage

- 配合保濕乳液或油
- 力度：輕至中度壓力
- 方向：垂直於疤痕方向
- 頻率：每日 2–3 次

### 伸展 Stretching

- 主動伸展（active stretch）
- 被動伸展（passive stretch）
- 牽引夾板（traction splint）輔助

## 特定關節運動 Specific Joint Exercises

### 頸部 Neck

- 屈曲、伸展、側彎、旋轉
- 壓力衣搭配頸部夾板
- 每日維持伸展位 ≥ 6 小時

### 腋窩 Axilla

- 外展（abduction）、前屈（flexion）、外旋（external rotation）
- 使用**腋下楔形夾板（axillary wedge splint）**
- 棍棒辅助伸展（wand exercises）

### 肘部 Elbow

- 屈曲與伸展
- 旋前（pronation）與旋後（supination）
- 配合前臂壓力衣

### 手部 Hand

- 掌指關節（MCP）屈曲、指間關節（IP）伸展
- 拇指外展與對掌
- 抓握與捏力訓練
- **爪形手預防（claw hand prevention）**

### 膝部 Knee

- 屈曲與伸展
- 低姿坐位練習
- 下樓梯訓練

### 踝部 Ankle

- 背屈（dorsiflexion）維持 90°
- **踮腳尖（tiptoe standing）**預防馬蹄足（equinus）
- 使用夜間背屈夾板

## 注意事項 Precautions

### 傷口感染 Wound Infection

::: warning
感染徵兆：紅腫加劇、滲出液增加或變色、發熱、異味。應立即通知醫療團隊。
:::

- 嚴格無菌換藥技術
- 監測白血球計數與培养結果
- 適時使用抗生素

### 自體植皮照護 Autograft Care

- 植皮後 5–7 天避免剪力與摩擦
- 供皮區（donor site）保持清潔乾燥
- 使用不沾黏敷料（non-adherent dressing）
- 避免陽光直射供皮區 6–12 個月

### 異位性骨化 Heterotopic Ossification (HO)

- 發生率：燒傷面積 > 30% TBSA 患者約 1–3%
- 好發關節：肘部、肩部、髖部
- 預防：早期關節活動、消炎痛（indomethacin）
- 治療：手術切除＋放射治療

### 心理社會支持 Psychosocial Support

- 燒傷倖存者**創傷後壓力症候群（PTSD）**發生率高達 30–45%
- 常見心理問題：憂鬱、焦慮、身體形象障礙（body image disturbance）、社交退縮
- 介入措施：
  - 團體治療（group therapy）
  - 認知行為治療（CBT）
  - 家族支持教育
  - 同儕支持（peer support）
  - 職業復健（vocational rehabilitation）

::: warning
心理支持應從急性期開始，貫穿整個復健過程。燒傷疤痕對外觀的影響可能造成嚴重的心理困擾。
:::

## 證據基礎 Evidence-Based Practice

### 燒傷復健臨床實踐指南 Clinical Practice Guidelines

| 指南來源                        | 重點建議                                        |
| ------------------------------- | ----------------------------------------------- |
| ISBI Practice Guidelines (2016) | 壓力治療應在癒合後立即開始，持續 6–12 個月      |
| ABA Guidelines                  | 早期復健介入（24–48 小時內），每日至少 2 次治療 |
| NICE Guidelines                 | 疤痕管理結合壓力衣、矽膠、按摩多模式治療        |
| Cochrane Reviews                | 壓力治療對增生性疤痕有中等品質證據支持          |

## 水療 Hydrotherapy

水療是燒傷復健的重要組成部分，利用水的浮力、溫度與阻力特性促進復健。

### 水療益處

- **浮力（buoyancy）**：減少關節負荷，便於早期活動
- **溫熱效應（thermaleffect）**：緩解疼痛、軟化疤痕、促進血液循環
- **流體阻力（fluid resistance）**：提供漸進式肌力訓練
- **心理放鬆（psychological relaxation）**：減輕焦慮、改善情緒

### 水療參數

| 項目 | 建議參數                   |
| ---- | -------------------------- |
| 水溫 | 33–35°C（體感舒適溫度）    |
| 時間 | 每次 20–45 分鐘            |
| 頻率 | 每週 3–5 次                |
| 深度 | 依患者耐受度，通常胸部深度 |

::: warning
水療需在傷口完全癒合後開始。若仍有未癒合傷口，應使用防水敷料保護，並避免長時間浸泡。
:::

## 疼痛管理 Pain Management

燒傷疼痛是影響復健參與度的主要障礙。有效的疼痛管理是成功復健的基礎。

### 疼痛類型

| 類型                  | 時間點        | 特徵                     |
| --------------------- | ------------- | ------------------------ |
| 急性疼痛 Acute        | 傷後即刻      | 劇烈、與組織損傷直接相關 |
| 慢性疼痛 Chronic      | 傷後 > 3 個月 | 持續性、可能為神經性     |
| 操作性疼痛 Procedural | 換藥、治療時  | 由醫療操作引發           |

### 疼痛介入策略

- **藥物治療**：鴉片類止痛藥、非鴉片類止痛藥、神經痛藥物（gabapentin、pregabalin）
- **非藥物治療**：分散注意力（distraction）、虛擬實境（VR）、催眠（hypnosis）
- **操作前止痛**：換藥前 30 分鐘給予速效止痛藥或吸入性止痛藥（nitrous oxide）

::: warning
操作性疼痛是燒傷患者最恐懼的經歷之一。有效的疼痛管理可顯著提高治療依從性。
:::

## 營養支持 Nutritional Support

嚴重燒傷會導致高代謝狀態，適當的營養支持對傷口癒合與復健至關重要。

### 營養需求

| 營養素   | 建議                                  |
| -------- | ------------------------------------- |
| 熱量     | 每日 25–35 kcal/kg + 根據燒傷面積調整 |
| 蛋白質   | 每日 1.5–2.0 g/kg                     |
| 維生素 C | 每日 500–1000 mg（促進膠原合成）      |
| 鋅       | 每日 40–80 mg（促進傷口癒合）         |
| 維生素 A | 每日 10,000–25,000 IU（上皮再生）     |

### 與復健的關係

- 充足營養支持可改善肌力與活動耐力
- 蛋白質不足會延緩傷口癒合、增加感染風險
- 營養不良會影響壓力衣治療的耐受度

## 兒童燒傷復健 Pediatric Burn Rehabilitation

兒童燒傷復健有其特殊考量，需結合生長發育特性。

### 特殊考量

- **生長板（growth plate）**：燒傷可能影響骨骼生長
- **皮膚面積比例**：兒童頭部佔比較大，需使用修正版 Lund-Browder
- **心理發展**：遊戲治療（play therapy）是兒童復健的重要手段
- **家長參與**：教育家長進行居家復健訓練

### 兒童壓力衣注意事項

- 需隨著生長定期更換
- 注意不影響呼吸與循環
- 選擇適合兒童的顏色與圖案以提高配戴意願

## 職業復健 Vocational Rehabilitation

職業復健的目標是協助燒傷倖存者重返工作崗位。

### 評估內容

- 工作需求分析（job demands analysis）
- 功能能力評估（functional capacity evaluation）
- 工作場所改造建議

### 介入措施

- 工作模擬訓練（work conditioning）
- 人機工學建議（ergonomic recommendations）
- 協助職場調整（workplace accommodation）

::: warning
職業復健應在復健中期開始規劃，與患者共同討論重返工作的目標與時間表。
:::

## 快速參考 Quick Reference

### 復健時程 Rehabilitation Timeline

| 時期                     | 時間      | 重點                             |
| ------------------------ | --------- | -------------------------------- |
| 急性期 Acute             | 0–2 週    | 定位、水腫控制、夾板、被動 ROM   |
| 癒合期 Healing           | 2–6 週    | 主動 ROM、疤痕管理、壓力衣、脫敏 |
| 疤痕成熟 Scar Maturation | 6 週–2 年 | 伸展、強化、功能訓練             |
| 重建 Reconstructive      | 隨時      | 放鬆術、持續疤痕管理             |

### 疤痕管理四步驟 Four-Step Scar Management

1. **壓力治療** Pressure therapy
2. **矽膠** Silicone
3. **按摩** Massage
4. **伸展** Stretching
