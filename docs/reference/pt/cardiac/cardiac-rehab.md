# 心臟復健

**Cardiac Rehabilitation**

## 概述

心臟復健是一項多面向的醫療計畫，專為心血管疾病患者設計，旨在透過運動訓練、教育及心理支持，恢復最佳生理與心理功能，降低再次心血管事件的風險。

<details>
<summary>禁忌症 Contraindications</summary>

::: warning 絕對禁忌症
**Absolute Contraindications (絕對禁忌症)**
| 類別 | 說明 |
| --------------------------------- | ------------------------ |
| Unstable angina | 不穩定型心絞痛 |
| Uncontrolled arrhythmias | 未控制的心律不整 |
| Severe aortic stenosis | 嚴重主動脈瓣狹窄 |
| Acute myocarditis or pericarditis | 急性心肌炎或心包膜炎 |
| Decompensated heart failure | 失代償性心臟衰竭 |
| Recent PE or DVT | 近期肺栓塞或深層靜脈血栓 |
| Uncontrolled systemic infection | 未控制的全身性感染 |
:::

::: warning 相對禁忌症
**Relative Contraindications (相對禁忌症)**
| 類別 | 說明 |
| -------------------------------- | ---------------- |
| Resting HR > 100 bpm | 靜止心率過快 |
| Systolic BP > 200 mmHg | 收縮壓過高 |
| Uncontrolled diabetes | 未控制的糖尿病 |
| Severe orthopaedic limitations | 嚴重骨科限制 |
| Significant cognitive impairment | 嚴重認知功能障礙 |
:::

</details>

## AHA/ACC 四階段模型 Four-Phase Model

美國心臟協會 (AHA) 與美國心臟病學會 (ACC) 提出的四階段心臟復健模型是目前國際上最廣泛採用的架構：

| 階段      | 名稱                          | 場所          | 時間                        |
| --------- | ----------------------------- | ------------- | --------------------------- |
| Phase I   | 住院期復健 Inpatient          | 醫院          | ACS/手術後數天              |
| Phase II  | 早期門診復健 Early Outpatient | 門診/復健中心 | 出院後 1-2 週開始，共 36 次 |
| Phase III | 延續照護 Extended Care        | 社區/居家     | Phase II 完成後             |
| Phase IV  | 維持期 Maintenance            | 自主管理      | 長期終身                    |

## 評估

<details>
<summary>心肺運動測試 Cardiopulmonary Exercise Testing (CPET)</summary>

CPET 是心臟復健中最重要的客觀評估工具，可同時評估心血管、肺部及代謝功能。

| 參數                                   | 意義                       |
| -------------------------------------- | -------------------------- |
| VO₂ peak                               | 運動能力的金標準指標       |
| AT (Anaerobic Threshold)               | 無氧閾值，用於設定運動強度 |
| VE/VCO₂ slope                          | 通氣效率，與預後相關       |
| Exercise oscillatory ventilation (EOV) | 運動震盪通氣，不良預後指標 |
| Blood pressure response                | 血壓對運動的反應           |

</details>

<details>
<summary>六分鐘步行測試 Six-Minute Walk Test (6MWT)</summary>

適用於無法進行 CPET 的患者，為次極量測試工具。

| 評估項目 | 正常參考值              |
| -------- | ----------------------- |
| 步行距離 | ≥ 400-500 公尺          |
| 變化意義 | > 30 公尺為臨床顯著改善 |
| Borg RPE | 記錄終止時主觀疲勞感    |
| SpO₂     | 監測是否有去飽和現象    |

</details>

<details>
<summary>運動處方制定 Exercise Prescription</summary>

#### 以心率儲備法 (HR Reserve / Karvonen) 計算

**Target HR = (HRmax − HRrest) × Intensity% + HRrest**

#### 以自覺費力程度 (RPE) 計算

| RPE (Borg 6-20) | RPE (Modified 0-10) | 描述                     |
| --------------- | ------------------- | ------------------------ |
| 11              | 2                   | 輕微 (Fairly Light)      |
| 12              | 3                   | 輕微至中等               |
| 13              | 4                   | 有些困難 (Somewhat Hard) |
| 14              | 5                   | 困難 (Hard)              |

::: warning 運動處方參數
**運動處方參數 Exercise Prescription Parameters**

| 參數               | 建議範圍                       |
| ------------------ | ------------------------------ |
| **強度 Intensity** | 40-80% HR reserve 或 RPE 11-14 |
| **時間 Duration**  | 20-60 分鐘（含熱身與緩和）     |
| **頻率 Frequency** | 每週 3-5 次                    |
| **型式 Type**      | 有氧運動為主，搭配阻力訓練     |

:::

</details>

## Phase I — 住院期復健

<details>
<summary>目標 Objectives</summary>

- 預防臥床相關併發症（如：肌肉萎縮、肺炎、深層靜脈血栓）
- 促進早期活動與功能恢復
- 患者及家屬教育
- 風險分層與出院規劃

</details>

<details>
<summary>早期活動方案 Early Mobilization Protocol</summary>

| 時間點                  | 活動內容                   |
| ----------------------- | -------------------------- |
| ACS 後 24 小時          | 床上翻身、四肢主動關節活動 |
| ACS 後 48 小時          | 坐起、床邊站立             |
| ACS 後 72 小時          | 室內緩步行走               |
| CABG 後 24 小時         | 坐起、下床                 |
| CABG 後 48 小時         | 走廊短距離步行             |
| PCI 後 6 小時（股動脈） | 輕度活動                   |
| PCI 後即刻（橈動脈）    | 限制較少                   |

</details>

<details>
<summary>日常生活活動教育 ADL Education</summary>

- 正確的起床與躺下方式
- 安全的穿脫衣技巧
- 個人衛生與沐浴注意事項
- 上下樓梯的策略
- 飲食與體重管理

</details>

<details>
<summary>出院前風險分層 Pre-discharge Risk Stratification</summary>

| 風險等級        | 特徵                             | 後續建議               |
| --------------- | -------------------------------- | ---------------------- |
| 低風險 Low      | 無併發症、無症狀、EF 正常        | 可進入社區式 Phase III |
| 中風險 Moderate | 輕微併發症或殘餘缺血             | 建議 Phase II 門診復健 |
| 高風險 High     | 嚴重併發症、心衰竭、電生理不穩定 | 需高度監測的 Phase II  |

</details>

## Phase II — 門診復健 (36 Sessions)

Phase II 是心臟復健的核心階段，通常包含 **36 次有監督的運動課程**，每次約 30-60 分鐘。

<details>
<summary>運動訓練結構 Exercise Training Structure</summary>

每次訓練包含三個部分：

#### 1. 熱身 Warm-up（5-10 分鐘）

- 低強度有氧活動（步行、固定腳踏車）
- 大肌群動態伸展
- 目的：預防運動誘導的心肌缺血、降低心律不整風險

#### 2. 主訓練 Main Training（20-40 分鐘）

**有氧運動 Aerobic Exercise**

| 訓練期間    | 強度              | 時間      | 進展策略           |
| ----------- | ----------------- | --------- | ------------------ |
| 第 1-4 週   | 40-50% HR reserve | 20-30 min | 建立習慣，適應訓練 |
| 第 5-12 週  | 50-70% HR reserve | 30-45 min | 逐漸增加強度與時間 |
| 第 13-36 週 | 60-80% HR reserve | 40-60 min | 維持或持續進步     |

**阻力訓練 Resistance Training**

::: warning 介入時機
阻力訓練應在有氧運動適應後（通常第 4-6 週）才開始加入。

| 參數     | 處方建議                                 |
| -------- | ---------------------------------------- |
| 強度     | 10-15 RM (Repetitional Maximum)          |
| 組數     | 1-3 組                                   |
| 頻率     | 每週 2-3 次，非連續日                    |
| 目標肌群 | 主要肌群（胸、背、肩、上臂、下肢、核心） |
| 呼吸技巧 | 避免Valsalva maneuver，用力時呼氣        |
| 初始階段 | 低阻力、高次數、建立正確動作模式         |

:::

**常見阻力訓練動作**

| 動作              | 目標肌群           | 備註               |
| ----------------- | ------------------ | ------------------ |
| Leg press         | 股四頭肌、臀肌     | 避免過深蹲         |
| Chest press       | 胸大肌、三角肌前束 | 可使用機器或啞鈴   |
| Lat pulldown      | 闊背肌             | 避免過度後仰       |
| Seated row        | 菱形肌、中斜方肌   | 改善姿勢           |
| Shoulder press    | 三角肌             | 注意肩關節活動度   |
| Biceps curl       | 肱二頭肌           | 輕重量高次數       |
| Triceps extension | 肱三頭肌           | 避免完全鎖定肘關節 |
| Core / abdominal  | 腹直肌、腹斜肌     | 避免仰臥起坐初期   |

#### 3. 緩和（5-10 分鐘）

- 逐漸降低運動強度
- 靜態伸展
- 目的：防止低血壓、降低心律不整風險

</details>

<details>
<summary>監測要求 Monitoring Requirements</summary>

| 監測項目         | 頻率                      |
| ---------------- | ------------------------- |
| 12-lead ECG      | Phase II 初期應有連續監測 |
| Blood pressure   | 每次訓練前、中、後        |
| Heart rate       | 每次訓練前與運動中        |
| RPE (Borg Scale) | 每 10-15 分鐘             |
| Symptoms         | 持續詢問                  |
| SpO₂             | 心衰竭患者應監測          |

</details>

<details>
<summary>心衰竭患者特殊考量 Heart Failure Considerations</summary>

心臟衰竭 (HFrEF) 患者的心臟復健需要特別注意：

| 考量事項       | 說明                               |
| -------------- | ---------------------------------- |
| 起始強度較低   | 從 40% HR reserve 或 RPE 9-11 開始 |
| 進展速度較慢   | 需更長的適應期                     |
| 立位低血壓風險 | 避免突然姿勢改變                   |
| 利尿劑使用     | 注意電解質平衡、補充水分           |
| 呼吸困難評估   | 使用 Modified Borg Dyspnea Scale   |
| 體重監測       | 每日體重，急劇增加需回報           |

</details>

## Phase III — 延續照護

<details>
<summary>目標 Objectives</summary>

- 維持 Phase II 達到的體能水準
- 從有監督過渡到自主運動
- 社區資源整合
- 長期健康行為建立

</details>

<details>
<summary>社區式復健 Community-Based Rehabilitation</summary>

| 模式         | 特點                         | 適用對象       |
| ------------ | ---------------------------- | -------------- |
| 社區運動班   | 醫師或治療師帶領，團體運動   | 中低風險患者   |
| 醫院門診追蹤 | 定期回診，運動指導           | 高風險患者     |
| 健身中心方案 | 有心臟復健資質的健身教練指導 | 低風險患者     |
| 運動APP追蹤  | 結合穿戴裝置，遠端監測       | 自律性高的患者 |

</details>

<details>
<summary>居家式復健 Home-Based Rehabilitation</summary>

居家復健適用於交通不便或偏好在家運動的患者：

- 個人化運動處方
- 定期電話或視訊追蹤
- 可搭配穿戴式裝置（如智慧手錶）
- 每 4-6 週進行功能性評估（如 6MWT）

::: warning 居家安全
**居家運動安全提醒 Home Exercise Safety**

- 確保運動環境安全、光線充足
- 事先準備飲用水
- 設定緊急聯絡人與方式
- 感到胸痛、嚴重呼吸困難、頭暈應立即停止

:::

</details>

## Phase IV — 維持期

<details>
<summary>目標 Objectives</summary>

- 終身維持規律運動習慣
- 降低心血管事件再發率
- 維持生活品質
- 自我健康管理

</details>

<details>
<summary>促進長期 adherence 的策略 Strategies for Long-Term Adherence</summary>

| 策略         | 具體做法                             |
| ------------ | ------------------------------------ |
| 目標設定     | 設定個人化、可達成的短期與長期目標   |
| 社會支持     | 加入患者支持團體、與家人共同運動     |
| 行為改變技巧 | 自我監測、問題解決、認知重建         |
| 活動多樣化   | 輪替不同運動型式（步行、游泳、騎車） |
| 環境調整     | 設定運動提醒、準備運動裝備           |
| 回饋機制     | 定期體能測試、數據追蹤               |

</details>

<details>
<summary>長期追蹤 Follow-Up</summary>

| 頻率        | 內容                             |
| ----------- | -------------------------------- |
| 每 3-6 個月 | 症狀評估、運動習慣確認           |
| 每年        | CPET 或 6MWT、心血管風險因子評估 |
| 必要時      | 調整運動處方、藥物評估           |

</details>

## 預防措施與警告信號

<details>
<summary>絕對應停止運動的信號 Absolute Indications to Stop Exercise</summary>

::: warning 立即停止
**立即停止運動並就醫！**
| 信號 | 說明 |
| -------------- | -------------------------------------------------- |
| 胸痛或胸悶 | Chest pain or discomfort |
| 嚴重呼吸困難 | Severe dyspnea not proportional to exercise |
| 頭暈或接近昏厥 | Dizziness or near-syncope |
| 心律不整 | Palpitations or irregular heartbeat |
| 面色蒼白或發紺 | Pallor or cyanosis |
| 運動後過度疲勞 | Excessive fatigue lasting > 1 hour post-exercise |
| 血壓異常反應 | Systolic BP drop > 10 mmHg with increased workload |
| ST 段改變 | ECG changes suggesting ischemia |
:::

</details>

<details>
<summary>相對應降低強度的信號 Relative Indications to Reduce Intensity</summary>

| 信號               | 說明                                      |
| ------------------ | ----------------------------------------- |
| 輕微呼吸困難       | Mild dyspnea                              |
| 異常疲勞           | Unusual fatigue                           |
| 頭痛               | Headache                                  |
| 下肢關節或肌肉不適 | Leg joint or muscle discomfort            |
| 心率過度升高       | Heart rate response greater than expected |

</details>

## 循證醫學證據 Evidence

<details>
<summary>主要研究結論 Key Evidence Summary</summary>

| 研究/統合分析                                | 主要發現                                                   |
| -------------------------------------------- | ---------------------------------------------------------- |
| **Cochrane Review (2016)** — Anderson et al. | CR 降低心血管死亡率 (RR 0.76) 及非致命性心肌梗塞 (RR 0.82) |
| **SMARTEX-HF Trial (2011)**                  | 運動訓練改善 HFrEF 患者 VO₂ peak，降低再住院率             |
| **Cochrane Review (2020)** — Dalal et al.    | 針對心衰竭患者，CR 改善生活品質與運動能力                  |
| **AHA/ACC Guidelines (2021)**                | 建議所有適合的心血管患者參與 CR (Class I, Level A)         |
| **ECSC Guidelines (2023)**                   | 歐洲心臟復健標準，強調多學科團隊合作                       |

</details>

<details>
<summary>統合分析數據 Meta-Analysis Data</summary>

| 結果指標         | 效應量 (95% CI)          |
| ---------------- | ------------------------ |
| 心血管死亡率降低 | RR 0.76 (0.64–0.90)      |
| 全因死亡率降低   | RR 0.87 (0.75–1.01)      |
| 再住院率降低     | RR 0.82 (0.70–0.96)      |
| 生活品質改善     | SMD 0.54 (0.30–0.78)     |
| VO₂ peak 提升    | +2.5 mL/kg/min (1.8–3.2) |

::: warning 實證結論
**Cochrane Review 結論 (Anderson et al., 2016)**
心臟復健與僅接受標準照護相比，顯著降低心血管死亡風險 (RR 0.76, 95% CI 0.64-0.90)，並改善運動能力與生活品質。建議所有符合條件的患者參與心臟復健計畫。
:::

</details>

## 心衰竭心臟復健的特別考量 Heart Failure Rehabilitation

<details>
<summary>SMARTEX-HF 試驗</summary>

SMARTEX-HF (Exercise Training in Congestive Heart Failure) 是一項重要的多中心隨機對照試驗：

| 試驗資訊 | 內容                            |
| -------- | ------------------------------- |
| 設計     | 多中心 RCT                      |
| 樣本     | HFrEF 患者 (LVEF ≤ 35%)         |
| 介入     | 12 週有監督的有氧運動           |
| 主要結果 | VO₂ peak 顯著提升、再住院率降低 |
| 隨訪     | 12 個月追蹤                     |

</details>

<details>
<summary>HFrEF 運動處方建議</summary>

| 參數     | 處方                           |
| -------- | ------------------------------ |
| 適應症   | 穩定的 NYHA Class II-III       |
| 起始強度 | 40-50% HR reserve 或 RPE 9-11  |
| 目標強度 | 60-70% HR reserve 或 RPE 12-14 |
| 有氧時間 | 30-45 分鐘（含熱身緩和）       |
| 頻率     | 每週 3-5 次                    |
| 阻力訓練 | 低重量、高次數（12-15 RM）     |

</details>
