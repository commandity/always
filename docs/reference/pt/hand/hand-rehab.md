# 手部復健 **Hand Rehabilitation**

手部是人體最精密的運動結構之一，包含 27 塊骨頭、超過 30 條肌肉以及複雜的肌腱和神經系統。手部功能障礙會嚴重影響患者的日常生活活動能力與生活品質。本文提供手部復健的全面性指南，涵蓋常見適應症、評估方法、分期治療計畫及具體運動處方。

## 適應症 **Indications**

手部復健適用於多種損傷與疾病狀態，包括但不限於以下常見適應症：

| 適應症                              | 常見病因                     | 預期復健目標                       |
| :---------------------------------- | :--------------------------- | :--------------------------------- |
| 肌腱修復 Tendon repair              | 割傷、撕裂傷                 | 恢復肌腱滑動能力、避免沾黏         |
| 腕隧道症候群 Carpal tunnel syndrome | 長期重複性動作、解剖構造異常 | 減輕神經壓迫症狀、恢復正中神經功能 |
| 骨折 Fractures                      | 跌倒、運動傷害、職業傷害     | 恢復關節活動度、骨癒合後肌力重建   |
| 截肢 Amputation                     | 外傷、疾病                   | 適配義肢、殘肢照護、功能性訓練     |
| 燒傷 Burns                          | 熱傷害、化學傷害             | 瘢痕管理、預防攣縮、恢復關節活動度 |
| 手部關節炎 Hand arthritis           | 類風濕性關節炎、退化性關節炎 | 減輕疼痛、維持功能、保護關節       |
| 手部肌腱炎 Tendinopathy             | 重複性壓力傷害               | 減輕炎症、恢復肌腱強度             |

## 解剖構造 **Anatomy**

### 屈肌腱系統 **Flexor Tendon System**

手部屈肌腱系統分為外在肌與內在肌兩大部分：

- **外在肌 (Extrinsic muscles)**：起源於前臂，提供主要的握力功能
  - 淺指屈肌 (Flexor digitorum superficialis, FDS)：屈曲近端指間關節
  - 深指屈肌 (Flexor digitorum profundus, FDP)：屈曲遠端指間關節
  - 拇長屈肌 (Flexor pollicis longus, FPL)：屈曲拇指指間關節

- **屈肌腱鞘 (Flexor tendon sheath)**：保護肌腱並提供營養
  - A1-A5 環狀滑車 (Annular pulleys)：防止弓弦現象
  - C1-C3 交叉滑車 (Cruciate pulleys)：允許肌腱在屈曲時適應

### 伸肌腱系統 **Extensor Tendon System**

伸肌腱系統位於手背側，結構較屈肌腱扁平：

- **外在伸肌**：
  - 橈側伸腕短肌 (Extensor carpi radialis brevis, ECRB)
  - 尺側伸腕肌 (Extensor carpi ulnaris, ECU)
  - 指總伸肌 (Extensor digitorum communis, EDC)

- **伸肌腱膜 (Extensor apparatus)**：包含中央束 (Central slip) 及外側束 (Lateral band)

### 腕隧道 **Carpal Tunnel**

腕隧道是由腕骨與屈肌支持帶 (Flexor retinaculum) 構成的骨纖維管道，通過的結構包括：

- 9 條屈肌腱（4 條 FDS、4 條 FDP、1 條 FPL）
- 正中神經 (Median nerve)

### 內在肌 **Intrinsic Muscles**

- **蚓狀肌 (Lumbricals)**：屈曲掌指關節同時伸展指間關節
- **骨間肌 (Interossei)**：手指外展與內收
- **拇指肌群 (Thenar muscles)**：包括拇短展肌、拇短屈肌、拇指對掌肌

## 評估方法 **Assessment Methods**

### 握力評估 **Grip Strength Assessment**

使用 **Jamar 握力計 (Jamar dynamometer)** 進行標準化測試：

| 測試位置          | 手柄位置 | 適用情況                   |
| :---------------- | :------- | :------------------------- |
| 第一階 Position 1 | 最窄     | 小型手掌                   |
| 第二階 Position 2 | 標準     | 大多數成人（標準測試位置） |
| 第三階 Position 3 | 中等     | 較大手掌                   |
| 第四階 Position 4 | 較寬     | 大型手掌                   |
| 第五階 Position 5 | 最寬     | 特大手掌                   |

::: warning
**注意**：測試時應採用標準姿勢——肩關節內收、肘關節屈曲 90 度、前臂中立位、腕關節中立位 0-30 度伸展。避免以「用力握」的口頭指示取代「盡可能用力擠壓」的標準化指令。
:::

### 捏力評估 **Pinch Strength Assessment**

使用 **B&L 捏力計 (Pinch gauge)** 測試：

| 捏力類型                    | 測試方法             | 主要肌肉           |
| :-------------------------- | :------------------- | :----------------- |
| 拇指-食指側捏 Lateral pinch | 拇指腹與食指橈側     | 拇長伸肌、拇短展肌 |
| 拇指-食指指尖捏 Tip pinch   | 拇指腹與食指指腹     | 拇長屈肌、指深屈肌 |
| 三指捏 Tripod pinch         | 拇指、食指、中指指腹 | 多條肌肉協同       |

### 關節活動度測量 **Range of Motion (ROM)**

使用 **量角器 (Goniometer)** 測量：

| 關節             | 屈曲正常範圍 | 伸展正常範圍 |
| :--------------- | :----------- | :----------- |
| 腕關節 Wrist     | 0-80° (掌屈) | 0-70° (背伸) |
| 掌指關節 MCP     | 0-90°        | 0-45°        |
| 近端指間關節 PIP | 0-110°       | 0°           |
| 遠端指間關節 DIP | 0-80°        | 0°           |
| 拇指腕掌關節 CMC | 前對掌 0-70° | 後對掌 0-70° |

### 功能量表 **Functional Outcome Measures**

**DASH 殘疾量表 (Disabilities of the Arm, Shoulder and Hand)**

- 30 個項目評估上肢功能
- 每項 1-5 分，總分轉換為 0-100 分
- 分數越高表示功能障礙越嚴重
- 包含可選的運動量表與工作量表

**密西根手部問卷 (Michigan Hand Questionnaire, MHQ)**

- 6 個維度：整體手部功能、日常活動、工作能力、疼痛、外觀、滿意度
- 分別評估雙手
- 針對手部功能的特異性更高

::: warning
**臨床提示**：建議同時使用通用量表 (DASH) 與疾病特異性量表 (MHQ)，以全面評估手部功能。初次評估應建立基線，並在治療過程中定期追蹤。
:::

## 分期復健計畫 **Phased Rehabilitation Protocol**

### 第一階段：保護期 (0-3 週) **Phase 1: Protection**

**目標**：保護修復組織、控制水腫、預防關節僵硬、開始早期被動運動。

**一般原則**：

- 維持手部抬高高於心臟水平
- 使用背側阻斷夾板 (Dorsal blocking splint) 固定
- 避免主動屈曲以保護修復的肌腱
- 開始被動關節活動度練習

**夾板固定原則 Splinting Principles**：

| 夾板類型                   | 功能                         | 適應症               |
| :------------------------- | :--------------------------- | :------------------- |
| 休息夾板 Resting splint    | 固定關節於功能位、保護軟組織 | 急性期、燒傷預防攣縮 |
| 動態夾板 Dynamic splint    | 利用彈力提供漸進式牽引       | 關節攣縮、肌腱延長   |
| 功能夾板 Functional splint | 允許特定動作同時保護結構     | 恢復期功能性訓練     |

**水腫控制 Edema Management**：

1. **抬高 (Elevation)**：患肢高於心臟水平，每天至少 20 分鐘
2. **加壓 (Compression)**：使用彈性繃帶或壓力手套 (Pressure garment)
3. **逆行性按摩 (Retrograde massage)**：從遠端向近端輕柔推摩
4. **肌肉幫浦 (Muscle pump)**：鼓勵輕柔的握拳-鬆手動作（在允許範圍內）
5. **冰敷 (Cryotherapy)**：每次 15-20 分鐘，避免直接接觸皮膚

**被動關節活動度練習**：

- 由治療師或患者健側手輔助
- 每個關節緩慢被動活動至輕微緊繃感
- 每天 2-3 回合，每回合 5-10 次

**肌腱滑動練習 (Tendon gliding exercises)**：

此階段僅進行被動式肌腱滑動，按以下順序執行：

1. **直拳 Straight fist**：MP 屈曲、IP 伸直
2. **鉤拳 Hook fist**：MP 伸直、PIP/DIP 屈曲
3. **全拳 Full fist**：MP/PIP/DIP 全屈曲
4. **桌面平放 Tabletop**：MP 伸直、PIP 屈曲 90°、DIP 伸直

::: warning
**警告**：肌腱修復後前 3 週內，禁止進行抗阻主動屈曲動作，以避免肌腱斷裂風險。所有活動應在治療師指導下進行。
:::

---

### 第二階段：活動期 (3-6 週) **Phase 2: Mobilization**

**目標**：逐步恢復主動關節活動度、開始瘢痕管理、進行脫敏訓練。

**關節活動度練習**：

- 從主動輔助 (Active-assisted) 逐步過渡至主動 (Active) ROM
- 開始進行 **差異滑動練習 (Differential gliding exercises)**
- 確保 FDS 與 FDP 的獨立活動能力

**差異滑動練習 Differential Gliding**：

| 練習動作   | 方法                            | 目標結構   |
| :--------- | :------------------------------ | :--------- |
| FDS 滑動   | 固定 DIP 於伸直位，主動屈曲 PIP | 淺指屈肌腱 |
| FDP 滑動   | 固定 PIP 於伸直位，主動屈曲 DIP | 深指屈肌腱 |
| 伸肌腱滑動 | 被動屈曲 MP，主動伸展 IP        | 伸肌腱     |

**阻指練習 Block Fingers**：

- 用健手固定一指於伸直位
- 被動或主動輔助活動其他手指
- 此練習有助於分離各指的肌腱活動

**瘢痕管理 Scar Management**：

- **硅膠貼片 (Silicone gel sheeting)**：每天 12 小時以上
- **瘢痕按摩 (Scar massage)**：使用乳液進行橫向與縱向摩擦
- **壓力治療 (Pressure therapy)**：定制壓力手套
- **關節周圍軟組織鬆動術**

**脫敏訓練 Desensitization**：

適用於截肢後、燒傷後或神經損傷後的感覺過敏：

1. 輕觸刺激（棉球、毛刷）
2. 不同質地材料接觸（棉布、絲綢、羊毛）
3. 振動刺激
4. 逐步增加接觸壓力

::: warning
**臨床提示**：此階段應密切觀察肌腱修復的完整性。若出現突發性疼痛、腫脹加劇或主動活動度突然下降，應立即停止並重新評估。
:::

---

### 第三階段：強化期 (6-12 週) **Phase 3: Strengthening**

**目標**：重建握力與捏力、恢復精細動作控制、開始功能性訓練。

**漸進式強化訓練 Progressive Strengthening**：

| 週數     | 訓練強度 | 訓練方式                 |
| :------- | :------- | :----------------------- |
| 6-8 週   | 輕度     | 橡皮筋抗阻、黏土塑形     |
| 8-10 週  | 中度     | 握力器訓練、加重練習     |
| 10-12 週 | 中重度   | 功能性握力練習、工作模擬 |

**握力強化練習 Grip Strengthening**：

- **全握 Full grip**：握持不同直徑的物體（瓶蓋→網球→握力器）
- **側捏 Lateral pinch**：使用捏力計或夾持紙張
- **指尖捏 Tip pinch**：拾取小物體（珠子、硬幣）
- **三指捏 Tripod pinch**：握筆、使用工具

**精細動作訓練**：

- 撿拾小物體（米粒、豆子）
- 按扣、鈕扣操作
- 拉鍊操作
- 书写、繪圖練習
- 使用筷子

**功能性訓練**：

- 日常活動模擬（開門、擰瓶蓋、握持餐具）
- 職業相關動作訓練
- 3D 列印物體操作練習
- 手工艺活動

::: warning
**進階提示**：強化期應以「無痛」為原則。若訓練後出現持續性疼痛或次日關節腫脹，應降低訓練強度。
:::

---

### 第四階段：功能回歸期 (>12 週) **Phase 4: Return to Function**

**目標**：回歸工作或運動、預防再傷害、維持手部功能。

**運動專項訓練 Sport-Specific Training**：

| 運動項目  | 重點訓練內容             |
| :-------- | :----------------------- |
| 球類運動  | 握拍力量、快速反應抓握   |
| 攀岩      | 指力強度、耐力握持       |
| 鋼琴/樂器 | 手指獨立性、速度與精確度 |
| 重量訓練  | 握力耐力、腕關節穩定性   |

**工作回歸評估 Work-Specific Training**：

- 評估工作中的手部需求
- 模擬工作動作進行漸進式訓練
- 工作體能測試 (Work hardening)
- 人因工程學建議

**預防再傷害策略**：

- 伸展與熱身計畫
- 休息間隔建議
- 正確姿勢與技巧教育
- 自我監測與何時求醫的指引

## 特殊練習詳解 **Specific Exercises**

### 肌腱滑動練習 Tendon Gliding Exercises

此組練習旨在促進屈肌腱在腱鞘內的滑動，預防沾黏形成。建議每天進行 3-4 回合，每回合 10 次。

```
順序：直拳 → 鉤拳 → 全拳 → 桌面平放 → 直拳
       ↓       ↓       ↓         ↓         ↓
     (1)     (2)     (3)       (4)       (1)
```

1. **直拳 Straight fist**：MP 屈曲 90°、PIP/DIP 伸直
2. **鉤拳 Hook fist**：MP 伸直、PIP/DIP 全屈曲
3. **全拳 Full fist**：MP/PIP/DIP 全屈曲
4. **桌面平放 Tabletop**：MP 伸直、PIP 屈曲 90°、DIP 伸直

### 差異滑動練習 Differential Tendon Gliding

**FDP 滑動**：

- 用健手固定患指 PIP 於完全伸直位
- 主動屈曲 DIP 至最大範圍
- 維持 3 秒後緩慢放鬆
- 每指 10 次 × 3 回合

**FDS 滑動**：

- 將所有手指的 DIP 固定於伸直位（可輕壓指背）
- 主動屈曲 PIP 至最大範圍
- 維持 3 秒後緩慢放鬆
- 每指 10 次 × 3 回合

### 握力強化練習 Grip Strengthening Progression

| 階段 | 方法         | 重複次數     | 頻率        |
| :--- | :----------- | :----------- | :---------- |
| 初期 | 橡皮筋伸展   | 15-20 次     | 每天 3 次   |
| 中期 | 握力球擠壓   | 10-15 次     | 每天 2-3 次 |
| 進階 | 彈簧握力器   | 8-12 次      | 每天 2 次   |
| 維持 | 可調式握力器 | 10 次 × 3 組 | 每天 1 次   |

### 捏力練習 Pinch Exercises

- **硬幣拾取**：從桌面拾取不同面值硬幣放入容器
- **橡皮筋拉伸**：拇指與各指捏持橡皮筋向外拉
- **黏土操作**：捏、拉、搓揉治療用黏土
- **夾持練習**：使用衣夾夾持紙張或布料

## 水腫管理原則 **Edema Management Principles**

手部水腫是影響功能恢復的重要因素。以下為綜合性水腫管理策略：

### 基本原則

| 方法                          | 技術要點             | 時間/頻率                  |
| :---------------------------- | :------------------- | :------------------------- |
| 抬高 Elevation                | 患肢高於心臟水平     | 每次 20 分鐘，每天 4-6 次  |
| 加壓 Compression              | 彈性繃帶或壓力手套   | 全天穿戴（移除清潔時除外） |
| 逆行性按摩 Retrograde massage | 從指尖向手腕方向推摩 | 每次 5-10 分鐘             |
| 肌肉幫浦 Muscle pump          | 緩慢握拳-鬆手動作    | 每小時 10-15 次            |
| 冰敷 Cryotherapy              | 冰袋包裹毛巾         | 每次 15-20 分鐘            |

### 進階水腫管理

- **對比浴 Contrast bath**：交替使用溫水 (38-40°C) 與冷水 (10-15°C)，各浸泡 3-5 分鐘
- **Coban 心型包紮法**：使用自黏彈性繃帶進行心型 (Figure-of-eight) 包紮
- **手部壓力手套**：定制 20-30 mmHg 壓力手套

::: warning
**禁忌症**：活動性感染、未控制的關節炎急性發作、深靜脈血栓、區域性疼痛症候群 (Complex regional pain syndrome, CRPS) 需謹慎使用加壓治療。
:::

## 證據基礎 **Evidence Base**

### Cochrane 系統性回顧

- **肌腱修復後 rehabilitation protocols**：早期被動運動與延遲運動的比較研究顯示，早期活動組的功能結果較佳，但肌腱斷裂風險略高 (Cochrane Database Syst Rev, 2015)
- **腕隧道症候群 conservative treatment**：夾板固定在症狀輕至中度患者中效果顯著，尤其是夜間穿戴 (Cochrane Database Syst Rev, 2020)
- **Hand therapy interventions for burns**：壓力治療配合矽膠產品對燒傷瘢痕管理有中度品質證據支持

### 證據等級建議

| 介入措施               | 證據等級  | 建議強度 |
| :--------------------- | :-------- | :------- |
| 肌腱修復後早期被動運動 | Level I   | 強烈建議 |
| 腕隧道夾板固定         | Level I   | 強烈建議 |
| 壓力手套治療瘢痕       | Level II  | 中度建議 |
| 動態夾板治療關節攣縮   | Level III | 中度建議 |
| 脫敏訓練               | Level IV  | 建議使用 |

::: warning
**循證實踐提醒**：手部復健的介入措施應基於最新證據、臨床專業判斷及患者偏好。治療計畫應個體化，並根據患者的反應與進展動態調整。
:::

## 常見併發症與處理 **Common Complications and Management**

| 併發症                     | 臨床表現                           | 處理策略                           |
| :------------------------- | :--------------------------------- | :--------------------------------- |
| 肌腱沾黏 Tendon adhesion   | ROM 下降、活動時緊繃感             | 加強滑動練習、超音波治療、手術鬆解 |
| 關節攣縮 Joint contracture | 無法達到完全伸直或屈曲             | 動態夾板、漸進式牽引、軟組織鬆動   |
| 區域性疼痛症候群 CRPS      | 過度疼痛、水腫、皮膚變色、運動受限 | 早期介入、藥物治療、階梯式脫敏     |
| 再斷裂 Re-rupture          | 突發性疼痛、主動活動能力喪失       | 立即就醫、可能需再次手術           |
| 瘢痕攣縮 Scar contracture  | 瘢痕增厚、皮膚緊繃、關節受限       | 壓力治療、矽膠、牽引、手術鬆解     |
| 神經卡壓 Nerve entrapment  | 感覺異常、刺痛、肌肉無力           | 神經滑動練習、夾板、手術減壓       |

::: warning
**警示**：出現以下情況應立即聯繫醫療團隊：修復部位突發劇痛、無法主動屈曲手指、手部顏色改變或溫度異常、感覺完全喪失。
:::

## 患者衛教 **Patient Education**

### 自我照護要點

1. **冰敷時機**：訓練後 15 分鐘內開始冰敷，每次不超過 20 分鐘
2. **抬高時機**：任何引起腫脹的活動後立即抬高患肢
3. **睡眠姿勢**：使用枕頭支撐患手，維持功能位
4. **營養補充**：充足蛋白質、維生素 C 促進傷口癒合
5. **戒菸**：吸菸顯著延緩肌腱癒合與傷口復原

### 何時恢復日常活動

| 活動類型                 | 建議恢復時間 | 注意事項               |
| :----------------------- | :----------- | :--------------------- |
| 輕度家務 Light housework | 4-6 週       | 避免用力扭絞動作       |
| 駕駛 Driving             | 6-8 週       | 確認握力與反應速度足夠 |
| 打字/滑鼠 Keyboard/mouse | 2-4 週       | 使用人因工程輔具       |
| 重物搬運 Heavy lifting   | 8-12 週      | 重量漸進增加           |
| 運動/嗜好 Sports/hobbies | 10-16 週     | 依個別需求評估         |

## 治療結果監測 **Outcome Monitoring**

建議在以下時間點進行功能追蹤評估：

1. **初評 Baseline**：治療開始前
2. **第 3 週**：保護期結束、進入活動期前
3. **第 6 週**：活動期結束、進入強化期前
4. **第 12 週**：強化期結束、評估是否可回歸功能活動
5. **第 6 個月**：追蹤長期功能維持狀況

每次追蹤應記錄：

- 握力與捏力百分比（與健側比較）
- 主動 ROM 測量值
- DASH 或 MHQ 分數
- 患者自覺功能等級 (Patient-rated evaluation)
