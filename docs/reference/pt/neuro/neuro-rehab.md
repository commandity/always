# 神經復健 **Neurological Rehabilitation**

## 概述 **Overview**

神經復健是針對中樞神經系統損傷後所引起的運動、感覺、認知及功能障礙，以系統性介入策略促進功能恢復與社會參與之專業領域。

**Neurological rehabilitation** encompasses a structured, evidence-based approach to restoring function, reducing disability, and maximizing independence following insult to the central nervous system (CNS). The three primary conditions addressed include:

| 疾病 **Condition**                            | 病因 **Etiology**                                          | 常見障礙 **Common Deficits**                                                       |
| --------------------------------------------- | ---------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| 腦中風 **Cerebrovascular Accident (CVA)**     | Ischemic or hemorrhagic inflection of cerebral vasculature | Hemiplegia, aphasia, neglect, dysphagia, cognitive impairment                      |
| 創傷性腦損傷 **Traumatic Brain Injury (TBI)** | Blunt or penetrating head trauma                           | Motor deficits, post-traumatic amnesia, behavioural changes, cranial nerve palsies |
| 脊髓損傷 **Spinal Cord Injury (SCI)**         | Traumatic or non-traumatic lesion to spinal cord           | Paraplegia/tetraplegia, autonomic dysreflexia, neurogenic bladder/bowel            |

::: warning 警告
**重要提醒 Important Notice**
神經復健需多學科團隊（physical therapist, occupational therapist, speech-language pathologist, physiatrist, neuropsychologist）協同介入。任何急性徵兆（意識改變、新發神經缺損、劇烈頭痛）應立即停止訓練並通報主治醫師。
:::

## 神經生理機制 **Neurophysiological Mechanisms**

### 神經可塑性 **Neuroplasticity**

神經可塑性（**neuroplasticity**）是神經復健之核心理論基礎，指中樞神經系統因應經驗、學習與損傷而產生結構與功能重組之能力。

| 機制 **Mechanism**               | 說明 **Description**                                        | 臨床應用 **Clinical Application**          |
| -------------------------------- | ----------------------------------------------------------- | ------------------------------------------ |
| **Cortical reorganization**      | 短皮質脊髓徑（CST）之突觸效率增加；對側及同側半球代償性活化 | Constraint-induced movement therapy (CIMT) |
| **Axonal sprouting**             | 健側軸突新生側芽支配失能區域之突觸靶點                      | 強迫性使用療法、.task-specific training    |
| **Unmasking of latent pathways** | 原本受抑制的替代性神經通路因損傷後去抑制而活化              | FES、rTMS、tDCS 輔助治療                   |
| **Synaptic strengthening (LTP)** | Hebbian learning — neurons that fire together wire together | 重複性任務練習、高強度訓練                 |

### 動作學習理論 **Motor Learning**

動作學習（**motor learning**）指透過練習與經驗獲得持久性動作能力改變之過程。其關鍵要素包括：

- **Task-specific training（任務導向訓練）**：練習內容應與目標功能高度相似
- **Variable practice（變化性練習）**：在不同環境、情境下練習同一任務
- **Feedback（回饋）**：trinsic feedback（結果回饋）與 intrinsic feedback（表現回饋）之平衡
- **Repetition & intensity（重複量與強度）**：足夠的練習量是神經重組的必要條件
- **Salience（意義性）**：對個案具功能性意義之任務提升學習動機與保留效果

### 任務導向訓練原則 **Task-Specific Training Principles**

1. **Meaningful, goal-directed tasks** — 以功能性任務為核心
2. **Repetitive practice** — 高重複量（建議 ≥ 450 次/訓練單元之重複性動作）
3. **Progressive task difficulty** — 依能力調整挑戰性
4. **Ecological validity** — 在接近真實生活的情境下練習
5. **Active problem solving** — 鼓勵個案主動探索解決動作策略

### 脊髓損傷專用評估 **SCI-Specific Assessment**

| 等級 **Grade**             | 定義 **Definition**                                        |
| -------------------------- | ---------------------------------------------------------- |

## 第一階段：急性期 **Phase 1: Acute (0–2 Weeks)**

此階段以預防併發症、維持關節活動度、促進神經恢復之早期刺激為核心目標。

::: warning 警告
**急性期注意事項 Acute Phase Precautions**

- 監測血流動力學穩定性（BP ≥ 90/60 mmHg, HR 60–100 bpm, SpO₂ ≥ 94%）
- 頭部外傷個案需排除顱內壓升高（ICP > 20 mmHg 為禁忌）
- 脊髓損傷急性期需維持脊椎穩定，避免不當搬動
- 體位性低血壓：漸進式姿勢轉換（床頭抬高 ≤ 30° 起始，每 1–2 天增加 10–15°）
- 深靜脈血栓（DVT）預防：穿著彈性襪、被動運動
  :::

### 定位與體位管理 **Positioning & Postural Management**

| 目標 **Goal**    | 方法 **Method**                                 | 頻率 **Frequency** |
| ---------------- | ----------------------------------------------- | ------------------ |
| 預防關節攣縮     | 功能位擺位（手肘伸展、腕背屈 20–30°、手指微屈） | 每 2 小時翻身/換位 |
| 預防褥瘡         | 定時翻身（30° 側臥交替）；使用減壓床垫          | 每 2 小時          |
| 維持心肺功能     | Semi-Fowler 位（床頭抬高 30–45°）               | 持續               |
| 預防肩關節半脫位 | 患側上肢以枕頭支撐，避免垂吊                    | 持續               |
| 抑制痙攣         | 抗痙攣擺位（患側伸展模式）                      | 持續               |

### 被動關節活動度運動 **Passive Range of Motion (PROM)**

| 運動 **Exercise**                                        | 方法 **Method**                                                                | 處方 **Dosage**                        |
| -------------------------------------------------------- | ------------------------------------------------------------------------------ | -------------------------------------- |
| 肩關節屈曲/外展 **Shoulder flexion/abduction**           | 治療師以一手固定肩胛骨，另一手引導肱骨進行全範圍屈曲（0–180°）及外展（0–120°） | 10 次/組，2 組/日，每關節 10–15 次重複 |
| 肩關節內/外旋 **Shoulder internal/external rotation**    | 肩關節外展 90°，肘屈 90°，緩慢進行內旋與外旋                                   | 10 次/組，2 組/日                      |
| 肘關節屈曲/伸展 **Elbow flexion/extension**              | 固定上臂，緩慢屈伸肘關節至終端感                                               | 10 次/組，2 組/日                      |
| 前臂旋前/旋後 **Forearm pronation/supination**           | 肘屈 90°，進行前臂旋轉                                                         | 10 次/組，2 組/日                      |
| 腕關節屈曲/伸展 **Wrist flexion/extension**              | 固定前臂，被動屈伸腕關節                                                       | 10 次/組，2 組/日                      |
| 髖關節屈曲/外展/內收 **Hip flexion/abduction/adduction** | 仰臥位進行屈曲（0–120°）、外展（0–45°）、內收                                  | 10 次/組，2 組/日                      |
| 膝關節屈曲/伸展 **Knee flexion/extension**               | 仰臥位，被動屈膝（0–130°）                                                     | 10 次/組，2 組/日                      |
| 踝關節背屈/蹠屈 **Ankle dorsiflexion/plantarflexion**    | 固定小腿，緩慢背屈（0–20°）及蹠屈（0–50°）                                     | 10 次/組，2 組/日                      |
| 趾關節屈曲/伸展 **Toe flexion/extension**                | 輕柔屈伸所有趾關節                                                             | 10 次/組，2 組/日                      |

::: info 資訊
**PROM 注意事項**

- 動作應緩慢、平穩，避免快速牽張引發痙攣
- 在終端感（end-feel）處停留 5–10 秒以進行靜態牽張
- 避免在肩關節外展 > 90° 時進行內旋（预防肩夾擠）
- 如個案出現明顯阻力（改良 Ashworth ≥ 2），減少活動範圍至無痛範圍
  :::

### 感覺刺激 **Sensory Stimulation**

| 刺激類型 **Stimulation Type**                | 方法 **Method**                                             | 處方 **Dosage**         |
| -------------------------------------------- | ----------------------------------------------------------- | ----------------------- |
| 本體感覺促進 **Proprioceptive facilitation** | 關節加壓（joint approximation）、肌腱敲擊（tendon tapping） | 每關節 5–10 次，2 組/日 |
| 觸覺刺激 **Tactile stimulation**             | 不同質地物品刷拂患側皮膚（冰、毛刷、棉花）                  | 每次 5–10 分鐘，2 次/日 |
| 前庭刺激 **Vestibular stimulation**          | 滾木運動（rolling）、緩慢體位變換                           | 每次 10–15 分鐘         |
| 視覺刺激 **Visual stimulation**              | 視線追蹤練習、雙眼協調訓練                                  | 每次 10–15 分鐘         |

### 早期床上活動 **Early Bed Mobility**

| 運動 **Exercise**                  | 方法 **Method**                                      | 處方 **Dosage**     |
| ---------------------------------- | ---------------------------------------------------- | ------------------- |
| 橋式運動 **Bridging**              | 仰臥，雙膝屈曲，抬臀維持 5 秒後緩慢放下              | 10 次/組，2–3 組/日 |
| 翻身訓練 **Rolling**               | 側臥位：上側腳置於下側腳前方，利用上肢及骨盆帶動旋轉 | 10 次/組，2 組/日   |
| 側臥至坐 **Side-lying to sitting** | 側臥位，雙腳垂於床緣，利用上肢推起至坐姿             | 5–10 次/組，2 組/日 |
| 仰臥至坐 **Supine to sitting**     | 先側臥，再以肘部及對側手支撐起身                     | 5–10 次/組，2 組/日 |

## 第二階段：亞急性期 **Phase 2: Subacute (2 Weeks–3 Months)**

此階段為神經恢復之關鍵窗口期（**critical window of recovery**），以主動運動訓練、任務導向練習、平衡及步態訓練、ADL 訓練為核心。

::: info 資訊
**亞急性期訓練原則 Subacute Training Principles**

- 由被動 → 主動輔助 → 主動 → 抗阻逐步進階
- 每次訓練至少 45–60 分鐘，建議每日 1–2 次
- 訓練強度應達個案能力之 60–80%（以自覺費力量表 RPE 12–14 / 6–20 量表）
- 目標設定：與個案共同訂定短期及長期功能性目標
  :::

### 上肢運動訓練 **Upper Limb Exercise**

| 運動 **Exercise**                                   | 方法 **Method**                                               | 處方 **Dosage**             |
| --------------------------------------------------- | ------------------------------------------------------------- | --------------------------- |
| 主動輔助肩屈曲 **Active-assisted shoulder flexion** | 患側手置於滾筒上，利用滾筒輔助進行肩屈曲                      | 10–15 次/組，3 組/日        |
| 肩胛骨穩定訓練 **Scapular stabilization**           | 四點跪位下，進行肩胛骨前突/後縮、上抬/下壓                    | 10 次/組，2 組/日           |
| 肘伸展抗阻 **Resisted elbow extension**             | 0.5–1 kg 啞鈴，仰臥位進行肘伸展                               | 10–15 次/組，3 組/日        |
| 前臂控制訓練 **Forearm control**                    | 前臂置於桌面，進行旋前/旋後控制練習                           | 10–15 次/組，3 組/日        |
| 抓握與釋放 **Grasp & release**                      | 抓取不同大小物體（網球、彈珠、紙夾）                          | 15–20 次/組，3 組/日        |
| 雙手協調訓練 **Bimanual coordination**              | 雙手同時進行揉捏麵團、擰毛巾、翻撲克牌                        | 每次 10–15 分鐘，2 次/日    |
| CIMT **Constraint-induced movement therapy**        | 健側手以手套/繃帶限制，強制使用患側手進行 6 小時/日功能性活動 | 2 週密集方案（6–8 小時/日） |

### 下肢運動訓練 **Lower Limb Exercise**

| 運動 **Exercise**                      | 方法 **Method**                              | 處方 **Dosage**      |
| -------------------------------------- | -------------------------------------------- | -------------------- |
| 直抬腿 **Straight leg raise (SLR)**    | 仰臥位，膝伸直抬高至 45°，維持 5 秒          | 10 次/組，3 組/日    |
| 橋式進階 **Bridging progression**      | 單腳橋式 → 穩定球橋式 → 抗阻橋式（彈力帶）   | 10 次/組，3 組/日    |
| 髋外展抗阻 **Resisted hip abduction**  | 側臥位，彈力帶套於膝上方，進行抗阻外展       | 10–15 次/組，3 組/日 |
| 膝伸展抗阻 **Resisted knee extension** | 坐姿，踝負重或彈力帶，進行膝伸展             | 10–15 次/組，3 組/日 |
| 踝背屈訓練 **Ankle dorsiflexion**      | 坐姿或仰臥，進行主動踝背屈（可加彈力帶抗阻） | 15–20 次/組，3 組/日 |
| 功能性蹲起 **Functional sit-to-stand** | 從不同高度椅子（50 cm → 45 cm → 40 cm）站起  | 10 次/組，3 組/日    |
| 踏步訓練 **Stepping training**         | 原地踏步 → 側踏 → 後踏 → 抗阻踏步            | 10–15 次/組，2 組/日 |

### 平衡訓練 **Balance Training**

| 運動 **Exercise**                        | 方法 **Method**                                    | 處方 **Dosage**         |
| ---------------------------------------- | -------------------------------------------------- | ----------------------- |
| 坐姿平衡 **Sitting balance**             | 坐於治療床緣，治療師给予前後左右輕推，訓練姿勢修正 | 每次 5–10 分鐘，3 次/日 |
| 坐→站轉移 **Sit-to-stand transfer**      | 雙手交叉置於胸前，從坐姿站起（可輔助）             | 10 次/組，3 組/日       |
| 站立平衡 **Standing balance**            | 助行器或平行杠內站立，重心左右/前後轉移            | 每次 5–15 分鐘，3 次/日 |
| 支撐面縮小 **Base-of-support reduction** | 站立：雙腳併攏 → T字站立 → 單腳站立                | 每次 10–30 秒，5–10 次  |
| 動態平衡 **Dynamic balance**             | 站立下進行體幹旋轉、側彎、觸碰對側腳趾             | 10 次/組，2 組/日       |
| 平衡板訓練 **Balance board training**    | 站立於平衡板/泡沫墊上維持平衡                      | 每次 2–5 分鐘，3 次/日  |

### 步態訓練 **Gait Training**

| 運動 **Exercise**                                                      | 方法 **Method**                                  | 處方 **Dosage**               |
| ---------------------------------------------------------------------- | ------------------------------------------------ | ----------------------------- |
| 平行杠內步行 **Parallel bar walking**                                  | 雙手扶杠，進行站立 → 邁步 → 擺動相練習           | 每次 10–20 分鐘，每日 1–2 次  |
| 助行器步訓 **Walker/crutch gait training**                             | 以四腳助行器或雙拐進行步態訓練（提示器律、步長） | 每次 10–20 分鐘，每日 1–2 次  |
| 減重 treadmilling **Body weight-supported treadmill training (BWSTT)** | 以懸吊系統減重 15–40% 體重，進行 treadmill 步訓  | 每次 20–30 分鐘，3–5 次/週    |
| 地面步態矯正 **Overground gait correction**                            | 治療師以手法提示骨盆控制、膝伸展、足廓清         | 每次 15–30 分鐘，每日 1 次    |
| 上下階梯 **Stair training**                                            | "好上壞下"（step-to pattern），扶手輔助          | 每次 2 層樓 × 3 組，每日 1 次 |

::: warning 警告
**步態訓練注意事項 Gait Training Precautions**

- 進行 BWSTT 時需 2 名以上治療師協助（1 名在後提示骨盆，1 名在側保護）
- 步態訓練中監測血壓（避免體位性低血壓）及心率
- 使用功能電刺激（FES）輔助步態時，注意電極片位置及皮膚反應
- 下肢深靜脈血栓（DVT）風險個案需先排除後方可進行高強度步訓
  :::

| ------------------------------------------- | -------------------------------------------------------------- | ------------------- |

## 第三階段：慢性期 **Phase 3: Chronic (>3 Months)**

此階段聚焦社區參與、體能維持、代償策略強化及次級預防。

::: info 資訊
**慢性期目標 Chronic Phase Goals**

- 維持或改善已恢復之功能
- 代償策略的熟練度與效率提升
- 體能（心肺適能、肌力）的持續訓練
- 社區參與：回歸工作、駕駛、休閒活動
- 次級預防：控制中風危險因子（高血壓、糖尿病、血脂異常）
  :::

## 特殊議題與併發症管理 **Special Considerations & Complication Management**

### 血壓監測 **Blood Pressure Monitoring**

| 情境 **Situation**                               | 監測要點 **Key Points**                                 | 處理 **Management**                                                 |
| ------------------------------------------------ | ------------------------------------------------------- | ------------------------------------------------------------------- |
| 體位性低血壓 **Orthostatic hypotension**         | 收縮壓下降 ≥ 20 mmHg 或舒張壓下降 ≥ 10 mmHg（3 分鐘內） | 漸進式角度訓練；腹帶/彈性襪；增加水分攝取                           |
| 高血壓反應 **Hypertensive response**             | 收縮壓 > 180 mmHg 或舒張壓 > 110 mmHg                   | 暫停高強度訓練；通知醫師；監測 Symonds 徵象（頭痛+心搏過緩+高血壓） |
| 自主神經反射失調 **Autonomic dysreflexia (SCI)** | T6 以上損傷；突發性高血壓、搏動性頭痛、出汗             | 立即坐起；移除誘發因子（尿管阻塞、緊繃衣物）；必要時藥物降壓        |

### 痙攣管理 **Spasticity Management**

| 介入方式 **Intervention**                       | 方法 **Method**                                                               | 適應症 **Indication**      |
| ----------------------------------------------- | ----------------------------------------------------------------------------- | -------------------------- |
| 牽張 **Stretching**                             | 持續性牽張（20–30 分鐘/次）；每日進行                                         | 輕至中度痙攣               |
| 夾板 **Splinting**                              | 靜態/動態夾板、夜間夾板                                                       | 預防攣縮；維持關節活動度   |
| 肉毒桿菌毒素 **Botulinum toxin**                | 局部肌肉注射（onabotulinumtoxinA / abobotulinumtoxinA）                       | 局部肌肉痙攣影響功能或照護 |
| 口服藥物 **Oral medications**                   | Baclofen (10–80 mg/day)、Tizanidine (2–36 mg/day)、Dantrolene (25–400 mg/day) | 廣泛性痙攣                 |
| 鞘內巴氯芬泵 **Intrathecal baclofen pump**      | 手術植入泵浦，持續輸注 Baclofen                                               | 嚴重泛發性痙攣             |
| 肌腱轉位/延長術 **Tendon transfer/lengthening** | 外科手術矯正固定性攣縮                                                        | 結構性攣縮（非可逆性）     |

### 異位性骨化 **Heterotopic Ossification (HO)**

| 特徵 **Feature**          | 內容 **Details**                                                              |
| ------------------------- | ----------------------------------------------------------------------------- |
| 發生率 **Incidence**      | CVA: 13–20%; TBI: 10–20%; SCI: 10–53%                                         |
| 好發部位 **Common sites** | 髖關節 > 肘關節 > 肩關節 > 膝關節                                             |
| 危險因子 **Risk factors** | 長期昏迷、痙攣、骨折、壓瘡                                                    |
| 預防 **Prevention**       | 早期被動關節活動度運動、Indomethacin 75 mg/day × 6 週（高風險個案）           |
| 治療 **Treatment**        | 輻射治療（低劑量）；手術切除（成熟後；以 ALP 及 Tc-99m bone scan 監測成熟度） |

### 肩關節半脫位 **Shoulder Subluxation**

| 特徵 **Feature**             | 內容 **Details**                                                                     |
| ---------------------------- | ------------------------------------------------------------------------------------ |
| 發生率 **Incidence**         | CVA: 40–80%（急性期）                                                                |
| 病因 **Pathophysiology**     | 三角肌/岡上肌癱瘓 → 肱骨頭失去主動支撐 → 重力牵拉                                    |
| 評估 **Assessment**          | 觸診（肩峰下可容一指）；X 光（前后位：患側 vs 健側 glenohumeral index）              |
| 治療 **Treatment**           | 功能電刺激（FES）至三角肌後束/岡上肌；貼布/taping 支撐；功能性肩膀懸吊帶（非環肩帶） |
| 疼痛管理 **Pain management** | 冰敷、TENS、肌效貼、口服NSAIDs、關節腔內注射                                         |

::: warning 警告
**肩關節半脫位注意事項 Shoulder Subluxation Precautions**

- 避免使用環肩帶（shoulder sling），可能加重半脫位及肩手症候群
- FES 需早期介入（發病 2 週內）效果最佳
- 勿從患側腋下拉抬個案（避免牵拉）
- 沐浴/更衣時使用前臂支撐，避免肩部垂吊
  :::

## 臨床實證摘要 **Key Clinical Evidence**

| 試驗 **Trial**                        | 設計 **Design**                                           | 結果 **Findings**                                                     | 臨床意義 **Clinical Implications**                                         |
| ------------------------------------- | --------------------------------------------------------- | --------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| **AVERT** (Bernhardt et al., 2015)    | RCT, n = 2,104; early mobilization (≤ 24 h) vs usual care | 早期大量 mobilization 無益甚至可能有害；14 天內以頻繁但低強度活動為宜 | 早期 mobilization 不宜過度激進；「頻繁、低強度」策略優於「長時間、高強度」 |
| **EXCITE** (Wolf et al., 2006)        | RCT, n = 222; CIMT vs 控制組                              | CIMT 組上肢功能顯著改善，效果維持至少 2 年；FMA 改善 ≥ 3.9 分         | CIMT 為 CVA 上肢復健之高品質證據推薦方案                                   |
| **LEAPS** (Duncan et al., 2011)       | RCT, n = 408; 電腦步訓 vs 物理治療 vs 家庭步訓            | 2–6 個月訓練後步速無顯著差異；進階平衡訓練組在 12 個月步速略優        | 步訓介入有效，但 BWSTT 未優於地面步訓；強度與時間比設備更重要              |
| **LOCOMOTOR** (Hidler et al., 2009)   | RCT, n = 145; BWSTT vs 等量地面步訓                       | 12 週後兩組步速改善無顯著差異                                         | 設備輔助步訓之效果與地面步訓相當                                           |
| **CAFE trial** (Gaubert et al., 2019) | RCT, n = 280; CIMT vs 一般照護                            | CIMT 改善上肢功能，尤其中度功能障礙者受益最大                         | CIMT 最佳適用對象：能主動伸腕 ≥ 20°、拇指外展 ≥ 10°                        |

## 吞嚥障礙管理 **Dysphagia Management**

| 介入方式 **Intervention**                              | 方法 **Method**                                                         | 適應症 **Indication**        |
| ------------------------------------------------------ | ----------------------------------------------------------------------- | ---------------------------- |
| 飲食調整 **Diet modification**                         | 依黏稠度分級：正常 → 糊餐 → 濃稠液 → 糊狀 → 半固體 → 液化（IDDSI 框架） | 所有吞嚥障礙個案             |
| 吞嚥姿勢技巧 **Postural techniques**                   | 下巴內收（chin tuck）：降低喉部前移，保護氣道                           | 喉部通過期延遲               |
| 吞嚥手法 **Swallowing maneuvers**                      | Mendelsohn 手法（延長喉部上提）、Shaker 運動（強化舌骨上肌群）          | 喉部上提不足、環咽肌功能障礙 |
| 口腔感覺促進 **Oral sensory stimulation**              | 冰酸刺激（thermal-tactile stimulation）、味覺刺激                       | 口腔期感覺減退               |
| 電刺激 **Neuromuscular electrical stimulation (NMES)** | VitalStim / FlyPoint：雙側胸鎖乳突肌/舌骨上肌群 NMES 配合吞嚥訓練       | 中重度吞嚥障礙               |

::: warning 警告
**吞嚥障礙注意事項 Dysphagia Precautions**

- 所有 CVA/TBI 個案吞嚥功能篩檢前禁止經口進食
- 使用增稠液需依製造商建議配比，避免過度增稠
- 仰臥位進食增加誤吸風險；建議 60–90° 坐姿
- 進食後維持坐姿 ≥ 30 分鐘
  :::

## 認知與神經心理復健 **Cognitive & Neuropsychological Rehabilitation**

| 執行功能 **Executive function** | 計畫性訓練（活動排程）、問題解決訓練（情境式任務）、自我監控     | 每日計畫表、拼圖、桌上遊戲、電腦化執行功能訓練                        |

| 語言 **Language**               | 失語症：命名訓練、語義聯想、語音回饋；構音障礙：口部運動訓練     | 命名卡片、語音辨識軟體、口肌阻力訓練器                                |

### Rancho Los Amigos 等級 **Levels of Cognitive Functioning (LCF)**

| 等級 **Level** | 名稱 **Name**          | 特徵 **Characteristics**               |
| -------------- | ---------------------- | -------------------------------------- |
| I              | Comatose               | 無反應                                 |
| II             | Generalized response   | 對刺激有反應，但反應不一致             |
| III            | Localized response     | 對刺激有明確反應（可轉頭、注視）       |
| IV             | Confused-agitated      | 混亂、激躁、定向力差、可能有攻擊行為   |
| V              | Confused-inappropriate | 混亂但較平穩，無法執行指令，記憶差     |
| VI             | Confused-appropriate   | 在引導下可執行熟悉任務，仍需協助       |
| VII            | Automatic-appropriate  | 可自動執行日常活動，判斷力仍差         |
| VIII           | Purposeful-appropriate | 可獨立執行目標導向行為，但抽象能力略差 |
| IX             | Strategic-appropriate  | 能主動規劃行為，但速度較慢             |
| X              | Purposeful-independent | 正常認知功能，可獨立生活               |

## 科技輔助復健 **Technology-Assisted Rehabilitation**

| ------------------------------------------- | ------------------------------------------------------ | ------------------------------------------------------- |

## 預後與恢復預測 **Prognosis & Recovery Prediction**

| 因子 **Factor**                      | 預測方向 **Predictive Direction**                                                                |
| ------------------------------------ | ------------------------------------------------------------------------------------------------ |
| 初始神經缺損嚴重度                   | 初始缺陷越輕，功能預後越好                                                                       |
| 年齡                                 | 年輕個案恢復潛力較大                                                                             |
| 損傷後第一個月功能改善速度           | 早期快速恢復預示良好長期預後                                                                     |
| 損傷部位與大小                       | 皮質病灶優於皮質下；大面積病灶預後差                                                             |
| 上肢 FMA 初始分數                    | FMA < 10 分：恢復至功能性手部控制之機率 < 10%                                                    |
| 10MWT 步速                           | < 0.4 m/s（ household ambulator）；0.4–0.8 m/s（limited community）；> 0.8 m/s（full community） |
| 認知功能                             | 認知障礙顯著影響 ADL 恢復及復健參與度                                                            |
| 合併症（Charlson Comorbidity Index） | 高合併症負荷降低功能恢復潛力                                                                     |
