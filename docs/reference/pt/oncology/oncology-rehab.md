# 腫瘤復健 **Oncology Rehabilitation**

---

## 1. 概述 Overview

癌症相關失能（cancer-related disability）是一種多面向、多階段的健康問題，影響患者的身體功能、心理狀態與生活品質。復健介入應貫穿整個癌症照護連續體（continuum of cancer care），包括：

| 階段 Phase                       | 定義 Definition    | 復健目標 Rehab Goals     |
| -------------------------------- | ------------------ | ------------------------ |
| **預復健 Prehabilitation**       | 確診後、治療前     | 提升功能儲備，降低併發症 |
| **治療期間 During treatment**    | 手術、化療、放療中 | 維持功能，減輕副作用     |
| **治療後 Post-treatment**        | 完成主要治療後     | 恢復功能，預防復發       |
| **晚期/安寧 Palliative/Hospice** | 疾病末期           | 舒適照護，維持尊嚴       |

> **核心理念 Core Principle**：復健不是「等治療結束才開始」，而是從診斷那一刻就應介入的全程照護。

## 2. 常見問題 Common Issues

### 2.1 癌因性疲乏 Cancer-Related Fatigue (CRF)

CRF 是癌症患者最常見且最困擾的症狀，盛行率 60–100%。

- **定義**：與癌症或癌症治療相關的、近期出現的、持續性的主觀疲倦感，與近期活動量不成比例，且休息後無法完全緩解
- **機制**：促發炎因子（IL-6, TNF-α）、下丘腦-垂體-腎上腺軸失調、代謝改變、心理因素
- **影響**：日常活動參與下降、情緒障礙、治療順從性降低

### 2.2 淋巴水腫 Lymphedema

- **好發**：乳癌術後（腋下淋巴結清除）、婦科癌症、前列腺癌
- **分期**：Stage 0（潛伏期）→ Stage I（可逆性）→ Stage II（不可逆性）→ Stage III（象皮病）
- **評估**：周徑測量、水體積測量（water displacement）、生物阻抗分析（bioimpedance）

### 2.3 疼痛 Pain

- **傷害感受性疼痛** Nociceptive pain：骨骼、軟組織侵犯
- **神經病變性疼痛** Neuropathic pain：化療引起周邊神經病變（CIPN）、腫瘤壓迫神經
- **混合型**：最常見

### 2.4 去適應化 Deconditioning

- 長期臥床或活動量驟降導致：
  - 肌力下降（sarcopenia）
  - 心肺適能降低
  - 平衡功能退化
  - 跌倒風險增加

### 2.5 化療誘發周邊神經病變 Chemotherapy-Induced Peripheral Neuropathy (CIPN)

| 常見藥物 Agents      | 神經毒性特徵 Features     |
| -------------------- | ------------------------- |
| 紫杉醇 Paclitaxel    | 感覺性為主，手套-襪子分佈 |
| 順鉑 Cisplatin       | 感覺運動混合，劑量累積    |
| 長春新鹼 Vincristine | 早期反射消失，感覺運動    |
| 硼替佐米 Bortezomib  | 小纖維神經病變，灼痛      |

### 2.6 癌症相關認知障礙 Cancer-Related Cognitive Impairment ("Chemo Brain")

- 發生率 17–75%，女性較多
- **受影響領域**：記憶力、專注力、執行功能、處理速度
- **可能機制**：神經毒性、神經發炎、微血管損傷、心理因素
- **功能影響**：用藥安全、工作能力、日常決策

    └── PG-SGA（Patient-Generated Subjective Global Assessment）
```

## 4. 預復健 Prehabilitation
### 4.1 定義與證據


預復健（prehabilitation）是在癌症治療開始前（通常為確診後 2–6 週），透過運動、營養、心理支持等介入，提升患者的功能儲備（functional reserve），以改善治療期間與術後的復原。

> **證據亮點 Key Evidence**
>
> - Cochrane Review (2021)：術前運動介入可改善術後功能狀態（SMD -0.53, 95% CI -0.91 to -0.15）
> - ACSM 立場聲明 (2019)：推薦術前運動作為標準照護的一部分
> - 多項 RCT 顯示預復健可縮短住院天數、減少術後併發症


### 4.2 預復健方案要素


| 要素 Component | 內容 Content                | 頻率 Frequency            |
| -------------- | --------------------------- | ------------------------- |
| **有氧運動**   | 步行、固定腳踏車            | 每次 20–30 min，3–5 次/週 |
| **阻力訓練**   | 彈力帶、啞鈴                | 每週 2–3 次               |
| **呼吸訓練**   | 膈肌訓練、激励式肺量計      | 每天 3–4 次               |
| **營養諮詢**   | 蛋白質攝取 1.2–1.5 g/kg/day | 每次治療前                |
| **心理支持**   | 療程、正念、認知行為策略    | 每週 1 次                 |
| **戒菸介入**   | 諮詢、尼古丁替代            | 至少術前 4 週             |


### 4.3 預復健時間線


```
確診 → 評估（1-2 週）→ 預復健開始（第 3 週起）
                            │
                            ├── 手術路徑：4-6 週預復健 → 手術
                            ├── 化療路徑：持續運動貫穿整個治療期
                            └── 放療路徑：配合治療排程調整
```

## 5. 按癌症類型的復健 Rehabilitation by Cancer Type

### 5.1 乳癌 Breast Cancer


#### 復健目標

- 恢復肩關節活動度（ROM）
- 預防/治療淋巴水腫
- 上肢功能重建
- 疼痛控制
- 體態恢復

#### 術後復健分期

| 階段 Phase   | 時間 Timing   | 介入 Intervention                      |
| ------------ | ------------- | -------------------------------------- |
| **急性期**   | 術後 0–1 週   | 鐘擺運動、深呼吸、手部活動、引流管管理 |
| **早期復原** | 術後 1–4 週   | 被動至主動輔助 ROM、肩胛穩定訓練       |
| **中期**     | 術後 1–3 個月 | 主動 ROM、漸進阻力訓練、上肢功能訓練   |
| **後期**     | 3 個月後      | 全面體能訓練、淋巴水腫管理             |

#### 淋巴水腫管理

- **完全消腫治療（CDT）**：徒手淋巴引流（MLD）+ 多層低張彈性繃帶 + 教育 + 皮膚照護
- **壓力衣**：Stage I–II 適用，20–30 mmHg
- **運動處方**：中等強度阻力訓練不會誘發或惡化淋巴水腫（證據等級 A）

::: warning 淋巴水腫預防
淋巴水腫患者應避免：高溫環境、緊身衣物、患側抽血/量血壓、蚊蟲叮咬
:::

#### 乳癌上肢運動處方

```
第一週（術後 0-7 天）：
  - 鐘擺運動 Pendulum：10 次 × 3 組，每日 2-3 次
  - 手指爬牆 Finger walk：每次 5-10 分鐘
  - 深呼吸練習 Diaphragmatic breathing：10 次 × 3 組

第二週（術後 7-14 天）：
  - 主動輔助肩屈曲 Active-assisted flexion：逐日增加 10°
  - 肩胛前突/後縮 Scapular protraction/retraction：10 次 × 3 組
  - 滑輪滑索 Pulley：10 次 × 3 組

第三-四週（術後 14-28 天）：
  - 主動全範圍 ROM
  - 彈力帶 Resistance band：輕阻力，10-15 次 × 2-3 組
  - 牆壁伏地挺身 Wall push-ups：10 次 × 2 組

第四週後：
  - 漸進阻力訓練
  - 低衝擊有氧運動
  - 淋巴水腫預防教育
```


### 5.2 肺癌 Lung Cancer


#### 復健目標

- 改善肺功能與運動耐量
- 減輕呼吸困難
- 預防術後肺部併發症
- 提升日常生活參與

#### 肺切除術後復健

| 時間 Timing       | 介入 Intervention                         |
| ----------------- | ----------------------------------------- |
| **術前**          | 預復健：有氧+阻力+呼吸訓練（至少 2-4 週） |
| **術後 0–2 天**   | 姿勢引流、咳嗽訓練、床上活動、早期下床    |
| **術後 3–7 天**   | 漸進步行、激励式肺量計、上肢訓練          |
| **術後 1–4 週**   | 有氧運動、呼吸再訓練、功能訓練            |
| **術後 1–3 個月** | 全面體能復原方案                          |

#### 呼吸訓練方案

```
1. 膈肌呼吸 Diaphragmatic breathing
   - 吸氣 4 秒（腹部隆起）→ 吐氣 6-8 秒
   - 每次 10-15 分鐘，每日 3 次

2. 缩唇呼吸 Pursed-lip breathing
   - 鼻吸 2 秒 → 嘴唇縮攏緩慢吐氣 4 秒
   - 適合呼吸困難發作時使用

3. 激励式肺量計 Incentive spirometry
   - 慢深吸氣至目標容量，維持 3-5 秒
   - 每次 10 下，每小時清醒時執行

4. 胸廓活動度訓練 Thoracic mobility
   - 胸椎伸展（over-roller）
   - 側彎、旋轉伸展
   - 每次 5-10 分鐘，每日 2 次

5. 上肢耐力訓練 Arm ergometry
   - 從低阻力開始，10-15 分鐘
   - 逐漸增加至 20-30 分鐘
```


### 5.3 大腸直腸癌 Colorectal Cancer


#### 復健重點

- 骨盆底功能訓練（尤其低位直腸癌術後）
- 造口照護與適應
- 核心穩定性重建
- 腸道功能復健

#### 骨盆底復健 Pelvic Floor Rehabilitation

| 干預 Intervention  | 適應症 Indications     | 方法 Method                                |
| ------------------ | ---------------------- | ------------------------------------------ |
| **骨盆底肌訓練**   | 大便失禁、排便功能障礙 | Kegel 運動，收縮 5-10 秒 × 10-15 次 × 3 組 |
| **生物回饋**       | 肌力不足、協調障礙     | 表面肌電圖引導                             |
| **電刺激**         | 嚴重肌無力             | 低頻電刺激，每次 20 分鐘                   |
| **直腸感覺再訓練** | 直腸感覺缺失           | 漸進式充氣球囊                             |
| **排便習慣訓練**   | 排便不規律             | 定時排便、排便姿勢指導                     |

#### 骨盆底運動處方

```
第一階段（第 1-4 週）：
  - 意識訓練：辨識骨盆底肌收縮與放鬆
  - Kegel 收縮 3 秒 × 10 次 × 3 組，每日 3 次
  - 腹式呼吸配合

第二階段（第 5-8 週）：
  - 延長收縮至 5-8 秒
  - 增加至 15 次 × 3 組
  - 加入快速收縮（quick flick）10 次 × 3 組

第三階段（第 9-12 週）：
  - 收縮 10 秒 × 15 次 × 3 組
  - 功能性整合：站立位、行走中收縮
  - 核心穩定訓練整合
```

### 5.4 頭頸部癌症 Head & Neck Cancer


#### 復健目標

- 恢復吞嚥功能
- 開口度訓練（trismus management）
- 肩頸功能維持
- 語言/溝通功能
- 放療後口腔衛生

#### 吞嚥復健方案

| 介入 Intervention   | 方法 Method                 | 頻率 Frequency                 |
| ------------------- | --------------------------- | ------------------------------ |
| **Shaker 運動**     | 仰臥抬頭看腳趾 + 平躺抬下巴 | 30 次 + 3 組 1 分鐘，每日 2 次 |
| **Mendelsohn 手法** | 吞嚥時維持喉部上抬 2-3 秒   | 10 次 × 3 組，每日 3 次        |
| **Masako 手法**     | 舌頭前伸抵下唇時吞嚥        | 10 次 × 3 組，每日 3 次        |
| **聲門上吞嚥**      | 深吸氣→閉氣→吞嚥→咳嗽       | 每餐前 5-10 次                 |
| **冰刺激**          | 冰棉棒軟顎/咽部刺激         | 每次 5-10 分鐘，每日 3 次      |
| **開口器訓練**      | TheraBite / 壓舌板漸進拉伸  | 每次 5-10 分鐘，每日 3-5 次    |

::: warning 放射性骨壞死風險
頭頸癌放療後患者有放射性骨壞死（ORN）風險，避免侵入性操作
:::


### 5.5 骨轉移 Bone Metastasis


#### 復健重點

- 疼痛控制
- 功能維持與安全移動
- 病理性骨折預防
- 骨骼保護運動

#### 骨轉移運動注意事項

| 轉移部位 Site | 允許的運動 Allowed       | 禁忌運動 Contraindicated  |
| ------------- | ------------------------ | ------------------------- |
| **脊椎**      | 仰臥位核心穩定、水中運動 | 過度屈曲/伸展、高衝擊活動 |
| **股骨近端**  | 坐姿運動、水中運動       | 重複性負重、深蹲、跳躍    |
| **肋骨**      | 輕度有氧、呼吸訓練       | 胸部直接阻力訓練          |
| **骨盆**      | 仰臥位運動、步行         | 跌倒風險高的活動          |

::: warning 骨轉移禁忌
**絕對禁忌 Absolute Contraindications**：病理性骨折已發生或有即時風險時，禁止患部負重
:::

### 6.1 ACS/ACSM 指南建議
## 6. 運動處方 Exercise Prescription


根據美國癌症協會（ACS）與美國運動醫學會（ACSM）2019 年聯合立場聲明：

| 族群 Population               | 建議 Recommendation                                                    |
| ----------------------------- | ---------------------------------------------------------------------- |
| **治療期間 During treatment** | 每週 150 分鐘中等強度或 75 分鐘高強度有氧 + 每週 2-3 次阻力訓練        |
| **治療後 survivors**          | 每週 150–300 分鐘中等強度或 75–150 分鐘高強度有氧 + 每週 ≥2 次阻力訓練 |
| **避免久坐**                  | 每天坐姿時間應中斷，每小時起身活動 3-5 分鐘                            |


### 6.2 運動強度分級


| 強度 Intensity | 心率 HR      | 自覺費力 RPE     | 說明 Description |
| -------------- | ------------ | ---------------- | ---------------- |
| **低**         | 40–54% HRmax | 11–12 (輕鬆)     | 可輕鬆對話       |
| **中等**       | 55–74% HRmax | 12–14 (有些吃力) | 可對話但稍喘     |
| **高**         | ≥75% HRmax   | 15+ (吃力)       | 難以對話         |


### 6.3 運動進階原則


```
階段一（第 1-2 週）：建立基線
  - 有氧：低強度 15-20 分鐘，3 次/週
  - 阻力：機器或彈力帶，1 組 × 10-12 次
  - 柔軟度：靜態伸展，每次 15-30 秒

階段二（第 3-4 週）：增加量
  - 有氧：中等強度 20-30 分鐘，3-5 次/週
  - 阻力：2 組 × 10-15 次
  - 平衡：站立平衡訓練

階段三（第 5+ 週）：維持與進階
  - 有氧：中等至高強度 30-45 分鐘，3-5 次/週
  - 阻力：2-3 組 × 8-12 次，漸增阻力
  - 功能性訓練整合
```

1. 膈肌呼吸訓練（10 分鐘）
   - 仰臥，膝蓋彎曲
   - 雙手置於腹部，鼻吸氣 4 秒（腹部隆起）
   - 嘴吐氣 6-8 秒（腹部凹陷）
   - 10 次 × 3 組

2. 胸廓活動度（5 分鐘）
   - 胸椎伸展 over foam roller：10 次
   - 側彎伸展：左右各 15 秒 × 3
   - 旋轉伸展：左右各 15 秒 × 3

3. 呼吸肌阻力訓練（5 分鐘）
   - 激励式肺量計：深吸維持 3-5 秒 × 10 次
   - 呼吸阻力閥門：呼氣阻力訓練

4. 姿勢訓練（5 分鐘）
   - 胸椎挺直坐姿
   - 肩胛後收 + 挺胸 × 10
   - 站立前鋸肌訓練 × 10

整合有氧：
  - 6MWT 每週 2 次追蹤進步
  - 漸進式：5 分鐘 → 10 → 15 → 20 → 30 分鐘步行
```


### 7.3 大腸直腸癌骨盆底方案 Pelvic Floor (Colorectal)

```
每日骨盆底訓練（15-20 分鐘）：

仰臥位（10 分鐘）：
  1. 膝蓋彎曲仰臥，放鬆骨盆底
  2. 慢速 Kegel：收縮 5 秒 → 放鬆 5 秒 × 15 次
  3. 快速收縮：1 秒收縮 → 1 秒放鬆 × 15 次
  4. 腹式呼吸配合：吸氣放鬆 → 吐氣收縮

坐位（5 分鐘）：
  1. 坐姿骨盆底收縮 × 15 次
  2. 骨盆前傾/後傾 × 10 次
  3. 側彎伸展：左右各 × 10 次

站立位（5 分鐘）：
  1. 站立 Kegel × 15 次
  2. 橋式運動 × 10 次
  3. 深蹲（浅） × 10 次

功能性整合：
  - 行走中骨盆底收縮
  - 咳嗽/打噴嚏時保護性收縮
  - 提重物時配合收縮
```

## 8. 注意事項與禁忌 Precautions & Contraindications


| 風險 Risk      | 預防措施 Precaution       |
| -------------- | ------------------------- |
| **病理性骨折** | 避免患部高衝擊/高扭力運動 |
| **脊髓壓迫**   | 避免脊椎過度屈曲/伸展     |
| **高跌倒風險** | 平衡訓練需有安全保護      |

### 8.2 血液學限制 Hematologic Considerations


| 病變 Complication                            | 運動限制 Exercise Restriction                              |
| -------------------------------------------- | ---------------------------------------------------------- |
| **嗜中性白血球減少 Neutropenia** (ANC < 500) | 避免公共場所運動、避免水中運動（感染風險）、室內低接觸運動 |
| **血小板減少 Thrombocytopenia** (< 50,000)   | 避免高衝擊/接觸性運動、避免抗凝血藥物同時使用              |
| **貧血 Anemia** (Hb < 8 g/dL)                | 降低運動強度、避免突然姿勢改變、監測血氧                   |
| **發燒 Fever** (> 38°C)                      | 暫停運動、評估感染源                                       |


### 8.3 其他重要注意事項


| 情境 Situation               | 注意事項 Precaution                      |
| ---------------------------- | ---------------------------------------- |
| **中心靜脈導管 PICC/CVC**    | 避免患側上肢高阻力訓練、游泳禁忌         |
| **放射性皮膚炎**             | 避免摩擦/壓迫照射部位、穿寬鬆衣物        |
| **嚴重噁心嘔吐**             | 避免治療當天高強度運動、選擇低強度活動   |
| **植入物/人工關節**          | 避免特定方向的高阻力                     |
| **心臟毒性藥物**（如蒽環類） | 運動前心臟評估、運動中監測、限制最大心率 |
| **淋巴水腫風險**             | 穿壓力衣運動、漸進增加強度、避免過熱     |


### 8.4 運動中止標準 Stopping Criteria


```
立即停止運動的情況：
├── 胸痛或嚴重呼吸困難
├── 心悸、頭暈、近暈厥
├── 收縮壓 > 250 mmHg 或舒張壓 > 115 mmHg
├── 血氧飽和度 < 88%
├── 新出現或加重的疼痛
├── 關節/骨骼異常響聲或劇痛
├── 嚴重噁心嘔吐
├── 意識改變
└── 跌倒或接近跌倒
```

## 10. 總結 Summary

```
腫瘤復健核心要點：
├── 全程照護：從預復健到長期追蹤
├── 個體化處方：根據癌症類型、治療階段、功能狀態
├── 多學科團隊：復健醫師、物理治療師、職能治療師、心理師、營養師
├── 安全第一：骨轉移、血液學限制、導管管理
├── 證據導向：遵循 ACS/ACSM 指南
├── 症狀管理：疲乏、疼痛、淋巴水腫、認知障礙
└── 生活品質：功能恢復 × 心理社會支持 × 自我管理
```

> **最終目標 Final Goal**：讓每一位癌症患者都能在治療全程獲得適當的復健介入，最大化功能恢復、最小化失能，最終回歸有意義的生活。
