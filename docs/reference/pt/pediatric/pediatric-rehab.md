# 兒童復健 **Pediatric Rehabilitation**

::: warning
本文件僅供教育參考，臨床決策應結合個案評估與醫療團隊專業判斷。
This document is for educational reference only. Clinical decisions should integrate individual assessment and interdisciplinary professional judgment.
:::

## 概述 Overview

兒童復健（Pediatric Rehabilitation）是一門跨專業的復健學科，專注於因先天或後天因素導致功能障礙之兒童與青少年。其目標為最大化兒童在家庭、學校及社區中的獨立性與參與度。

### 常見診斷 Common Diagnostics

| 診斷 Diagnosis                           | 描述 Description                                                                     |
| ---------------------------------------- | ------------------------------------------------------------------------------------ |
| 腦性麻痺 Cerebral Palsy (CP)             | 一組永久性運動與姿勢發展障礙，導致活動限制，歸因於胎兒或嬰兒腦部非進行性傷害         |
| 發育遲緩 Developmental Delay             | 運動、語言、認知或社會情緒發展未達年齡標準                                           |
| 遺傳症候群 Genetic Syndromes             | 如唐氏症（Down syndrome）、威廉斯症候群（Williams syndrome）等影響肌肉骨骼與神經系統 |
| 後天性腦損傷 Acquired Brain Injury (ABI) | 腦外傷（TBI）、腦瘤切除後、缺氧性腦病變等                                            |

### 腦性麻痺的流行病學 CP Epidemiology

- 活產發生率約 2–3 / 1,000
- 雙胞胎風險較單胎高 4–6 倍
- 低出生體重（<1500g）與早產（<32 週）為主要危險因子
- 近年因新生兒照護進步，輕度 CP 占比增加

## GMFCS 分級 **Gross Motor Function Classification System**

GMFCS 以兒童在日常環境中執行粗大動作能力的自主運動功能進行分級，為 CP 兒童最常用的功能分類系統。

| 等級 Level    | 描述 Description                       | 功能特徵 Functional Characteristics                                    |
| ------------- | -------------------------------------- | ---------------------------------------------------------------------- |
| **Level I**   | 能執行所有動作，包括跑步與跳躍         | 功能接近同齡正常兒童；動作速度與協調略差                               |
| **Level II**  | 能行走但有限制；可能無法跑步或跳躍     | 可使用輔具行走；上下樓梯需扶手；可能需要手動輪椅長距離移動             |
| **Level III** | 使用輔具行走有困難；身體支撐輔具可執行 | 使用助行器（walker）或前臂拐杖（forearm crutches）行走；可能需手動輪椅 |
| **Level IV**  | 自我移動受限；倚賴輔具或他人輔助       | 使用手動或電動輪椅；可能使用支撐站立架（stander）                      |
| **Level V**   | 所有粗大動作技能均受限                 | 依賴輪椅；需他人搬運與協助；頭頸控制受限                               |

::: warning
GMFCS 應以**功能觀察**而非年齡為分級依據。五歲後分級穩定度高，建議至少觀察至六歲後確認。
:::

## 評估工具 **Assessment Tools**

| 工具 Tool                                               | 評估項目 Domain            | 適用對象 Population | 說明 Description                                                           |
| ------------------------------------------------------- | -------------------------- | ------------------- | -------------------------------------------------------------------------- |
| **GMFM** (Gross Motor Function Measure)                 | 粗大動作功能               | CP 兒童             | 涵蓋躺/翻身、坐、爬/跪、站、行走/跑步/跳躍五大面向；GMFM-88 或簡版 GMFM-66 |
| **PEDI** (Pediatric Evaluation of Disability Inventory) | 功能性技能與照護者協助需求 | 6 個月–7.5 歲       | 評估自我照顧、移動、社會功能；含能力量表與協助量表                         |
| **Bayley** (Bayley Scales of Infant Development)        | 認知、語言、動作發展       | 嬰幼兒（1–42 個月） | 發展里程碑標準化評估                                                       |
| **WeeFIM** (Wee Functional Independence Measure)        | 日常功能性獨立             | 6 個月–12 歲        | 18 項活動（自我照顧、轉移、運動、溝通、社會認知）                          |
| **GMFCS** (Gross Motor Function Classification)         | 粗大動作功能分級           | CP 兒童             | 5 級分類系統（見上表）                                                     |
| **MACS** (Manual Ability Classification System)         | 上肢手部功能分級           | CP 兒童（4–18 歲）  | 5 級分類，描述雙手操作物品的能力                                           |
| **CFCS** (Communication Function Classification System) | 溝通功能分級               | CP 兒童             | 評估日常溝通能力                                                           |

::: warning
評估時應結合標準化工具與臨床觀察，並考慮文化、語言及環境因素對測驗結果的影響。
:::

## 腦性麻痺分型 **Cerebral Palsy Subtypes**

### 痙攣型 Spastic CP（約 80%）

| 類型 Type               | 受影響肢體 Distribution | 常見特徵 Common Features                          | 捐傷部位 Lesion Site |
| ----------------------- | ----------------------- | ------------------------------------------------- | -------------------- |
| **雙癱 Diplegia**       | 雙下肢為主              | 膝張力增高、尖足（equinus）、剪刀步態、髓外展不足 | 雙側腦室旁白質       |
| **偏癱 Hemiplegia**     | 同側上下肢              | 上肢屈曲模式、手指抓握受限、步態不對稱            | 對側大腦半球         |
| **四肢癱 Quadriplegia** | 四肢及軀幹              | 嚴重運動受限、可能伴隨吞嚥困難、認知影響、癲癇    | 雙側廣泛性腦傷       |

### 非痙攣型 Non-Spastic CP

| 類型 Type                          | 張力特徵 Tone            | 運動特徵 Movement                                         | 描述 Description |
| ---------------------------------- | ------------------------ | --------------------------------------------------------- | ---------------- |
| **失張力型 Dyskinetic (Athetoid)** | 低至中度，伴隨不自主運動 | 手足徐動（athetosis）、肌張力障礙（dystonia）、舞蹈樣運動 | 基底核損傷       |
| **失調型 Ataxic**                  | 低至正常                 | 平衡不佳、辨距不良（dysmetria）、意向性震顫               | 小腦損傷         |

### 混合型 Mixed CP

常見為痙攣型與失張力型或失調型之混合，需依個別臨床表徵分別處理。

## 介入策略 **Intervention Approaches**

### 1. 神經發展治療 Neurodevelopmental Treatment (NDT / Bobath)

- **核心理念**：通過抑制異常反射模式、促進正常運動反應
- **技術**：關鍵點控制（key points of control）、反射抑制模式（reflex inhibiting patterns）
- **目標**：改善姿勢控制、促進功能性動作
- **證據等級**：臨床證據有限，但仍廣泛使用；建議結合以任務為導向的訓練

### 2. 限制誘發運動療法 Constraint-Induced Movement Therapy (CIMT)

- **適應症**：偏癱型 CP，患側上肢有一定活動能力
- **方法**：限制健側上肢（手套、吊帶），強迫使用患側
- **劑量**：每日 6–8 小時，持續 2–3 週（改良式 CIMT 可縮短）
- **證據**：A 級證據支持改善患側上肢功能

### 3. 馬術治療 Hippotherapy

- **機制**：馬背的三維節律性運動模擬正常步態模式
- **效果**：改善核心穩定性、姿勢控制、軀幹張力
- **證據**：B–C 級證據；建議每週 1–2 次

### 4. 水療 Aquatic Therapy

- **優勢**：浮力減輕關節負荷、水溫促進放鬆、阻力提供感覺輸入
- **適應症**：張力調節困難、疼痛、關節活動度限制
- **證據**：B 級證據支持改善粗大動作功能

### 5. 序列石膏 Serial Casting

- **目的**：漸進增加軟組織長度（主要用於踝關節背屈不足）
- **方法**：每 5–7 天更換石膏，逐步增加角度
- **配合**：與肉毒桿菌素注射結合效果更佳
- **證據**：B 級證據

::: warning
序列石膏使用時應密切監測皮膚完整性、血液循環及兒童舒適度。石膏移除後應立即開始功能訓練以維持改善效果。
:::

## 年齡別介入重點 **Age-Based Intervention Focus**

| 年齡 Age      | 發展重點 Development Focus       | 主要介入 Primary Interventions                    | 預期目標 Expected Goals          |
| ------------- | -------------------------------- | ------------------------------------------------- | -------------------------------- |
| **0–6 個月**  | 姿勢控制、頭頸穩定、原始反射整合 | 姿勢管理、家屬衛教、感覺運動經驗、自然環境策略    | 正常姿勢張力模式、促進手眼協調   |
| **6–12 個月** | 坐姿平衡、爬行、抓握             | NDT、姿勢控制訓練、輔具評估（如需要）             | 坐姿獨立、雙手操作、早期移動     |
| **1–2 歲**    | 站立、行走啟始、手部精細動作     | 站立架使用、步態訓練、CIMT（偏癱）、遊戲介入      | 站立平衡、輔助行走、手部功能     |
| **2–4 歲**    | 行走、跑跳、日常活動             | 功能訓練、馬術治療、水療、序列石膏、適形鞋墊/支具 | 功能性行走、參與遊戲、ADL 獨立性 |
| **4–6 歲**    | 學齡前準備、團體活動參與         | CIMT、功能導向訓練、輔具調整、社會參與            | 學校參與、同儕互動、書寫與操作   |
| **6–12 歲**   | 學齡期功能、體能、社會參與       | 功能維持與強化、體能訓練、輔具升級、轉銜計畫      | 校園獨立、體能維持、社交能力     |
| **12–18 歲**  | 青春期發育、自決能力、社區參與   | 自我管理技能、職能訓練、成年轉銜、骨骼健康管理    | 獨立生活、職業準備、健康自主     |

## 依 GMFCS 分級之介入 **Interventions by GMFCS Level**

### GMFCS Level I

- 一般體能訓練：跑步、跳繩、球類活動
- 協調性訓練：平衡板、動態穩定任務
- 特殊運動機會：游泳、體操、腳踏車
- **目標**：最佳化體能與運動參與

### GMFCS Level II

- 步態優化：速度與耐力訓練
- 上下樓梯與不平坦地面行走訓練
- 競速輪椅運動（長距離）
- **目標**：功能性行走維持、社區參與

### GMFCS Level III

- 輔具行走訓練（助行器、前臂拐杖）
- 站立架使用與站立耐力
- 替代移動方式（輪椅技能訓練）
- **目標**：輔具下功能性移動、站立參與

### GMFCS Level IV

- 電動輪椅評估與訓練
- 站立架、適形坐墊
- 姿勢管理與壓力釋放
- **目標**：環境控制、姿勢舒適、社會參與

### GMFCS Level V

- 全面姿勢支持與姿勢管理
- 輔助科技（AAC、環境控制系統）
- 搬運訓練與照護者教育
- **目標**：最大化舒適度、環境參與、照護品質

::: warning
GMFCS Level IV–V 兒童應特別注意壓瘡預防、呼吸功能管理及營養狀態監測。定期多團隊評估建議每 6 個月。
:::

## 矯形器與輔具 **Orthotic & Assistive Devices**

### 下肢矯形器 Lower Extremity Orthoses

| 矯形器 Orthosis                           | 適應症 Indications         | 功能 Function                      |
| ----------------------------------------- | -------------------------- | ---------------------------------- |
| **AFO** (Ankle-Foot Orthosis)             | 尖足、踝關節不穩定         | 控制踝關節角度、改善步態、預防攣縮 |
| **DAFO** (Dynamic AFO)                    | 中度張力異常               | 提供足弓支撐、允許些微背屈/跖屈    |
| **HKAFO** (Hip-Knee-Ankle-Floor Orthosis) | GMFCS III–IV、髓關節不穩定 | 支撐下肢三關節、輔助站立           |
| **KAFO** (Knee-Ankle-Forthosis)           | 膝關節鎖定不穩             | 穩定膝關節、輔助站立與行走         |
| **SAB (Supramalleolar Orthosis)**         | 輕度足部變形               | 穩定足踝、預防內翻/外翻            |

### 上肢矯形器 Upper Extremity Orthoses

| 矯形器 Orthosis                         | 適應症 Indications | 功能 Function                |
| --------------------------------------- | ------------------ | ---------------------------- |
| **休息位手夾板 Resting Hand Splint**    | 痙攣嚴重、張力障礙 | 維持手腕手指功能位、預防攣縮 |
| **功能型手夾板 Functional Hand Splint** | 偏癱上肢功能訓練   | 促進手指伸展、輔助抓握       |
| **拇指擋板 Thumb Spica**                | 拇指內扣           | 維持拇指外展位、促進抓握     |
| **動態夾板 Dynamic Splint**             | 關節攣縮、術後     | 漸進牽張、維持關節活動度     |

### 輔具與適形設備 Assistive Technology

- **電動輪椅**：GMFCS III–IV，上肢控制能力有限時
- **AAC（替代溝通系統）**：平板電腦溝通軟體、眼控系統
- **姿勢支撐座椅系統**：客製化坐墊與背靠
- **環境控制系統**：語音控制、開關控制燈光/電視

::: warning
矯形器使用應定期追蹤（建議每 3–6 個月），確認適配性、皮膚完整性及功能效果。兒童生長快速，需頻繁更換。
:::

## 家庭中心照護 **Family-Centered Care (FCC)**

### 核心原則 Core Principles

1. **尊重家庭**：家庭是兒童生活的主要來源與穩定力量
2. **資訊分享**：提供完整、客觀的資訊支持家庭決策
3. **合作夥伴**：專業團隊與家庭是合作夥伴，共同制定目標
4. **個別化服務**：介入應基於兒童與家庭的獨特需求
5. **社區融合**：支持兒童與家庭參與社區活動
6. **家屬賦能**：強化家庭照顧能力與信心

### 實務策略 Practical Strategies

| 策略 Strategy                               | 說明 Description                               |
| ------------------------------------------- | ---------------------------------------------- |
| 家庭優先目標設定 Goal Setting               | 由家庭主導設定復健目標，治療師提供專業建議     |
| 自然環境策略 Natural Environment Strategies | 在家庭、學校、社區進行治療與訓練，而非僅限診所 |
| 家長教導 Coaching                           | 治療師指導家長在日常生活中融入復健策略         |
| 同儕支持 Peer Support                       | 連結其他 CP 家庭經驗分享                       |
| 轉銜服務 Transition Planning                | 學齡前→學齡、學齡→成年之系統性轉銜             |

::: warning
家庭中心照護不等於「家長要求什麼就做什麼」。專業團隊仍有責任提供最佳證據建議，並在尊重家庭選擇的前提下引導決策。
:::

| ----------------- | --------------------------------- | ------------------------------ |
| 髖關節半脫位/脫位 | GMFCS III–V、痙攣型、內收肌張力高 | 髖關節X光追蹤、適時手術介入    |

### 癲癇管理 Seizure Management

- 約 40–50% CP 兒童伴隨癲癇
- 治療中應注意癲癇發作前兆
- 避免感覺過度刺激、維持規律作息
- 教導照護者發作時之處置流程
- 癲癇藥物與復健治療之交互作用需與神經科團隊協調

## 呼吸與營養考量 **Respiratory & Nutritional Considerations**

### 呼吸功能 Respiratory Function

- 痙攣四肢癱兒童因呼吸肌無力，咳嗽能力不足
- 胸廓變形（脊椎側彎）可影響肺功能
- 胸廓物理治療、姿勢引流、呼吸訓練為必要介入
- 睡眠呼吸中止症篩檢建議納入評估

### 營養狀態 Nutritional Status

- 口腔運動功能障礙可導致進食困難
- 吞嚥造影（VFSS）評估與飲食調整
- 低張型 CP 兒童常伴隨胃食道逆流
- 體重過輕或過重均需營養團隊介入

## 循證實踐 **Evidence-Based Practice**

### AACPDM 證據分級系統

美國腦性麻痺與發展障礙兒童醫學學院（AACPDM）提供介入措施的證據等級分類：

| 等級 Level | 定義 Definition    | 意義 Meaning |
| ---------- | ------------------ | ------------ |
| **A 級**   | 足夠的高品質證據   | 強力建議使用 |
| **B 級**   | 有限的高品質證據   | 建議使用     |
| **C 級**   | 混合品質證據       | 可考慮使用   |
| **D 級**   | 有限證據，可能有害 | 不建議使用   |
| **E 級**   | 證據不足或無證據   | 無法建議     |
| **I 級**   | 證據無法進行分類   | 需更多研究   |

### CP 實踐指南關鍵建議 CP Practice Guidelines

1. **評估**：使用標準化工具（GMFM、GMFCS、MACS）進行全面評估
2. **目標設定**：以家庭和兒童為中心，使用 ICF 框架
3. **介入選擇**：結合證據等級與個案需求
4. **劑量**：高重複性、高強度訓練优于低強度被動治療
5. **功能導向**：以任務為導向的介入優於純被動手法
6. **家庭參與**：自然環境介入與家屬教導
7. **定期重評**：每 3–6 個月全面評估、每年調整介入計畫

::: warning
證據等級不是唯一決策依據。個案偏好、臨床經驗、文化脈絡及資源可及性均應納入考量。
:::

## 遊戲為基礎之介入 **Play-Based Interventions**

遊戲是兒童的自然學習媒介，復健介入應將治療目標融入遊戲情境。

### 治療性遊戲分類 Therapeutic Play Categories

| 遊戲類型 Play Type             | 治療目標 Therapeutic Goals       | 活動範例 Examples          |
| ------------------------------ | -------------------------------- | -------------------------- |
| 感覺運動遊戲 Sensorimotor Play | 張力調節、本體感覺輸入、姿勢控制 | 觸覺箱探索、球池、按摩遊戲 |
| 建構性遊戲 Constructive Play   | 手眼協調、雙手操作、認知概念     | 積木堆疊、拼圖、黏土塑形   |
| 象徵性遊戲 Symbolic Play       | 語言發展、社交互動、認知表徵     | 角色扮演、娃娃屋、廚房遊戲 |
| 規則性遊戲 Rule-Based Play     | 指令遵循、輪替概念、社交規則     | 桌遊、團體競賽、音樂遊戲   |
| 感冒運動遊戲 Locomotor Play    | 粗大動作、平衡、空間意識         | 追逐遊戲、障礙跑、滑梯     |

### 遊戲介入策略 Play Intervention Strategies

- **循環任務安排**：設置 3–4 個站點，每站針對不同治療目標
- **選擇權提供**：讓兒童在兩個活動中選擇，促進動機與自主性
- **挑戰分級**：依據最近發展區（ZPD）調整活動難度
- **社交中介**：利用同儕或兄弟姊妹共同遊戲，促進社交互動
- **家長參與**：指導家長在家中設計治療性遊戲活動

## MACS 上肢手部功能分級 **Manual Ability Classification System**

| 等級 Level    | 描述 Description                   | 手部功能 Hand Function                     |
| ------------- | ---------------------------------- | ------------------------------------------ |
| **Level I**   | 能輕易抓握物品，速度與準確性不受限 | 操作物品不受限；可能略慢                   |
| **Level II**  | 能抓握物品，但準確度與速度受限     | 處理中等大小物品有困難；快速操作受限       |
| **Level III** | 處理物品有限，能操作簡單物品但受限 | 需輔具或他人輔助完成大部分手部活動         |
| **Level IV**  | 處理物品嚴重受限                   | 需輔具支撐或他人協助；可能使用少數穩定抓握 |
| **Level V**   | 無法使用手部處理物品               | 手部功能嚴重受限；完全依賴他人協助         |

::: warning
MACS 側重「日常手部操作能力」而非「解剖範圍」。與 GMFCS 一樣，MACS 是功能分級而非肌力或關節活動度的測量。
:::

## 步態分析重點 **Gait Analysis Considerations**

| 步態偏差 Gait Deviation       | 常見成因 Common Causes          | 介入方向 Intervention               |
| ----------------------------- | ------------------------------- | ----------------------------------- |
| 尖足步態 Equinus Gait         | 腓腸肌/比目魚肌攣縮、脛前肌無力 | AFO、序列石膏、肉毒桿菌素、跟腱延長 |
| 剪刀步態 Scissor Gait         | 內收肌痙攣                      | 內收肌伸展、肉毒桿菌素注射、AFO     |
| 反膝步態 Recurvatum Gait      | 股四頭肌無力、膝關節過伸        | HKAFO、膝關節支撐、強化訓練         |
| 扁平足步態 Crouch Gait        | 腘繩肌攣縮、膝伸肌無力          | 矯形手術、AFO、肌力訓練             |
| 骨盆前傾 Anterior Pelvic Tilt | 髂腰肌緊縮、腹肌無力            | 伸展、核心穩定訓練、適形坐墊        |

步態分析應結合觀察分析（Gait Observation）與儀器分析（Instrumented Gait Analysis，如 3D Motion Capture），以制定精準的介入計畫。

## 疼痛管理 **Pain Management**

CP 兒童的疼痛問題常被低估，研究顯示 50–75% 的 CP 青少年報告有慢性疼痛。

| 疼痛來源 Pain Source            | 表現 Presentation      | 處理 Approach                        |
| ------------------------------- | ---------------------- | ------------------------------------ |
| 肌肉骨骼疼痛 MSK Pain           | 關節痛、背痛、步態改變 | 姿勢管理、伸展、適形座椅、必要時藥物 |
| 痙攣相關疼痛 Spasticity-Related | 肌肉痙攣、夜間不寧     | 肉毒桿菌素、口服解痙劑、巴氯芬泵     |
| 壓力性疼痛 Pressure Pain        | 壓瘡、皮膚破損         | 壓力釋放、適形坐墊、定時姿勢變換     |
| 姿勢相關 Postural Pain          | 脊椎側彎、骨盆傾斜     | 脊椎矯形、手術評估、適形支撐         |

::: warning
疼痛評估應使用適齡工具（如 Wong-Baker FACES、Numeric Rating Scale），並考慮無法口語表達兒童的非語言疼痛行為指標。
:::
