# 肩關節物理治療 — 分期復健運動計畫

**Shoulder Physical Therapy — Phased Rehabilitation Protocol**

## 概述 Overview

本文件為肩關節疼痛與活動受限（如冰凍肩／沾黏性關節囊炎、旋轉肌袖病變）之**分期居家復健運動計畫**，共分三期：由**疼痛控制與被動活動**（第 1–2 週）漸進至**主動輔助與主動活動**（第 3–6 週），最後進入**肌力強化**（第 6 週以後）。每一階段皆需在**無明顯疼痛加劇**的前提下再往下一期推進。

::: warning ⚠️ 洗腎與動靜脈瘻管注意事項（Dialysis / AV Fistula）
本計畫針對**有動靜脈瘻管（AV fistula）之血液透析（HD）病患**特別調整。所有運動皆應**避開瘻管側肢體的負重與拉扯**，並於復健前告知透析團隊。詳見下方「應避免事項」。
:::

## 分期總覽 Phase Summary

| 分期 Phase                  | 時程 Timeline           | 目標 Goal                        |
| --------------------------- | ----------------------- | -------------------------------- |
| **Phase 1** 疼痛控制與鐘擺  | 第 1–2 週               | 維持被動關節活動度、減少發炎     |
| **Phase 2** 主動輔助 → 主動 | 第 3–6 週（疼痛改善後） | 恢復主動活動度、正常化肩胛力學   |
| **Phase 3** 肌力強化        | 第 6 週以後（疼痛極少） | 漸進式強化旋轉肌袖與肩胛穩定肌群 |

## Phase 1：疼痛控制與鐘擺運動（第 1–2 週）

**目標：** 維持被動關節活動度、減少發炎
**⚠️ 禁忌：** 不主動舉手、不過頭伸取、不做拉扯動作（No active lifting / overhead reaching / pulling）

| 運動 Exercise                           | 做法 How                                                                 |
| --------------------------------------- | ------------------------------------------------------------------------ |
| **鐘擺運動 Pendulum (Codman)**          | 身體前彎，讓患肢像布娃娃般自然下垂；輕晃軀幹帶動手臂擺盪。順時針後逆時針 |
| **仰躺被動外轉 Supine passive ER**      | 平躺，雙手握木棍／掃把柄，用健側手把患側手往外側推（手肘 90°、夾緊身側） |
| **仰躺被動前舉 Supine passive flexion** | 同一根棍子，用健側手緩緩將雙臂舉過頭頂                                   |
| **冰敷 Ice**                            | 運動後冰敷 15 分鐘，每日 3–4 次                                          |

## Phase 2：主動輔助 → 主動（第 3–6 週，疼痛改善後）

**目標：** 恢復主動活動度、正常化肩胛力學

| 運動 Exercise                       | 做法 How                                                      |
| ----------------------------------- | ------------------------------------------------------------- |
| **爬牆前舉 Wall walking (flexion)** | 面對牆壁，手指沿牆往上爬，以牆面提供支撐                      |
| **肩胛後收 Scapular retraction**    | 將兩側肩胛骨向後、向下夾緊，維持 5 秒                         |
| **側躺外轉 Side-lying ER**          | 側躺於健側，患肘 90° 貼身側，前臂向上轉（可握小罐頭增加阻力） |
| **俯臥 T/Y Prone T/Y**              | 俯臥，雙臂先成 T 字、再成 Y 字，夾緊肩胛                      |

## Phase 3：肌力強化（第 6 週以後，疼痛極少）

**目標：** 漸進強化旋轉肌袖與肩胛穩定肌群

| 運動 Exercise                    | 起始 Start                     | 進階 Progress     |
| -------------------------------- | ------------------------------ | ----------------- |
| **彈力帶內外轉 ER/IR with band** | 腰部高度輕阻力彈力帶（紅／黃） | 綠／藍            |
| **划船 Rows**                    | 彈力帶固定於前方，將手肘往後拉 | 漸進加阻力        |
| **俯臥後伸 Prone extension**     | 俯臥、拇指朝上，手臂向後上抬   | 加 1–2 磅腳踝負重 |

## 應避免事項 What to Avoid — especially with AV fistula

- ❌ **過頭推舉／軍事推舉** Overhead press / military press
- ❌ **臥推** Bench press（負重下的外轉會壓迫棘上肌 supraspinatus）
- ❌ **引體向上／頸後下拉** Pull-ups / lat pulldown behind neck
- ❌ **睡在患側** Sleeping on the affected side（必要時於腋下墊枕）
- ❌ **瘻管側肢體負重或穿刺當日過度運動** — 透析（HD）當日若疲累，**勿安排物理治療**；並確保**透析團隊知悉復健計畫**
- ❌ 任何造成瘻管受壓、拉扯之動作（彈力帶勿纏繞或錨定於瘻管側手臂）

## 警訊 — 停止復健並就醫 Red Flags → Stop PT and Consult

- 🚩 突發劇痛，伴隨**「啪」聲或撕裂感**（audible pop / tear）
- 🚩 運動後**無法對抗地心引力舉起手臂** → 疑似旋轉肌袖斷裂（rotator cuff rupture）
- 🚩 發燒、紅腫、全身性症狀 → 疑似化膿性關節炎／膿瘍（septic joint / abscess）
- 🚩 **瘻管功能異常**：thrill／bruit（震顫／雜音）改變、出血時間延長
