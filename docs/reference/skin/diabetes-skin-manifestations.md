# 糖尿病皮膚表現

**糖尿病皮膚表現 Skin Manifestations of Diabetes Mellitus**

## 概述 Overview

- DM（type 1 & 2）患者 **30–70%** 一生中會出現皮膚併發症
- 皮膚變化可反映 **血糖控制狀況**，有時甚至是未診斷 DM 的首發徵象
- 五大類：① 強相關皮膚表現 ② 非特異性徵象 ③ 相關皮膚病 ④ 常見感染 ⑤ 藥物相關變化

## 一、與糖尿病密切相關的皮膚表現 Skin Manifestations Strongly Associated with DM

### 黑色棘皮症 Acanthosis Nigricans

- **表現**：灰棕色、天鵝絨樣／疣狀斑塊，邊界不清，對稱分布
- **好發部位**：頸後、腋下、肘、腹股溝、臍周、乳房下皺褶
- **病理機轉**：高胰島素 → 活化 IGF-1 受體 → 角質細胞＆纖維母細胞增生
- **臨床意義**：胰島素阻抗的皮膚標誌；出現時應篩檢 DM 及相關內分泌疾病
- **治療**：首重生活型態調整（飲食、運動、減重）＋ metformin；局部可用 retinoid、vitamin D 類似物、keratolytic

### 糖尿病皮病變／脛前斑 Diabetic Dermopathy

- **盛行率**：高達 **50%** — 最常見的 DM 皮膚表現
- **表現**：圓形紅丘疹 → 1–2 週內演變為萎縮性棕色斑，表面細鱗屑
- **好發部位**：雙側脛前（亦可見於前臂、外踝、大腿）
- **病程**：18–24 個月消退，遺留色素沉著＋輕度凹陷；不同病灶可處於不同階段
- **關聯性**：常伴隨 microvascular disease（視網膜病變、腎病變）；與 CVD 相關
- **治療**：無需治療（自限性）；但需加強篩檢微血管併發症

### 糖尿病足 Diabetic Foot Syndrome

- **發生率**：年發生率 1–4%，終身盛行率 4–10%
- **常見病灶**：
  - 神經性潰瘍（最常見）— 無痛，好發於胼胝處、骨突起處
  - 缺血性潰瘍 — 較少見但疼痛
  - Charcot foot — 中足塌陷 → rocker-bottom foot 變形
  - 爪狀趾 — 內在肌萎縮＋足底筋膜增厚
- **病理機轉**：神經病變＋動脈粥狀硬化＋傷口癒合障礙（AGEs、發炎、氧化壓力）
- **關鍵數字**：15–25% DM 患者會發生足潰瘍；50% 三年內復發
- **治療**：跨團隊照護、預防為上（每日檢查、適當鞋具）；傷口照護、抗生素、清創、血管重建

### 糖尿病厚皮 Diabetic Thick Skin

#### 類硬皮症變化 Scleroderma-like Skin Changes

- **盛行率**：10–50% DM 患者，type 1 > type 2
- **表現**：無痛、硬結、蠟樣皮膚增厚，對稱分布於手指背側（sclerodactyly）
- **特殊徵象**：**Prayer sign**（雙手無法平貼）；易與 Dupuytren contracture、trigger finger 合併出現
- **治療**：嚴格血糖控制可能改善；物理治療維持關節活動度

#### Limited Joint Mobility（LJM，糖尿病性關節病變）

- **盛行率**：DM 患者 8–58%
- **表現**：手指 MCP、PIP 關節屈曲攣縮，由第五指開始向外擴散
- **徵象**：Prayer sign、**Tabletop sign**（前臂垂直桌面時手掌無法貼平）
- **治療**：控制血糖＋伸展運動；NSAIDs / 局部類固醇注射

#### 糖尿病性硬腫症 Scleredema Diabeticorum

- **盛行率**：DM 患者 2.5–14%
- **表現**：對稱性、瀰漫性皮膚硬化增厚 — 上背、肩、頸後最常見；遠端肢體通常不受影響
- **嚴重 case**：可限制軀幹活動，甚至導致限制性呼吸問題
- **治療**：**2024 EDF guideline** — 一線：UVA1 / PUVA 光療；二線：methotrexate；進階：immunosuppressants、IVIG

### 類脂質漸進性壞死 Necrobiosis Lipoidica

- **盛行率**：僅 0.3–1.6% DM 患者；但 25% NL 患者同時有 DM
- **表現**：紅褐色邊緣＋黃棕色萎縮性中心（可見 telangiectasias）之斑塊
- **好發部位**：雙側脛前（幾乎 always）；Koebner 現象
- **併發症**：~1/3 會潰瘍 → 續發性感染或 SCC
- **治療**：困擾 — 多數治療反應不一；類固醇（局部／病灶內／口服）、calcineurin inhibitors、anti-TNF；控制血糖未必改善皮膚病灶

### 糖尿病性水疱 Bullosis Diabeticorum

- **盛行率**：約 0.5% DM 患者
- **表現**：突發性、非紅斑性、緊張性無菌水疱（0.5–5 cm），常 overnight 出現
- **好發部位**：雙側下肢遠端（acral）
- **預後**：2–6 週自癒，但易復發
- **治療**：無需治療 — 保護皮膚、避免感染；大水泡可穿刺減壓

## 二、非特異性皮膚徵象與症狀 Nonspecific Dermatologic Signs and Symptoms

| 徵象                                      | 表現                           | 好發部位                | 備註                                            |
| ----------------------------------------- | ------------------------------ | ----------------------- | ----------------------------------------------- |
| **Ichthyosiform Changes**                 | 乾燥、魚鱗狀脫屑               | 雙側脛前                | 22–48% type 1 DM                                |
| **Xerosis**                               | 異常乾燥、龜裂                 | 足部                    | 高達 40% DM 患者；肥胖者更嚴重                  |
| **Acquired Perforating Dermatosis (APD)** | 角化過度臍狀丘疹＋角質栓，極癢 | 四肢伸側、軀幹          | 常合併慢性腎衰竭                                |
| **Eruptive Xanthomas**                    | 光澤性粉黃色丘疹 1–4 mm        | 伸側、臀部              | TG > 2000 mg/dL（DM 患者可能更低）              |
| **Acrochordons（skin tags）**             | 軟性有柄贅生物                 | 頸、腋、眼眶周          | 75% skin tags 患者同時有 DM                     |
| **Pruritus**                              | 局部或全身搔癢                 | 頭皮、踝、足、生殖器    | 更常見於有乾皮症或神經病變者                    |
| **Huntley's Papules（finger pebbles）**   | 無症狀小丘疹，可融合成斑塊     | 手指 MCP 關節背側、甲周 | 與 type 2 DM 強相關；可能是厚皮早期表現         |
| **Keratosis Pilaris**                     | 毛囊角化丘疹＋周圍紅暈         | 上臂後側、大腿          | DM 患者更常見且範圍更廣                         |
| **Pigmented Purpuric Dermatoses**         | 非可壓白之古銅色斑             | 脛前、足背              | 常與 diabetic dermopathy 共存                   |
| **Palmar Erythema**                       | 對稱性手掌發紅發熱             | 手掌（大小魚際）        | 微血管病變所致                                  |
| **Periungual Telangiectasias**            | 甲褶紅斑＋毛細血管擴張         | 近端甲褶                | 高達 50% DM 患者                                |
| **Rubeosis Faciei**                       | 慢性臉／頸部潮紅               | 臉、頸                  | ~7% DM 患者（住院 >50%）；淺色皮膚更明顯        |
| **Yellow Skin & Nails**                   | 無症狀黃色變                   | 手掌、腳掌、臉、趾甲    | 可能與 carotene、glycosylated proteins 堆積有關 |
| **Onychocryptosis**                       | 嵌甲                           | 大腳趾                  | type 2 DM 相關；與 BMI、創傷、血管供應有關      |

## 三、與糖尿病相關的皮膚疾病 Dermatologic Diseases Associated with DM

- **Generalized Granuloma Annulare（GGA）** — 膚色／紅色環狀丘疹或斑塊，中央消退、邊緣隆起；軀幹＋四肢對稱分布；10–15% GGA 患者有 DM

- **Psoriasis** — 與 DM 風險增加相關（尤其年輕＋嚴重乾癬患者）
- **Lichen Planus** — 多角形紫紅色丘疹，手腕／腳踝常見；~25% LP 患者有 DM
- **Vitiligo** — 境界清楚之脫色斑；type 1 DM 更常見；應篩檢自體免疫甲狀腺疾病

- **Hidradenitis Suppurativa（HS）** — 腋下／腹股溝反覆發炎結節、膿瘍、瘻管；HS 患者 DM 盛行率為一般人群 3 倍

- **Glucagonoma** — 罕見神經內分泌腫瘤；70–83% 出現 **Necrolytic Migratory Erythema（NME）**；20–40% 在診斷 glucagonoma 前已確診 DM

- **Melanoma** — type 2 DM 雖不增加發生率，但診斷時分期更晚、更易出現 ulceration

## 四、糖尿病常見皮膚感染 Common Skin Infections in DM

- DM 患者皮膚感染盛行率約 **20%**；易反覆感染

### 細菌性感染 Bacterial

| 感染                              | 特徵                                                                          | 治療                               |
| --------------------------------- | ----------------------------------------------------------------------------- | ---------------------------------- |
| **Erysipelas／Cellulitis**        | DM 常見；MRSA 定殖不少見                                                      | 口服／靜脈抗生素                   |
| **Diabetic Foot Infection**       | 最常見之軟組織感染；可能導致骨髓炎、截肢                                      | 抗生素＋清創                       |
| **Necrotizing Fasciitis**         | 致死率 ~20%；痛感與理學檢查不成比例；**Fournier's gangrene** 為會陰部 variant | 緊急清創＋廣效抗生素               |
| **Erythrasma（C. minutissimum）** | 紅棕色細鱗屑斑塊；Wood's light 下呈 **coral-red fluorescence**                | 局部 erythromycin／clindamycin     |
| **Malignant Otitis Externa**      | 罕見但致命（未治療死亡率 ~50%）；P. aeruginosa                                | 長期 anti-pseudomonal 抗生素＋HBOT |

### 黴菌性感染 Fungal

| 感染                             | 特徵                                                       | 治療                         |
| -------------------------------- | ---------------------------------------------------------- | ---------------------------- |
| **Candida**                      | 口腔（thrush）、間擦部位、指甲；反覆發作應篩檢 DM          | 局部／口服抗真菌藥＋控制血糖 |
| **Dermatophytes（Tinea pedis）** | 最常見之黴菌感染；可續發細菌感染、fungemia                 | 局部／全身抗真菌藥           |
| **Mucormycosis**                 | type 1 DM＋DKA 重大風險；**Rhino-orbital-cerebral** 最常見 | 緊急清創＋IV amphotericin B  |

## 五、糖尿病藥物相關皮膚變化 Cutaneous Changes Associated with DM Medications

- **Insulin**：Lipohypertrophy（最常見，~30%）→ 延遲吸收、血糖起伏不定。Lipoatrophy（現代胰島素已少見）。過敏反應罕見（全身性 0.01%）
- **DPP-4 inhibitors**（vildagliptin 等）：**Bullous pemphigoid**、Stevens-Johnson syndrome、angioedema
- **Sulfonylureas**：maculopapular rash（前 2 個月）；chlorpropamide → 飲酒後臉部潮紅
- **SGLT-2 inhibitors**：生殖器黴菌感染↑、**Fournier's gangrene**（罕見但嚴重）、Sweet syndrome
- **GLP-1 receptor agonists** (semaglutide, liraglutide etc)：高劑量 → 感覺異常、落髮；快速減重 → **"GLP-1 face"**（面部脂肪流失、皺紋、眼窩凹陷）；但可 off-label 用於 psoriasis、HS、acanthosis nigricans
- **Diabetes Devices**（CGM／CSII）：接觸性皮膚炎；CSII 有胰島素相關皮膚變化風險

## 七、臨床重點 Clinical Key Points

1. **Acanthosis nigricans** + **acrochordons** → 胰島素阻抗標誌 → 應篩檢 DM
2. **Diabetic dermopathy** 為最常見皮膚表現 — 不需治療，但提示 microvascular disease，需加強併發症篩檢
3. **糖尿病足**是可預防的重大併發症 — 每日檢查＋適當鞋具＋血糖控制為關鍵
4. **Necrobiosis lipoidica** 未必隨血糖控制改善；潰瘍病灶有 SCC 風險
5. **全身搔癢** → 排除感染（特別是 candidiasis）及糖尿病神經病變
6. **DPP-4 inhibitor** 使用中出現水疱 → 考慮 bullous pemphigoid
7. **SGLT-2 inhibitor** → 生殖器黴菌感染↑；罕見但致命：Fournier's gangrene
8. **GLP-1 receptor agonist** — 高劑量可能引起感覺異常、落髮；快速減重導致 **"GLP-1 face"**（面部脂肪流失）
9. **Scleredema diabeticorum** 緩慢進展、治療棘手；2024 EDF 指引：一線 UVA1/PUVA
10. **多重皮膚表現同時出現** → 應全面評估血糖控制及 DM 併發症
