# 濕疹鑑別診斷與臨床思考流程

**Differential Diagnosis of Eczema & Clinical Algorithm**

> **核心概念**：「濕疹（eczema）」非獨立疾病，而是一組以**搔癢、紅斑、滲出、脫屑**為共同表現的皮膚炎症候群。臨床上須系統性排除各種原因，才能確立診斷與治療方向。

## 第一步：確認是否為濕疹樣形態（Eczematous Morphology）

在進入鑑別診斷前，先確認病灶符合濕疹樣形態：

| 急性期（Acute）         | 亞急性期（Subacute）   | 慢性期（Chronic）             |
| ----------------------- | ---------------------- | ----------------------------- |
| 紅斑（erythema）        | 鱗屑（scaling）        | 苔蘚化（lichenification）     |
| 水疱（vesicles）        | 結痂（crusting）       | 皮膚增厚（skin thickening）   |
| 滲出（weeping／oozing） | 丘疹（papules）        | 色素沉著（hyperpigmentation） |
| 劇癢（pruritus）        | 搔抓痕（excoriations） | 皸裂（fissures）              |

若形態吻合，進入以下鑑別流程。

## 第三步：主要鑑別診斷分類

### 一、原發性濕疹（Primary Eczema）

| 疾病                                                          | 好發部位                                                   | 特徵性表現                                                                     | 關鍵鑑別點                                                         |
| ------------------------------------------------------------- | ---------------------------------------------------------- | ------------------------------------------------------------------------------ | ------------------------------------------------------------------ |
| **異位性皮膚炎**（Atopic dermatitis, AD）                     | 嬰兒：臉頰、頭皮、伸側；兒童及成人：肘窩、膕窩、頸部、手腕 | 劇癢、慢性復發、苔蘚化、乾燥皮膚（xerosis）                                    | 個人或家族異位史（哮喘、過敏性鼻炎）；Hanifin-Rajka 診斷準則       |
| **接觸性皮膚炎 — 刺激性**（Irritant contact dermatitis, ICD） | 手部最常見（尤其指間、手背）                               | 紅斑、脫皮、皸裂；燒灼感 > 搔癢                                                | 長期接觸刺激物（水、清潔劑、溶劑）；停止接觸後改善                 |
| **接觸性皮膚炎 — 過敏性**（Allergic contact dermatitis, ACD） | 接觸部位（形狀常對應接觸物：如項鍊、耳環、錶帶形狀）       | 延遲性超敏反應（接觸後 48–72 小時出現）；邊界清晰的紅斑水疱                    | 貼片試驗（patch test）陽性；鎳、香料、橡膠、防腐劑為常見致敏原     |
| **脂漏性皮膚炎**（Seborrheic dermatitis）                     | 頭皮、鼻唇溝、眉毛、耳廓、前胸（皮脂腺豐富處）             | 油膩性黃色脫屑（greasy yellow scales）、紅斑、搔癢                             | 與 Malassezia 相關；頑固難治時注意 HIV                             |
| **汗皰疹**（Dyshidrotic eczema／pompholyx）                   | 手掌、手指側面、腳掌                                       | 深層小水疱群（tapioca-like vesicles）、劇癢、水疱破後脫皮                      | 夏季好發；與金屬過敏（鎳）、真菌感染（遠端id reaction）相關        |
| **錢幣狀濕疹**（Nummular eczema／discoid eczema）             | 小腿伸側、手背、軀幹                                       | 界線明顯的圓形或橢圓形（錢幣狀）濕疹性斑塊                                     | 常伴皮膚乾燥；冬季好發；易誤診為環癬（tinea corporis）→ KOH 鏡檢   |
| **鬱積性皮炎**（Stasis dermatitis／venous eczema）            | 雙側小腿下 1/3，內踝附近                                   | 紅褐色含鐵血黃素沉著、靜脈曲張、水腫                                           | 靜脈功能不全為基礎；可進展為脂肪皮膚硬化症（lipodermatosclerosis） |
| **神經性皮炎**（Lichen simplex chronicus, LSC）               | 頸後、踝部、外陰、頭皮（容易搔抓的部位）                   | 苔蘚化斑塊（lichenified plaque）、明顯皮溝皮嵴（skin markings）、搔癢–搔抓循環 | 搔抓行為為核心；心理壓力誘發                                       |
| **結節性癢疹**（Prurigo nodularis）                           | 四肢伸側（手臂、小腿）                                     | 多個堅硬的結節（firm nodules）、表面脫皮結痂                                   | 慢性搔抓所致；頑固難治                                             |

### 二、感染性及感染誘發（Infectious & Infection-related）

| 疾病                                                     | 特徵                                                                               | 鑑別線索                                                                      |
| -------------------------------------------------------- | ---------------------------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| **感染性濕疹樣皮炎**（Infectious eczematoid dermatitis） | 環繞感染灶（膿瘍、竇道、潰瘍）向外蔓延的濕疹樣紅斑                                 | 有明確感染灶；細菌培養陽性                                                    |
| **癬菌疹**（Dermatophytid／id reaction）                 | 身體某處有真菌感染（如足癬），免疫過敏反應導致遠端部位（手掌常見）出現汗皰疹樣水疱 | 足癬（tinea pedis）同時存在；id 病灶 KOH 鏡檢陰性（水疱是過敏反應，不含真菌） |
| **疥瘡**（Scabies）                                      | 指間、腕部屈側、乳暈、陰囊等；隧道（burrows）                                      | 夜間劇癢；家庭群聚；皮膚鏡（dermoscopy）見疥蟲                                |
| **膿皰病**（Impetigo）                                   | 面部、鼻周、口周                                                                   | 蜂蜜色結痂（honey-colored crusts）；細菌繼發感染                              |

### 三、系統性疾病相關濕疹（Systemic Disease-related）

| 疾病                                                           | 皮膚表現                                                                     | 系統性線索                                                           |
| -------------------------------------------------------------- | ---------------------------------------------------------------------------- | -------------------------------------------------------------------- |
| **乳糜瀉相關**（皰疹樣皮炎 Dermatitis herpetiformis）          | 劇癢群聚性水疱，分布於肘伸側、膝、臀、頭皮                                   | 對無麩質飲食反應；小腸切片：絨毛萎縮；IgA 抗 tTG 陽性                |
| **系統性接觸性皮炎**（Systemic contact dermatitis）            | 經口攝入或靜脈注射致敏原後引發全身性濕疹樣反應（如鎳、藥物）                 | 對應貼片試驗致敏原；低鎳飲食改善                                     |
| **肝臟疾病相關搔癢**（Cholestatic pruritus）                   | 全身搔癢 → 搔抓 → 繼發濕疹化（eczematization）                               | 黃疸、膽汁鬱積；肝功能異常                                           |
| **甲狀腺功能低下**（Hypothyroid xerosis）                      | 皮膚乾燥粗厚→ 繼發濕疹化                                                     | TSH 升高；眉毛外側脫落（madarosis）                                  |
| **腎臟病相關搔癢**（Uremic pruritus）                          | 全身搔癢 → 繼發濕疹化                                                        | 腎功能異常；透析患者                                                 |
| **POEMS 症候群**                                               | 色素沉著、皮膚增厚                                                           | M蛋白、器官腫大、多發性神經病變                                      |
| **修格蘭症候群相關乾燥**（Sjögren xerosis）                    | 嚴重皮膚乾燥 → 全身搔癢 → 繼發濕疹化（eczematization）                       | 口乾、眼乾（sicca symptoms）；anti-Ro/La 陽性；淚腺唾液腺功能減退    |
| **慢性移植體抗宿主疾病**（Chronic GVHD）                       | 苔蘚樣丘疹斑塊（lichenoid）、硬皮症樣（sclerodermoid）；可呈廣泛濕疹樣紅皮症 | 造血幹細胞移植史；口腔黏膜白紋（lichenoid GVHD）；肝功能異常         |
| **自體免疫黃體素皮膚炎**（Autoimmune progesterone dermatitis） | 月經週期前後規律發作的濕疹樣丘疹水疱、蕁麻疹、多形紅斑樣病灶                 | 黃體期（luteal phase）反覆發作；對自體黃體素過敏；卵巢抑制治療有效   |
| **成人史迪爾氏症**（Adult-onset Still's disease, AOSD）        | 短暫橙紅色斑丘疹（evanescent salmon-pink rash），伴隨發燒高峰出現            | 間歇高燒、關節痛、咽喉痛；鐵蛋白（ferritin）顯著升高；皮疹隨發燒消退 |

### 四、副腫瘤性及血液疾病相關（Paraneoplastic）

| 疾病                                                             | 特徵                                                              | 警示線索（Red flags）                                                     |
| ---------------------------------------------------------------- | ----------------------------------------------------------------- | ------------------------------------------------------------------------- |
| **後天性魚鱗癬**（Acquired ichthyosis）                          | 廣泛乾燥脫屑（魚鱗状）、搔癢                                      | 成人新發；何杰金氏淋巴瘤、NHL                                             |
| **何杰金氏淋巴瘤搔癢**（Hodgkin lymphoma pruritus）              | 全身頑固搔癢 → 繼發濕疹樣變化                                     | 體重減輕、盜汗、淋巴結腫大（B症狀）                                       |
| **副腫瘤性天皰瘡**（Paraneoplastic pemphigus, PNP）              | 嚴重口腔糜爛 + 多形性皮膚病灶                                     | 嚴重口腔病灶；CLL、NHL、Castleman disease                                 |
| **皮肌炎**（Dermatomyositis）                                    | 技工手（mechanics hands）可呈濕疹樣                               | 向光性紅斑、Gottron丘疹、近端肌無力；肺癌、卵巢癌篩查                     |
| **藍格罕細胞組織球增生症**（Langerhans cell histiocytosis, LCH） | 脂漏性皮炎樣紅斑丘疹（seborrheic-like），好發於頭皮、腋下、腹股溝 | 淋巴結腫大、骨痛、肺部浸潤、尿崩症；皮膚活檢：CD1a+/Langerin+             |
| **壞死性遊走性紅斑**（Necrolytic migratory erythema, NME）       | 軀幹四肢遠端遊走性邊界不清紅斑，中央水疱糜爛結痂，類似嚴重濕疹    | 高血糖（glucagonoma）、體重減輕、舌炎、蛀牙；血漿 glucagon 升高           |
| **嗜酸性白血球增多症候群**（Hypereosinophilic syndrome, HES）    | 廣泛搔癢性濕疹樣丘疹斑塊、蕁麻疹、紅皮症                          | 嗜酸性球 >1500/µL 持續6個月以上；心臟（Loeffler心內膜炎）、神經、肺部侵犯 |
| **成人T細胞白血病／淋巴瘤**（ATLL）                              | 廣泛性濕疹樣紅斑丘疹、可進展為紅皮症（erythroderma）              | 來自 HTLV-1 流行區；淋巴腺腫大、高血鈣、LDH 升高；皮膚活檢見捲曲核淋巴球  |
| **白血病皮膚浸潤**（Leukemia cutis）                             | 單發或多發紅褐色丘疹結節；可呈廣泛濕疹樣斑塊（罕見）              | 周邊血液抹片異常；依白血病類型不同（CLL、AML 最常見皮膚表現）             |

### 五、藥物疹（Drug Reactions）

| 類型                                                      | 特徵                                      | 常見致因藥物                                              |
| --------------------------------------------------------- | ----------------------------------------- | --------------------------------------------------------- |
| **固定性藥疹**（Fixed drug eruption）                     | 反覆在同一部位出現的紅褐色圓形斑塊        | NSAIDs、磺胺類、抗生素                                    |
| **泛發性藥物疹**（Morbilliform/eczematous drug eruption） | 廣泛對稱性紅斑丘疹                        | 抗生素（ampicillin + EBV 感染幾乎必發）、抗癲癇藥         |
| **DRESS/藥物過敏症候群**                                  | 廣泛性皮疹 + 發燒 + 淋巴結腫大 + 內臟侵犯 | 抗癲癇藥（carbamazepine、phenytoin）、allopurinol、磺胺類 |
| **光敏感性藥物疹**（Photo drug eruption）                 | 僅曝光部位出現濕疹樣反應                  | Thiazide、tetracycline、amiodarone、NSAIDs                |

### 六、需排除的類濕疹疾病（Eczema Mimics）

| 疾病                                                       | 外觀相似點                             | 鑑別要點                                                                                     |
| ---------------------------------------------------------- | -------------------------------------- | -------------------------------------------------------------------------------------------- |
| **乾癬**（Psoriasis）                                      | 脫屑性斑塊                             | 境界清楚、銀白色厚鱗屑、Auspitz徵（刮除後點狀出血）、好發肘伸側／膝／頭皮                    |
| **環癬**（Tinea corporis）                                 | 環形脫屑性紅斑                         | 邊緣有活躍性丘疹水疱、中央改善、KOH 陽性                                                     |
| **亞急性皮膚紅斑狼瘡**（SCLE）                             | 環形或多環形脫屑性斑塊                 | 光敏感分布（臉、上胸、背、伸臂）；anti-Ro/La 陽性；對類固醇反應差                            |
| **圓盤狀紅斑狼瘡**（Discoid lupus, DLE）                   | 境界清楚紅色斑塊伴黏著性鱗屑           | 毛囊角栓（follicular plugging）、萎縮、色素紊亂、疤痕性禿髮；活檢見 interface dermatitis     |
| **扁平苔癬**（Lichen planus）                              | 紫紅色多角形扁平丘疹                   | Wickham striae（白色網狀紋）；口腔黏膜侵犯；劇癢；好發腕屈側、小腿、下背                     |
| **環狀肉芽腫**（Granuloma annulare）                       | 環形膚色或淡紅色丘疹群                 | 無鱗屑（non-scaly）；好發手背、足背、肘膝伸側；自限性；組織學：palisaded granuloma           |
| **多形性紅斑**（Erythema multiforme）                      | 靶形病灶（target lesions）             | 口腔黏膜侵犯常見；與 HSV 感染相關（EM minor）或藥物相關（EM major/SJS）；急性發作            |
| **類肉瘤病**（Sarcoidosis）                                | 紅褐色或紫褐色浸潤性斑塊               | 丘疹結節沿疤痕分布常見（scar sarcoidosis）；肺門淋巴腺腫大、眼葡萄膜炎；活檢見非乾酪性肉芽腫 |
| **硬化性苔癬**（Lichen sclerosus）                         | 白色萎縮性丘疹斑塊、皮膚皺縮           | 好發外陰、肛周（引起慢性搔癢灼痛）；非生殖器部位亦可見；活檢：homogeneous 膠原               |
| **腸病性肢端皮炎**（Acrodermatitis enteropathica／鋅缺乏） | 肢端（手、足、口周）溼疹樣紅斑水疱糜爛 | 腹瀉、落髮、生長遲緩；血清鋅降低；補鋅後迅速改善                                             |
| **玫瑰糠疹**（Pityriasis rosea）                           | 軀幹橢圓形脫屑斑                       | 先兆斑（herald patch）、聖誕樹型分布（Christmas tree pattern）、自限性                       |
| **天皰瘡**（Pemphigus）                                    | 水疱                                   | 鬆弛性水疱、Nikolsky 徵陽性、口腔黏膜侵犯；DIF 陽性                                          |
| **大皰性類天皰瘡**（Bullous pemphigoid）                   | 水疱、搔癢                             | 緊張性大水疱（tense bullae）；老年人；DIF：線狀 IgG/C3                                       |
| **Paget 病**（Extramammary Paget disease）                 | 肛門生殖器溼疹樣紅斑                   | 長期不癒的單側紅斑；活檢見 Paget cells                                                       |
| **Bowen 病**（Bowen disease／in situ SCC）                 | 界線清楚脫屑性紅斑斑塊                 | 緩慢擴大、對局部類固醇無反應；活檢確診                                                       |
| **皮膚T細胞淋巴瘤**（CTCL／Mycosis fungoides）             | 慢性濕疹樣斑塊、頑固搔癢               | 對標準治療無反應；皮膚活檢多次才確診；Sézary cells                                           |

## 第四步：診斷輔助工具（Diagnostic Workup）

### 基礎評估

| 工具                                  | 適用情境                                          |
| ------------------------------------- | ------------------------------------------------- |
| KOH 直接鏡檢（KOH direct microscopy） | 懷疑真菌感染（tinea、candida）；排除環癬          |
| 皮膚鏡（Dermoscopy）                  | 疥瘡（疥蟲、隧道）、色素性病灶評估                |
| 刮片培養（Skin scraping culture）     | 細菌、真菌培養                                    |
| 皮膚活檢（Skin biopsy）               | 診斷不明確；懷疑 CTCL、Paget 病、自體免疫水疱疾病 |
| 貼片試驗（Patch test）                | 懷疑過敏性接觸性皮膚炎；找出致敏原                |

### 選擇性實驗室檢查

| 檢查                                       | 懷疑診斷                                 |
| ------------------------------------------ | ---------------------------------------- |
| CBC + differential                         | 嗜酸性球增多（過敏性疾病）；血液惡性腫瘤 |
| Total IgE、特異性 IgE（RAST/ImmunoCAP）    | 異位性皮膚炎、食物或吸入性過敏原         |
| ANA、anti-dsDNA、anti-Ro/La                | 懷疑 SLE、亞急性皮膚紅斑狼瘡             |
| TSH、fT4                                   | 甲狀腺功能異常相關皮膚搔癢               |
| BUN、Creatinine                            | 尿毒症搔癢                               |
| LFT、膽汁酸（bile acids）                  | 膽汁鬱積性搔癢                           |
| 血清蛋白電泳（SPEP）、尿液蛋白電泳（UPEP） | 懷疑 POEMS、輕鏈病                       |
| Tryptase、histamine                        | 肥大細胞增多症（mastocytosis）相關搔癢   |
| 直接及間接免疫螢光（DIF/IIF）              | 水疱性自體免疫疾病                       |
| 抗 tTG IgA、抗 endomysial 抗體             | 皰疹樣皮炎（乳糜瀉篩查）                 |

## 第五步：特殊情境的鑑別重點

### 成人新發嚴重濕疹（New-onset severe eczema in adult）

```
成人新發難治性全身濕疹
        │
        ├─► 排除藥物疹（近3個月內新藥？）
        │
        ├─► 排除接觸性皮膚炎（職業史、新產品？）→ 貼片試驗
        │
        ├─► 排除感染誘發（疥瘡？繼發感染？）
        │
        ├─► 系統性疾病篩查
        │       ├─ 搔癢為主 → TSH、LFT、腎功能、CBC
        │       └─ 體重減輕＋B症狀 → 淋巴瘤篩查（CBC、LDH、CT）
        │
        └─► 皮膚活檢排除 CTCL（Mycosis fungoides）
```

### 局部頑固不癒的「濕疹」

```
單一部位長期不癒（> 3 個月）的「濕疹」
        │
        ├─► 乳頭 → Paget 病（乳腺管癌）
        │
        ├─► 肛門生殖器 → Extramammary Paget 病、Bowen 病
        │
        ├─► 頭皮單側 → Discoid lupus、脂漏性皮炎、CTCL
        │
        └─► 任何部位對類固醇無反應 → 活檢必做
```

### 搔癢為主、皮膚病灶不明顯（Pruritus sine materia）

```
全身搔癢但皮疹不明顯
        │
        ├─► 皮膚因素：乾燥症（xerosis）、疥瘡（nocturnal itch）
        │
        ├─► 腎臟：BUN/Cr → 尿毒症搔癢
        │
        ├─► 肝臟：LFT、膽汁酸 → 膽汁鬱積性搔癢
        │
        ├─► 血液：CBC → 真性紅血球增多症（aquagenic pruritus）
        │              　　何杰金氏淋巴瘤
        │
        ├─► 甲狀腺：TSH → 甲亢（warm moist）/ 甲低（dry）
        │
        ├─► 神經性：纖維肌痛、帶狀疱疹後神經痛
        │
        └─► 心因性（排除器質性後）
```

## 第六步：異位性皮膚炎診斷準則（Hanifin-Rajka Criteria）

**主要準則（Major criteria）— 需符合 3 項以上：**

1. 搔癢（Pruritus）
2. 典型形態與分布（Typical morphology and distribution）
3. 慢性或慢性復發病程（Chronic or chronically relapsing course）
4. 個人或家族異位史（Personal or family history of atopy）

**次要準則（Minor criteria）— 需符合 3 項以上（共 23 項，列舉重要者）：**

- 皮膚乾燥症（xerosis）
- 魚鱗癬、毛囊角化症（ichthyosis vulgaris, KP）
- 即時型（I型）皮膚試驗反應（immediate skin test reactivity）
- 血清 IgE 升高（elevated serum IgE）
- 早發年齡（early age of onset）
- 反覆皮膚感染傾向（tendency to skin infections）
- 手部或足部非特異性皮膚炎（nonspecific hand or foot dermatitis）
- 乳頭濕疹（nipple eczema）
- 眼周濕疹（periorbital darkening／Dennie-Morgan folds）
- 白色皮膚劃紋症（white dermographism）

::: warning 臨床警示：何時必須皮膚活檢？

- 對局部類固醇治療 4–6 週無反應
- 單一部位頑固病灶（懷疑 Paget 病、Bowen 病）
- 疑似皮膚 T 細胞淋巴瘤（CTCL）
- 水疱性病灶（確認是否為自體免疫水疱疾病）
- 全身搔癢伴有全身症狀（B 症狀、體重減輕）
