# 濕疹鑑別診斷 **Differential Diagnosis of Eczema**

> **核心概念**：「濕疹（eczema）」非獨立疾病，而是一組以**搔癢、紅斑、滲出、脫屑**為共同表現的皮膚炎症候群。臨床上須系統性排除各種原因，才能確立診斷與治療方向。

## 第一步：確認是否為濕疹

在進入鑑別診斷前，先確認病灶符合濕疹樣形態：

| 急性期（Acute）         | 亞急性期（Subacute）   | 慢性期（Chronic）             |
| ----------------------- | ---------------------- | ----------------------------- |
| 紅斑（erythema）        | 鱗屑（scaling）        | 苔蘚化（lichenification）     |
| 水疱（vesicles）        | 結痂（crusting）       | 皮膚增厚（skin thickening）   |
| 滲出（weeping／oozing） | 丘疹（papules）        | 色素沉著（hyperpigmentation） |
| 劇癢（pruritus）        | 搔抓痕（excoriations） | 皸裂（fissures）              |

若形態吻合，進入以下鑑別流程。

## 第三步：主要鑑別診斷分類

### 一、原發性濕疹

| 疾病                                                          | 好發部位                                                   | 特徵性表現                                                                     | 關鍵鑑別點                                                         | 治療原則                                                 |
| ------------------------------------------------------------- | ---------------------------------------------------------- | ------------------------------------------------------------------------------ | ------------------------------------------------------------------ | -------------------------------------------------------- |
| **異位性皮膚炎**（Atopic dermatitis, AD）                     | 嬰兒：臉頰、頭皮、伸側；兒童及成人：肘窩、膕窩、頸部、手腕 | 劇癢、慢性復發、苔蘚化、乾燥皮膚（xerosis）                                    | 個人或家族異位史（哮喘、過敏性鼻炎）；Hanifin-Rajka 診斷準則       | 局部類固醇＋保濕；Tacrolimus；嚴重者 dupilumab           |
| **接觸性皮膚炎 — 刺激性**（Irritant contact dermatitis, ICD） | 手部最常見（尤其指間、手背）                               | 紅斑、脫皮、皸裂；燒灼感 > 搔癢                                                | 長期接觸刺激物（水、清潔劑、溶劑）；停止接觸後改善                 | 避免刺激物；局部類固醇；保濕                             |
| **接觸性皮膚炎 — 過敏性**（Allergic contact dermatitis, ACD） | 接觸部位（形狀常對應接觸物：如項鍊、耳環、錶帶形狀）       | 延遲性超敏反應（接觸後 48–72 小時出現）；邊界清晰的紅斑水疱                    | 貼片試驗（patch test）陽性；鎳、香料、橡膠、防腐劑為常見致敏原     | 避免致敏原；局部類固醇                                   |
| **脂漏性皮膚炎**（Seborrheic dermatitis）                     | 頭皮、鼻唇溝、眉毛、耳廓、前胸（皮脂腺豐富處）             | 油膩性黃色脫屑（greasy yellow scales）、紅斑、搔癢                             | 與 Malassezia 相關；頑固難治時注意 HIV                             | 抗真菌洗劑（ketoconazole）；局部類固醇；鈣調磷酸酶抑制劑 |
| **汗皰疹**（Dyshidrotic eczema／pompholyx）                   | 手掌、手指側面、腳掌                                       | 深層小水疱群（tapioca-like vesicles）、劇癢、水疱破後脫皮                      | 夏季好發；與金屬過敏（鎳）、真菌感染（遠端id reaction）相關        | 局部類固醇；乾燥療法；避免金屬過敏原                     |
| **錢幣狀濕疹**（Nummular eczema／discoid eczema）             | 小腿伸側、手背、軀幹                                       | 界線明顯的圓形或橢圓形（錢幣狀）濕疹性斑塊                                     | 常伴皮膚乾燥；冬季好發；易誤診為環癬（tinea corporis）→ KOH 鏡檢   | 局部類固醇＋保濕；避免乾燥刺激                           |
| **鬱積性皮炎**（Stasis dermatitis／venous eczema）            | 雙側小腿下 1/3，內踝附近                                   | 紅褐色含鐵血黃素沉著、靜脈曲張、水腫                                           | 靜脈功能不全為基礎；可進展為脂肪皮膚硬化症（lipodermatosclerosis） | 壓迫治療＋抬高患肢；局部類固醇                           |
| **神經性皮炎**，俗稱牛皮蘚（Lichen simplex chronicus, LSC）   | 頸後、踝部、外陰、頭皮（容易搔抓的部位）                   | 苔蘚化斑塊（lichenified plaque）、明顯皮溝皮嵴（skin markings）、搔癢–搔抓循環 | 搔抓行為為核心；心理壓力誘發                                       | 打破搔抓循環；局部強效類固醇；抗組織胺                   |
| **結節性癢疹**（Prurigo nodularis）                           | 四肢伸側（手臂、小腿）                                     | 多個堅硬的結節（firm nodules）、表面脫皮結痂                                   | 慢性搔抓所致；頑固難治                                             | 局部強效類固醇；抗組織胺；UVB；thalidomide               |

### 二、感染性及感染誘發

| 疾病                                                     | 特徵                                                                               | 鑑別線索                                                                      | 治療原則                                      |
| -------------------------------------------------------- | ---------------------------------------------------------------------------------- | ----------------------------------------------------------------------------- | --------------------------------------------- |
| **感染性濕疹樣皮炎**（Infectious eczematoid dermatitis） | 環繞感染灶（膿瘍、竇道、潰瘍）向外蔓延的濕疹樣紅斑                                 | 有明確感染灶；細菌培養陽性                                                    | 治療原發感染灶（抗生素）；局部類固醇緩解炎症  |
| **癬菌疹**（Dermatophytid／id reaction）                 | 身體某處有真菌感染（如足癬），免疫過敏反應導致遠端部位（手掌常見）出現汗皰疹樣水疱 | 足癬（tinea pedis）同時存在；id 病灶 KOH 鏡檢陰性（水疱是過敏反應，不含真菌） | 治療原發真菌感染（口服抗真菌藥）；id 病灶自癒 |
| **疥瘡**（Scabies）                                      | 指間、腕部屈側、乳暈、陰囊等；隧道（burrows）                                      | 夜間劇癢；家庭群聚；皮膚鏡（dermoscopy）見疥蟲                                | Permethrin 5% 乳膏（全身塗抹）；全家同時治療  |
| **膿皰病**（Impetigo）                                   | 面部、鼻周、口周                                                                   | 蜂蜜色結痂（honey-colored crusts）；細菌繼發感染                              | 局部抗生素（mupirocin）；廣泛者口服抗生素     |

### 三、系統性疾病相關

| 疾病                                                           | 皮膚表現                                                                     | 系統性線索                                                           | 治療原則                                                      |
| -------------------------------------------------------------- | ---------------------------------------------------------------------------- | -------------------------------------------------------------------- | ------------------------------------------------------------- |
| **乳糜瀉相關**（皰疹樣皮炎 Dermatitis herpetiformis）          | 劇癢群聚性水疱，分布於肘伸側、膝、臀、頭皮                                   | 對無麩質飲食反應；小腸切片：絨毛萎縮；IgA 抗 tTG 陽性                | 嚴格的無麩質飲食；Dapsone 快速控制搔癢水疱                    |
| **系統性接觸性皮炎**（Systemic contact dermatitis）            | 經口攝入或靜脈注射致敏原後引發全身性濕疹樣反應（如鎳、藥物）                 | 對應貼片試驗致敏原；低鎳飲食改善                                     | 避免致敏原（口服、注射）；局部類固醇                          |
| **肝臟疾病相關搔癢**（Cholestatic pruritus）                   | 全身搔癢 → 搔抓 → 繼發濕疹化（eczematization）                               | 黃疸、膽汁鬱積；肝功能異常                                           | 針對肝病原因；Cholestyramine、naltrexone、rifampin            |
| **甲狀腺功能低下**（Hypothyroid xerosis）                      | 皮膚乾燥粗厚→ 繼發濕疹化                                                     | TSH 升高；眉毛外側脫落（madarosis）                                  | 甲狀腺素補充（levothyroxine）；加強保濕                       |
| **腎臟病相關搔癢**（Uremic pruritus）                          | 全身搔癢 → 繼發濕疹化                                                        | 腎功能異常；透析患者                                                 | 優化透析；局部保濕；Gabapentin、UVB                           |
| **POEMS 症候群**                                               | 色素沉著、皮膚增厚                                                           | M蛋白、器官腫大、多發性神經病變                                      | 治療潛在漿細胞病變（化療、自體幹細胞移植）                    |
| **修格蘭症候群相關乾燥**（Sjögren xerosis）                    | 嚴重皮膚乾燥 → 全身搔癢 → 繼發濕疹化（eczematization）                       | 口乾、眼乾（sicca symptoms）；anti-Ro/La 陽性；淚腺唾液腺功能減退    | 加強保濕；人工淚液／唾液；羥氯喹                              |
| **慢性移植體抗宿主疾病**（Chronic GVHD）                       | 苔蘚樣丘疹斑塊（lichenoid）、硬皮症樣（sclerodermoid）；可呈廣泛濕疹樣紅皮症 | 造血幹細胞移植史；口腔黏膜白紋（lichenoid GVHD）；肝功能異常         | 局部類固醇／Tacrolimus；全身免疫抑制（類固醇、MMF）           |
| **自體免疫黃體素皮膚炎**（Autoimmune progesterone dermatitis） | 月經週期前後規律發作的濕疹樣丘疹水疱、蕁麻疹、多形紅斑樣病灶                 | 黃體期（luteal phase）反覆發作；對自體黃體素過敏；卵巢抑制治療有效   | NSAIDs、抗組織胺；嚴重者卵巢抑制或抗黃體素治療                |
| **成人史迪爾氏症**（Adult-onset Still's disease, AOSD）        | 短暫橙紅色斑丘疹（evanescent salmon-pink rash），伴隨發燒高峰出現            | 間歇高燒、關節痛、咽喉痛；鐵蛋白（ferritin）顯著升高；皮疹隨發燒消退 | NSAIDs；全身類固醇；IL-1/IL-6 抑制劑（anakinra、tocilizumab） |

### 四、副腫瘤性及血液疾病相關

| 疾病                                                             | 特徵                                                              | 警示線索（Red flags）                                                     | 治療原則                                                       |
| ---------------------------------------------------------------- | ----------------------------------------------------------------- | ------------------------------------------------------------------------- | -------------------------------------------------------------- |
| **後天性魚鱗癬**（Acquired ichthyosis）                          | 廣泛乾燥脫屑（魚鱗状）、搔癢                                      | 成人新發；何杰金氏淋巴瘤、NHL                                             | 治療潛在惡性腫瘤；加強保濕＋角質溶解劑                         |
| **何杰金氏淋巴瘤搔癢**（Hodgkin lymphoma pruritus）              | 全身頑固搔癢 → 繼發濕疹樣變化                                     | 體重減輕、盜汗、淋巴結腫大（B症狀）                                       | 治療淋巴瘤（化放療）；症狀性抗組織胺、局部類固醇               |
| **副腫瘤性天皰瘡**（Paraneoplastic pemphigus, PNP）              | 嚴重口腔糜爛 + 多形性皮膚病灶                                     | 嚴重口腔病灶；CLL、NHL、Castleman disease                                 | 治療潛在腫瘤；全身類固醇＋rituximab；免疫抑制                  |
| **皮肌炎**（Dermatomyositis）                                    | 技工手（mechanics hands）可呈濕疹樣                               | 向光性紅斑、Gottron丘疹、近端肌無力；肺癌、卵巢癌篩查                     | 全身類固醇；免疫抑制（MTX、MMF）；防曬；篩查潛在惡性腫瘤       |
| **藍格罕細胞組織球增生症**（Langerhans cell histiocytosis, LCH） | 脂漏性皮炎樣紅斑丘疹（seborrheic-like），好發於頭皮、腋下、腹股溝 | 淋巴結腫大、骨痛、肺部浸潤、尿崩症；皮膚活檢：CD1a+/Langerin+             | 局部類固醇（局限型）；全身化療（廣泛型）；vemurafenib（BRAF+） |
| **壞死性遊走性紅斑**（Necrolytic migratory erythema, NME）       | 軀幹四肢遠端遊走性邊界不清紅斑，中央水疱糜爛結痂，類似嚴重濕疹    | 高血糖（glucagonoma）、體重減輕、舌炎、蛀牙；血漿 glucagon 升高           | 切除 glucagonoma；補充胺基酸、鋅；全身類固醇                   |
| **嗜酸性白血球增多症候群**（Hypereosinophilic syndrome, HES）    | 廣泛搔癢性濕疹樣丘疹斑塊、蕁麻疹、紅皮症                          | 嗜酸性球 >1500/µL 持續6個月以上；心臟（Loeffler心內膜炎）、神經、肺部侵犯 | 全身類固醇；imatinib（FIP1L1-PDGFRA+）；mepolizumab            |
| **成人T細胞白血病／淋巴瘤**（ATLL）                              | 廣泛性濕疹樣紅斑丘疹、可進展為紅皮症（erythroderma）              | 來自 HTLV-1 流行區；淋巴腺腫大、高血鈣、LDH 升高；皮膚活檢見捲曲核淋巴球  | 化療（CHOP）；抗病毒（zidovudine＋IFN-α）；莫努匹克（限局部）  |
| **白血病皮膚浸潤**（Leukemia cutis）                             | 單發或多發紅褐色丘疹結節；可呈廣泛濕疹樣斑塊（罕見）              | 周邊血液抹片異常；依白血病類型不同（CLL、AML 最常見皮膚表現）             | 治療原發白血病（化療、標靶治療）；局部放射治療                 |

### 五、藥物疹

| 類型                                                      | 特徵                                      | 常見致因藥物                                              | 治療原則                                             |
| --------------------------------------------------------- | ----------------------------------------- | --------------------------------------------------------- | ---------------------------------------------------- |
| **固定性藥疹**（Fixed drug eruption）                     | 反覆在同一部位出現的紅褐色圓形斑塊        | NSAIDs、磺胺類、抗生素                                    | 停用致病藥物；局部類固醇加速消退                     |
| **泛發性藥物疹**（Morbilliform/eczematous drug eruption） | 廣泛對稱性紅斑丘疹                        | 抗生素（ampicillin + EBV 感染幾乎必發）、抗癲癇藥         | 停用致病藥物；抗組織胺；局部／全身類固醇             |
| **DRESS/藥物過敏症候群**                                  | 廣泛性皮疹 + 發燒 + 淋巴結腫大 + 內臟侵犯 | 抗癲癇藥（carbamazepine、phenytoin）、allopurinol、磺胺類 | 立即停藥；全身類固醇；支持療法（住院）；監測器官功能 |
| **光敏感性藥物疹**（Photo drug eruption）                 | 僅曝光部位出現濕疹樣反應                  | Thiazide、tetracycline、amiodarone、NSAIDs                | 停藥；嚴格防曬（物理性防曬）；局部類固醇             |

**高風險藥物分類速查**（依藥物類別，非依反應形態）：

| 高風險藥物                                            | 最常見反應類型                               | 注意事項                                                   |
| ----------------------------------------------------- | -------------------------------------------- | ---------------------------------------------------------- |
| **Penicillins／Amoxicillin**                          | 泛發性斑丘疹（morbilliform）                 | EBV 合併用 ampicillin → 接近 100% 發疹（非過敏）           |
| **Sulfonamides**（磺胺類）                            | Morbilliform、SJS/TEN、DRESS                 | HIV 患者風險更高                                           |
| **抗癲癇藥**（Carbamazepine、Phenytoin、Lamotrigine） | DRESS、SJS/TEN、Morbilliform                 | HLA-B\*1502（亞洲人 SJS/TEN 篩查）；Lamotrigine 需緩慢加藥 |
| **Allopurinol**                                       | DRESS、SJS/TEN                               | HLA-B\*5801（亞洲人風險篩查）；腎功能不全者風險↑           |
| **NSAIDs**                                            | 固定性藥疹、蕁麻疹、Morbilliform             | 最常見固定性藥疹的致因藥物之一                             |
| **Vancomycin**                                        | 紅人症候群（Red man syndrome）、Morbilliform | 紅人症候群非過敏（infusion rate 相關）                     |
| **Doxycycline／Tetracyclines**                        | 光敏感性皮膚炎（photo-onycholysis）          | 停藥後消退                                                 |
| **Thiazide 利尿劑**                                   | 光敏感性藥疹、Lichenoid 藥疹                 | 潛伏期可長達數月                                           |
| **Piperacillin-Tazobactam**                           | Morbilliform（住院患者最常見）               | 常被誤認為感染本身惡化                                     |

### 六、類濕疹

| 疾病                                                       | 外觀相似點                             | 鑑別要點                                                                                     | 治療原則                                                                            |
| ---------------------------------------------------------- | -------------------------------------- | -------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| **乾癬**（Psoriasis）                                      | 脫屑性斑塊                             | 境界清楚、銀白色厚鱗屑、Auspitz徵（刮除後點狀出血）、好發肘伸側／膝／頭皮                    | 局部類固醇／維生素D衍生物；UVB；生物製劑                                            |
| **環癬**（Tinea corporis）                                 | 環形脫屑性紅斑                         | 邊緣有活躍性丘疹水疱、中央改善、KOH 陽性                                                     | 局部抗真菌藥；廣泛者口服 terbinafine                                                |
| **亞急性皮膚紅斑狼瘡**（SCLE）                             | 環形或多環形脫屑性斑塊                 | 光敏感分布（臉、上胸、背、伸臂）；anti-Ro/La 陽性；對類固醇反應差                            | 防曬；局部類固醇；羥氯喹；全身免疫抑制                                              |
| **圓盤狀紅斑狼瘡**（Discoid lupus, DLE）                   | 境界清楚紅色斑塊伴黏著性鱗屑           | 毛囊角栓（follicular plugging）、萎縮、色素紊亂、疤痕性禿髮；活檢見 interface dermatitis     | 強效局部類固醇／Tacrolimus；羥氯喹                                                  |
| **扁平苔癬**（Lichen planus）                              | 紫紅色多角形扁平丘疹                   | Wickham striae（白色網狀紋）；口腔黏膜侵犯；劇癢；好發腕屈側、小腿、下背                     | 局部類固醇；嚴重者口服類固醇、acitretin、UVB                                        |
| **環狀肉芽腫**（Granuloma annulare）                       | 環形膚色或淡紅色丘疹群                 | 無鱗屑（non-scaly）；好發手背、足背、肘膝伸側；自限性；組織學：palisaded granuloma           | 觀察；局部類固醇或冷凍治療                                                          |
| **多形性紅斑**（Erythema multiforme）                      | 靶形病灶（target lesions）             | 口腔黏膜侵犯常見；與 HSV 感染相關（EM minor）或藥物相關（EM major/SJS）；急性發作            | 治療 HSV（若相關）；停藥（若藥物）；支持療法                                        |
| **類肉瘤病**（Sarcoidosis）                                | 紅褐色或紫褐色浸潤性斑塊               | 丘疹結節沿疤痕分布常見（scar sarcoidosis）；肺門淋巴腺腫大、眼葡萄膜炎；活檢見非乾酪性肉芽腫 | 局部類固醇（局限型）；全身類固醇、MTX、抗 TNF-α（廣泛型）                           |
| **硬化性苔癬**（Lichen sclerosus）                         | 白色萎縮性丘疹斑塊、皮膚皺縮           | 好發外陰、肛周（引起慢性搔癢灼痛）；非生殖器部位亦可見；活檢：homogeneous 膠原               | 強效局部類固醇；Tacrolimus；長期追蹤（鱗癌風險）                                    |
| **腸病性肢端皮炎**（Acrodermatitis enteropathica／鋅缺乏） | 肢端（手、足、口周）溼疹樣紅斑水疱糜爛 | 腹瀉、落髮、生長遲緩；血清鋅降低；補鋅後迅速改善                                             | 口服鋅補充（2–3 mg/kg/day）；症狀數日內改善                                         |
| **玫瑰糠疹**（Pityriasis rosea）                           | 軀幹橢圓形脫屑斑                       | 先兆斑（herald patch）、聖誕樹型分布（Christmas tree pattern）、自限性                       | 支持療法；UVB 用於嚴重或頑固搔癢                                                    |
| **天皰瘡**（Pemphigus）                                    | 水疱                                   | 鬆弛性水疱、Nikolsky 徵陽性、口腔黏膜侵犯；DIF 陽性                                          | 全身類固醇；Rituximab；免疫抑制（MTX、MMF、azathioprine）                           |
| **大皰性類天皰瘡**（Bullous pemphigoid）                   | 水疱、搔癢                             | 緊張性大水疱（tense bullae）；老年人；DIF：線狀 IgG/C3                                       | 局部／全身類固醇；Doxycycline＋nicotinamide；Rituximab                              |
| **Paget 病**（Extramammary Paget disease）                 | 肛門生殖器溼疹樣紅斑                   | 長期不癒的單側紅斑；活檢見 Paget cells                                                       | 廣泛局部切除；Mohs 手術；局部 imiquimod（不適合手術者）                             |
| **Bowen 病**（Bowen disease／in situ SCC）                 | 界線清楚脫屑性紅斑斑塊                 | 緩慢擴大、對局部類固醇無反應；活檢確診                                                       | 手術切除；冷凍治療；局部 5-FU／imiquimod                                            |
| **皮膚T細胞淋巴瘤**（CTCL／Mycosis fungoides）             | 慢性濕疹樣斑塊、頑固搔癢               | 對標準治療無反應；皮膚活檢多次才確診；Sézary cells                                           | 局部類固醇、氮芥、UVB（早期）；全身治療（晚期）：干擾素、brentuximab、mogamulizumab |

### 基礎評估

| 工具                                  | 適用情境                                          |

## 第六步：異位性皮膚炎診斷準則
**Hanifin-Rajka Criteria**

**主要準則（Major criteria）— 需符合 3 項以上：**

1. 搔癢（Pruritus）
2. 典型形態與分布（Typical morphology and distribution）
3. 慢性或慢性復發病程（Chronic or chronically relapsing course）
4. 個人或家族異位史（Personal or family history of atopy）

**次要準則（Minor criteria）— 需符合 3 項以上（共 23 項，列舉重要者）：**

- 皮膚乾燥症（xerosis）
- 魚鱗癬、毛囊角化症（ichthyosis vulgaris, KP）
- 即時型（I型）皮膚試驗反應（immediate skin test reactivity）
- 血清 IgE 升高（elevated serum IgE）
- 早發年齡（early age of onset）
- 反覆皮膚感染傾向（tendency to skin infections）
- 手部或足部非特異性皮膚炎（nonspecific hand or foot dermatitis）
- 乳頭濕疹（nipple eczema）
- 眼周濕疹（periorbital darkening／Dennie-Morgan folds）
- 白色皮膚劃紋症（white dermographism）

## 第七步：環形病灶鑑別診斷

> 多種皮膚疾病皆可呈現環形病灶，正確鑑別需注意：**邊緣活性（scaly border vs. smooth border）、中央變化（clearing vs. intact vs. atrophy）、分布部位及伴隨症狀**。以下依臨床常見度整理。

| 疾病                                                | 環形形態特徵                                       | 關鍵鑑別點                                      | 首選檢查                                   | 治療原則                                     |
| --------------------------------------------------- | -------------------------------------------------- | ----------------------------------------------- | ------------------------------------------ | -------------------------------------------- |
| **環癬**（Tinea corporis）                          | 環形紅斑脫屑，邊緣活躍丘疹水疱，中央消退           | 最典型環形病灶；搔癢                            | KOH 鏡檢                                   | 局部抗真菌藥；廣泛者口服 terbinafine         |
| **錢幣狀濕疹**（Nummular eczema）                   | 圓形錢幣狀斑塊，邊界清楚                           | 表面濕疹樣（無中央消退），乾燥皮膚背景          | 病史、外觀；必要時 KOH 排除環癬            | 局部類固醇＋保濕；避免刺激原                 |
| **圓盤狀紅斑狼瘡**（Discoid lupus, DLE）            | 境界清楚紅色圓形斑塊，黏著性鱗屑                   | 毛囊角栓、萎縮、疤痕性禿髮                      | 皮膚活檢（interface dermatitis）           | 強效局部類固醇／塔克莫司；羥氯喹             |
| **環狀肉芽腫**（Granuloma annulare）                | 環形膚色／淡紅色丘疹群，**無鱗屑**                 | 好發手足背／肘膝伸側；多無症狀；自限性          | 多為臨床診斷；活檢（palisaded granuloma）  | 觀察；局部類固醇或冷凍治療                   |
| **多形性紅斑**（Erythema multiforme）               | 靶形病灶（target lesion：同心圓三環結構）          | 中央水疱或糜爛；常伴 HSV 感染                   | 病史（HSV／藥物）；活檢                    | 治療 HSV（若相關）；停藥（若藥物）；支持療法 |
| **固定性藥疹**（Fixed drug eruption）               | 單發或多發圓形紅褐色斑塊                           | 反覆**同一部位**發作；消退後殘留灰紫色色素沉著  | 用藥史；停藥觀察                           | 停用致病藥物；局部類固醇加速消退             |
| **亞急性皮膚紅斑狼瘡**（SCLE）                      | 環形或多環形脫屑性紅斑，對稱分布                   | 光敏感分布（臉、上胸、背、伸臂）；anti-Ro/La(+) | ANA、anti-Ro/La                            | 防曬；局部類固醇；羥氯喹；全身免疫抑制       |
| **玫瑰糠疹**（Pityriasis rosea）                    | 先兆斑（herald patch）→ 聖誕樹型分布之橢圓形丘疹斑 | 自限性（4–8 週）；軀幹為主                      | 外觀診斷                                   | 支持療法；UVB 用於嚴重或頑固搔癢             |
| **環狀乾癬**（Annular psoriasis）                   | 環形斑塊，銀白色脫屑邊緣                           | 多見於兒童；肘膝／頭皮伴典型乾癬病灶            | 外觀；必要時活檢                           | 局部類固醇／維生素D衍生物；UVB；生物製劑     |
| **離心性環狀紅斑**（Erythema annulare centrifugum） | 環形紅斑，邊緣脫屑（trailing scale），向心性擴大   | 中央完全消退；輕癢或無症狀                      | 需排除潛在原因（真菌感染、藥物、惡性腫瘤） | 針對潛在原因；局部類固醇緩解症狀             |

::: warning 臨床警示：何時必須皮膚活檢？

- 對局部類固醇治療 4–6 週無反應
- 單一部位頑固病灶（懷疑 Paget 病、Bowen 病）
- 疑似皮膚 T 細胞淋巴瘤（CTCL）
- 水疱性病灶（確認是否為自體免疫水疱疾病）
- 全身搔癢伴有全身症狀（B 症狀、體重減輕）
