# 兒童皮疹鑑別診斷

**Pediatric Rash Differential Diagnosis**

> **核心概念**：兒童皮疹原因廣泛，從自限性病毒感染到致命性敗血症皆有可能。**最重要的第一步是評估孩子是否 toxic（病懨懨／中毒樣）**，而非急著判斷皮疹形態。Toxic appearance + 皮疹 = 緊急情況，需立刻處理。

## 第一步：快速分診（Rapid Triage）

### Red Flags — 需緊急評估

| Red Flag                           | 可能診斷                                           |
| ---------------------------------- | -------------------------------------------------- |
| 病懨懨／中毒樣（toxic appearance） | 敗血症、腦膜炎球菌血症、TSS、SSSS                  |
| 紫斑／瘀斑（petechiae / purpura）  | 腦膜炎球菌血症、DIC、ITP（well child）、rickettsia |
| 快速進展（數小時內惡化）           | 腦膜炎球菌血症、TSS、SJS/TEN                       |
| 頸部僵硬（nuchal rigidity）        | 腦膜炎、腦膜腦炎                                   |
| 低血壓休克                         | TSS、敗血症                                        |
| 意識改變                           | 腦膜炎、腦炎、敗血症相關腦病變                     |
| 呼吸窘迫                           | 肺炎併發敗血症、麻疹重度肺炎                       |
| 黏膜侵犯（口腔、眼、生殖器）       | SJS/TEN、EM major、Kawasaki、HSV                   |

## 第二步：發燒有無 → 鑑別方向

```
兒童皮疹
    │
    ├── 有毒癥象（toxic appearance）
    │       └── 緊急處理，不須等待確診
    │
    ├── 有發燒 ──┬── 疹子與發燒同時出現
    │           │     → 麻疹、scarlet fever、Kawasaki、enterovirus
    │           │
    │           ├── 疹子在退燒後出現
    │           │     → Roseola（HHV-6/7）經典表現
    │           │
    │           └── 疹子在發燒數日後出現
    │                 → Measles、EBV、Dengue、Gianotti-Crosti
    │
    └── 無發燒（或微燒）
              ├── 搔癢為主 → 異位性皮膚炎、疥瘡、蕁麻疹、接觸性皮炎
              │
              ├── 水疱 → 膿痂疹（impetigo）、molluscum、tinea、contact
              │
              └── 無症狀 → 病毒疹恢復期、pityriasis rosea、milium、血管瘤
```

## 第三步：皮疹形態分類（Morphology Approach）

### 一、發燒 + 斑丘疹（Maculopapular / Morbilliform ± Fever）

| 疾病                                                    | 發燒特徵                               | 皮疹特徵                                                                         | 關鍵鑑別點                                                                                    | 治療原則                                                    |
| ------------------------------------------------------- | -------------------------------------- | -------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- | ----------------------------------------------------------- |
| **麻疹**（Measles / Rubeola）                           | 高燒 39–40°C，持續 4–7 天              | 暗紅色斑丘疹，耳後 → 頭臉 → 軀幹 → 四肢（由上往下）；退疹順序同出現順序          | 3C：咳嗽（cough）、流鼻水（coryza）、結膜炎（conjunctivitis）；Koplik spots（頰黏膜白色斑點） | 支持療法（退燒、水分）；Vitamin A（重症）；通報             |
| **德國麻疹**（Rubella）                                 | 低-中度發燒                            | 粉紅色斑丘疹，臉 → 軀幹 → 四肢，3 天消退                                         | 耳後／枕部淋巴結腫大（post-auricular/occipital LAP）；關節痛（成人/青少年）                   | 支持療法；避免接觸孕婦                                      |
| **玫瑰疹**（Roseola infantum, HHV-6/7）                 | **高燒 3–5 天（39–40°C），退燒後出疹** | 玫瑰色斑丘疹，軀幹 → 頸→四肢                                                     | 退燒後出疹是經典特徵（defervescence = rash）；好發 6–24 月嬰兒；熱性痙攣常見                  | 支持療法；退燒；熱性痙攣處理                                |
| **傳染性紅斑**（Erythema infectiosum, Parvovirus B19）  | 低-中度發燒（有時無）                  | **拍打頰紅斑**（slapped cheek）→ 軀幹四肢 lace-like 網狀紅斑                     | 皮疹於遇熱（洗澡、日曬）後加重；關節痛（成人）；胎兒水腫（pregnant高危）                      | 支持療法；避免遇熱加重皮疹；孕婦需追蹤胎兒超音波            |
| **猩紅熱**（Scarlet fever, Group A Strep）              | 高燒 38–40°C，伴喉嚨痛                 | **砂紙樣紅疹**（sandpaper rash），軀幹 → 全身；壓之褪色；皮膚皺褶处 Pastia lines | 草莓舌（strawberry tongue）、扁桃腺化膿、頸部淋巴腫大；快速鏈球菌抗原檢測或喉頭培養陽性       | Penicillin 或 Amoxicillin 10 天；支持療法                   |
| **川崎病**（Kawasaki disease）                          | 高燒 ≥5 天，抗生素無效                 | 多形性紅斑（軀幹四肢）；會陰部脫屑                                               | **雙眼結膜炎無分泌物**、草莓舌、手腳腫脹脫皮、頸部淋巴 ≥1.5 cm；冠狀動脈瘤風險                | IVIG（2 g/kg）＋高劑量 Aspirin；類固醇（IVIG 無反應者）     |
| **登革熱**（Dengue fever）                              | 高燒（saddle-back fever pattern）      | 全身斑丘疹；可伴點狀出血                                                         | 頭痛、眼窩後痛、肌肉關節痛（breakbone fever）；血小板↓；tourniquet test +；流行區旅遊         | 支持療法；水分補充；避免 NSAIDs（出血風險）；警示期嚴密監測 |
| **傳染性單核球增多症**（EBV／Infectious mononucleosis） | 中度發燒，持續 1–2 週                  | 全身斑丘疹（特別是服用 ampicillin 後）                                           | 疲憊、淋巴腺腫大、扁桃腺白色滲出物；heterophile Ab +；肝功能異常                              | 支持療法；避免 ampicillin；短期類固醇（扁桃腺嚴重腫大阻塞） |
| **Gianotti-Crosti 症候群**（嬰兒丘疹性肢端皮炎）        | 低-中度發燒（常見）                    | 對稱性扁平紅色丘疹，分布於**臉頰、臀部、四肢伸側**，持續 2–8 週                  | 與 HBV、EBV、CMV 等多種病毒感染相關；無口腔黏膜病灶                                           | 支持療法（自限性）；局部類固醇（嚴重搔癢）                  |
| **非特異性病毒疹**（Viral exanthem, nonspecific）       | 低-中度發燒                            | 全身散在粉紅色斑丘疹，非特異分布                                                 | 排除以上特徵性疾病後診斷；自限性                                                              | 支持療法；退燒；自限性                                      |
| **DRESS 症候群**（藥物過敏）                            | 高燒，皮疹在用藥後 2–6 週出現          | 廣泛濕疹樣斑丘疹，可進展為紅皮症                                                 | 嗜酸性球↑、淋巴腺腫大、肝功能異常；特定藥物（AEDs、allopurinol、磺胺類）                      | 立即停藥；全身類固醇；住院支持療法；監測器官功能            |

### 二、發燒 + 水疱（Vesiculobullous ± Fever）

| 疾病                                                              | 發燒特徵                | 皮疹特徵                                                                                   | 關鍵鑑別點                                                               | 治療原則                                                      |
| ----------------------------------------------------------------- | ----------------------- | ------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------ | ------------------------------------------------------------- |
| **手足口病**（HFMD, Enterovirus 71 / Coxsackie A16）              | 高燒 38–39°C            | 手掌、腳掌、臀部、口腔水疱潰瘍                                                             | 口腔疼痛導致拒食；典型分布（手+足+口）；EV71 可併發腦幹腦炎              | 支持療法；口腔疼痛控制；觀察 EV71 重症（抽搐、意識改變）      |
| **水痘**（Varicella / Chickenpox）                                | 中度發燒，出疹後 3–5 天 | **向心性**（centripetal）：軀幹 → 頭臉 → 四肢；**不同時期病灶共存**（斑疹→丘疹→水疱→結痂） | 經典「露珠在玫瑰花瓣上」（dew drop on rose petal）；劇癢；可接種疫苗預防 | 支持療法（止癢、退燒）；Acyclovir（重症、青少年、免疫不全）   |
| **單純疱疹**（HSV gingivostomatitis）                             | 高燒（初次感染）        | 口腔黏膜群聚水疱糜爛；少數擴散至口周皮膚                                                   | 口腔疼痛嚴重、流口水、拒食；年幼兒童初次感染表現最重                     | Acyclovir 口服或靜脈；口腔疼痛控制（lidocaine 凝膠）          |
| **葡萄球菌燙傷樣皮膚症候群**（SSSS, Staph scalded skin syndrome） | 中度發燒                | 廣泛紅斑 → 大水疱 → 大片脫皮（燙傷樣）                                                     | Nikolsky sign +；**黏膜不受侵犯**（與 SJS/TEN 鑑別）；好發 <5 歲兒童     | 住院；IV 抗葡萄球菌抗生素（oxacillin、clindamycin）；皮膚護理 |
| **毒性休克症候群**（TSS, Staph / Strep）                          | 高燒（>39°C）           | 瀰漫性日燒樣紅斑（sunburn-like diffuse erythroderma）→ 後期脫屑（手掌腳掌）                | 低血壓、意識改變、多器官功能不全；黏膜充血、草莓舌                       | 住院；IV 抗生素；升壓劑、大量輸液；感染源清除（引流、清創）   |

### 三、發燒 + 紫斑／瘀點（Petechiae / Purpura ± Fever）

| 疾病                                                        | 發燒特徵                       | 皮疹特徵                                                                            | 關鍵鑑別點                                                                    | 治療原則                                                    |
| ----------------------------------------------------------- | ------------------------------ | ----------------------------------------------------------------------------------- | ----------------------------------------------------------------------------- | ----------------------------------------------------------- |
| **腦膜炎球菌血症**（Meningococcemia）                       | 高燒，快速進展                 | 非壓迫性紫斑（non-blanching purpura），軀幹下肢；可伴網狀青斑（livedo reticularis） | **病懨懨（toxic） + 紫斑 = 緊急**；腦膜炎徵象（頸僵、意識改變）；PCN 緊急治療 | **緊急住院**；IV Ceftriaxone 或 Penicillin；隔離；支持療法  |
| **免疫性血小板低下**（ITP）                                 | 通常無發燒（或病毒感染後微燒） | 皮下出血點（petechiae）、瘀青（ecchymosis）                                         | **孩子看起來很好！**（well-appearing）；無肝脾腫大；血小板↓；骨髓正常         | 觀察（輕度）；IVIG 或類固醇（Plt <20K 或活動性出血）        |
| **過敏性紫斑**（Henoch-Schönlein purpura / IgA vasculitis） | 低度發燒（常見）               | 觸診性紫斑（palpable purpura），主要分布於**下肢／臀部**                            | 關節炎（膝踝）、腹痛、類血便；小兒最常見血管炎；IgA 沉積於 DIF                | 支持療法；NSAIDs（關節痛）；類固醇（嚴重腹痛、腎炎）        |
| **登革出血熱**（Dengue hemorrhagic fever）                  | 高燒                           | 點狀出血、紫斑；tourniquet test 陽性                                                | 血漿滲漏（hematocrit ↑）、血小板↓；警示期（critical phase）在退燒時           | **住院**；IV 輸液監測；避免 NSAIDs；血小板輸注（嚴重出血）  |
| **立克次體感染**（Rocky Mountain spotted fever, RMSF）      | 高燒                           | 紫斑性皮疹，始於**手腕腳踝** → 往軀幹擴散（centripetal）                            | 蜱蟲叮咬史；頭痛、肌肉痛；doxycycline 治療                                    | Doxycycline（兒童可用）；早期治療可降低死亡率               |
| **瀰漫性血管內凝血**（DIC）                                 | 視原發疾病                     | 廣泛紫斑、瘀斑、穿刺處出血                                                          | 血小板↓、PT/PTT↑、D-dimer↑、fibrinogen↓；潛在敗血症或器官衰竭                 | 治療原發病因；支持療法（血小板、FFP、cryoprecipitate 補充） |
| **新生兒狼瘡**（Neonatal lupus）                            | 不一定發燒                     | 環形紅斑性脫屑斑塊（母體 anti-Ro/La 抗體通過胎盤）                                  | anti-Ro/La 陽性（母體及嬰兒）；心臟傳導阻滯（CHB，不可逆）；皮疹數月自癒      | 皮疹觀察（自限性）；防曬；CHB 需 pacemaker；心臟科長期追蹤  |

### 四、發燒 + 蕁麻疹樣（Urticarial ± Fever）

| 疾病                                                 | 特徵                                                                     | 鑑別要點                                                                     | 治療原則                                                      |
| ---------------------------------------------------- | ------------------------------------------------------------------------ | ---------------------------------------------------------------------------- | ------------------------------------------------------------- |
| **急性蕁麻疹**（Acute urticaria, infection-induced） | 感染（尤其是 URTI）誘發的過敏反應                                        | 搔癢性膨疹（wheals），單個 <24h；自限性                                      | 抗組織胺；治療原發感染（若明確）                              |
| **多形性紅斑**（Erythema multiforme, EM）            | 靶狀病灶（target lesions），伴發燒及黏膜侵犯                             | 與 HSV 相關（EM minor）；藥物亦可引起（EM major）；通常無 SJS 的嚴重全身症狀 | 治療 HSV（若相關）；停藥（若藥物）；支持療法                  |
| **血清病樣反應**（Serum sickness-like reaction）     | 用藥後（常見 cefaclor）7–14 天出現蕁麻疹樣皮疹 + 發燒 + 關節痛           | 良性自限；無血管炎或腎臟侵犯（與 HSP 鑑別）                                  | 停藥；抗組織胺；嚴重者類固醇短期療程                          |
| **史迪爾氏症**（Still's disease, AOSD）              | 間歇高燒 + 褪色性鮭魚紅疹（evanescent salmon-pink rash）伴隨發燒高峰出現 | 關節痛、咽喉痛、血清 ferritin 顯著升高                                       | NSAIDs；全身類固醇；IL-1/IL-6 抑制劑（anakinra、tocilizumab） |

## 第四步：無發燒（或微燒）皮疹鑑別

### 五、無發燒 — 搔癢性皮疹

| 疾病                                                | 皮疹特徵                                                         | 鑑別要點                                                                 | 治療原則                                                 |
| --------------------------------------------------- | ---------------------------------------------------------------- | ------------------------------------------------------------------------ | -------------------------------------------------------- |
| **異位性皮膚炎**（Atopic dermatitis／Eczema）       | 乾燥、紅斑、脫屑、苔蘚化；嬰兒：臉頰伸側；兒童：肘窩膕窩         | 劇癢、慢性復發、個人或家族過敏史（哮喘、鼻炎）                           | 局部類固醇＋保濕；Tacrolimus；抗組織胺；嚴重者 dupilumab |
| **疥瘡**（Scabies）                                 | 隧道（burrows）、丘疹水疱；分布於指縫、腕屈側、腋下、腹股溝      | **夜間劇癢**；家庭群聚；皮膚鏡見疥蟲；嬰兒可出現手掌腳掌水疱             | Permethrin 5% 乳膏（全身頸到腳）；全家同時治療           |
| **蕁麻疹**（Urticaria, allergic）                   | 搔癢性膨疹（wheals），單個病灶 <24小時                           | 可找到過敏原（食物、藥物、物理刺激）；慢性自發性（>6 週）常找不到原因    | 抗組織胺；避免過敏原；嚴重者短期類固醇                   |
| **丘疹性蕁麻疹**（Papular urticaria／Insect bites） | 群聚性紅色丘疹或水疱，常見於暴露部位                             | 蟲咬過敏反應（跳蚤、蚊子最常見）；中央可見叮咬點                         | 抗組織胺；局部類固醇；防蟲措施                           |
| **接觸性皮膚炎**（Contact dermatitis）              | 紅斑水疱脫屑，分布於接觸部位（形狀對應刺激物）                   | 常見原因：植物（poison ivy）、鎳、化粧品、尿布                           | 避免刺激物／過敏原；局部類固醇                           |
| **尿布疹**（Diaper dermatitis）                     | 臀部、會陰紅斑；可伴糜爛                                         | 刺激原為主（尿液糞便）；續發念珠菌感染常見（衛星病灶 satellite lesions） | 勤換尿布；氧化鋅藥膏；抗黴菌藥膏（續發念珠菌感染時）     |
| **汗疹**（Miliaria／Heat rash）                     | 細小透明水疱（miliaria crystallina）或紅色丘疹（miliaria rubra） | 悶熱環境、過度穿衣；頸部、胸部、皮膚皺褶處常見                           | 降溫通風、減少衣物                                       |
| **白色糠疹**（Pityriasis alba）                     | 臉頰境界不清的淺白斑、輕度脫屑                                   | 與皮膚乾燥、日曬相關；常見於異位性體質兒童；非黴菌                       | 保濕；局部低效類固醇（發炎期）                           |

### 六、無發燒 — 非搔癢性／無症狀皮疹

| 疾病                                                      | 皮疹特徵                                                       | 鑑別要點                                                                 | 治療原則                                                      |
| --------------------------------------------------------- | -------------------------------------------------------------- | ------------------------------------------------------------------------ | ------------------------------------------------------------- |
| **玫瑰糠疹**（Pityriasis rosea）                          | 先兆斑（herald patch，較大橢圓形斑）→ 聖誕樹型分布的細小脫屑斑 | 軀幹為主，四肢近端；自限性（4–8 週）；無需治療                           | 支持療法；UVB（嚴重搔癢）                                     |
| **傳染性軟疣**（Molluscum contagiosum）                   | 膚色蠟樣圓頂形丘疹，中央有臍凹（central umbilication）         | 分布腋下、肘窩、腹股溝；可自發消退（但需數月至數年）；異位性兒童更易傳播 | 觀察（自限）；冷凍／刮除（選擇性）；局部免疫調節劑            |
| **膿痂疹**（Impetigo）                                    | 蜂蜜色厚結痂（honey-colored crusts）；水疱膿疱                 | 非搔癢但具傳染性（contagious）；金黃色葡萄球菌為主                       | 局部抗生素（mupirocin、fusidic acid）；廣泛者口服抗生素       |
| **體癬**（Tinea corporis）                                | 環形脫屑性紅斑，邊緣活躍（active border）                      | KOH 陽性；接觸動物（貓狗）或感染者                                       | 局部抗真菌藥；廣泛者口服 terbinafine                          |
| **單純性血管瘤**（Infantile hemangioma）                  | 鮮紅色凸起斑塊（草莓狀），出生後數週出現，1歲後慢慢消退        | 好發頭頸部；位於面部 midline 或腰骶部需超音波排除內臟血管瘤              | 觀察（多數消退）；Propranolol（高危位置：眼周、鼻、口、肛門） |
| **鮭魚斑**（Salmon patch / Nevus simplex）                | 後頸、眼瞼、眉間淡紅色斑，哭時更明顯                           | 良性胎記；大多數（眼瞼、眉間）數月內消退                                 | 無需治療                                                      |
| **蒙古斑**（Mongolian spot）                              | 藍灰色斑，好發臀部、腰骶部                                     | 亞洲裔常見；數年內逐漸消退；勿誤認為瘀青                                 | 無需治療                                                      |
| **葡萄酒色斑**（Port wine stain / Nevus flammeus）        | 邊界清楚的紫紅色斑塊                                           | 不會消退；分布 V1 區域需注意 Sturge-Weber 症候群                         | Pulsed dye laser；神經科／眼科評估（V1 分布）                 |
| **新生兒毒性紅斑**（Erythema toxicum neonatorum）         | 出生後 1–3 天出現的紅斑基底上黃白色丘疹膿疱                    | 好發軀幹、臉；**預後良好**，1–2 週自癒；嗜酸性球浸潤                     | 無需治療（自限性）                                            |
| **新生兒頭部膿疱病**（Neonatal cephalic pustulosis）      | 額頭、臉頰、下巴紅色丘疹膿疱（新生兒痤瘡）                     | 出生後 2–4 週出現；數月內自癒；不需治療                                  | 觀察（自限性）；嚴重者局部抗黴菌（ketoconazole 乳膏）         |
| **嬰兒脂漏性皮膚炎**（Seborrheic dermatitis／Cradle cap） | 頭皮油膩黃色脫屑（乳痂）；可延伸至臉、耳後、腋下               | 出生後數月常見；自限性；與 Malassezia 相關                               | 頭皮軟化（嬰兒油）＋溫水清洗；抗黴菌洗劑（ketoconazole）      |

## 第五步：診斷輔助工具（Diagnostic Workup）

### 基礎評估

| 工具                                          | 適用情境                                                                           |
| --------------------------------------------- | ---------------------------------------------------------------------------------- |
| CBC + differential                            | 白血球分類（病毒感染→lymphocyte↑；細菌→neutrophil↑）；嗜酸性球（寄生蟲、藥物過敏） |
| CRP、PCT（Procalcitonin）                     | 細菌感染（PCT >0.5 ng/mL 支持細菌）vs 病毒感染                                     |
| 血液培養（Blood culture）                     | 懷疑敗血症、腦膜炎球菌血症                                                         |
| 喉嚨快篩（Throat rapid Strep test / culture） | 懷疑猩紅熱                                                                         |
| KOH 顯微鏡檢查                                | 懷疑黴菌感染（tinea）                                                              |
| 皮膚活檢（Skin biopsy）                       | 無法確診的不明皮疹；懷疑血管炎（HSP）、自體免疫                                    |
| DIF（直接免疫螢光）                           | 懷疑自體免疫水疱疾病                                                               |
| 血清學（Serology）                            | EBV（EBV VCA IgM）、Parvovirus B19（IgM）、Measles（IgM）、Dengue（NS1、IgM）      |
| 血小板計數（Platelet count）                  | 懷疑 ITP、Dengue、DIC                                                              |
| Coagulation profile（PT/PTT/D-dimer）         | 懷疑 DIC                                                                           |
| PCR／病毒培養                                 | 懷疑特定病毒（HSV、VZV、Enterovirus 71、Measles）                                  |
| 心臟超音波（Echocardiogram）                  | 川崎病 → 排除冠狀動脈瘤                                                            |

## 第六步：年齡層特殊考量

| 年齡層                  | 好發診斷                                                                            |
| ----------------------- | ----------------------------------------------------------------------------------- |
| **新生兒（< 1 月）**    | Erythema toxicum、新生兒痤瘡、HSV（生產時感染）、念珠菌尿布疹、細菌性敗血症相關皮疹 |
| **嬰兒（1–12 月）**     | 異位性皮膚炎（2月起）、玫瑰疹（6–24月高峰）、脂漏性皮炎、尿布疹、血管瘤             |
| **幼兒（1–5 歲）**      | HFMD、水痘、玫瑰疹、猩紅熱、Gianotti-Crosti、烏龜/寵物相關 tinea、疥瘡              |
| **學齡兒童（5–12 歲）** | Parvovirus B19、猩紅熱、川崎病、ITP、HSP、傳染性軟疣、玫瑰糠疹                      |
| **青少年**              | EBV、多形性紅斑、藥物疹、DRESS、膿瘍性痤瘡、STI 相關皮疹（HIV、梅毒、HSV）          |

| 有   | 紫斑／瘀斑（toxic）  | Meningococcemia（緊急）                   |

## 關鍵鑑別要點（Key Diff Tips）

1. **Toxic appearance = 敗血症／Meningococcemia，先救命再診斷**
2. **玫瑰疹經典：高燒 3–5 天→退燒→出疹（other way around）**
3. **麻疹：3C（cough, coryza, conjunctivitis）+ Koplik spots**
4. **川崎病：高燒 ≥5 天 + 眼睛紅 but no discharge + 手指腫脹脫皮**
5. **猩紅熱：砂紙樣皮疹 + 草莓舌 + Pastia lines**
6. **HSP：lower extremity palpable purpura + arthralgia + abdominal pain**
7. **SSSS vs SJS/TEN：黏膜是否侵犯（SSSS 不傷黏膜）**
8. **手足口病：手掌腳掌水疱 + 口腔潰瘍（但膝蓋臀部也常見）**
9. **異位性皮膚炎：慢性復發 + 劇癢 + 過敏家族史 + 特定分布**
10. **疥瘡：夜間搔癢 + 家庭群聚 + 指縫隧道**

::: warning 臨床警示：何時必須立即轉診／住院？

- Toxic appearance + 皮疹
- 非壓迫性紫斑／瘀斑（non-blanching purpura）
- 快速進展的皮疹（數小時內擴散）
- 高燒 + 意識改變、頸部僵硬
- 低血壓、休克、呼吸窘迫
- 疑川崎病（發燒 ≥5 天 + 任意 4 項）
- 疑 SJS/TEN（黏膜侵犯 + 水疱 + 表皮剝離 > 30%）
- 使用抗生素後出現廣泛性紅疹 + 嗜酸性球升高（疑 DRESS）
