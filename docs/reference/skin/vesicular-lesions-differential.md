# 水疱性病灶鑑別診斷與治療

**Differential Diagnosis of Vesicular Lesions by Associated Symptoms & Treatment**

> **核心概念**：水疱（vesicle，< 0.5 cm）與大水疱（bulla，≥ 0.5 cm）的鑑別，需結合**分布形態、伴隨症狀、發病速度、年齡層**綜合判斷。本文以伴隨症狀為主軸，建立臨床思考路徑。

## 第一步：水疱基本形態學分類

| 分類依據 | 類型 | 代表疾病 |
|---|---|---|
| 水疱深度 | 表皮內（intraepidermal） | 天皰瘡、病毒性水疱、汗皰疹 |
| | 表皮下（subepidermal） | 類天皰瘡、皰疹樣皮炎、大疱性紅斑狼瘡 |
| 水疱張力 | 鬆弛性（flaccid） | 天皰瘡（Nikolsky 徵陽性） |
| | 緊張性（tense） | 類天皰瘡 |
| 分布形態 | 群聚（grouped） | 疱疹病毒感染、皰疹樣皮炎 |
| | 散在（scattered） | 水痘、藥物疹 |
| | 單側皮節（dermatomal） | 帶狀疱疹 |
| | 環狀邊緣（annular/peripheral） | 環癬合併水疱、線狀IgA病 |

### 依「最主要伴隨症狀」進一步分流

```
水疱 + 發燒/前驅症狀
    │
    ├─ 單側皮節分布 + 疼痛在先 → 帶狀疱疹
    ├─ 全身散在 + 同心圓分布（向心性離心擴散）→ 水痘
    ├─ 口腔潰瘍 + 手足水疱 → 手足口病
    ├─ 黏膜大量受損 + 標靶病灶 → SJS/TEN、多形性紅斑
    └─ 皮膚廣泛剝離 + Nikolsky徵 → SSSS（金黃色葡萄球菌燙傷樣皮膚症候群）

水疱 + 劇癢
    │
    ├─ 肘膝伸側群聚 + 腸胃症狀 → 皰疹樣皮炎（乳糜瀉）
    ├─ 老年人 + 緊張性大水疱 → 大皰性類天皰瘡
    ├─ 妊娠晚期 + 臍周開始 → 妊娠類天皰瘡
    ├─ 手掌足底深部小水疱 → 汗皰疹
    └─ 線狀排列 + 戶外活動史 → 接觸性皮膚炎（毒藤/蟲咬）

水疱 + 黏膜侵犯（口腔/眼/生殖器）
    │
    ├─ 鬆弛性水疱 + 口腔糜爛在先 → 天皰瘡
    ├─ 用藥史 + 標靶病灶 + 黏膜廣泛剝離 → SJS/TEN
    └─ 反覆發作 + 生殖器或口唇 → 單純疱疹病毒（HSV）

水疱 + 無痛/無癢，慢性病程
    │
    ├─ 糖尿病史 + 突發無痛水疱 → 糖尿病性水疱（bullosis diabeticorum）
    ├─ 肝病/光敏感 + 手背 → 遲發性皮膚紫質症（PCT）
    └─ 摩擦部位 + 近期穿新鞋/運動 → 摩擦性水疱
```

## 第三步：依伴隨症狀詳細分類表

### A. 水疱 + 發燒／全身性前驅症狀

| 疾病 | 伴隨症狀 | 病灶特徵 | 好發族群 | 治療 |
|---|---|---|---|---|
| **水痘**（Varicella） | 發燒、倦怠、頭痛先於皮疹 1–2 天 | 「露珠狀」水疱（dewdrop on rose petal），同時期出現不同階段病灶（斑疹→丘疹→水疱→結痂），向心性分布（軀幹多於四肢） | 未接種疫苗兒童、免疫不全成人 | • 兒童輕症：support care、止癢（爐甘石、抗組織胺）<br>• 免疫不全/成人/重症：**Acyclovir 800mg 5x/day PO** 或 IV acyclovir 10mg/kg q8h（重症）<br>• 避免 aspirin（Reye症候群風險）<br>• 接觸者：水痘疫苗或 VZIG（高風險者） |
| **帶狀疱疹**（Herpes zoster） | 疼痛/灼熱感先於皮疹數天、單側皮節分布 | 群聚水疱沿單一皮節，不過中線 | 中老年人、免疫抑制者 | • **Acyclovir 800mg 5x/day** 或 Valacyclovir 1000mg tid 或 Famciclovir 500mg tid，共7天，發疹72小時內開始效果最佳<br>• 疼痛控制：gabapentin、pregabalin（預防帶狀疱疹後神經痛）<br>• 眼侵犯（HZ ophthalmicus）需眼科會診<br>• 50歲以上建議施打 帶狀疱疹疫苗（Shingrix）預防 |
| **手足口病**（Hand-foot-mouth disease） | 發燒、咽痛先於皮疹 | 口腔潰瘍 + 手掌足底及臀部橢圓形水疱，周圍紅暈 | 5歲以下兒童，可群聚感染 | • 支持性治療（症狀緩解、補水）<br>• 口腔疼痛：局部麻醉漱口水、避免酸辣食物<br>• 通常 7–10 天自癒；無特效抗病毒藥<br>• Coxsackievirus A6/A16、EV71 注意重症徵兆（嗜睡、肌躍型抽搐） |
| **史蒂芬強生症候群／毒性表皮溶解症**（SJS/TEN） | 高燒、廣泛黏膜疼痛（口腔、眼、生殖器）先於皮疹 | 標靶樣病灶 → 大水疱 → 表皮剝離，Nikolsky徵陽性，受損面積 SJS<10%、TEN>30% BSA | 近期新藥史（2–8週內）：抗癲癇藥、磺胺類、allopurinol | • **立即停用致病藥物**<br>• 轉入燒燙傷中心/加護病房（類燒傷處理）<br>• 支持性治療：液體補充、感染控制、傷口照護<br>• 眼科、泌尿科會診預防黏膜沾黏<br>• IVIG／cyclosporine（爭議性但常用）<br>• Steroid 使用具爭議，早期高劑量可能有幫助 |
| **金黃色葡萄球菌燙傷樣皮膚症候群**（SSSS） | 發燒、皮膚廣泛壓痛 | 鬆弛性水疱 + 廣泛表皮剝離，Nikolsky徵陽性，**黏膜不受侵犯**（與TEN鑑別要點） | 嬰幼兒、腎功能不全成人 | • 住院、**抗葡萄球菌抗生素**（oxacillin/nafcillin 或 vancomycin若MRSA疑慮）<br>• 傷口照護如燒傷處理（避免摩擦剝離的皮膚）<br>• 液體電解質補充<br>• 不需全身性類固醇 |
| **瀰漫性淋病菌血症**（DGI） | 發燒、遊走性關節痛、肌腱炎 | 肢端少量膿疱性水疱，中央出血或壞死 | 性活躍年輕成人 | • **Ceftriaxone 1g IV/IM qd**，住院治療，同時治療披衣菌<br>• 性伴侶通知與治療 |

### B. 水疱 + 劇癢為主

| 疾病 | 伴隨症狀 | 病灶特徵 | 好發族群 | 治療 |
|---|---|---|---|---|
| **皰疹樣皮炎**（Dermatitis herpetiformis） | 劇癢（搔抓常使水疱破裂，臨床上常僅見搔抓痕）；可伴腹瀉、腹脹 | 肘伸側、膝、臀、頭皮對稱性群聚小水疱 | 乳糜瀉患者（celiac disease），多見白種人 | • **Dapsone 100–150mg/day**（48小時內止癢，需監測G6PD、CBC、MetHb）<br>• **嚴格無麩質飲食**（gluten-free diet）為根本治療，數月後可減量dapsone<br>• 篩查並治療乳糜瀉 |
| **大皰性類天皰瘡**（Bullous pemphigoid） | 劇癢，常先於水疱數週至數月出現蕁麻疹樣斑塊 | 緊張性大水疱，好發屈側、下腹、大腿內側；**口腔黏膜較少侵犯**（與天皰瘡鑑別） | 老年人（>70歲） | • 局部：**super-potent topical steroid**（clobetasol）可作為輕中度首選<br>• 全身：Prednisolone 0.5mg/kg/day，逐漸減量<br>• 免疫調節劑：doxycycline + nicotinamide（輕症）、azathioprine、mycophenolate（類固醇減量輔助）<br>• Rituximab（頑固型） |
| **妊娠類天皰瘡**（Pemphigoid gestationis） | 劇癢，妊娠中後期發作 | 始於臍周，逐漸擴散至軀幹四肢，緊張性水疱 | 懷孕婦女（妊娠中後期或產後） | • **局部及全身性類固醇**（依嚴重度），多數安全使用於妊娠<br>• 嚴重時口服 prednisolone<br>• 產後通常自行緩解，但下次懷孕易復發<br>• 新生兒可能短暫出現病灶（被動抗體傳遞） |
| **汗皰疹**（Dyshidrotic eczema） | 劇癢，夏季加重 | 手掌、手指側面深層小水疱（tapioca樣），對稱性 | 異位體質者、金屬過敏（鎳）、多汗症 | • 急性期：**高效局部類固醇**（避免長期使用於手部薄皮膚區）<br>• 冷敷、Burow's溶液濕敷收斂<br>• 嚴重反覆發作：光照治療（PUVA）、口服類固醇短期<br>• 找出並避免誘發因子（鎳、清潔劑） |
| **接觸性皮膚炎（毒藤/蟲咬）**（Allergic contact dermatitis） | 劇癢，接觸後 24–72 小時發作；線狀排列病灶常見（毒藤刷過皮膚） | 線狀或地圖狀紅斑水疱，邊界對應接觸物 | 戶外活動者、園藝工作者 | • 局部類固醇（中強效）<br>• 嚴重廣泛者：口服 prednisone 0.5–1mg/kg/day，**療程需 2–3週逐漸減量**（避免反彈）<br>• 抗組織胺控制搔癢<br>• 移除過敏原、清洗接觸部位 |
| **線狀IgA皮膚病**（Linear IgA bullous dermatosis） | 中度搔癢 | 環狀排列水疱，呈「珍珠項鍊徵」（string of pearls sign） | 兒童（慢性兒童大皰性疾病）及成人（常為藥物誘發，如vancomycin） | • **Dapsone** 為首選<br>• 找出並停用可疑藥物（成人型）<br>• 局部類固醇輔助 |

### D. 水疱 + 無痛或輕微症狀，慢性病程

| 疾病 | 伴隨症狀 | 病灶特徵 | 好發族群 | 治療 |
|---|---|---|---|---|
| **糖尿病性水疱**（Bullosis diabeticorum） | 通常**無痛無癢**，突發 | 緊張性水疱突然出現於肢端（足、小腿），清澈液體 | 長期糖尿病患者（常合併神經病變） | • 通常**自限性**，2–6週自行癒合<br>• 避免弄破水疱（保護屏障），若大則無菌抽吸<br>• 嚴格血糖控制<br>• 監測繼發感染 |
| **遲發性皮膚紫質症**（Porphyria cutanea tarda） | 輕微疼痛或無症狀，皮膚脆弱易破 | 曝光部位（手背）小水疱、糜爛，癒合後形成粟粒疹（milia）及疤痕 | C型肝炎、酒精、雌激素使用、血色素沉著症患者 | • 治療誘因：**戒酒、治療C肝、停用雌激素**<br>• 反覆放血治療（phlebotomy）降低鐵蛋白<br>• 低劑量 hydroxychloroquine（200mg biw）<br>• 嚴格防曬（UVA防護） |
| **摩擦性水疱**（Friction blister） | 局部疼痛，明確物理摩擦病史 | 單一或少數水疱於摩擦部位（足跟、足趾） | 運動員、新鞋穿著者 | • 保護性敷料，避免持續摩擦<br>• 大水疱可無菌抽吸但保留疱皮（天然敷料）<br>• 預防：合適鞋具、磨擦部位貼防護貼 |
| **昆蟲咬傷反應**（Bullous arthropod reaction） | 搔癢或疼痛，戶外活動或居家蟲害史 | 中央咬痕 + 周圍水疱或大水疱，常見小腿 | 兒童較常見（過敏反應較強） | • 局部類固醇 + 抗組織胺止癢<br>• 大水疱可抽吸減壓<br>• 預防：環境除蟲、防蚊液 |

## 第四步：危險訊號（Red Flags）— 需立即處理／轉診

| 警示徵象 | 可能診斷 | 處置 |
|---|---|---|
| 黏膜（口腔/眼/生殖器）廣泛受損 + 皮膚剝離 | SJS/TEN | **立即停藥**，轉燒燙傷中心 |
| Nikolsky徵陽性 + 發燒 + 皮膚廣泛壓痛 | SSSS、TEN | 住院，鑑別兩者（TEN有黏膜侵犯，SSSS無） |
| 眼周/鼻翼水疱（三叉神經眼支皮節） | 帶狀疱疹侵犯眼部（HZ ophthalmicus） | 立即眼科會診，避免角膜失明 |
| 新生兒/嬰兒廣泛水疱 + 發燒 | 新生兒單純疱疹病毒感染、SSSS | 緊急住院評估 |
| 免疫抑制患者 + 廣泛水痘樣皮疹 | 瀰漫性帶狀疱疹／原發水痘 | IV acyclovir，住院隔離 |
| 水疱 + 呼吸道症狀 + 結膜炎 | 嚴重多形性紅斑、SJS前驅期 | 密切觀察黏膜進展 |

## 第五步：基礎診斷工具對照

| 工具 | 用途 |
|---|---|
| **Tzanck smear** | 快速床邊檢測，見多核巨細胞 → 提示 HSV/VZV（無法區分兩者） |
| **病毒培養/PCR**（水疱液） | 確診 HSV、VZV，可區分型別 |
| **Nikolsky徵測試** | 鬆弛性水疱（天皰瘡、SSSS、TEN）陽性；緊張性水疱（類天皰瘡）陰性 |
| **直接免疫螢光（DIF）** | 天皰瘡：細胞間 IgG（魚網狀）；類天皰瘡：基底膜帶狀 IgG/C3；皰疹樣皮炎：真皮乳突顆粒狀 IgA |
| **皮膚活檢（病理）** | 確認水疱深度（表皮內 vs 表皮下）、發炎細胞種類 |
| **血清自體抗體** | Anti-desmoglein 1/3（天皰瘡）；Anti-BP180/BP230（類天皰瘡） |
| **抗 tTG IgA** | 篩查皰疹樣皮炎相關乳糜瀉 |

## 速查摘要表

| 主要伴隨症狀 | 優先考慮 | 第一線檢查 | 第一線治療思路 |
|---|---|---|---|
| 發燒 + 單側皮節 | 帶狀疱疹 | 臨床診斷／Tzanck/PCR | Acyclovir/Valacyclovir |
| 發燒 + 全身散在 | 水痘 | 臨床診斷 | 支持治療 ± Acyclovir |
| 發燒 + 黏膜廣泛剝離 | SJS/TEN | 用藥史、皮膚活檢 | 停藥 + 燒傷中心照護 |
| 劇癢 + 肘膝對稱 | 皰疹樣皮炎 | DIF、anti-tTG | Dapsone + 無麩質飲食 |
| 劇癢 + 老年緊張性大水疱 | 大皰性類天皰瘡 | DIF（BMZ線狀） | 局部/全身類固醇 |
| 黏膜先發 + 鬆弛性水疱 | 天皰瘡 | DIF（細胞間IgG） | 全身類固醇 + 免疫調節 |
| 無痛 + 糖尿病史 | 糖尿病性水疱 | 臨床診斷 | 觀察自癒 |
| 摩擦/咬傷病史明確 | 物理性水疱 | 臨床診斷 | 保護性處置 |

::: warning 臨床警示
- **Nikolsky徵陽性 + 黏膜侵犯**：優先排除 SJS/TEN 與天皰瘡，這是皮膚科急症
- **老年人新發搔癢性蕁麻疹樣病灶**：即使尚未出現典型水疱，也應懷疑早期大皰性類天皰瘡（pre-bullous phase）
- **任何不明原因水疱合併新藥使用史**：詳細用藥時間軸（過去 8 週內）為關鍵
:::
